绝地武士训练

名称：绝地武士训练
本质：特异
分类：可升级血统
概述：绝地（Jedi）武士起源于最初发现原力的一群科学家、哲学家、历史学家构成的学术团体。他们最先发现并开始研究原力，为了避免原力被心怀不轨者所滥用并带来灾难，他们选择建立起一个严格遵守戒律、为维护和平并引导人类正确使用原力的特殊组织。他们是信仰和遵从原力光明面的原力使用者，在银河历史上，绝地武士团一直为共和国所服务，在漫长的岁月里，绝地武士一直担任着保卫银河系和平的正义卫士。
绝地武士常常会被派遣前往执行最困难的任务（也被称作绝地任务），但绝地武士们往往会以最完美的方式将任务完成。平民听见“绝地事物”四字，他们总是心安；而罪犯则会知道自己已经无路可走。他们也是最优秀的谈判专家，当议会人员无法处理一些棘手的外交事务时，绝地委员会将派遣绝地武士前往谈判，凭借优秀的外交技巧和绝地之名，哪怕是再利欲熏心的野心家也会退让——即使真有动刀弄枪的麻烦人物，绝地武士也会以原力与高超的光剑剑技迅速平定暴力。
绝地武士并非单纯的战士，他们也需要学习凡世的知识，从驾驶星际战机到混迹凡夫俗子间参与耕作、从钻研高深的科学理论到磨练剑术技艺，他们都必须精通与熟练。
在绝地武士悠长的历史中，他们并非总是作为一个武装组织而出现，其恪守的戒律令他们把原力应用在了更多元、非战争的正义之途。为了避免拥有原力的绝地武士相互结合形成一个以血缘关系为纽带的超凡阶级，绝地武士在立下誓约后常常避免结婚生子。

总纲：
当你强化本训练，你需要遵守绝地武士戒律。
当人物不遵循绝地武士戒律时，人物将无法使用原力技能树下分类为【光明】的技艺，同时人物的光剑颜色会变为倾向红色调的紫色或者粉色。
当人物违背了绝地武士戒律，本训练所提供的好处不再生效。且可能导致原力的性质变为中立甚至黑暗。
注：ST拥有对绝地武士是否遵循、违背戒律的裁断权。

绝地武士戒律总则：
There is no emotion, there is peace.
无需激情，平静心智。
There is no ignorance, there is knowledge.
勿随愚昧，顺从真知。
There is no passion, there is serenity.
勿纵情欲，沉静明意。
There is no chaos, there is harmony.
虽有混沌，安谧仍存。
There is no death, there is the Force.
无有灭亡，唯行原力

以下是对前四条的通俗解释，也是人物需要遵循的戒律细述：
1、人物将自身从情感中解脱而出，这意味着：人物不能为一时的激愤所进行暴力杀戮行为，即使这可能是正当的；人物需要以平和的心态去面对眼前事物，即使是常人眼中难以接受的。
2、人物在任何时刻都需要保持一颗谦逊求知的心，并远离愚昧盲从。即使是在最困难的环境中，也不可放弃对知识的探求，人物在每部影片中都需要学习并开启学识、生存、器用、手艺四项技能下至少一项的尚未拥有的专业。
3、人物不得娶妻生子，也不可因为个人利益与情感驱使而进行杀戮、抢夺、欺诈等伤害他人的恶意行为。
4、人物在影片内面对混乱的局势时，应该尽自己所能去令局势往更和平或缓和的方向改变。（注：绝地需要根据自身实力水平选择性的遵循这一条戒律：或许一名学徒难以改变整个世界的局势，但能令一个村落暂时的脱离战火袭扰，这不算违背戒律。一名绝地大师可以强行停止两个国家的战争，却选择困地自保不愿涉足险境，则是明显违背了戒律。）

D级
等级名称：绝地幼童（Youngling）
价格：D+600
特性：【冥思】【原力亲和】【原力】
描述：绝地武士们发现表现出极强原力天赋的幼童体内细胞的纤原体的含量远高于常人，绝地武士常常会将这些具有天赋的幼童在父母同意的情况下带回绝地圣殿进行培养。幼童们会在绝地圣殿经受严苛的思维训练，在令他们远离情绪影响的同时培养他们对原力的亲和性。
效果：
【原力】
你开启能量池：原力（属性为光明）。

【冥思】
你可以在长休息时进行一个不计算其他属性加值的沉着检定将自身的精神状态协调稳定，该沉着检定被称之为冥思检定，接着你将这次检定的出目记录下来，这些成功数被称之为冥思。在这次长休息结束后直到下一次长休息期间，若冥思没有消耗完毕，亦不会积累。多次进行冥思检定，只会记录最后一次检定的出目。
当你进行沉着、操控、感知属性检定时你可以消耗任意点冥思以获取等额的修行加值。除此之外，当你承受情绪异常点数时，你同样可以消耗任意点冥思并减少等额的情绪异常点数。

【原力亲和】
每当你获得一次冥思，你的原力能量池上限就会增加1点。每场影片中，因此而增加的原力能量池上限不能超过你感知、沉着、操控三者属性上附加成功的总和。

C级
等级名称：绝地学徒（Padawan）
价格：C+1200
特性：【提升—冥思】【原力追随者】【光剑共鸣】【战型兼习】【博识】
描述：当幼童们经过数年的思维训练、知识学习以后，绝地圣殿会派遣一名绝地大师带领并着手训练一名绝地幼童，此时幼童也从而晋升为绝地学徒。绝地学徒们往往在头部右侧扎有发辫以作为学徒的标志，若没有头发，也会佩戴些许标志表明。
【博识】：绝地武士并非他人刻板印象中的只知晓挥舞光剑的战士，他们更是各领域的专家，武力对于绝地武士而言只是保障必要安全的工具，一名绝地武士对宇宙理解的越是深刻，其便越加靠近原力。
效果：
【提升—冥思】
你现在可以在短休息时进行冥思检定以获得相应的冥思。多次进行的冥思检定，只会记录最后一次检定的出目。当你需要进行意志豁免时，你可以消耗任意点冥思以在意志豁免上获取双倍消耗冥思的修行加值 。

【原力追随者】
只要人物不违背绝地武士戒律，人物在公开自身绝地武士的身份以后，非敌对势力与人物的初始态度上升一级（至多为友善），且获得10点对该势力的临时声望。当人物履行了绝地武士戒律并完成一方势力的委托，则人物的事迹会在影片世界中流传，此时这些临时声望会变为人物的永久声望。一场影片中你至多从该特性中获取10点永久声望。
（注：对于不了解绝地武士的势力可能需要进行一定的社交检定来表明绝地武士的立场）

【光剑共鸣】
当你使用光剑时，你可以额外支付任意点原力，令光剑的武器伤害提升你所支付的原力点，提升的武器伤害持续一轮。额外支付的原力至多等同于感知、沉着、操控三者中最低者。

【博识】
计算人物在学识、手艺、器用、生存技能下不同种类专业的数目（不计算专业等级），若这四项技能下的各专业数目均相等，则人物在这四项技能检定上获得等同于专业种数的修行加值。人物的原力能量池上限增加专业种数*3点。计算本特性的专业种类数量时，数量上限不能超过这四项技能较高者的附加成功。
（如：人物在学识下拥有3种专业、在手艺下拥有3种专业、在器用下拥有3种专业、在生存下拥有3种专业，由于专业种数相等，人物在这四项技能检定上获得3点修行加值，且原力上限增加9点。）

【战型兼习】
你在使用等级为【基要】的光剑战型时，你每具有除当前使用战型以外的两式其他等级为【基要】的战型，就能在这次攻击中结合一式其他战型的【基要】效果。但是在一轮内，你不能重复使用同一种战型或者以此效果令一式已经以此特性生效的战型多次生效。

B级
等级名称：绝地武士（Knight）
价格：B+2400
特性：【提升—冥思】【原力掌握】【授业】【危机反射】【战型融会】
描述：从学徒进阶为武士并非易事，即使有大师的指导，学徒也需要完成银河中最困难的任务才能被圣殿授予武士的头衔。从学徒进阶为武士并非刻苦修行的结束，而是另一场艰难试炼的开始。年轻的武士，愿原力与你同在。
效果：
【冥思】
现在你可以将自身的感知属性加入冥思检定，同样不计算其他的属性加值。当你需要进行反射豁免时，你可以消耗任意点冥思以在反射豁免上获取双倍消耗冥思的修行加值。

【授业】
成为绝地武士以后，你可以着手于在各个世界传导原力之道。在影片中，你可以向关系为友善的单位或组织传述原力的理论，以此引导具有原力天赋的人进入绝地武士的修习之途，这需要花费十天的时间，这期间被传授知识者需要凭借自己的悟性对原力进行领悟，具体流程为：被传授者在传授期间，每日可进行一次感知检定，将成功数记录下来，若成功数总和达到自身的心智属性与沉着属性之和，则其开启原力能量池。若其血统与训练模板未被占据，他还能以对应等级的经验修习绝地武士训练。

【原力掌握】
你在运用消耗原力的能力时，可减少任意点能量消耗（至少为1），但每轮合计减少的能量消耗数不能超过【感知+沉着+操控】属性上的附加成功点。

【危机反射】
多次绝地任务所磨砺的素养令绝地武士在遭遇险境时能从容应对。角色免疫A级先制，每轮获得2点只能用于转化反射动作的AP。

【战型融会】
你已经将光剑的各式技艺习练并熟练掌握，这令绝地武士发展出属于自身的独特战斗风格。你在使用等级为【精通】的光剑战型时，你每具有除当前使用战型以外的两式其他等级为【精通】的战型，就能在这次攻击中结合一式其他战型的【精通】效果。但是在一轮内，你不能重复使用同一种战型或者以此效果令一式已经以此特性生效的战型多次生效。

A级
等级名称：绝地大师（Master）
价格：A+4800
前提：这一级别无法直接购买，只有当绝地武士成功培养出另一名绝地武士后，人物才能购买本级别训练。 （即你需要以【授业】特性令一名其他角色获得绝地武士训练且该单位的训练等级提升至B级）
特性：【提升—冥思】【原力代行者】【绝地英灵】【光剑大师】
描述：从本质而言，大师与其他绝地武士并无不同，他们遵从原力的指引，从一而终。
效果：
【冥思】
你现在可以将操控属性加入冥思检定，同样不计算其他属性加值。每轮一次，你消耗冥思以获取修行加值时，可以将成功数提升一半。

【原力代行者】
绝地武士在银河各势力间享有盛名，即使是兵刃相见的敌人，也常对绝地大师赞赏有加。只要你不违背绝地武士戒律，你在公开自身绝地武士的身份以后，你在影片内的听闻你事迹的各方势力中均获得25点临时声望，这不会改变你与他们之间的初始态度。影片结束时，若你履行了绝地武士戒律并完成各势力的委托，则你可以将临时声望转变为永久声望，每部影片，你至多从该特性中获得50点永久声望。

【光剑大师】
现在你获得8点AP以使用光剑七式。你在使用等级为【掌握】的光剑战型时，你每具有除当前使用战型以外的两式其他等级为【掌握】的战型，就能在这次攻击中结合一式其他战型的【掌握】效果。但是在一轮内，你不能重复使用同一种战型或者以此效果令一式已经以此特性生效的战型多次生效。

【绝地英灵】
无有灭亡，唯行原力。对于大师而言，死亡并不意味着一切的结束，当你死亡时，你从肉体的拘束中解脱，这也令你前所未有的靠近原力，你的灵魂会不可避免的脱离身躯并转换为灵体，由于形体尚未稳固，此时你无法使用任何能力，也不能主动进行攻击行为，这是A级别环境来源的复活效果。与一般灵体不同的是，具有原力的单位可以正常看见你并与你进行沟通。当你化为灵体一周以后，你可以正常使用原力技巧，但因不具有肉体，你无法使用任何需要肉体的能力。转变为灵体是永久性的，而即使你在之后的生涯中通过其他方式找回了肉身，本特性的复活效果也不会多次生效。
