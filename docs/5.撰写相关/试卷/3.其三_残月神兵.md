# 其三：残月神兵

  

残月神兵  
拉露恩制作了五把枪型皎月兵器，使用月光作为驱动能量与弹药。当获得这些皎月兵器时，通常情况下只会展示出其中一把，以通碧、断魄、坠明、荧焰、折镜的顺序供给，处于队列最前端的两把枪视为符合一个攻击组合时的两把枪，但在单纯进行射击时只计算手中的枪械拥有的特性和伤害。每当一把残月神兵中带有的子弹完全用光时就会自动进入武器队列的最末端中并切换至顺位的下一把残月神兵。因此，残月神兵不能进行无限子弹改造，也不能获得超过附魔上限的附魔。但一把残月神兵获得的附魔和精制品等强化会同时作用于所有其他残月神兵上。  
在通常意义上，主手枪械和副手枪械是指排在枪械队列最前端的两把枪械。每把残月神兵都会拥有一个独特的被动，它们都需使用枪械：残月神兵专业进行检定。

传知者与真知者  
D+1500  
在使用残月神兵时，无论其实际上为何种武器，统一使用射击-专业：残月神兵进行检定。  
通碧在枪型上为狙击枪  
断魄在枪型上为手枪  
坠明在枪型上为加农炮  
荧焰在枪型上为喷火器  
折镜在枪型上为飞轮刃

月相轮转I  
前提：传知者与真知者  
C+2500  
购买后解锁五把残月神兵，在此阶段下，残月神兵会占据拥有者至多3点意志上限

通碧  
18L  
射程400米 初始子弹数10  
这是一把通体绿色，流转着绿色月光的长型枪械，或许可以被称之为狙击枪。它在一般情况下拥有6点高速及5点破甲  
通碧在通常情况下不能附带招式进行攻击，以通碧进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，若如此做，则此时进行的攻击拥有【歼敌】特性，此次攻击伤害若大于被攻击者体积，则会附带伤害数减对方体积的流血点数，这些流血点数使用标准豁免进行减免。  
当通碧进行一次射击且命中任意敌人时会将其标记，在下一次对拥有标记的敌人攻击时可以使用副手枪械对该敌人进行一次攻击并去除其拥有的标记，前提为在副手枪械射程范围内。当通碧为副手枪械时这个效果会变为使用主手枪械。特殊的，当通碧进行一次攻击后，攻击者可以在回合外消耗一个反射动作预支下回合的标准动作使用副手枪械对那个敌人发起攻击。

  
断魄  
11L  
射程70米 初始子弹数10  
这是一把通体红色，散发出红色血光的短型弯刀，或许可以被称之为手枪。它在一般情况下拥有【能量武器】特性  
断魄在通常情况下不能附带招式进行攻击，以断魄进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，若如此做，则此时进行的攻击拥有【吸血】特性，此次攻击会为攻击者回复3L/1A（医疗点32）

  
坠明  
射程500米 初始炮弹数10  
这是一把通体紫黑色，逸散出紫黑色暗光的手提式加农炮，尾端禁锢着一个充满暗能量的弹囊，弹囊中储藏有至多十次炮击的能量。它在一般情况下拥有3点高速和2点破甲。并拥有【自动制导】特性，并使用44+6DP的预设值进行攻击。坠明的力量需求是5  
坠明在通常情况下不能附带招式进行攻击，以坠明进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，若如此做，则此时进行的攻击拥有【衰败】特性，此次攻击伤害若大于被攻击者体积，则会附带伤害数减对方体积的衰败点数，衰败是一个影响人物自身身体，使人物的身体失去行动能力的效果。衰败中的生物在运动检定、攻击检定、基本速度、基础防御+闪避防御+格挡防御的总和上受到减值（按照格挡防御、闪避防御、基础防御的顺序进行减少），这个异常点数使用意志豁免进行对抗。当进入毁灭时视为进入定身效果。特殊的，被坠明攻击的敌人将会降低20点基础移动速度持续一轮（最多降低至1点）

  
荧焰  
8L2A  
射程100米 初始子弹数10  
这是一把通体蓝色，闪烁着蓝色幽光的长型枪械，尾端有一颗蓝色的暗晶，暗晶中储藏有至多十次喷射的能量，或许可以称之为喷火器。荧焰的力量需求是4  
荧焰在通常情况下不能附带招式进行攻击，以荧焰进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，若如此做，则此时进行的攻击拥有【燃烧】特性，此次攻击将会对攻击范围内第一个碰到的角色造成等同于武器伤害的固定伤害，使用反射豁免进行对抗。紧接着，荧焰的伤害会传递给离该单位最近的三个在射程之内的单位，这些单位在进行反射豁免时视为降低10点能量减值进行检定

  
折镜  
12L  
射程70米 初始子弹数10  
这是一把通体白色，照射出白色寒光的半轮短刃，除了中心的环状握把外，上下各分布着半轮状刀刃。通常情况下，折镜只能发射特殊的飞轮状弹药。它在一般情况下拥有6点高速  
折镜在通常情况下不能附带招式进行攻击，以折镜进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，若如此做，则此时进行的攻击发射的飞轮在击中敌人后会造成被攻击者视为已经被攻击过一次。特殊的，当折镜进行连射攻击时，每个飞轮都会视为进行过一次攻击（最高为6）。被发射出的飞轮在那次攻击结束时会盘旋飞回攻击者身边进行盘旋，攻击者每次以折镜进行攻击时都还会额外获得等同于盘旋的飞轮数的DP（最高为6）

武器技能

所有的武器技能都分别对应着一把武器，只有在使用此武器的前提下才能使用对应的武器技能，除还魂驻灵及清辉夜凝外，所有技能的价格为C+1000。还魂驻灵的价格为B+4500，清辉夜凝的价格为B+5000。  
使用任何除清辉夜凝外的武器技能时都会消耗该枪械现有的一半数量的子弹（向上取整）。特殊的，清辉夜凝只需要购买一次就可以获得所有枪械的使用效果

还魂月闪（通碧）  
此招式需结合在仅以通碧进行的一次攻击中。对射击范围内的一个目标进行一次单体攻击，在表现上视为射出一道翠绿色能量，在击中敌人后会削减其最多等同于当前使用者最高支线数一半+5的基础防御（向上取整），这个效果将持续存在一轮。在此次攻击命中后会为被击中者增加一个标记

擘分对影（断魄）  
此招式需结合在仅以断魄进行的一次攻击中。此次攻击将改为以断魄攻击以自身为中心，35米半径范围内的所有敌人，此次攻击额外获得8点医疗点（上限为16，累计使用），所有在此次攻击中需扣除的医疗点优先从额外获得的部分中扣除

地霜暗蚀（坠明）  
此招式需结合在仅以坠明进行的一次攻击中。此次攻击将额外削减所有在此次攻击中被【衰败】影响的角色40点移动速度（最多降低至1点）

地霜暝涌（荧焰）  
此招式需结合在仅以荧焰进行的一次攻击中。此次攻击结束时，记录所有被【燃烧】影响的角色，在持有者的回合外时可以使用一个反射动作以调用下个回合的标准动作，使用副手武器对所有已被记录的角色进行一次攻击。但只能攻击到副手武器射程范围一半之内的角色

还魂驻灵（折镜）  
此招式需单独使用，前提为持有者的主手枪械必须为折镜。使用一个标准动作，立即在折镜攻击范围内的地面上部署一个月之驻灵，它本身拥有持有者此时副手枪械的所有效果和特性，拥有【自动制导】，并使用的40+3的预设值进行自动攻击，这个驻灵自身拥有30点结构值及40基础防御，被击破或经过一个场景后自动消失。特殊的，驻灵会占据使用者此时的副手枪械，攻击时自动消耗子弹，虽然无法使用但可以更换当前副手枪械。特殊的，还魂驻灵不会为购买者提供属性点

清辉夜凝  
特殊的，清辉夜凝不会为购买者提供属性点，它在通常情况下会以一道淡淡的蓝色魂影萦绕在购买者的身后，并为购买者提供下一把即将到来的枪械（在未购买前都会直接从武器队列中供给）  
清辉夜凝本身会为所有枪械提供一种独特的技能效果，在使用清辉夜凝时必须结合对应的主手武器进行的一次攻击而使用。  
使用一个标准动作进行释放，对一个射程70米内的目标释放，持有者的枪械中射出一道浓缩着月光的爆裂能量，在击中第一个敌人后立即爆炸，对以其为中心35米半径内的所有敌人造成一次相同的爆裂，并使用主手武器立即对所有命中的敌人再进行一次攻击
