# 命运的撰写笔记

  

结合TMD:新秩序008  
撰写相关-基础、动作、限制与能量 使用  
在此之前需先了解模板和特性位的相关内容 至少要知道技艺面板

最开始容易出现的误区是同级技艺面板（例如C级技艺模板写作6+6）和同级特性（例如C级特性写作6）的混淆

拥有多个同级特性的模板在计算基础指标时仍然只计算同级两特的数量（即技艺模板）  
例：B级的血统模板 提供3个B级特性（也就是B+B+B，换算12+12+12），但是指标只计算其中的同特两级（12+12）

1D特=3DP 每1D特中有1指标  
同时又因为1高一级支线特性=2低一级支线特性  
（注：此处仅为高级特性降级，非次级特性升级的换算）  
2同级支线特性+每2个D级特性\*2DP=1高一级支线特性 可以理解为  
2同级特性+2同级特性指标之和的DP= 1高一级支线特性

C特=2D特=2x1指标=2指标  
以此推算出  
1C级特性中有2指标  
1B特性中有4指标  
1A特性中有8指标  
1S特性有16指标

由此可得支线等级指标等于2\*上一支线等级指标

向下内容中的“但此价格为连续购买价格，当一个内容中想临时切换或获得其他仅此等级模板内容时，需额外再支付2DP”  
例如：你准备撰写一把武器，它平时是近战的冷武器，但是也可以切换成热武器，那么你需要先行购买一把近战冷武器的模板，之后再购买热武器的面板，最后额外支付2DP，切换武器动作默认为掏出武器。

未提供属性的模板与指标之和会在每个等级提供一个同级同特（不包括未成特性的部分），提供的同级特性不能获得权值  
其中“未成特性”指额外加钱买的特性和后续提到的负特换取的。（也称之为散DP，加钱每250分可购买1条无支线特性，也就是1dp，上限为：D3/C6/B12/A24/S48）

判断一个资源能否获得比其当前级别更高的特性的判断方式为指标DP数是否达到该等级额定数量。  
（如D级特性升C级特性需满足C级技艺模板6DP+6DP，也就是4指标）

  
动作权值部分——

从移动动作（包括移动动作）开始，每个动作的消耗都会额外视为具有动作倍率\*逐级合成税DP的指标【又写作动作升指标×动作权值，这是目前唯一算式中可以提升指标的部分】  
同时扣除当前等级（未升级前支线等级）的税值（不计入指标），多个动作累加使用时只提供最高倍率动作的指标，同时只扣除一次。完全消耗动作在权值基础上提升一倍，【但不提升指标】动作权值乘数得到的DP可自动向上升至多一级。（注：动作权值乘数得到的DP可自动向上升至多一级 这句话在规则书中并无意义，如果硬要理解，这句话是由于早期规则较为混乱未能及时删除，后确立结果只有dp和指数，所以这句话失去了意义）  
注：D级也需交2DP税 C级4DP税 B级8DP 以此类推...... 此税值已处于公式计算中 无需特意记录 这些税值位于（动作升指标）中

以上的不完全消耗动作和完全消耗动作的概念（不完全消耗动作按照正常权值倍率）  
使用标准动作进行一次攻击，并且此次标准动作用于换取动作权值，此为不完全消耗动作。  
使用标准动作仅换取权值，期间无标准动作用于其他行为，此为完全消耗动作。

限制权值：  
此部分建议累计或询问知情人士，如若没有笔记或者明确写明，请勿擅自决定倍率  
其中消耗品部分，对于仅生效几次的特性，也是视作消耗品。

“难权不会自动升税，但会自动权值升级”  
此处意为，限制权限的倍率可以升级支线等级，但是不提供指标。（但是由于后续版本公式只存在dp和指标，所以这句话中仅有不提供指标这一部分规则是正确的）

时间权值：  
无笔记

能量：  
“若超出了这部分支付上限，必须写成单独的增幅方式。”  
此处意为，若是要超出D2C6B10A14S18的能量提供等数量DP的部分，需要额外购买增幅上限（据说这特性还没写上规则书，我不好说了，但是已经有了限制规则，位于TMD：新秩序009-资源分区-魔法-法术总纲中的法术的耗能、威力、增幅部分 无论是后续出现的特性dp购买，还是由法术精熟获取的 增幅上限 都需要总计增幅上限后遵守D2C6B10A14S18的增幅上限规则）  
但仍旧存在提升增幅上限的办法，在TMD：新秩序009-资源分区-魔法-法术总纲 中的 法术精熟中描绘了如何获取的额外增幅上限，但限制较大，简略描述

首先，你需要拥有一个法术派系（一般而言，法术会被统筹归入六大类派系之中，这些派系分别是：核心、自然、物理、黑暗、咒击、万变。使用一个法术时，需先学习过对应的派系，且派系造诣等级达到或超过该法术等级才能正常使用。  
所有种类派系必须从最低级开始掌握，并逐级进行掌握。派系造诣等级分五个等级，分别为D/C/B/A/S级，派系造诣一般由能力/法术书/魔导书赋予。）

其次你需要在影片中释放过该法术攻击敌人才可以获得计数。（此规则位于在TMD：新秩序009-资源分区-魔法-法术总纲 中的 法术精熟部分）  
初级熟练（一场影片）：使用这道法术时获得1点施法上的增强加值，并至多增加1点增幅上限。  
中级熟练（两场影片）：使用这道法术时再获得1点施法上的增强加值，并至多增加1点增幅上限。  
高级熟练（三场影片）：使用这道法术时再获得1点施法上的增强加值，并至多增加1点增幅上限。  
传奇熟练（四场影片）：使用这道法术时再获得1点施法上的增强加值，并至多增加1点增幅上限。  
注意，这里要求是对于释放过的法术才能享受，这对于撰写未存在的攻击法术限制十分明显。

  
负特：  
无笔记

公式：

基础DP   
+（基础DP×动作权值-【动作升指标×动作权值】）/时间权值  
+基础DP×限制权值（※）÷时间权值  
+能量DP/时间权值  
\=总DP

注：若动作权值低于移动动作则动作升指标代入为0  
其中：基础DP为代入的DP，可以是C级技艺模板（6+6），可以是C支线特性（6），可以是使用了部分dp的C级记忆模板（6+4），或者是散dp  
基础1B特 负特12 加钱12都可以代入

公式中仅存在着：dp和指标

※：“难权不会自动升税，但会自动权值升级”  
（限制权限中难以上倍率是1以及朝上，可以倍率后升级特性等级，但是不提供指标，但是公式中特性都变成了DP，同时限制部分本身就不提供指标，我真的不好说了。  
若是不能理解，可以理解为这句话尚未被删除，在规则书中毫无作用）

【动作升指标×动作权值】是唯一可以提升指标的部分，得出提升的指标数和模板指标相加就是当前的指标数。

动作升指标：（D2/C4/B8/A16/S32）原模板中同等级技艺模板的指标。

公式范例：  
代入基础DP为C级技艺模板（6+6）  
基础DP（6+6）（括号中的数字为分乘）  
+（基础DP（6+6）x 动作权值（不完全标准动作 1）-动作升指标（2+2）x动作权值1）=（6+6-4）=8DP

+基础DP（6+6）x 限制权值（难权2倍）÷时间权值（一轮 1）=（6+6）x （2+2）÷ 1 =（12+12）=24 DP

+能量DP(无）÷时间权值（一轮1）=0DP  
\=（6+6）+8+24 =44DP

指标：基础C技艺模板（6+6）  
因为C特=2D特=2x1指标=2指标  
所以C技艺模板=（C特+C特）=（6DP+6DP）=（2指标+2指标）=4指标  
又因为此处使用的是不完全动作，可以提升指标  
【动作升指标×动作权值】=（6DP+6DP）x1=（2指标+2指标）x1=(2指标+2指标）=4指标  
C技艺模板4指标+升指标4=8指标

最终得44DP 8指标

范例2：  
代入基础DP为C级技艺模板（6+3）（其中第二个C级特性被拆成两个D级特性并且使用完）  
基础DP（6+3）  
+（基础DP（6+3））x动作权值（不完全标准动作1）-动作升指标（2+2）x动作权值1=（6+3-4）=5DP  
+基础DP（6+3）x限制权值（难权2倍）÷时间权值（一轮1）=（6+3）x（2+2）÷1=（12+6）=18DP  
+能量DP（无）÷时间权值（一轮1）=0DP  
\=（6+3）+5+18=32DP

指标：基础C技艺模板（6+6） （即使只是代入部分，依旧享受全体升指标）  
因为C特=2D特=2x1指标=2指标  
所以C技艺模板=（C特+C特）=（6DP+6DP）=（2指标+2指标）=4指标  
又因为此处使用的是不完全动作，可以提升指标  
【动作升指标×动作权值】=（6DP+6DP）x1=（2指标+2指标）x1=(2指标+2指标）=4指标  
C技艺模板4指标+升指标4=8指标

由此得知 公式得到的额外指标实际上只与动作权值且要求不完全使用有关  
注：多个动作累加使用时只提供最高倍率动作的指标，所以一个模板的指标上限就是模板指标x不完全整轮动作（1.5），也就是模板指标的2.5倍

结合TMD:新秩序008  
撰写相关-DP转换 使用（注，DP转换中的特性为散特性且部分收税，通用特性表中的特性为支线等级特性，无需收税）

在DP转换过程中，每有3DP被用于同种散特性可购买的能力中（包括攻击DP，防御等）就需抽走1DP税  
每有1个同级特性被用于同种散特性可购买的能力中（包括全吸收、生命等），就需抽走1DP税。  
特殊的是购买附加成功、伤害（包括瞄准上限）、异常点、包含含税字样、属性等级、技能等级无需抽税。

其中“在DP转换过程中，每有3DP被用于同种散特性可购买的能力中（包括攻击DP，防御等）就需抽走1DP税”  
指散DP每重复购买3DP散特性就需要收1DP税，在购买下次3DP前要求先交这1DP（例如，1DP=2先攻，购买3次2先攻，花费3DP，购买第四次需要交1税，再购买第四次，3+1+1）  
其中“每有1个同级特性被用于同种散特性可购买的能力中（包括全吸收、生命等），就需抽走1DP税。”  
这个仅针对DP转换中价格为3DP的部分，并且不太好说。  
（例：D级特性是3DP，购买1次提升1.5生命上限需交1税。  
         C级特性是6DP，购买2次提升1.5生命上限需交1税。以此类推）
