# 能量池

  

获得能量池：根据决定大小的关键属性不同，价格不同。  
归属主属性：力量、智力、敏捷  
归属副属性：感知、决心、耐力、沉着  
归属通常属性：风度、操控  
这种分类方式仅在获得能量池时使用。  
通常属性+通常属性、通常属性+副属性：2dp  
副属性+副属性、主属性+通常属性：2DP  
主属性+副属性：3DP2指标  
主属性+主属性、某种属性的两倍：4DP2指标  
强化能量池（属性无限制）：6DP2指标，在具有能量池的基础上将能量池升级为强化能量池4dp2指标
