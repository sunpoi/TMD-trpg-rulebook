# 特性表-多等级

特性表-多等级

如无特殊说明，多等级特性只需具有对应等级整特性便可购买。  
如无特殊说明，多等级特性在本表中标注写法为：  
【XX】特性名 Y+  
Y为购买该多等级特性的最低级，Y+代表有后续等级。

免疫某个效果：但免疫能力一般也具有自己的基础生效等级，高于这个等级的效果无法免疫，例如免疫目盲的能力至少是D级（因为目盲效果的基础等级是C级），但是无法免疫B级的目盲效果。  
免疫范围更小的则为低1或更多级的单特性，例如免疫措手不及、免疫困难地形。  
免疫某来源的效果根据来源出现的频率分为二种价格  
低一级单特性：变形、附身、诅咒/祝福、毒素/药物、信息、疾病  
同级别特性的一半（即如果在低级别购买高级别的免疫，需提升指标）：创伤、影响心灵、环境、时空、命运、幻象  
免疫单一能量（魔力、内力、灵力等）的负面效果：同级别半特  
免疫某种具体的效果（例如免疫时间停止、衰老、即死等）可以被认为为低二级单特性，详见下方的效果-免疫栏；  
免疫范围更大的则为高1或更多级的单特性，例如免疫心智/身知/精魄类能量驱动的效果影响为高1级单特性，免疫所有能量则为6类能量免疫的价格之和。  
免疫某种特定能量驱动的效果影响为同级特性的一半（例如免疫魔力驱动的负面效果）

免疫第一档异常点带来的减值及严重状态：3DP2指标  
免疫第二档异常点带来的减值及严重状态：6DP4指标  
异常效果替换某种异常点的严重效果，或者替代其毁灭效果，并像毁灭效果一般无法永久持续且无法免疫。（异常效果均按持续时间均为永久计算，因此异常效果部分无法获得动作权值带来的DP）

变形D+（免疫变形价格低1级）  
D级：在撰写时指定一个特殊形态（例如老虎、狮子），在效果持续时间内，单位变化为目标形态，变化后单位的种族与目标形态相同，此形态体积可以相较于原本人物体积放大或者减少至多2点，这部分体积不会带来生命值变化。变化的体积需要在撰写时指定为增大或者缩小。  
C级：在撰写时指定一个特殊形态，在效果持续时间内，单位变化为目标形态，变化后单位的种族与目标形态相同，此形态体积可以相较于原本人物体积放大或者减少至多10点，这部分体积不会带来生命值变化。变化的体积可以在使用时指定为增大或者缩小。  
B级：在撰写时指定至多3个特殊形态，在效果持续时间内，单位变化为目标形态，变化后单位的种族与目标形态相同，此形态体积可以相较于原本人物体积放大或者减少至多20点，这部分体积不会带来生命值变化。变化的体积可以在使用时指定为增大或者缩小。  
A级：在撰写时指定至多5个特殊形态，在效果持续时间内，单位变化为目标形态，变化后单位的种族与目标形态相同，此形态体积可以相较于原本人物体积放大或者减少至多80点，这部分体积不会带来生命值变化。变化的体积可以在使用时指定为增大或者缩小。  
S级：在撰写时指定至多10（或者更多只要你写得出来）个特殊形态，在效果持续时间内，单位变化为目标形态，变化后单位的种族与目标形态相同，此形态体积可以相较于原本人物体积放大或者减少至多320点，这部分体积不会带来生命值变化。变化的体积可以在使用时指定为增大或者缩小。

旅行D+（旅行不能免疫，因为其不能对非友方角色使用）  
这是一个非战斗时生效的特性，其无法使用动作权值购买。  
D级：城市级旅行 这一级别旅行可以令使用者在一小时内从城市的一端至另一端，也可以在一小时内从一座城市抵达另一座相邻城市。  
C级：大陆内旅行 这一级别旅行可以令使用者在一小时至一日内从大陆的一端至另一端。这一级别的旅行速度等效于500公里的时速。  
B级：大陆间旅行 这一级别的旅行可以令使用者在一小时内横穿整个大陆，在一日内从星球的一端抵达另一端，甚至在条件允许的情况下进行地月系间的通行。这一级别的旅行速度等效于2000-8000公里一小时。  
A级：位面内旅行，这一级别的旅行可以令使用者在具有坐标系的情况下一小时抵达影片世界内的任意一点。  
旅行特性在使用时大致分为两种类型：  
1、传送类：使用者进行一个十分钟-1小时的旅行准备（视距离决定准备时间长短），在准备期间使用者不可进行其他行为，准备完成后即可开启一个传送门、传送阵以供使用，这一传送阵通常是单向的。需要注意的是，这不代表1小时准备即可立即达成一日的旅行路程，例如C级别旅行，其传送应当为1小时对应500公里或者近似区间。  
2、加速类：使用者进行一个持续十分钟的加速准备行为，在这期间不可中断准备行为，在准备完成后即可以旅行速度进行旅行。当处于旅行状态时，人物默认失去所有动作，不可进入战斗轮（换句话说就是进入战斗则旅行速度失效）。
