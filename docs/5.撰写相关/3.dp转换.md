# DP转换

  

※转化对应表   
  
1DP=1任意检定加值=1防御（任意类型，包括盔甲、格挡、掩蔽等）=意志、强韧、反射豁免检定各+1=单项豁免检定+2。  
1DP=扩大3点通用能量池  
2DP=1.5武器伤害提升（不含税）  
1DP=1高速/破甲=1破魔=1威猛  但4DP=5某种破防（已含税）  
1DP=2先攻  
1.5DP=1瞄准上限（含税）   
1DP=增加某个医疗类能力的8点医疗点数。   
1DP=5声望  
2DP=1点机运骰加值  
1DP=3虚拟体积（不提供生命值）  
4DP=1全检定加值=5外貌（1dp=1外貌，满4送1），这一特性的税会在每购买3点全检定加值时一并收取  
3dp=熔毁2（含税），每购买4点熔毁送1点。  
3dp=1点意志力用法为检定提供的加值提升  
3dp=攻击命中后1点意志力流失  
1dp=1点任意能量流失=4点指定能量流失  
6dp=5点指定破防属性，若该破防属性结算后仍有剩余则转化为攻击上的xx加值，至多转化5点。  
1dp=3点指定破防属性转化（在进行攻击前指定一种破防属性将其转化为另一种破防属性）  
1dp=2抵消破甲/高速/破魔/威猛 1dp=抵消2点在范围攻击中避开单位时需受到的减值  
在DP转换过程中，对于以上特性每有3DP被用于同种散特性可购买的能力中（如攻击DP，防御等，不包括提升属性、瞄准上限、扩大能量池、声望）就需抽走1DP税  
  
—————————————————  
  
  
3DP=1附加成功=1.5生命上限=1.5意志池上限=1.5临时生命（默认一场景刷新）=1招架伤害抵消=1快速治疗（并自带12点医疗点数）=0.75全伤害吸收=0.75异常点数吸收=1.2DR/-=1.2全能量抗力=2.5单能量抗力=2某类型物理伤害吸收=1.5DR/某类型（DR/魔法可以视为2.5，因为法破特性太便宜了）=1.2封闭=1吸血  
6DP=1异常点治疗（并自带24点医疗点数）  
  
3DP=提升1技能等级

1韧性伤害=1.5严重伤害，将1点严重伤害转为韧性为3DP。    
  
2DP=基因锁其中一个骰子加1成功数  
  
2.5dp=1近战伤害吸收=1远程伤害吸收  
3dp=2点指定异常点数恢复（医疗点12）  
1dp=每场景一次，忍耐2点指定异常点数至战斗结束后生效  
2.5DP=1体积  
2dp=狂怒1  
3DP=1点某伤害性质易伤  
2dp=1点伤害降低一级 （冲击-严重-韧性）  
3dp=1点活体护甲  
3dp=2点抵消意志力流失效果  
对于以上特性每有1个同级特性被用于同种散特性可购买的能力中（包括全吸收、生命等，不包括提升技能、外貌，载具模板购买体积无需交税），就需抽走1DP税。特殊的是购买附加成功、伤害、异常点无需抽税

注：通用射程是指近战武器的攻击射程、人物的触及范围等比较通用的涉及射程相关的购买方式。技艺射程是指基于新建攻击的初始15米基础射程，只能用于技艺中购买射程，并且不与通用射程叠加。  
1、在基础模板的基础上，射程跨阶需要至少高出等于【阶数差】的支线等级（D1~S5）。例如，基于狙击枪（200米）撰写跨阶至区间501~1500米的狙击枪，则该狙击枪至少是D级；基于白刃撰写跨阶至100米的飞剑，则该飞剑至少是B级。这只关系到物品/技艺定价的等级。  
2、对于远程而言，每在模板的基础射程上增加1个基础射程，就需要相当于3DP的特性；或者根据物品等级查找DP换算射程表，获得的射程取高。特殊的，对于步枪和狙击枪无法3DP增加1个基础射程，而是被1DP=10米的买法取代。  
3、对于近战而言，根据资源等级查找DP换算射程表。  
4、对于可以影响一定范围的能力而言，其能力的范围购买方式为：线形（或者宽度为1米的长方形）=1倍同级射程的长度；扇形、锥形（或者宽度不大于长度一半的长方形）=0.5倍同级射程的半径（或长度），圆形（或者宽度大于等于长度一半的长方形）=0.25倍同级射程的半径（或长度）。举例而言，A级的一个影响范围能力，可以花费1DP购买20米线形长度，或者10米锥形半径，或者5米圆形半径。  
在圆形的基础上，每支付3DP，可以让数值上升一档（即在1/4圆形时支付2DP，可以换成数值1/2的圆形，再支付3DP可以变为该数值的圆形）；此外，如果这个效果是对队友使用的buff，则该数值每有12DP，就要额外支付1DP税。对于持续时间达到一定程度（也即是说，可以通过多次施展来达到实质上作用于多个单位目的）的能力，也需要依照此方式购买范围。
