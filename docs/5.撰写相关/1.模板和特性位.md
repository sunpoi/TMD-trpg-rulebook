# 模板和特性位

  

  
模板与特性  
  
模板的定义：每个资源都有其基础模板，模板意味着资源在撰写前就已被决定了具有的同级特性总数。  
  
笔者将模板简单的分为三类，较为笼统，如有新的分类心得可联系笔者。  
  
1.内在模板：即会提供自由属性点的，不会因非遗忘类效果而失去的规整能力。一般包括：血统（训练）、改造（功法）、瞳术、技艺（称号）、能量池等。  
  
2.外在模板：即不会提供自由属性点的，会因非遗忘类效果（如击落武器，失去所有权等）而失去的规整物品。一般包括：物品（武器、护甲）、载具（随从）、奇物等。  
  
3.特殊模板：因多种原因而存在的特殊模板，无法一而概之，如模板特性不规整（无支线物品/能力、专长），或为特殊强化（如见证物品/强化、见证专长、天命）。  
  
内在模板：  
内在模板又可笼统的分为【可复数强化】与【不可复数强化】两大类。  
  
可复数强化模板包括：技艺（称号）、能量池  
  
不可复数强化模板包括：血统（训练）、改造（功法）、瞳术  
  
不可复数强化模板一般意味着能够提供更多的同级特性。  
血统（训练）/改造（功法）/瞳术的通用价格为：  
D+600/C+1200/B+2400/A+4800/S+9600  
  
非血统（训练）/改造（功法）/瞳术的通用价格为：  
D+500/C+1000/B+2000/A+4000/S+8000

此外，还可以在选择模板时额外选择一次性不可复数强化模板。一次性不可复数强化模板意味着该资源只能有一个主要等级，但可以有非不可复数强化模板的下属技艺/物品。  
一次性不可复数强化模板将会额外提供1同级特性。  
特殊的：一次性D级血统会共提供5D特，一次性C级血统会共提供4C特。  
  
笔者的话：血统、改造应尽可能避免沦为攻击招式发射器，可具有增益类主动能力，但尽可能避免所有等级不断的向增益类主动能力中添油的行为。  
  
可复数强化模板是所有模板的雏形，这意味着所有的模板只会在可复数强化模板的基础上增加特性和效果。

可复数强化模板即使只有一个主要等级，也不存在一次性等级收益。  
  
其中，能量池模板必须购买一项能量池（不可少于也不能多于），无论其组成方式。  
  
外在模板：  
外在模板可笼统分为【物品模板基底】与【奇物模板基底】两类。这意味着使用模板基底不同，提供的特性数亦不相同。  
  
物品模板基底包括：武器、防具、爆炸物。  
  
奇物模板基底包括：奇物、消耗品、载具（随从）。

关于奇物：  
以规则来看，不特殊注明的情况下，奇物不具有体积（可以是概念武装）、不占据装备位，因此，对于占据装备位的奇物将给予税收优待，从D级起，依次免除D1/C3/B5/A7/S9点税（同特税+三一税而非二者分别享受）。  
  
笔者的话：请勿选择奇物模板，并通过购买属性的方式冒充技艺。奇物应尽量不提供主动招式或能力。  
  
关于消耗品：  
消耗品本身具有难易程度（即难权，详见5.2基础、动作、限制）。

购买移动速度/轮时，在DP换算移速的基础上，移速=2×移动速度/轮。  
固定移动速度的轻型载具具有以下模板效果：防御减半，最终移动速度加倍。本特性只会作用于资源下的购买项目。B级及以上等级的轻型载具可以选择为驾驶者提供全遮蔽效果，但是需要占用一个每级的同级特性。撰写者可以选择不使用此模板效果，但是依旧归属与轻型载具分类。  
  
重型载具、战斗机器人具有以下模板效果：  
每为驾驶者和乘客提供全遮蔽效果，并额外为驾驶者和乘客提供遮蔽防御，其数值为载具体积/被遮蔽者体积取低，这个效果会在每个等级占用一个载具同级特性。  
  
最高速度与加速度：  
可进行加速的载具是一个需要额外消耗分数购买的特性，其价格为（最高速度/基础速度-1）×250分，而加速度通常为基础速度/4，可支付3DP（2税）使加速度翻倍一次，最多可翻倍两次使加速度等于基础速度。  
  
2.对于生物载具（随从）：  
对于生物载具和召唤物，根据主、副属性计算，对应如下：  
给这些召唤物分配2个主属性，3个副属性，其他4项皆为通常属性；有2个主技能，4个副技能，2个主专业，5个副专业，其他技能皆为1。之后，再给予其8点专长点（或对应的DP）  
给予的专长点兑换DP皆为散DP，专长同专长规则。也可以选择直接将这些专长点兑换为1个B特  
无支线：主属性为5，副属性为3，通常属性为1；主技能/专业为3，副技能/专业为1（特殊的，无支线随从不提供自带专长点）  
D级：主属性为6，副属性为4，通常属性为2；主技能/专业为5，副技能/专业为3  
C级：主属性为11，副属性为9，通常属性为6；主技能/专业为8，副技能/专业为5  
B级：主属性为16，副属性为14，通常属性为11；主技能/专业为11，副技能/专业为8  
A级：主属性为21，副属性为19，通常属性为16；主技能/专业为14，副技能/专业为11  
S级：主属性为36，副属性为34，通常属性为31；主技能/专业为20，副技能/专业为17  
S+XS级：主属性为36+XS×15，副属性为34+XS×15，通常属性为31+XS×15；主技能/专业为20+XS×6，副技能/专业为17+XS×5，X为额外S支线数量  
不关注载具的风度、操控属性，智力为0。优先分配感知+生理属性及体积，其他平均分配给决心和沉着。  
不关注载具除豁免以外的技能。实际上，在有驾驶员的情况下，生物载具同样可以用驾驶员的豁免作为豁免。  
该生物基础体积为5，若超过，每一点体积需要额外支付500分。  
支付750分，可以使其自由行动。  
  
生物载具同样可以遵循轻型和重型载具的模板（见下文），但可以选择不使用它。相对应的，也无法获得对应的好处和承受对应的负面。  
  
笔者的话：  
1.生物载具/随从的天生优势为可自主行动，若自主行动的生物载具/随从为飞宠/骑宠，在驾驶者不使用移动动作进行指挥移动的情况下，驾驶者必须经过一个与生物载具/随从进行的攀附对抗赖在生物载具/随从身上被携带移动，需写明。  
2.生物载具/随从一般而言不能拥有通用强化或技艺，只能被生物载具/随从使用的技艺不提供自由属性点（本质奇物）。  
3.针对召唤物而言（本质生物载具/随从），召唤时不要使用动作权值（详见5.2基础、动作、限制）。  
4.现已禁止将机械载具写作一次或多次吸收伤害器，即虚假机械载具，随身携带以利用全遮蔽保护自身。  
  
特殊模板：  
特殊模板因其特殊性而无法分类，特殊模板部分也可能被在规则层面归为某资源类型（如：无支线模板物品被归为物品资源）。  
  
无支线模板：  
无支线模板即其起始价格不足2000分的模板，一旦总花费价格达到2000分，则必须升为D级对应模板。无支线模板不存在无支线血统/改造/瞳术（被视为血统/改造/瞳术时，不占用不可复数强化模板位，可以与不可复数强化模板并存），无支线模板不提供自由属性点，无支线模板除消耗品外，不能应用非限制权值。  
无支线模板以每250分提供1DP散特性的方式提供DP。

套装模板：  
基于多个物品或强化组成的特殊模板，会根据物品等级或强化等级使同一套装内的所有物品或强化获得额外特性，通常情况下每个人只能使一个套装的效果生效，其它套装会被视为不成套的散件，不会获得额外特性，甚至失去所有效果。套装等级等于套装内支线等级最高的物品或强化。  
  
套装效果需要具有至少2个及以上的套装内同级别物品被穿戴才可生效，此时套装会根据组成套装物品的支线等级提供等量的额外特性。例如：由2件C级别物品、一件D级别物品组成的套装，在计算套装效果时会获得2个C级别特性与1个D级别特性。  
  
从A级开始的套装，武器与护甲不计入套装的物品数量  
  
见证物品/强化/专长模板：  
见证物品/强化/专长模板属规则书层面特殊模板，不提供自由属性点，无有效等级划分，其基础换算为：  
1见证点价值10DP  
当资源价值1见证点时，视为D级资源，提供3D级特性与具有D级特性效力的1DP  
当资源价值2见证点时，视为D-C级资源，提供3C级特性与具有C级特性效力的2DP  
当资源价值3见证点时，视为C-B级资源，提供5C特  
当资源价值4见证点时，视为B级资源，提供3B特与具有B特效力的4DP  
当资源价值5见证点时，视为B-A级资源，提供4B特与具有B级特性效力的2DP  
当资源价值6见证点时，视为B+-A级资源，提供5B特  
当资源价值7见证点时，视为A级资源，提供2A特与具有A级特性效力的22DP  
  
天命模板：  
天命视为价值1C特的能力，若可与特5共存，则需要承受价值3DP的负面效果。
