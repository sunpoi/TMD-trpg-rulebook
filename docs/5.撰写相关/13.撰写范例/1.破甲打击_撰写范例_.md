# 破甲打击（撰写范例）

名称：破甲打击  
本质：魔幻  
分类：技艺  
价格：D+500  
动作：标准动作  
范围：触及范围  
目标：单一目标  
持续：立即  
描述：使用力量来一场硬碰硬的较量吧！  
效果：你进行一次普通攻击，本次攻击具有2点表现加值，附带自身等同【力量】属性一半的【破甲】。

\======  
D级技艺2D特

2D特转6DP

6DP+（6DP×（标准动作权值）1-（动作税收）2\*（2\*1/2））/时间权值1（立即归属于1轮）=10DP

其中10DP为4指标，具有C特效力。  
2DP加值

（6+11+16+21+26）/5=16（平均值），16/2（属性一半）=8，8\*1（每点破防价格）=8DP购买自身等同【力量】属性一半的【破甲】  
（参考成长性能力-等级成长，将力量作为主属性进行平均值计算，）
