# 吾等的梦醒之泪（撰写范例）

名称：吾等的梦醒之泪  
本质：特异  
分类：消耗品

价格：250/4滴  
描述：  
早已忘记那些虚幻之物，早已忘记那些可怖之兽，早已忘记那些幸喜之景。

但眼泪记得一切。  
效果：  
每轮一次，以一个花招动作饮用一滴吾等的梦醒之泪，饮用者获得4点意志豁免上的增强加值，持续一轮。

\======  
无支线消耗品，250分，1DP

持续1轮，只能用4到9次的限制权值为1倍。

即1DP+1DP\*1（限制权值）/1（持续1轮）=2DP

2DP购买2次2点某豁免检定加值。

记得标效果持续时间。
