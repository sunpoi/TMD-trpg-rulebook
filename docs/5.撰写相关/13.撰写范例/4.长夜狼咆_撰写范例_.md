# 长夜狼咆（撰写范例）

名称：长夜狼咆  
本质：魔幻  
分类：技艺  
价格：D+500  
动作：标准动作  
目标：-  
消耗：2点生理类能量  
持续：一场景

描述：  
于长夜之际，化为夜之梦魇。

效果：  
只能在位于夜间的情况下使用。

每场景一次，以一个标准动作发出咆哮，将狼人之力呼唤在自己身上，本场景内。你的体积提升3点（需视为虚拟体积，不会提升生命值），天生防御提升2点，你的强韧豁免获得2点增强加值。你的近战攻击获得【余波】和【9加骰】。

\======  
D级技艺2D特

2D特转6dp

6DP+（6DP×（完全消耗标准动作权值）2-（动作税收）2\*（2\*1/2））/时间权值1.5（一场景）+6DP×0.5（仅夜间）/1.5（一场景）+2（能量dp）/1.5（一场景）=16DP

其中14.6667dp 为4指标 1.33333dp为散DP

1dp买3点虚拟体积。（如果需要区分，这里用掉了散DP）  
2dp买2点天生防御  
1dp买2点强韧豁免加值  
6dp4指标买余波  
6dp4指标买9加骰
