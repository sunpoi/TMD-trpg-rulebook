# 铁之心，石之花（撰写范例）

名称：铁之心，石之花  
本质：魔幻  
分类：奇物  
占据位置：饰品位  
价格：D+500  
描述：一朵石雕的玫瑰花，当你佩戴时，它的荆棘将扎入你的心间  
效果：  
只要佩戴铁之心，石之花，你的生命值上限就降低1点。

你获得1点快速医疗（12医疗点）。

在佩戴铁之心，石之花的情况下，每经过一场影片，在影片结束后，你佩戴铁之心，石之花就额外具有2点在豁免上的器械加值，最多以此法获得12点器械加值。

\======  
D级奇物模板 3D特

生命值上限降低1点换1点永久DP

1D特转3dp买1点快速医疗，占据位置的D级奇物有1点免税，这里用于购买快速医疗的同特税。

剩下的2D特转6dp和负特取得的1dp组合成7DP

使用影片成长，一影片条件成长20（不要滥用）

成长效果为2dp三豁免加值，需要2.66667DP（需要计入三一税，所有本来需要根据情况缴纳税的效果需计算计入税后的平均值作为成长效果）

成长次数实际为12/2=6次

将该条件输入成长性计算器（撰写相关-称手工具-成长性计算器），得到需要支付6.678544040546358DP

根据影片成长规则，若使用影片成长后具有小数，则额外向上舍入0.5，入为7DP
