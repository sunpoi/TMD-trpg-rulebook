# 撰写范例

  

撰写范例  
D级技艺  
消耗1点魔力，标准动作进行一次斩击，此次斩击获得XDP检定加值，并额外击退被攻击者，反射豁免对抗击退二者差值  
｛（3+3）\*（1+1）-2｝+1=6+6-2+1=6+4（具有C特效力）+1 4税  
击退同冲撞，3指标3DP  
每3点攻击检定加值就需抽走1DP税，易得X=6 被抽走2DP  
  
D级血统  
开启一个魔力池，其容量为智力+感知之和。每经过一场影片，这个魔力池便+1上限  
并获得等同于人物最高支线等级数的意志豁免（D1C2B3A4S5）  
（3+3+3） 2指标  
开启一个主副能量池3DP2指标  
每经过一场影片扩大1能量池上限分两步，第一步扩大1能量池上限=0.334DP，四舍五入0.3；第二步成长效果0.3、成长条件8，易得价格2.512DP，约等于3  
成长性D1C2B3A4S5，（1+2+3+4+5）/5=3与D级相比，等于D级特性，D级一次付清1D特=3DP  
  
D级防具  
轻甲，护甲5/5【防弹】，被进行一次有造成伤害的攻击作用后可有一次，获得快速治疗1（医疗点24）  
（3+3）\*（1+0.1） 2指标  
被一次造成伤害的攻击后才能发动的能力难权极易，0.6抹去  
轻甲模板2/2，D级自动上升3点  
防弹2DP2指标  
快速治疗3DP，抽走1DP税
