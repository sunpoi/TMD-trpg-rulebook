# 幕间时间与影片成长

  

幕间时间与影片成长  
  
由于部分时间成长类资源与造物类资源的存在，每个影片之间在主神空间内的幕间时间被固定为7天，在此期间那些一影片成长一次的资源会成长一次，然后在影片中往往会停止成长，而角色在次期间也可以自由的行动，去结交不同的角色、去制造自己所需要的物品、去进行某种活动。
