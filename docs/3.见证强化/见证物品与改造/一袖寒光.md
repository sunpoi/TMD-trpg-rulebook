# 一袖寒光

  

一袖寒光  
1见证点数  
前提：第二种武器（3）  
一袖寒光本质上是基于一件衣服/暗器袋等可以用来储存暗器的物品改造，属特异本质。一件衣服/一个暗器袋只能进行一次改造。在改造后，即使你不在衣袖中/暗器袋中储存任何暗器，也可以在试图取出暗器时取出想要的现实暗器。  
一袖寒光在购买时会将第二种武器3级的效果提供的先制替换为D级先制，但相应的，随着人物内在强化最高等级支线的提升，会逐级升级为C/B/A/S级先制。
