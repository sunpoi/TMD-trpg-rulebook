# “国土防卫者”机群

“国土防卫者”机群

5见证点 

R-Romeo  
在冷战最剑拔弩张的年代，欧亚国家们曾秘密研发出一种名为“国土防卫者”的无人机群。它们不是普通的飞行器，而是一群被赋予“有产者灵魂”的钢铁哨兵——24小时不间断巡逻国土的每一寸边缘，连一只越界的大洋国松鼠都会被它们用红外扫描仪拍成高清证件照。

项目总设计师是据称“比老大哥更疯”的欧亚国科学家爱德华·奥威尔（其坚称有前英帝国王室血统）。他在立项报告中写道：“传统防空系统的问题在于，人类士兵会打瞌睡、会思乡，还会偷偷用雷达给女友发摩尔斯情书。而无人机……只需要一块刻着‘战争至上’的EPROM芯片。”

特殊的，“国土防卫者”机群需使用升级版支援无人机进行换购，在使用支援无人机经历三场影片后，可以使用标注的见证点数换购“国土防卫者”机群。

“国土防卫者”无人机群拥有无从计数的无人机组，在未下达任何命令时，无人机组将自动盘旋在使用者的周围，使用者将总是享受无人机群带来的紧急守护效果（仅限类同格挡）。

“国土防卫者”机群可以被当作武器使用，也可以作为独立行动的随从，在作为武器使用时，使用智力+器用：无人机+辅助专业：弹道学进行攻击，武器伤害22L，基础射程为100米，速射等级为连射，攻击拥有【高速5】。在作为独立行动的随从时，它具有74+8的预设DP，，基础射程为100米，并拥有【高速5】。其始终拥有可以搭载你进行飞行（机动性为完美），并具有曲线攻击的能力。  
“国土防卫者”机群的移动速度为每轮80米，结构值52，护甲防御28。

“国土防卫者”机群具有两种特殊的攻击方式，在使用这两种攻击方式时，都需要使用者以一个整轮动作指挥。

●焦土政策  
无人机群朝着指定的位置进行俯冲轰炸，投下微型核弹，进行一次以智力+器用：无人机+辅助专业：弹道学进行攻击的攻击检定，其出目为这枚微型核弹造成的伤害，其落点半径35M内的敌人均需进行一次强韧豁免进行对抗，并受到二者差值的伤害，这颗微型核弹还会附带等同于造成伤害四分之一的燃烧点数和四分之一的毒素点数，使用强韧豁免进行对抗。核弹爆炸后还会在爆炸范围内留下同样大小的辐射区C级困难地形，持续存在一轮。

●蝇式袭扰  
无人机群朝着指定敌人进行机动扫射，进行一次预设攻击检定，正常攻击那名敌人的防御。那名敌人若被此次攻击命中造成伤害，则立刻陷入【虚弱】状态中，直到你的下个行动轮开始，他的所有攻击检定将受到等同于此次攻击造成的伤害的士气减值。  
同时，根据其可能拥有的异常点数或异常状态，进行如下结算，若不满足括号内的条件则跳过：  
获得1点霜火（并移除1点冻结点数，若有）  
获得1点电浆（并移除潮湿，若有）  
获得1点排异（并移除1点毒素点数，若有）
