# 麦田里的守望者

  

麦田里的守望者  
  
3见证点  
  
西边山上，九只母狼  
东边山上，一轮月亮  
反复抱过的妻子是枪  
枪是沉睡爱情的村庄  
  
特殊的，麦田里的守望者需使用猎手之枪进行换购，在使用猎手之枪经历三场影片后，可以使用标注的见证点数换购麦田里的守望者。  
  
武器采用基础远程武器中的狙击枪模板，具有全部相关特性，在其基础上提升9L武器伤害，并额外具有1点高速。  
  
这把狙击枪，整体呈深麦色，如若不仔细分辨，可能会将它错认作一片倒塌的麦丛，如仔细观察，还会发现木制枪柄上生出一圈又一圈的年轮。  
  
大地之子  
“祖父死在这里，父亲死在这里，我也会死在这里。  
你是唯一一块埋人的地方。”  
当你身处于草地、田野、森林等有大量植被的环境中时：  
①你的身周将快速催长出大量常见植物，如杂草、不知名的野花，这些催生出的植物将会连成一道仅保护你的壁垒，能够为你提供5点遮蔽防御。  
②并获得C级自然本质的反侦测能力，同时，被你亲手埋葬于这类环境中的躯体将始终视为遗体保存完好。  
  
无尽守望  
“大风从东吹到西，从北刮到南，无视黑夜和黎明。  
你说的曙光究竟是什么意思。”  
一场景一次，一次行动轮开始时，若你无法被场景中的任何敌人定位，无尽守望将开始生效，此时你始终视为处于先攻队列的最末端。在每个行动轮开始时，需选择是否跳过你自身的回合，若不跳过，则这时无尽守望将会失效；若跳过你的回合，则：  
①每一个行动轮结束时，若无尽守望还在生效，根据无尽守望生效的轮次，立即获得等同于你的传奇感知\*8/4/2点的瞄准上限。同时，若你的基础AP高于14点，则还会每超过1点，就再额外获得6点瞄准加值。这些获得的上限和加值将在无尽守望失效后的下一次攻击结束时失去。  
②当无尽守望失效时，这一能力将立即开始生效，你每跳过一轮，就在这回合开始时额外获得1点仅能用于施展射击技艺的AP，这一轮结束时立即失去这些ap。
