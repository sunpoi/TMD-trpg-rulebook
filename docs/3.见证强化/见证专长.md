# 见证专长

  

无双剑围  
4见证点数  
前提：无双剑士（2）  
“没人能越过我的剑围。”  
你必须手执两柄剑才能使用此招式。  
当使用者使用标准动作进行攻击时，还可以同时喊出“左击”或“右击”，并选择左击或者右击，若选择的和攻击的方向相同，则额外附带基于敏捷属性一半的威猛,若获得威猛在计算破防以后仍有剩余，则可以将它们转化为攻击上的增强加值；若选择的和攻击的方向不同，则额外附带一个C级措手不及效果。  
你每轮从以下三种架势中选择一种使用，选择的架势不可与上一轮选择的架势相同。使用架势是一个标准动作。  
1、使用一柄剑攻击、一柄剑防御，进行一次攻击与一次格挡，在此次攻击中获得武技等级一半的技艺加值，在格挡上获得武技等级一半的格挡加值。  
2、使用两柄剑防御，进行一次全防御与格挡，在全防御期间免疫多次攻击减值、获得基于格挡防御的【坚韧】并对触及范围的一个友方单位提供全遮蔽。  
3、使用两柄剑攻击，你可以选择其中一把武器作为主导，并将未加入攻击的另一把武器的基础属性一半与所有特性加入攻击中，攻击同时对触及范围内所有单位造成伤害，这次攻击中你会同时获得左击与右击中猜测方向相同与不同的好处，并且获得的威猛提升至敏捷属性点、措手不及提升至B级。  
  
巨人杀手  
2见证点数  
巨人杀手仅有在面对体积至少大于等于其自身体积3的生物敌人时才能生效，这意味着它无法在面对机械载具等情况时正常生效。巨人杀手在对体积至少大于等于其自身体积3的敌人时，会额外为巨人杀手的购买者提供在对那个敌人攻击时可以将攻击结合在移动中的能力，正常消耗动作，表现为巨人杀手灵活的进行攻击。且巨人杀手对巨人的攻击还视为具有【毁灭】效果。同时，巨人每比巨人杀手的体积高出3，巨人杀手在对巨人进行攻击检定时，就会获得额外的1DP表现加值，至多获得巨人杀手决心属性点。  
  
毁灭冲击  
2见证点数  
前提：驾驶轻载具或生物载具  
现在，骑乘智慧生物或可独立攻击载具时（不包括你的轮回者队友或者人），可以以一个整轮动作发起冲锋，冲锋上限仍然遵循驾驶者的上限，但同时扣除自身和载具的基础防御（若有），在冲锋的移动步骤结束时，被骑乘的智慧生物或可独立攻击的载具会随驾驶者一同对其攻击的目标进行一次攻击，正常以该智慧生物或可独立攻击的载具的基础攻击方式进行攻击。  
  
传奇正盛  
2见证点数  
前提：至少为D级传奇轮回者  
每当你获得见证点数时，若此见证点数并非你兑换的，而是在影片结束时自然获得的，那么你立即再获得5点永久声望，并检视下一项是否能满足条件，若可以，则获得其效果，必须逐项检视，某一项不符合时则跳过剩余的，直至不符合或所有的都被获得过为止。  
●若上一场影片中，你获得了临时声望，则再获得5点永久声望。  
●若这场影片结束时，你获得的奖励资源最多，则再获得5点永久声望。  
●若这场下场影片开始前，你的评级为同队最高（前提为有队伍），则再获得5点永久声望。  
  
犬马之劳  
2见证点数  
前提：大镖客（3）  
在购买此专长后，你的身体将会不可避免的发生异变，你的身上将会逐渐出现如同金属般的斑状物，身体也会变的僵硬，如同金属一般，并逐渐失忆，这种症状被称为“金尸化”。同时，因为金尸化，这也会为你带来无需睡眠、进食的能力，同时获得基于你天生防御的【防弹】。由于对于炼金术的深入研究，在你使用手枪进行射击检定时，还可以额外将辅助专业：炼金术（学识）加入检定中进行检定，如ST允许，还可以以造物规则制造特异本质子弹，但不能制造出无限版本。而黄金时代结束后，炼金术师的异变就已开始无法遏止，金尸化还会永久的减少1点你的意志上限。  
  
吸元秘术  
2见证点数  
前提：劫命圣手（2）  
现在，你掌握了新的只能在长线治疗中施展的疗法，用一次长线治疗的时间施展，正常进行长线治疗，但治疗的所有伤害不正常治愈，而是转移到你的身上，若这次长线治疗不少于一个小时，则可以将转移的伤害中最多不超过你的耐力属性等级的伤害数下降一级，优先下降韧性伤害，不可将同1点下降两次。在使用吸元秘术时，治疗者必须保持与被治疗者体液交换的形式，这可以代替长线治疗的正常治疗方式，但治疗者只能同时治疗一个被治疗者。  
  
末日歌者  
2见证点数  
前提：电台主持（3）  
现在，当你使用风度作为主属性进行攻击时，将改为造成音波伤害。在表现形式上，视为你唱出的歌声成为杀伤敌人的音波能量。你的攻击范围扩展为半径为15米的圆，若你的攻击射程大于15米，则改为半径为攻击射程的圆。在范围内的所有敌人将使用意志豁免对抗你的攻击出目。但相应的，末日歌者也会在购买时令你承受1点永久存在，不可豁免的沮丧点数。  
  
[热线电话（Hotline）](https://music.163.com/song?id=27690423&userid=1539218849)  
3见证点数  
前提：接线员（2）  
在每场影片中，由ST安排下，会有至多一位乐意雇佣你的人以各种形式（如打电话、面见等）找到你，并为你发布一个比你的人物支线等级困难一级的刺杀/屠戮任务。完成这个任务会为你带来雇佣者赠予你的资金（具体世界具体裁定，由ST决定，总数值不会低于正常标准的2.5W金钱）和5点声望（这一般是恶名）。  
选开：ST也可以为接线员提供更加困难的任务，但这只会提升获取的金钱数量。  
  
无踪死神  
5见证点数  
前提：断魂诡影（3）  
“无形的镰刀一直在等待时机，一个夺走你性命的时机。”  
此专长必须使用狙击步枪才能施展，在进入战斗场景时，若所有敌人无法定位你，你可以选择不投掷先攻骰，而是随时可以以一个反射动作在目标的回合对其进行攻击，这次攻击具有【先制】。目标需以意志/强韧豁免检定中较高者来对抗造成的伤害，若失败，则这次进行的动作会被打断，这视为一个B级创伤来源的震慑效果。之后投掷先攻骰，按照基础规则正常加入战斗轮。  
【先制】的等级等同于沉着属性的附加成功，D1C2B3A4S5。  
  

逻辑与推理的实验剧场  
前提：侦缉探查（3）安乐椅神探或现场调查  
3见证点数  
只能在非战斗状态下使用。与至少一名愿意一同进行推理的角色共同进行一场逻辑与推理的实验剧场。参与者全程需要进行类似舞蹈的动作，即使没有舞蹈相关的技能，尬舞也是舞嘛。作为发起者的你需要首先阐述自己的一段推理，即使这个推理错得离谱。之后所有与你共舞的参与者将与你一起，围绕着你的推理，为了指正其中的错误而进行讨论，与你共舞的参与者将在你的启发下进行深度的思考，而你也在与其的探讨中获益良多。当前场景下包括你在内的所有人【为了验证你推理的正确与否进行的非战斗的智力与感知的检定】获得等同于你智力属性-1的表现加值。（举例：一般情况下有回忆检定、搜索检定、解读检定、追踪检定、操作电脑等）以及，需要注意的是，修复，制造物品的检定与载具驾驶相关检定一般与进行推理无关，一般不应受到剧场的加成，侦查检定则几乎在所有情况下都不应受到该效果加成。但具体仍由ST判定。

  
你是在看,而我是在观察,这有很明显的差别。  
前提：侦缉探查（3）现场调查或隐形探访者  
2见证点数  
观察已经成为了你的日常习惯。在非战斗情况下，你无需刻意即进行了搜索检定与侦查检定，这视为在你需要进行这两种检定时自动进行过了一次检定，且此次检定结果直接由DP的三分之一加上附加成功作为结果，即使失败，你也会意识到自己有什么没有察觉到，随后可以立刻选择刻意执行一次这种检定。此外，你可以用智力代替感知进行侦查检定，这代表你从目前发生的变化中通过逻辑揣测出了即将到来的新变化。当你轻易地进行了搜索或者侦查时，你可以立刻将自己的观察结果述说出来大声告诉给现场所有人，所有见识到或得知了你此番表演的人都将为你的表演所折服。这将为你在当前轮回对得知这番表演的人来说额外具有10声望。  
  
持卫的裁决  
前置：坚定风采（2）  
2见证点数  
唯有那些最勇敢，最坚定，最强韧的家伙才能被称作裁决使，无论何种伤势，何种攻击，他们都不会被击倒。  
现在你的意志会在危机时刻，强化你外在的抗性，这表现为你能更好的应对敌人的攻击，当你的完好在一半以下时，获得3DR/-。  
所有的裁决使都要学会在战斗中为自己的队友创造机会，方式就是借袭击敌人来保护队友。  
你现在在使用坚定风采中的格挡方式时，一轮一次，可以以一个自由动作移动耐力属性上的附加成功\*5米，现在你的冲撞具有打断性。  
  
不破壁垒  
2见证点  
前提：披坚执锐（2） 耐力6 力量11  
仅在身着的盔甲是重甲时才能生效，你无视8点破甲，你盔甲减值减少4点。  
你在穿戴重甲后，厚重的盔甲给你带来的不止是安全，还有对敢于挑战你的人的反击。  
你在受到近战攻击后，若仍然保留有盔甲防御，则对该单位反伤等同力量附加点力场伤害。  
  
银背族长  
1见证点  
前提:银背猩猩（1） 体积6 力量6  
waaaaaaaaaay——你晋升为了族长！  
你现在在场上是最亮的崽。  
同时你无视多次攻击减值。  
你的体积再提升2点。  
  
疾风骤雨  
前置条件：同时持有白驹过隙，捷足/先登其中之一  
2见证点  
“步履如飞，疾如闪电。”  
只有在不着甲或是着轻甲的时候才能生效。  
  
现在你额外获得敏捷上的传奇属性\*1个先攻骰，但是这些先攻骰带来的先攻只有在你本轮移动之后才会保留，否则你在回合结束时失去这部分先攻，并且基础移速下降20M，  
  
在每轮开始时，你获得1层【疾行】。  
在你移动时，你可以消耗1层【疾行】在一轮内为你带来15M的额外移速；在你受击时，你可以用自由动作消耗1层【疾行】在一轮内获得3点闪避防御。  
  
箭无虚发  
2见证点数  
前提：手稳就准  
射箭本身没什么技巧，只要保证命中就行。  
对于弓箭专长α/β/γ而言  
你每拥有1项其中的（2）级专长，就在弓箭射击上获得2点破甲。  
你每拥有1项其中的（3）级专长，就在弓箭射击上获得1点附加成功。  
你每拥有1项其中的（4）级专长，就在弓箭射击上获得1点附加成功。  
现在，你施展手稳就准的动作降低至花招，且手稳就准提供的加值将变为你的传奇敏捷\*3点洞察加值，你每轮使用手稳就准的次数提升至传奇敏捷次。  
  
巨人之擘  
1见证点数  
前提：破军重武（2）  
对此，阿特拉斯只是耸了耸肩。  
当你使用双手武器时，破军重武将会自动生效。当你进行一次攻击时，本次攻击你可以消耗1点意志力，你可以选择至多等同于你的体积/武器体积中较低者点攻击上的DP失去，并额外获得等量的附加成功，但不能超过5点。巨人之擘将为你额外提供1点体积。  
  
自由射手  
2见证点数  
前提：使用狙击枪/来复枪/步枪  
冷静到近乎冷酷，无差别的将子弹送入每一个敌人的脑中，当他们看到狙击镜的反光时，一切都晚了。  
当你因瞄准获得攻击检定上的加值时立刻触发，可以再额外消耗1点意志，可以选择失去本次获得的瞄准加值中的至多5点加值，每失去1点，本轮内你的下一次攻击将会额外获得1点附加成功。  
自由射手将为你额外提供传奇沉着\*2点瞄准上限。
