# 死神之眼

  

死神之眼【The Dead Eye】  
  
3见证点数  
前提：拥有大镖客（3）  
  
这个见证强化会占据你的瞳术位  
只有传说级别的牧马客才具有的眼瞳。拥有死神之眼的荒野行者即使背对着六名持枪匪徒，亦能在下一刻吹散枪口的硝烟，转枪入袋，纵身上马离去——在旷野上留下的，是一声枪响与散落的六枚黄铜弹壳。  

注：死神之眼提供的好处仅能在使用者持用不具有【能量武器】词条的远程武器时生效。  

强化本资源后，强化者获得一个特殊的刻度——【核心值】。  
【核心值】：核心值上限为敏捷+沉着，最低为0。每次长休息后核心值恢复满值。  
当使用者通过爆头/头部部位攻击的方式击杀一名敌方单位后可恢复1点核心值。（理所当然的，使用者通常不能随意的将路人、平民或者动物认定为敌人并杀死以获取核心值，除非St切实的支持此类行为。）  
死神之眼开启时，核心值会因为使用者的行为而减少，这是使用者精神专注程度被消耗而导致的。  
具体消耗核心值的行为如下：  
使用迅捷/反射动作—2点  
使用移动/杂技动作—4点  
使用标准动作—6点  
每标记一个目标—1点  

  开启/关闭死神之眼是一个自由动作。  
  使用者只有在核心值不为0的情况下才能开启死神之眼。死神之眼开启时，使用者眼中的世界将会覆上昏黄的滤镜，同时场景内会响起一道来自正午时分、厚重悠长的钟声与“喀～哒、喀～哒”轻微而缓慢的怀表转动声。使用者的核心值被消耗的越多，钟声就越是消散，怀表转动的频次也会越接近正常。若使用者的核心值被消耗至0，则钟声彻底消散且怀表转动频次恢复为正常，死神之眼也将被迫关闭。  
  
特殊的，购买者在强化这个瞳术的时候可以选择将钟声替换为低沉轰鸣的雷声/野风吹拂荒漠的呼啸声、怀表声替换为缓慢且有力搏动的心跳声等此类独特的声音。  
  使用者需要进行一个敏捷检定以对抗在场所有人（无法定位、不在视距范围内的单位不在此列）的敏捷/沉着（取高者）检定，这之后对抗失败者在使用者眼中会同样被覆盖上昏黄色滤镜，其动作甚至是枪口射出的子弹也变得迟缓凝滞。若目标对抗成功，则不会被死神之眼的任何效果所影响，其在使用者眼中也不会覆盖上昏黄色滤镜。  
  
死神之眼实际效果如下：  
1、受到死神之眼影响的目标在你的回合中视为被你掌握【先制】，对于掌握先制的一方而言，对被掌握先制的一方具有措手不及，但这种措手不及并不会导致被掌握先制的一方失去基础和闪避防御，而是无法应对你的攻击使用应对技，或利用反射动作调用动作。这个措手不及的效果初始为B级，当你的人物最高内在支线等级提升至B/A时，这个措手不及等级将自动提升为A/S。  
2、使用者攻击受影响目标时，瞄准上限提升敏捷附加成功数\*2点。  
3、受影响目标对使用者进行的攻击失去5点高速。  
4、使用者可用一个自由动作标记单个或多个受死神之眼影响的目标，标记的目标在使用者眼中会被打上红色的X标记，标记在进行一次攻击后会被消耗。使用者攻击被标记的目标会获得敏捷检定成功数点瞄准加值（这部分加值不计入瞄准上限）。多次标记一个目标不会重复获得瞄准加值。
