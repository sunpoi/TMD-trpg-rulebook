# 对特定部位攻击

  

对特定部位攻击  

这是指定攻击敌人身上某处要害的动作。  
对特定部位攻击一般来说视为一次普通的攻击，如无特别说明，则你可以在这次攻击中使用任何特殊能力或招式。需要注意的是，只有指定具体目标的攻击可以进行部位攻击，而范围性攻击若无特殊说明通常不能指定部位攻击。  
视攻击部位而定，你在这次攻击中受到一定的减值。若被攻击者意识到了你的这次攻击且保有基本防御（这意味着他没有处于措手不及或类似定身的状态），他在防御上获得与你受到的减值数量相同的闪避加值。抵御以此法造成的异常点数（包括肢体妨碍）时，使用标准豁免进行对抗  
攻击特定部位会有不同的特殊效果，视情况而定，除去击落武器外，这些特殊效果均视为C级创伤来源的效果。以下是一些范例：  
攻击头部：减值3，每2点伤害造成1点晕眩，以耐力-求生-强韧对抗。  
攻击腹部：减值2，每1点伤害造成1点恶心，以耐力-求生-强韧或决心-求生-强韧对抗。  
攻击肋骨或鼻梁：减值3，每2伤害造成1点剧痛，以耐力-求生-强韧对抗  
击落武器：减值3，不造成伤害，以力量-武技-对应武器专业对抗，失败则武器脱手。若攻击方以手或类似结构的天生武器进行攻击，可夺取武器。  
攻击肢体：减值2，每2点伤害造成指定肢体的1点肢体妨碍，以耐力-求生-强韧或敏捷-求生-强韧对抗
