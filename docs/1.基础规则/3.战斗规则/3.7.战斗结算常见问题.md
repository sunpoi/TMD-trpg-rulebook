# 战斗结算常见问题

  

战斗结算常见问题  
  
Q1：附加之间会相互抵消吗  
A：会  
  
Q2：若反击具有打断性，攻击者被反击死亡，如何结算  
A：他的那次攻击没有成功结算  
  
Q3：吸血的生效时间  
A：20步  
  
Q4：一般的反击生效时间  
A：具有打断性则在第二步生效，不具有打断性则在攻击结算后生效  
  
Q5：在反击过程中，被反击者进入全遮蔽状态，如何结算  
A：正常攻击用于全遮蔽的物体或人  
  
Q6：在攻击过程中，被攻击者进入全遮蔽状态，如何结算  
A：正常攻击用于全遮蔽的物体或人，这不包括被攻击者用攻击者自身全遮蔽，如出现此情况，正常攻击  
  
Q7：防御转换成的伤害抵消可以被以某种能力无视或抵消吗  
A：不可以  
  
Q8：类冲锋能力是否也如同冲锋能力一样会被移动出范围所取消攻击  
A：是  
  
Q9：是否可以边格挡边招架  
A：理论上来讲，只要动作足够，可以，但二者效果不会叠加计算，仅生效最后执行的防御行为。  
  
  
Q10：如果我与队友都购买了某个使范围内的友方单位获得增益的资源，能否进行叠加？  
A：不能，一个单位同时只会受到同一个能力(即相同资源带来的能力）的影响，即使这个能力由多个不同的单位同时释放/同时购买也是如此。  
  
Q11:如果我一次攻击能够同时攻击多个单位，且我的攻击能够造成5米内的范围伤害，那么我的攻击在同时攻击两个相邻5米内的单位时，这两个单位是否会因为一次攻击受到2次伤害？  
A：不能，通常而言，一次攻击影响了多个单位的能力不会重复结算。  
  
  
Q12:如果我的冲锋能够重复途径同一个单位，我能否以此方式造成多次伤害？  
A：不能，与上一条类似，同一次攻击不会重复影响同个单位。  
  
  
Q13:如果我的格挡所获得的防御由格挡防御改为其他防御类型，那么因其他资源或者特性带来的格挡防御是否会一并更改？  
A：不会，更改格挡所获得的防御类型仅限于格挡本身行为所带来的格挡防御生效，因此诸如【格挡5】、格挡时获得x点格挡防御等特性将不会随着变更。  
  
  
Q14：触及范围与周围/前方/面前X米的能力是什么关系？如何进行互动？  
A：通常而言，若一个能力的描述是你面前5米/周围5米，这意味着以你当前的触及范围内的一点为起点往前方延长5米作为生效范围。  
  
  
Q15：传送移动与移动中攻击特性是否能达成在传送中途移动接着进行传送的效果？  
A：不能，这是因为传送特性是无视效果线直接抵达终点决定的，当你使用传送特性进行移动时，这一传送效果会令你直接抵达移动的终点而无视路径（也就是效果线）上的障碍而无法在移动的途中进行移动中攻击行为，类似的是冲锋移动时进行传送且附加了攻击路径上触及范围内所有单位的特性，只会在移动的起点与终点达成攻击触及范围内敌对单位的效果。  
  
Q16：如果我同时有施加一档肢体妨碍与二档肢体妨碍异常点数的能力，能否同时生效？若一个单位同时承受一档肢体妨碍与二档肢体妨碍，该如何计算严重状态的点数？  
A：一档肢体妨碍与二档肢体妨碍本质是一种异常点数，因此在一次攻击中不会同时施加，而是选择一种施加。一档肢体妨碍意味着这一异常点数是施加给某一肢体的，二档肢体妨碍则通常是对所有肢体生效，因此在计算一档肢体妨碍的严重状态时，可以将二档肢体妨碍点数加入其中计算，但反之则不行。  
  
  
Q17:假如我具有多个固定生效的转化防御能力，由哪些能力优先生效？  
A：无法自由切换的固定效果外在资源先生效，比如用盾格挡、用剑挥砍，则这个盾牌的转格挡类型、剑的固定伤害性质等。如果都是内在/外在资源，则资源等级较高者优先生效。  
如果资源等级相同，则最后强化的资源优先生效（即覆盖先前的同级别固定强化）。  
  
  
Q18:我有一个自由动作进行一次属性检定，在一场景内获得出目点攻击加值的能力，能否通过多次应用来令自己获得更高的加值？  
A：不行，对于同一来源（即相同资源带来的能力）具有持续时间的效果起效后，只有在这一效果的持续时间结束后才可再被同一效果所再次影响。也就是说，对一个持续时间为2轮的效果而言，无法在享受这一效果的第一轮后接着使用来令这一效果的持续时间延长，而必须等到持续时间结束后重新使用才可以再度享受效果。
