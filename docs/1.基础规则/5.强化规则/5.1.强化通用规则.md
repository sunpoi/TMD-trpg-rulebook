# 强化通用规则

  

强化通用规则  
  
强化是指在每场影片结束后/幕间进行的增强自身的行为，通过资源购买强化以强化自身。  
通常而言，强化分为内在强化和外在强化两类。内在强化一般是指直接作用于轮回者本体，永久性存在，且会提供属性点的强化，包括血统/训练、改造/功法、瞳术、技艺/称号；外在强化一般是指被轮回者购买并利用的物品、装备等资源，在丢失这些物品/装备时便会失去对应强化，外在强化包括武器、防具、奇物、物品、载具、随从等。  
内在与外在强化的通用价格参考支线、分数、XP与金钱一节，未注明价格，只写明支线等级的资源通常被视为通用价格。  
购买同一资源条目下的其他资源：若没有特殊说明，某一资源条目下的其他资源需要轮回者在购买了主要资源后才能进行购买。（主要资源是指资源条目下第一个可供购买的资源）  
无论是何种类型的资源，在没有特殊注明的情况下，必须逐级购买。在没有特殊能力注明的情况下，轮回者无法在影片中购买强化以强化自身。如无特殊能力或特殊说明，每个轮回者只能具有一项血统/训练强化、一项改造/功法强化、一项瞳术强化。  
  
内在强化会开启技能等级上限、并提升轮回者的最高支线等级。内在强化通常分为以下几种：  
  
\-血统与训练：  
血统是与生俱来的，先天具有的某种能力，只要拥有某种血统，即使不加以锻炼也可以掌握血统带来的力量。  
  
\-改造与功法：  
改造是指对轮回者身体进行的永久的，不可逆的改造，强化自身能力，或是直接改变其物种和特性，一般以改变肉体的形式出现。  
  
\-瞳术：  
瞳术是基于轮回者眼部的一种永久存在的能力，只有具有眼睛的轮回者才能使用瞳术。由于瞳术的特殊性，如无特殊说明，即使轮回者有成千上万只眼睛，也只能强化一种瞳术。  
  
\-技艺与称号：  
通过后天学习、长期锻炼而习得的内在强化被称为技艺或称号，技艺与称号能同时拥有多项，并附带一些技能树或是专长作为额外解锁的可购买项，技艺与称号是构成轮回者能力的主要组成部分。  
  
\-能量：  
能量包括能量池及应用方式，单独购买的能量一般会具有能量池应用。能量如无特殊说明，则一般可以复数兼修。能量根据其记述方式还会提供恢复方式。  
  
外在强化不会开启技能等级上限、也不会提升轮回者的最高支线等级。外在强化通常分为以下几种：  
  
\-物品：  
物品中包括武器、防具、奇物及物品本身四种类型，消耗品也属于物品。物品一般会提供一些基于外在基础的能力，在丢失物品时，也会随之丢失它的效果。  
  
\-载具：  
载具中包括轻载具、重载具、生物载具、战斗机器人及随从，具体请参考载具规则一节。
