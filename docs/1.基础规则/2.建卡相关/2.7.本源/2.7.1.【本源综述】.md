# 本源综述

  

本源综述  
所谓本源，即末法时代中，轮回者行走求生而逐渐觉醒的心源之力，技能方面的强化会让轮回者逐渐找回这股力量，以保证自身在末法时代结束之前还有足够的能量继续行走。  
除3级技能外，每达到1项技能的提供附加等级，轮回者就会获得3点本源点，可以在各本源体系中挑选本源主流与本源支流强化之。  
本源主流的强化需要至少拥有2项该本源的支流技能，强化一项本源主流需要12点本源点，强化一项本源支流需要3点本源点（或其标注点数），强化一块本源碎片需要1点本源点，同一大类的本源碎片最多可以强化该大类下属系本源支流技能数\*3块。  
（例：轮回者魔术师拥有9点本源点，他首先购买了心智类-兵者诡道中的终局战术和生理类-披荆斩棘的科技蛮族，一共花费了6点本源点，接下来他可以强化3块心智系本源碎片或者生理系本源碎片，但无法强化互动系的本源碎片。）  
本源技能共分心智、生理、互动3大类，轮回者可以拥有多系本源技能，这些本源技能之间互不冲突。
