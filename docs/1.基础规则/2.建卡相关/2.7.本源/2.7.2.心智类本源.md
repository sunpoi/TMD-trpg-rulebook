# 心智类本源

心智类本源主流：  
1.兵不厌诈：  
在影片开始时选择这项本源主流后，在一场战斗中，当：  
敌人具有过异常点数/异常状态、或场上存在过由你造成的幻象、或存在过由你造成的环境效果、或已有敌人被解析过、或使用过打断性能力、或已有敌人流失过能量/意志时，根据以上效果数触发下列效果，高级效果会包括低级效果。

1种：你获得等同于具有兵不厌诈触发效果种类数\*1+1级的环境来源免疫（无支线1/D2/C3/B4/A5 最高为A级）。

3种：每个行动轮开始时，你额外获得只能用于心智类检定的2AP，这些AP在你的回合结束时立即消耗。

5种：一名携带异常点数或被异常状态影响的敌人因你的攻击而死亡时，由你选择另一名敌人，使用你的攻击检定与其进行意志豁免对抗，若你胜出，则令其承受那名死亡敌人死亡时携带的异常点数数量或异常状态，标准豁免（若无则仍为意志豁免）。

2.读万卷书：  
在影片开始时选择这项本源主流后，你在心智类技能上每具有1点平均附加成功（四项心智技能均有1点附加成功或附加成功总和为4），就根据以上效果数触发下列效果，高级效果会包括低级效果。当你拥有法术书/魔导书/神话典籍，在计算本能力的效果时，心智类技能上的平均附加成功就视为增加1点。  
1个：你获得等同于拥有的平均附加成功数量等级的影响心灵效果免疫（无支线1/D2/C3/B4/A5）。  
现在，你始终视为拥有一项你未获得的1级法术/奥法真理专长，获得读万卷书时立即选择。

3个：获得读万卷书时立即选择选择一种你未掌握的派系，在你达到这一条件时，立即掌握其最低等级的造诣，若你已掌握所有种类的学派，则选择一种已掌握且不为最高等级造诣的派系，提升一级造诣（万变系无法选择这个效果）。  
现在，你始终视为拥有一项你未获得的2级法术/奥法真理专长，获得读万卷书时立即选择。

5个：你使用的所有法术/仙术/神术视为具有等同于你拥有的心智类本源支流\*1点法术强度。

3.神来一笔：  
选择神来一笔需在影片开始前就选择，且进入影片时不能更改。每场影片幕间限一次，若你在幕间进行过同级及以上的造物，则根据你的每次延长检定造物结果触发下列效果：  
第一次（成功）：在下一场影片中，你的造物检定的主属性将获得免疫【劣化1】。  
第二次（成功）：在下一场影片中，你的造物检定的主属性将在检定时获得每天一次的重投机会，但必须接受第二次的结果。  
第三次（成功）：在下一场影片中，每天一次，你可以进行一次智力+器用，不具有附加成功，不能获得任何加值的检定，获得等同检定出目的医疗点，这些医疗点仅能用于进行快速维修。  
第四次（成功且合计超过需求1/2）：在下一场影片中仅一次，若你的造物检定主属性进行的某次检定具有【毁灭】/【湮灭】/【回春】/【起死】中的一个，则你可以选择令其无条件触发。  
若你在幕间进行了多次造物，则取结果最好的一次结算。

4.机器学习：  
在影片开始时选择这项本源主流后，若你具有解析能力，则你的解析能力将无法再指定解析结果；你将无法再进行群体解析；进行一次解析时，只能同时解析出至多三种信息。这一本源只能在使用者为构造体时才能生效。  
（例：玩家【逆位】命运之轮具有解析被攻击者【防御】的能力，在获得机器学习后，玩家【逆位】命运之轮进行的所有解析对应的信息将为随机解析，可能解析结果为被攻击者的【攻击】能力，或某项招式的信息，由主持人决定，不会解析出重复结果）  
根据随机解析的结果，获得以下效果，每个效果每个战斗场景对于每个目标只能触发一次，所有效果均可由你选择触发：  
解析结果为攻击构成：仅对于那个被你解析出攻击构成的目标而言，你对其进行的攻击具有【加骰上升一级（最高为8）】。此效果将持续存在一个场景。

解析结果为防御构成：仅对于那个被你解析出防御构成的目标而言，其对你进行的攻击需再进行一次重掷，并接受后者结果。此效果将持续存在一个场景。

解析结果为某招式信息：仅对于那个被你解析出某招式信息的目标而言，你可以选择其中一项非攻击技，学习之直至战斗场景结束。ST有权禁止此项能力。

解析结果为某特性：仅对于那个被你解析出某特性的目标而言，你可以选择一项该目标具有但你没有的，ST允许的特性，获得之。此效果将持续存在一个场景。

已获取所有解析结果：仅对于那个被解析所有结果的目标而言，你对其视为还具有独立的生命值，其上限为你的结构值，在其对你进行攻击时，需先攻击独立的特殊生命值，直至消耗殆尽。你不能治疗这些独立生命值，也不能与其互动。

心智类本源支流：  
1.终局战术  
若场景中仅有你一名轮回者，则此效果开始生效直至场景结束。本场景内，你的攻击命中后会随机揭示一项对方已有的攻击加值、防御类型相关的信息。当你揭示了对方至少两项信息后，对方对你进行的攻击将承受【劣化1】。此能力是一个等同于心智类本源支流数量等级（无支线1/D2/C3/B4/A5 最高为A级）的信息来源的解析效果。

2.自学成才  
购买时从心智/生理/互动类本源中选择一种，不可更改。  
每场影片结束时，若在获得影片奖励阶段获得了2000点分数及以上，则立即获得所选择类型的1点本源点，并再获得该类型对应技能类型的1点自由技能点。

3.幸存者偏差  
若战斗场景中有队友或自身机运检定出现无加值10及以上或1及以下时，则此本场景内，你的下一次非基因锁检定可以额外重投一次，两次结果由你任选一个生效。这一效果一轮只能生效一次。

4.混沌理论  
若你的攻击或能力中存在需进行随机判定的不确定效果，且在使用时触发了不确定效果（无论是正面还是负面），则你下一次转化花招动作不消耗AP，并投掷rd2，出目为1则改为转化成一个自由动作；出目为2则在转化出花招动作的基础上额外获得1点只能用于转化花招动作的AP。这一效果每轮只会触发一次。

5.操控现实  
当你受到一次攻击时，如你还具有可以在本场景内使用的传奇智力效果，则你可以对本次攻击使用传奇智力，改为将攻击者的检定中的等量成功骰转化为失败骰。这一效果一轮只能使用一次，正常占用传奇智力使用上限。

6.了如指掌  
你可以看懂并阅读所有你见到的文字，即使你从未学习过这种语言，但不代表你能使用它；在观察一件你不知道为何物的道具或物体时，你可以得知它的名称和简单用途及这件物品的支线等级，但不能得知如何使用；当你进行回忆检定时，你会额外获得1DP回忆检定上的表现加值，此外，若ST允许，则你可以略过一些简单的回忆检定而直接想起某事。

7.稳操胜券  
当你释放一道来自于魔法书/魔导书/神话典籍的派系法术时，若你同时掌握了超过2种派系，且这些派系的造诣等级均超过或等于你使用的法术等级，你获得一种额外的增幅方式，额外支付该法术消耗的能量，你可以选择至多一种非此法术的派系加入这次攻击之中。若如此做，这道法术额外获得你的传奇智力\*1点法术强度。

8.奇门风水  
若你具有东方能量池，且掌握以东方能量驱动的法术/神术/仙术，则你在释放符合奇门风水条件且具有攻击范围的法术/神术/仙术时，还会在法术持续时间内额外制造一片范围等同于法术范围的C级环境来源的困难地形。

9.费诺斯定理  
若某战斗场景中存在AP点数多于24的敌人，则此效果开始生效直至所有满足条件的敌人死亡。你成功造成伤害的对象需要进行意志豁免，DC为伤害值，失败则在计算AP值带来的可额外转化的所有动作数量时，所需的AP值翻倍计算，这个效果持续一轮。  
此能力是一个等同于心智类本源支流数量等级（无支线1/D2/C3/B4/A5 最高为A级）的信息来源的【迟滞】效果。

10.黑暗启蒙  
一天一次，你可以使用电子设备连接局域网，并进行一个为期4小时，每1小时检定一次的延长检定，使用智力-器用-专业：计算机使用进行检定，延长检定中每次达到10成功数时，你可以通过网络找到一条有关于你想了解的信息内容；每次达到20成功数时，你可以再找到一条关于某人的某项能力。4小时结束时，无论还差多少成功数生效下一个效果，都立即结束这次信息搜寻。获得的信息由ST决定.此能力是一个等同于心智类本源支流数量等级（无支线1/D2/C3/B4/A5 最高为A级）的信息来源的解析效果。

11.谋划专家  
每个战斗场景限一次，若你执行过紧急规避或紧急守护，则你下一次同样执行紧急规避或紧急守护时，立即获得X点偏斜防御直至战斗结束（X为拥有的心智类本源支流数\*1）。

12.心灵堡垒  
每当你的属性等级/技能等级/专业等级在某场景内降低时，少降低1级，这一效果每场景至多触发传奇智力\*1次。你始终忽略X点心智类属性及其相关检定的减值。（X为拥有的心智类本源支流数量\*1）

13.未雨绸缪  
每场影片结束，当你获得见证点时，若获得时你每具有未使用的2见证点，就可以获得4点仅可以用来购买专长的XP。

14.深谋远虑  
战斗场景开始时，立即选择接下来的某一轮（最低为第1轮），那轮你的回合开始时，你获得等同于选择轮数\*2点偏斜防御直到场景结束。选择的轮数最多不能超过X轮。（X为拥有的心智类本源支流数量\*1）

15.书页风暴  
当你释放一道来自于魔法书/魔导书/神话典籍的派系法术时，若这道法术在此战斗场景中未被你使用过，则你在本场景内立刻【激昂】、【激昂】。  
若你拥有其他【激昂】能力，则转化为2点本场景内使用法术的法术强度。

16.精神过载  
当你首次支付心智类属性的能量时，你可以选择消耗1点意志力，以此代替支付的至多4点心智类能量。  
当你不以意志力的基础用法消耗意志力时，你还可以消耗至多4点心智类能量，以代替支付1点意志力。  
这两项效果每轮只能使用其中一个。

\======================  
心智类本源碎片：  
1.获得1点心智类技能检定上的增强加值。  
2.指定一种心智能量，提升6点能量上限。  
3.获得0-6点意志豁免检定内在加值，每次意志豁免前先投掷此项选择次数的RD5+1决定获得的加值数。  
4.心智类能力射程增加2米通用射程。
