# 相貌

  

相貌

较为特殊的是，建卡风度也会决定人物的初始相貌，人物的初始相貌最高不能超过建卡风度等级。  
  
相貌是评定一名轮回者长相的关键指标，在建卡相貌基础上每高出5点就会获得1点全检定上的加值。过高的相貌可能会导致轮回者无法在影片中正常行走，相应的，在表现力上，过高的相貌可以让一名轮回者更容易讨得他人欢心（特别是异性）。  
相貌大致评级标准：  
1-3正常人水平  
4-5俊美  
6+倾城级
