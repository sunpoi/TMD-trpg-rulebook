# 建立人物卡以继续游戏

  

continued  
建立人物卡以继续游戏  
在建立人物卡的开始阶段，首先自然而然的需要一张人物卡（[请参照后录](2.2.范例空白卡.md)）  
无论如何，在建立一张人物卡时，都需要首先在意的是人物的设定而非强度，不要为了选择某些开卡方式而刻意去追求强度车卡，在娱乐游戏的基础上适当进行这一活动。  
关于人物姓名、种族、国籍、身高、体重等基础设定部分，本质上以建卡者喜好自行填写。  
概述部分填写人物背景，角色概念部分填写人物外观。

* * *

在建卡前，需先选择是否启用天命，若选择启用，则以方式1进行开卡，天命内容详见2.8 天命一节。若不启用，则使用方式2进行开卡。

属性部分：  
▓▓生理属性  
■力量  
■敏捷  
■耐力  
▓▓心智属性  
■智力  
■感知  
■决心  
▓▓互动属性  
■风度  
■操控  
■沉着  
在建卡之初，便需将总共21点自由属性分配入其中，当分配结束时，若某一种属性上的等级仍为0，则这张卡在所有包含那种属性的检定均无法进行检定，自动视为失败且出目为0，直至那个属性上获得1级为止。在分配过程中，属性的最高等级不能超过5，当建卡时属性希望从4点提升至5点时，这次提升需要1点额外的自由属性。  
  
▓▓▓▓技能  
▓▓生理系  
■运动  
■求生  
■射击  
■武技  
  
▓▓心智系  
■学识  
■器用  
■手艺  
■专注  
  
▓▓互动系  
■洞察  
■隐秘  
■表达  
■社交  
在建卡之初，便需将总共24点自由技能点及12点自由专业点分配入其中，当分配结束时，若某一种技能上的等级仍为0，则这张卡在所有包含那种技能的检定均无法进行检定，自动视为失败且出目为0，直至那个技能上获得1级为止。  
在分配过程中，任意技能的等级至多不能超过3级，任意一个技能在建卡时希望从2级升至3级时都需要1点额外的自由技能点。  

建卡结束后，若仍然存在0属性或0技能，那么ST有权拒绝收卡。  
  
在建卡结束后，[属性](2.4.属性概述.md)的购买为1级500点，[技能](2.5.技能概述.md)的购买为1级500点。  
  
在分配技能后：若为方式1进行开卡，则获得6专长点以购买任意专长。若为方式2进行开卡，则获得9点专长点以购买任意专长；选择专长最基础的需求是符合人物特点以建卡。

在购买专长时，1级专长需以1点专长点购买，而2级专长需以3点专长点购买，其余参考[支线分数XP与金钱](../5.强化规则/5.3.支线分数xp与金钱.md)一节。

在分配完属性、技能、专业、专长后，按照范例空白卡上的公式计算衍生属性值，其正常成年人在建卡时体积视为5，并且无法超过5，未成年人可以视情况减少。（1体积=40cm）
