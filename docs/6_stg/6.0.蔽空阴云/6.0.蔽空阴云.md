# 蔽空阴云

  

我像麝鹿一样在林萌中奔走，为着自己的香气而发狂。  
夜晚是五月正中的夜晚，清风是南国的清风。  
我迷了路，我游荡着，我寻求那得不到的东西，我得到我所没有寻求的东西。
