# 死人经

  

死人经  
D+600  
一次性功法  
**_魂升九天，魄落九渊。生者悲苦，死者平安。_**  
**_欲令人死，先由己死，形若枯槁，心如死灰，坐忘我形，杀生者不死，生生者不生。  
_**修炼死人经的前提为已经掌握内力，并使用剑进行施展，死人经内记载着两部分内容，一部分为剑招，一部分为内力运用。  
死人经中虽记载着二十九式剑招，但其实只有一式，练成后的剑招无不怪奇诡异，不是杀人，就是杀己。  
练习死人经的内功需要以杀人为目标，杀更多的人来砥砺剑道，磨练杀意。因此，在影片中，运用死人经外功杀人成为了它独特的武学精熟。死人经的武学精熟没有上限，每场影片结束时，若《死人经》的修行者在影片内斩杀了8个非友方或中立的单位，那么死人经的修炼就会再进一层，在使用死人经推动武学外功招式时就会获得1点检定上的附加成功。  
相反，若没有杀足够的人，那么死人经的功力就会倒退，失去一层死人经武学精熟得到的好处，当死人经的武学精熟好处全部失去且还需扣除时，则失去死人经的修炼功法，需重新购买并学习。  
死人经中记载的外功招式虽有二十九种，但实则只有一种，其核心思想便是每个人的身上都有可以感知却无法看见的气，用剑斩断对方的气节，就可以达到杀死敌人的效果。修炼死人经会为修炼者提供一种特殊的感知能力，这可以让他即使看不见敌人，也能感觉出淡淡的“气”，随着敌人的运动，他能感知到气也在运动，这视为一种弱感知效果。  
死人经的剑招并不是固定的，因此，死人经也可以结合在其他外功剑招中使用，在使用时，视为正在攻击一种特殊的部位，承受3点部位攻击减值，同时被攻击者正常获得对应的闪避加值。这个部位是概念上的部位，并不能被直接看见，被称为“气节”，攻击气节并不会带来异常点数，也不被视为C级创伤来源，而是会将等同于死人经内功修炼层数点严重伤害等量转换为韧性伤害。  
同时，修炼死人经也会对修炼者产生极深的影响，会让修炼者不由自主的好杀，充满负面感情和杀意，而随着深入修炼，这种影响也会越来越深，难以自拔，每修炼一层死人经，修炼者就会不可避免的获得1点无法豁免永久存在的亢奋点数。
