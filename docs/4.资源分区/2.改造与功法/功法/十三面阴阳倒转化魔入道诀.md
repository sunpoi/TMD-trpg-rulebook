# 十三面阴阳倒转化魔入道诀

名称：十三面阴阳倒转化魔入道诀  
分类：功法  
本质：魔幻本质

概述：  
限制1：非人类种族无法兑换，因特殊能力获得其中效果，必然心魔入体。  
限制2：见利忘义者，心神不定者，或目光短浅者，若无有真心反省，无论自述聪慧几何，机缘几重，修则入魔，无有他法。  
书记：  
本法乃望道仙人李氏亲传，以人心百变道化魔心独一，合四九重数，三六变化，噬魔气反化灵气，以助修行。  
非人者，心窍塞闭，少者，思绪亦有不通，反受魔制，故不宜修行。  
此法本做无上法门，然，李氏圆月飞升时，其下弟子受邪修红中氏损害，道统零落，勉强留存残篇，又经修补，与原法迥异。

D级  
价格：D+600  
描述：修此术者，善聚而不善散，故修功至简，而修心为余。  
道境修行：炼此法者，当炼心为先，炼人为先。

在修行之前就有着能够克服外魔的心境，与识破伪诡之物的慧眼，二者缺一不可。

特性：【聚灵气】  
效果：  
你获得C级幻象与影响心灵免疫能力。

【聚灵气】：  
初步了解修炼，《十三面阴阳倒转化魔入道诀》不取天地，化无形魔气，在丹田之中汇聚灵气，成就自己，最终超脱。  
开启真灵能量池，但不获得基础用法。

真灵：真灵是一种类似真气的东方通用能量，你开启了真灵池，大小相当于（耐力+决心），真灵会在长休息后回满，或通过打坐调息的方式，在一次短休息中恢复关键属性判定成功书都真灵，每天恢复量最大值三倍。  
选开：若身处黑暗生物聚集，或邪魔外道之类的场景下时，每三个小时就可以恢复等同于上限的真灵。

C级  
价格：C+1200  
描述：魔者，神也，无形亦无质，念生则起，念绝则亡，凡尘世多少生者，则倍有多少魔者，几近无穷。不知何来，无入轮回，亦不能不知何去又能化诸伪，祸乱善法、道心，使人入魔化魔，恰如人心周转。

特性：【化相入伪】  
效果：  
【化相入伪】：  
你视为具有C级心灵影响来源的读心能力，对于没有相应免疫的对象，在你的眼中，就如同在阳光下一样无所遁形。  
即他人心中活动的情绪会如渲染在白布上的油彩一样，但即使颜色再多再模糊也能轻易辨别，只靠看颜色你就能了解到对方目前的心理活动，能够任意挑选一个颜色（情绪）的方向进行具体读取心中包含着那段情绪时想到的事情，会如漫画一样浮现在白布（他人的灵台）之上，虽然只有你看得到。  
这皆需要以自由动作流转心念，运作功法威能，此外一旦这个能力因心灵影响而被免疫，你会看见一张白布，或者对方根本无心智，你只能看见黑色而看不见其他颜色。

此外，在感知属性上，还会因此获得2DP加值。

B级  
价格：B+2400  
描述：藏阁主评之，如今天下，人心浮躁，心定稀缺。  
故见修此法者，少有可成大道，皆做邪修，可怜火旺仙君，大道零落如今，魔人之分仍无退消。  
化魔入相：修此道者，魔中之魔，更胜常魔千百倍。

效果：  
你具有B级幻象来源的变形能力，能够如外魔般变化自己，身躯能够以不可思议的方式变化，有一种如邪魔之流的大恐怖，见之常有不适。  
具体而言，你能将皮肤与躯体捏橡皮泥一样的塑造成一个你曾经杀死、见过的单位，且其他人在观察你时，都会误以为是你所变化的单位。  
如果目视你的单位无视B级幻象来源/变形能力的效果，则会发现你的外型是一个单薄的人影，只有脸皮处血淋淋的构成了一个“中”字。  
此外，你还可以一个自由动作由肉体变为虚体或由虚体变回肉体，这是外魔的特征之一。但在使用本能力的那一回合内，不能立即变回虚体/肉体。

在与其他单位战斗并取胜后，你可以选择留手不杀死对方，且对方不具备对应的免疫等级时，对其进行道心种魔。  
此时，你以风度/操纵+说服+交涉蛊惑，或选择以智力+学识+道法强行施法，对方以意志豁免对抗，成功则“种魔”，允许进行修改对方的记忆，并将其合理化，除非这种修改威胁到对方的生命或是严重冲击三观。  
这视为一个B级别影响心灵来源的思维控制能力。

关于记忆修改，你可以在对方的记忆里做添加或者模糊化，对方通常会对此深信不疑/无法正确认知。  
举例而言，本来是和山贼的战斗，你可以让对方获得一段你是他远隔万里老乡的青梅竹马的记忆，并且有着基本完善的经历，今天相遇后在某个地方叙旧之类的，虽然实际态度可能因为对方性格有差异。  
而对应的，目标会无法正确认知其中不合理的部分，尽可能的在记忆中模糊掉这次与你遭遇时的正确情景，打斗造成的伤口会当做今天劫道时和其他人战斗受的伤，会尽可能的找个合理的理由对自己或者他人进行说明，但其他人是否相信就是另一回事了。

A级  
价格：A+4800  
描述：你终于明晓了一丝功法本来的真意，不再执着于魔，或终有一日，这门功法得以补全。  
神通引：炼魔即炼人，去魔即去我，魔我两忘，唯念存之。  
化伪做实，化意为物，炼假为真，欺天瞒海，以证大道。

效果：  
虽然你仍没有那么强大的能力，但仍可以化身真正的魔头，这需要满足以下条件：  
对方已被“种魔”（如果ST允许，剧情中被其他人以其他手段种魔的NPC也能满足要求）。  
此时，你以风度/操纵+说服+交涉再进行蛊惑，或选择以智力+学识+道法的施法，对方以意志豁免进行对抗，若其对抗失败，你将以外魔无形之姿遁入其灵台，这需要具备虚体性质。本能力为A级附身来源的效果。

这个状态下，被附身/心控者无法察觉到自己被“附身”，记忆会自动合理化（如B级举例所述，把与你相遇被施法的事情基本忘记，并为自己为何待在这里找一个合适的理由）。  
此时你视为全遮蔽状态，这是一个A级附身来源的胁迫效果，你会藏匿于那人的脑中，共享那人的视野，并且随时如同满足了B级思维控制前提下的对某些发生的事情进行模糊化，或者添加不存在的虚构记忆，如果ST认为添加的内容过于不合理，可以要求你承受惩罚值的再次判定豁免对抗。

如果对他发动攻击并造成伤害，则自动解除附身/心灵控制，且被弹出灵台。  
若施加负面效果而可能陷入生命危险、进入异常状态严重/毁灭效果时，也会解除这个心控控制/附身效果，并且被弹出灵台。  
驱使对方做的事情会使他受到极大的冲击，如：自己的生命陷入危险，没有杀过人的人去杀人，或者要求他去屠杀，也会自动解除，并且被弹出灵台。

当有其他人发现时，若那人受到提醒，双方立刻以意志检定对抗，对方胜出，你被弹出灵台。

虚体无法被听到/闻到，通常意味着他人需要以视觉或者其他方式来发现附身状态下的你。但即使他人看见了你存在于被控制的人脑内，除非其免疫A级别影响心灵或B级别思维控制能力，否则须以感知+洞察+侦查或者智力+学识+仙术/魔术，来对抗你的风度+潜藏+伪装，一旦对抗失败，则视为被外魔惑乱了心神，无法做出正确的认知（即无法发现附身）。

若你因各种原因死亡时，则立刻转换为灵体，选择一个敏感范围内曾经被道心种魔过，且后续不曾察觉或解除的角色，视为发动A级效果一般在他身上藏匿，期间你除了观察和移动什么也做不了，这样为期一周后会正式复活，你从其脑内复归为虚体。期间你以意志值作为生命值，意志池清空则你立刻被摧毁，视为复活失败。这是一个A级别附身来源的复活效果。
