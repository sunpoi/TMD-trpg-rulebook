# 逍遥真意

名称：逍遥真意  
分类：功法  
概述：凡人碌碌所求，当真是为了财侣法地本身吗？  
凡人求财，要的是生活富裕。修士求财，为的是修为精进。  
若言，凡人为何要让求富，修士为何要求进？或许这又能引得诸多解答，但归一言：

“凡人碌碌一生，是为过上自己希望的生活。修士与天争道，是以行己所欲行之道，即【得一切想得之物，成一切想成之事】，我称它为“逍遥”。”

D级  
价格：D+600  
描述：“下到饥民贱隶，上到天尊魔皇，若终有不成之事，便都是一样不得逍遥。反之，清心寡欲者，纵是草庐劣田，诸道不通，也照样怡然自得，获了逍遥。”

特性：【逍遥值】  
效果：  
【逍遥值】：逍遥值代表“践行己道，满足所欲”的动力和成果，因此逍遥值不具有正常恢复模式。  
在这一阶段，逍遥值的上限等同你的内在支线等级\*1点（D1C2B3A4S5），获得该功法并初次进入影片时，逍遥值默认为【0】。

你需要明确一项符合自身核心欲望（修炼、复仇、守护、探索、享乐等），每当在推进该目标取得显著进展，或完成一次目标时，你获得1点逍遥值。在这一阶段，你的目标需为可以是微小（例如仅仅是当天吃饱饭等目标），但必须是真切的个人愿望。

逍遥值不会自然下降。你每具有1点【逍遥值】，就可以在使用意志力增加检定DP时，额外再获得1点同类型的加值。

C级  
价格：C+1200  
描述：“然而，这般轻得的逍遥是成不了什么事的。为了求得逍遥，就去压制自己的欲望，以我来看是不可的，如同缩纳天风，其形增减，但其量无变，纵被强压暂得，若遇丝隙，即功亏一篑，届时其造成的毁坏，非一风耳。”

特性：【欲壑难填】【乘风破浪】

【欲壑难填】：你的逍遥值上限扩大决心属性的四分之一。你获得逍遥值的目标范围将会扩大，这会使你的核心欲望还会包括提升实力、获取关键资源、完成有意义的任务，只有进行这些行为你才能获得逍遥值（通常由ST判断，不过显然的、你通过兑换提升实力时可以满足上述条件），当你完成这些进阶目标时，所获得的逍遥值从1点改为【传奇决心】点。

但是，当你主动拒绝，或强行压制一个符合本心且有能力追求的欲望（尤其是功法中提到的“财侣法地”相关欲望）时，你的逍遥值将会下降【传奇操纵】\*1点。

【乘风破浪】：在你追求一个明确且符合本心的短期目标（如击败眼前强敌、夺取关键物品、通过某项检定）时，你可以在自己的回合内使用一个花招动作并消费X点意志力，进行一次1DX的检定，X＝当前逍遥值。

直到该场景结束为止，你在符合目标的相关检定中获得等同检定结果的表现加值，因此获得的表现加值不会超过4点。同一场景内你只能进行一次这样的检定（并且显然的，这是一次自然骰，无法享受任何其他加值，也无法因效果调整出目或重骰结果）。
