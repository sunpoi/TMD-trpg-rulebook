# 炼铁录下属技能树

【可兑换金属】  
概述：以下金属的皆为主神提供的为特供版，专门为炼铁录吸收使用，因此价格也比起常规材料更加昂贵。这些金属等同于产生永久性效果的消耗品，只能吸收一次，多次吸收不会提供效果。

圣宝石  
价格：D+500  
分类：奇物  
描述：一种神秘的宝石矿物，具有能量活性的特殊物质，作用于物品使会使其通体呈现出宝石一样的质感与色泽。  
效果：你在以天生武器攻击时，破魔增加2点，并获得【法破】特性；你在受到一次攻击时，可以支付1点任意通用能量，额外吸收1点这次攻击造成的伤害。

秘银  
价格：D+500  
分类：奇物  
描述：常在奇幻世界中出现的材料之一，这种闪耀着银光的金属虽然比钢铁轻，却有同样的硬度。  
效果：你的基础移动速度提升10米，承受的盔甲减值降低3点计算；你的天生防御提升3点。

黑石英  
价格：D+1000  
分类：奇物  
描述：在修真世界中出现的高硬度材料，通体黑色的石英晶体，有天然金刚石的硬度，掺入锻造使用能大幅度提升其他金属的性能。  
效果：你在以天生武器攻击时，额外获得1点破甲，并能无视至多1点任意伤害减免；你获得1点天生防御，与基于天生防御的DR1/-。

玄铁  
价格：D+500  
分类：奇物  
描述：常在武侠世界中出现的高强度金属，黑色，性状如铁，但更加沉重坚硬。  
效果：你的天生武器伤害提升2点，额外获得【威猛5】。

疫金  
价格：D+500  
分类：奇物  
描述：在核废土或后现代人类文明断层时代有可能出现的奇异金属，被它碰到的人身体无不溃烂发臭，盖革计数器在疫金旁也会响个不停，某些新兴宗教甚至会供奉这种特殊材质。  
效果：你在以天生武器攻击且命中敌人时将额外附带1点不可豁免的毒素点数；在你进行了一次招架后，将对触发此次招架的攻击者造成1点不可豁免的毒素点数。

陨铁  
价格：C+1000  
分类：奇物  
描述：一种特殊的金属，本质是具有特殊结构的钢铁。陨铁内部的铁原子堆叠成多角螺旋的晶粒，这使得陨铁内部被弹簧状的细微组织所充斥着。  
陨铁可以承受外界的冲击力，并吸收所有的动能，存储在内部的弹簧状组织内，转化成其内部对外力的抗性。  
效果：每轮一次，你受到攻击后获得DR1/-，每五的倍数次会DR2/-，而不是DR1/-。  
次数抵达10后无法将继续获得，该效果仅在进入战斗后有效，脱离战斗后将失去获得的效果。

万年寒铁  
价格：C+1000  
分类：奇物  
描述：一种天地间诞生的奇特矿石，外表为带有微弱蓝色的金属，即是打造装备时掺杂一些都足以大幅度提高其品质，其性质冰寒刺骨，即使是间接接触都会导致严重冻伤。  
效果：  
你的天生武器伤害类型转变为寒冷伤害。若在攻击时额外支付1点能量，则在命中后给予目标【寒冷易伤：3】；对你造成伤害的单位将承受1点反伤，不可豁免但可以被伤害吸收/减免等效果影响，反伤伤害类型为寒冷伤害。

流银飞金  
价格：C+1000  
分类：奇物  
描述：其形如水，其状如银，刀切不断，火烧不化，若是舀出一小部分也不会断开，始终会有细丝相连，是一种绝佳的锻造兵器的好材料。  
用它为辅佐材料锻造出来的装备，不仅更加锋利，更具灵性，并且可以缓慢的吸收灵气，自动修复装备上的损伤。  
效果：  
你获得【快速维修3】，维修点数44。快速维修可以如同快速医疗一样生效（但无法对除你以外的其他生命值生效，对结构值有效）：你获得【异常点数恢复1】，医疗池24点。

高达尼姆合金  
价格：C+2000  
分类：奇物  
描述：具有的在极近距离中也能把连续发射的120mm炮弹弹开的刚性，而且它的耐热，耐磨损性以及轻量性，都使得在地球圈外等残酷环境中和在磨损严重的关节，轴承等领域中被有着很高的实用性。  
效果：你获得B级环境免疫的效果；你获得等同自身耐力一半的天生防御，以及基于天生防御的【防弹】特性，与【全伤害吸收1】。

【专属能力】  
锻造学  
价格：D+1000  
类型：技艺  
分类：被动  
描述：开发炼铁炉的功能，进一步加强自身的锻造能力。  
效果：你指定一项造物技能或专业，每经历一场影片，就能在对应技能或专业相关检定上获得1个额外附加成功；你在造物相关检定上获得1DP修行加值。
