# 虚实之握

虚实之握（改造）

这个实验的目标之一就是要克制源石感染对该个体产生的不良影响。  
他们做到了，即使是暂时的。  
从立项开始，他们就想要生产一个没有被矿石病感染的感染者，一个不用施术单元的术师，一个无需精深源石技艺理论培训的大师。  
感染器官不会提高她体内源石结晶的密度。这个特殊个体，某种意义上，“并没有被感染”。  
——————————  
D级  
价格：D+600

人工施术单元：这个特殊个体也许可以被称作是感染者，但这很难说是矿石病。  
处于她脑干处的感染器官并非天然生成。没错......人工植入。参考过Raidian的探测报告以后，我们可以确定，这个装置是百分百的人造产物。  
你的脑内被植入了一个特殊的施术单元，通过该单元，你拥有了一种新的能力，当然，一切都是有代价的。

从现在起，你的意志上限永久-1。

你获得一种新的攻击方式，你可以用意念将可投掷武器举起，随后扔向你想攻击的目标，该攻击方式等同于投掷，但使用【决心+学识+投掷+源石学（学识下属）】来进行检定，这种攻击将附带【破甲5】的效果。

当你的攻击完毕后，你也可以将你进行攻击投掷的物品再利用你的能力取回来。使用投掷武器时，投掷武器视作具有【回力】。

【记事】：你对实验室感到恐惧。你对医疗器材感到憎恨。你还能轻松把强度数倍于标准护甲的装置在半空压成拳头大小的一团…….在那个施数单元的影响下，你的回忆检定将必定失败，当你的意志低于你意志上限的一半时，你获得【狂怒1】。

——————————  
C级  
价格：C+1200

【思绪立场】：她的法术既能延展成一张能覆盖很大范围的、感知范围内物体运动状态的“网”，范围内的个体则一般对这种“触摸”毫无知觉；她的法术也能单纯地作用在单一个体上，我觉得可能很像一只大手......即使要求她把法术控制在最小范围内，也会至少影响好几平方米的大小。

你获得一种新的感知方式：思维感知，这视作一种强感知方式，可以精确定位。

那双大手可以握住人和物，同样的……包括你自己。  
你获得飞行能力，机动性为普通。

【冲动阻断】：在现在，你已经可以更好地利用你的力量去“握”住你想要握住的东西，也能更准确的，破坏你想要破坏的东西。  
你的决心检定获得【9加骰】。

——————————  
B级  
价格：B+2400

【思维崩毁】：在测试过程中，根据“手”和“握”这个意象，我们大致推测，她在注意力极度集中的情况下最多能运用“四只”巨大但无形的“手”。测算结果好像是根据法术活性和深水测试来计量的。无论是速度测量还是力度测量，都证明了她的法术力蕴含着庞大的能量，但我们在它运作并打碎什么东西前，连法术形成的征兆都观测不到。

从现在起，那无法被观测的法术形成使得你的攻击拥有B级【措手不及】效果，同时你C级获得的加骰等级上升一级。

【思维膨大】：此刻起，你也无需再为了使用你的能力而停下脚步，如同随手握住一个东西将其扔过去一般，简单，方便，有效。

你获得仅可用于攻击的4AP，且可以在移动的同时进行攻击。

——————————  
A级  
价格：A+4800

如你所愿：为什么是四只，为什么是无形的“手”呢？是因为她就是这么想象的吗？确实，在检查目标结构形变的过程中，我们几乎会觉得，有些凸起就像是指缝的位置......。  
温差较大的情况下，她在使用法术时似乎也会被冻伤和灼伤......至少在神经活动上是的。没有任何症状显现，只是她会说......“冷”......或者“疼”，而且根据她大脑皮层的活跃程度，这是真的。  
我听过一个很不靠谱的说法......甚至有些可怕。  
他们说她的兄弟还活在她的身体里。她的法术就是她的兄弟在......伸出手，拥抱土地，或者用力捶打。  
我不知道这是从谁开始这么说的。真令人不安。

从现在起，你的攻击将不再受到约束，这意味着你的“手”的活动范围已经无限扩大，你的射程范围提高至你的感知范围。

末梢阻断；当那些被你所握住的事物砸向他人时，产生的波动将会令对方受到更严重的伤害。

一轮一次，你的攻击检定出目增加一半。  
你的攻击将附带等值的晕眩点数和肢体妨碍点数，正常豁免。

B级  
价格：B+2000

实体化的思绪会带来重压，也会带来力量。罗德岛工程部的尖端设计在此体现，一切都如你所愿。

该武器伤害再增加7L。

专属技艺：

念力  
D+500

你的伤势将不会自然恶化，同时，  
以一个迅捷动作消耗1点意志力启用，在本场景中，你每在心智类属性上具有1点传奇附加，就获得1点破甲。

投射  
D+500

动作：标准动作  
操控你的“手”，将你的武器以更用力的方式投射出去，你的本次攻击将获得【高速5】和【破甲5】的效果，并获得2dp的速度加值。

当使用该技艺进行攻击时，你的本次攻击将获得【连射】特性，本次攻击将获得【高速5】【破甲5】【熔毁5】效果，并获得2dp的速度加值。
