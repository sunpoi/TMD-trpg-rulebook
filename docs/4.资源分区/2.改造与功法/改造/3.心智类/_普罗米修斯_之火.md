# “普罗米修斯”之火

名称：“普罗米修斯”之火  
本质：特异  
分类：可升级改造  
概述：以盗火者普罗米修斯为名，接受了这一改造的人将获得能够自由自在的操纵火焰的能力，同时也具有极高的火焰抗性，与种种不同性质的能力。  
这种火焰被称为“普罗米修斯”之火，与使用者的生命力息息相关，也会因为强烈的信念而大幅度增加威力。

C级  
价格：C+1200  
特性：【火种】  
描述：最初的普罗米修斯之火并无特别之处，只会不断焚烧受改造者原本脆弱的肉身，提供再生能量，使其身体可以承担使用这一火焰的能力而不至于彻底崩溃。这种再生需要消耗受改造者的体力，主神优化了这一内容，使你不至于因体力耗尽而化作雪白的灰烬。  
效果：  
【火种】  
你开启名为一种通用能量池，关键属性为【耐力+决心】，这种能量池被称作【普罗米修斯之火】，也可以简称为【火种】。  
当你在一次短休息中通过大量进食补充体力后，可以进行一个关键属性检定，恢复成功数点【普罗米修斯之火】。  
当你在进行一次长休息后，【普罗米修斯之火】将自动回满。  
一日恢复的上限为能量池上限的三倍。  
——  
你常时具有3点快速医疗点数，医疗池36点。如果你额外支付2点【普罗米修斯之火】，则在该场景内，你额外获得1点异常点数恢复能力，医疗点24。

B级  
价格：B+3400  
特性：【种子】  
描述：随着对于火焰能力的掌握，你学会了一种新的用法，即塑造火焰本身的性质。枪械，机车，大炮，利刃。一如普罗米修斯将火种赠予人类，人类就此推进了文明的发展，你手中的火正是象征文明与造物的【种子】。  
效果：  
【种子】  
你获得了对【普罗米修斯之火】新的使用方法。  
消耗6点能量，在该场景内，你可以将【普罗米修斯之火】塑造为武器或是载具使用，这些事物不能同时存在，你可以每轮一次的使用自由动作进行切换。  
——  
作为武器使用时，普罗米修斯之火的武器伤害为18L，使用决心+专注：控火+学识：科学进行攻击检定，伤害类型为火焰伤害，无论塑造远程武器还是近战武器，其攻击类型皆为远程攻击，射程15+你的传奇决心\*5米，可以对范围内的所有敌对目标造成伤害，目标使用反射豁免检定而非正常防御流程。  
你可以自行决定武器的外观，但其本质依旧是你操纵的火焰，因此不会真正影响性能。  
——  
当作为载具使用时，普罗米修斯之火视为基础移动速度160米/轮，防御值14，结构值52的轻型载具，加速度为80米，最高速度320米/轮。  
载具形态下，计算米罗米修斯之火的载员最大可以以52体积进行处理（而且神奇的不会伤害到承载的单位），如果在此期间被击破，那么该影片内你无法重新变化出载具形态的普罗米修斯之火，需要回到影片后主神免费修复普罗米修斯之火的变化能力。

A级  
价格：A+4800  
特性：【火衣】  
描述：当充分掌握塑造的力量后，你所能唤出的火焰会逐渐化作内敛的蓝色，虽然在接触时只有温暖的触感，甚至可以塑形化作外衣披在身上，但其性质却比之前更加炽热——无论是物理的冲击，还是冷热的变化，都会在这火焰面前被焚烧殆尽。  
效果：  
【火衣】  
你可以使用自由动作为自己披上一件由普罗米修斯之火塑造的外衣，或者说防具，持续时间永久（可无需动作取消）。其与你穿戴的盔甲叠加计算，提供等同你决心数值的力场防御，以及基于力场防御的【超级坚韧】【适应】特性。如果你愿意在塑造时支付14点能量，则该场景内还会额外获得A级环境来源效果的免疫能力。  
——  
在这一等级下，你的普罗米修斯之火攻击范围可以扩大等同自身传奇决心\*20米，攻击伤害可以转换为等量的燃烧不良点数，正常豁免，可叠加。
