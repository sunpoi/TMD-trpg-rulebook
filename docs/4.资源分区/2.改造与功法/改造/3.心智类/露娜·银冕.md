# 露娜·银冕

露娜·银冕  
改造  
前提：女性，基础外貌不低于4  
描述：这个机体基本上维持了露娜当初被改造时的素体模样，其他外部装备基本都是由帕弥什异聚而成，不但防御力出众而且惊人的轻。露娜能够通过改变异聚体的形状制造出不同的武器，从而实现多种多样的攻击，暴风骤雨般的攻击或是滴水不漏的防守，都只在露娜的瞬念之间。

帕弥什病毒  
「帕弥什」病毒能够直接破坏生物体细胞，或通过操控逻辑电路使机械体呈现被感染状态，并对任何具有人类意识行为的个体，展现出极其强烈的攻击欲望。但是在主神空间必须深入开发才能窥见其一小部分的威能。  
帕弥什病毒的关键属性是操控和沉着  
恢复方式：  
1当人物进行一次长休息，恢复所有帕弥什病毒。  
2当人物进行一次短休息后恢复关键属性较高者点帕弥什病毒。

基础用法：消耗2点帕弥什病毒，使下一次攻击获得2点能量加值

D：初阶  
价格：D+600  
描述：“我将带领他们革新这个世界，而首先，就是绝对不能让空中花园那样的东西继续存在······”  
露娜认为一切的灾祸根源来自于人类的傲慢和自私，他们高高在上，妄图置身事外，甚至还想反过来夺回因为他们而陷入绝望的大地。

【病毒感染】：你开启帕弥什病毒能量池，并获得基础用法  
【初阶构造体】：身为构造体你已经脱离的人类的范畴，你不需要呼吸，你获得【防弹】特性，黑暗视觉。露娜喜欢身处宇宙的感觉，无重力且近乎绝对的洁净，她的躯体总是处于漂浮的状态，你免疫摔落伤害，获得等同于基础移动速度的水面速度

C：中阶  
价格：C+1200  
描述：作为代行者，露娜有资格也有义务选择有可能成为新的升格者的构造体和机械体，但遗憾的是只要极少数能够成功，大多数都只能沦落为感染体，而对于那些未能够得到升格网络眷顾的感染体，露娜也曾为其感到过悲伤  
【极星增幅】：露娜对升格网络力量的运用更为成熟，你获得任一施法专业上的9加骰，获得该改造时绑定专业，每经过一影片，你在施放黑暗/万变派系法术或伤害类型为精神攻击时获得1点附加成功，最多以此效果成长5次。  
【中阶构造体】你被彻底的改造，现在的你除了外形，本质上与人类已经相差甚远，你的生命值现在被转化为结构值，获得普通飞行能力

B：高阶  
价格：B+2400  
描述：“姐姐······如果可以的话，在我睡着之前请就留在我的身边。”  
在阿尔法成为升格者之后，露娜才能真正得到了片刻的平静和安稳——她虽然是升格者的首领，但是在姐姐的面前，或许仍是那个害怕孤独的妹妹。  
【升格网络】：身为升格网络的代行者，借助升格网络的算力，入侵低级的网络轻而易举，在有相对应设备的情况下，你可以随时消耗1点帕弥什病毒，骇入整个互联网，或是直接骇入离自己较近（局域网区）内的机密设备，这是B级网络效果  
【银霭坍缩】：宇宙中每时每刻都有星球诞生，同时有星球走向消亡，与露娜而言，地球的湮灭只是其千万之一，露娜从中感悟出湮灭，你在【极星增幅】中获得的9加骰提升为8加骰。  
【高阶构造体】帕弥什病毒让轻便的构造体也能获得极强的防御力。每经过一影片，获得2点基于闪避防御的dr/-，最多成长3次，第4个影片开始获得2点生命上限，最多成长2次。在你招架时，你额外支付3点能量，使你这次招架生效时，视作已经移出攻击范围。

A：终阶  
价格：A+4800  
描述：露娜曾经下定决心，在毁灭一切之前，她必须要去再找一次那另一位姐姐，让她成为自己的伙伴——即使是要用强迫性的手段。露娜相信等到一切终结之后，她会明白自己的。  
【湮灭聚变】：现在，当你以【极星增幅】中绑定的专业进行攻击检定时，额外获得【湮灭】效果，并且获得等于智力属性\*1的修行加值。  
【代行者】：升格网络拥有革新涤荡世界的力量，在有相对应设备的情况下，你可以随时消耗1点帕弥什病毒， 可以随意骇入所有网络，不再被距离限制随意骇入机密设备，这是A级网络效果。  
【终阶解放】：完全掌握帕弥什病毒的力量，使自身的进化趋于完美，你的结构值提升至生命值的1.5倍。
