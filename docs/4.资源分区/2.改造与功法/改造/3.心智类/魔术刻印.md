# 魔术刻印

魔术刻印（Magic Circuit）  
魔幻本质  
改造  
描述：魔术刻印是将魔术师花费一生来锻炼并固定化的神秘刻印化而成的东西。亦可说是让魔术师家系成为魔术师家系的，魔术性的遗产。刻在里面的魔术，只要魔力流过就能发动。不需要复杂的咏唱手续。运转起来就可以使魔术这辆车跑动，终极的加速装置。也正因为如此，魔术刻印只有在使用时才会浮现出来。

残缺刻印  
D级  
D+600  
描述：你移植了残缺的魔术刻印，其残缺不全的魔术回路没法让你掌握刻印中记载的魔术。  
特性：【魔术回路】【残缺应用】

【魔术回路】：你因此获得【魔术回路】。【魔术回路】是一种变体魔力，可以如常作为魔力相关前置。

在获得魔术回路后，你将获得等同自身（耐力+智力）\*1条魔术回路，每1条魔术回路可以在计算能量消耗时为你提供1点能量，因此获得的能量视为【魔力】；当使用过魔术回路对应的魔力后，那些魔术回路会暂时失效，视为你耗尽了其中蕴含的个人魔力。

恢复方式：  
1、经过一次完整的长休息后，恢复所有失效的魔术回路。  
2、经过一次短休息后，恢复关键属性（耐力/智力）较高者\*1条失效的魔术回路。

此外，你还需要选择一种除万变派系以外的法术派系作为你移植的刻印拥有的刻印属性，你将因刻印属性掌握对应所选派系的D级造诣，一经选择无法更改，并会影响该改造后续特性所指内容。

【残缺应用】：虽然刻印残缺，但你依然可以从中获益。你获得【灵感视觉】，并且可以随时以自由动作开启或关闭，表现为你的双眼微微发光，并且你在施法检定上获得传奇智力\*1DP表现加值。

你获得了以下魔术回路的基础用法：  
在承受一次攻击时消耗1点魔术回路提供的魔力，针对这次攻击获得1点偏斜防御。

不协调刻印  
C级  
C+1200  
描述：你移植了完整的魔术刻印，但你的身体与刻印出现了排斥反应，没法完全掌握刻印中的魔术。  
特性：【招牌魔术】【回路调用】

【招牌魔术】：你的刻印属性对应的派系造诣提升至C级，虽然你无法完全掌握刻印，但你加以运用。选择一个你已掌握的，最高不超过C级的非仪式法术储存在魔术回路中，将其视作自己的招牌魔术，一经选择无法更改。  
招牌魔术将转变为与D级所选的刻印属性相同的法术派系。  
相对的，不协调的刻印带来了痛苦的排斥反应，这是永远无法摆脱的折磨。你将承受2点永久存在不可消除的剧痛点数，该剧痛点数可以与其他来源的剧痛点数叠加计算，可以进入毁灭性后果。

【回路调用】：魔术回路中的力量依然可以为魔术师增强自身。  
你额外获得智力附加\*2点意志豁免检定上的表现加值。  
现在，回路魔力能量池上限根据你的最高内在强化等级会依次增加D5、C7、B9、A11、S13点。

在施法时你可以从与自己的生命链接在一起的魔术刻印中获得增幅，当你进行施法检定时，你身上的魔术刻印将亮起淡蓝色的光芒汇聚力量，为你额外提供你拥有的最高内在强化等级\*2dp（D2C4B6A8S10）的施法检定上的表现加值。

你获得了以下魔术回路基础用法：  
进行一次施法检定时消耗3点回路魔力，获得额外1点附加成功，若如此做，你可以将这次检定中的3DP转化为1点附加成功，转化的上限等同于学识技能上的附加成功，只计算技能本身提供的附加。

完美刻印  
B级  
描述：你移植了完整的魔术刻印，你的身体与刻印经过协调后，可以完整地发挥魔术刻印的力量与增幅，并且完全掌握刻印内蕴含的魔术。  
特性：【完整掌握】【魔术起源】  
【完整掌握】：你的刻印属性对应的派系造诣提升至B级，你需要额外支付2XP，并且当你的【招牌魔术】的法术精熟达到传奇熟练后，你会掌握该派系的A级造诣，这视作你提前对这一身魔术刻印的完全开发。

【魔术起源】：现在，你已经可以掌握刻印内的魔术，除了最初所选的刻印属性外，你还能从刻印中溯源自己的起源。  
你在购买时立即获得3点【起源点】用于购买自己的起源，以下简称pt。  
无论拥有怎样的起源，你都可以从中提取强化，你的三豁免检定再获得等同于智力附加\*2点表现加值，在进行三豁免检定时，你的身上会展露出魔术回路的光芒。

【起源】：一切果之因，一切答案的问题，一切生灵在出生前就已经因起源确定了自己的存在，是起源让该个存在成为了该个存在本身。

每个起源都需要消耗1pt用于开启，若无特殊说明，可以多次购买同一个起源，提升起源等级以解锁更高等级的内容，单一起源最多购买3次，高等级效果替换低等级。

\[切断：斩断、破坏、分离、解析等概念集合的起源\]  
拥有该起源的魔术师其术式拥有极强的破坏力，并且会在不自觉中逐渐与原有的人际关系相远离。  
效果：在以法术攻击命中敌人时将额外附带1/2/3点韧性伤害，这视作以魔术切断其生命力本身；从破甲/高速/威猛中选择其一，一经选择无法更改，法术附带所选5/10/15点所选类型破防，三次购买后再提升5点所选的破防类型。

\[结合：相连、融合、紧密、生育等概念集合的起源\]  
拥有该起源的魔术师其术式拥有极强的融合性，并且会在不自觉中逐渐与陌生的人际关系相贴近。  
效果：你获得1/2/3点快速医疗（每1点快速医疗12医疗点数），选择一种快速医疗能力，使其医疗点数增加24/48/72点。你的增益类能力作用范围额外提升15/30/45米。

\[禁忌：离经叛道、行凶、罪、一切恶的集合的起源\]  
拥有该起源的魔术师天生就有着与世俗常理背道而驰的念头，这个世俗不只是凡人的世界，也包括魔术师的世界。  
效果：你在施法时提升2/4/6点法术强度，在施展【招牌魔术】进行攻击时，在自身生命值不会因此而降低至0或以下的情况下，可以对自身造成1/2/3点不可豁免不可吸收的严重伤害，若如此做，你可以在这次法术检定中额外获得2/4/6点附加成功，并且可以将检定中总计3DP任意加值转化为1点附加成功。  
拥有禁忌起源会使你额外获得2/4/4点额外的先攻加成。

\[进食：生存、贪婪、自私、饥饿等概念集合的起源\]  
拥有该起源的魔术师其术式具有极强的吞噬能力，并且会拥有难以满足的食欲和饥饿感。  
效果：你的法术攻击总是视为具有1/2/3点【吸血】，攻击命中后，使对方的1/2/3点快速医疗在本轮内无法生效，若成功因此使至少1点快速医疗无法生效，则自己在本轮内获得1/2/3点快速医疗（每1点快疗12医疗点数），在强韧豁免上获得2/4/4点内在加值。

\[剑：即利刃、武器、战场、荣耀等概念集合的起源\]  
拥有该起源后，你的所有魔术都可以具象化为【剑】的形态，这是起源对于剑这一事物本身的具象。  
效果：拥有该起源后你在进行法术攻击时，可以将法术的形态具象为剑型，这不会改变法术原本的效果，但会让火球术变成剑光爆炸，寒冰射线变成冰锥飞剑，闪电链变成穿梭雷剑等等；除此之外，你的攻击获得等同于你智力一半/等同于智力/等同于智力的技艺加值，并获得1/2/4点额外的【破魔】，但副作用是你会额外具有1/2/3点永久存在不可消除的精神束缚点数。  
当你将【剑】起源叠加至三层时，施展【招牌魔术】前可以先进行一个数值等同于智力属性一半的纯属性检定，你与将检定出目等量的附加成功加入这次施法检定，这次纯属性检定无法计入属性提供的附加，无法计入任何加值，也无法因任何效果调整成功数出目。

\[虚无：无动力、无活力、无生命等概念集合的起源\]  
对存在、生命、物质、精神乃至一切事物的否认，如空壳般的本源，是没有具象的具象。  
该起源必须消耗3pt获取。  
效果：在承受攻击时，出于对攻击的否定，敌人的攻将击视为承受基于偏斜防御的【劣化2】，在进行攻击时，出于对生命，抑或是对战斗的否定，你自身的攻击会视为承受【劣化2】，在施展【招牌魔术】时，每轮一次，出于对结果的否定，你可以重投一次检定，并取高结算，当你死亡时，出于对死亡本身的否定，你的生机在肉身被完全摧毁前不会真正失去，而是在长达一个月的虚无中找到基于现实的锚点，成功否定自身的死亡后复活，在此期间，你只能以类灵体的形态存在，只能观察和移动，这是一个B级命运来源的复活能力。
