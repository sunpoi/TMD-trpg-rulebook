# 虚实之握下属技能树

虚实之握下属技能树

专属武器：  
总纲：特殊的，本武器为逐级升级，若无提及，则效果与前几级叠加而非替换。

名称：矢量战术装备

D级  
价格：D+500  
描述：  
为迷迭香专门配置的战术装备，如同“剑”一般被迷迭香的能力举起，飘在她的身旁。

一个巨型物体。

武器伤害：6L  
力量需求：1  
武器体积：10  
基础特性：高速1 威猛1 投掷武器  
效果：  
以本武器进行投掷攻击时，力量需求-6。

C级  
价格：C+1000  
描述：  
经过一定的特殊改造，使得该战术装备拥有了更强的效果。

武器伤害：14L  
力量需求：1  
武器体积：10  
基础特性：高速1 威猛1 投掷武器  
效果：  
当你使用该武器进行投掷时，将会对落点处，半径射程/4的圆形内，所有目标造成伤害。

同时，你使用该武器投掷时，你的攻击将变为垂直型攻击。

B级  
价格：B+2000  
描述：  
实体化的思绪会带来重压，也会带来力量。罗德岛工程部的尖端设计在此体现，一切都如你所愿。

当你使用该武器攻击时，这把武器也将融入进你的“手”中。

武器伤害：21L  
力量需求：1  
武器体积：10  
基础特性：高速1 威猛1 投掷武器  
效果：  
你以本武器进行的攻击将附带接触攻击效果。

每轮一次，你的攻击检定可重掷一次，取其中较高的一次进行结算。

你的攻击将附带【幻灭】效果。

专属技艺：

名称：念力  
价格：D+500  
分类：被动技艺  
效果：  
你的伤势将不会自然恶化，同时，  
以一个迅捷动作消耗1点意志力启用，在本场景中，你每在心智类属性上具有1点传奇附加，就获得1点破甲。  
——支付11XP升级该技艺，不在需要迅捷动作启用，你的攻击如同具有【能量武器】一样结算。

名称：投射  
价格：D+500  
分类：攻击技  
动作：标准动作  
描述：  
操控你的“手”，将你的武器以更用力的方式投射出去  
效果：  
你的本次攻击将获得【高速5】和【破甲5】的效果，并获得2dp的速度加值。

——研发：支付10XP，升级为增幅投射。  
动作：标准动作  
效果：  
当使用该技艺进行攻击时，你的本次攻击将获得【连射】特性，本次攻击将获得【高速5】【破甲5】【熔毁5】效果，并获得2dp的速度加值。

名称：心绪力场  
价格：D+500  
分类：增强技  
动作：杂技  
消耗：1意志力  
持续：一场景  
描述：  
展开你的思绪，将其化作一个特定的力场，来阻碍敌人的攻击。  
效果：  
消耗一个杂技动作和1点意志力启用，本场景内，你每在决心属性上具有一点传奇属性，那么就获得全伤害吸收1。  
——研发：支付20XP，效果升级为，本场景内，你每在心智类属性上具有一点传奇属性，那么就获得全伤害吸收1。

名称：记忆流失：  
价格：C+1500  
分类：增强技  
动作：花招  
效果：  
当你发动攻击时，可消耗一个花招动作的同时令自身受到2点严重伤害，你每在心智类属性上具有一个传奇附加，就在本次攻击上获得2点速度加值。

专属奇物：

名称：迷迭香之拥  
价格：C+1000  
描述：这是一个奇特的如同凳子一样的物品，一个金色的小球被悬挂在其中  
效果：  
当持有该奇物时，你的攻击获得【余波】特性  
同时，每满足一个条件：  
改造等级达到C  
武器到C  
购买首个下属C技艺

则逐级获得：  
1：获得【毁灭】  
2：获得【碾压】  
3：【毁灭】升级为【湮灭】
