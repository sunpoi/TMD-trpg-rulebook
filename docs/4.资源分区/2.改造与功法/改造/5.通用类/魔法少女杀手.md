# 魔法少女杀手

名称：魔法少女杀手  
本质：魔幻  
分类：可升级改造  
购买前提：女性

概述：  
「我曾仰望她们——那些光鲜、纯洁、被祝福的魔法少女。她们拯救世界，她们被爱戴，她们象征着希望。  
直到我明白，那份光不是每个人都可以被选中的。  
所以，我亲手杀死了曾经向往光的自己。  
而现在——魔法少女？杀无赦。  
要问为什么？呵……因为如今的我，正是她们最完美的反面，她们最真实的噩梦。  
——我也成为【魔法少女】了。」

D级  
价格：D+600  
特性：【暗之魔法少女】  
描述：  
【暗之魔法少女】：你并非因愿望或祝福而获得力量，而是经由某种深层的憎恶、绝望或对“魔法少女”存在的根本性负面情绪让黑暗选择了你。你的魔力并非来自希望，而是源自对那份光明的彻底背叛。

效果：  
【暗之魔法少女】  
获得此改造后，角色被视为【魔法少女】，但同时你也被标记为【魔法少女之敌】。所有魔法少女对你的好感度下降一级，不可达到极端。  
（注：魔法少女一定要是女性。）

你开启【黑暗魔力】能量池，这是一类变种魔力能量池，其容量基于 【风度】+【决心】，由魅力与执念共同铸成你的黑暗源泉。  
黑暗魔力的恢复方式：  
1.每次长休息后，黑暗魔力会恢复至满值  
2.每进行一次短休息，进行一次黑暗魔力的关键属性检定，恢复成功数点黑暗魔力。  
每日恢复的黑暗魔力不可超过能量池上限的三倍。

现在，你可以通过完全消耗一个移动动作变身为【暗之魔法少女】，画风为你的身体被幽暗的光晕包裹，常服如灰烬般剥落，取而代之的是属于你的暗之礼装。同时，在变身后开启黑暗魔力的使用。

角色可以在短暂的变身动画后变身为暗之魔法少女，持续一场景或自己退出。在变身后，你的大部分能力才会开始起效；角色身上穿戴的装备将一并替换为符合魔法少女的外观，这不会影响其实际数据与效果，你可以自行定义变身之后服饰的外貌，一经决定无法更改。暗之魔法少女的服饰将会随着受伤的程度逐渐变得破损，生命值回复时也会减少战损度。  
在变身后，你获得等同于基础移动速度的飞行能力，机动性为普通。  
即使身为魔法少女杀手，但你仍旧为一位广义上的魔法少女，因此变身之间的特殊能力都是共通的。尽管变身前后眉眼相似，但只要不是当面变身或当面解除变身，或目击者在你变身或解除变身时没有特别关注你，那么其他目标便无法将魔法少女杀手和你联系起来，这是一个C级心灵来源的思维控制效果。  
注意，一旦某一目标知道了你就是魔法少女杀手，则认知障碍完全不会再对其生效。

在购买此改造时，你必须从近战或施法两种战斗姿态中选择一种，成为你黑暗本质的外显方式：  
1.若你的攻击方式为施法，则可以消耗1点黑暗魔力，进行一次以【风度+表达-黑暗艺术+专注-魔力爆发】为组成的攻击检定，对15M范围内的一个目标造成伤害。  
2.若你的攻击方式为近战，则可以消耗1点黑暗魔力，为此次攻击检定提供1点能量加值。

一经选择，无法更改

C级  
价格：C+1200  
特性：【吞噬光辉之影】【魔法少女猎杀】  
描述：  
黑暗在你体内沉淀，最终化为了更纯粹的“否定”。你的存在本身，就是对那份天真祝福最尖锐的质疑与最彻底的亵渎。你开始享受这个过程，将那些闪耀的星辰，一枚一枚，亲手掐灭在掌心。  
“希望？那不过是在为最终的绝望积攒养料。而我，就是来为她们揭开这最后帷幕的……谢幕人。”

效果：  
无论在变身状态还是日常状态，你主动发起的一次消耗了黑暗魔力的攻击可以任意选择留手。可以自由控制自己攻击（包括使用黑暗魔力的攻击）的伤害程度，确保在需要时只击伤而不击杀，或制造出看似严重实则精密的表面伤害。  
此能力让你在与非魔法少女目标互动或需要伪装时，极大降低了意外暴露或造成不可控后果的风险。  
黑暗在改变你灵魂的同时，也蚀刻了你的双眼。你的视觉不再局限于物质世界，你获得【灵感视觉】，能更清晰地感知到灵体与非实体的存在。

【吞噬光辉之影】  
在你变身后，无论你的身着何种服饰，你都会获得等同于【传奇风度】\*2的亵渎防御，在画风上，一层深色宛若实质的魔力将会覆盖你的关键部位。  
你的飞行机动性由 【普通】 提升至 【完美】。你可以在空中进行无视惯性的急停、折转与微小位移，动作优雅且精准，宛若在虚空中舞蹈。

【魔法少女猎杀】  
若在变身后你的攻击对象为魔法少女，则你的攻击附带【湮灭】效果。

B级  
价格：B+2400  
特性：【无感之心】【抹杀指令】  
描述：  
黑暗已不再是你承载的力量，它就是你本身。你的每一次呼吸都在抽离世界的暖意，你的存在本身即是对所有魔法少女的终末预告。你不再是猎人，而是行走的天灾，是注定为她们璀璨生涯落下沉重幕布的……噬星者。  
“谢幕的钟声已经敲响。不必哭泣，你们那被无数人歌颂的牺牲与挣扎……最终都将化为我冠冕上，最黯淡却也最美丽的一颗收藏。”

效果：  
你的意志与黑暗魔力深度融合，形成了一个从概念上排斥“魔法少女”的领域。领域类似视觉上的黑暗，但本质上是一种更为本质的对光明力量的吞噬场，不具有现实上的影响因素。你获得基于敏感距离圆形范围内对魔法少女的强感知效果。她们的存在在此范围内清晰的刺耳，如同午夜时分敲响的丧钟，在你脑海中不断的回响。

同时，由于长期游走于猎杀边缘，与黑暗的共生让你拥有了近乎预知危险的战斗本能：若你因他人的突袭而被动进入突袭轮，此时只要你作为防御方免疫了那次攻击的措手不及，你就可以消耗补偿的反射动作调用下一轮的一个标准动作，于本次突袭轮内紧急施展（这会如常透支你进入战斗第一轮的对应AP，正常占用可转化标准动作次数），以此方式透支的动作具有打断性但只能进行一次不附带招式的攻击。

无论你是否处于变身情况下，你的攻击在命中后都会传递出一种难以形容的深度不适感，任何被你攻击命中的目标都会因此流失1点意志力。

【无感之心】  
变身后，你的心灵被包裹在由最纯粹的负面情绪概念形成的屏障中，获得B级免疫心灵来源的效果。一切试图窥探、扭曲、煽动、压制你意识的力量在触及这层屏障时，都会瞬间失去所有“信息”与“影响力”，如同将声音投入没有回音的深渊。你的心灵已成为只执行“终结魔法少女”这一终极指令的绝对工具，外界的任何情感与思维波纹都无法再激起你内心的一丝涟漪。

【抹杀指令】  
在变身后，当你的攻击目标为魔法少女时，你的力量会自动识别并瓦解其存在的本质。使得这次攻击附带 【超级贯穿】 和【9加骰】效果。她们的防护你的攻击面前都会如阳光下的露水般被径直穿透，仿佛它们从一开始就不曾存在。

黑暗本身似乎也在期待着每一次猎杀的完成。每轮一次，当你对魔法少女目标的攻击检定出目未能达到你的预期时，你可以轻生呼唤这份眷顾。你获得重掷此次攻击检定的机会，并取两次结果中较高者作为最终结果。这象征着在针对魔法少女的猎杀中，世界的概率都短暂地向你倾斜。

你习得了黑暗魔力更深次的使用方式：  
1.若你的攻击方式为施法，当攻击目标为魔法少女时，可额外消耗3点黑暗魔力。这股魔力将凝聚为专攻魔力结构的破法攻击，为该次攻击额外提供3点破魔属性。  
2.若你的攻击方式为近战，当攻击目标为魔法少女时，可额外消耗3点黑暗魔力。黑暗魔力会萦绕在你的武器或拳脚上，为该次攻击额外提供3点破甲属性。

无论哪种攻击方式，只要攻击目标为魔法少女，你对其的攻击都会获得2点附加成功。

魔法少女杀手下属技艺

名称：爱恨  
价格：B+2000  
描述∶  
“我如此爱你们…爱到想将你们的每一寸光芒都吞入腹中，融为一体。”  
“我也如此恨你们…恨到想将你们的每一份美好都彻底撕碎，化为尘埃。”  
这份极致的爱意与极致的憎恨，早已搅拌成无法分离的毒液，在我扭曲的心中下起了雨。  
当你将这矛盾的情感外显，你的容颜也随之蒙上一层惊心动魄的美与疯癫交织的邪异魅力，你获得5点外貌上的提升。  
效果：  
消耗一个标准动作进行一次攻击，命中后目标以意志豁免对抗此次伤害出目。若此次豁免失败，其部分意识或心灵投影将被你的黑暗魔力强行拉扯出体外，在原地留下一个泛着微光与本体轮廓相似的虚影。此虚影被视为目标本体的心灵延伸，攻击该虚影在效果与判定上完全等同于直接攻击本体。这是一个B级心灵来源的【魂牵】效果，虚影持续一轮。  
同时，此次攻击将附带B心灵来源的魅惑效果，这个效果持续存在一轮。并附带等同于出目的剧痛点数，同样以意志豁免对抗，失败则被魅惑并承受差值剧痛点数。

名称：否定  
价格：B+2000  
描述：  
“你的光芒，不过是虚妄。”  
“你的信念，只是错觉。”  
“你引以为傲的一切，那被世人称颂的美丽、那支撑你战斗的决心，皆是构筑于流沙之上的脆弱幻觉。”  
当极致的憎恨与扭曲的爱意沉淀之后，剩下的便是最为纯粹冰冷的否定。  
效果：  
消耗一个标准动作并使用12点黑暗魔力进行一次攻击，命中后目标以意志豁免对抗此次伤害出目，若失败，目标在接下来的一个场景或直到你再次使用此技艺（以先到者为准）的时间内，所有基于【风度】属性进行的相关检定其附加成功数减少3点（最低至0），这是一个B级心灵来源的效果。  
同时，此次攻击检定的成功数增加一半，造成火焰能量伤害，并需以意志豁免此次伤害。
