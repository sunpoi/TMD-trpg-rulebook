# 富江

富江细胞  
分类：可升级改造  
本质：特异本质

概述：  
富江是充满神秘感的少女，没有人说得清她的来历，她拥有让所有男人为之痴狂的美貌，最为奇特的是她的特异体质，可以凭身体的细胞永无止境地繁殖，购买该资源会为你植入富江细胞。

D级  
价格：D+600  
描述：  
细胞还在沉睡，但那难以掩盖的魔性已然足以令人脱胎换骨。  
特性：【恶之美】【富江】  
效果：  
【恶之美】：  
购买该改造后，你获得10点来源于富江的恶名，你拥有的声望，以及从今往后可能获得的声望全都转化为恶名，恶名的影响稍微发生了改变，任何人若知晓你的恶名，则会随时间推移对你产生厌恶与排斥，但倘若知晓你的恶名时对你的态度已经达到了亲密或以上，这种厌恶会随着态度的降低表现为对你的爱恋、信仰、怜惜或渴求等亲密情感的萌发，但会以贪婪的、扭曲的、独占的、病态或恶意的迫害与残忍的虐杀欲为体现。  
现在，每当你在与人交往时，富江细胞就会逐渐活性化，当你连续两场影片与至少六个人交流过，在影片结束后，你的外貌永久增加5点，最多以此成长2次。

【富江】：  
购买该改造后，你的外貌无法避免地发生转变，发色将转变为纯黑，头发生长齐腰，睫毛修长，脸型完美，肤质细腻挑不出一丝瑕疵，左眼角下有一颗美人痣，你的身体也将保持完美的身材比例，哪怕是男性；富江细胞独特的增殖性也令你获得【快速医疗1】（12医疗点数）。

C级  
价格：C+1200  
描述：  
细胞分化的速度逐渐加快，摄人心魄的魅力伴随着细胞一同苏醒。  
特性：【富江-提升】  
效果：  
【富江-提升】：  
活性化的富江细胞开始展现出病态的魅力，活化的细胞增殖能力会令你再获得【快速医疗1】（12医疗点数），富江自满的性格底色令你免疫C级及以下影响心灵来源负面效果的影响。现在，当你与异性进行社交检定后，若其对你的好感等级不低于温和，交流结束后，其视为承受1点不可豁免的针对你的魅惑点数；若你以带有社交技能构成的攻击检定命中敌人并造成至少1点实际伤害，敌人也会承受1点不可豁免的魅惑点数。

现在，当敌人因你造成的、针对你的魅惑点数陷入迷情严重状态时，迷情状态将发生改变，与寻常的迷情不同，此时，陷入迷情严重状态的敌人会不由自主地陷入对你的病态沉迷，想要将你占为己有，想要听你的赞赏、辱骂、惨叫，感受你的快乐，悲伤，临死前的恐惧，想要你的眼中只有一个人的身影，为此，其会在接下来的战斗中，优先以你为目标进行攻击，这替换原有的魅惑异常点数严重状态。

B级  
价格：B+2400  
描述：  
富江的恐怖在于，她高度活性化的细胞。  
几乎不死不灭的，恶性美人。  
哪怕被碾碎，哪怕被分尸，哪怕被吃下。  
富江总会重生，为了变得完美，踏上自毁的道路；路上牵扯进来的无辜者，尽皆陷入空虚与绝望。  
特性：【富江-提升】  
效果：  
【富江-提升】：  
你的细胞彻底苏醒，你再次因富江细胞获得额外的【快速医疗3】（36医疗点数）。  
富江的妖艳魅力开始逐步展现超越想象的力量，当首次与你相见的男性对你的态度不低于冷淡，则其对你的态度立即提升一级（无法以此提升至超过友善），当你承受一次攻击，且攻击你的目标可以看到你的存在，你受伤时的诡异美感与施虐欲望的满足会令人如饮鸩酒，立即对对方反击造成1点精神伤害并附带1点不可豁免的魅惑异常点数。

死亡不是结束，而是富江又一次新生的开始。现在的你死亡后细胞依然会保持活性，但你确实视作死亡；现在你的尸体会获得等同于生命值的结构值，若你的尸体结构完整，则可以在一个月后修复并复活，你在复活期间视为【灵体】且失去一切战斗能力，仅能在影片中游荡，除了移动和观察无法进行其他任何动作。

但如果你的尸体遭受了肢解、分尸等等造成肉体分离的情况，你的尸体结构值将会被平均分在这些分离的肉体之上，质量越大拥有的结构值就越多，该等级下至多分为五份，你需要从中选择其中一块作为自身重生的基础，被分离的尸体将会分别开始重生进程，根据尸块的缺失情况，最晚需要六个月才会完成复活（具体时间由st裁定），这些分别复活的个体皆视为【富江】，且拥有和你相同的所有内在强化与富江特性提到的所有特质，但性格和人格有些微的不同，将会完全脱离你的掌控（即作为中立敌对npc存在）。  
所有的【富江】毫无疑问地敌视自己之外的所有富江，会本能地想尽办法找到并杀死其他富江。这是一个B级别、诅咒来源的复活效果。

st在阅过该特性后，有权禁止该特性中被分尸后的特殊复活情况。
