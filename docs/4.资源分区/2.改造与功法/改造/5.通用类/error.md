# ERROR

名称：改造--Error  
本质：科技  
概述：  
最早发现于“云端世界”的数据异常，经研究，确定其是作为‘不可准确预测’、‘无序’、‘可能性’的概念的具象化存在。  
而关于Error的研究一开始也只是探究对其的高效清理方案，但在发现其具有独特的能力，包括但不限于从微观层面上对能量的精细操作、自身的混沌性等等，研究的倾向便隐秘改为如何在现实中复刻出类似的效果，让人类得以掌握这份力量并用于军事。  
在主神与其研究者进行了交易之后，将其进行优化，使这份技术可以以改造形式供轮回者购买。

D级  
等级名称：诞生  
价格：D+600  
前提：在购买本级改造前，购买者的种族须为人类

描述：  
改造的第一阶段，将对受术者进行为期三十天的全维度数据锚定观测：在此期间，完整捕获受术者的全脑神经放电模式、记忆突触连接图谱，乃至本身具备的特殊能力底层逻辑数据，以此构建出你的意识完整镜像，确保改造过程中意识的连续性。  
观测完成后的下一步，选定机械躯体，并将躯体的核心处理单元替换为一枚特制的微型约束力场发生器。这枚发生器的作用，是在极小的空间内构建约束力场，令微观粒子在力场的精密引导下，在宏观层面拟合出与原大脑完全一致的思维活动规律。  
经观察，改造成功后，受术者将保留了原有的思考能力与以及绝大部分记忆，同时能初步复现 Error 的部分特性，例如在行为上的难以预测性等。  
但因为其存在的不稳定，故为了保证其存在，限制了其能量的输出功率。

效果：  
在获得本级改造之后，将你的种族强制修改为【Error】，开启下属D级技艺和物品的购买权，并且获得如下特性：  
能量引导Ⅰ  
你开启一个名为熵的能量池，由【Error关键属性+操控】决定其上限，  
这其中的Error关键属性为任意心智类属性，须在购买本级改造时选择且选择后不能更改，后续提及的Error关键属性与本级改造中选择的一致。  
熵的恢复方式：  
1.在一次短休息后，进行一次关键属性检定，恢复等额的熵。  
2.在一次长休息后，恢复全部的熵。  
每日最多恢复能量上限三倍的熵。  
熵的基础用法：  
1.消耗2点熵，获得持续一场景的1点快速维修（维修点12）。  
2.消耗2点熵，在一轮内进行的Error关键属性的相关检定上获得2点能量加值。  
3.消耗1点熵，在本轮内你可以【Error关键属+学识：量子力学+洞察下属辅助专业：能量感知】进行攻击  
并且，当你第一次购买TFP系列的特殊武装之后，依照其武器的类别，此后【Delete】将作为你使用那个武装进行近战/远程攻击的攻击方式，造成的伤害取决于武装。

机械构造Ⅰ  
将你的所有生命值永久转化为等额的构造值。

不确定性Ⅰ  
现在，你在进行任何检定时，将会不可避免的失去1点附加成功，随后，进行一次rd2，若出目为1，则在本次检定中，获得【因本特性减少的附加成功】\*4的幸运加值，  
你可以在每轮开始时额外支付1点熵，使本轮内因不确定性失去的附加成功增加1点。

C级  
等级名称： 适应  
价格：C+1200  
描述：  
改造的第二阶段，逐步解除上一阶段为了稳定意识而设置的过度束缚，让受术者真正开始以 Error 独有的方式干涉现实。  
这一阶段的核心目标，是逐步解放约束，让原本被约束在意识模拟层的粒子，初步具备对外干涉的能力，以微观层面的无规则能量扰动对目标进行可观的打击。  
同时，受术者的存在将变得更倾向于不可确定，经实验，常规的预测手段对于受术者变得无效。

在获得本级改造之后，开启下属C级技艺和物品的购买权，并获得如下特性：  
能量引导Ⅱ  
在一级的基础上，  
现在，你的熵能量池的基础用法3将不再需要消耗能量。  
仅一次，支付3XP，在下一场影片中，若你在本次影片中以【Delete】杀死了一个同级单位，那么在这场影片结束后，你在进行Error关键属性的检定时，获得8加骰；

不确定性Ⅱ  
在上一级的基础上，  
现在，不超过C级的预知、解析、读心在尝试获取你的信息时仅会获得一段无序杂乱且无用的信息，这视为一个C级预知、解析、读心免疫效果。  
并且你在每场景开始时，可以支付3点熵，随后本场景中，因不确定性失去的附加成功增加【传奇感知】\*1点。

B级  
等级名称：掌握  
价格：B+2400  
描述：  
Error改造的第三阶段，也是意味着受术者完全适应Error的特殊形态，开始展现其更多不可思议的特性。  
对于普朗克常数不可被忽略的微观世界，粒子往往会呈现与宏观世界不同乃至完全相反的特性，  
而受术者可以尝试解开尺度的限制，在宏观世界复现这些只属于微观的特性和规则。

在获得本级改造之后，并获得如下特性：  
机械构造、自我迭代  
现在，你可以在下一场影片开始前支付1XP，若你在这场影片中赢下了一场高一级战斗并且在这场战斗中失去了3/4的结构值，那么在影片结束后，你的结构值增加2点。  
若你购买了下属物品里机械构造系列强化内的任意B级及以上强化，那么你可以将生命值永久转化1.5倍的结构值。

不确定性Ⅲ  
在上一级的基础上，  
提供的预知、解析、读心免疫等级提升至B级  
现在，每轮一次，支付8点熵，你下一次【Delete】攻击时，会随机触发【时间反演对称】、【瞬时熵减】、【叠加态】三个效果之中的一个。  
【时间反演对称】：若你可以进行移动，在本次攻击之后，你会随机移动到平面内半径30m内的任意一点，并且本次移动视为C级传送移动。  
【瞬时熵减】：在触发时会额外扣除你1点熵，若成功扣除且在本场景中你已经使用了熵能量池的基础用法1，那么你额外获得快速维修4（维修点48），这些快速维修在生效一次后便失去，直到你下一次触发瞬时熵减。  
【叠加态】：你进行的本次攻击可以在进行攻击之后，至多重新投掷一次，两次结果取高处理。
