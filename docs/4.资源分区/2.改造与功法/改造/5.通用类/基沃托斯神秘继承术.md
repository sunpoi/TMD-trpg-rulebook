# 基沃托斯神秘继承术

基沃托斯神秘继承术  
分类：可升级改造  
本质：魔幻本质  
购买前提：只有女性可以进行该强化。

概述：  
在未来，名为基沃托斯的巨大学园都市，生活在其中的少女，继承了来自古老神明的神秘，神秘化成的光环给予了她们不一样的力量。  
特殊的，在进行了该强化后，你的头顶会出现一个特殊的光环，当你失去意识时，光环消失，同时你无法获得本改造除属性以外的所有特性。你可以自由隐藏展示光环，这是一个移动动作。

D级  
等级名称：基础神秘继承  
价格：D+600  
描述：  
仅仅继承了基础的神秘，稀薄的神秘甚至无法在神话中留下自己的姓名，就像神话中的称呼“天使们/恶魔们”。而在基沃托斯，你这种存在通常被称为“杂鱼酱~”。

特性：基沃托斯枪械学【初级】光环庇护【初级】  
效果：  
基沃托斯枪械学【初级】:  
你的射击技能提升1级，最多到3。若已到3则改为提供1点射击技能相关检定上的附加成功。

光环庇护【初级】：  
你获得2点天生防御和【防弹】特性，同时你无视冲击点数带来的减值。

C级  
等级名称：进阶神秘继承  
价格：C+1200  
描述：  
你似乎继承了不错的神秘，成功摆脱了杂鱼酱的名号，总而言之欢迎来到基沃托斯。在这里，你或许可以加入一两个不错的社团，也许可以成为一篇故事的主角。  
特性：  
效果：基沃托斯枪械学【中级】光环庇护【中级】  
基沃托斯枪械学【中级】：  
你的射击技能提升2级，最高为5，若已到5则改为在射击技能检定获得【9加骰】。

光环庇护【中级】：  
你的神秘进一步改造了你的身体，你也许长出来了天使或者恶魔的翅膀，又或者拥有了鬼的角，又或者显示了部分动物的特征，总而言之你似乎不一样了？  
你在D级获得的天生防御提升至5点，并且获得基于天生防御的【防能量武器】与【全伤害吸收1】。

B级  
等级名称：唤醒失落众神  
价格：B+2400  
描述：  
你的神秘在次获得了提高，在神话中，你会是经常会出现的不重要天使或者恶魔。而在基沃托斯，你会是这段独属于你们的青春故事的常客。  
特性：基沃托斯枪械学【高级】【专武】  
效果：  
基沃托斯枪械学【高级】：  
你的射击技能提升3级，最高为8，若已到8且满足C级变更条件，则你在C级获得的加骰提升到8加骰，同时在射击技能相关检定上再取得1点附加成功。

【专武】：  
选择你使用超过24小时的一把枪械，通过12小时的祈祷与接触，将神秘引导于枪械之上，这把枪械将成为你的专武。  
特殊的，你只有在持有专武时才能获得下列好处：  
1）你的枪械如同获得【能量武器】特性一样处理，但不影响枪械本身击发结构（即实弹武器在获得能量武器特性后依旧击发实弹）。  
2）该枪械的武器伤害提升你的【传奇敏捷+传奇耐力】\*1点。  
3）你每有1点传奇决心，就可以选择使该枪械的【破甲】或【高速】或【破魔】其中一种提升5点，必须在购买时决定会获得的种类。

A级  
等级名称：名为学生的奇迹  
价格：A+4800  
描述：  
在我面前的究竟是格黑娜的风纪委员长日奈，还是圣三一的茶会成员未花，又或者是千年的那个小个子女仆？总之你的力量已经处于基沃托斯的最顶峰了，那么有兴趣接待奇迹的来临吗？  
特性：【专武强化】光环庇护【高级】【所有奇迹的始发点】  
效果：  
【专武强化】：  
你在使用B级提及的专武进行远程枪械攻击时，计算属性带来的附加成功时翻倍计算，同时获得等同属性附加成功数量的士气加值。

光环庇护【高级】：  
你失去该兑换在之前等级提供光环庇护的所有效果，改为获得【传奇耐力+传奇敏捷+传奇感知】\*1点全伤害吸收，以及两倍的天生防御。

【所有奇迹的始发点】：  
你每轮额外获得一个只能用来进行远程枪械攻击的标准动作。
