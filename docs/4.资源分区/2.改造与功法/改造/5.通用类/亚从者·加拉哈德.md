# 亚从者·加拉哈德

【亚从者改造·加拉哈德】  
所谓【从者（英灵）】即是境界记录带……是将其人【作为英雄得到确认并活跃的时间段】记录下来的产物。  
所谓【亚从者】，乃是通过魔术基因工程，将从者的灵基植入人的肉体中，以人的身躯发挥英灵的伟力，并且保有人的成长空间的存在  
本资源植入的英灵灵基为【加拉哈德】，其身为完美的骑士，被人称颂为背负次代圆桌的少年。——虽然得到了圣杯，但也因他无欲无求将圣杯还诸于天，他自己本人也归还天上。

【亚从者】  
D+600  
接受此改造后，你初步融合了英灵的灵基，但它目前尚处于沉寂状态，只有当你的身心成长到一定程度，才能得到对方的认可并获取其完整力量。  
目前，成为【亚从者】的你可以随自己心意让身上浮现紫灰色的盔甲，这仅提供画风上的效果

改造到达该等级后，你获得【从者之躯】、【技艺成长】特性  
【从者之躯】：你无需饮食、休息和呼吸也可维持正常行动，从者的力量就是这么方便的东西

【技艺成长】：作为活在当下的人类，你具有一般英灵没有的成长能力。每场影片过后，你使用【盾】专业进行的攻击额外获得1dp修行加值，使用【盾】专业进行格挡时额外获得1点格挡防御。

【真名解放】  
C+1200  
你体内【加拉哈德】的灵基完全觉醒，它观察你一路的旅途后认可了你，将自身的能力全部予以托付。  
你开启下属技艺技能树

改造到达该等级后，你获得【宝具展开】技能

【宝具展开·已然遥远的理想之城】：  
英灵·加拉哈德拥有的宝具。  
使用白垩之城卡美洛的中心，  
圆桌骑士们围坐的圆桌作为盾的究极防守。  
其强度与使用者的精神力相呼应，  
只要心不屈服，城墙就绝不会崩塌。

以一个标准动作，你持盾全力防御并进行一次格挡，并对触及范围内的友方提供【全遮蔽】效果，  
宝具维持阶段中，你每剩余1点意志值就为触及范围内的所有友方提供1点遮蔽防御，至多获得8点遮蔽防御。  
这个效果持续至你的下一轮开始，除非你继续使用宝具展开。

【奥特瑙斯】  
B+2400  
在战斗告一段落了后，【加拉哈德】的灵基陷入沉寂，然而全新的敌人袭来，为了对抗，你选择身披强化装甲【奥特瑙斯】再度踏上前线。  
其结果，就是加拉哈德的灵基重新活性化了。  
——每次用奥特瑙斯战斗时，身体都会受到损伤，而这些损伤正是由【加拉哈德】的灵基来修复、支撑的。

随着战斗的持续，你与【加拉哈德】灵基之间的融合也不断地加深，  
如果照此发展下去，最终的结果极有可能是——  
你将在【仍然活着的人类】的状态下，成为【英灵】。

改造到达该等级后，你获得【Shielder的加护】特性，并且【宝具展开】技能获得强化。

【Shielder的加护】  
【加拉哈德】留给你的遗产，守御者的加护，对一切外来的恶性污染都具有抗性。  
你免疫B级及以下的【诅咒】、【毒素】、【疾病】来源的效果。

【宝具展开·依然存在的梦想之城】  
——那瞬间。  
尽管只有短暂的一瞬，但齿轮确实喀嚓一声啮合了。  
沉重的盾牌闪耀着光辉，唇间轻吐浮现于脑海的启示。  
『——世上没有永恒不朽的城塞哦，■■。如若存在，那一定是重整旗鼓的心态。纵使腐朽也试图奋起的身影，宛若永恒。』

以一个标准动作，你持盾全力防御并进行一次格挡，并对50m圆形范围内的友方提供【全遮蔽】效果，  
宝具维持阶段中，  
你每有1点剩余的意志值就为范围内的所有友方提供1点遮蔽防御，  
每有1点消耗的意志值就为范围内所有的友方提供1点洞察防御，  
以此方法获得的防御合计值不超过26点。  
这个效果持续至你的下一轮开始，除非你继续使用宝具展开。

【帕拉丁】  
A+4800  
并非借用英灵【加拉哈德】的力量的从者，  
而是你自身存在意义的觉醒催生出的全新职阶，  
在获得了自我存在的理由之后，  
你与英灵【加拉哈德】彻底诀别，建立了属于自己全新的灵基。

改造到达该等级后，你的【Shielder的加护】特性提升为【Paladin的加护】，并且【宝具展开】技能获得更进一步的强化。

【Paladin的加护】  
你对【诅咒】、【毒素】、【疾病】来源的效果的免疫提升至A级。  
此外，对【混乱】、【胁迫】、【支配】效果也获得B级及以下的免疫。

宝具初始为【盾】，使用过一次后转变为【剑】，而后转变回【盾】，无论如何每轮只能使用以下两种宝具的一种

【宝具展开·构筑希望的人理之盾】  
抵达了旅程的终点后，你所得出的答案。  
就算自己的人生是被刻意设置好的人生，你也愿意相信——人类的旅程是“正确”的。  
在前方展开强力的守护障壁。

以一个标准动作，你持盾全力防御并进行一次格挡，并对50m圆形范围内的友方提供【全遮蔽】效果，  
宝具维持阶段中，  
你每有1点剩余的意志值就为范围内的所有友方提供1点遮蔽防御，  
每有1点消耗的意志值就为范围内所有的友方提供1点洞察防御，  
以此方法获得的防御合计值不超过26点。  
此外，你还为范围内的所有友方提供三豁免上的12点士气加值  
这个效果持续至你的下一轮开始

【宝具展开·明证希望的人理之剑】  
将圆盾变形为炮身，打出强力的一击。  
看起来像是魔力形成的能量弹，其本质实际上是时空修正效果。  
融合了黑色炮身（Black Barrel）的【测定物质极限的力量】和Paladin的【阻塞空间的力量】，  
能够将错误的历史、歪曲的未来修复回正确的形态。  
是用于修补伤口、治疗目的的灵子转移。  
别名“人理装填炮”。

以一个标准动作，你以【盾】专业进行一次攻击，这次攻击对武器射程范围内的所有敌人造成的伤害全部转为韧性伤害，此次攻击具有【破魔10】和【8加骰】。  

【专属技艺】

【现为脆弱的雪花之壁】  
C+1000  
对队伍全体发动的防御增益。将使用者的精神力变换为物理防御的存在。

以一个移动动作发动，你对触及范围内的友方单位提供4点【DR/-】，持续一个场景，同一单位不能重复获得此效果的好处。

【荣光坚毅的雪花之壁】  
差价升级：B+2000

在C级的基础上，你获得【坚韧】特性，当你持有技艺下属的盾牌进行防御时，攻击者受到基于格挡防御的【劣化1】效果。

【荣光远扬的雪花之盾】  
差价升级：A+4000

你的【坚韧】提升为【超级坚韧】，【劣化1】提升为【劣化2】，每轮第一次招架不需要转化移动动作。

【时为朦胧的白垩之壁】  
A+4000  
时为朦胧，正如其名，将对象的时间轴一时错开从而回避攻击。  
阶级高的情况下即便是来自高次元的攻击也可以忽略掉。

每场影片一次，当你遭受到致死攻击时可以发动，你获得一次A级时空来源的【无敌】效果。  
通过错开不同的时间轴，可以保证自己不被敌人完全命中，你免疫接触攻击效果。  
此外，可以微妙地从时间轴上卡入多次攻击的间隙，你免疫多次攻击带来的减值。  
你每轮获得4点AP以进行防御行为。

【测定时间的紫弹之薪】  
A+4000  
测定正确的时间与历史，若对方乃人理（人类的历史）之敌，便可发挥圣骑士的力量，爆发出无与伦比的威力。

每轮一次，你施展【明证希望的人理之剑】时，若目标持有包含【时空】来源的特性/技艺，则你此次造成的伤害翻倍。  
当你以【盾】专业攻击命中目标时，可以在原地产生一个无法移动的目标的时间虚像，攻击虚像等同于攻击目标本身，这是一个B级时空来源的【魂牵】效果。  
此外，你还可以将人理的力量化作柴薪，获得【生息】效果。

【决意奋起之盾】  
C+1000  
暂时性提升自身的防御力，吸引敌方攻击的技能。  
奋起勇气，无论面对的是谁都不会让出后方的所守护着的人。

当你以【盾】专业进行的命中后，目标需要以意志豁免对抗出目，若你胜出，则对方只能对你发起攻击  
这是一个C级影响心灵来源的【胁迫】效果  
此外，你免疫擒抱状态

【悲壮奋起之盾】  
差价升级：B+2000

当你在持盾状态下遭受攻击时，攻击方的出目每有1点就使你在本轮的下一次使用【盾】专业的攻击上获得1点士气加值。  
以此方法获得的士气加值至多不超过9点。

【专属装备】

【亚从者之盾】  
D+500  
分类：重盾  
属性：武器伤害6L，具有【双手】、【格挡5】、【防能量】特性  
特性：

【放心吧，我用的是盾背】  
你使用【盾】专业进行攻击时，可以选择任意留手

【圆桌之盾】  
差价升级：C+1000  
分类：重盾  
属性：武器伤害9L，具有【双手】、【格挡7】、【防能量】特性  
新增特性：

【英雄聚集之地】  
影片内足以称得上【英雄】的单位对你的好感度上升1级，最高不超过友善。  
与你接触过的【英雄】接触的【英雄】，会在一个月或更久之后其对你的好感度上升1级，这是一个D级命运来源的【模因】效果，同时只能作用于一个目标。

【铁骑之盾】  
差价升级：B+2000  
分类：重盾  
属性：武器伤害12L，具有【双手】、【格挡11】、【防能量】特性  
新增特性：

【软银替身】  
你可以用一个自由动作在你的触及范围内召唤一个由液态水银构成的替身（同一时间只能存在一个），  
它可以完美模拟你的外表和细节，拥有自主行动的能力，但其行动不会对现实造成实际影响。  
但是该替身无实体，被攻击后自动消散，这是一个B级环境来源的幻象效果。

【传说之盾】  
在你以这个盾格挡或招架的时候，一轮一次，敌人需重投其出目，并且你从中选择一项对你较有利的生效。

【圣骑士之盾】  
差价升级：A+4000  
分类：重盾  
属性：武器伤害18L，具有【双手】、【格挡14】、【防能量】特性和攻击上的2点附加成功  
新增特性：

【悖论构造体】  
概念武装黑色炮身（Black Barrel）。

只要这宇宙中还存在寿命，就不存在没有【终结】的事物。  
有形之物必定存在【终结】（存在极限）。  
而黑色炮身能观测极限，并将其物质化。  
换言之就是根据对方的质量誊写魔力，并直接向对方轰出的武器。

你使用【盾】专业进行的攻击改为对敏感范围内的任一目标进行攻击，表现为从该盾牌中伸出黑色炮管发射能量炮。

在你被攻击时，若你使用盾牌格挡或招架，那么你获得基于格挡防御的【反震】效果，造成光能伤害，你可以一个自由动作将【反震】切换为基于洞察防御的【折射】，若如此做，则造成伤害切换为亵渎伤害。
