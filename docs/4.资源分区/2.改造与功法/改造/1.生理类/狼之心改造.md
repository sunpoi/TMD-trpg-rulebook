# 狼之心改造

狼之心改造  
特异本质  
传说中的恶魔之果，鲜红的外表，如同狼之心脏般的果实，具有诅咒的狼桃。传言中可以沟通恶魔，使人能化为饿狼的果实。  
为了力量，吃下它吧……

D级：狼心怪谈  
D+600  
体魄增强，血液流速加快，身体的毛发增长，牙齿，指甲略微变长，在外人看来，你就像一个强壮的幽鬼在面前。  
强化后获得【狼之皮】，且从【狼之怪力】/【狼之咆哮】，两个特性选择其一，之后不可更改。  
【狼之皮】：具有诅咒的狼桃给与的狼的力量是如同传说的狼人一般的，身上具有的坚韧的毛发只能由银器破除。  
你获得1点天生防御，且获得1点基于天生防御的1dr/银，当你受到具有【银】的武器伤害，你具有的快速医疗能力在本轮不生效。

【狼之怪力】：传说中的狼人力量，随着时间的经历在你身上体现。  
第一场影片结束时，你的力量相关检定获得1点表现加值与1个额外附加成功数。  
在那之后，每经过一场影片，你就在力量相关检定上再获得1点表现加值。  
此时你已经可以将巨石击碎，劈开木门，在城墙上快速攀爬，成为夜里的怪物（纯描述）。

【狼之咆哮】：午夜，狼嚎，群狼震撼众兽，随时间奔驰，咆哮会更加强大。  
你学会了模仿狼咆，其使用操控+社交+专业：胁迫+表达下属-演绎（辅助专业）进行攻击检定，能力的攻击射程为20米，基础武器伤害为3L，造成精神伤害，并具有1个额外攻击附加成功数。

支付12XP：你的咆哮已经如雷灌耳，令人生畏，你的狼咆以及其他使用操控属性作为攻击方式的攻击具有【贯穿】，若支付1点意志力，则可以将攻击转化为造成恐惧不良点数，意志豁免。

C级：午夜诡物  
C+1200  
你的身体回到的从前的模样，但具有诅咒的狼桃给与你的力量并未消失，而是内化于你体内，等待着你野性的唤醒...  
强化后获得【狼化】，且根据上一阶段你选择的特性进一步获得【洁白独狼】，或是【群狼幽首】。  
【狼化】：在你进行一个移动动作可以进行的基础行为后（这不是一个招式），你可以激发野性，变为狼人，你将在头上长出一对狼耳，脊椎尾延申出狼尾，头部变为狼型，全身长满毛发，肉体变的更强大，这会使你获得提高3点体积（需视为虚拟体积，不会提升生命值），并额外获得2点天生防御，【狼化】持续一个场景。

【洁白独狼】：选择【狼之怪力】获得，你的白狼血脉提供给你一点圣洁的力量，你具有1点异常点治疗（医疗点24），你在【狼化】时需要额外支付1点意志力，长出的狼毛发为白色毛发，【狼化】状态下你使用力量属性进行的攻击额外具有【碾压】特性，这是你的强大怪力轰击导致的结果。

【群狼幽首】：选择【狼之咆哮】获得，你的咆哮已至，你使用狼咆的攻击可以对以你前方射程一半的扇形范围内的敌对单位同时造成伤害，你在【狼化】时需要额外支付2点意志力，毛发呈现出半透明的黑，看起来如同幽灵，这会使你免疫擒抱状态，并在操控属性相关检定上的加骰提升一级，至多提升到8加骰。

B级：传言永存  
B+2400  
蛮荒之地，巫术横行之处，生长的果实应为恶魔之果。  
但，事实并非如此。  
吃下变为怪物的传言即将逝去，但你仍选择寻找着足以变强的力量  
即使变为真正的怪物，让这传言永不逝去。  
该改造现在名字改为“番茄怪谈”  
强化后获得【非魔】，且根据上一阶段你选择的特性进一步获得【圣兽降世】，或是【蛮荒幽灵】  
【非魔】：你现在明白了，你并非真正的狼人，你仍然是人，即使在外人看来你狼化时的模样多么恐怖，你的本质仍然为人，而非魔，但你的狼人怪谈仍为你阻碍着想要探究你本质的存在，你免疫A级的读心和解析效果。

【白兽降世】：选择【狼之怪力】后获得，你向果实的起源之地寻求力量，回应你的便是呼啸身体的战意，你化为纯粹的野兽，你的力量属性相关检定获得等同于你力量属性/2+1的内在加值。且每轮一次，你以力量属性进行的检定重掷一次，取其中较高的一次进行结算 。

【蛮荒幽灵】：选择【狼之咆哮】后获得，你向果实的起源之地寻求力量，回应你的便是伴随身体的恶灵，恶灵使你在现实中如同虚体一般存在，但你为人的本质还是阻止了你进行飞行，穿墙等能力，你现在不会受到纯物理类型的伤害，但是例如具有【能量武器】、【法破】、【扼魂】或是【非物理伤害】的攻击都可以伤害到你。  
你的狼咆以及其他使用操控属性的攻击方式具有令人胆寒的肃叫，所有被你进行上述类型攻击命中的单位，需要进行一次对抗伤害出目的意志豁免，失败则陷入一轮，B级信息来源的混乱状态，在进行攻击时会选择尽量避开你，选择其他目标进行攻击。
