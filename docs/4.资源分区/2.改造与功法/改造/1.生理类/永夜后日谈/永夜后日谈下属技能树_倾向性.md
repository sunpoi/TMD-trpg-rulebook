# 永夜后日谈下属技能树-倾向性

永夜后日谈下属技能树-倾向性

倾向性  
概述：倾向，相当的程度决定了你的气质，甚至说正是气质也不为过。  
以预设的想法而言，倾向等同于你在队伍中扮演怎样的角色。尽管不是不能重复，发生你以不重复为前提下，每个人会展现出不同的面相。  
倾向等于你在队伍中扮演怎样的角色。尽管不是不能够重复，但是在以不重复为前提之下，每个人展现出不同的面向。每种倾向都有着自己才能办得到的事情与场合。

既然成为倾向，这个东西其实是「比较」出来的。  
在队伍中，众多羞涩易惊的人群中，总是有其中那么一个比较具有长辈风范；在狂野的人群里头，总是有比较可能得人信赖受到推崇的那个；总是有那么一个沉默寡言，不轻易表现出感情的那么一个人。

总纲：你可以从以下倾向中选择一种，从而可以购买下属的相应能力。  
【爱丽丝(アリス/Alice)】  
以一名人类、少女的身份，治愈大家心灵的女孩。  
【狂人(ホリック/Holic)】  
仿若疯狂一般高涨的欲求与自我，几近失控的捉摸不定。  
【自律人偶(オートマトン/Automaton)】  
学会扼殺自身、承受痛苦的战斗人偶。  
【废弃品(ジャンク/Junk)】  
哪怕继续坚持的只剩下自己，也绝对不会放弃的老练战士。  
【幕僚(コート/Court)】  
冷静分析状况，下达正确判断的参谋角色。  
【班长(ソロリティ/Sorority)】  
凝聚大家身心，合众为一的领队。

虽然拥有战斗的力量，但这并不是你生下来的目的。如果因为绝望而扭曲了自己，那个时候，自己还是自己吗？

只要你具有【狂气值】，即使你受到的严重伤害填满生命值也不会昏迷，如果已经变为构造值，则该效果改为获得基于天生防御的【坚韧】特性。

【只要你因此透支了AP，就要承受2点不可避免的严重伤害，同时损失2点意志力。】  
【显然的，你无法因此反复透支AP（这意味着若你已经透支过下一轮的动作，则无法继续透支，且在次轮承受透支的AP减少后，才能再因透支减少AP的回合结束后，才能继续使用这一效果）。】

当消耗一个迅捷动作，你可以进行一次侦查检定对抗对方的潜藏检定。若成功，则立刻获得对方的防御构成，生命值构成，这是一个C级环境来源的解析能力。  
若你成功确认了对方的防御构成，则可以使一轮内自身的攻击增加【高速10】，同时可以将自身最多不超过9点任意破防能力改为另一项破防能力。  
——  
当场上有敌对单位进行攻击时，你同样消费一个反射动作，使用侦查检定对抗对方的潜藏检定，若成功，立刻得知对方本次攻击构成，与附带招式的信息，这是一个C级环境来源的解析能力。

临时调整后的防御类型必须是你具有的防御类型，最多可以调整15/对方破防数值（取低）点防御，只要进行了调整，你们还会额外获得1点基于洞察防御的【全伤害吸收】能力（不具有洞察防御的情况下，也可以挂钩在调整后的任一防御类型上）。

若无特殊说明，肢体残障的影响同样持续一场景。

画面表现上，你在进行攻击后通过对手的反馈，精准的测量出其攻击模式，从而为同伴提供建言，使其可以躲避最致命的攻势。

即使无法理解这个世界背后的意义，还是能够理解同伴的心。  
感情、压抑、恐怖、疯狂。  
你是所有人中，最能理解大家内心的人。以支持与领导守护着同伴，化解争端，合而为一。  
只要大家团结在一起，队伍也该变得更为强大……所以你必须成为战斗之中，指引同伴的明灯，以及圆润处理队伍当中的紧张气氛。虽然这样的工作常会让人感觉晕头转向，不过这种负有责任的立场会使你变得更强。总有一天，你会成为驱散狂气、抵御绝望的角色。
