# 永夜后日谈

名称：永夜后日谈  
本质：特异  
分类：可升级改造  
概述：人类灭亡了。  
只剩下阔步在世上的群群死者，还有操纵它们的「死灵法师」。  
与受到操纵的尸体「活死人」。然后是活死人之中，被賦予心灵与记忆的特例——那便为「娃娃」。  
活在你的注目之下的存在，于坏灭的世界之中苏醒的可怜女孩们。

【活死人的身体并非活物。】  
【已然死绝的身体，无论受到怎样都伤害，都无法停止她们的活动。既不会疲劳、也不会疼痛。不需要睡眠、也不用进食。就连呼吸也不是十分必要的東西。这样的她们，连想要生病都是天方夜谭。】  
【拥有发达的感觉，无论是暗处还是噪音，基本上都不会对娃娃的行动造成阻碍。】  
【而且为了强化战斗能力，许多的娃娃都移植过怪异的器官，甚至部分的身体成為机械。身体上刻印着自己不曾记忆的战斗技巧。】  
【由死者的肉块构成的身体，结合肉体与肌腱，出乎意料地容易修复。对于创伤不過是「修完了」的程度。】  
【娃娃的身体，在不让心灵崩溃的状况下，可以永远地活动下去。】  
【永永远远……】

购买前提：女性。  
总纲：【主神提供的版本是试做版本，与原版并不完全一致。虽说性能有所区别，但也不具有原典中的负面作用，在自我层面也与常人与异，不会因为处于自我次元而过于稀薄。】

D级  
等级名称：最初  
价格：D+600  
效果：  
·你的生物类型转变为不死生物，但你的生命值并不会被替换为结构值，而是正常具有生命槽。  
·你免疫D级毒素、疾病、药物影响；你不需要呼吸、饮食、睡眠。  
·你不会自然衰老，这视为寿命不会自然流逝的能力，你在对抗衰老来源效果时视为具有C级的免疫能力。

C级：  
等级名称：死者的自觉  
价格：C+1200  
描述：  
朦胧的死前记忆。  
肉体无需睡眠与进食。  
可是内心依然渴求着。  
欺骗着自己仍然在世。  
哪怕睡眠仅止于冥想般的时间浪费。  
哪怕进食只是给予味觉的自慰行为。  
娃娃们还是努力地试着像个人  
……然而，如果你接受了身为死者的残酷自觉就会发现，如今的身体是多么的方便。  
效果：  
·你的生命值转为等额构造值。  
·你免疫剧痛、疲乏不良点数带来的减值及严重状态。  
·你可以从听觉、视觉中选择一种升级为强感知，并获得对应感官的【灵体感知能力】与【黑暗视觉】。  
·你的意志力上限永久性减少1点，同时获得1点快速维修，医疗点数12。这体现为你的身体即使被切开也能通过简单的拼接行为“黏合”。

B级：  
等级名称：即使如此  
价格：B+2400  
描述：  
必须面对的只有绝望的黑暗过去？  
或许就这样让疯狂支配，比较轻松也说不定？  
还是干脆不要回想起来，比较幸福也说不定？  
如果沒有互相见面的話，比较好过也說不定？  
【……即使是在毁灭的世界之中，在这样永远的后日谈的世界之中，你也绝对不是那个被疯狂吞噬的，不是那些肉体被破坏殆尽的。】  
【请在这个在毁灭的世界之中，演绎自己的故事；沒有任何逼迫，抵达属于自己的后日谈吧。】  
【沒有绝望、脱离疯狂。】  
【坚定坚强、保持自我。】  
效果：  
·你获得对于心灵来源的效果免疫，最高可以免疫不超过B级的效果。  
·你的肢体，内脏，包括头颅可以立刻再生。在画面上是在被切下之后也能迅速接合，碎肉可以重组，只有在遭到不可逆转的破坏才会自行再生。你在C级获得的快速维修提升至3点，医疗点数36。  
·当你死亡后，身体会自行在无意识的驱动下以无法辨别的速度缓慢恢复。此时你的遗体具有等同你意志值的结构值，在一个月后恢复完整，使你的意识恢复，这是一个B级变形来源的复活能力，期间如果结构值被清空，则复活失败。
