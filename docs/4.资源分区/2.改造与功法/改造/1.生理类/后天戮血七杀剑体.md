# 后天戮血七杀剑体

后天戮血七杀剑体  
魔幻本质  
改造  
描述：  
天生万物以养人，世人犹怨天不仁。  
不知蝗蠹遍天下，苦尽苍生尽王臣。  
人之生矣有贵贱，贵人长为天恩眷。  
人生富贵总由天，草民之穷由天谴。  
忽有狂徒夜磨刀，帝星飘摇荧惑高。  
翻天覆地从今始，杀人何须惜手劳。  
……  
世事不平等，有的人先天而来便注定要成为剑道宗师，身怀剑气，胸有剑意；  
而凡人穷极一生，或许都无法追上所谓天才之脚步，  
但如果，有一条注定伴随着痛苦与挣扎的道路摆在面前，  
只要踏上即可开辟从未有过的剑道未来，直达剑体。  
那么，区区杀戮与恶念，又有何不可呢。

特殊的，该资源与“先天通明无极剑体”无法同时获得。

D级：  
D+600  
描述：不忠之人曰可杀！不孝之人曰可杀！  
你获得【后天剑骨】【戮血剑气】特性  
【后天剑骨】：你浑身的骨骼被人为替换为人造剑骨，经脉之中也穿梭着锋锐的剑气，无穷的杀气逐渐浸透你的心灵，你的意志池减少1点，当你拥有该资源之后，你将始终承受着不可豁免永久存在不可消除的2点亢奋点数，判断来源时视为创伤，与其他来源的亢奋点数叠加。并在恐吓之外的社交检定上永久承受改造等级\*2dp减值。  
你的肉体因这种后天的剑体改造产生了强烈的排斥反应，当你持有所有非剑类武器时，剑气会狂乱地产生排斥，使你立即不可避免地承受1点韧性伤害，若你持续持握，则视作每回合承受1点不可豁免不可吸收的韧性伤害。  
但这一切对你而言都是值得的，体内不断孕育的剑气就是最好的佐证，只是现在剑气未成火候。

【戮血剑气】：你开启“戮血”计量槽，这不视为一个能量池，无法满足任何资源的前置需求，也无法以任何提升能量池的手段提升，该计量槽的基础上限为1，仅根据“后天戮血七杀剑体”的改造等级额外扩大（D3 C6 B9 A12 S15）点（即D级为1+3=4点戮血上限，B为1+9=10点戮血上限）  
初次开启时，该计量槽为空，之后获得的戮血在影片结束后可以保留。  
戮血剑气不会自然补充，这意味着无论是短休还是长休都无法补充该能计量槽。  
仅当自身在战斗中亲手击杀过同级或更高级别的敌人后，可以将其体内的鲜血抽出，从战斗中汲取杀意，化为己用，沉练戮血，增加1点戮血，同步获得戮血的益处。  
随着戮血的累计，你在战斗时，身上会浮现出随戮血计量层数逐渐加重的血红杀气。  
每当你具有1点戮血，你就在以武技-剑专业进行的攻击检定上额外获得1点破甲，若在计算破防后有剩余的破甲存在，则可以将等同于自身当前戮血数量的破甲转化为攻击上的士气加值。

C级：  
C+1200  
描述：不仁之人曰可杀！不义之人曰可杀！  
你获得【修罗血衣】特性  
【修罗血衣】：此时，你体内剑气归于平静，与气血融为一体，气若游丝，但并非虚弱，而是以杀气为丝线，以血气为纱布，将杀意织作血衣，血气冲天，杀意弥漫，生命源源不息，杀机腾腾不止。  
花招动作开启，持续一场景，效果持续时间内，你身上的戮血剑气会凝为实质，如一件血色罗衣覆盖全身，这件血衣的外观会因人而异，但都会提供相同的效果；当你穿着修罗血衣时，你从戮血计量槽中获得的破甲数量翻倍，可转化为士气加值的破甲数量提升为当前戮血\*2，并且获得1点快速医疗（12医疗点数）。

具有【修罗血衣】特性后，每经历一场亲手击杀过至少为高一级敌人的影片，就记为一次磨炼成长，当磨炼成长的次数达到三次后，该特性将彻底完善，效果提升为：当你穿着修罗血衣时，你从戮血计量槽中获得的破甲数量翻倍，可转化为士气加值的破甲数量提升为当前戮血\*2，你在能应用戮血增益的攻击上再额外获得【威猛5】，从该特性中获取的威猛在计算破防后若任有剩余则可以转化为攻击上的表现加值，至多转化5点；并且获得1点快速医疗（12医疗点数），每当你拥有3点戮血，就可以将攻击检定中的2dp转化为1点附加成功，除此之外，你需要在该特性完善时，从天生/力场/偏斜三种防御类型中选择其一，你获得基于所选防御类型的【防能量】特性，一经选择不可更改。

B级：  
B+2400  
描述：不礼不智不信人，大西王曰杀杀杀！  
你获得【逆断五雷】【七杀修罗】【后天剑骨-提升】特性  
【逆断五雷】：所谓五雷，即为五脏，五雷逆断，五脏逆转，戮血倒悬，伤敌一千，自损八百，此为燃命杀伐之法；以一个花招动作对自身造成1点韧性伤害开启，本轮获得【逆断五雷】状态，持续一轮，在该状态下，你进行的攻击加骰将会提升一级（至多为8加骰），在以武技-剑专业进行攻击时，可以凝聚杀气化剑气，每一次挥砍都会通过黑色剑气向外延伸，触及范围额外提升15米，并且在攻击上获得2dp能量加值。除此之外，该状态每维持一轮都会额外变强一分，每维持一轮【逆断五雷】效果，就在攻击上额外获得1dp能量加值，至多以此获益7轮。  
该状态每轮都需要以同样的花招动作对自身造成1点韧性伤害维持，若在开启后不继续主动维持或因任何原因无法继续维持而结束，该场景内将无法再次开启【逆断五雷】，同时承受1+逆断五雷开启轮次数的疲乏点数。

【七杀修罗】：在【修罗血衣】状态下，可以再以一个花招动作展开一轮纯粹由黑色杀意凝结而成的漆黑圆环，该状态持续一场景，或以一个花招动作主动关闭，这圆环似功德光环般从后脑环绕至后背，如高温火焰般令空气变形，亦会使修罗血衣出现黑色杀气织就的漆黑纹路，在展开杀气圆环之后，你从戮血计量槽中获得的破防数量再增加一倍，可转化为士气加值的破甲数量提升为当前戮血计量\*3，漆黑杀气会缠绕在手中的剑上，令手中长剑锋锐更甚，提升2L武器伤害。代价是杀气会逐渐侵蚀肉身，开启【七杀修罗】后，每轮都会承受1点不可豁免不可吸收的严重伤害。

【后天剑骨】：植入剑骨至今，杀气早已融贯全身，但这一切对你而言都是值得的，体内不断孕育的剑气就是最好的佐证，如今的剑气已经可以令你熟练地调用自身的力量，基于D级效果，你不再承受来自【后天剑骨】的意志池减少与负面点数，并且额外获得仅能用于转化为迅捷动作的2AP。

A级：  
A+4800  
描述：我生不为逐鹿来，都门懒筑黄金台，  
状元百官都如狗，总是刀下觳觫材。  
你获得【修罗血衣-提升】【后天剑骨-提升】特性  
【修罗血衣】：将杀气凝聚化形，在此道你已臻至化境；开启【修罗血衣】的动作等级降低为自由动作，并且在原有特性基础上，状态持续时额外获得3点快速医疗（36医疗点数），除此之外【逆断五雷】和【七杀修罗】的效果可以直接容纳于【修罗血衣】之中同时开启并维持，【逆断五雷】不再占用花招动作且不再对自身造成韧性伤害即可维持，如常获得轮次维持收益；【七杀修罗】不再占用花招动作，在开启时额外获得3点快速医疗（36医疗点数）。  
现在，同时开启【逆断五雷】【七杀修罗】时，【修罗血衣】会额外将血气与杀气汇聚成一条环绕腋下与后脖颈的黑底红云纹披帛，该状态下每维持一轮【修罗血衣】都会获得2点快速医疗（24医疗点数）与2点临时生命，这些临时生命在被击破后下一轮开始时刷新，至多受益2轮。  
【后天剑骨】：在B级的基础上，你再额外获得仅能用于转化为移动动作的2AP、仅用于转化为标准动作的2AP与仅用于以【剑】施展攻击技艺的2AP，这代表从【后天剑骨】中获得的AP总量提升为8AP。

S级：  
S+11600  
描述：传令麾下四王子，破城不须封刀匕。  
山头代天树此碑，逆天之人立死跪亦死！  
购买该等级后，你获得【剑冠首】特性，并永久失去2点意志上限。  
【剑冠首】：在进入影片后，你可以随时选择进入【剑冠首】状态并挥出一剑，在判断动作时视为标准动作，进行一次不附带技艺且不造成伤害的白刃攻击检定（这意味着开启剑冠首就必须要执行这次攻击检定），记录那次攻击检定的出目与剩余的破防能力，那之后戮血剑气将化为实质；表现为猩红的戮血剑气与无边的黑色杀气相融，随后化为所有你一直以来击杀过的敌人之兵器形状，这些兵器会在你的敏感范围内诞生，并袭击所有除了你与你认定的队友之外的生物，在战斗中，每个轮次开始时，这些剑气会以你记录的攻击出目作为伤害对范围内的所有敌人造成一次物理-挥砍严重伤害，正常计算破防。

在【剑冠首】状态下，若你完全展开【修罗血衣】【逆断五雷】和【七杀修罗】，则在该状态下，你从戮血计量槽中获得的破防数量再次提升，可转化为士气加值的破甲数量提升为当前戮血计量\*4，你可以从黑金斗笠、华褛披肩、将军战袍、浑天斗篷和龙纹水袖中选择其一，一经选择不可更改，你在展开【剑冠首】状态时杀气与血气会塑造出黑底红纹的戮血饰品加持你身，这些饰品不会真正占据对应部位，可以正常与部位上的饰品叠加，当获得戮血饰品加持时，你在力量属性相关检定上获得等同于当前戮血剑气数量的附加成功，每以此获得1点附加成功，就可以将这次攻击中的3dp转化为1点额外附加成功。
