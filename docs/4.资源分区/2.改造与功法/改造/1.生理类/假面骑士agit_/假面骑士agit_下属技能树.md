# 假面骑士AgitΩ下属技能树

假面骑士AgitΩ下属技能树

C级技能树：  
除非特别声明，技能树下的能力如同技艺一般会提供属性

名称：Rider Kick/Punch  
价格：C+1000  
分类：攻击技  
动作：整轮  
射程：特殊  
目标：单一目标  
描述：  
大地形态之下AgitΩ的最强一击，伴随着头部的交叉之角展开，巨大的AgitΩ的标记浮现于地面上，紧接着纵身跃起，发挥全部力量做出一击。  
效果：  
你以一个整轮动作消耗1点意志力，进行一次冲锋肉搏攻击。这次攻击会将光之力直接贯入敌人体内，因而是一次【接触攻击】，不会因冲锋而受到基础防御减值，并且这次冲锋攻击的移动是一次跳跃移动，这意味着你可以享受到跳跃越过障碍的好处。  
在发动这次攻击的这整个回合中（包括这次攻击本身），因为你浑身所有的能量都汇聚在攻击之中，这次攻击还会具有【余波】效果。

名称：暴风形态  
价格：C+1500  
分类：形态  
描述：  
寄宿着风之力量的AgitΩ的派生形态，是精神方面得到了进化的形态，无论遇到什么危机也不会慌张，与其他形态相比，速度与跳跃力较为优秀，能够使用“暴风战戟”来进行战斗。  
效果：  
因为是对AgitΩ已有力量的特化与派生，所以强化暴风形态并不会使你获得自由属性点。

当你变身为假面骑士AgitΩ后，可以一个自由动作变身为暴风形态。暴风形态中，你通过【变化圆环】取得的轻甲所带来的盔甲防御改为闪避防御。  
你获得“暴风战戟”作为武器，这是一柄具有10L点武器伤害和3点【高速】和的戟。“暴风战戟”被储存在变化圆环的异空间，你只能在变身圆环显现时使用一个迅捷动作将其取出。  
由于“暴风战戟”是以AgitΩ的力量构筑而成的武器，因而AgitΩ对其驱如臂使，可以以拳或其他一个肉搏攻击的专业来代替戟专业进行攻击检定。  
【超越精神之蓝】：这一特性具有两种效果——一种是由贤者之石直接提供给人物的进化效果，另一种则是仅在变身后进一步激活贤者之石力量生效的形态效果。  
通常而言，形态效果不会与【超越肉体之金】或者【超越感官之红】的形态效果共同起效。  
进化效果：影片结束时，若你在这一场影片中取得了至少两次同级别难度或以上的战斗的胜利，则你在意志豁免上获得6dp的内在加值，这个效果最多获得2次。  
形态效果：当你变身为假面骑士AgitΩ后，暴风之力将进一步的在你体内涌动，此时你每具有1点传奇敏捷，你就会在攻击上获得3点高速。

名称：烈焰形态  
价格：C+1500  
分类：形态  
描述：  
寄宿着火之力量的AgitΩ的派生形态，是感觉方面得到了进化的形态，视力和听力等感觉都很敏锐，能准确地捕捉在远方或是隐藏在某处及动作迅速的敌人。以俊敏性为牺牲，强化了腕力。右腕具备操纵火的力量。能够使用“烈焰军刀”来进行战斗。  
效果：  
因为是对AgitΩ已有力量的特化与派生，所以强化烈焰形态并不会使你获得自由属性点。

你可以在大地形态中，以一个自由动作变身为烈焰形态。烈焰形态中，你通过【变化圆环】取得的轻甲所带来的盔甲防御改为天生防御。  
你获得“烈焰军刀”作为武器，这是一柄具有10L点武器伤害的马刀，具有4点破甲和【重型武器】特性。“烈焰军刀”被储存在变化圆环的异空间，你只能在变身圆环显现时使用一个迅捷动作将其取出。  
由于“烈焰军刀”是以AgitΩ的力量构筑而成的武器，因而AgitΩ对其驱如臂使，可以拳或其他一个肉搏攻击的专业来代替刀专业进行攻击检定。  
【超越感官之红】：烈焰之力不断在你的体内增长，你的感官会随着每一次战斗而进化。这一特性具有两种效果——一种是由贤者之石直接提供给人物的进化效果，另一种则是仅在变身后进一步激活贤者之石力量生效的形态效果。  
通常而言，形态效果不会与【超越精神之蓝】或者【超越肉体之金】的形态效果共同起效。  
进化效果：影片结束时，若你在这一场影片中取得了至少两次同级别难度或以上的战斗的胜利，则你在感知检定上获得3dp的内在加值，这个效果最多获得2次。  
形态效果：当你变身为假面骑士AgitΩ后，烈焰之力将进一步的在你体内涌动，此时你每具有1点传奇力量，你就会在攻击上获得2点洞察加值。

特殊的，当你同时具有暴风形态和烈焰形态时，你使用“暴风战戟”或“烈焰军刀”进行的攻击获得1点附加成功。

名称：Halberd Spin/Saber Slash  
价格：C+1000  
分类：攻击技  
动作：标准  
射程：触及  
目标：单一目标  
描述：  
将自变化圆环中出现的AgitΩ的武器的威力发挥到极限的必杀攻击。  
效果：  
这个技艺有两种用法。  
暴风形态下，这个技艺为Halberd Spin，消耗1点意志力，以一个标准动作用“暴风战戟”进行一次攻击，这次攻击具有B级措手不及，且额外具有1点附加成功。  
烈焰形态下，这个技艺为Saber Slash，消耗1点意志力，以一个标准动作用“烈焰军刀”进行一次攻击，这次攻击具有熔毁10，且额外具有1点附加成功。

B级技能树：  
除非特别声明，技能树下的能力如同技艺一般会提供属性

名称：暴风形态II  
前置：暴风形态  
价格：C+1000  
分类：形态  
描述：  
对于暴风形态的风之力量有了进一步的掌握。  
效果：  
因为是对AgitΩ已有力量的特化与派生，所以强化暴风形态并不会使你获得自由属性点。

你从C级技能树中获得的暴风形态的效果获得下述提升：  
你获得的“暴风战戟”【高速】提升到8点。暴风形态中，你通过B级【变化圆环】取得的轻甲所带来的盔甲防御改为闪避防御。  
【超越精神之蓝II】：这一特性具有两种效果——一种是由贤者之石直接提供给人物的进化效果，另一种则是仅在变身后进一步激活贤者之石力量生效的形态效果。  
通常而言，形态效果不会与【超越肉体之金】或者【超越感官之红】的形态效果共同起效。  
进化效果：影片结束时，若你在这一场影片中取得了至少两次同级别难度或以上的战斗的胜利，则你在意志豁免上获得8dp的内在加值和1点附加成功，这个效果最多获得2次，并且与C级别的效果叠加。  
形态效果：当你变身为假面骑士AgitΩ后，暴风之力将进一步的在你体内涌动，在C级的基础上，每轮一次，你可以一个迅捷动作进行一次移动。

名称：烈焰形态II  
前置：烈焰形态  
价格：C+1000  
分类：形态  
效果：  
因为是对AgitΩ已有力量的特化与派生，所以强化烈焰形态并不会使你获得自由属性点。

你从C级技能树中获得的烈焰形态的效果获得下述提升：  
你获得的“烈焰军刀”【破甲】提升到10点。烈焰形态中，你通过B级【变化圆环】取得的轻甲所带来的盔甲防御改为天生防御。  
【超越感官之红II】：这一特性具有两种效果——一种是由贤者之石直接提供给人物的进化效果，另一种则是仅在变身后进一步激活贤者之石力量生效的形态效果。  
通常而言，形态效果不会与【超越精神之蓝】或者【超越肉体之金】的形态效果共同起效。  
进化效果：影片结束时，若你在这一场影片中取得了至少两次同级别难度或以上的战斗的胜利，则你在感知鉴定上获得1点附加成功，这个效果最多获得2次，并且与C级别的效果叠加，当你获得两次这个效果后，你还会获得A级别的免疫措手不及效果。  
形态效果：当你变身为假面骑士AgitΩ后，烈焰之力将进一步的在你体内涌动，在C级的基础上，你可以一个花招动作消耗1点意志力，令你本轮内的攻击加骰提升1级（最高为8加骰）。

当你同时具有暴风形态II和烈焰形态II时，你获得购买三位一体形态的权利。

名称：三位一体形态  
价格：B+2000  
分类：形态  
描述：  
将大地，烈焰，暴风的力量集合于一身的形态，能够同时操控“暴风战戟”和“烈焰军刀”进行战斗，即使是复数的尊者，也可以与之战斗。  
效果：  
现在，在AgitΩ的变身中，你可以通过一个自由动作同时启动烈焰形态和暴风形态。并且你可以同时具有【超越肉体之金】，【超越精神之蓝】，【超越感官之红】三者的形态效果。  
并且你的【超越精神之蓝】的形态效果中，每一点传奇敏捷获得的3点【高速】改为5点【高速】，【超越感官之红】的形态效果中，每一点传奇力量获得的2点洞察加值改为3点洞察加值。  
你的变化圆环附带的储存武器的异空间变为C级，你只需要一个花招动作，就可以从中取出物品。  
当你以“暴风战戟”或“烈焰军刀”进行攻击时，你可以将另一方的基础属性的一半加入到攻击中。

名称：燃烧形态  
价格：B+4000  
分类：形态  
描述：  
AgitΩ进化理论上的终点，宛如自大地之中爆发的熔岩般熊熊燃烧的赤红战士，以牺牲灵活度为代价，获得了惊人的刚力与坚不可摧的身躯。  
效果：  
现在，在AgitΩ的变身时，你可以额外消耗3点意志和一个移动动作，宣言变身成燃烧状态。燃烧形态的效果会持续一整个场景。  
燃烧状态下，你通过【变化圆环】获得的B级轻甲改为B级重甲，具有32/32的盔甲防御和基于这个盔甲防御的【防弹】【防能量】【适应】【超级坚韧】，但具有12点盔甲减值。  
燃烧形态下，你仍旧适用大地形态的形态效果，盔甲防御的类型不会因为烈焰或者暴风形态而改变。并且额外具有等同于耐力一半的天生防御和等同于力量一半的攻击检定上的内在加值。  
燃烧形态是AgitΩ的第一最终形态，这意味着你的力量完全凌驾于AgitΩ已有的姿态。  
当你持有三位一体形态时，三位一体形态的效果也依然在燃烧形态下生效，但是效果发生以下变化。  
【绝对刚力】：燃烧形态舍弃了行动的灵活，获得了强悍无比的刚力。你三位一体形态效果中基于暴风形态的形态效果获得的【高速】在燃烧形态下强制变为【威猛】，这部分威猛能够如同【破甲】、【破魔】一般破除其他防御。并且如果在破防结算完成后，还有额外溢出的【威猛】的话，你可以将这些【威猛】转化为等额攻击上的内在加值。  
【熔岩翻涌】：燃烧形态比起来通过洞察敌人的弱点，击破对手，更加倾向于直接利用熔岩般的高热能量将对手击垮。你三位一体形态的效果中基于烈焰形态的形态效果获得的洞察加值全部转化为火焰能量加值，并且攻击获得【超级贯穿】效果。每轮一次，你在进行攻击时可以额外消耗1点意志力，使得这次攻击造成火焰伤害，并且检定的出目增加一半。  
然而，由于燃烧形态下，你拥有的光之力过于强大，暴风形态和烈焰形态时召唤出来的武器，都无法承载其强大的威力，因而你在燃烧形态下无法召唤“烈焰军刀”和“暴风战戟”或者说是即使召唤了也无法使用。

名称：Burning Rider Kick/Punch  
价格：B+2000  
分类：增强技  
消耗：1意志力  
描述：  
燃烧形态之下AgitΩ进化过后的必杀技，拥有更为强大的破坏力。  
效果：  
这个技艺只有在燃烧形态中才能使用。在C级别的Rider Kick/Punch的基础上，追加消耗1点意志力，你还会在这次攻击上额外获得【碾压】和【幻灭】以及【湮灭】特性。并且在施展这个技艺时，你的天生武器的武器伤害还会额外提升传奇力量\*3点。

名称：闪耀王剑召唤  
前置：三位一体形态  
价格：B+2000  
分类：形态  
描述：  
你获得了召唤闪耀王剑的能力。所谓的闪耀王剑，是“暴风战戟”与“烈焰军刀”进化后的姿态，也是AgitΩ拥有的最强威力的武器。  
效果：  
燃烧形态下，你可以花费一个花招动作，你可以从变化圆环的异空间之中将闪耀王剑取出，闪耀王剑是一柄具有22L武器伤害，和【重型武器】，【双手】特性的重剑，是B级武器，具有一切相关的硬度和结构值。同“烈焰军刀”和“暴风战戟”一样，AgitΩ对闪耀王剑驱如臂使，可以以拳或其他一个肉搏攻击的专业来代替剑专业进行攻击检定。并且由于闪耀王剑是“烈焰军刀”和“暴风战戟”进化过后的姿态，所以同时持有暴风形态和烈焰形态带来的一点攻击上的附加成功，你在使用闪耀王剑时也能享有。  
【Burning Bomber】：闪耀王剑中的炎之力完全进行解放的必杀技，即使是位列尊者中最高位的水之上神也被这一招当场斩杀。  
燃烧形态下，以一个整轮动作消耗3点意志力发动，你首先进行一次冲锋攻击，与对手交错而过，以闪耀之刃的威力破开防御，这次攻击无视冲锋攻击的减值，对手需要进行一次DC等同于你的攻击出目的强韧豁免，如果豁免失败，对方立刻陷入A级别、创伤来源的【震慑】状态,持续一轮。这次攻击造成的伤害增加一倍。
