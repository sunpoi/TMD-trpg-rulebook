# 偷盗者

名称：偷盗者  
本质：魔幻本质  
分类：可升级改造/血统

概述：  
途径是一系列魔药配方。在第一块亵渎石板之后出现。魔药服用者会获得非凡能力，成为非凡者，完成生命层次的晋升，并最终成神。  
偷盗者途径不是普通的窃贼，而是一位偷盗者，欺诈者。他们偷窃人心，窃取能力，欺诈规则，瞒骗命运，而到最后，成为一切错误的化身。

总纲：  
途径会同时占据购买者的血统位与改造位

【非凡污染】：魔药的本质是源于不可名状存在所分离出的非凡特性，当人物强化了这一途径的资源后，会根据序列所对应的支线等级（D/C/B/A/S级）永久性的承受（2/4/8/16/32）点不可避免的精神束缚点数。例如：阿蒙购买了D级的序列9与序列8的魔药，会承受合计4点的不可避免的精神束缚点数。  
在使用消耗灵性的能力时，角色可以承受至多沉着附加点只能被长休息移除的不可避免的精神束缚点数来替代沉着附加点灵性。  
当角色因非凡污染承受的精神束缚点数超过了角色的意志池，会彻底的失去自身的控制权，从而屈服于非凡特性的本能，在判定上，视为人物被怪物取代而导致的不可避免的死亡。因此死亡的轮回者视为不可复活。  
【扮演法】：非凡特性会带来极为严重的负面影响，非凡者们为了减免或者抵消这些影响，总结出以主动或者被动形式的扮演方式去迎合非凡特性的潜在要求，最终达成对魔药的契合的扮演法。  
每个序列等级均具有不同的扮演法，当角色完成了扮演法的需求后，会移除一定数值的因服用对应序列魔药而带来的永久性精神束缚异常点数。  
若无特殊说明，则一个序列等级的扮演法只能生效一次。

D级  
名称：序列9 偷盗者  
价格：D+600  
特性：【灵性】【敏锐察觉】【敏捷之手】【偷窃】  
描述：很难将这些非凡者和普通的盗贼、小偷区分开来，也许他们在手段上会更加厉害，而他们偷盗财物的目的不是为了享受，也不是为了生存，更像是在履行一个使命。无论什么途径的非凡者在这个阶段，与凡人的并没有太多的区别。

【灵性】：  
灵性代表着一个有心智的生物所具有的神秘学精神体，是生物最根本的精髓。除精神体之外，一个生物还具有星灵体、心智体、以太体。  
星灵体是生物沟通灵界、星空的途径，也是占卜、释放魔法的媒介。  
心智体负责与肉体结合、牵涉推理、思考、洞察等方面，是生物对事物认识能力的综合体现。  
以太体在最外层，是生命能量和肉体状态的表现，外向显征为气场。

【敏锐察觉】：对危险和不怀好意之人的警觉，对非凡之物的敏感，是偷盗者一次次完美窃取的关键所在。

【敏捷之手】：魔药强化了偷盗者的手，使偷盗者可以更好的完成行窃。  
【偷窃】：想要窃取无形的命运，先从有实的物体开始，虽然失手常有。偷盗者在进行的偷窃是无声无息，令人防不胜防。

效果：  
【灵性】：  
你开启灵性能量池，灵性的关键属性为：沉着+感知/智力/风度（购买时自行选择）

具体的关键属性由开启灵性能量池的能力决定  
沉着象征着最根本的精神体  
感知象征着沟通外界的星灵体  
智力象征着综合认知的心智体  
风度象征着外向气场的以太体

特殊的，当你承受精神束缚异常点数时，你的灵性能量池会减少等同精神束缚异常点数的上限，直到这些异常点数被移除。

灵性的恢复方式  
1、每次长休息后，灵性会恢复至满值。  
2、每进行一次短休息，进行一次关键属性检定，恢复成功数点灵性。

灵性的基础使用方式：  
在进行灵性关键属性检定时，支付1点灵性以获得1点对应属性的内在加值。

【敏锐察觉】：  
偷盗者依凭灵性，能更好的观察事物，去了解他们。你进行的洞察技能检定获得传奇感知\*1的内在加值。

【敏捷之手】：  
偷盗者的敏捷检定获得敏捷附加\*1点增强加值。  
你获得一个新的攻击方式：敏捷+手艺+偷窃+辅助专业：潜藏，可以使用体积不超过2的具有【轻灵】特性的武器（例如匕首、小刀）。

【偷窃】：  
你现在能够偷窃触及范围内无支线等级的物品，通常而言这是一个整轮动作。进行一次偷窃行为在他人看来是一个敌对行动。若被他人发现或者偷窃失败，通常意味着一次战斗。  
若这一物品具有主人，根据对方对于你的定位等级分为两种情况：  
1、对方无法定位你：对方需要以反射豁免对抗你的近战攻击检定，根据物品的支线等级，你的胜出数需要超过对方至少（无支线8/D18/C32/B44/A58/S80）点才能获得这一物品的所有权。这是一个D级别、命运来源的夺权效果。  
2、对方具有你的模糊定位或精确定位：对方需要以攻击检定对抗你的近战攻击检定，根据物品的支线等级，你的胜出数需要超过对方至少（无支线8/D18/C32/B44/A58/S80）点才能获得这一物品的所有权。这是一个D级别、命运来源的夺权效果。  
每部影片你只能通过这一方式带出一个物品。

【扮演】：在不为人发现的情况下以偷窃获得一个有主物品的所有权。完成后会移除1点因本序列魔药带来的永久性精神束缚异常点数。

等级名称：序列8 诈骗者  
价格：D+600  
特性：【偷窃提升】【欺诈】【非凡之躯】  
描述：“在一些诈骗案里，我们发现了非凡者的痕迹，他们以欺诈别人为乐。”  
在别人眼中，诈骗者可能是一位谈吐文雅的绅士，或者是路边偶遇的幽默风趣的陌生人，可惜的是，他们的口中，满是谎言。  
【偷窃提升】：长期简单的潜入偷盗已经满足不了得到晋升的诈骗者，于是他们有了新的目标，利用谎言去窃取，去诈骗。

【欺诈】：欲望恶魔的阴影笼罩在每一个人的心中，我们做的不过是引导，不是吗？诈骗者的言语具有极强的煽动性和欺骗性，往往他们只需要言语就可以引发一场灾难。

【非凡之躯】：灵性和身体素质二度得到提升，诈骗者向非凡更进一步。  
效果：  
【偷窃提升】  
诈骗者在非战斗场景下，与他人进行至少十分钟的社交检定后可以附带进行一次偷窃，这次偷窃按照对方无法定位你进行对抗，表现为诈骗者利用语言分散了目标的注意力。  
现在偷窃可以窃取的物品等级提升至D级别。  
角色在进行偷窃检定时，获得操控附加点表现加值。

【欺诈】  
角色在与一个单位初次进行社交检定时，该单位对角色的初始态度提升一级，不能到达极端。

【非凡之躯】  
角色获得1点快速治疗（12点医疗点数）

【扮演】：在一场影片内，角色成功欺骗/误导了一方势力，令角色从中获得了重大收益（即至少完成一份支线任务）。完成后会移除1点因本序列魔药带来的永久性精神束缚异常点数。

C级  
等级名称：序列7 解密学家  
价格：C+1200  
特性：【灵感直觉】【推演】【知识赠礼】  
描述：  
一位解密学者可以同蛛丝马迹还原神秘学事件的真相或者预见事情的发展，可以仅凭一些小小的痕迹反推出事物的本质。解密学者是谜题，是规律的研习者、探寻者，日后，则是它们最大的破坏者。

【灵感直觉】：借由提升的灵性，解密学家对危险，对未知愈发敏感。  
解密学者可以利用灵性被动的感知周围可能出现的危险。

【推演】：一次蝴蝶的振翅，便可能引起一场灾难性的风暴；某一天的无意之举，也可能会在日后影响深远。而解密学者们，凭这场风暴，便可溯源到那只振翅的蝴蝶，仅凭蛛丝马迹，解密学家也会找到事件的本源。

【知识赠礼】：魔药的带来的可不只是对肉体凡胎的改造，更加可贵的，是那些无数人求之而不得的，知识。  
解密学家利用自己的博学，优化自我能力的使用，体现为使用更好的灵性使用回路，或者能力使用的熟练等。  
效果：  
【灵感直觉】  
这使得解密学者免疫措手不及等级提升一级，至多为B级。

【推演】  
角色可以花费一个整轮动作进行【感知+洞察+侦查】检定来获取遗留在场景内的物品、战斗痕迹、残留能量等线索，接着对场景内已经发生的行为进行大致的还原，每5个成功数会获得1项线索直接关联的信息，至多获取3条信息。  
举例而言，如果是一个拳头攻击留下的凹陷，角色会获得进行攻击单位的力量属性值；如果是远程射击而留下的弹坑，角色会获得攻击的入射角、所使用的弹药对应的武器类型；如果是能量烧灼留下的残骸，角色会获得所使用的能量类型。  
若需要对抗，这是一个C级别、信息来源的解析效果。

【知识赠礼】  
当角色通过【推演】揭示了主线任务背后的详细情况后，在影片结束时会按照以下顺序依次获得好处，每场影片只会获得1次好处，各选项的好处只能获取1次。  
1、提升-【偷窃】：可以偷取的物品等级提升至C级，夺权的来源等级提升至C级。  
2、提升感知属性点灵性能量池上限。  
3、偷窃检定获得8加骰。  
4、提升-【灵感直觉】：角色可以敏捷取代感知进行侦查检定。

【扮演】：解密学家运用思维，辅以自己的【推演】能力，成功的还原出一个重大事件（st裁定）的真相。完成后会移除2点因本序列魔药带来的永久性精神束缚异常点数。

B级  
等级名称：序列6 盗火者  
价格：B+2400  
特性：【偷窃提升】【能力窃取】  
描述：在世人的分类中，序列6的非凡者自己的某些能力超脱凡人，被称为中序列，代表着非凡者正式通过了意志，身躯的双重考验，步入真正的非凡。  
盗火者的偷盗能力大幅度提升，现在，他们已经不只能偷盗凡物，甚至是他人的非凡，他人的能力。就如同神话中的普罗米修斯，但盗火者盗取的，是他人的火，照亮的是自己的未来。

【偷窃提升】：那些飘渺无形，又不可言说的非凡能力也成为了偷盗者的目标，它们已不是遥不可及。

效果：  
【偷窃提升】  
可以偷取的物品等级提升至B级，夺权的来源等级提升至B级。

【能力窃取】  
标准动作消耗10点灵性启动，选择一个目标当前场景内使用过的、需要动作使用的能力，以敏捷+手艺+偷窃+辅助专业：神秘学（学识）对操控\*4米距离内的目标进行一次施法攻击检定（若对方处于你的触及范围内，你可以用近战攻击替代这一施法攻击方式），这次攻击不会造成伤害，受影响单位以意志进行豁免，若受影响单位失败且你的胜出数超过目标能力等级对应的DC（无支线8/D18/C32/B44/A58/S80），则你会在十分钟/一场景内获得这一能力的使用权，你可以一个自由动作将这一能力归还给目标。  
在这次攻击检定中，根据你对于目标能力的了解程度会获得以下增益：  
1、知晓这一能力的使用动作：获得等同感知属性点增强加值。  
2、知晓这一能力的具体效果：获得等同操控属性点增强加值。  
3、知晓这一能力的对应本质：获得等同沉着属性点增强加值。  
被你偷取能力的单位在你归还这一能力前无法使用这一能力，这在判定上是一个基于【偷窃】特性的扼魂效果。因此若一个单位免疫偷窃特性或者扼魂，将无法偷取其能力。你在同一时刻只能持有一个偷取的能力。  
你在使用被偷取能力时，若这一能力需要消耗能量，以灵性代替能量消耗。若这一能力具有限制，需要遵循使用限制（例如每场景一次使用的能力，因为其已经使用了这一能力，你即使偷窃了这一能力也无法再次使用）。若这一能力具有使用前提，需要遵循使用前提。

【扮演】：偷盗者窃取了一个同级别单位的能力并以窃取后的能力将其战胜。完成后会移除3点因本序列魔药带来的永久性精神束缚异常点数，这一扮演可以完成2次。

等级名称：序列5 盗梦人  
价格：B+2400  
特性：【非凡之躯提升】【思维窃取】  
描述：能力能涉及的领域是可以代表非凡者层次的高低，盗梦人就可以将自己的偷窃的目标扩大到他人的思绪。在无声中，将他人的想法，纳入自己的脑中。  
【非凡之躯提升】：低序列到中序列，非凡者身体素质得到较大提升，普通的攻击已经无法对他们造成太多伤害。

【思维窃取】：窃梦人现在可以窃取他人的可以偷窃并让目标遗忘脑海内正要付诸实践的那个想法，转而令自己获取先机。  
效果：  
【非凡之躯提升】  
序列8时获得的快速医疗提升2点，医疗点提升32点。再获得基于天生防御的2dr/斩击。

【思维窃取】  
这一能力具有二种使用方式，但一轮内只能使用其中一种。  
1、消耗10点灵性以迅捷动作使用，对操控\*3米内的目标进行一次敏捷+手艺+偷窃检定，目标以意志豁免，若你胜出则对方需要将其一轮内的想法告知于你，例如：如何分配AP与使用动作的行动方式。这是一个B级别、影响心灵来源的读心效果。  
若处于战斗内，被读心的单位还会因为被你窃取了想法而失去4点AP，这是一个耗竭效果。在窃取对方想法的下一轮，你需要按照上一轮中你窃取的目标的行动方式来分配Ap并使用目的相同的动作，直到所有的行动都执行完毕。例：你窃取了对方当前的想法：分配8点ap转化出标准动作来进行防御，则下一轮你也需要分配8点ap来转化出一个标准动作以进行防御。  
2、在受到一次攻击时，消耗10点灵性以反射动作使用，你会如同招架一般抵消这次攻击的伤害并在应对这次攻击时视为移出了攻击范围。

【扮演】：偷盗者通过思维窃取以达成两个条件：知晓一个任务有关的重要信息、赢得同级别战斗。完成后会移除3点因本序列魔药带来的永久性精神束缚异常点数，这一扮演可以完成2次。
