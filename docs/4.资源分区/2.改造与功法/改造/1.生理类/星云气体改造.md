# 星云气体改造

公元2008年，载人火星登陆器成功降落在火星表面，人类首次踏上火星的表面并发现了一万年前的火星遗迹，在这里，宇航员石动发掘出了一个在此后深远的影响了世界进程的东西---潘多拉魔盒。  
同一年，宇航员带着潘多拉魔盒重回地面，人们将该物品当做21世纪最重大的发现而举行记者会并展示这个盒子，也就是这个时候，一切悲剧的源头诞生了。潘多拉魔盒被突然发狂的石东触碰后放出了耀眼的白色光芒，在这光芒中，三道高耸入云的天空之壁拔地而起，将整个国家强行分割为了三份......  
大约9年后，天才物理学家葛城巧在天空之壁下发现了不知从何泄露而出的，地球上从未见过的物质，这种物质呈气状并伴有硫醇一般的难闻气味（俗称煤气味），他发现这种物质可以大幅度强化人体以及改变生命形态，并且还有其他的神奇作用，便开始了那禁忌的试验......  
星云气体，出自作品《假面骑士build》，从天之壁下提取的未知物质，向人体注射后由星云气体强化改造身体，身体素质与生命力得到增强。对强化者的身体素质和生命力存在要求，身体素质不足者无法完成强化。注：兑换该改造后会获得一个特殊评级危险等级，危险等级未达到5.0时，被改造者仍被视为人类，到达5.0及以后，被改造者成为半人类半Blood族，不再是纯种人类。

  
星云气体改造  
D+600  
前提：耐力6+，生存5+  
在确定自己拥有足够的身体素质后，你选择接受了接受星云气体的改造。主神会在你的轮回空间的房间中开辟出一个实验室，里面有一个可以让你躺进去的“棺材”，躺入“棺材”后自动注入星云气体，这种气体虽然无毒但具有浓重的硫醇气味，这可能会是你在生理上承受严重的不适，但此时你想后悔也不可能了，因为主神的“棺材”哪怕你是S级也打不破！ 在注入了大约半小时后，初步改造即可完成，星云气体会强化你的身体机能，让你远超凡人。 现在你获得了一个新的评定自身等级的数值--危险等级，你现在的危险等级为：3.0。  
【生命强化】：在一次应用到敏捷/力量/耐力属性的检定中，对于本次检定而言，获得1DP表现加值。经过星云气体的侵蚀，你的免疫能力大增，强韧豁免检定获得1点附加成功。

  
【危险等级】：在战斗中，星云气体将会加快与身体的融合，每影片一次，危险等级在你战胜一次与同级的敌人后增加0.2，以此类推，成长上限为4.0。危险等级每比3.0高0.2就会获得2DP力量检定上的增强加值。（即直到4.0为止，你会合计获得10点力量检定加值，后续不再累积）

幻影液体改造  
C+1200  
强化前提：危险等级4.0  
幻影液体是两个地球融合的时候，旧世界存在的星云气体因为碰撞而受到高浓度压缩液化的产物，本质上是星云气体的另一种形式，但浓度及性能远高于星云气体。看来星云气体的力量已经满足不了你了，渴求力量的你再次躺进了棺材里，接受了下一步的改造。

  
【危险等级】：危险等级4.0后，你进入了新的进化阶段，得益于超高浓度的幻影液体，你的危险等级上限提升至4.9，但是在这个阶段，普普通通的战斗再难使你变得更强，你的成长方式改为，每部影片一次，经历一次有比你更高一级的敌人存在的战斗，你的危险等级上升0.3，你至多以此方式将危险等级提升至4.9。每经过一次这样的成长，你获得2天生防御和力量检定上的1附加成功。（即直到4.9为止，你合计会获得6点天生防御与3点力量检定时的附加成功）  
【生命强化】：你现在的生命力更加旺盛顽强，再严重的伤势都不能使你失去战斗力，即使你的生命槽被严重伤害填满，你也不会晕厥。同时，你获得1点异常点数治疗，医疗池24。  
【我的身体已经菠萝菠萝哒】：浓度过高的星云气体无时不刻都在侵蚀你的身体，你永久性承受1点无法减免，无法恢复的剧痛不良点数。

  
Blood因子改造  
B+2400  
前提：危险等级4.9  
看来就算是最高浓度的星云气体都无法满足你对力量的追求了啊，如果做好了觉悟的话，就再躺进那个棺材去吧，是时候告别人的身份了。主神亲自出手为你抓住了一个Blood狠狠榨取，现在，你的身体被注入了大量因子，改造完成，拥抱进化吧。  
【危险等级】：你的危险等级提升0.1，即从4.9提升至5.0。这并不会为你带来额外的收益，只是提醒你，你已经不在是纯粹的人类了。  
【生命强化】：你已经具备了宇宙生物的特性，你获得了在宇宙空间生存的能力，你获得黑暗视觉，无需饮食，全环境呼吸，自适应长短休基于天生防御的三防和2天生防御。  
【Blood因子】：你的身体经过改造之后，已经逐渐拥有了一些Blood一族的能力，现在，你能够将四肢进行一定程度的液化，你基于四肢天生武器的攻击获得接触攻击的特性。同时的，你拥有了飞行能力，机动力为完美。

  
Blood完全改造  
A+4800  
力量就像毒药，品尝到Blood因子带来的力量之后，你就踏上了一条不归路，渴求力量的你祈求着主神，祈求着更多力量，恭喜你，现在你真真正正的成为了一个Blood，100%纯度，不含一丝人味。  
【危险等级】：你的危险等级来到了恐怖的6.0，不过这不会带来任何额外的好处。  
【生命强化】：你已经不做人了！生命力得到了进一步的提高，你获得了基于天生防御的【超级坚韧】和【适应】，你获得4点天生防御，同时，你不再受到星云气体侵蚀的烦恼。  
【Blood因子】：你拥有了像蛇皮怪一样化为Blood因子藏于生物体内的能力，现在的你虽然不能像E总一样熟练的附身别人，但是却能做到化为因子藏在生物身体上。当你死亡，你可以将具有自己的Blood因子的碎片混入当下接触范围内的友方单位体表，或者留在自身死亡的位置，这将如同一个标记一样处理，如果被仔细搜查会被发现这一标记的碎片。这种具有Blood因子的碎片需要保存一周时间才能复活，其结构值为你的生命值上限，被摧毁则复活失败。这是一个A级别附身来源的复活能力，复活过程中除了观察和移动外无法进行更多的动作。  
【天生的破坏欲】：Blood一族生来就对一切事物都有着极强的破坏欲，甚至于不惜破坏掉自己的母星。而你，作为一名后天的蛇皮怪，你也受到了这种影响。每次长休，你都需要进行一次dc为10的意志豁免检定，失败则获得1点不可豁免的亢奋点数，杀死生命和破坏物品都能让你免受该特性影响。在执行攻击检定或是毁坏物品的检定上，每轮一次检定成功数增加一半。

  
Blood之王  
S+9600  
【危险等级】：你的危险等级上升到了7.0这个极限。  
【生命强化】：即使在血族里，你的生命力也堪称顶级，现在想要杀死你无疑是极为困难的。  
你获得5点全伤害吸收，以及具有9点快速医疗，医疗点数108点。  
【Blood因子】：你现在可以分裂出部分Blood因子侵入到其他生物体内进行附身，你可以完全掌握他的思维。若想如此做，你需要以操纵+专注（控制）+辅助专业：生物学（学识）进行操纵，这一操纵需要在以杀死对方为目的的战斗时将其击败，成功清空其生命值时，选择使对方不会死亡，而是改为使用意志豁免操纵检定，豁免失败则被你控制，如同承受了S级附身来源 思维控制能力的效果。同时，被控制的人也可以成为你复活的媒介，你可以在死亡后选择融入曾经操纵过的任一单位体内，在一周后复活，这是一个S级附身来源的复活效果。  
【能量形体】：你现在可以将自身转化为能量流体，能够在不改变自身体积的前提下自由改变自己的形状，可视为超微型集群。
