# 披缠著夜幕的漆黑少女

披缠著夜幕的漆黑少女  
资源类型：可升级改造  
本质：魔幻本质  
购买前提：此改造限定女性

概述：  
这是英雄传说的开始。序夜的第一幕。  
舞台设定。燃烧著天空的红，以及渲染著大地的红。被名为战争的恶魔的爪痕、名为战乱的病魔所侵蚀而逐渐衰退消瘦的世界。无穷的憎恶与无止境的悲叹与无处可逃的疯狂。  
渴求著救赎的世界的沙哑吼叫以及诅咒般的祈求。英雄、英雄、英雄、使英雄降临此地、赐与我们的故事英雄吧。英雄、英雄、英雄――  
舞台中央。尸与尸与尸的绒毯上满布著不再呼吸的残骸，以及与那差不了多少的却尚未变成尸体的人们。连小鸟的羽毛亦无法吹起的微弱气息，那每一次呼吸也渐渐地丧失那剩余的温暖。  
既不是主角，也不是配角，只是单纯的背景美术的其中一个零件――梦想溃散的少年。  
无意义的死。本来就是无意义的生命。理当是这样的。  
\[这里是梦的\]  
对那微不足道的，像是个尘埃的弱小的生命，有个身影正凝视著他。穿著夜幕的漆黑少女。而因为她的登场，故事也面临了戏剧性的变化。舞台装置的齿轮倒转了。  
\[不被言语的故事的最后之夜与闭幕。不被回忆亦不被遗忘，也就是未曾存在过的故事现在所看见的幻影\]  
特殊的，恋人并不限定为男性。

D级  
等级名称：黑鳞生命的继承者  
价格：D+600  
特性：【射杀一切所见之物的蛇之眼】【传承记忆】  
效果：  
【射杀一切所见之物的蛇之眼】：你的瞳孔会变成如黑渊般深邃幽暗的竖瞳，你也可以以一个自由动作将其回复正常的样子，但使用时会变回原样。  
夜幕少女具有一种新的攻击方式，可以选择一个15米之内的目标，用一个标准动作，进行一次风度+社交+专业：凝视+辅助专业：魔力爆发（专注）的攻击检定，对目标造成成功数点物理钝击严重伤害。表现形式为眼中出现了一条黑龙咆哮的虚影，扫向敌人。

【传承记忆】：已经重复了数次与几百几千几万遍了，这是无数次以鲜血和生命编制出的血腥之恋的记忆的轮回，在连轮回之龙亦被风乾的长久时光后，故事仍将继续下去。  
你获得了寿命不会随时间自然流失的效果，同时外貌提升5点，但永久性减少1点意志上限，视为对精神负担的影响。

C级  
等级名称：夜幕的少女  
价格：C+1200  
特性：【只要想着与你的吻，即便是心脏里插著一把尖刀，也觉得甜蜜】【只有一人知晓的真名】  
描述：  
夜幕的少女将脸凑近，与他口吻相交。对那微弱气息的肺腑深处，灌入了龙之吹息。  
\[我的王啊。龙王啊。以你的灵魂铭记住我的真名吧。妾是黑鳞生命的继承者。既是披缠暗夜的舞蹈者，亦为奔驰於夜晚的凶风。妾是清纯之黑的龙姬……\]  
效果：  
【只要想着与你的吻，即便是心脏里插著一把尖刀，也觉得甜蜜】：获得2点快速治疗【医疗点24】，可通过接吻度气把本血统提供的快疗暂借给他人，直到结束接吻为止，你的意志力池将会再被永久减少1点上限。  
此外，你的外貌再提升5点。  
特殊：如果是与恋人接吻，在结束时会受到1点不可豁免的沮丧点数，可叠加，不会进入毁灭后果，可通过某些特殊（比如S\*X）方式消除。与恋人的其他亲密行为可以造成类似效果。

【只有一人知晓的真名】：少女对少年宣告著\[以你的灵魂铭记住我的真名吧。\]对那衰老的白龙。对那一天的少年。那是在这个世界之中，只有他才知道的，你的名字。\[妾的名字是――\]  
选择你的一个队友（通常为恋人）作为唯一知晓你真名的对象，当其位于你敏感范围内时，你的攻防获得3点神圣加值，但其一旦离开你的敏感范围，所有加值变为减值。

B级  
等级名称：披缠暗夜的舞蹈者  
价格：B+2400  
特性：【披缠暗夜】【提升C射杀一切所见之物的龙之瞳】【黑羽之翼】  
描述：  
他望向脚边。被枪弹击中的夜幕的少女已经死了。与其说那是残缺的尸体，还不如说那是被切成烂肉与骨与黑布的某物。但是...  
\[师团长！那、那个……！\]  
\[什么！？\]  
夜幕的少女的残骸，化为黑雾而逐渐挥发。自那黑雾之中，无数的乌鸦与蝙蝠、蛇与昆虫涌出，聚集，硬是结合在一起。  
乌鸦与蝙蝠与蛇与昆虫的群体模傲著少女的外貌。  
他追著她。但却为时已晚。少女翻舞著那以乌鸦羽毛、蝙蝠翅膀、蛇与昆虫所编织出的黑色洋装，  
摺叠於乙太海中的龙翼扩展开来。  
摇曳著黑翼的洋装，少女对他露出了微笑。  
\[被不识风情地打扰了呢。下次换个气氛更好的地方，再来约会吧\]  
效果：  
【披缠暗夜】:你身边围绕着一层由乌鸦羽毛、蝙蝠翅膀、蛇与昆虫所编织出的幽暗浓雾，挡开那些不怀好意的家伙。  
你不会因为完好生命全部失去而昏迷，同时获得2点全伤害吸收。

【提升C射杀一切所见之物的龙之瞳】:你的瞳孔化为鲜红欲滴的血色龙之瞳，被龙之瞳瞪视的生命体会遭受高位幻想种生命等级碾压带来的有如实质的无形威压。  
你使用龙之瞳发动的攻击可以选择不造成伤害，而是改为造成等量精神束缚点数，强韧豁免，可叠加。

【黑羽之翼】:你花费一个移动动作，展开置于以太海中的黑色龙翼，获得获得等同于你基础速度的飞行速度，机动性为完美持续一场景。

A级  
等级名称奔驰於夜晚的凶风  
价格：A+4800  
特性：【恋爱的追踪者】【黑渊的凶风】【爱意如宝石】【血腥的轮回之恋】  
描述：  
无数的飞禽与走兽与虫群从那黑暗的角落不住涌来。冲击声与啼声与振翅声与匍匐前进声攻击著你的耳朵。不协调音。  
但唤起生理上的厌恶的那声音像是要配合节奏似地，逐渐地拥有了方向性。地狱的不协调音，化为了奇妙的合唱，化为了一道旋律、终於变成了言语。  
\[请您高兴点吧、我的君王\]  
飞禽与走兽与虫群集结而结合。以飞禽与走兽与虫群所编织而成，夜幕的少女被创造而出。  
\[虽然有点多事，但我来接您了\]  
自乌鸦眼窝中长出的头部，少女露出了微笑。你因欢喜而颤抖。苦涩的爱恋化为毒素紧抓著你的心脏，甜美的憎恶化为麻药融化著你的大脑。  
她的裙摆翻舞著，无数的飞禽飞出。。为了将你的肉啄食殆尽，化为黑风而杀来。

效果：  
【恋爱的追踪者】:由于脑海中的记忆，你变得分外敏锐。  
你的敏感范围扩大一倍，同时能够以一个移动动作，获得以嗅觉侦测的非线性感知手段（允许精确定位），启动时一次性支付10点能量，同时获得敏感范围的盲视，并在侦查检定上获得1DP加值，持续一场景

【黑渊的凶风】:夜幕少女能够以一个反射动作，对自身造成1点不可豁免的冲击点数，将自己转化为为一团由无数的乌鸦与蝙蝠或昆虫组成的黑雾（视为微型集群生物），主动将自己由集群生物转化回原本的生物类型是一个自由动作，持续一场景。

【爱意如宝石】:苦涩的爱恋化为毒素紧抓著你的心脏，甜美的憎恶化为麻药融化著你的大脑。  
你心中一直满溢着爱意，这视作一种陷入毁灭性后果的情绪异常状态，无法被任何手段消除，但不会带来任何减值，只是在判定你人物的状况时视为你永久具备一项情绪异常状态，你需要在扮演时表现出这一点。你免疫A级以下影响心灵的负面效果。

【血腥的轮回之恋】:在无尽轮回中，交互死的接吻、刃的爱抚吧。如同以往一般的，如同之后一般的。  
你C级获得的快疗提升到3点【医疗点36】，并获得1点异常治疗【医疗点24】。

S级  
等级名称：清纯之黑的龙姬  
价格：S+9600  
特性：【黑昼之暗】【幻想终结】  
描述：  
你因惊愕而睁大了龙之瞳。  
他微笑着，露出了自己的喉咙。在那里埋着发出金属光芒的白鳞。  
――逆鳞。  
你的形体逐渐地崩溃。  
于是，折叠于以太海中的巨大质量被展开了。  
伴随着冲击的白色闪光渲染着黄昏的天空。  
你己经不算是人类了，硬要说的话，只不过是己经逝去的故事的亡灵，现在正在叙说的这个故事，要将其比喻的话，就像是宛如吊歌一般的事物――  
你现在成为了一种概念，一种幻想，不再被视为生物。  
相互啃咬，相乎残杀而相爱的两只龙。  
相互啃食着对方，相互贪求着对方，像是描绘着螺旋般地不断回旋，两只龙朝着天空爬升。  
看见了这个景象、啊啊――简直就是轮回之龙一般啊，她如此想着。  
终于相互地将对方啃食至连一滴血也不剩，全都回归于虚无了。  
幻想总是这样，总是什么都不留下而后离去。  
能够将幻想结束的只有幻想。这就是与众不同的屠龙者的世界观。  
而当那一切都回归于虚无的瞬间――白夜之光与夜幕爆裂开来。于这渲染天空的黄昏之红之中。  
人类战士的将领，只能呆然地仰望着。仰望着两条龙融合的爆炸中心。仰望着那一切的开始之地。

效果：  
【黑昼之暗】:当你展开翅膀时，你可以选择化为一只美丽而失去矜持，优雅而焦躁的黑龙，此时你的体积会增加，以敏感范围计算增加，这不会影响到相关数据，但会影响到你的触及范围（即射程等同敏感范围），并具有对应的碰撞体积。（虽然表现上很奇怪，但这不意味着你可以同时攻击范围内的所有敌人，直到购买并使用一个挂钩敏感范围的攻击技艺为止）（此外，也不影响敌人的触及，打不到的仍然无法攻击到）。  
如果在这个状态下使用射杀一切所见之物的龙之瞳，攻击时将会附带等额的精神束缚异常点数，标准豁免，这一异常点数升级为四档异常。

【幻想终结】:你的攻击检定获得【传奇】特性。  
选开：仅针对超自然事物时生效。

黑鳞逆鳞  
本质：魔幻本质

价格：D+500  
她自钢筋上跳入空中。无视於重力，轻飘飘轻飘飘地，就连翻舞黑色洋装的裙摆声也没有於地面著地。  
\[我的您啊\]她仰首露出了喉咙。在那里埋著发出金属光芒的黑鳞。将逆鳞――龙的要害露出，夜幕的少女倾诉著。  
\[就请您疼爱我吧、如同以往一般地\]以那因恋爱而焦躁的声音，如同诅咒一般地倾诉著。
