# 绝响之物-绝响

绝响  
本质：特异本质

这是一台带有立架的，黑红双色的麦克风，所有的声音经过它的处理会扭曲失真，似怨灵的哀嚎，又有傀儡丝抽动的声音。  
幕后人操控，  
傀儡空舞梦一场，  
线断魂魄丧。

D级  
价格：D+500  
类型：新建  
特效：  
武器伤害6L，现在你可以使用决心+表达-歌唱+专注-魔力洞察进行攻击，射程20米，造成物理-音波伤害，这种攻击具有【法破】特性。  
这种检定也被称为【和歌】检定，在画面表现上，数道傀儡丝从各处蔓延而出，对敌人进行纠缠的攻击。

逐级提升：

C级  
价格：C+1500  
类型：新建  
特性：【和歌出世】  
特效：  
武器伤害提升8L，

上一等级中使用【和歌】检定进行攻击的射程，现在提升至20+【传奇决心\*10米】。

【和歌出世】：当你进行表达相关检定时，具有9加骰。

B级  
价格：B+2500  
类型：新建  
特性：【和歌出世】、【和歌离尘】  
特效：  
武器伤害提升7L

上一等级中使用【和歌】检定进行攻击的射程，现在提升至20+【传奇决心\*20米】。

【和歌出世】：当你进行表达相关检定时，加骰提升1级处理（最高8加骰）。

【和歌离尘】：在【和歌】攻击命中后，目标需要使用意志豁免对抗伤害出目，失败则受到B级命运来源的耗竭影响，立刻损失8点AP，如同在本轮内正常使用了这些AP（即使并没有真正使用）一样处理。
