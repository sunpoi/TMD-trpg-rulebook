# 0.人机改

基础改造  
类型：一次性改造  
本质：科技本质

价格：D+600  
描述：  
在一场手术后，你的胃和肠道被动力炉取代，骨骼和血管被替换为合金和电线，肌肉成为了弹簧和液压泵，你全身上下剩下的或许只有大脑和为数不多的表皮。

效果：  
你的生命值被替换为结构值。  
注意：如果选择成为构造体，部分改造的效果会不一样，请慎重选择。  
【机械之躯】：  
你的身体变得结实。  
增加1点天生防御的同时获得基于天生防御的防弹特性，你的强韧豁免获得6点器械加值

【动力炉】：  
虽然你现在几乎不是血肉之躯，但维持这副身躯并非无需能量。因此，你体内的动力炉会为你带来行动需要的能量。  
开启能量池【电力】  
电力的上限=决心+耐力  
电力不适用于长休，如果像是正常人一样摄入三餐，电力将会回满，并且一整日不会被动消耗。但如果未能正常摄入三餐，每日将会被动消耗2电力，如果未能以待机模式长休，则额外消耗1电力，同时大脑也会感到疲劳。  
但你可以通过任意方式摄入任意有机物来获得电力补充，例如摄入250g枯枝烂叶或500ml的任意液体可以获得5电力。

义体与插件

总纲：  
人体能够搭载的义体和插件有限，如果强行搭载过多可能会引起各功能混乱，乃至于变成赛博精神病，初始肢体、器官、躯体都各具有3个改造槽。  
智力上的附加成功决定了器官能够搭载的最多改造数：1点智力上的附加成功能开启2个新的器官改造槽，1点力量上的附加成功能开启2个新的肢体改造槽，1点耐力上的附加成功能开启2个新的躯体改造槽。  
人体所有成对的肢体和器官在购买对应的改造时一般都直接成对植入，可以选择只要其中一半，但也不会降低价格，单个购买的成对义体不会因植入两次而叠加效果。除成对的器官和肢体可以分别搭载外，所有其他部分只能同时具有一种，如想搭载一种新的，就需要卸下旧的。特殊的，搭载义体有可能会占用瞳术位，如改造的部位为眼球，则会占用。  
带有【手术】字样的改造只能接受一种。

在轮回者需要且愿意付出潜在代价的前提下，最多还能超出改造槽上限，视为暂时具有等同于其风度上的附加成功\*1的改造槽，可以随意分配进三项之中，这种情况下视为超负荷改造，每具有1项占据临时槽位的超负荷改造，在影片中就需要以半周为单位，每半周开始时对每一项超负荷改造进行一次意志豁免，DC为超负荷改造数\*占据的临时槽位\*5，若未能成功通过，则这项改造会立即毁坏，回到主神空间自动取出，在回到主神空间前，每有一项毁坏的超负荷改造，就会占据你D3C6B9A12S15点意志池上限。所有人机改造不会提供属性点。

购买所有人机改的前提为具有基础改造，每一种义体或插件都会占据一定的电力能量池上限，若超过上限，则视为陷入超负荷状态。不存在任何意志池上限时，会进入迷失自我的状态，由主持人决定你接下来的状态与生死。
