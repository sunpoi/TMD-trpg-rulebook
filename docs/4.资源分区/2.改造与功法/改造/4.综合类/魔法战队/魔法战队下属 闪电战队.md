# 魔法战队下属 闪电战队

【魔法战队下属装备】  
魔法战队-闪电战队  
“直到有一天，我突然意识到。如果从一开始就能把核能转化为电能用于维持装甲的运转，那为什么我不选择直接把电能用在武器输出上？”

【D级“闪电战队”动力盔甲】  
  
代号：迅雷  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：迅雷是一件大型外骨骼动力装甲，其外观以土黄色为主，外表各处存在外置护甲与机械装置，持有可以喷射电弧的长鞭类武装。  
外置装备：【电鞭】

【电鞭】：视为一件专业为长鞭的武器，在着装盔甲时会同步被你持有，武器伤害3L，轻灵，体积3，伤害类型：物理-钝击。

你可以使用一个迅捷动作并消费1点核能启动电鞭，这会使电鞭的武器伤害上升至6L，具有【高速5】，伤害类型转为雷电能量伤害，持续一轮。  

代号：电网  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：电网是一件大型外骨骼动力装甲，其外观以土黄色为主，外表各处存在外置护甲与机械装置。电网的武装以发射通电的网状弹药麻痹目标，进行束缚敌人的方式进行战斗；由于网状弹药使用了高强度切割用钢丝绳，该武装依旧可以正常造成伤害。  
外置装备：【电网发射器】

【电网发射器】：电网发射器视为一件霰弹枪专业武器，使用【敏捷属性+射击+专业：霰弹枪+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。

电网发射器的武器伤害为4L，武器体积4，伤害类型：物理-挥砍，具有【高速5】（已计入远程武器自带的高速），【攻击距离不超过15米内的单位时获得2威猛】，速射等级【长点射】，射程40米，弹容量【11+1】。

启动电网发射器的放电功能是一个迅捷动作，你需要消耗1点核能，这会使电网发射器的攻击附带1点不可豁免的麻痹不良点数；若如此做，在攻击后电网发射器将会自动回收电网，如同该武器具有【无限子弹】一样处理。  

代号：雷霆  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：雷霆是一件大型外骨骼动力装甲，其外观以土黄色为主，外表各处存在外置护甲与机械装置。雷霆的武装是由高能力场约束的不稳定电浆，通过投掷的方式进行杀伤。  
外置装备：【电浆抛射器】

【电浆抛射器】：电浆抛射器视为一件特殊的投掷类武器，正常采用轻投掷规则，在着装盔甲时会同步被你持有，由于电浆抛射器会源源不断的产生被高能力场约束的不稳定电浆团（由于约束力场，你可以正常将其投掷），每轮你都需要使用一个迅捷动作为电浆抛射器充能，这会使其具有【无限子弹】特性。

电浆团作为投掷武器而言，视为轻投掷类武器，武器伤害为6L，武器体积1，伤害类型：雷电能量，具有【高速2】（已计入远程武器自带的高速），若你使用一个迅捷动作在攻击前对其进行充能，则提升至【高速5】，速射等级【短点射】，射程等同正常投掷武器射程。
