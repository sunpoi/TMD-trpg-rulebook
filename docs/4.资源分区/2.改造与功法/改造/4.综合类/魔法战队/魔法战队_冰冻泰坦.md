# 魔法战队-冰冻泰坦

【魔法战队下属装备】

魔法战队-冰冻泰坦  
类型：可升级防具  
本质：魔幻本质

概述：  
“冰冻泰坦是我少数在物理性质上还原能量攻击所进行的尝试之一。不过，由于低温本身在大部分场合并不算特别实用……这一类型的护甲仍然具有物理攻击的性质。”

等级名称：【D级“冰冻泰坦”动力盔甲】

代号：神盾  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：  
神盾是一件大型外骨骼动力装甲，其外观以浅蓝色为主，装甲厚重而庞大。神盾装甲携带的巨盾能够有效阻碍目标攻击。  
特性：【斯巴达之盾】  
效果：  
【斯巴达之盾】：  
视为一件盾专业武器，在着装盔甲时会同步被你持有。斯巴达之盾的武器伤害为3L，武器体积4，具有【格挡5】【双手】，伤害类型：物理-钝击。

你可以消费一个移动动作进行基础规则中的格挡行为，随后在该场景内，你从斯巴达之盾中获得的防御增加1点；当你使用斯巴达之盾攻击并命中目标后，目标需要使用强韧豁免对抗伤害出目，失败则在该场景内只能攻击你，这是一个D级创伤来源的【胁迫】效果。这一效果同样需要你先使用移动动作进行格挡后才能生效。

代号：镜像  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：镜像是一件大型外骨骼动力装甲，其外观以浅蓝色为主。介于低温在战斗中的非显著效果，镜像配备了纯物理伤害的武装，用于在特定场合下降低战斗压力。  
特性：【两面神火炮】  
效果：  
【两面神火炮】：两面神火炮视为一件炮专业武器，使用【敏捷属性+射击+专业：炮+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。两面神火炮的武器伤害为8L，武器体积5，伤害类型：物理-穿刺，具有【高速5】（已计入远程武器自带的高速），【破甲3】【无限子弹】【双手】，速射等级【无】，射程200米。  
两面神火炮在攻击时可以额外选择一个目标（无法选择同一目标）。

代号：冰甲  
分类：轻甲  
基本属性：盔甲防御2/2  
特殊属性：无  
描述：冰是一件大型外骨骼动力装甲，其外观以浅蓝色为主。除了装甲结构外，冰甲减轻了大部分结构以安装低温发射器，用于迅速冻结目标。  
特性：【低温发射器】  
效果：  
【低温发射器】：低温发射器视为一件手枪武器，使用【敏捷属性+射击+专业：手枪+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。低温发射器的武器伤害为3L，武器体积2，伤害类型：物理-穿刺，具有【高速5】（已计入远程武器自带的高速），【无限子弹】，速射等级【短点射】，射程30米。

启动低温发射器需要一个移动动作，随后在该场景内你可以正常使用这一武器进行攻击；低温发射器命中后会使目标承受1点不可豁免的冻结不良点数。

支付D+1000：斯巴达之盾提供的【全遮蔽】效果可以同时对触及范围内的友方目标生效，购买这一能力如同正常购买了一项D级技艺一样处理，因此会正常提供自由属性。

你随时可以使用一个花招动作降低低温发生器的功率，从而失去增加冻结不良点数的效果，也无需继续额外承受相应不良点数（已承受的不良点数正常生效）。

你随时可以使用一个花招动作降低低温发生器的功率，从而失去增加冻结不良点数的效果，也无需继续额外承受相应不良点数（已承受的不良点数正常生效）。
