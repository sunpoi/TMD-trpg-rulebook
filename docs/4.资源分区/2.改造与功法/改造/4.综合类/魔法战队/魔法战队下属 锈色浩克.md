# 魔法战队下属 锈色浩克

【魔法战队下属装备】  
魔法战队-锈色浩克  
“在收集了前两次的数据后，我对旧时代的火炮进行了科技迭代。介于之前的护甲不得不降低防护性的经验，这次我结合了迷雾放射器装置，其中遍布的干扰粒子应该能起到良好的效果。”

【D级“锈色浩克”动力盔甲】  
代号：战机  
分类：轻甲  
基本属性：盔甲防御2/2  
特殊属性：无  
描述：战机是一件大型外骨骼动力装甲，其外观主要以橙色为主，在外表各处存在外置轻量化护甲，是少有的可飞行装甲。  
外置装备：【外挂式机炮】【喷气式飞行器】

【外挂式机炮】：外挂式机炮视为一件机枪类武器，使用【敏捷属性+射击+专业：机枪+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。

外挂式机炮的武器伤害为4L，武器体积4，伤害类型：物理-穿刺，具有【高速5】（已计入远程武器自带的高速），【贯穿】，【无限子弹】，速射等级【连射】，射程100米。

【喷气式飞行器】：启动喷气式飞行器是一个移动动作，随后在该场景内你获得等同基础移动速度的飞行能力，机动性普通；在这一等级下，飞行器的防护较为脆弱，因此你获得【物理易伤1】。

代号：重炮  
分类：重甲  
基本属性：遮蔽防御4/4，盔甲减值3  
特殊属性：无  
描述：重炮一件大型外骨骼动力装甲，其外观主要以橙色为主，在外表各处存在外置护甲与迷雾装置。重炮的武装毫无新意，但实践证明它永远足够“实用”。  
外置装备：【穿甲火炮】【迷雾放射器】

【穿甲火炮】：穿甲火炮视为一件炮专业武器，使用【敏捷属性+射击+专业：炮+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。  
穿甲火炮的武器伤害为8L，武器体积5，伤害类型：物理-穿刺，具有【高速5】（已计入远程武器自带的高速），【破甲3】【无限子弹】【双手】，速射等级【无】，射程200米。

【迷雾放射器】：重炮提供的防御并非是盔甲防御，而是永久性改为遮蔽防御；任何应用于重甲并带来盔甲防御提升的效果，对于重炮而言皆改为提升遮蔽防御，但无法使其他效果中基于盔甲防御的特性改为基于遮蔽防御。

代号：蜘蛛  
分类：轻甲  
基本属性：遮蔽防御2/2  
特殊属性：无  
描述：蜘蛛一件中型外骨骼动力装甲，其外观主要以橙色为主，在外表各处存在外置轻量化的护甲与迷雾装置。蜘蛛武装的特色在于它能抛射多个自走型炸弹，并在爆炸后形成大片迷雾。  
外置装备：【子母弹】【迷雾放射器】

【子母弹】：子母弹视为一件冲锋枪类武器，使用【敏捷属性+射击+专业：冲锋枪+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。

子母弹的武器伤害为3L，武器体积3，伤害类型：物理-穿刺，具有【高速5】（已计入远程武器自带的高速），【无限子弹】，速射等级【连射】，射程20米。

在这一等级下，子母弹攻击后会额外分裂出一个小型自走炸弹，对于你攻击范围内的一个敌对目标进行一次相同的攻击（等于多目标攻击能力，无法选择同一目标），若不存在额外敌对目标则无法分裂；当你使用子母弹进行攻击后，你获得1点掩蔽防御，持续一轮。

【迷雾放射器】：蜘蛛提供的防御并非是盔甲防御，而是永久性改为遮蔽防御；任何应用于蜘蛛并带来盔甲防御提升的效果，对于蜘蛛而言皆改为提升遮蔽防御，但无法使其他效果中基于盔甲防御的特性改为基于遮蔽防御。
