# 魔法战队-钢铁柔道家

【魔法战队下属装备】

魔法战队-钢铁柔道家  
类型：可升级防具  
本质：魔幻本质

概述：  
“随着战况推进，传统的动力推进器显得格外沉重，在应对高速机动单位的表现上差强人意，因此我设计了一套重力发生装置用于取代原本的推进器……后来发现不如直接制造一套专门使用的装甲。”

等级名称：【D级“钢铁柔道家”动力盔甲】

代号：柔道家  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：  
柔道家是一件大型外骨骼动力装甲，外观如同钢铁一样呈现出冷硬的深灰色。虽然装甲较为庞大，但其安装的重力发生装置可以一定程度的利用这一点。  
特性：【罪恶拳套】  
效果：  
【罪恶拳套】：视为一件拳套类武器，在着装盔甲时会同步被你穿戴，这会使你的肉搏攻击获得1个额外附加成功数，同时造成的伤害由冲击转化为严重，伤害类型：物理-钝击。  
支付1点核能，本轮内当你使用罪恶拳套攻击并命中后，目标需要使用正常绊摔流程对抗这次伤害出目，失败则陷入【伏地】状态；当你攻击处于【伏地】状态下的目标时，你的攻击获得1DP器械加值与额外1个附加成功数。

代号：围城  
分类：重甲  
基本属性：盔甲防御5/5，盔甲减值3  
特殊属性：无  
描述：围城是一件大型外骨骼动力装甲，外观如同钢铁一样呈现出冷硬的深灰色。作为极少数将重力发生装置直接应用在武装上的装甲，围城代表了其能够应对多目标攻击的性质。  
特性：【重力炮】  
效果：  
【重力炮】：重力炮视为一件炮专业武器，使用【敏捷属性+射击+专业：炮+辅助专业：弹道学+武器伤害+其他加值（若有）】进行攻击，在着装盔甲时会同步被你持有。

重力炮的武器伤害为5L，武器体积5，伤害类型：力场伤害，具有【高速5】（已计入远程武器自带的高速），【破甲3】，【双手】，【无限子弹】，速射等级【无】，射程200米。  
重力炮可以同时攻击等同你【传奇敏捷】\*1个目标，无法重复选择同一目标，这是一个多目标攻击能力。

代号：引力  
分类：重甲  
基本属性：盔甲防御4/4，盔甲减值3  
特殊属性：无  
描述：引力是一件大型外骨骼动力装甲，外观如同钢铁一样呈现出冷硬的深灰色。引力的武装并不是为了杀伤而设计的，其主要目的是限制目标攻击，以及控制目标的移动能力。  
特性：【引力发生器】  
效果：  
【引力发生器】：引力发生器使用【智力属性+学识+专业：重力发生器+辅助专业：重力学（学识下属专业）+武器伤害+其他加值（若有）】进行攻击，引力发生器在着装盔甲时会同步被你持有。

引力发生器的武器伤害为6L，体积2，伤害类型：力场伤害，射程15米。

当你使用引力发生器攻击并命中后，目标使用反射豁免对抗伤害出目，失败则目标在该场景内因重力而陷入D级环境来源的【困难地形】影响，持续一场景。

这一效果需要消费一个标准动作启动，你可以藉由引力发生器获得【改变重力】效果，若需要处理其效果来源视为A级环境来源。

这意味着你可以使自身移动速度增加该武器支线等级（5）\*20＝80米，或当你使用引力发生器攻击并命中后，你可以让目标使用反射豁免对抗伤害出目，失败则你可以让目标在该场景内陷入【超重】或【失重】状态。

失重等同异常状态中的失重；超重意味着目标移动时视为处于A级（及以上）困难地形中，若目标拥有飞行，则飞行能力下降一级。载具的机动性能力不变，但加速度降低一半。

你可以额外支付C+1000分，如同购买了一项C级技艺一样正常提供自由属性，让你施加该武器提供的失重或超重，又或者加速能力时，可以同时作用于范围内的所有单位，此时不再需要处理豁免，而是范围内的单位直接承受效果，但这一效果会延迟一轮后发动（注意，如果你和队友没有A级环境来源的效果免疫，也会被失重或超重影响）。
