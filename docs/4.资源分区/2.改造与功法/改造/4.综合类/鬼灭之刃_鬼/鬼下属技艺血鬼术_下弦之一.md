# 鬼下属技艺血鬼术-下弦之一

血鬼术  
下弦之一.魇梦.强制昏睡催眠.凝视  
C+1000  
通过眼睛对视让指定的目标陷入沉睡幻境的血鬼术，只有通过结束在自己梦里的生命才能脱离。  
强化时需要绑定一个通过眼睛进行攻击的攻击方式才能使用，绑定后不能更换，该种攻击方式视为【血鬼术】  
你的攻击在命中后可以选择不造成伤害，而造成等量C级心灵来源的精神束缚点数，目标以反射豁免对抗攻击出目，你的眼神更加摄人心魂，该种攻击方式具有C级措手不及。
