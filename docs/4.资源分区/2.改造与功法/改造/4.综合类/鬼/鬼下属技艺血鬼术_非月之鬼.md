# 鬼下属技艺血鬼术-非月之鬼

技艺名称：血鬼术-朱纱丸.手球  
价格：C+1000  
描述：  
某个把自己说死的鬼的血鬼术  
能够让自己投出的手球具有极其强大的威力，而且能够一定程度上的自由控制。  
效果：  
你投掷球形物体进行的攻击变为【曲线型】且具有【贯穿】和1点额外的附加成功。你投掷球形物体进行的攻击视为【血鬼术】。  
由于TMD主神阻止了涩情要素，所以额外从侧胸部分长出手臂的效果被砍掉了。

技艺名称：血鬼术-矢琶羽.红洁之箭  
价格：C+1000  
描述：  
某个眼睛长手上的鬼的血鬼术  
能够从自己的眼睛里发出具有力量的无实体红色箭矢使目标进行移动的能力。  
效果：  
你可以用眼睛发射出红色的箭矢进行攻击，这种攻击方式以【力量+射击+能量控制+辅助专业：弹道学】进行攻击，具有20M射程，3点武器伤害和一个额外的附加成功，造成立场伤害。  
该种攻击方式无法应用于攻击多个单位或者转变为范围攻击的能力。  
你可以将箭头方向转化为向上，该次攻击检定不造成伤害，命中后，对方以反射豁免对抗你剩余攻击出目，失败则被你击飞胜出的成功数。这是一个D级创伤来源的击飞效果。  
同时的，每轮一次，你可以通过箭头表现出自己的力量，击中目标后你可以用自由动作通过箭头进行一次特殊的拖拽，对方以反射豁免抵抗。
