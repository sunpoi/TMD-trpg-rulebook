# 鬼下属技艺血鬼术-下弦之二

技艺名称：下弦之二.佩狼.卤获腔.影狼  
价格：C+1000  
描述：如狼般的无形黑影从影子中冒出，或探出枪械，或撕咬敌人。  
效果：  
现在你的触及范围内会生出等同你传奇操控只狼型黑影，每个黑影能够如同你一条肢体一般的持有枪械类武器。

如果你选择不让所有黑影持有任何武器，其会帮助你撕咬你的敌人，你使用体积3以上的枪械武器不再因为处于对方触及范围内高速减半。

技艺名称：下弦之二.佩狼.卤获腔.战祸阵狼  
前置：下弦之二.佩狼.卤获腔.影狼  
价格：C+1000  
动作：移动  
消耗：2意志  
持续：一场景  
描述：将所有影子集合，凝聚自身，化为如同狼人的模样，覆盖于身的影子将给与强大的吸收力。  
效果：  
以一个移动动作消耗2点意志力凝聚影子上身，在本场战斗中，你额外获得8点盔甲防御，以及4点基于盔甲防御的dr/-。
