# 为你纹身

  

为你纹身   
C+1200  
  
替身外貌：  
替身的外形是覆盖了持有者头发和后背的报纸状纹身，于替身持有者的后脑上额外附带了一张苍白的小脸。  
  
替身数据：  
破坏力 无  
速度E（偏斜防御）  
射程C  
持续力B  
精密动作性E  
成长性E  
替身对应主属性：操控  
  
替身能力：  
进出纹身：觉醒为你纹身的替身使者可以将替身共享给其他人，使其他人背上也出现报纸状纹身，可共享总数为【替身使者的操控属性附加×3】人。持有纹身的人可以以同伴身上的纹身作为出入口，自由进入同伴体内的纹身空间，进出纹身空间是一个结合在移动的动作，并且不能在一个动作同时进行进出行为。每个人纹身内的容纳人数没有限制，但是所有人纹身内的总容纳体积上限为【替身使者的操控属性×50】体积。当同伴死亡时，潜伏在同伴体内的其他人不会受到影响，可以在同伴的体内缓慢地移动尸体。此外在替身使者射程范围内的纹身空间是联通的，并且可以自由决定从那个纹身出来，若替身使者处于某一个同伴体内的纹身空间时，以那个同伴为起点计算射程并共享纹身空间。  
以上能力是一个C级别、时空来源的异空间效果。
