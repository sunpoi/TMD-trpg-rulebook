# 恶行易施

  

恶行易施/极为轻易行出的肮脏行径/D4C  
  
SSS+28800（替身价格）  
  
替身外貌：主体为宝蓝色加粉红色，外貌是个类似机器人的人型替身，有着巨大直立的角，如同兔子的耳朵，他的下颚被面具遮盖，全身布满着粉红色镶边。  
  
替身数据：  
破坏力A  
速度A（偏斜防御）  
射程A  
持续力A  
精密动作性A  
成长性C  
  
替身对应主属性：决心  
  
替身能力：  
近距离力量型：近距离力量型都具有的基础能力，控制替身直接对目标进行肉搏攻击，检定方式为【决心—专注—主专业：替身使用—副专业：精神力（专注下属副专业）】。

平行世界穿越：只要能力射程范围即【决心×5】米内的人或物品被夹在两个物体之间就可以将其拉入或拉出平行世界，两个物体中的其中一个能包裹住被夹物体时便可发动能力，例如门和后面的墙壁、地面和国旗，甚至是水滴和地面，而平行世界有无限个且D4C可以自由穿梭，这视为SSSS级异世界效果。并基于平行世界的存在，有一条世界规矩，除了觉醒恶行易施的替身使者以外的不同世界的同一事物出现在同一世界时，并且之间没有阻碍且距离在10米以内时会互相吸引并湮灭，此时湮灭产生的碎末为门格海绵状，此时湮灭产生的碎末为门格海绵状，而湮灭的威力足以消灭湮灭的对象，但其对周围的破坏力不大，这视为SSS级时空来源的即死效果，即便具有相关豁免抗性也会也承受【决心】点不可豁免的力场伤害。关于平行世界的细节方面情况，因为具有无限的平行世界，因此可以总能找到各种各样的平行世界，但所连接的每个平行世界中另一个自己所觉醒的恶行易施都不具有任何替身能力，只具有基础的替身数据，而另一个自己也只有平凡人的身体素质与能力。此外一些极其关键的物品在所有平行世界中只会存在一个，其余平行世界中会以其它物品代替其对应的位置。

本体更换：觉醒恶行易施的替身使者可以自由与另一个平行世界的自己交换替身，使其成为新的本体，新的本体将继承之前本体的记忆和能力，并且新的本体一般是完全健全的状态，因此可以通过这种方式恢复自身状态，视为SSS级时空来源的类复活能力。

觉醒能力：  
爱之列车 S+8000  
不幸转移：使自身周围出现光之幸运壁垒后，处于壁垒的保护时会完全切断外部对其的攻击，并将试图攻击自身的行为视为【不幸】并被转移到其他无辜的人身上，相应的如果要攻击壁垒外的敌人，自身攻击的部分也得离开壁垒。光之幸运壁垒是一个次元的缝隙，用正常的方法是绝对无法突破的，因此这视为基于平行世界穿越能力延伸的SSSSS级异世界效果与S级命运效果的结合。
