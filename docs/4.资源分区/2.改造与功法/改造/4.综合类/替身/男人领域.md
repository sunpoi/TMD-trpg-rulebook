# 男人领域

  

男人领域   
S+9600  

替身数据：  
破坏力 无  
速度A（闪避防御）  
射程无  
持续力E  
精密动作性无  
成长性C  
  
  
替身能力:使用替身能力会使时间回溯，使一切恢复到3秒钟前的状态，每次发动后必须等时间流逝到上次发动的时间点经过一轮后才能再次发动。在这种回溯下觉醒男人领域的替身使者与指定人物可以保留回溯前的记忆。  
此外在满足一些条件后回溯的时间会变长，将一个特殊的动作作为触发点，做出此动作时立即进行回溯，这个动作一般需要一个花招动作执行，可以再增加3秒回溯时间；决心属性达到26点时可以再增加3秒回溯时间。  
此外，当人物表现出足够具有「漆黑意志」时，可以提议将对决变得公平，这样有助于人格的成长：主动将替身能力告知每一个正要战斗的敌人，这场战斗中，可以再增加3秒回溯时间。这是一个S级别、命运来源的回溯效果  
由于公平决斗的意志在指引着替身的主人，因此，男人领域至多能每个行动轮回溯时间一次，该效果的冷却时间以使用这一能力时的行动轮次为准。
