# 愚者

  

愚者  
  
B+2400  
替身外貌：  
愚者拥有机械动物的外型，与类似汽车底盘的东西结合在一起，前脚是机械兽爪，后脚部分则是轮子，头部造型类似部落土著的面具。  
替身数据：  
破坏力B 速度C（格挡防御） 射程D 持续力C 精密动作性D 成长性C  
替身对应主属性：操控  
替身能力：  
沙子操控：具有操控沙子的能力，能以一个标准动作通过替身使用检定操纵沙子引起沙子暴风或者用沙子攻击射程范围内的单个或多个目标，同时这种情况下可以视为一个无特殊效果的D级环境。  
元素化替身：愚者的身体是沙子构成的，攻击愚者只会导致沙子被打散，不会对本体造成任何伤害。反过来，当本体被击败时，愚者会变回一摊沙子。  
  
沙子伪装：能以自由动作操控沙子变形为自己指定的形态和颜色，以此来达到伪装事物的目标，变形后的目标大小不能超过替身的射程范围，这是一个B级环境来源的伪装效果。  
他人可以通过调查检定来对抗你的伪装检定，你可以决心替代敏捷进行伪装检定。  
除非你伪装后的事物离开了你的替身射程，否则其只有在被攻击后才会化成沙砾。  
  
替身能力开发  
  
沙子堡垒 B+2400  
能以一个移动动作操控沙子制作出一个像爱斯基摩冰屋形态一样的防护设施暂时躲避外侧的攻击，其大小上限为替身能力射程范围，持续一个场景。  
处于沙堡内的单位无法对外部进行攻击，除非其离开沙堡所在区域。  
沙子堡垒将为处于内部的所有单位在受到沙堡以外攻击时提供等同于使用者【操控】点的临时生命，只要沙堡内任一单位的临时生命被消耗完毕，其余单位将无法再享受沙堡所带来的好处。  
人物可以重新消耗移动动作制作出新的沙堡。  
沙堡只能同时存在一个。  
  
沙子滑翔翼 D+600  
愚者可以伸出一个滑翔翼来带着替身使者进行飞行，需要以标准动作或杂技动作维持，视为机动性为良好的飞行方式。
