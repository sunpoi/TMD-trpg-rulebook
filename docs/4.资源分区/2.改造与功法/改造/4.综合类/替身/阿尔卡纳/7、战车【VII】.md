# 战车

  

银色战车  
  
A+4800（替身价格）  
替身外貌：  
主体银色，外型类似穿着中世纪盔甲的瘦小机器人，肩垫和肘部都附带重型装甲，头盔前额嵌有一颗宝石。  
替身数据：  
破坏力C 速度A（闪避防御） 射程C 持续力B 精密动作性B 成长性C  
替身对应主属性：决心  
替身能力：  
近距离力量型：近距离力量型都具有的基础能力，控制替身直接对目标进行白刃攻击，检定方式为【决心—专注—主专业：替身使用—副专业：精神力（专注下属副专业）】。  
  
【ホラホラ】：由于银色战车的速度达到了A，因此在进行白刃攻击时可以连打。消耗一个标准动作，对目标快速攻击三次。  
  
西洋剑剑术：银色战车可以像正常使用白刃武器那样使用自身所携带的西洋剑，可以用其格挡。  
  
银色甲胄：银色战车穿着厚重的铠甲，能提供【决心】点盔甲防御，此外也能以一个移动动作脱下铠甲，牺牲铠甲提供的盔甲防御为代价获得极快的速度，转而在替身使用检定上获得【决心】点速度加值，也能用一个移动动作穿回铠甲，之外无论如何穿上铠甲和脱下铠甲必须间隔一轮。  
  
剑身射出：将仅有一个的剑身像子弹一样发射出去，发射出的剑身可以反弹数次后击中目标。但这个招数因为危险性极大，需要手动去回收剑身否则就无法再使用剑，需要结合在一次替身攻击当中使用，将这次攻击的距离提升【决心×5】米。
