# 女帝

  

女帝  
  
A+4800  
女帝在寄生前几乎微不足道，然而在寄生后，女帝就会渐渐长大，最初这只是一个类似于虫子叮咬，小而痒的疣，之后它随着时间生长成一个小人形在妮娜，除此之外，女帝也能以相对于它的大小的外部来源为食而成长，无论是小昆虫、水果还是整只鸡。当它发育成熟时，看起来像一个从受害者的肉体中长出来的小女人，同时长出双臂且双臂肌肉会随着成长变得健壮，力量也会变得更强。  
  
替身外貌：  
主体红色，外观看起来像一个没有头发的壮硕女性，头上有两个头巾和一条披肩。  
  
替身数据：  
破坏力C  
速度E（格挡防御）  
射程A  
持续力A  
精密动作性D  
成长性D  
  
替身对应主属性：耐力  
  
替身能力：  
远距离操纵型：作为远距离操纵型，射程距离为【耐力×10】米，能在这个距离控制替身，对一个你能感知到的单位主动使用血液寄生特性，这需要目标满足以下条件之一：  
1、直接接触替身使者的新鲜血液。  
2、你经过一个攻击命中检定，让自己血液与目标接触。  
3、对方对你造成了非精神伤害，且对方处于你的触及范围范围。  
4、他人使用沾有你血液的物品攻击命中了一个目标。  
  
血液寄生：女帝是绑定在血液中的替身，以一个标准动作控制女帝寄生到目标身上，目标以强韧豁免对替身使者的替身使用检定，不造成伤害，对抗失败会被女帝寄生。目标需要以侦察检定对抗你的【决心+隐秘+伪装】，若失败则无法发现其被女帝所寄生，这是一个A级创伤来源的伪装效果。  
寄生成功后，替身使者选择目标躯体上任意一个区域，在该区域生长出一个体积为1的【寄生体】，通常而言这个寄生体样貌类似疣、水泡、痣。  
每轮一次，替身使者可命令寄生体食用目标触及范围内的无支线食物，每食用1体积的食物，女帝就获得1点每轮刷新的AP。这一行为可能会引起被寄生者的警觉。  
每经过十分钟，女帝会获得1点每轮刷新的AP。  
女帝至多具有替身使者耐力点AP。  
女帝的AP用途：女帝能且仅能将AP转化为标准动作，女帝的标准动作仅能进行影响被寄生者触及范围的行为。  
女帝进行的任何检定均由替身使者进行，进行的攻击检定采用替身使用检定。  
当被寄生者发现女帝后，被寄生者每轮可消耗一个标准动作进行一次强韧豁免对抗寄生时的检定成功数，对抗成功可脱离其寄生，除此之外在寄生期间女帝视为与被寄生者为同一个个体，但只有替身才能单独伤害女帝，其余类型的攻击均视为攻击被寄生者，以上效果视为A级创伤来源的附身效果。  
  
尸体伪装：可以将因女帝寄生而死的目标尸体挖空，以供替身使者用来伪装，并且可以被尸体生前的声音说话、行动，目标可以用调查检定以识破你的伪装检定，这是A级别创伤来源的伪装效果。
