# 星星

  

白金之星  
  
SSS+27200  
替身外貌：  
主体为蓝紫色，外观为留着长发的壮硕男性。  
  
替身数据：  
破坏力A  
速度A（偏斜防御）  
射程C  
持续力A  
精密动作性A  
成长性A  
  
替身对应主属性：决心  
  
替身能力：  
近距离力量型：近距离力量型都具有的基础能力，控制替身直接对目标进行肉搏攻击，检定方式为【决心—专注—主专业：替身使用—副专业：精神力（专注下属副专业）】。  
  
オラオラ：由于白金之星的破坏力与速度都达到了A，因此在进行肉搏攻击时可以连打，消耗一个标准动作对目标快速攻击四次。并且可作为一种应对技，每轮4次，不以防御对抗对方的伤害数，而是进行一次普通攻击，以攻击伤害数抵消对方，与其它应对技一样需要反射动作调用，这是一个SSS级时空来源的无敌效果。  
  
白金之星·世界：白金之星具有对时间效果的抗性与免疫力，视为SS级时空来源免疫。当遇到S级及以上的时间停止能力时，免疫后将会解锁令时间停止的替身能力。  
动作：迅捷  
范围：特殊  
成分：言语  
目标：场景内除自身外的所有单位  
效果：  
使用本能力后的下一行动轮开始时，人物率先进行一个特殊的回合，但不会因进入此回合获得AP点数（无论是基础还是额外的），取而代之的是人物可以将自身回合可使用的至多4点AP分配至此回合内。人物的耐力与决心属性上每合计有1个附加成功，可分配给此特殊回合的AP值就增加1点。  
受影响单位在应对人物在这一特殊回合内进行的任何行为时都处于SS级时空来源的时间停止状态。  
若有单位免疫此能力的影响，则其也会如同你一般获得特殊回合。在判定执行此特殊回合的先后顺序时，以先攻顺位为准。
