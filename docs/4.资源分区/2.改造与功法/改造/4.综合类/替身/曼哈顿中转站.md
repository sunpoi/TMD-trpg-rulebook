# 曼哈顿中转站

  

曼哈顿中转站   
C+1200  

替身外貌：  
主体蓝色。曼哈顿中转站是由一个圆形金属核心组成，四个矩形机翼围绕并固定在核心，下面还悬挂了两个钥匙状的天线。  

替身数据：  
破坏力E（不具备任何攻击力）  
速度E（闪避防御）  
射程A持续力A  
精密动作性A  
成长性C  
替身对应主属性：感知  
  
替身能力：  
子弹重定向：能使替身使者的子弹改变一次方向，但是需要精准地射中曼哈顿中转站的子弹才能从曼哈顿中转站所在的位置改变方向继续发射，改能力只能改变方向，不会影响子弹原有的任何效果。  
探测气流：精确地感知周围空气的流动，并将信息传递给替身使者。这视为曼哈顿中转站拥有一种感知气流的非线性感知方式，感知范围等于替身使者的感知范围，并且与替身使者共享。并且当替身使者主动封闭直接视觉时可以更为精准地感知气流，视为强感知方式。
