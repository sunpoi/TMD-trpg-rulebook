# 大冰

大冰（称号） B+2000

  
获得这个称号后，你可以给自己改名叫大冰。

  
我很少用人山人海来形容一个人，但......

  
你会获得主持人、作家、民谣歌手、油画画师、皮匠、银匠、手鼓艺人、黄金左脸、老背包客、法国骑士、禅宗弟子......等等头衔，你也可以自行添加。是的，进入影片后你自动获得如上的职业身份，不用担心自己没有相应技能，主神已经给你准备好了相应的证书、记忆以及对应的社会背景关系，如果影片背景内没有相应的职业，则会自动将其替换为相近的职业。

  
任何人在第一时间都觉得你是人间至宝，黑夜一望无际的大海上给迷途的船只指引的灯塔，在烈日炎炎的夏天吹来的一股清风，在寒风刺骨的冬天里燃起的阵阵簧火！不过，对方如果觉得不太对的话，可以进行一个DC为16的侦查检定，成功的话对方会发现你只是个写鸡汤的——这不妨碍你的书异常畅销到能出盲文版，然后摸两个小时才能发现自己摸的是简介；

  
而失败的话对方会被你迷地五体投地，并自动在你的名字后面加一个老师的尊称。在拜读你的作品后，对方自动学会大冰两个字的写法，学会阿弥陀佛的读音，记住你走过的西藏路线，并对诋毁你的人说一句：你坏！对方会发自自内心获得如下感受：大冰有一抹媚劲。感觉真不是那种小男生能比的。温柔的声音，让我感触到激情和温暖，携带着潜在的危险，撩拨我内心暗藏的火焰。其实大冰就是这种有男人味的男人，不需要长的如何倾国倾城，而是隔着屏幕就能感受到的，男人的香味。懂的自然懂，不多说了。你们真的是纯纯被国内白幼瘦的网红脸把审美给带歪了，攻击人品也就算了，攻击长相的，但凡多看看时尚杂志，也知道这种长相是难得的高级脸超模脸，大冰相貌艳而不俗，有辨识度，当超模完全绰绰有余了。

  
觉得职业有点多了吗？没关系，你随时可以将自己变为集群生物，通常数量可以直接与你的头衔挂钩。非战斗状态下每一个体都会占据相应的体积，这会让你被视为一个由不同头衔的你所组成的集群，虽然不会影响属性，而且我并没有在玩大冰梗，但我还是想说：大冰一出现就发狠了，忘情了，没命了！百十个身份的残影，如百十块被强震不断击起的石头，狂舞在你的面前。酒吧掌柜一样，是急促的鼓点；手鼓艺人一样，是飞扬的流苏;民谣歌手一样，是蹦跳的脚步；摄影大师一样，是闪射的瞳仁；法国骑士一样，是强健的风姿。在黄金左脸上，爆出一场多么壮阔、多么豪放、多么火烈的变化哇—大冰！
