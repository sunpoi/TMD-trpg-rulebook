# 简易长矛组装术

名称：【蕾忍派秘传·简易长矛组装·术】技艺  
价格：C+1000  
本质：特异本质  
描述：  
当你想要消灭碍眼的敌人的时候，再也不用因为手里的武器长度不够而痛苦了  
来自绝密监狱岛的传奇技艺——【蕾忍派秘传·简易长矛组装·术】，将会让你摆脱这一困境  
下面就让我告诉你如何制造这一神器：

第一步，找一个你要握持的主武器（必须是近战冷兵器，不然你为什么要提高射程呢？）  
第二步，找到绳子、胶布、胶水、发带、订书机等其他任何可以用于固定的东西  
第三步，拿起你周围的另一备用武器（同样是近战冷兵器），把它们绑在一起  
效果：  
你需要以一个自由动作来进行以上组装，组装后持续一个场景，以此方法组装后的武器名称将会变成【简易长矛】，这意味着如果有涉及武器真名之类的效果将会失效  
【简易长矛】在你握持的主武器基础上，额外拥有你添加的另一备用武器的【一半武器伤害】与【所有特性】，且触及范围提升3米  
此外，【简易长矛】还可以满足【长柄】、【双手】、【临时武器】的要求，额外具有【破甲5】，但是使用它的专业仍为主武器对应的专业，并且无论如何必须用双手持用

名称：【简易版·简易长矛组装·术】技艺  
价格：D+500  
描述：  
有人反应原版的技艺太过深奥，把武器一个个绑起来的手法还是太吃操作了  
主神，主神，有没有更简单更轮椅的提升冷兵器射程的技艺呢  
有的，兄弟，有的，这里是【简易版·简易长矛组装·术】  
该版本的操作方法和原版一致，并且因为太过简易，可以单手持用了。此外，如果使用时太用力，可能会散架。  
效果：  
组装后的【简易版简易长矛】的效果改为【武器伤害】提升3L，触及范围提升1m，但失去了额外的【破甲5】

该技艺无法与【蕾忍派秘传·简易长矛组装·术】共存，但是可以差价升级为【蕾忍派秘传·简易长矛组装·术】
