# 呼名即应

名称：呼名即应  
价格：c+1000  
本质：特异本质  
描述：  
姓名，是生物区分与标示他物的产物  
真名，是一个人生来被赋予的事物  
真名含有力量，真名具有灵性  
而性有灵者，以其名呼它，当有反应  
呼名则应，呼名即应

【本性灵名】：他人的真名亦有灵性，通过呼唤，能达成各种效果。

特性：【本性灵名】【呼名则应】【呼名即应】  
效果：  
【本性灵名】：你的真名拥有灵性，当他人以你的真名称呼你时，你将亦有所感，知晓其大致方向与所在地区，这是一个命运来源c级预知效果。

【呼名则应】：当有人以名唤你时，当你们在同一座城市中时，你将可以进行十分钟的旅行准备来召唤一扇传送门，传送门是单向的，让你能够抵达【本性灵名】所有所感应的区域，这是一个d级的命运来源传送门类旅行效果。

【呼名即应】：当你呼唤其他人的真名时，你将与其搭建沟通的渠道，距离无限，并且无需借助其他事物便能以文字、语音、图片、视频等方式进行沟通，当然，对方也可能拒绝进行沟通，这是一个d级的命运来源信息共享效果。
