# speed战吼

  

speed战吼  
500  
魔幻本质  
概述：说真的，当主神空间第一次在一个平行时空中观测到这个申必的油管黑人狠活主播时，就连无敌的主神都差点哼起了阳光彩虹小白马。最终，在几位轮回者的不懈努力下——他们在强化归零的情况下重复了这位主播整的所有狠活后，完成了这个堪称史的支线任务。在那之后，一个神秘的资源悄悄出现在了主神空间的资源兑换表中。  
【甲亢】：在购买该技艺后，你立即获得speed语1级，并且你的语言不可避免地转变为speed语言风格，这意味着你所有的语言行为都会带有不可避免的大声吼叫，这在旁人看来如同你得了甲亢一般。这也意味着你会承受1点不可豁免的亢奋点数。  
唯一的好处是当你购买该语言技艺后，你在耐力检定上获得1dp表现加值。这大概是这种语言里隐藏的某种模因吧。
