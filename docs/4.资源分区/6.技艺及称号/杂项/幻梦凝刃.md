# 幻梦凝刃

  

幻梦凝刃  
本技艺为差价升级  
本技艺是一项武器的隐匿之法，掌握这门技艺的少女们，能够将她们手中的武器随心境幻化作不同形式隐匿。无论是蝶翼，花瓣还是星光，武器都能隐匿于这些唯美的幻象之中，优雅与神秘，将会成为其武器的代名词。  
当技艺的使用者进入战斗状态时，武器的锋芒并非未消失，而是与这些梦幻元素融为一体。在她们的手中，武器可以化作任何梦幻的形态，从轻盈的花瓣，到流动的星光，甚至是静静围绕在身旁的蝶翼。当武器在手中时，表面依旧保持无形，敌人只会看到这些美丽的元素在她们的周身萦绕，而一旦她们发动攻击，唯美与杀机交织，武器瞬间展现出难以察觉的致命锋芒。  
在战斗之外，少女们的武器则可化为她们的饰品或装饰隐藏于周身，如同若隐若现的幻影。武器可以幻化成随身饰品，亦或是化作优雅的纹路轻柔地浮现在肌肤表面，点缀着她们的四肢、肩膀或背部，如同是自然形成的纹路一般柔和（因为是暂时视觉效果，所以并不会影响使用者考公~）。  
  
  
【幻化之饰-伪】  
1000  
在非战斗状态下，使用者可以将其武器根据其喜好，化作诸如手镯戒指等装饰品点缀在身上，也可以化为优雅的花纹，如花瓣装饰或者星光纹路，像轻抚肌肤的丝绸般悄然浮现在使用者的身体表面，或者附着在其衣物上形成瑰丽的图案。  
在任何能够正常拔出武器的时刻，使用者可以使用移动动作，让武器自周身轻盈现身于手中，如同从异空间中将武器取出。只是从视觉上看，其仿佛幻若星光、蝶翼或花瓣等元素围绕在使用者的周身，随她们的动作翩翩起舞，仅有在使用者需要武器的一瞬间，周身的元素才会紧随其动作凝结成武器的幻影，之后将再度随光辉消散消失于风中。 而当使用者将武器用作招架或格挡，视觉表现上则为这些元素短暂凝结成护盾，如花瓣飞舞形成屏障或星光闪烁出无形守护，并紧接在被攻击后散落于空中。  
  
  
D级 D+500  
掌握这项技艺的少女们，仿佛行走在梦与现实的交界，武器在她们手中不再只是冰冷的钢铁，而是随心情和意念舞动的美丽幻象。无论是淡淡的花瓣飘舞、星光闪耀，还是轻盈的蝶翼环绕，她们的武器都隐匿在这些流光溢彩的奇景中，静待致命一击的时刻。  
  
【幻化之饰】  
通过能量，使用者可以将幻化的武器在瞬息间拔出。消耗2点任意能量，你拔出武器的动作变为正常动作。伴随着绚丽的视觉效果，武器从花瓣或星光中瞬间凝聚显形。  
  
【幻光指引】  
当“幻梦凝刃”的使用者陷入黑暗或无法看清前方的困境时，她们可以将萦绕在身周的梦幻元素释放出去，化作一道照亮前路的光芒。这些如流光般的花瓣或蝶翼，在空中翩然舞动，逐渐凝聚成一束温柔却明亮的光，照亮她们前方的路径。  
消耗2点能量，使用者可以将将幻梦凝刃的元素释放出来，化作一道蝶影或星光飞向前方，照亮周围场景内的环境。这道光芒可以将当前攻击发出位置周围5米内的亮度提升1级，持续时间为一个场景。之后其会化作流光，消散于风中。特殊的，非战斗状态下使用者可以一次呼唤更多的流光飞翔于天幕，虽然这不会提升整体的场景亮度与持续时间，但又有谁会拒绝漫天的流光闪烁。  
  
【幻刺贯穿/延展】  
随着修习，幻梦凝刃带来的不仅是一种藏匿武器的效果，亦是一种将周围的流光化作光流，贯穿敌人光流的独特技艺。  
你在习得本技艺时从下方选择一项：  
  
幻刺贯穿：使用幻梦凝刃取出的武器将获得【贯穿】效果。武器在攻击时，周围的元素幻化出一道如梦似幻的光流，蝶影与花瓣向着其身后从敌人身体穿行而过。  
  
幻刺延展：使用幻梦凝刃取出的武器触及范围提升1米，并且攻击时可以选择2个额外的范围内目标。伴随攻击动作，武器周围的幻象元素迅速扩散，花瓣和星光如同波纹般在空气中荡漾，蝴蝶的虚影翩翩飞舞，幻影瞬间覆盖更大的空间，武器以虚影为延展，使敌人难以招架。
