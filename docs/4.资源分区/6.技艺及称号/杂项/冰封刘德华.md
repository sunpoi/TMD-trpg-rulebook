# 冰封刘德华

名称：冰封刘德华  
价格：C+1000  
本质：特异本质  
描述：  
新春将至，那个男人也即将解封……  
在你使用该技艺时，你身上的衣物配色将自动替换为喜庆的红色与金色，并且各种装饰也会自动变成金元宝、鲤鱼、鞭炮、鲜花、“福”字等形状，  
你的外形变为一位体积为5的帅气亚洲成年男性，并且在效果持续期间内一直保持着开心的笑容。  
效果：  
以一个整轮动作使用，持续一个场景，你获得如下效果：  
对自身造成使你进入【冰封】状态的不可豁免的【冻结】不良点数，并获得等量的盔甲防御，最多以此法获得5点防御。  
以此方法获得的【冻结】不良点数在脱离战斗后仍然存在  
并且在该场景中，以此方法获得的【冻结】不良点数减少时，以此方法获得的盔甲防御也等量减少

当你的【冻结】不良点数减少，使你退出【冰封】状态后，你会自动唱起《恭喜发财》这首歌，并获得以下效果：

①“我恭喜你发财~我恭喜你精彩~最好的请过来~不好的请走开~哦，礼多人不怪~”

每轮一次，你可以唱着“最好的请过来~”，以一个自由动作发起【拖拽】  
每轮一次，你可以唱着“不好的请走开~”，以一个自由动作发起【冲撞】

②“我祝满天下的女孩，嫁一个好男孩，两小口永远在一块~”

购买时选定一项攻击检定方式，选定后不可更改，当你以这种攻击方式命中目标后，对方额外以意志豁免对抗你的攻击出目  
若你胜出，则对方会爱上你，对方可以接受你提出的，对方日常不会做，但是不挑战其基本价值观的事  
这是一个C级命运来源的【思维控制】效果

③“我祝满天下的小孩，聪明胜过秀才，智商充满你脑袋~”

每轮一次，你使用【智力-学识-专业：解读】进行检定时，可以再骰一次，你必须接受重骰的结果

④“我祝尊敬的姑奶奶，三十六圈的比赛，气不喘面容不改~”

你的移速提升30m，免疫【疲乏】不良点数带来的减值及严重状态

⑤“我祝三叔公的买卖，生意扬名四海，财运亨通住豪宅~”

在该技艺持续时间内，视作你具有C级命运来源的【背景身份】效果，  
你获得一个能被查到过往经历的身份，并附带记忆，但是该身份不会带来任何地位上的好处，不附带人际关系，仅附带足够生存的金钱  
以此方法获得的金钱在效果结束时会自动消散，所以请把握好该效果的使用方法
