# 我会永远看着你

【我会永远看着你】  
C+1000  
购买时绑定一种你已经拥有的攻击检定方式，选定后不可更改。  
当你以绑定的攻击方式命中一个目标时，你会在对方身上留下一个标记，当目标处于你的敏感范围内时，视作你拥有针对目标的【真实视觉】  
你同时只能维持对一个目标的【真实视觉】，如果你同时命中了多个目标，由你选择其中一个施加【真实视觉】  
当你对新的目标施加【真实视觉】后，原本目标的【真实视觉】效果消失。

  
被你施加【真实视觉】的目标称为你的【凝视对象】  
每轮一次，当你受到【凝视对象】的攻击时，可以选择受到1点不可豁免的【亢奋】，获取临时的1点全伤害吸收。

  
每轮一次，当你使用绑定的攻击方式对【凝视对象】进行攻击时，可以附加1点不可豁免的【恶心】点数。  
你向【凝视对象】移动时，你的移速提升5M。
