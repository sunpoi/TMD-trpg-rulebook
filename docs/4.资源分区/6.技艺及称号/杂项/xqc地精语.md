# XQC地精语

名称：XQC地精语  
价格：500  
本质：魔幻本质  
描述：  
说真的，当主神空间第一次在一个平行时空中观测到一个申必的老鼠台澳大利亚主播时，就连无敌的主神的翻译能力都受到了一次冲击，最终，在几位轮回者的不懈努力下，终于在一个平行世界的地精种群中找到了近似的语言对照表，完成了这个堪称史的支线任务。在那之后，一个神秘的资源悄悄出现在了主神空间的资源兑换表中。

【超速地精语】：在购买该技艺后你的语言将不可避免地转变为XQC语言风格，这意味着你的语速被加速到一秒钟内可以塞入22个音节，任何可以连读并缩略的句子都会被尽可能地缩写，而且在缩写后会产生不可避免的多次结巴（脏话除外）。

特性：【超速地精语】  
效果：  
【超速地精语】：你立即获得XQC语1级，并且你和任何未掌握XQC地精语的人进行社交检定都会承受4dp表现减值。 作为补偿，你在施法检定上获得1dp表现加值。
