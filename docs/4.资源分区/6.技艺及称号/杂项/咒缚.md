# 咒缚

名称：咒缚  
价格：500  
本质：特异本质  
描述：  
有的时候我们需要点比毒誓更毒的东西。  
效果：  
以自由动作支付1点意志力，角色立下某种承诺，这种承诺会立即变成仅能对角色自身生效的咒缚，并且在角色违反承诺时立即产生反噬，使角色立即死亡，这不被视为一个即死效果，因此即死免疫对此无效。  
咒缚本身是一个无法被常规能力解除的效果，将永久缠绕在角色身上，无论角色之后是否失去了本能力，即便复活或更换身体等也无法逃脱咒缚的约束，只有超越世界之上的存在（对就是说主神）可以将其免费解除。  
当角色如此做的时候，能够感知到角色行动的任何角色都能够立即理解咒缚的全部效果和内容，无视语言和感知方式等的隔阂，并意识到这些内容全部真实无虚。  
举例：轮回者安南以自由动作施展本技艺，立下“我今后从此不再饮酒”的承诺，随后承诺将成为安南的咒缚，当安南尝试喝酒时，咒缚会立即反噬，使安南在喝到酒之前前死去；即便出于非自愿的情况下触犯了咒缚，如在不知情时饮用了酒精饮料，也会立即死亡。
