# 好硬的嘴

名称：好硬的嘴  
价格：750  
本质：特异本质  
描述：  
“我们这块呢是管肾的，要是按上去疼就说明你可能肾比较虚”  
“嘶……一点不疼，师父你是不是没使劲啊”

在启动该技艺后，角色会强行将自身的表情和语气维持在放松状态下，即便承受不良点数和伤势，或者面对心中伤痛的情景，也可以在表情和行动上表现得若无其事，在外人看来如同三无少女或铁血硬汉一般毫无变化。  
效果：  
以一个自由动作启动并维持此技艺后，角色免疫C级针对伤情的获知效果，针对角色的诊疗检定自动失败，在对抗察言观色检定时也可以获得1点额外的附加成功。另一方面，启动此技艺时，角色在唬骗以外的社交相关判定上承担2DP表现减值。
