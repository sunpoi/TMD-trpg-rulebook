# 童年游戏节选

名称：童年游戏节选  
本质：特异本质  
描述：  
仅致以美好的童年。  
在那个电子设施并不先进的年代，娱乐并不匮乏。

名称：跳房子  
价格：250  
描述：  
玩家需单脚或双脚交替跳跃通过数字标记的格子，同时踢动沙包或石块，过程中不得踩线或双脚落地，最终以完成所有格子或占据最多“房子”为胜。  
效果：  
你在【运动】下开启专业【跳房子】，并提升为1级

名称：跳房子  
价格：D+500  
描述：  
房子当然是【房子】，跳房子当然也可以是【跳房子】  
效果：  
你现在可以使用【跳房子】专业来代替【跳跃】专业来进行检定。  
你现在可以在垂直平面上进行移动，但若回合结束时没有落脚点会正常坠落。

名称：滚铁圈  
价格：1000  
描述：  
在使用此技艺的时候你会自动获得一个前段为U形的铁丝和一个圆铁环。  
效果：  
使用者进行一个持续十分钟的加速准备行为，在这期间不可中断准备行为，在准备完成后即可以旅行速度进行旅行。进行旅行时你需要以铁丝来推着铁环行动，同时在旅行前进行一个【平衡】检定，你可以在一小时内旅行成功数×5公里的距离，上限为200公里每小时，这是一个D级旅行效果。

名称：丢沙包  
价格：750  
效果：  
你现在获得了一个武器伤害为0B的沙包，现在你可以花费标准动作以投掷攻击检定丢出沙包来进行一次命中检定，这次不造成伤害，若是命中，则对方获得1点别的可豁免的沮丧点数，这是无支线心灵来源。沙包可以在使用此技艺时取出。
