# 调休

调休 1000/4XP  
描述：“所以我说调休这个词有玄机在里面的，休了吗？调休。它真休了吗？嗳，调休~"  
效果：角色可以在24小时内进行两次长休息，并获取相应收益，这将允许角色突破每天仅能进行一次长休息的限制。但相对应的，下一个24小时将无法进行长休息，也需要因此承受未进行长休息的劣势。  
特殊：若角色背景为21世纪后的华夏民族者，则此能力可视为一个2级背景专长，在建卡时支付专长点获取。
