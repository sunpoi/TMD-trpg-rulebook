# 夏季活动限定皮肤！

夏季活动限定皮肤

技艺/称号

1250分

夏天正是在海边穿泳装的时间！虽然现在可能不是夏天，但是众所周知，一年四季都可以有夏季活动。

在购买了这个技艺后，只要你穿着了任何服装或护甲，都可以使用一个手势把这些服装变形成泳装的样子，这是一个D级变形来源的变形效果。主神提供多种泳装的款式，包括分体式、连体式等等。如果你不满意的话，可以自行在空间更改泳装的样式。

这种变形只是视觉上的效果，不会提供任何其他功能，所以你最好不要真的去游泳。不过原服装的功能仍然会被保留。

周围人不会因为你在任何地方穿着泳装而感到奇怪。这是一个无支线心灵控制效果。当然你能够以一个想法让这个效果失效，这样周围的人就会因为你穿着泳装而感到奇怪了。
