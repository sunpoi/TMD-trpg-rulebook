# 转生卡车！！

名称：转生卡车！！  
价格：B+2000  
本质：魔幻本质  
描述：  
介绍：有无数个异世界面临着灭亡的命运，这时候就需要有着特殊能力的勇者去拯救他们。  
女神们发出召唤，让那些死于卡车的灵魂转移到异世界获得新生……只是对于轮回者来说，这些异世界可能没那么美好。

效果：  
购买本能力后，你可以选择以下两种之一作为表现获得。  
1.你进入新的影片并不是以主神传送的方式进入。因为在传送前你的天命会先召唤一辆疾驰的卡车先一步将你撞死，使你的灵魂前往异世界转生。  
2.你在非战斗场景下，可以随时选择召唤一辆疾驰的卡车将你撞死，使你的灵魂前往异世界转生。

如果你选择第一种，那么你会在死后，本应进入的下一个影片内复活，成长；并在队友到达这个影片时，再次成为加入队伍成为轮回者。  
你会因此在影片中获得能解释自身能力来源的合法身份，拥有实际经历，对应记忆，以及普通人的人际关系，自身应有的财产（包括但不限于：一套装修过的租房、一辆需要还贷款的车）。

如果你选择第二种，那么主神的大手会在你完成转生前强行把你抓回本影片，让你以被卡车撞死前一刻的状态复活，使你无法从这个世界脱离，不过你依旧可以获得下列提及调整兑换的好处。  
作为补偿，你可以额外在下列提及计算支线与分数（或经验）时额外获得等价C+1000的资源量，只能用于兑换或合成兑换技艺类资源，因此获得的资源不会因为多次转生叠加，但可以被转生效果所调整兑换内容。

不管是哪一种，由于转生的原因你可以选择将自身两项内在兑换能力，这两项能力与相关内容（如血统，改造，或瞳术，单一称号，以及对应主干下属相关专长、技艺等），将全额转换为等量的支线与分数（或是经验），用于在转生时向女神许愿获得能力，就好像是你在主神空间进行兑换一样。

你不能因此调整自身的属性分配与技能等级。  
你可以进行一次自身相貌/身高/体重的调整，同时重新以当前风度属性决定一次基础外貌的数值。

注1：购买这个能力不会获得自由属性。  
注2：转生卡车的能力一影片只能发动一次。  
注3：ST有权要求你事先写好修改资源后的角色卡，以及要求你只能为自己准备一张修改资源后的角色卡备用。

变体：  
价格：A+5000  
效果：  
购买这个能力不会获得自由属性。  
购买这个变体，则可以召唤出更高级的转生卡车，以这种方式召唤出的转生卡车可以在战斗中出现，在你受到致死伤害时抢先一步把你撞死，让你的灵魂前往异世界进行转生。  
此时依旧遵循非变体转生卡车限制与条件，将可转换的内在兑换从【2】降低至【1】项，只能选择第二种转生方式，而且无法获得额外支线与分数作的补偿。

作为代替，你可以立刻将负责转生的女神击倒，从异世界杀回这个场景，但显然该影片内无法找到第二个愿意负责给你转生的女神了，所以变体在一影片内只能发动一次，这是A级命运来源的复活能力，会使你满状态复活而不是以被撞死前一刻的状态复活。
