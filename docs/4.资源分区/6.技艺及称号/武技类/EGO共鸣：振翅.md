# EGO共鸣：振翅

EGO共鸣：振翅  
D+1000

他的内脏和眼球遭到不明生物啃食，受损严重。此外，我们察觉到精灵们的嘴角沾有某人的血肉。

以一个整轮动作启用，你选定一名可怜的受害者，对其进行一次冲锋攻击，本次攻击的伤害类型转为挥砍伤害，并且这次冲锋攻击中你的基础防御不会降低，在你第一次斩击切下其血肉之后，你恢复1点完好或韧性，这是一个【快速医疗1（医疗点12）】效果。随后，你调转方向，对其进行重复的斩击，重复攻击的次数为投掷4次rd2，若出目为1则视作成功重复攻击一次，每次重复攻击仅会造成额外的1点多次攻击减值而不视作一次完整的攻击，本次攻击的实际伤害会平分至首次攻击和每次重复攻击之中。（如轮回者A投掷出4个1，那么本次攻击造成5点多次攻击减值，本次攻击造成实际伤害为10点，那么则变为受到2点伤害5次）

——解析II：额外支付D+1000，开启贪食被动：  
当你使用振翅后，在本场景内，当你对某位敌人造成实际伤害时，若这不是其第一次受到的实际伤害，那么每造成3点实际伤害，则为你恢复1点完好值或韧性值，每回合最多恢复3点，这是一个【吸血3】效果。  
——解析III：额外支付C+1500，  
现在在你使用振翅时，你每成功进行一次重复攻击，那么你本次攻击的实际伤害倍率提升0.4，但是你本次攻击的基础伤害倍率降低至0.4倍。（即当你成功重复攻击的次数达到1/2/3/4时，实际伤害倍率提升至0.8/1.2/1.6/2.0倍）  
并且你每成功进行一次重复攻击，那么你获得1点仅在本次攻击中生效的吸血。  
——解析IV：额外支付B+2000，  
在解析III中获得的效果改为：你每成功进行一次重复攻击，那么你本次攻击的实际伤害倍率提升0.6，但是你本次攻击的基础伤害倍率降低至0.6倍。（即当你成功重复攻击的次数达到1/2/3/4时，实际伤害倍率提升至1.2/1.8/2.4/3.0倍）  
并且你每成功进行一次重复攻击，那么你获得2点仅在本次攻击中生效的吸血。  
此时振翅会附带等量的流血异常点数
