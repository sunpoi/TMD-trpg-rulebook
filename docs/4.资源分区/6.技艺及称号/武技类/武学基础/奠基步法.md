# 奠基步法

奠基步法  
资源类型：武技技艺  
本质：特异本质  
购买前提：武学基础-步法入门

价格：D+500  
描述：  
厮杀争锋、奔逃亡命，如遇争斗，最【速】者便处于不败之地，籍于此，步法一脉便逐步发展，也有了超脱于其的内涵，而只论技法，无非【走】与【跃】耳，此法奠基最为容易，无需钻研费神，只需勤练便可有成，恒持之，或得逍遥。

特性：【\*步法修行】【\*小纵跃功】  
效果：  
兑换后可以修心步法精要，将开始：  
【\*步法修行】：  
每当修习者经历一个影片，其对步法的掌握便更进一步，愈发熟练的动作将使其增加五米的移速。

此外，修习者将得到一种基础的纵跃赶路之法，即：  
【\*小纵跃功】：  
使用时将消耗一点内力，用移动动作进行移动，本次移动若进行跳跃，如同已助跑一样进行处理，自由控制跳跃的落点和距离而无需检定，但每次跳跃的高度不能超过移动距离的1/2；回合结束时仍然需要站在稳固的落脚点上，否则就会坠落。

表现上视为运气内力连续纵跃数次到达目的地，同时在下蹲/伏地攀爬时也可正常移动，不受到此两者的移动减益效果影响。
