# 基础武学-指法

  

基础武学-指法  
江湖中最最基础的武功招法，在结合不同的基本功和兵器招式之后，大囧朝廷通过总结推演出新，创新性的为百姓和天下所有武学新人创造了这些更接近于实战的基础武功，并划分江南漠北地区二十八县推广推陈，力求普及，为未来扫平江湖提前埋下另一道伏笔。

本系列下所有能力皆需要内力能量池才可购买。

基础指法 D+1250  
练法是，将一块厚约寸余的青石板搁在板凳上，高及人的腹部。练功者将指甲剪净，距石板半尺成马步，两手屈于胸前，自然下垂，以两手手指在石板上缓慢来回抚摩。用意不用力自然和缓。半小时以后，可用十指在石板各处点打200次。击打时要用腕力，要有弹性，点打分明，点后即起。用力不可太猛，然而意念须击石而穿。练久后，指尖会慢慢磨平，十指发烫，偶尔还会发硬。直到五指在木板、石板、青砖上能击打出凹印时，指力已十分锋利，就大功告成了。  
完成训练之后，你的肉搏天生武器获得3L武器伤害。  
指力已成的武者不在光凭一时血勇而战，此时你的双手经脉已通气走其中，激战之时运使内力流转，指力随内力散射，凝而不散力贯其中，这就是指法武功基本功已成的气势。  
花费标准动作进行天生武器攻击，消耗1内力，你的可攻击范围额外增加3米，并且具有5威猛。

这是一个攻击技。
