# 五毒

技艺  
需要关键属性含有生理属性的能量池  
五毒-壁虎功  
D+500  
出自《五毒》，四师兄所学之功法，壁虎功身轻矫健，随处皆可攀援，  
习得者可在垂直平面上进行移动，但回合结束时没有落脚点会正常坠落。  
同时壁虎功攀墙伏壁时可以借劲力反弹，以增加本身的劲力。  
在垂直平面上时以一个自由动作发动，你借力增强自己的力量，你免疫坠落伤害持续1轮，额外消耗2点能量获得，你的力量属性临时+2，临时属性不能超过当前等级上限。

技艺  
需要关键属性含有生理属性的能量池  
五毒-蜈蚣功  
D+500  
出自《五毒》，大师兄所学之功法，习得蜈蚣功后出招犹如百足千手，出招极为迅速，你以拳，腿等部位发动的肉搏攻击具有高速5  
以一个自由动作发动，你消耗2点能量，再次强化你的出招速度，你以拳，腿等部位发动的肉搏攻击具有【连射】，如果你不具有天生武器伤害，则你临时具有3l天生武器伤害，持续1轮。

技艺  
需要关键属性含有生理属性的能量池  
五毒-蛤蟆功  
D+500  
出自《五毒》，五师兄所学之功法，习得蛤蟆功后，可以刀枪不入。  
你获得基于格挡防御的近战伤害吸收2点  
以一个自由动作发动，你消耗2点能量，运足自身真气，使全身肌肤如铁，具有断木折铁之功，你获得2点格挡防御，持续1轮。

技艺  
需要关键属性含有生理属性的能量池  
五毒-蝎子功  
D+500  
出自《五毒》，三师兄所学之功法，手如蝎钳腿似钩，  
无需通过检定也可进行攀爬移动，以腿发起的肉搏攻击具有【贯穿】  
以一个自由动作发动，你消耗2点能量进行准备，在进行肉搏攻击时将自己的蝎钩腿攻向另一个敌人，你的下次攻击额外在你的肉搏攻击范围内新增一个其他攻击目标，攻击具有破甲1，持续1轮。

技艺  
需要关键属性含有生理属性的能量池  
五毒-灵蛇功  
D+500  
出自《五毒》，二师兄所学之功法，右为蛇头，左为蛇尾，长于游走，胜在灵。  
伏地时可全速移动，取消潜行行走速度限制  
以一个自由动作发动，你消耗2点能量，你的肉搏攻击改为造成穿刺伤害，并且这次攻击中，你的天生武器伤害提升2点，持续1轮。
