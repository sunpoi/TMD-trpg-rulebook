# 燃木刀法

燃木刀法  
资源类型：武技技艺  
本质：特异本质  
购买前提：武学基础-战技入门

价格：C+1000  
描述：  
少林七十二绝技之一，练成后在一根干木旁快劈九九八十一刀，刀刃不损木材丝毫，刀上发出的热力却可将木材点燃生火，故得此名。

效果：  
习得此刀法，可在使用任意刀法进行攻击时消耗两点内力，在画面表现上可以瞬息间连劈敌人九九八十一刀，不过最开始只能劈出两刀使本次攻击获得两点技艺加值以及一点高速。

此后，基于武学总纲，当此武学精熟程度达到以下程度时，将额外获得：  
【\*初窥门径】：  
五点高速，两点技艺加值。在画面表现上 你现在可以连劈九刀。

【\*渐入佳境】：  
本次攻击升级为【连射】，增加一点附加成功和两点技艺加值，本次攻击中你的力量临时上升1点。在画面表现上，你现在可以连劈二十七刀。

【\*开宗立派】：本次攻击附带一点不可豁免的燃烧不良点数，可叠加，并且可以攻击接触距离内的两个目标。在画面表现上，你现在可以连劈四十五刀。

【\*武道传奇】：  
本次攻击再增加五点高速以及一点技艺加值。在画面表现上，你神功大成，可以连劈九九八十一刀。
