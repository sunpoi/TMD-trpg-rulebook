# 基础武学-枪法

  

基础武学-枪法  
江湖中最最基础的武功招法，在结合不同的基本功和兵器招式之后，大囧朝廷通过总结推演出新，创新性的为百姓和天下所有武学新人创造了这些更接近于实战的基础武功，并划分江南漠北地区二十八县推广推陈，力求普及，为未来扫平江湖提前埋下另一道伏笔。

本系列下所有能力皆需要内力能量池才可购买。

基础枪法 C+1500  
年刀月棍一辈子枪，长枪是所有兵器之中入门最难的一项，却也是最适合战场杀伐的技艺，考虑到了大囧军中武艺尚枪，本阶段功法经过额外的修正，最终推崇出以“长枪之法，始於杨氏，谓之曰梨花，天下咸尚之；其妙在於熟之而已，熟则心能忘手，手能忘枪；圆精用不滞，又莫贵於静也，静而心不妄动，而处之裕如，变幻莫测，神化无穷“口诀传下习练，并且严格限制民间唯有民兵武丁可以操练此法，更多用于军中兵士习练战场穿插机动之用。  
花费一个标准动作和2内力进行长枪攻击，你的这次攻击视为一种使用长枪的横扫攻击，这次攻击可以攻击触及范围内所有敌人，进行反射豁免并具有9加骰。

这是一个攻击技。
