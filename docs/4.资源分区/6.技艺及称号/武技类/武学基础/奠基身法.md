# 奠基身法

奠基身法  
D+500  
前置：武学基础-身法入门  
所谓身法，便是方寸挪移之法，与步法唯速迥异，身法以变为妙，讲究生死一步隔，故更为险要，此法奠基乃好斗者之重，唯重练此法，方能先制克敌、出奇制胜。

  
兑换后可以修心身法精要，将开始：  
\*身法修行：每当修习者经历一个影片，其对身法的熟练便更进一步，愈发难以捉摸的行动轨迹将为其提供两点的先攻加值。

  
此外，修习者在经历三场影片的修行后，将习得一种基础的挪移之法，即：  
\*方寸步：这是一种类招架能力，使用时将额外消耗一点内力，然后进行一次纯运动技能检定，本次检定无法获得任何其他能力带来的额外加值。  
使用者会在招架的基础上额外减免等同于成功数的伤害，表现上视为修行者在瞬息间闪出原本的方位，籍以此来减少收到的伤害。
