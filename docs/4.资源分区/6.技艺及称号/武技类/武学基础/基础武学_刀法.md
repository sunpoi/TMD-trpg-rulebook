# 基础武学-刀法

基础武学-刀法  
资源类型：武技技艺  
本质：特异本质  
购买前提：具有内力池

概述：  
江湖中最最基础的武功招法，在结合不同的基本功和兵器招式之后，大囧朝廷通过总结推演出新，创新性的为百姓和天下所有武学新人创造了这些更接近于实战的基础武功，并划分江南漠北地区二十八县推广推陈，力求普及，为未来扫平江湖提前埋下另一道伏笔。

技艺名称：基础刀法  
价格：D+1000  
描述：  
为刀者霸道刚猛，其技法讲究：  
一者、“刀招沉猛”，与剑相比，刀法大开大阖，变化较少而威力不减。  
二者、“单刀看手，双刀看走”，在刀法中持刀手及步法极为重要，其中又以扫、劈、拨、削、掠、奈、斩、突八式为刀法精要习练者会当领却刀法之猛烈霸力，意为绵绵之力层层叠叠如浪激流，而非一时血勇之力。

效果：  
花费一个标准动作和1内力进行攻击，你的本次刀法攻击获得【连射】特性，并且你在使用本刀法攻击的时候，可以同时使用移动动作移动，使用本技能时，你的刀术攻击额外具有D级措手不及  
这是一个攻击技。
