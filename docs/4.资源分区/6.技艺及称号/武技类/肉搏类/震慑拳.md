# 震慑拳

技艺名称：震慑拳  
价格：B+2000  
本质：特异本质  
能耗：10点东方能量  
动作：标准动作  
描述：  
通过阻碍敌人体内能量的流转，达到干扰敌人的攻击

效果：  
你以一个标准动作发动一次肉搏攻击，本次攻击检定出目的成功数增加一半。若被命中的目标体内具有能量，则需要进行一次强韧豁免，dc等同于本次攻击伤害值，失败则进入B级创伤来源的震慑状态一轮。
