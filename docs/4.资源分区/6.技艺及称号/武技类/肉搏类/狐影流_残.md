# 狐影流-残

狐影流  
技艺  
狐影流是没有历史的，非要说的话，狐影流也属于古代柔术一种，是传承400多年的古代柔术，但由于先前的大战使得门下弟子几乎死绝导致其技术已经失传。现在狐影流的招式是有继承宗家之血平良山将以前的文献收集起来，重新编写的招式。虽然可以使用的招式屈指可数，但平良山依然将仅有的两招教给了自己的弟子桐生刹那，希望狐影流在这个武学天才手中发扬光大。

狐影流.瞬  
分类：步法  
C+1250  
狐影流.瞬 -是瞬间从敌人的视角范围内消失，然后出现在对方侧面或后面对敌人进行出其不意打击的步法。其原理是利用对手眨眼瞬间，快速移动到视觉死角，从而给予敌人消失的假象。  
狐影流.瞬-使用自由动作维持。对于与你交战的敌人而言，你的移动看起来就像瞬间消失了一般，但这只是敌人眼中的假象，你实际上只是利用视觉死角快速移动，并不是空间传送，所以不能获得空间传送带来的任何的好处，你可以在维持期间额外获得3点闪避防御，移动速度增加15米。同时对于无法看破狐影流.瞬的敌人，维持狐影流.瞬-还可以使针对敌人的攻击获得C级的措手不及效果。  
（st选开：对于360度视野的生物，或者眼豆之类无法眨眼的生物无效）

狐影流.罗刹掌  
A+4000  
所谓的罗刹掌是将通过背-肩-上臂所传达的力量通过前臂的回旋增幅，将其转换为强烈的扭转之力打击,在击中对方后，扭转之力将穿透皮肤扭曲对方被击中部位肌肉以及内脏。  
罗刹掌虽然称之为掌，但其实也可以使用腿脚，甚至是指尖来施展，对于拥有特殊构造的生物（比如脖子可以旋转）也可以使用头或者其他部位来施展罗刹掌。  
你控制肌肉扭转，扭转之力将从背-肩-上臂一直回旋增幅到你的手掌，  
你消耗一个标准动作进行一次肉搏攻击，你的攻击携带的扭转之力内劲将穿透铠甲与皮肤直接对内脏造成伤害，无视对方的盔甲与天生防御，这是一个A级别、创伤来源的负面效果，同时扭转的攻击还将破坏对方的格挡架势，这次攻击额外具有【威猛25】。  
使用罗刹掌时，你造成的伤害转为韧性伤害。
