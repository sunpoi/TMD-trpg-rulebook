# 蛇形刁手

技艺名称：蛇形刁手  
本质：特异本质  
价格：D+500  
使用限制：需佩戴技艺赠送的绿色蛇形长筒袜  
描述：  
本质是戴上蛇袜进行自我暗示和催眠，使自己认为蛇形刁手已经成为了一条有自己思想，攻击性极强并带有致命毒液的眼镜蛇。

效果：  
购买该技艺时会得到一条滑稽的绿色蛇形长筒袜，只有戴上这条蛇袜才能发挥蛇形刁手的完全力量。

在施展蛇形刁手时，你需要以一个迅捷动作为右手戴上蛇袜并以一连串的动作拉动右臂进行自我催眠，之后以一个标准动作使用蛇形刁手进行一次肉搏攻击，这是一次针对头部的特殊部位攻击，本次攻击获得2点武器伤害，并且减少3点特殊部位减值，获得3DP的技艺加值，并且受到伤害的目标需要进行一个DC=伤害数的反射豁免，失败则被立即击倒。  
当你免疫催眠，暗示等效果时，攻击变为一次普通的针对头部的特殊部位攻击。  
并且，你的右臂会被任何针对蛇的效果影响，包括但不限于蛇癖，印度耍蛇人的笛子……并且你无法解除自我暗示，直到攻击结束。
