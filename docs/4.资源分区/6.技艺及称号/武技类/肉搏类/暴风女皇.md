# 暴风女皇

名称：柔道家-技艺  
类别：技艺组  
本质：特异本质

描述：  
她们是通过锻炼拳、脚、肩、膝等身体的一部分，主要以身体进行战斗的战士。因身体娇小，经常被人们误认为成弱小的女性。她们会迅速靠近进行攻击，使对方窒息或筋骨断裂，特别是在与她们贴近时，她们会有接近必杀的攻击力 。强大的腰部力量与女性特有的韧性，利用投技可瞬间给对方带来致命伤害。  
柔道家精通多种翻摔投掷技能，可瞬间对敌人造成致命伤害。掌握的“螺旋彗星落”、“背摔”等技能，可在任何时候，将对手轻易翻摔坠落。

技艺名称：柔道家  
价格：D+500  
描述：  
柔道家擅长使用抓取技能，她们可以使用手臂、腿脚、关节等来进行擒抱

特性：【善于擒抱】【抓轰炮】  
效果：  
【善于擒抱】：  
在使用抓取技能、擒抱时，可以视为天生武器的武器伤害为3L。

【抓轰炮】：  
你获得1个名为【抓轰炮】的攻击技，效果具体如下

技艺名称：抓轰炮  
效果：  
以标准动作进行一次擒抱检定，如果检定成功，可以选择不进行擒抱，而是以力量+武技，专业摔技进行攻击，并在命中目标后将目标击退成功数米，目标以反射进行豁免。抓轰炮在施展时，可以选择不进行击退，而是将目标向上进行D级击飞。

技艺名称：柔道家  
价格：C+1000  
特性：【暴力抓取】【强制-抓轰炮】  
效果：  
【暴力抓取】：  
你可以在一次攻击中抓取多个目标，在进行擒抱检定时，可以对接触范围内的所有目标进行擒抱（依旧需要遵循擒抱规则），如果使用抓轰炮，则可以选择将这些目标全部朝向一个方向进行击退。

【强制-抓轰炮】：  
你获得1个名为【强制-抓轰炮】的应对技，其具体效果如下

技艺名称：强制-抓轰炮  
效果：  
在自己的回合外，如果有目标进入了攻击范围内，可以调用下个回合的标准动作，对该目标以反射动作施展一次【抓轰炮】，但是此次的【抓轰炮】不会造成伤害，只会击退目标。

技艺名称：暴风眼  
价格：B+2000  
特性：【末日风暴】【滑行抓取】  
效果：  
【末日风暴】：  
在施展【抓轰炮】时，可以使用特殊的抛投技法，将目标抛去空中，目标以反射进行对抗，如果失败目标则会陷入B级的击飞状态，这是一个B级环境来源的效果。这会替代D级的击飞效果，并且目标的反射豁免降低2。

【滑行抓取】：  
你获得1个名为【滑行抓取】的攻击技，其具体效果如下

名称：滑行抓取  
效果：  
朝向一个方向进行一次冲锋，并将沿途攻击范围内的所有目标进行击退，这些目标最远可以被击退15米/冲锋距离取低的距离，并在距离达到最远后发动【抓轰炮】。

技艺名称：暴风眼  
价格：A+4000  
特性：【地狱摇篮】【彗星螺旋落】  
效果：  
【地狱摇篮】：  
在施展一次【抓轰炮】并造成伤害后，可以将目标击退向特定位置，如果在途中击中其他目标、障碍物，则被投出的目标还会再受到一次伤害。

【彗星螺旋落】：  
选择一个被击飞的目标，跳向该目标并夹住目标令其提前结束击飞状态，并立即坠落地面，除了造成坠落伤害之外还会造成一次额外伤害，但是该能力不能作为位移、攻击飞行目标的能力进行使用。

技艺名称：暴风女皇  
价格：S+8000  
描述：  
普通人所谓的战斗不过是打架而已，都只是街头混混的把戏罢了。  
真正的战斗，是对武道最终极的诠释。猛烈、迅捷、华丽！一旦抓住机会，就会对敌人进行狂风骤雨般地攻击，绝不会给对方任何喘息的机会，任何的失误都将让你陷入万劫不复的境地。  
只有这种热血贲张的战斗，才是这大陆上最完美的艺术，这种绝妙的美感可不是那些肤浅的人能够体会得到的。  
战斗的极致就是对美的最终追求，这就是我的武道。  
而能达到这种境界的人，就只有我——暴风女皇！

【苍宇彗星落】：  
使用拳头击打地面，并产生强大的冲击波击飞所有人！之后将所有人贯入地面，这巨大的冲击可能会导致他们晕眩

特性：【苍宇彗星落】【极手奥义】  
效果：  
【苍宇彗星落】：  
将敏感范围内的所有目标进行击飞，目标以强韧进行对抗，如果失败，则目标会眩晕，这是一个S级创伤来源的效果。

【极手奥义】：  
在施展【强制-抓轰炮】时，会正常造成伤害，并且每经过一场恐怖片，就可以令使用擒抱、【抓轰炮】时的攻击范围提升50米。
