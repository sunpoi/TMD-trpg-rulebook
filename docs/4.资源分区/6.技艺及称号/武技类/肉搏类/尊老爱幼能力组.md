# 尊老爱幼能力组

  

“尊老爱幼”D+500  
购买这个能力后，其他人在看到你后会下意识认为你是个尊老爱幼的优质人才，这是一个D级思维控制的效果。你的容颜永驻，永远保持在购买该能力的一瞬间，且寿命不会因为时间自然流逝。  
想当年我拳打南山敬老院，脚踢北海幼儿园。一米以下七十以上完全不放眼里，双手插兜，不知道何为对手。  
尊老敬老只能对年龄大于70的老人使用（老人的判定方式为剩余寿命少于上限的一半）  
尊老拳  
C+1000  
人生最美妙的结束自然是毫无病痛的老去。秉承着这个想法，某位尊重老人不愿透露姓名的轮回者自创了这套拳法，目的是送遇到的老人一个安详的结局。  
标准动作对触及范围内的一个目标进行一次肉搏攻击，命中时不会造成伤害，而是会造成等同于伤害数的疲乏点数，正常豁免。目标立刻衰老10年，至多为寿命上限的四分之一，持续一个场景。若目标因此攻击失去所有剩余寿命，则会立即安详去世。可以进行一次不受任何效果影响的机运骰，若出数在8以上，st可以根据目标生前的事迹掉落一些标志性物品。比如作恶多端什么都不会掉落，而多行善事的人会掉落舍利子，这也许会对剧情人物有吸引力。  
只要在影片中通过这个方式超度过老人，在影片结束后对老人使用尊老敬老时获得1dp的技艺加值。  
  
敬老掌  
C+1000  
老而不死是为贼，自古宝物有德者居之，老头，你的时代已经结束了，快爆金币！  
标准动作对触及范围内的一个目标进行一次肉搏攻击，命中后提供C级夺权效果。在夺权对抗中，你可以把超度老人获得的dp加入到夺权对抗中，并且额外获得5dp的技艺加值。

  
爱幼护幼只能对年龄小于自己的目标使用。  
爱幼脚  
B+2250  
一个潜力无限的强敌就在眼前，最好的解决问题的办法是什么？此子断不可留！  
标准动作对触及范围内的一个目标进行一次肉搏攻击，命中后目标进行强韧豁免，若失败则会幼化，体积下降差值的数值，至少降为1。持续一个场景，这视为一个A级命运来源的幼化效果。  
若你在一部影片中成功用爱幼脚幼化一个目标并将其扼杀在萌芽中，则在影片结束后被爱幼脚命中的目标强韧豁免减少2dp。  
  
护幼腿  
C+1000  
幼儿是种族延续的未来，但为了种族的未来不长歪，只能痛下狠手，切除掉长歪的枝干  
进行肉搏攻击时可以选择不造成伤害，而是造成等量的剧痛点数，正常豁免。若进入战斗轮时敌人中有年龄小于自己的目标，你可以增加两个D10到自己的先攻骰中。对付长错枝干的未来，下手必须要快准狠，孩子不打不听话。
