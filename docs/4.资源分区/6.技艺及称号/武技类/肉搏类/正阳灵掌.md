# 正阳灵掌

正阳灵掌 D+500  
前提：指定一项自己拥有的阳性能量池（如阳性内力，苏摩等），一旦选定不可更改，本技艺消耗的能量皆基于该能量池。  
描述：来自《武灵》世界的武学，以自身纯阳正气，借用大日阳气化解阴晦邪法的技艺，是纯阳正气的基础招式。  
效果：本技艺只能在阳光最为炽热的时刻（通常是正午前后半小时内），于阳光可以直接照射的环境下施展。每次施展后需要间隔24小时后才可再次施展。  
消耗5分钟时间准备，使用者对触及范围内的一个目标拍出一掌，如常进行肉搏攻击判定，但是不造成任何伤害，也不触发攻击命中后的任何效果。若该次攻击成功命中，受术者身上不超过B级的毒素、疾病、药物、附身、影响心灵来源的效果都会被这一掌直接驱散，每驱散一个，都会强制消耗规定的能量（D1C2B3）。  
这不会使得使受术者再次受到相应能力时拥有免疫。  
画面效果中，受术者的身上将蒸腾出热气，隐约能看出被驱散的阴邪之气。
