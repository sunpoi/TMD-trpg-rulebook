# 太吾绘卷-无门无派

名称：无门无派  
类别：技艺组  
本质：特异本质

————————————

名称：内功-【沛然诀】  
价格：D+500  
描述：  
沛然诀于内力收发引化之法中另辟蹊径，纳气不吐反复调练，所谓：“博观而约取，厚积而薄发”，功力愈用愈多，越是消耗，越显精强。  
效果：  
你开启了内力能量池，但并不获得【基础内功】中内力的基础用法。  
现在你即便不处于适合休息的环境，只要周围没有敌方单位存在，你便可以进行长休息/短休息。

名称：身法-【小纵跃功】  
价格：D+500  
描述：  
小纵跃功乃是以连续的纵跃快速行动的粗浅轻功，虽然修习法门十分简单，但修习过程极长，非数十年寒暑不能练就，故而功法虽简，修习者却寥寥无几。  
效果：  
你的移动速度提升15米。  
你现在可以免疫自然坠落伤害。

名称：拳法-【太祖长拳】  
价格：D+500  
动作：标准动作  
使用限制：使用专业：拳  
描述：  
太祖长拳共有三十二式，朴实无华，简单纯粹，虽然招式粗浅，却已含拳术至理，故而人称“百拳之母”。  
效果：  
这是一个混元属性的拳法。  
对敌人进行一次肉搏攻击检定，造成钝击伤害。这次攻击具有5dp的技艺加值以及1点来自于技艺的附加成功。
