# 匕首使

名称：匕首使  
类别：技艺组  
本质：魔幻本质  
购买限制①：该称号下属技艺须以对应的称号等级作为前置  
购买限制②：若无特殊说明，该称号下属技艺皆为逐级购买

使用限制：若无特殊说明，该称号及下属技艺中的效果皆须以单手持用【匕首】类武器进行近战攻击，且副手不持用任何物品才能生效。

描述：  
灵巧，轻便，便于携带，出手快速，直至要害……匕首的优点数不胜数，它是精通于近身搏斗的武器，是最适合刺杀的致命工具。

等级：D/C/B/A/S  
价格：D+500/C+1000/B+2000/A+4000/S+8000（逐级购买，效果叠加）  
购买限制：在购买时，需要额外支付等同于称号等级(D1/C2/B3/A4/S5)的XP。  
效果：  
【通用】：  
你获得等同于称号等级的闪避防御，使用匕首进行的攻击获得等额DP的技艺加值。

【D级】：  
你使用匕首进行的攻击额外获得【高速5】。

【C级】：  
每轮首次进行部位攻击时，可以免去至多2点部位攻击减值。  
现在，你可以在每回合开始进行一次敏捷属性为DP，传奇敏捷为附加成功的检定，且不计入任何加值与改变出目的能力。记录那个出目，那个回合中的攻击额外获得等同于出目的速度加值；若每回合开始时未进行该检定，则不适用该效果。

【B级】：  
你的攻击获得等同于自身敏捷属性点的【高速x】，这些高速在计算破防后若仍有剩余，则可以转化为等量的攻击上的速度加值。  
除此之外，你每拥有1点传奇敏捷，都可以在攻击时将攻击检定中的1点附加成功转换为3dp速度加值。

【A级】：  
当你在施展该称号下属技艺时，每个回合至多可以有3次，将需要消耗花招动作的行为改为自由动作。  
所有的匕首在你手中都如臂指使，额外提升1L武器伤害。现在，你在以匕首进行的攻击上获得额外等同于自身生理属性附加成功总和\*2点附加成功。

【S级】：  
你使用匕首进行的攻击额外获得1点附加成功与【高速5】，在满足称号条件的情况下，你免疫S级措手不及效果。每轮一次，你以匕首发动的攻击出目增加一半。  
若在发起一次攻击时，敌方无法对你进行精准定位，你的那次攻击额外具有【传奇】特性，获得2点额外的附加成功与1dp速度加值

【匕首使下属技能树】

技艺名称：快速调整  
价格：D+500  
动作：花招动作  
描述：  
在一次正手攻击之后，抽出武器的惯性会导致手腕难以快速完成反手切换，从而导致下一击的力道衰减。

效果：  
本技艺需在一次攻击结束后，自身还留有一次可以进行攻击动作的标准动作时可以施展，那么你可以结合该技巧使用标准动作进行一次攻击，结合该技巧的这次攻击额外提升2L武器伤害，并具有【高速5】。

技艺名称：袭击  
价格：D+500  
分类：被动  
描述：  
从专业的角度进行攻击，哪怕是近身的匕首也可以起到突袭的效果，出其不意，防不胜防。  
效果：  
当你在近身以匕首进行搏斗时，可以快速捅刺斩戳出数次，这视为你的攻击额外具有【连射】特性。若是你在敏捷属性上至少拥有1点附加成功，则在攻击上额外获得2dp高速加值，以速度与角度将武器的劣势化为优势。  
除此之外，每轮首次攻击前，你都能以一个花招动作变换角度，若如此做，你的这次攻击额外获得1dp速度加值，至多变化两次角度，连续变换两次时，额外获得1dp速度加值。  
技艺名称：精挡准打  
价格：D+500/C+2000/B+4000/A+8000  
描述：  
提前以目标的攻击轨迹预判其攻击落点，在闪避的同时以匕首偏斜攻击路径，一旦成功将获得绝佳的安全性，相对的，该能力十分考验使用者的运气与直觉

效果：  
D级：  
在招架一次攻击时，你可以以一个自由动作宣言一个数字，在表现上，你提前将匕首刺向某个角度；当目标这一次攻击造成的实际伤害等同于你宣言的数字时，你可以立即使这次攻击的伤害降低一级(冲击-严重-韧性)，至多以此降低7点。

C级：  
现在你不再承受冲击点数带来的负面影响，在D级的基础上，你可以在降低伤害等级(冲击-严重-韧性)时再额外降低10点，并且在猜出准确的伤害数值后，你的下次攻击可以额外获得1dp高速加值(不可叠加)。

B级：  
在C级的基础上，你可以在降低伤害等级(冲击-严重-韧性)时再额外降低14点，那之后针对这一次攻击伤害获得2点全伤害吸收。

在适用该效果后，你的下次攻击可以额外获得1点附加成功(不可叠加)，并具有【8加骰】；同时在攻击时，可以针对对方手中持有的一把武器进行击落武器，这不会导致进行的攻击效果产生变化，视为附带击落武器，以那次攻击伤害为dc，对方进行反射豁免，若失败则武器被击落。击落武器的效果每回合限一次。

A级：  
在B级的基础上，你可以在降低伤害等级(冲击-严重-韧性)时再额外降低28点，那之后针对这一次攻击伤害额外获得3点全伤害吸收。

在适用该效果后，你的下次攻击可以再额外获得1dp高速加值与1点附加成功(不可叠加，不同级效果可叠加)。此时，若你需要，每轮一次，你可以选择立即以手中的武器对触及范围内的一个目标进行一次不附带任何技艺的标准攻击。

技艺名称：一击必杀  
价格：D+500/C+1000/B+2000/A+4000/S+8000  
购买限制：该技艺的高等级效果与描述替换低等级效果。  
描述：  
匕首的一种传统使用方式就是用作隐蔽的暗杀武器，在现代，匕首更多是被用作于更加激烈且迅捷的近身格斗当中，而在刺客这一名词诞生之前，它们常常陪着行凶者一起蹲在茅坑里袭击毫无防备的后庭……言归正题，通过打磨与训练，以匕首对要害进行攻击是非常轻易的行为，若是在不被人发现的情况下，匕首将使得一击致命变得可能。

效果：  
【D级】：  
在发动一次攻击前，若敌方完全没有针对你的感知定位，你的这次攻击具有【9加骰】与额外的1dp速度加值和1点附加成功。

【C级】：  
在发动一次攻击前，若敌方完全没有针对你的感知定位，你的这次攻击具有【9加骰】与额外的4dp速度加值和7点附加成功。

拥有的每1点传奇敏捷都会在这次攻击中额外获得1点附加成功，每以此获得1点附加成功，都可以将攻击中的2dp转换为1点附加成功。

【B级】：  
在发动一次攻击前，若敌方完全没有针对你的感知定位，你的这次攻击具有【8加骰】与额外的10dp速度加值和9点附加成功，这次攻击额外视为具有【接触攻击】特性与B级【措手不及】效果，拥有的每1点传奇敏捷都会在这次攻击中额外获得2点附加成功，每以此获得1点附加成功，都可以将攻击中的2dp转换为1点附加成功。首次投掷此攻击检定时可以重投一次取优结算。

【A级】：  
在发动一次攻击前，若敌方完全没有针对你的感知定位，你的这次攻击具有【8加骰】与额外的14dp速度加值和14点附加成功，这次攻击额外视为具有【接触攻击】特性与A级【措手不及】效果，无视对方的盔甲与闪避防御，这是一个A级【创伤】来源的无视防御特性。首次投掷此攻击检定时可以重投一次取优结算。

对方以这次攻击伤害为DC进行强韧豁免，若失败则立即死亡，这是一个【创伤】来源的A级【即死】效果。拥有的每1点传奇敏捷都会在这次攻击中额外获得2点附加成功，每以此获得1点附加成功，都可以将攻击中的2dp转换为1点附加成功。

【S级】：  
在发动一次攻击前，若敌方完全没有针对你的感知定位，你的这次攻击具有【8加骰】与额外的14dp速度加62值和点附加成功，这次攻击额外视为具有【接触攻击】【传奇】特性与A级【措手不及】效果，无视对方的盔甲与闪避防御，这是一个A级【创伤】来源的无视防御特性。首次投掷此攻击检定时可以重投一次取优结算，且伤害增加两倍。

对方以这次攻击伤害为DC进行强韧豁免，若失败则立即死亡，这是一个【创伤】来源的S级【即死】效果。拥有的每1点传奇敏捷都会在这次攻击中额外获得2点附加成功，每以此获得1点附加成功，都可以将攻击中的2dp转换为1点附加成功。

技艺名称：迅捷白刃战  
价格：D+500/C+1000/B+2000  
描述：  
与大部分人所理解的有些偏差的是，在贴身白刃战中最具优势的武器一直是匕首，但在具有超凡能力的情况下，匕首的体术往往被人忽视，只有真正专精于匕首格斗者才能将其融入自身战斗方式当中并熟练运用，以此创造优势。

效果：  
【D级】：你在先攻上获得1枚10面骰，现在，你可以在一次攻击前以一个花招动作迅速调整姿态，若如此做，你将额外获得1点附加成功，一轮至多以此消耗2个花招动作。

【C级】：在D级的基础上，你在先攻上获得1枚先攻骰。你可以在一次攻击前以一个花招动作调整姿态，令这次攻击加骰提升一级(至多为8加骰)，并额外获得2dp速度加值。

【B级】：在D级的基础上，以该种方式获得的附加成功数量翻倍，每次使用都会令这次攻击额外无视2点抵高速。在C级的基础上，你在先攻上获得1枚先攻骰。现在，当你进行任何消耗花招动作的行为时(哪怕它被补充为自由动作)，你都可以额外获得1点附加成功与1dp速度加值(每轮至多3次)。

技艺名称：施手反击  
价格：D+500/C+1000/B+2000  
描述：  
由于其迅捷的出手速度，匕首是最适合后手反击的武器，甚至可以不用加上之一；通过熟练地掌握各类武器的进攻路线，从而可以达到在对方进行攻击时立即进行反击，以进攻作为防守

效果：  
【D级】：  
以一个移动动作，不消耗而是将其保留至回合外，如此保留的移动动作仅能用作进行回合外的招架行为，进入【施手反击】状态，你额外获得1点洞察防御。

该状态下进行招架时，不抵消伤害，转而对攻击者造成应抵消伤害量的不可豁免的物理-穿刺伤害，可以被吸收，若攻击来源超过你攻击的触及范围，理所当然地无法进行反击，如常结算招架抵消伤害。

【C级】：  
现在，你可以以自由动作宣言进入【施手反击】状态而无需提前预备一个移动动作，但这并不代表你可以不消耗移动动作进行招架，依然需要如正常招架般在回合外保留移动与反射动作才能进行。

现在，无论你是否进行招架，只要承受攻击伤害，都可以立即对攻击目标造成2点物理-穿刺反击伤害，不可豁免，可以被吸收，你无法对来源超过你触及范围的目标造成反击伤害。

【B级】：  
你可以以一个花招动作进行调整，若如此做，在下回合开始前，你进行招架时可以额外抵消1点伤害，一回合内你至多以此调整4次。

现在，你C级的反击伤害提升至4点，当你招架时，针对招架的那一次攻击，令其重投取劣结算。
