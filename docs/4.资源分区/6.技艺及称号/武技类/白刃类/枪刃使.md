# 枪刃使

名称：枪刃使  
类型：称号  
本质：魔幻本质  
概述：通过同时具有枪械与兵刃两种特性的武器枪刃，进行风格多样战斗的战士，对于个体素质和灵活度都有很高的要求。

D级：  
价格：D+500  
描述：初出茅庐的枪刃使通常每日都需要消耗大量的时间掌握在挥砍时开枪并稳定自身的态势，在攻击时掌握不好扣动扳机的时间是新手最容易犯的错。

效果：拥有该等级称号后，你的武技-枪刃专业等级上升一级（至多为5），在以枪刃进行攻击时，获得等同于自身传奇力量的技艺加值。  
枪刃换弹后发起的首次攻击额外获得2dp增强加值。

C级：  
价格：C+1000  
描述：能够在战斗中同时兼顾攻击、换弹与防御才算是一个枪刃使足以独当一面的表现，通常到此为止，枪刃使就有了独立作战的能力。

效果：拥有该等级称号后，你的武技-枪刃专业等级上升一级（至多为8），使用枪刃进行换弹的动作下降一级；进行格挡时，若此时枪刃内不具有晶弹，则可以额外视为具有【格挡4】。  
枪刃换弹后发起的首次攻击额外获得1点附加成功。

B级：  
价格：B+2000  
描述：对于一个成熟的战士来说，运用枪刃战斗早已刻入本能，可以用如臂指使来形容，真正的强者往往在于其能够攻守兼备，不露破绽。

效果：拥有该等级的称号后，在使用枪刃进行格挡时，额外具有【看破】特性，并具有仅能用于为枪刃换弹的2AP，当你拥有至少3个仅在使用枪刃时可以生效、触发或施展的B级技艺时，这些AP仅可以用于施展枪刃技艺。  
枪刃换弹后发起的首次攻击加骰额外提升一级（至多为8加骰）。
