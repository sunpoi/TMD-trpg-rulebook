# 废土奇物

  

迈克特达尔的扫描仪  
10分  
科技本质  
迈克特达尔博士制作的垃圾佬专用版扫描仪。只有半个手掌大小，使用两节一号电池即可工作20小时。以一个标准动作起动，扫描仪会自动扫描周围2千米内是否存在废料堆并在拼制的次品显示屏上显示出来。你只有具有废土守望者称号本身才可利用垃圾。  

提纯！  
对于你收集到的每种垃圾都可以利用迈克达特尔博士制作的提炼机进行提纯，提炼机价值1000分，需提前购买才能在主神空间中进行提炼，每影片只能提炼一次。每次开始提炼时，垃圾总价值必须等同于500分的垃圾，不足或多于500分时都无法进行提纯。对于每种垃圾而言，提炼结束后，该种垃圾将变为原先两倍的价值。你可以选择在提炼时额外掷一次D100，若大于50，则获得100点数的副产物垃圾作为额外收益（种类任选）；若小于50则无变化，已提纯的垃圾不可再重复提纯。  

拾荒者指环  
1500分  
科技本质  
这是一个破旧而无光的指环，环身刻着这样一句话：“勇气与荣光从不庇佑失落者。”这个指环将占用戒指位。以一个自由动作起动，使用者可以将手心中不超过10立方厘米的物品存储入指环中，这个指环的最大储存空间为3立方米，当使用者需取出储存空间中的物品时需要一个移动动作。额外的，这个指环将为佩戴者提供4DP偏斜防御。
