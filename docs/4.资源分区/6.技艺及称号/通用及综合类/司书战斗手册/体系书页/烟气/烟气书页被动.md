# 烟气书页被动

  

烟气书页被动  
  
  
  
烟气缠绕  
  
价格：750  
消耗：1被动点  
效果：现在你的烟气不使自身受到负面效果。  
  
围捕烟气  
  
价格：1500  
消耗：2被动点  
效果：现在你的攻击击中敌人时，追加【自身烟气层数/3】点不可豁免的冲击点数。  
  
致瘾烟气  
  
价格：1500  
消耗：1被动点  
效果：击中至少拥有8层【烟气】的目标时，对其施加1点不可豁免的麻痹点数。  
  
清醒烟气  
  
价格：1000  
消耗：1被动点  
效果：若自身拥有【烟气】，则在受击的时候获得等同于【自身烟气层数/2】的格挡防御  
  
内燃机  
  
价格：1250  
消耗：1被动点  
效果：每轮一次，使用书页获得【烟气】时额外获得一层【烟气】  
  
血肉齿轮  
  
价格：1000  
消耗：2被动点  
效果：使用【烟气】书页的第一颗“进攻型”骰子击中时，额外造成2点钝击严重伤害。  
  
  
供应过剩  
  
价格:：1500  
消耗：2被动点  
效果：若自身拥有10层烟气，则击中目标时额外造成2点钝击严重伤害。  
  
烟气过度  
  
价格：750  
消耗：2被动点  
效果：每一轮结束时，如果自身拥有至少八层【烟气】，则攻击获得+1附加成功。  
  
烟气精通  
  
价格：2000  
消耗：4被动点  
效果：在使用具有【烟气】关键词的书页作战时，攻击获得+2附加成功。  
此外，所有闪避/防御骰，拼点成功时都会给予1点闪避/格挡防御。
