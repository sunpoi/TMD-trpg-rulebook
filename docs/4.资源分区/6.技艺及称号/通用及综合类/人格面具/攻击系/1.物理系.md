# 物理系：![](./物理.jpg)

  

物理系：  
在判断物理系的伤害类型时，都为物理伤害下属的三种伤害类型范围内。物理系技能在释放时不像除枪击系以外的技能一样消耗SP，而是消耗HP（血量），消耗的HP无法通过除武道的诀窍以外的任何方式被减免、吸收，并且还能将使用者使用的近战武器加入到技能释放当中，释放的技能加入近战武器的武器伤害、特性。虽然无法使用近战武器的射程，但是射程能够根据人格面具的体积获得加成。
