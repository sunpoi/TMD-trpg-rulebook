# 核热系

核热系：

【芙雷】  
等级：D  
习得提供能力值：4  
消耗SP：1  
动作：标准  
射程：1米×【决心】  
范围：个体  
持续：立即  
对抗：防御  
效果：附带【小量】检定加值，若被攻击者具有【燃烧】、【冻结】、【触电】中任意一种效果，则还具有【小量】威猛效果。造成纯能量伤害。  
遗忘提供能力值：16

【芙雷拉】  
等级：C  
习得提供能力值：8  
消耗SP：3  
动作：标准  
射程：2米×【决心】  
范围：个体  
持续：立即  
对抗：防御  
效果：附带【中量】检定加值，若被攻击者具有【燃烧】、【冻结】、【触电】中任意一种效果，则还具有【中量】威猛效果。造成纯能量伤害。  
遗忘提供能力值：33

【芙雷达因】  
等级：B  
习得提供能力值：16  
消耗SP：5  
动作：标准  
射程：5米×【决心】  
范围：个体  
持续：立即  
对抗：防御  
效果：附带【中量】检定加值，若被攻击者具有【燃烧】、【冻结】、【触电】中任意一种效果，则还具有【中量】威猛效果，若具有的【燃烧】、【冻结】、【触电】累计达到10点及以上，触发【TECHNICAL】。造成纯能量伤害。  
遗忘提供能力值：66

【原子火焰】  
等级：S  
习得前提：拥有认知要素【核】  
习得提供能力值：32  
消耗SP：7  
动作：标准  
射程：50米×【决心】  
范围：个体  
持续：【战斗结束/15分钟】取低  
对抗：防御和反射  
效果：附带【特大量】检定加值。造成纯能量伤害。并使被攻击者流失等同于造成伤害量的能量，被吸收的能量能化作蓝色恒星虚影，虚影会通过重力将被攻击者控制在恒星中央，这是A级时空来源的定身效果。  
反射豁免对抗本效果。无论定身效果是否生效恒星虚影都会保留下来。  
恒星虚影、定身效果均持续【战斗结束/15分钟】取低。若被攻击者具有【燃烧】、【冻结】、【触电】，则触发【TECHNICAL】效果。  
遗忘提供能力值：101
