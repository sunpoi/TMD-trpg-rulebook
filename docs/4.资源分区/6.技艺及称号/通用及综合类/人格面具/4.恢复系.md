# 恢复系

恢复系技能卡：  
治疗HP（血量，优先恢复韧性）、异常状态、无法战斗（死亡）的技能，对应能力值为【魔】，若无特殊说明，恢复系技能的效果对不同的目标每轮只能生效一次。

【迪亚】  
等级：C  
习得提供能力值：8  
消耗SP：2  
动作：迅捷  
射程：1米×【决心】  
范围：个体  
持续：立即  
对抗：无  
效果：恢复目标【中量】HP。  
遗忘提供能力值：33

范围：个体
