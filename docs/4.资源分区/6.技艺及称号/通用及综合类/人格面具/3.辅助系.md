# 辅助系

辅助系技能卡：  
使自己和友方获得增益效果、使敌方承受减值的技能，对应能力值为【魔】。  
【攻击力上升】、【防御力上升】、【命中·回避率上升】、【暴击率上升】、【攻击力下降】、【防御下降】、【命中·回避率下降】

【塔尔卡加】  
等级：C  
习得提供能力值：8  
消耗SP：2  
动作：迅捷  
射程：5米×【传奇决心】  
范围：个体  
持续：【战斗结束/15分钟】取低  
对抗：无  
效果：使目标获得【攻击力上升】的增益效果，持续【战斗结束/15分钟】取低。  
遗忘提供能力值：33
