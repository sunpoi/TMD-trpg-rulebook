# 炽焰天穹专属物品

【炽焰天穹专属物品】  
“是不是物有所值呢？~那就赶紧去赚钱~你这混账东西~”

偏导护盾  
泛着淡蓝色光芒的蛋壳型能量护罩，可以制作成各式各样蓝色小装饰佩戴在身上，提供偏导护盾保护炽天使部队的成员。可以消耗护盾能量进行短距离传送。  
偏导护盾将如同【重甲】一样占据对应的位置，可以享受重甲相关资源/专长带来的好处。  
可以使用的偏导护盾等级不高于已强化的炽焰天穹下属称号等级。  
偏导护盾需要逐级提升为高等级偏导护盾，虽然高等级会替换低等级效果，但增幅器效果始终生效，不会因此被替换。

D+500  
给新兵在模拟器中进行模拟训练用的偏导护盾，传送效果由模拟器模拟所以没有加装，降低了成本。  
训练用偏导护盾为装备者提供5/5的力场防御，以及【力场护盾（1）】  
【力场护盾（1）】：力场护盾（X）会提供等同X点的力场防御，以及X点的护盾，护盾可以1：1的抵消伤害。这些护盾在一场战斗中若被击毁，则不会再恢复直至这场战斗结束。  
在这场战斗结束后，力场护盾将会缓慢恢复它带有的护盾，以每20分钟恢复1点的速度自动恢复，也可以使用快速医疗或是生命值恢复效果对它进行加速治疗。  
在对抗擒抱检定时，力场护盾此时每剩余1点未被击毁的护盾，就会额外提供1点对抗擒抱检定上的力量加值。

训练用偏导护盾具有一个增幅器插槽。  
赠送一个增幅器【第一步】。

C+1000  
炽天使部队的标准型号偏导护盾，只要消耗盾量就可以进行传送。  
标准型护盾为装备者提供5/5的力场防御，以及【力场护盾（6）】  
在进行移动时，你可以消耗1点护盾将本次移动变更为传送移动，这是C级时空来源的传送能力，如同对应数量的力场护盾被击破了一样。  
特殊的是，因传送移动被消耗的护盾会在你的下一次回合开始时回复至满值。  
标准型偏导护盾具有一个增幅器插槽。

B+2000  
负责攻坚的炽天使精英小队特供型号偏导护盾。能量出力更稳定，传送不再需要消耗盾量。  
特供型护盾为装备者提供5/5的力场防御，以及【力场护盾（22）】  
在进行移动时，你可以将本次移动变更为传送移动，这是B级时空来源的传送能力。  
偏导护盾具有一个增幅器插槽。

现在该护盾获得另一种充能方式：每场景一次，装备者可以将【传奇决心】属性\*6点完好生命值转换为韧性，每具有1点【传奇决心】，就需要额外支付1点意志力。  
装备者每因此将6点生命值转换为韧性，则可以在该场景内获得【力场护盾（2）】，与等量的力场防御，当因此累计获得【力场护盾（6）】时，只能获得4点而不是6点力场防御，此后也无法再因此获得额外力场防御，只能获得力场护盾。

【增幅器：第一步以作为1000分抵扣，用于差价购买更高级的增幅器，因此抵扣的价格会减少对应的评分，即增幅器：第一步本身与对应的抵扣额度不会计入评分。】

宝珠  
只有强化了至少C级炽焰天穹下属称号后方可购买。  
特殊的，这不是一个成长类资源，购买宝珠等同于购买了奇物。

C+1000  
在战斗场景骰先攻时决定是否使用此奇物。  
战斗开始时消耗1意志，战斗轮每经过【5】轮再消耗1意志。由此方式消耗意志最低只会使意志变为1。意志已经为1且仍需消耗时，转为将生命骰的五分之一（去尾计算取整）恶化一级（完好→恶性→韧性伤害），优先从完好开始转化。  
战斗结束时：战斗第一轮开始时，以及战斗每经过【5】轮时，获得一枚【宝珠】，不足五轮的不计算；这次战斗每消灭（不需亲自造成最后一击）一名同级或以上评级的敌人，获得一枚【宝珠】。  
【宝珠】强化等级已满后，使用【宝珠】的效果无需在进行消费意志力。

分别消耗1、2、3、4、5枚宝珠（即累计1、3、6、10、15枚），可以提升一级宝珠强化等级，这一行为无需在主神空间进行。不同炽焰天穹下属称号根据宝珠强化等级获得的强化效果不同，如下：
