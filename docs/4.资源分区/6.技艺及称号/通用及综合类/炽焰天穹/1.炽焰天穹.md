# 炽焰天穹

炽焰天穹  
本质：科技本质  
人类存亡的危机迫在眉睫  
反抗命运的是  
背负着最后希望的少女们

纵使迷惘、纵使抗争  
无休止的战争也仍在继续……  
但正因如此，  
我们才能发现此刻的意义。  
「大家可都要活下去啊。」

由于主神的改良，不是少女也可以进行炽焰天穹下属强化。  
但即使如此，也只有使用生命值（而非结构值）的单位可以正常强化该资源。

称号的购买价格分别是D+1000/C+1000/B+2000，各级效果叠加，你只能选择一种炽焰天穹下属分类进行强化（如攻击者，减益者，破盾者等等，只能选择一种进行强化），在D级时除了获得选择分类的效果外，还具有一项通用效果。  
若以下效果具有范围，则所有范围皆为以自身为中心展开的效果，除非另有特殊说明。

D级通用效果：  
选择一个（或双手持用的一对）武器作为你的专属炽天使武器。  
你获得【电子军人手册】，外形是手掌大的智能掌上电脑，其除作为正常的电脑使用外，可以用于存放你的专属炽天使武器（自选已有的武器绑定），以及作为最大10立方米的个人储物空间使用，只能存放死物。

设定一个你自己的专属口号。  
在你手持【电子军人手册】时，你可以使用迅捷动作将你的专属炽天使武器拿在手上。其画风为你喊出你的专属口号，你的上方虚空打开虫洞，武器出现并落入你的手中。  
你可以在主神空间更换你的专属炽天使武器，但是其武器类型不能改变。（例，你可以把无支线长剑换为D级长剑；或把自动手枪换为左轮手枪，等。但你不能把手枪换成霰弹枪，或者把长剑换为镰刀，等）

攻击者（ATTACKER）-主力负责在敌方护盾被破除后输出的职业。  
D  
你的攻击检定获得2DP修行加值。  
如果你攻击的目标在计算防御破除后，能量防御与力场防御、偏斜防御均为0，则你的这次攻击检定获得3DP修行加值。  
C  
你的专属炽天使武器获得【毁灭】。  
如果你攻击的目标在计算防御破除后，能量防御与力场防御、偏斜防御均为0，则你的这次攻击检定再获得3附加成功。  
B  
你的专属炽天使武器的攻击可以视为接触攻击。  
如果你攻击的目标在计算防御破除后，能量防御与力场防御、偏斜防御均为0，则你的这次攻击检定上再获得2点附加成功与【幻灭】。

破盾者（BREAKER）-主力负责输出破除敌方护盾的职业。  
D  
你的攻击检定获得2DP修行加值。  
只要你攻击的目标存在能量防御与力场防御、偏斜防御中的任意一种，则你的这次攻击检定获得2DP修行加值。  
C  
你的专属炽天使武器获得【毁灭】。  
只要你攻击的目标存在【生命槽上限】（结构值上限）以外的生命/结构值（如临时生命、护盾、载具结构值等），则你的这次攻击检定再获得2附加成功。  
B  
如果你攻击的目标存在【生命槽上限】（结构值上限）以外的生命/结构值（如临时生命、护盾、载具结构值等），则你的这次攻击检定再获得8附加成功。

破坏者（BLASTER）-破除敌方外壳，使其受伤加深的职业。  
D  
你的攻击检定获得2DP修行加值。  
购买时从所有伤害性质中选择一种，被你的攻击成功命中的目标获得1点对应的【易伤】，持续二轮。  
每1点易伤会使目标受到所选择性质的伤害增加1点。  
C  
你的攻击检定获得5DP修行加值。  
被你的攻击成功命中的目标，其获得的【易伤】提升1点，在易伤生效期间，目标受到的所有非韧性伤害会有1点转为韧性伤害  
B  
你的专属炽天使武器进行攻击命中后，目标需要进行一个强韧豁免，若失败，则目标会在原地被剥离出一块可供攻击的灵魂碎片，这是一个B级创伤来源的【魂牵】效果，持续二轮。  
被你的攻击成功命中的目标，其获得的【易伤】提升2点，在易伤生效期间，目标受到的所有非韧性伤害会有2点转为韧性伤害。

增益者（BUFFER）-加强我方战斗力的职业。  
D  
你的智力相关检定+2DP修行加值。  
你的先攻增加D10。  
C  
你获得技能【攻击提升】  
技能名称：【攻击提升】  
动作：标准动作  
消耗：无  
范围：15m半径圆形  
目标：范围内的自愿友方单位（包括自己）  
持续：一轮  
效果：目标在攻击检定上获得8DP增强加值，以及这个数值一半的攻击附加成功。  
B  
你获得技能【暴击提升】，【暴击提升】将替换【攻击提升】的效果。  
技能名称：【暴击提升】  
动作：迅捷  
消耗：无  
范围：15m半径圆形  
持续：一轮  
目标：范围内的自愿友方单位（包括自己）  
效果：目标的攻击获得【湮灭】效果，攻击检定上获得9DP增强加值，以及2点攻击附加成功。

减益者（DEBUFFER）-削减敌方能力的职业。  
D级  
永久指定一项检定，当你进行以敌对单位为目标的，该检定获得4DP修行加值。  
C  
你获得技能【破防加持】  
技能名称：【破防加持】  
动作：自由动作  
消耗：无  
持续：一轮  
效果：你在下次攻击时获得5DP的器械加值，同时在命中后给予该敌方单位【熔毁5】。  
B  
你获得技能【灾厄加持】  
技能名称：【灾厄加持】  
动作：自由  
消耗：无  
效果：你在下次攻击时获得1DP器械+1附加成功，同时造成伤害出目等额的精神束缚点数（标准豁免，可叠加）。

治疗者（HEALER）-治疗我方角色的职业。  
D  
你选择一个可以以友方单位为目标的技能/能力，使其快速医疗点数增加1点，医疗池扩大20点。  
C  
你获得技能【恢复】  
技能名称：【恢复】  
动作：标准动作  
消耗：无  
射程：20m  
范围：射程内指定一点为中心，半径10m  
效果：立刻将范围内所有友方目标（包括自身）6点伤害值恢复到完好（共用医疗点72）。  
B  
你获得技能【饱和恢复】，【饱和恢复】将替换【恢复】的效果。  
技能名称：【饱和恢复】  
动作：标准动作  
消耗：无  
范围：20m半径圆形  
目标：自愿友方单位（包括自己）  
效果：立刻将目标身上12点伤害恢复至完好（共用医疗点144）。

防御者（DEFENDER）-减少我方受到伤害的职业。  
D  
你被攻击时忽略对方的5点破甲/高速/破魔/威猛破防能力，必须在购买时从四种之一忽略中选择一种获得，同时提升1点天生防御。  
C  
你获得技能【保护】  
技能名称：【保护】  
动作：移动动作  
消耗：无  
范围：3.75m半径的圆形  
持续：一轮  
效果：你可以为范围内所有友方单位提供全遮蔽效果。  
B  
你获得技能【强效保护】，【强效保护】将替换【保护】的效果。  
技能名称：【强效保护】  
动作：迅捷动作  
消耗：无  
范围：20m半径的圆形  
效果：你可以为范围内所有友方单位（包括自己）提供4点全伤害吸收效果，以及5点闪避防御的提升。此外，你可以对范围内不包括自己在内的友方单位提供全遮蔽效果。
