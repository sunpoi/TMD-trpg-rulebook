# 面具杀人狂

面具杀人狂  
描述：恶灵以强烈的情感为食粮，它创造了各个噩梦世界作为审判之地，让杀手和逃生者都被困在幻境之中，杀手要做的事情很简单：让逃生者饱尝恐惧，收割他们的灵魂献祭给恶灵。  
【Death is not an escape（死亡不是解脱）。】

  
D级：惊声尖叫  
杀手面具：你获得了一个概念面具，你可以定制它的形象但必须符合恐怖片的概念，面具无法被打破，且不占用面具/头盔栏，而是使你的面具/头盔转变为对应的样式。

  
独行杀手：当你独自面对一个独行的敌人时，你对他的洞察，隐秘，攻击检定获得2DP的表现加值。

  
无踪鬼影：你在潜行的时候可以以正常速度移动并不受到限制。

  
C级：万圣前夜  
当众谋杀：现在你在D级获得的【独行杀手】特性获得了提升，你可以使用一个迅捷动作，消费1点意志力进入这一状态。处于该状态下，你在洞察，隐秘，攻击上获得5DP的表现加值。

  
编号不明：每轮一次，你可以在攻击前以自由动作进行一次虚招，如同正常虚招流程的内容一样进行对抗。  
在画面表现上，对方对你手中的凶器猝不及防，被你一击捅穿。

  
梦魇化身：仅限处于【当众谋杀】状态下，若你具有自然本质的反侦测能力时，若该反侦测能力是无支线/D级，则将对应等级提升至D级/C级。  
若你具有其他类型的D/C级反侦测能力，也可以选择将该效果改为获得C级空间来源的传送移动能力。

  
B级：13号星期五  
一刀了断：众所周知，你出刀就不用擦刀，往往最需要一刀就能将逃生者处决。  
你在处于【当众谋杀】状态下时，进行的攻击附带【幻灭】特性。  
你也可以选择将这一效果替换为【魂牵】，表现为在攻击后使对方留下一个标记，只要攻击标记就等同于攻击对方，这是一个B级附身来源的效果。

  
恶灵附身：死亡不是解脱，不光对于其他人而言，对于你也是一样。  
如果你因任何原因死亡，则将在一个月后在尸身位置复活。复活期间，你处于灵体状态，只能进行观察或者是移动的行动，以意志值作为结构值，期间构造值被清空则复活失败。

  
专属技能  
电锯狂魔  
价格：C+1000  
分类：被动  
描述：血腥桥段中最经典的一环就是使用电锯分割受害者的身体，无论是从视觉冲击还是从真实体验来说都是极为震撼的内容。  
效果：  
你获得2点，只能用于使用冲锋所需AP的，这2点AP会将本次冲锋的移动动作升级为杂技动作。  
无论如何，本次冲锋时的有效距离将取最大值、冲锋的移动行为结束时即使对方已经离开了攻击范围也可以按照脱离攻击范围一般对目标进行攻击。

  
恐怖鬼影  
分类：被动  
价格：C+1250  
描述：你的行动就像是被剪辑过的影片，没有过程，只有屠刀落下的结果。  
效果：  
你现在只需要一个迅捷动作就能试图重新进入潜行状态。同时，在你攻击任意一个会因你的攻击而陷入措手不及状态的单位前，你可以消耗1点意志力进行擦刀，随后在本场景内，使自身攻击获得等同于自身传奇力量或传奇敏捷\*1点附加成功，两者取高计算。

  
职业杀手  
购买前提：接线员  
价格：20XP  
描述：你是雇佣杀手与自由射手，以杀戮为生，以此为乐。你与其他杀手的唯一区别是――你效忠于恶灵，而并非人类。  
效果：  
若你在建卡后不具备接线员身份，则你可以支付4XP购买接线员身份，你的声望将永久性转为等额的负数声望，负数声望即负面声望。  
当你具有15点负面声望时，你获得D级附身来源的接线员/杀手身份，不附带过往经历，仅附带能证明身份的物品（身份证，健康码）。  
通常情况下，一般民众并不会认识这种身份，治安/军事机构则会在你表露身份的时候，立刻将你扣押。  
――  
当你具有90点负面声望时，你获得C级附身来源的，能被查到过往经历的接线员/杀手身份身份，并附带记忆，但该身份不会带来任何地位上的好处，不附带人际关系，仅附带足够生存的金钱。  
通常情况下，一般民众并不会认识这种身份，治安/军事机构则会在你表露身份的时候，立刻将你扣押，转交司法机构审理。  
当然，特殊影片会有特殊处理，并不一定是这样。  
――  
当你具有180点负面声望时，你获得B级附身来源的，有实际经历的接线员/杀手身份，并附带记忆，附带通常程度的人际关系，附带正常应有的财产，但是有限（包括但不限于：一套装修过的租房、一辆需要还贷款的车）。  
通常情况下，一般民众可能会认识并且惧怕这种身份，治安/军事机构则会在你表露身份的时候，酌情介于将你压制转交司法处决还是当场击毙的区间中。  
当然，特殊影片会有特殊处理，并不一定是这样。  
――  
当你具有300点负面声望时，你获得A级附身来源的，有实际经历的接线员/杀手特殊身份，解释自身能力来源，并附带记忆，附带特殊渠道的人际关系，附带正常应有的财产，附带符合安排身份的消耗品（需要ST临时安排，无法带出影片）。  
通常情况下，只要你展露身份，一般民众一定会认识以及恐惧这种身份，治安/军事机构则会在你表露身份的时候，考虑将你请走或者收容，这是因为你的名声太过于恐怖以及超规格。  
当然，特殊影片会有特殊处理，并不一定是这样。  
――  
特殊的，购买这一技能不会获得自由属性。
