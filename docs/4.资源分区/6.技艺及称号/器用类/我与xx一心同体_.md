# 我与XX一心同体！

我与XX一心同体！  
资源类型：器用技艺  
本质：魔幻本质

价格：D+500  
效果：  
驾驶员与载具一心同体，载具和驾驶员具有的免疫效果可以互通使用，但无论如何同时只能有其中一方能够受到免疫的好处，且载具作为结构体获得免疫效果以及无心智载具的心灵免疫无法被驾驶员互通使用。作为代价，未被免疫的任何具有来源的负面效果，驾驶员和载具要同时承受，正面效果则一般只能由其中一方接受。
