# 海虎武学

【电磁力】下属技艺-【海虎】系列武学  
购买本系列技艺的共通前置要求为：至少拥有同等级的【电磁力】能量池

自磁场力量现世以来，无数强者诞生。在那之中，海虎便绝对是一个特立独行的人，他不慕名利，无谓理想，仅凭本心而挥拳。

【海虎战纹】  
战纹是一种随着磁场强者的战斗意志而浮现的纹身，它通常代表了磁场强者的性格与武艺特点。  
拥有任意海虎系列武学技艺后，当你施展技艺时可以浮现出海虎战纹  
海虎战纹的初级形态为额头上的三道尖锐的三角形赤红纹路，宛如鲨鱼的利齿  
随着你的【电磁力】等级提升，该战纹也会延申，使你的脸颊、胸口、手臂等部位也浮现赤红的利齿状纹路。

【海虎爆破拳/海虎爆破腿】  
海虎的标志性招式，以磁场力量推动拳脚轰出，命中后劲力会隐藏在敌人体内，随使用者心意迫发，产生诸多妙用。  
购买时需要选择【拳】或【腿】专业之一作为技艺绑定的攻击方式，以下技艺介绍以【拳】为例。

若需要处理来源，海虎爆破拳相关效果的来源皆视为【创伤】来源（例如击飞等效果来源）。

D级：D+500  
爆破拳的最初形态，这个阶段你仅仅只是会借助磁场力量加大拳劲  
以一个标准动作，你使用【拳】专业发起近战肉搏攻击，此次攻击获得5点【威猛】，该破防属性结算目标防御后若仍有剩，则这些剩余威猛转化为本次攻击检定上的技艺加值。  
此外，你可以在使用此技艺时消耗至多2点【电磁力】，每消耗1点就使你获得1点技艺加值，每累积消耗2点再额外获得1点士气加值

C级差价升级：C+1000  
当你真正步入【磁场转动】的境界时，你也便能真正接触到爆破拳的精奥之处  
现在你使用的爆破拳不仅具有D级的效果，而且在命中目标后，将会在其身上留下1道【爆破劲】；你可以立刻选择引爆它并获得对应效果，或将其存留等待后续的攻击再引爆，【爆破劲】持续存在一场景/直到下次攻击（取低）。

当引爆【爆破劲】时，目标需要以强韧豁免对抗你的天生武器伤害点数，失败则承受差值的冲击点数；多次攻击留下的【爆破劲】可以同时存在，但每次【爆破劲】造成的冲击点数都需要单独豁免，这是一个类【余波】效果。

此外，你现在在该技艺中可消耗的【电磁力】上限提升为6。

额外支付C+2000+1XP：从现在开始，根据你在目标身上留下的【爆破】劲数量，你在引爆【爆破劲】的那次攻击中会获得额外效果；你每轮将只能在每个目标身上留下1道【爆破劲】，你留下的【爆破劲】将持续存在一场景而非与下次攻击取低。  
1道：本次攻击可以转化为C级【击飞】效果，这会使攻击不造成伤害，目标使用反射豁免对抗伤害出目，失败则被向上击飞差值/米。  
2道：本次攻击获得【9加骰】。  
3道：本次攻击附带【熔毁5】。

B级差价升级：B+3000。  
你的磁场力量已经达到了反地心引力境界，在这个阶段，你的爆破拳可以融入更多属性和招式变化。  
现在你使用的爆破拳不仅具有C级的效果，其中附带的【威猛】还会提升至10，并且附带B级击飞效果（这意味着攻击命中后，目标需要使用反射豁免对抗伤害出目，失败则被击飞差值/米，正常具有B级击飞的效果）。  
此外，你现在在该技艺中可消耗的【电磁力】上限提升为10。

额外支付B+3000+3XP（由C级额外支付价格差价升级）：

取代C级额外支付获得的效果；从现在开始，根据你在目标身上留下的【爆破】劲数量，你在引爆【爆破劲】的那次攻击中会获得额外效果；你每轮将只能在每个目标身上留下1道【爆破劲】，你留下的【爆破劲】将持续存在一场景而非与下次攻击取低。  
1道：此次攻击获得【8加骰】。  
2道：此次攻击附带B级【重伤】效果。  
3道：此次获得【熔毁5】和【威猛5】，该破防属性结算目标防御后若仍有剩余，则这些剩余威猛转化为本次攻击检定上的技艺加值。

A级差价升级：A+5500  
前置：已购买海虎爆破拳下属B级额外支付

你的磁场力量达到了原子分类境界，一拳轰出鲜有人可以抵挡。

取代之前所有等级（包括额外支付支获得的效果）：你可以以一个标准动作，使用【拳】专业发起近战肉搏攻击，此次攻击获得10点【威猛】，该破防属性结算目标防御后若仍有剩，则这些剩余威猛转化为本次攻击检定上的技艺加值。

此外，你可以在使用此技艺时消耗至多14点【电磁力】，每消耗1点就使你在该场景内近战肉搏攻击获得1点技艺加值，每累积消耗2点再额外获得1点士气加值；在你使用近战肉搏攻击命中目标后，将会在其身上留下【传奇决心\*1】道【爆破劲】；你可以立刻选择引爆它们并获得对应效果，或将其存留等待后续的攻击再引爆，【爆破劲】持续存在一场景。

当引爆【爆破劲】时，目标需要以强韧豁免对抗你的天生武器伤害点数，失败则承受差值的冲击点数，这是一个类【余波】效果；根据你在目标身上留下的【爆破】劲数量，你在引爆【爆破劲】的那次攻击中会获得额外效果。

需要注意的是，从A级开始海虎爆破拳是一次性引爆所有【爆破劲】，因此即使你通过多次攻击留下多次【爆破劲】，引爆时也只会处理一次上述提及的类余波效果。

1道：此次攻击获得【8加骰】与【湮灭】效果。  
2道：此次攻击附带B级【重伤】效果。  
3道或以上：此次攻击额外获得【熔毁10】和【威猛10】。

【磁场天锁】  
A+10000  
运用力量封锁目标的磁场，使得对方的能量也随之被封印的特殊技艺  
以一个整轮动作，你可以在该场景内进行攻击时，对被命中的目标施加磁场天锁。对方若有能量池，你在攻击命中指定对方任意一种能量池，施加至多为你的【每万匹磁场转动匹数（即1~100）】的【磁场天锁】。

处于【磁场天锁】状态时，该能量池立刻被清空（若对方剩余能量大于你的磁场转动匹数，则只消耗等同于匹数的量），并且以任何手段恢复的能量都会被【磁场天锁】立刻消耗，直到对方累积消耗了等于这个【磁场天锁】匹数的能量为止，该效果持续一个场景。  
同一角色的同一时间只能被施加一个【磁场天锁】。

【急速子弹拳】  
B+3000  
运用磁场力量加快出拳速度，宛如机关枪一样将拳头疯狂射出的技艺。  
你使用消耗【电磁力】的【拳】或【腿】（购买时选择一种）攻击技艺时，你可以额外消耗至多等同于你的【电磁力能量池等级数-1（即d0c1b2a3s4）】点AP。  
那之后该攻击可以将造成的最终伤害平均拆分为【d1c2b3a4s5】次，只需要计算一次防御与减伤相关内容，以及附带的效果。  
你的此次攻击获得【高速5】，每拆分出一次攻击额外获得【高速5】，该破防属性结算后若仍有剩余则转化为检定上的技艺加值

【五极力量】  
B+4500  
前置：B级爆破拳  
在与奥加大战之前，海虎为了丰富自己招式的多变性而开发了将元素力量融合进招式的技艺，习得此技艺后，你可以在爆破拳中融入风火雷电雪五种不同的力量  
以一个标准动作并消耗9点【电磁力】，你使用磁场力量调动周围环境中的元素，接下来一个场景内，你获得以下好处：  
当你使用海虎爆破拳时，可以选择以下一种元素效果附加，每轮你只能选择其中一种：  
①风：你的此次攻击触及距离增加60米，可以触及距离内的所有敌人造成伤害。  
②火：你的此次攻击获得【毁灭】特性，伤害转化为等量的【燃烧】不良点数。  
③雷：你的此次攻击附加等量的【目眩】、【耳鸣】不良点数，命中目标将以强韧豁免对抗两次伤害出目，失败则承受差值的对应不良点数。  
④电：你的此次攻击获得【能量武器】特性，并附带等同伤害出目的【麻痹】不良点数，正常豁免。  
⑤雪：你的此次攻击命中后，目标在本轮内攻击你时需要承受【劣化1】，这一劣化基于你的偏斜防御，你的这次攻击附带等同伤害出目的【冻结】不良点数，正常豁免。
