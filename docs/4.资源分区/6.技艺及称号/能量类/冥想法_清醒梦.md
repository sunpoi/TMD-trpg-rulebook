# 冥想法-清醒梦

冥想法-清醒梦  
前提：具有魔力(精神魔力)  
魔幻本质  
技艺  
D+500  
描述：...在那场梦后，那些觉醒者不断的尝试回忆，而在迷茫与探索中，一种似梦非梦的冥想法被开发了出来，被主神记录下来。  
效果：学会后，你可以在一次长休息时选择冥想来在清醒的意识状态下进入梦境来进行额外的精神修行。这种技巧探索的梦境有不同的倾向，购买时从下述梦境倾向中选择一项，之后不可更改。  
1.心灵迷宫：在你应用了冥想法时，直到下一次长休息时，你将你的内心用冥想时建造的迷宫封闭，心思不行于表面，你具有B级读心效果免疫。  
2.幻象自我：在你应用了冥想法时，直到下一次长休息时，你在梦境窥视你的过去回忆的行为将会给你的所有专注技能检定提供1点附加成功。  
且当你修行一段时间后，冥想法对修炼者的精神的增强会体现出来。在购买后应用了冥想的效果的第一场影片后，你获得2点智力相关检定的内在加值，你的精神魔力能量池上限提高7点。
