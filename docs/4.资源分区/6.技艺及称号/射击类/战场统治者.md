# 战场统治者

  

战场统治者

弹药专家D+1000  
手雷与子弹的主宰者，大范围的手雷与各种子弹纷飞的交叉射击中，没有目标可以逃脱。

超负荷装填：为子弹装填特殊弹药，每回合一次，可以改变一次弹药伤害类型，从火、冰、电中选择一个获得。  
弹药改良：增加子弹的穿透力和杀伤力，在使用枪械类武器时增加3点武器伤害，如果为步枪则额外提升1点高速。  

弹药专家C+1000  
“只有外行人才仅仅看重枪械，子弹的力量一样重要。”

穿甲弹：使用枪械时提升子弹的稳定性与穿透力，手中的枪械攻击时射程增加一倍，并获得贯穿特性，特殊的，若使用的枪械为步枪或狙击枪，则只能提升30m射程。  
交叉射击：双手交叉，令武器形成交叉火力，一回合一次，在使用战场统治者的技艺时，可以以弹幕形式发动攻击，被命中的目标需要进行一次DC等于伤害值的反射豁免，若失败则被击退失败数\*1+15m。

大将军B+2000  
与其他神枪手不同，他们的子弹有着特殊的功能，可以瞬间穿过对手或将对手火化，甚至将多个敌人心脏瞬间穿透。

聚合弹：改良弹药的火药，使子弹更容易发挥作用，在进行攻击时，可以把本次攻击视为进行了一次长点射，若已拥有连射等级，则额外提升1点高速，并且额外获得1DP技艺加值。  
C4爆弹：选择不超过手中武器射程的一个或多个目标，使用标准动作向其丢出C4爆弹，进行一次不附带招式的攻击检定这视为一次攻击但不造成伤害，所有的成功数都会被等量转化为C4爆炸时的爆炸伤害，并额外再提升4点伤害。只要存在一个成功数，C4就会附着于目标身上，目标可使用“智力+学识：爆炸物”来对抗那次攻击的成功数，成功即可拆除。每回合投掷的C4爆弹数目不能超过称号等级（D1~A4）且C4最多存在4个，当场景内的C4达到上限时无法进行投掷，C4最多存在一个场景且在离开战场统治者视线之后消失。使用战场统治者的技艺造成伤害时C4会发生爆炸并让被伤害目标承受【已附着C4爆炸伤害】的额外伤害，若引爆的C4为1个，则使用技艺引爆C4的那次攻击中至多会有四分之一伤害直接攻击被攻击者的韧性槽，每多出1个，这个伤害数将提升一倍，最高所有成功数都将直接攻击韧性。在那次攻击过后，此次攻击中涉及的C4将因爆炸而消失。特殊的，在战斗场景外，安装一个C4只需要6s  

  
战场统治者A+8000  
无论在何时何地，榜样与领袖都是不可缺少的，战场统治者就是士兵的榜样与领袖。

火力支援：在战斗的时候召唤炮击支援，每次行动之前战场统治者会率先对射程中的所有敌方发动一次炮击。进行一次不带招式能量的攻击检定，出数作为炮击的伤害数值，被命中的目标以反射进行对抗，并再以意志豁免对抗一次，若失败，则陷入一轮的震慑状态。  

  
技能树：  
D级：

弹夹改装：D+500  
进行一次延长检定，DC为手中武器支线等级的两倍，以智力+学识，专业：枪械进行检定，周期为小时。改装成功后，当这把枪械在你手中时视作具有无限弹药特性，但该效果的原理是直接消耗你身上的弹药而无需装取弹药，因此你身上需要携带有足量有弹药，否则该特性无法生效。同时也因为这个原理，这个无限子弹特性可射击特殊子弹，前提是你身上有携带特殊子弹。  
此外被你改造的枪械在你使用时速射等级+1（单发－短点射－长点射－连射）。  

手雷精通：D+1500  
技能类型：攻击技  
以一个标准动作使用，使用手中的枪械进行一次射击攻击检定（不通过正常防御，也不会对他人进行攻击），并生成一个手雷，之后可以以自由动作丢出手雷。本次射击检定会获得1点技艺加值。  
手雷的伤害为本次攻击检定的成功数，拥有10米的爆炸范围，并以反射进行对抗。  
手雷最远可以被丢出手中枪械射程的四分之一米。使用基础射程低于20的枪械时无法使用此能力。  
在回合结束时，无论手雷是否丢出都会爆炸。  

C级：

冰冻弹：D+1500  
技能类型：攻击技  
处于冰属性弹药时可以使用。  
在使用冰属性弹药攻击时，那次攻击每有一个出数为10就可以为目标累计1个【霜地】点数，每个点数降低目标5点移速，持续一轮，上限5点。攻击存在【霜地】点数的目标时造成一点冰属性额外伤害。  

B级：

爆炎弹：B+2000  
技能类型：攻击技  
处于火属性弹药时可使用。  
子弹在触及目标时发生一次爆炸并在穿透之后立即再发生一次爆炸。  
使用标准动作进行两次攻击，两次攻击均会爆炸并对射击目标周围10m内所有单位造成伤害，被波及的单位需要以反射进行对抗。  

地狱烈炎：B+2000

以标准动作向站立位置发射一颗充满不稳定燃料的爆弹，爆弹会覆盖战场统治者站立地点周围的区域并造成范围伤害，范围伤害的区域半径大小为手中枪械的射程，范围伤害数等于这次攻击检定（不通过正常防御）的成功数，区域内的目标以反射对抗伤害，并且本次反射承受10DP环境减值。  
爆弹会在爆炸区域产生一个同样范围内的熔岩地区，熔岩的属性取决于弹药属性，并视为一个持续一个场景的B级环境来源困难地形。熔岩地带不可以存在多个，生成新的地带时前一个无条件移除。

弹药主宰 B+2000

当你进行攻击时，你的这次攻击造成的伤害在原先的伤害性质与你当前使用的【超负荷装填】对应的伤害性质中取优生效。  
C4爆弹可设置数量+1。炎爆弹命中目标时造成额外伤害+3。冰冻弹的【霜地】点数累计上限提升5点。

  
战场勋章：A+4000  
如果不能做到吾日三省吾身，怎么能让士兵活下来？只有做到这些，才具有统筹战场的资格。如果一个人的名字就可以令敌人闻风丧胆，那他一定是一个神话般的人物。

当击杀敌人、完成任务、亦或是其他可以振奋军心的事情，都可以为自己佩戴一枚勋章，这些勋章会忠实的记录事迹和成就，也会在交战中震慑敌人，在对目标造成伤害时还会为目标造成等同伤害的恐惧点数，使用标准豁免进行对抗。  
不同的勋章还会产生不同效果，但勋章一旦佩戴就无法摘下或更换，但如果主神允许，可以放弃所有荣誉从头再来。总共有5个槽位可以放置获得的勋章，每块勋章占用一个槽位，满足一次以下条件时，可以从中选择一块勋章获得，获得的勋章总数最多不能超过本称号等级（D2C3B4A5），每种勋章至多选择一次。  
1、击杀难度等级至少为自身二倍的大型BOSS  
战神勋章：战场统治者在使用枪械进攻时获得8加骰。  
风采勋章：在进行攻击检定时可以重骰一次，两次出目可以任选其一结算。  
2、在一场战斗难度高于自身两倍的战斗中胜利  
战损勋章：获得3点快速医疗（医疗点96）。  
战地勋章：攻击命中目标时若可以击退敌人，则额外再击退60m。  
3、购买两个D级称号/技艺  
战场恶鬼：目标在承受战场统治者的攻击时降低12点反射。  
战杰勋章：提升9点洞察防御。  
战场修罗：目标承受战场统治者的伤害时额外4点转变为直接攻击韧性。
