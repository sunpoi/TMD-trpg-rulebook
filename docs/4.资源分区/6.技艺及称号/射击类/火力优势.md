# 火力优势

火力优势  
购买前提：枪械技能3+  
描述：无论是怎么样的位置，只需要负责清空所有的弹仓，在作战中通过集中和增强火力，以绝对的优势压倒敌人，就能实现战场上的胜利，这就是最简单粗暴版本的火力优势学。  
“人命关天，子弹价廉。”

  
特殊：火力优势学下属的武器是逐级购买，效果替换。

  
价格：D+750  
超远程射击：安全距离之下才能合理的发挥火力，因此你在使用枪械武器进行攻击时，武器的基础射程+1倍。不过，如果使用的武器类型是步枪或狙击枪，则将这个效果改为提升你的内在支线最高等级\*10米的武器基础射程。

  
精准落点：丰富的经验和观测让你可以打出诡异的弹道，这种弹道可以忽略全遮蔽以外的遮蔽效果，但因此忽略的防御不会超过5点，这个效果也可以对无名防御生效。

  
火力支援技能树：  
祝福语  
价格：D+500  
描述：在子弹上刻下一些祝福的话语，当这颗子弹被发射出去时，那些祝福也被送予了上苍。  
通常来说，你不必担心子弹的体积和字迹的问题，该系列的子弹稍微会大一些。  
效果：仅能在非战斗状态下对实弹弹药使用该技艺，这些被加工过的弹药只有你使用的时候才会生效。  
花费15分钟，用你在子弹的金属表面刻上一些祝福语，如“这是送你们的圣诞节礼物！”等内容。每15分钟可以为一个弹匣写上，而当这些子弹被使用时，可以无视目标除基础、闪避、洞察防御外任意5点防御。

  
弹性射击  
价格：D+1000  
描述：通过调整枪口的角度使得发射的子弹可以在空气中折射轨道来进一步越过障碍物攻击敌人的小技巧……  
效果：仅当你使用实弹射击的枪械类武器攻击时才能生效。  
在攻击时，你可以选择将射击轨道变更为曲线型，这意味着如果你拥有透视等能力，同时对方处于你的射程内而不处于全遮蔽状态下，你可以越过障碍物命中敌对目标，哪怕子弹的飞行轨道会异常离奇。

  
驱魔铳击  
价格：D+500  
描述：火药乃是至阳之物，因此铳击可以驱魔也是理所当然的事情，至少被轰炸过的坟地就从来没有闹过鬼。  
效果：你在使用枪械攻击时获得【法破】特性，此外还可以在针对灵体、虚体等正常物理攻击无法命中的单位，使本次攻击额外获得3DP器械加值。

  
快速开火  
价格：D+500  
描述：长年累月的射击让你仅凭身体本能便可完成动作。  
效果：你在使用枪械进行攻击时，获得C级先制效果，同时在进行处理你拔出武器的动作时，只需要一个迅捷动作就拔出随身携带的武器。

  
饱和攻势  
价格：C+1000  
动作：结合在任意一次攻击当中  
能耗：无  
射程：等同于武器射程  
限制：这个技能每轮只能使用一次。  
描述：在短时间内进行多次快速攻击，对多处目标或同一目标进行饱和式打击。  
效果：你仅能在使用枪械类武器时使用本技能。  
你进行一次枪械攻击，同时攻击落点5米内的所有敌人。  
如果对结果不满意的话，可以进行第二次攻击，但两次攻击中只有一次会造成伤害，且如果进行第二次攻击则必须接受这次的出目，目标用反射检定本次攻击。

  
购买饱和攻势不会获得自由属性。

压缩舰炮：剧情中这把武器可以使用自由动作切换武器化作原本的大小，全长约9米……理所当然，这需要足够的空间才能切换，计算你持有时依旧按照5体积计算，切换回5体积是一个自由动作。
