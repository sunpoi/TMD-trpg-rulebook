# 魔弹射手

技艺名称：魔弹射手  
价格：D+1500  
本质：特异本质  
描述：  
获得魔弹射手的能力必须先与恶魔萨米耶签订契约，萨米耶将为购买者铸弹七发，但第七发所射中的东西归恶魔所有  
特性：  
【守林人】【农夫】【被引诱者】【猎户】【少女】【隐士】【恶魔】  
效果：  
魔弹射手是一种通用射击技巧，这意味着它可以应用在任何一次带有射技的攻击上。  
当购买者发射第七发魔弹时，将由ST决定这枚魔弹的攻击对象。这项强化至多可以重复购买三次，重复购买带来的加值会正常累加，但“被引诱者”不会如此，而是改为重复购买额外获得1点自由属性点。

七枚魔弹按照顺序，分别名为：守林人、农夫、被引诱者、猎户、少女、隐士、恶魔

七枚魔弹每天都只能发射一次，在发射后立即消失，发射时，必须按照顺序进行发射，第七次发射魔弹必然只会发射恶魔。当午夜降临时，已发射的魔弹会重新回到购买者手中。

根据发射魔弹的不同，依次参考下列效果进行结算。

·守林人：重新分配此次攻击上的所有破防能力，但不能将一种类型的破防转化为一种此次攻击原本没有的破防类型（例：玩家“战车”使用了守林人魔弹，本次攻击具有5点高速和3点破甲，他可以重调整为1点高速和7点破甲，但不能调整为3点高速、2点破魔和3点破甲），并再额外获得等同于敏捷附加\*1点的威猛，这些威猛也可以被守林人的效果所调整。

·农夫：本次攻击额外获得1点攻击上的技艺加值、1点环境加值、1点幸运加值，若你已不再拥有任何完好，则这个数字提升至每项2点。

·被引诱者：本次攻击视为连射攻击，但无需消耗额外的子弹，若已经连射，则改为本次攻击将视为具有法破。

·猎户：本次攻击额外获得1点攻击上的附加成功，若在本场景中，你已杀死过敌人，则这个数字提升为2点。

·少女：若本场景中，你已发射过其他魔弹，则选择本场景中已发射过的一种魔弹，使用它的效果。若无，则你的下一颗魔弹额外获得等同于敏捷附加成功\*1点的攻击上的技艺加值。

·隐士：本次攻击至多还可以同时再攻击两个目标，若如此做，则本次攻击获得1点攻击上的表现加值。

·恶魔：这枚子弹的效果由ST决定，攻击目标同由ST决定。
