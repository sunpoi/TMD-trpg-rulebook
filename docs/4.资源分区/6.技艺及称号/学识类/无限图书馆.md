# 无限图书馆

无限图书馆  
资源类型：学识技艺  
本质：特异本质

概述：  
无限图书馆内拥有无限数量与无限种类的书籍，因此每个人都能在图书馆内找到符合自身需求的书籍。每个人都能在无限图书馆内花费资源购买或借阅书籍，但是由于版权等限制，所购买或借阅的书籍只能由购买者或借阅者观看，其余观看者只能看到是一本书。

无限图书馆不会为购买者提供属性。

物品名称：提升技能的书籍  
价格：500  
效果：  
看完能提升指定技能1级的书籍，重复观看并不能重复提升指定技能。

物品名称：提升专业的书籍  
价格：250  
效果：  
看完能提升指定专业1级的书籍，重复观看并不能重复提升指定专业。

物品名称：普通借阅卡  
价格：D+500  
效果：  
能在无限图书馆内借阅书籍，每过一影片就能在无限图书馆处借阅价值1000分数的书籍,前提是将之前的书籍还回去。
