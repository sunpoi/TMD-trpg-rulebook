# 魔法专长

  
魔法专长的价格为：1级1点、2级3点、3级6点、4级12点专长点  
  

能量爆发  
1级：击发  
你可以使用标准动作将纯粹的能量直接从手中爆发出来，该攻击视为一种无派系法术，法术成分为动作，伤害类型为纯能量，使用施法检定，其射程为专长等级\*5M。特殊的，发射能量是最基础的手段，可以匹配任何派系，你在购买该专长时若已经具有某个派系基础，则你可以将能量爆发改为对应派系的法术习得，此时改能力消耗的能量为任意一种通用能量，不可混合支付。  
2级：强烈  
你懂得了在爆发时使用技巧，令其产生更加剧烈的反应。能量爆发现在具有【法破】效果，但在使用时你也不可避免地会因为暴动的能量被轰退智力\*1M或者触碰到障碍而无法再向后移动，如果在使用这个攻击时你就已经处于完全无法向后移动的状态，那么你会受到1点不可豁免的剧痛点数，因为没有距离来削减能量爆发后坐力的影响，那么就会对你的身体产生伤害。  
  
能量掌握  
1级：破解  
对于能量的掌握让你善于寻找能量的漏洞，你的任何攻击均获得具有等同于专长的等级的【破魔】  
2级：律动  
你的眼睛现在能观察到空间中特殊能量的变化，你可以借此洞悉灵体的存在，获得【灵感视觉】，也能借助能量的轨迹追踪到敌人的存在，这是一种D级魔法本质的侦测效果。  
3级：辐射  
通过精妙的操控使用几乎可以忽略的能量即可将自己推进到空中，这是一种普通的飞行能力。在平时也能将自身的少量能量化为细微的粒子向周围辐射，根据返回的粒子如同雷达一般精确定位到蕴含对应能量的人或物，例如若你具有魔力池，你便能感知到具有魔力池的生物以及具有魔法效果的物品，具有真气池则能感应到同样修炼了真气的修士或者法宝，但能量粒子会被障碍物所阻挡。这视为一种基于敏感范围的强感知方式。  
  
派系理论  
1级：派系基础  
你从核心、自然、物理、黑暗、咒击、万变中选择一种派系，你掌握该派系最低级的造诣，当你获得一道更高级的对应派系法术时，自动掌握那个等级的派系造诣，但前提是你已经掌握了低一级的派系造诣。每回合一次，你在使用该派系法术时可以让该派系能耗下降专长等级点，但至少为1。  
2级：派系实操  
通过不断训练来提升法术的掌握，你的法术攻击获得【法破】特性。在一场影片结束时，若你将一道或多道属于上一级专长中选择的派系的法术进行了法术精熟，则你在使用该派系法术时永久获得1点基于自身的法强。  
3级：精益求精  
在各派法术共通的基础部分之后，各派系的特色便显现了出来，通过基础理论的学习你懂得如何根据各派系的特色发挥技巧。根据你在1级专长中选择的派系你获得对应的效果。  
核心派系：核心派系通过释放出大量爆发性能量以用于杀伤敌人，了解过基础理论的你知晓如何引发能量的链式反应，你使用的核心派系法术获得【湮灭】。  
自然派系：自然派系遵循自然规律，还原自然逻辑，借助自然的力量以进行施法，了解过基础理论的你知晓怎么做自然才会成为你的帮手，每轮一次，在使用自然派系法术时你可以额外进行一次投掷，取较高者，这表现为自然意志，大道，世界命运之类的规则用一丝力量对你的法术进行了修正。  
物理派系：物理派系将物理融入进法术之中，懂得基础理论的你知道如何将物理学应用于魔法，在施展物理派系魔法时你额外获得等同于专业：物理学的修行加值。  
黑暗派系：黑暗派系汲取黑暗能量，将其通过恶毒的解释方式进行使用，不择手段的你在了解了相应的性质后用敌人死亡时的绝望来补充力量，在一场同级及以上的战斗结束后你通过死亡时的绝望补充施法检定主属性附加成功数两倍的魔力。若你如此恢复了魔力，则你能通过一小时的冥想消化残余的杂质，这将为你永久提供1点修行加值，这一效果一影片只会生效一次。  
咒击派系：咒击法术是纯粹的法术表现形式，了解了基础理论的你对于这种纯粹相对简单的类型得心应手，这让你在使用咒击派系的法术进行的施法检定获得了8加骰。  
万变派系：万变法术的运行方式捉摸不透，形态千变万化，只能简单的总结出一定规律。了解基础理论的你能大致摸清万变法术的变化规律，你可以使你的万变派系法术朝三种方向改变性质：令万变派系法术从射程内的目标体内生成并造成伤害，这将会令此次攻击只能以强韧豁免对抗，或是令万变派系法术变得虚幻，如幻影一般穿透敌人的防御袭击目标，这将令此次攻击只能以反射豁免对抗，或是令万变派系法术变为幻觉，从灵魂层面袭击目标，这将令此次攻击只能以意志豁免对抗。  
  
触类旁通：若你将该专长升到了3级，你可以再次购买第3级专长，选择不同派系的效果获得，这同时会让你习得对应的派系造诣，这种方式获得派系造诣仍为最低级的派系造诣，当你获得一道更高级的派系法术时，自动掌握那个等级的派系造诣，但前提是你已经掌握了低一级的派系造诣。并且令1、2级的效果同样对该派系生效。但需要注意的是，3级专长只对对应派系的法术生效，所以一般来说多次购买第3级专长获得的效果不能在同一法术中生效，但通过奥法真理专长的派系兼修之类的能力使同一法术具有多个派系效果时，第3级专长可以同时生效。此外，你购买了3级专长的所有分支，则1、2级的效果还会对无派系法术生效。  
  
战地施法者（1-3）  
描述：亲临前线的经验让你知道如何在接身中稳固自身的法术优势。  
1级：循环施法  
效果：当你维持一个需要“专注”的法术时，你在三豁免检定上获得等同于该专长等级的专长加值。  
同时，每轮维持需要“专注”的法术时，你可以将那个法术所需的能耗降低1点，至多以此降低能耗为1。  
2级：熟练应对  
效果：当你维持一个需要“专注”的法术且处于敌人的触及攻击范围内对你造成威胁时，你视为获得2点洞察防御，威胁你的敌人对你发起攻击时，你额外获得洞察防御上的1点附加成功。  
3级：近外兼顾  
效果：只要你维持“专注”，2级专长中给予的洞察防御改为：获得自身传奇感知\*2点洞察防御  
同时，1级专长中给予的能耗降低改为降低5点，至多以此将能耗降低为1。
