# 隐形法球

名称：隐形法球  
类型：增强技  
价格：C+1000  
派系：无  
花费：4  
成分：动作  
动作：移动  
距离：自身为中心  
效果：5m范围内全体选定目标  
持续：一场景/15分钟  
效果：一道圆形的朦朦胧胧的球形力场笼罩内部所有目标，这使得所有目标在范围内具有C级魔幻本质的反侦测效果。
