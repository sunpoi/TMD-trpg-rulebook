# TMD隐者私营小学特供教材

名称：TMD隐者私营小学特供教材  
本质：魔幻本质  
概述：  
欢迎来到TMD主神空间隐者私营法师小学，希望你能在这里不要好好学习，多多留级，多交贵得要死的学费。顺便，本小学不教幼儿园课程，魔力池请自行学习。除此之外为了赚更多的钱，啊不是，为了服务更多法师，所以本学校也提供【东方版】，此时所有法术的消耗能量由魔力改为灵气，但请注意，这需要在购买时选择，且之后不可更换，毕竟我们隐者小学的一贯宗旨就是概不退换。

TMD隐者私营小学特供教材  
价格：750  
分类：奇物  
描述：  
由隐者经过层层转包后由小作坊印刷出来的教材，虽然纸质差，印刷不清晰，缺页漏字，但能用，反正老师讲的和教材上写的关系也不大。不过请放心，部分教材上才有的内容老师会故意不讲，所以教材还是有用的捏。  
效果：  
课程中的部分魔法需要参阅过特供教材才能掌握施展，但即使购买了多个这样魔法，只需要购买一份教材就够了，对此隐者表示非常痛心，因为少赚了不少教材费。

小学魔法课程  
效果：  
特殊的，你可以老老实实地一个一个地支付D+500购买以下法术，也可以选用TMD主神空间隐者私营法师小学辅导班才教的速成版本，若选择速成版本，第一次你需要用D+500购买一个法术，第二次你可以用D+500购买两个法术，第三次你可以用D+500购买三个法术，第四次可以用D+500获得剩余所有法术。但速成版本虽然能应付考试，但因为根基不牢，无法自由使用，战斗中你在同一轮只能使用一种你购买法术。每次购买只提供一个D级技艺的属性点。在购买相应法术后自动获得相应派系的D级法术造诣
