# 无支线魔法

  

御水术  
类型：增强技  
价格：250  
派系：自然  
花费：1  
成分：动作  
动作：标准  
射程：接触范围内一个目标  
效果：单体  
持续：一场景  
御水术在成为具有支线等级的魔法前不能享受派系带来的好处。对接触范围内的一个目标使用御水术，在接下来的一个场景中，令其可以在水面上如常行走，且可以站立在水面上。  
法术升阶：支付9XP，将御水术升级为D级，消耗的魔力额外提升1点。使用御水术还可以令其在水中游泳时能以全速进行移动，且可以在水中进行呼吸。
