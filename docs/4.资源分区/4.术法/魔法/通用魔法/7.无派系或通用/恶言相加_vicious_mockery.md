# 恶言相加｜Vicious Mockery

名称：恶言相加｜Vicious Mockery  
价格：250  
派系：无派系  
等级：无支线  
花费：1  
成分：言语  
动作：标准  
射程：15米  
目标：一个生物  
持续：立即  
对抗：无  
效果：  
你对施法距离内一个你能看见的生物连珠炮式的说出一串蕴涵魔力的咒骂（需要rp出来），并进行一次施法攻击。如果目标能听见你说话（不一定要听懂），则该次施法攻击对其造成精神伤害，该次攻击检定会因为你的辱骂程度获得获得1-3dp表现加值，具体加值多寡由ST判定，一般来说，具有实际内容的辱骂演出足以拿满加值（例如并非宽泛的垃圾话，而是针对对方的心理弱点打击）。
