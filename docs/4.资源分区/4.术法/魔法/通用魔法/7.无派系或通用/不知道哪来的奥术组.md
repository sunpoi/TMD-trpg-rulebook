# 不知道哪来的奥术组

不知道哪里来的奥术们  
阿蒙不知道从哪个大系列里找（偷）来的奥术们，虽然不算惊艳，但总归是能用。  
【注：这个奥术组的攻击检定为智力-学识：施法-洞察下属辅助专业：魔力感知】  
【注：这个奥术术组的购买价格无特殊说明，都是标准价格】  
【购买下列奥术的前提均为具有魔力池，购买下列奥术均会附带对应派系相同等级的派系造诣】

炼狱光束  
类型：攻击技  
等级：C  
派系：核心  
花费：6  
成分：姿态  
动作：标准动作  
射程：10+（传奇智力-1）\*15m  
目标：面前以自身为中心的，半径等同于射程/2的120°扇形内的所有敌人  
持续：立即  
汇聚能量，炽热的射线凝集，射线略过处，你的敌人被尽数扫除。这次攻击造成火焰伤害。

灼热皇冠  
类型：攻击技  
等级：B  
派系：咒击  
花费：10  
成分：语言（火焰啊，为我带上桂冠！）、动作  
动作：标准动作  
射程：智力\*5M  
目标：以自身为中心的，半径为射程的圆形内除自身外的所有目标  
持续：立即  
以魔力为引，操纵火焰，为自己编织这灼热的皇冠。这次攻击造成火焰伤害，目标以反射豁免。这火如此强烈，以至于连生命都能焚尽，因此本次攻击还会附带B级别重伤与2点不可豁免的燃烧点数。

跳动火焰/穿孔喷射/刀藤/雷电轨道/巨潮  
（当然，每种基础奥术是独立购买）  
类型：攻击技  
等级：D  
派系：咒击  
花费：2  
成分：姿态  
动作：标准动作  
射程：智力/m  
目标：单体  
持续：立即  
丢出跳动的火球/释放穿刺性的气流/释放像似带有刀片的树藤挥砍/引导短促的电流/发射水流组成的涟漪，这都是奥术的基础用法，好消息是这些法术可以连发，虽然命中率感人。这次攻击获得【连射】，1点附加成功，随后依据选择，获得3点能量加值/5高速/5威猛/5破魔/使对方流失3点任意能量。

闪避微风（伪）  
类型：防御技  
等级：D  
派系：无  
花费：2  
成分：姿态  
动作：移动动作  
射程：自身  
目标：自身  
持续：一场景  
传说，完整的闪避微风，可以让施法者只要保持移动，便无法被任何攻击击中，甚至可以凌空飞行！但很可惜，这不是完整版。在持续时间内，获得传奇敏捷\*2的闪避防御，以及4点抵消高速

马赫昏迷  
类型：攻击技  
等级：C  
派系：自然  
花费：6  
成分：姿态  
动作：标准动作  
射程：15m  
目标：单体  
持续：立即  
发出一连串的声波，使被击中的敌人陷入眩晕。这次攻击对方使用强韧豁免来抵消出目，攻击命中后对方以强韧豁免豁免对抗攻击出目，若失败则在强韧豁免上受到4点减值。随后，这次附带等同伤害的耳聋点数，并将这次攻击的伤害转化为为眩晕点数，分别进行强韧豁免。

剧毒流星  
类型：攻击技  
等级：B（特殊的，这个奥术需要b+2500购买）  
派系：黑暗  
花费：10  
成分：姿态  
动作：标准动作  
射程：60  
目标：单体  
持续：立即  
掷出由毒蔓组成的流星，紧紧缠住敌人。这次攻击对方改为使用反射豁免抵消出目，随后对方需以反射豁免对抗伤害出目，若失败，则陷入b级毒素来源定身效果。

螺旋地鸣（伪）  
类型：攻击技  
等级：C  
派系：物理  
花费：6  
成分：语言  
动作：标准动作  
射程：30m  
目标：单体  
持续：立即  
好吧，阿蒙又拿（偷）了一个不完整的过来。释放一条由岩石与泥土构成的地钻，打击并穿刺敌人！这次攻击具有贯穿、碾压特性，以及4点破甲。特殊的，当这4点破甲在破除对方防御后仍有剩余，则可以转化为攻击上的技艺加值。
