# 强力魅惑术

名称：强力魅惑术  
类型：攻击技  
价格：D+500  
派系：无  
花费：2魔力  
成分：动作  
动作：标准  
射程：15米  
效果：单体  
持续：立即  
效果：该法术有两种用法，施法前选择其一：  
①对范围内可视目标使用进行一次施法攻击检定，该次检定有额外1点附加成功同时不造成伤害而是转为等量的魅惑点数，正常豁免。  
②对范围内可视目标使用进行一次施法攻击检定，该次攻击额外获得4点内在加值并且转为精神伤害。
