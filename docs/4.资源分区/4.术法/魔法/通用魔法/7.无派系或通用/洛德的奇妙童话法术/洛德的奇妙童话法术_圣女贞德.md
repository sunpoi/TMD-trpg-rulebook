# 洛德的奇妙童话法术：圣女贞德

洛德的奇妙童话法术：圣女贞德  
价格：C+1000  
派系：咒击  
花费：6  
成分：动作  
动作：标准  
射程：20米  
效果：见下  
持续：立即  
描述：  
洛德评价：不可倾听那个自称能听到神的声音的没文化的村姑的诉说。  
效果：  
指定射程内的目标对其进行一次施法攻击，火焰从地下涌出，直冲天际。  
这个法术最多同时攻击3个目标。 此次攻击对目标造成【湮灭】性的火焰伤害打击。每有1点智力上的附加成功，这次攻击检定中获得2DP施法检定上的灼热能量加值，以及1点不因智力附加而增加的附加成功。

每场战斗中，从第二次施放这个法术以后，每施法一次，你都需要进行一次D10，出1则连带自己一同攻击。
