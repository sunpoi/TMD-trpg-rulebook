# 咒枣之术

咒枣之术  
类别：自然  
等级：C  
价格：C+1000  
消耗：3  
动作：标准  
成分：枣（红枣蜜枣，只要是枣类皆可）  
射程：无  
目标：见说明  
持续：见说明  
描述：  
修行者，当澄心洁己寂静端坐，叩齿三遍。存变法身是天仙，想元辰默朝金阙上帝俨赐仙丹。将枣七枚，排面前几案，“嘘、呵、呼、呬、吹”五炁于枣上，以手引弓水，书符于枣上，每枣书一勅八字，然后施咒。后与病人食之，即癒；与老者食之，焕神。  
效果：花费3分钟，将至多七枚枣变为仙丹，名为咒枣，咒枣必须在10分钟之内吞服，否则变为凡物。吞下咒枣者将治愈自身不高于B级的疾病和毒素效果；同时，获得持续一天的B级【免疫衰老】。这种【免疫衰老】不会怎加其寿命，也不会令其外貌改变重回年轻，其作用仅为使其不受因衰老导致的生理属性减值和力竭，精气神入青年一般旺盛，在具有以此获益的【免疫衰老】特性时，也不会承受疲乏点数与其严重后果的负面效果。  
特殊的，虽该术法视为自然派系，但在施术时不会获得自然派系的益处。  
研发：  
每日食枣三枚，如此坚持一个甲子，则可支付3点xp，便可使得该免疫衰老效果固化，即使暮年依然身体康健。
