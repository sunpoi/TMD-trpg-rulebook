# 梦芽

名称：梦芽  
分类：法术  
本质：魔幻本质

概述：  
在钢铁浇筑的丛林中，纯白无暇的建筑内。一墙之隔的人在徘徊踱步，在周围的白色注视下，一颗普通且平凡的芽慢慢诞生。  
你可以使用任意通用能量来施展梦芽魔法。  
只有购买萌生才能购买晨露、霜寒、燃火、余烬。

萌生  
价格：B+2000  
类型：被动  
描述：  
芽，无知无畏地诞生，好奇地看着一切。  
聆听你的言语与邀请，芽在你所期望的地方出现。  
效果：  
你可以用施法检定对施法检定主属性\*3M范围内的所有敌人进行一次攻击，造成穿刺伤害，作为一种基础的施法攻击方式，不消耗能量，可以结合在其他法术释放之中。

晨露  
B+2000  
类型：攻击技  
等级：B  
派系：自然  
花费：10  
成分：动作、语言  
动作：标准  
射程：施法检定主属性\*3M  
目标：圆形范围内所有敌人  
持续：一场景  
描述：  
诞生不久的芽啊，风雨被巨木遮挡，无拘无束，无忧无虑，晨露悬挂在芽尖，折射阳光，散发七色的光芒。  
效果：  
攻击技，使用标准动作，让施法检定主属性\*3M范围内泛起白雾，迅速凝聚为露水，显露着芽的稚嫩。以施法检定对范围内的敌人进行一次攻击，造成钝击伤害，他们需要意志豁免对抗此次伤害出目，若失败，他们会被露水沾染，陷入与新生的芽相同的稚嫩，这让他们在当前场景中陷入B级命运来源的B级潮湿状态，并且在当前场景中意志豁免承受你的施法检定主属性一半的减值。

霜寒  
价格：B+2000  
前提：只有在当前场景的上一次攻击使用晨露的情况下才能使用一次。  
类型：攻击技  
等级：B  
派系：自然  
花费：10  
成分：动作、语言  
动作：标准  
射程：施法检定主属性\*3M  
目标：圆形范围内所有敌人  
持续：一轮  
描述：  
蒙昧的芽啊，离开巨木的保护，惴惴不安地踏足天空之下，霜雪与寒风无情地让新生的芽伤痕累累。  
新生的芽啊，这是你的道路上必经的苦难，面对他，战胜他。  
落下无情的霜雪，踏入天空之下的芽，注定面临这样的困难，若芽在此冻结，命运便也到此为止  
效果：  
攻击技，使用标准动作，对施法检定主属性\*3M范围内，以施法检定敌人进行一次攻击，造成寒冰伤害，此次攻击出目变为1.5倍，附带伤害出目等量命运来源的冻结点数，意志豁免。若在豁免后因此承受了至少1点冻结点数（其他能力在此次攻击产生的冻结点数不计其中），无论冻结点数是否生效，这困顿难行的命运便已来到其面前，其在本轮中进行的下一次攻击检定需要额外投掷一次取劣计算，这是一个B级命运来源的效果。

燃火  
价格：B+2000  
前提：只有在当前场景的上一次攻击使用霜寒的情况下才能使用一次。  
类型：攻击技  
等级：B  
派系：自然  
花费：10  
成分：动作、语言  
动作：标准  
射程：施法检定主属性\*3M  
目标：圆形范围内所有敌人  
持续：一轮  
描述：  
跨越命运无情的寒风，芽，已经足够坚强，他熊熊燃烧，所有存在都不得不为之侧目  
复现坚韧的芽，熊熊燃烧，这炽热的火焰，正是芽熊熊燃烧的命运。  
效果：  
攻击技，使用标准动作，对施法检定主属性\*3M范围内的敌人以施法检定进行一次攻击，造成火焰伤害，此次攻击的出目变为1.5倍，且伤害转化为等量的燃烧点数，并立刻执行一次基于燃烧点数的激爆，若对方还具有冻结点数，则先结算激爆再转化为霜火。

余烬  
价格：A+4000  
前提：只有在当前场景的上一次攻击使用燃火的情况下才能使用一次。  
类型：增强技  
等级：A  
派系：自然  
花费：14  
成分：动作、语言  
动作：标准  
射程：特殊  
目标：圆形范围内所有友方（不含自己）  
持续：一轮  
描述：  
熊熊燃烧过的芽啊，做出一番光辉伟业的芽啊，终于，你漫长的人生来到了尽头，燃尽了……化作白色的灰……  
效果：  
以标准动作使用，如同燃尽一般，你的身体不断掉落白色的灰，直到你的下一轮开始，只要你没有陷入无助状态，范围内所有你以外的，其敏感范围内有你存在的你的友方单位，因为看到你燃烧直到变成灰烬的激情，在本轮中仅一次机会可以让一次攻击获得【传奇】、并造成韧性伤害。
