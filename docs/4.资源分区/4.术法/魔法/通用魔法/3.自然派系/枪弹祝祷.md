# 枪弹祝祷

【枪弹祝祷】  
c+1000  
增强技  
派系：自然  
成分：语言，动作  
动作：迅捷动作  
消耗：6  
描述：由新兴教国结合传统祝祷神术诞生的新兴神术，能够稳定安抚枪魂并使其短暂的能够全功率运行一段时间，当然，在事后也需要虔诚的对枪械进行维护与保养。枪魂大喜。  
闭上双眼，祈求枪弹女神的护佑，随后，祂将降临。  
效果：消耗一个迅捷动作，为手中的一个枪械赋予【枪弹祝祷】效果，该效果持续一场景  
此时，该枪械视为无限弹药，无需换弹，速射等级升级为连射  
当枪弹女神祝福你的时候，便不要在乎这些子弹是从哪里来的，也不要管它的这种构造是怎么射出这种弹雨。  
以及，在你泼洒弹药时，你可以额外对两个目标进行攻击  
最后，每轮一次，若你对攻击检定结果不满，你可以对枪魂祈祷，对该次攻击检定进行一次重投，但必须接受重投结果。
