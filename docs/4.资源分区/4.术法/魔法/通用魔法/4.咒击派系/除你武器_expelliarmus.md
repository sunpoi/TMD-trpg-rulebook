# 名称：除你武器 Expelliarmus

名称：除你武器 Expelliarmus  
价格：C+1000  
类型：攻击技  
等级：C  
派系：咒击  
消耗：6  
成分：语言，姿态  
动作：标准  
射程：15M  
目标：单体  
持续：一轮  
魔法界在现代决斗中常用的缴械魔咒  
效果：  
每轮仅可使用一次，以一个标准动作使用，对射程内的敌人进行一次施法攻击，该次施法攻击将附带击落武器的效果，且如果施法者的先攻高于目标，该次攻击将额外具有B级措手不及效果。
