# 故障机器人

故障机器人  
一具产生了自我意识的战斗型机器人。它装载的远古科技让它可以操控充能球。  
故障机器人有着以下三个充能球类型，分别具有被动和激发俩种效果。  
只有位于充能球栏位中的充能球，才会在描述节点触发被动效果。  
每个充能球栏位最多存放一个充能球，多个充能球栏位独立计算，互不干扰，一般来说，最多拥有8个充能球栏位。  
当充能球栏位中已有充能球时，额外获得充能球，可以选择替换或是保留任一充能球，随后没有位于充能球栏位的充能球直接消失。

  
【电击充能球】  
被动：在你的回合，结合在一次攻击中，在计算额外伤害环节时触发一次，对你本轮攻击的随机1名敌人造成1点雷击严重伤害。  
激发：不能主动激发，需要结合在资源提供的技艺中激发，消耗该充能球，结合在一次攻击中，在计算额外伤害环节时触发一次，对你本轮攻击的随机1名敌人造成1点雷击严重伤害，额外重复1次。

  
【冰霜充能球】  
被动：在回合开始时触发一次，你在本回合中获得1点力场防御。偶数回合时，额外获得1点抵破魔。  
激发：不能主动激发，需要结合在资源提供的技艺中激发，消耗该充能球，使在本回合中获得1点力场防御，以及1点基于力场防御的附加成功，每激发3个充能球，就会因为过热减少一点力场防御。

  
【黑暗充能球】  
被动：在回合开始时触发一次，使该充能球的计数+1。  
激发：不能主动激发，需要结合在本资源提供的技艺中激发，消耗该充能球，使这次攻击获得充能球计数\*1点DP，这是能量加值。

  
【集中】  
对充能球栏位起效，使充能球栏位中的充能球触发被动时，额外触发一次。

  
【等离子充能球】  
最特殊的充能球，它没有激发效果，不能也不需要放于充能球栏位中，在每轮开始时，每个等离子充能球触发一次被动，之后所有等离子充能球自动销毁。  
被动：在回合开始时触发一次，获得1点仅能用于释放故障机器人下属技艺的AP。

每场影片结束时，若你在该影片中使用过至少1次遗传算法，那么遗传算法生效时的格挡防御额外获得1点附加成功。
