# 资本游戏

资本游戏  
资源类型：魔导书  
本质：特异本质

价格：D+500  
描述：  
对我们来说，这只是一场打发时间的游戏。这是一本概念魔导书，需要展露时可以在拥有者面前以虚影显示。若具有资本家训练，则也可以在黑影手中显现。虚影表现为一本朴素的黑色真皮封面，只有几个烫金的字写着资本游戏的笔记本。

特性：【资本入局】【合法雇佣】【资本积累】【建立资本】  
效果：  
获得资本游戏下属能力购买权。  
【资本入局】：  
用你擅长的方式参与这场游戏。从现在开始你可以用操控+社交+专业：驱使+辅助专业：资本理论（学识下属）进行施法检定。

【合法雇佣】：  
你获得一个基础无派系攻击型资本魔法，在手中生成一张纸钞飞向操控\*1M内的一名敌人进行攻击，使用施法检定，造成腐蚀伤害。你只是通过合法雇佣就用工作侵蚀了他的身体，用金钱买下了他的健康。

【资本积累】：  
资本游戏中最重要的便是足够的资本。每过1场影片金钱游戏积累的资本力量就能为施法检定提供2器械加值。

【建立资本】：  
你开启能量池：财力  
关键属性：操控+智力

基础恢复方式：  
经过一次短休后，若短休期间持有或使用任何奢华物品，或拥有价值20分以上的现金，则恢复关键属性较高者的财力，否则恢复关键属性较低者的财力。

若在奢华的环境（奢华的床上或者奢华的房间或旁边有价值200分以上的奢侈品）进行长休，则在一次长休后恢复所有财力，否则需要再等待1小时财力才能完全恢复。

特殊的，若你拥有上流人士（2），则可以通过幻象满足奢华环境与奢华物品的条件。  
基础用法：每日一次，你可以支付1点财力，产生相当于当前世界5分的现金，因此生成的现金影片结束时无论如何都无法带出。

因为这个魔法的手段不以杀死为贱民为目的，所以【不具有黑暗派系的效果】。使用一些财力发出你进行炮制的“觉醒”的声音，对范围内的所有敌人进行一次攻击，造成音波伤害。之后他们需要意志豁免对抗此次攻击伤害出目，若失败，他们将陷入你特意炮制的幻觉，也就是所谓的，信息茧房中，他们相信自己的行为是被众人赞同的，他们相信反对他的人是邪恶的，他们变得固执，他们听不懂他人的声音，他们说出的声音也不被他人理解，于是他们无法协作，他们无法联合。这视为陷入S级幻象来源的沉默效果。随后，他们开始对立，他们开始敌视自己的同胞，肤色？性别？总之是因为你随便挑唆的理由，于是他们开始高举正义的大旗互相残杀，完全忽视了你和任何你不想让他们伤害的同伙，你可以随时炮制新的理念，以至于让他们心甘情愿地为正义的大旗，自杀。这是一个A级幻象来源支配效果。

类型：攻击技  
等级：S  
价格：S+8000  
派系：黑暗  
能耗：18  
成分：语言、动作  
动作：标准

效果：范围内全体敌人  
持续：一场景  
描述：  
聚敛整个国家的财富，从每个公民手中提取价值，提高必需品的价格，降低员工的相对工资，用舆论骗走他们手中的辛苦钱，让贱民，永远是贱民。  
效果：  
私人领地范围吹起一股萧瑟的风，浮现出都市的虚影，医院、公司、水电、房贷、学校……每一个建筑都有办法剥削，每一个建筑代表一种伤害性质，你选择其中任意一个方法来剥削财富，这意味着这次攻击的伤害性质可以在发动攻击时自由选择，此次攻击的成功数会增加一半，且在命中后，对方需要用意志豁免对抗此次攻击的实际伤害出目，若失败，这些敌人成为你的被剥削者，你的下回合开始时，这些目标的意志豁免承受等同于你的操控属性一半的士气减值，并且每过一轮，在你的回合开始这个减值就会再次增加等同于你的操控属性一半的数值，直到战斗结束或者其脱离你的私人领地范围，这是一个S级命运来源的效果。当前场景下，你获得操控属性一半的吸血，但只对被剥削者生效。理所当然的，这个效果不会叠加生效，毕竟你不能将一个已经是被剥削者的人转化成被剥削者，你要相信你榨干油水的能力。
