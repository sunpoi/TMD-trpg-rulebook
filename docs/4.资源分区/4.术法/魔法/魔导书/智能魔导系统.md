# 智能魔导系统

智能魔导系统  
资源类型：魔导书  
本质：科技本质

价格：D+500  
描述：  
魔法与科技，并不冲突。当人类用科技的力量飞向太空时，他们见到了外星的大魔导师们，这是一次历史性的接触，随着两个星球的往来，双方的知识互相交流，两个星球的生活就此改变。  
智能魔导系统，是一个手机大小的电子屏幕，平时以液态金属的形式隐藏在使用者体内，需要时会在使用者手上自动悬空显现。智能魔导系统始终通过液体金属触须与使用者身体连接，因此不能更换使用者。

效果：  
智能魔导系统不占据装备栏。  
获得智能魔导系统下属能力购买权。  
会根据使用者的能力调整出力，因此为使用者提供人物最高内在支线等级\*2的法强，因为你能借助智能魔导系统施法，所以你可以将施法检定变为智力+器用+专业：操作电脑+专业：精密操作

购买智能魔导系统后解锁旗下魔法的学习。消耗能量均为魔力。

在购买任意魔导程序后获得相应派系相应等级的派系掌握。

【维生编程】：  
根据不同人的体质维生程序应该进行相应的适应变化，优秀的现代软件应当重视用户的个人需求，在购买软件时从以下选项中选择一个获得，之后不可更改：

1>压缩程序：  
通过引导魔力的运动提升空间利用率，从而增强使用者存储魔力的能力，使用者的魔力能量池上限增加智力\*1

2>环绕程序：  
引导魔力环绕与使用者体表，这意味着使用者的施法检定获得智力上附加成功\*2的防御上的能量加值

3>应激程序：通过强行刺激神经系统与内分泌系统临时提升身体的速度，每回合使用者获得只能转换为反射动作，且不能转化为迅捷动作的2AP

4>生长程序：引导魔力作为刺激源活化身体的自愈能力，对较轻的伤势是足够的，使用者视为具有2快速治疗（医疗点24）

技艺名称：魔导战术软件  
价格：C+2500  
描述：  
这是一款警用战斗软件，主要用途是辅助使用者进行战场分析，但使用者需要根据战场状况适时切换战术软件的运行模式，要想利用好战术软件，必须有良好的决策能力。

效果：  
使用者每次回合开始时可以用一个自由动作让战术软件在以下三个模式中切换：  
1>镇压模式：  
用于应对大量混混团体等场景的模式，此模式下软件专注于分析敌人的攻击速度、攻击模式与攻击可能性，并将结果传递到使用者的视网膜上，辅助使用者以合适的顺序反应来袭的攻击，从而免疫多次攻击减值。此外使用者获得智力上的附加成功\*1的防御上的洞察加值

2>缉捕模式：用于追捕逃犯的模式，在进行一次消耗魔力的攻击后，若命中了目标，战术软件会专注于锁定这次攻击残留的魔力，意味着使用者对目标具有真实视觉，战术软件只能最终一个目标，因为魔力的不稳定性，一小时后战术软件便无法继续追踪，真实视觉效果消失。对于被锁定的目标，使用者进行攻击时获得智力上附加成功\*1的洞察加值

3>侦查模式：  
用于在战斗前观察目标区域人员流动情况等信息的模式，战术软件读取使用者的视觉信息，将其放大并进行AI修复后再反馈到使用者的视网膜，这相当于让视觉感知范围增加1倍，战术软件能够智能分析与处理视觉信息，从而消除浓雾等的干扰，这视为让视觉变为强感知方式

类型：攻击技  
等级：A

描述：  
零日漏洞或零日威胁是指软件开发者有零日或没有时间来修复的漏洞，因为恶意行为者已经可以使用这个漏洞来访问易受攻击的系统。  
效果：  
使用施法检定对工作区内的全部敌人进行一次攻击，首先选择此次魔导病毒的载体  
1>意识链接：  
通过精神直接发送破坏病毒，此时对方用意志豁免这次攻击

2>压缩硬盘：  
将病毒浓缩后直接发射，触碰到对方就会直接进行侵蚀，因此对方只能用反射豁免应对攻击

3>硬编入：  
直接在对方体表构建病毒进行入侵，无法规避，凭借肉体的防护可以阻止初期病毒的入侵并避免后面的影响，此时对方用强韧豁免应对攻击。

之后选择此次病毒破坏方式  
1>精准爆破：  
将破坏性集中于几处，虽然威力没有增大，但产生了更难以修复的破坏效果，此次攻击具有S级重伤效果。

2>完全感染：  
放弃破坏性，而是着眼于让对方思维过载毁灭对方的心灵状态，此次攻击附带等量的心灵来源的四档疲乏异常点数，且具有心灵来源的扼魂效果。

最后选择底层逻辑  
1>爆破 以能量的爆炸为主要方式，此次攻击造成纯能量伤害  
2>凝实 直接用魔力凝实的固体进行冲撞切割，此次攻击造成挥砍伤害  
3>过载 通过过载损耗产生热量，此次攻击造成火焰伤害
