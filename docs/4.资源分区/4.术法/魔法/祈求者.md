# 祈求者

  

祈求者

Quas，风暴的精华。Wex，暴风雪之心。Exort，凤凰的灵魂。在他的子民们分崩离析到世界的各个角落去之后，Kael被遗弃了，陪伴他的是他曾经创造的的大世界仅存的遗物。对复仇的渴望，让他将自己的灵魂出卖给了耐奥祖，交换来的是近乎无限的力量。披上古代血法师的斗篷，创世者只带着他支配的元素跨入了这场战争。他能够将元素熔合成致命的咒语，在狂怒中撕裂天空。他的天才无穷无尽，近卫军团将会体会到创世者的愤怒。

  
元素球  
D+500  
这个强化使你能够掌控三种不同的元素。初始三类元素等级都为0，第一次购买后，选择一类元素提升1级，之后每次购买该能力，都可以选择一类元素，使其提升1级。你最多可以将每类元素提升至7级。  
祈求者可以以一个自由动作令一个元素球出现在他的身边漂浮，他将享受这种元素球为他带来的效果，同种元素球不会多次提供效果，元素球出现的上限为3个，当希望出现第4个时，需使用一个花招动作切换一个已有的元素球。  
冰元素（Quas）：每级提升2点智力检定上的表现加值和1点智力上的附加成功。  
雷元素（Wex）：每级提升15米移动速度和2点闪避防御  
火球（Exort）：每级提升2点耐力检定上的表现加值和1点临时生命。  
当你的所有种类元素球都已经提升至7级时，你可以再额外支付4XP，此时，你即使同时释放了多颗相同的元素球，也可以同时享受它们带来的效果。（例：玩家星星释放了三颗冰元素球（QQQ），在他未解锁这个效果时，只能至多获得「提升2点智力检定上的表现加值和1点智力上的附加成功\*冰元素等级」，但在他解锁后，三颗冰元素球提供的能力将变为「提升2点智力检定上的表现加值和1点智力上的附加成功\*冰元素球等级\*3」）

  
奥术宝典  
B+4750  
一次性购买卡尔的27个法术，特殊的，它并不会为购买者提供属性点。  
同时，你掌握奥术宝典中所有所记载的派系的对应等级造诣，若你没有造诣基础，获得之。

  
融合技能：  
将冰球（Quas）、雷球（Wex）、火球（Exort）根据记载的顺序排列，可以得到一个待融合的技能，它不能被使用，只能在融合后使用，融合法术是一个自由动作。在融合一道法术后，那道法术的元素球组合提供的基础效果将会被固定至这道法术之中，只有在使用掉这道法术后，才能产生新的元素球。你至多只能同时拥有一道已经被融合的技能。  
在融合一个技能后，你拥有的元素球会消失，需重新释放。  
祈求者融合技能后，视为立即获得了一个组合出的法术，它们的前缀都是【召唤：】  
奥术宝典中记载的技能如无特殊说明，则一个战斗场景中只能使用一次。
