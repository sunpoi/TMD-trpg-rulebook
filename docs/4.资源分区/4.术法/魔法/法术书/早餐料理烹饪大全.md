# 早餐料理烹饪大全

早餐料理烹饪大全  
资源类型：法术书  
本质：魔幻本质

价格：D+500  
描述：  
这是一本神奇的法术书，当然不怕死的话也可以当作它是一本烹饪教程。  
奇怪的是书本封面和内容都没有留下作者的信息，可能只是一个无名小卒单纯誊写下某些疯狂存在的杰作。  
魔法书里记录了一些早餐法术和研发方式，人们难以想象这些法术是在什么情况下被创造出来的。

效果：  
早餐料理烹饪大全会为购买者在释放法术时，一轮一次，降低人物支线等级\*1点能耗，能耗最少为1。  
此等奇异法术必然会让敌人惊诧不已，你获得1枚先攻骰。

技艺名称：豆汁大肠汁  
类型：攻击技  
价格：D+500  
派系：黑暗  
能耗：2  
成分：动作  
动作：标准  
射程：20米  
范围：单体  
持续：立即  
描述：  
早餐法术中最为恶毒的法术之一，由一个患有异食癖的京爷所创。  
效果：  
召唤出人神共愤的不可名状之液体朝敌人攻击，刷新敌人的三观，这通常会让被攻击者与施法者不死不休。攻击附带等额的【恶心】点数，本次攻击具有9加骰。

一次研发10xp：  
效果：  
能耗改为6点，升级为C级法术豆汁大肠酱，更为浓稠更为惊世骇俗的攻击，令早餐脑袋们闻风丧胆，对于洁癖脑袋们会起到百倍的伤害，为了防止酱汁扩散而强迫自己不剧烈运动，攻击附带等额的【恶心】点数，攻击附带二分之一【精神束缚】点数，射程提升至30米。

二次研发20xp：  
效果：  
能耗改为10点，升级为B级法术浓缩豆汁大肠精华，早餐科技制造的超越认知的产物，只要闻闻就能熏死一头幼年巨噬蠕虫，攻击附带等额的【恶心】点数，攻击附带等额【精神束缚点数】，每轮一次，该攻击检定的成功数增加一半，因为过于浓稠，射程反而降低至20M。

技艺名称：跳跳糖小笼包  
类型：攻击技  
价格：D+500  
派系：万变  
能耗：2  
成分：动作  
动作：标准  
射程：15米  
范围：单体  
持续：立即  
描述：  
在身前召唤出一个五颜六色的小笼包，以迅雷不及掩耳之势飞出，造成力场伤害，因为是跳跳糖馅，被攻击的敌人会和小笼包产生剧烈的碰撞。  
效果：  
攻击视为拥有【余波】特性，本次攻击具有高速5。

第一次研发10xp：  
效果：  
能耗提升为6点，升级为C级法术整笼跳跳糖小笼包，富有的你召唤出整整一笼小笼包，一股脑全部丢出，造成立场伤害，让目标被小笼包压得喘不过气，毕竟一次点一笼小笼包实在是太奢侈了，攻击视为拥有【余波】【碾压】【高速5】，射程提升至20米。

第二次研发20xp：  
效果：  
能耗提升为10点，升级为B级法术进击的跳跳糖大笼包，量变引起质变，使用法术召唤多个巨大无比的大笼包沿直线重重砸下，如同巨人的脚步快速逼近，压迫感十足，就算是法棍也会被狠狠砸扁，攻击视为拥有【余波】【碾压】【高速5】，射程提升至30+传奇智力\*15米,攻击范围改为等于射程的直线。

技艺名称：玛莎拉生蚕蛹苦瓜领域  
类型：攻击技  
价格：C+1000  
派系：黑暗  
能耗：6  
成分：动作  
动作：标准  
射程：25米  
范围：以施法者为中心，以射程/2为半径的圆形范围  
目标：范围内所有敌人  
持续：一场景  
描述：  
对于战士来说，最绝望的无疑是身上黏黏糊糊还要进行激烈的战斗，施法释放玛莎拉生蚕蛹苦瓜领域，不可名状的糊状物将铺满四周，污染所看到的一切。  
效果：  
本次攻击的范围内视为一个C级环境来源的困难地形，该困难地形由意志豁免对抗，持续一个场景

一次研发20xp：  
效果：  
能耗改为10点，提升为B级法术玛莎拉生蚕蛹苦瓜修罗场，范围进一步扩大，范围升级为以施法者为中心，以射程为半径的圆形区域，再圣洁的战士也会在这充满玛莎拉生蚕蛹苦瓜的无间地狱中沉沦，攻击范围内的困难地形提升为B级环境来源，受困难地形影响的敌人，在攻击检定上具有【劣化1】，射程35米

技艺名称：加点猛料  
类型：增强技  
价格：D+500  
派系：无  
能耗：2  
成分：动作  
动作：移动  
射程：自身  
范围：单体  
持续：立即  
描述：  
通过往料理中从容，随意，自然地加入一些神奇，黑暗，恐怖的配料，从而让法术的冲击力，威力，味道？都更上一层楼。  
效果：  
使用后，你下一次释放的本法术书所记录的法术 ，射程增加20米，具有【熔毁5】
