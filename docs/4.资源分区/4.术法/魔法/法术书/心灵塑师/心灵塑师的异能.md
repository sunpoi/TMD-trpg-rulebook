# 心灵塑师的异能

  

反击（Counter）  
反击法术允许使用者去反击其他的法术，让它不能生效。这种法术必须在目标法术使用过程中，使用反射动作去调用一个可调用的标准动作以进行这次反击。如果一道法术被成功反击了，那么它的效果将完全无效化。一般而言，某个等级的反击法术只能反击最多等同于同级及以下等级的法术，并且只能反击检定时主专业为心智系、副专业为互动系的法术。进行一次反击使用智力-学识-专业：施法-辅助专业：反击（洞察）进行检定，若检定出目大于施法者检定出目，则成功反击，无效那道法术。

风暴（Storm）  
当你施放具有风暴特性的法术时，本场景内于它之前每施放过一个法术，便将此法术复制一次，在一道拥有风暴特性的法术使用过后，风暴计数清零。风暴计数会随着使用的法术等级提升而获得更多计（D1C2B4A8S16）

通联古咒X（Spilce onto Arcane）  
于施法者使用一道类别为古咒的法术时，可以再支付X点对应法术力，将一道非释放法术的，拥有通联古咒特性的法术效果加入那次施法中，释放一次古咒中能使用的通联古咒数量不限，但不可重复，会正常计算风暴计数

命悬一刻（Fateful hour）  
前提为施法者濒死状态时使用可以触发的效果

延缓X（Suspend）  
当某一道法术以延缓X的形式结算时，它将会在X个回合后被立即释放出来（而非行动轮），被释放时会正常结算它的效果，视作插入结算
