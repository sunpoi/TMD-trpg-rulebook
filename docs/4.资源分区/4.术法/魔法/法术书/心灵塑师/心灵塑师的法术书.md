# 心灵塑师的法术书

  

心灵塑师的法术书  
“人们的思考涌向了我，有时候我不知道究竟是他们在思考还是我在思考。”  
心灵塑师的法术书中记录着他研究所得的成果，心灵塑师的法术书中的每道法术都不会提供内在属性点

D+500  
特异本质  
解锁一个名为蓝色法术力的能量池，其上限为智力+决心，在一场战斗结束后会恢复至满，每日以此法恢复上限不能超过总上限三倍。特殊的，法术力能量池无法被以任何除属性提升外的方式提升上限，也无法被其他手段恢复。将所有下属于心灵塑师的法术射程提升至智力\*1米。 不同等级的法术各自有其使用上限，每级每天限用对应次数，根据其等级的提升，获得的更高等级的法术上限会越来越少（D6C4B2A1），通联古咒法术在通联古咒时不视为使用，书中记载的所有法术不会被任何其他效果而增加使用次数  
  
在未获得前三级专长前，无法购买任何其他法术内容

  
专长：  
汲法涡流（1）  
1级：当你使用一道由火花力驱动的法术时，可以改为消耗1点额外的该色法术力。  
魔能指针（2）  
2级：当你使用一道反击法术时，将无需满足法术中的动作与语言前提便可直接释放。  
流光重刻（3）  
3级：每轮一次，使用一道反击法术时，可以至多重掷一次那道法术的结果，无论结果如何，都必须接受。  
时灵道标（4）  
4级：获得仅能用于释放这道法术的2AP，只要你足以支付对应的标准动作。在使用一道反击法术时，可以每轮至多一次，重掷那道法术的结果，两次结果取高进行结算。  
灾祸匣（5）  
5级：一轮一次，当你成功反击一道法术时，可以立即对其进行一次攻击，这次攻击可以正常使用本法术书中记载的法术。  
毁灭挽歌（6）  
6级：当你使用一道C级及以上的反击法术时，获得仅能用于释放这道法术的8AP，但同一道反击法术在一轮内至多使用一次，所有的反击法术在使用时再上升出目一半的检定出目。

D级  
逆转宿命  
技能类型：应对技  
花费：1点蓝色法术力+1点火花力  
成分：动作  
施法：反射动作调用标准动作  
射程：智力\*2米  
范围：个体  
对抗：对抗对方  
效果：反击已被释放的一道法术，无视最多该法术的4点成功  
命悬一刻～改为无视12点成功  
「晚安。」  

  
阻滞（古咒）

技能类型：应对技  
花费：1蓝色法术力（或3蓝色法术力）  
成分：动作  
施法：反射动作调用标准动作（或无）  
射程：智力\*1米  
范围：个体  
对抗：对抗对方  
效果：反击一道已被释放的法术，在下一个你的回合开始前，视为拥有2点偏斜防御。  
通联古咒3～变为3点  
  

水面院秘咒（古咒）  
技能类型：攻击技  
花费：1蓝色法术力  
成分：动作  
施法：反射动作调用标准动作（或标准动作）  
射程：智力\*2米  
范围：个体  
对抗：意志豁免  
效果：指定一个目标，对其进行一次施法检定，那次检定获得3点额外的法强，对方使用意志豁免抵抗，受到二者差值的伤害，这道法术被通联古咒时，至多为每一道通联古咒减少1点法术力消耗

  
拖延  
技能类型：应对技  
花费：1蓝色法术力  
成分：动作  
施法：无需动作调用，但仍需调用标准动作  
射程：智力\*1米  
范围：个体  
对抗：对抗对方  
效果：反击一道已被释放的法术，若反击成功，则将其延缓3  

  
C级  
无路可逃  
技能类型：应对技  
花费：3点蓝色法术力  
成分：动作  
施法：反射动作调用标准动作  
射程：智力\*2米  
范围：个体  
对抗：对抗对方（及意志豁免）  
效果：反击已被释放的一道法术，这次检定额外获得6点法强，若成功反击，则其还需使用意志豁免对抗你的反击出目，若其对抗失败，则将其放逐至异空间内直至其回合结束，这视为一个B级心灵来源的放逐效果

  
智之钉  
技能类型：应对技  
花费：3点蓝色法术力  
成分：动作  
施法：反射动作调用标准动作  
射程：智力\*2米  
范围：个体  
对抗：对抗对方  
效果：反击已被释放的一道法术，对方可以选择流失30点其拥有的能量池以抵消反击法术的效果。若不选择那么做，则此次反击至多无视其4点附加成功并升级为长点射  

  
雾漫羽翼（古咒）  
技能类型：防御技  
花费：3点蓝色法术力  
成分：动作  
施法：反射动作调用标准动作（或无）  
射程：自身  
范围：无  
对抗：无  
效果：反射动作调用标准动作对自身使用，浓雾将弥漫在自身周围，浓雾将为自身提供15点偏斜防御，它拥有32点稳定值，你承受到的所有攻击将由稳定值承受，当其稳定值被完全击破时将自动散去。你也可以随时收回这道法术，浓雾将以每6s恢复3点稳定值的速度缓慢恢复稳定值  
通联古咒3  

  
B级  
躁乱风暴  
技能类型：应对技  
花费：5点蓝色法术力  
成分：动作  
施法：反射动作调用标准动作  
射程：智力\*40米  
范围：个体  
对抗：对抗对方  
效果：反击已被释放的一道法术，风暴  

  
无尽化身（古咒）  
技能类型：防御技  
花费：5点蓝色法术力  
成分：动作  
施法：反射动作调用标准动作  
射程：智力\*5米  
范围：以自身为中心半径为射程的圆  
对抗：意志豁免  
效果：反射动作调用标准动作对自身使用，立即产生无数与自身几乎一致，细节部分几乎无异，可自由行动且无实体的幻象。这些幻象将持续存在一个场景，当你首次在身边存在幻象时进行反击法术时，若那次反击法术成功，则对方需以意志豁免对抗那次反击法术的出目，若失败，则令其陷入B级心灵来源的混乱状态中，你周围的幻象立即全部消失  
通联古咒5  

  
A级  
卷时乱流  
技能类型：应对技  
花费：8蓝色法术力  
成分：动作  
施法：反射动作  
射程：智力\*100米  
范围：个体  
对抗：对抗对方（或意志豁免）  
效果：反击一道已被释放的法术（或选择一个目标），若成功反击（或对其进行一次施法检定），则对方使用意志豁免对抗意志出目。若对抗失败，则立即结束其回合，这视为一个S级的心灵来源支配效果，并在其回合结束时为自身开启一个独有的新回合，此回合获得14点AP，仍可正常使用任意本法术书中记载的法术  

  
扭时伟力（古咒）  
技能类型：应对技  
花费：8蓝色法术力  
成分：动作  
施法：反射动作  
射程：自身  
范围：无  
对抗：无  
效果：在受到一次攻击时，可以立即使用反射动作以应对，立即将自身回溯至本一轮开始时的状态（这并不代表会重置使用过的法术已占用的当日上限），这视为一个S级的心灵来源回溯效果，并在攻击者的回合结束时为自己开启一个独有的新回合，此回合获得16点AP，仍可正常使用任意本法术书中记载的法术      * * *
