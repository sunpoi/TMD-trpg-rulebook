# 怎样腌菜

怎样腌菜  
法术书  
D+500  
本质：魔幻本质  
封面上绘画出了上古灵植的幻影。每一片叶片都似蕴含着天地灵气，其上点缀着橙色的浓缩的灵阳印记，但书的名字是...怎样腌菜？  
怎样腌菜为你的施法提供了一些法术特质与一个特殊的法术。  
【人与时光与自然】：充满关于自然变化的知识让你进一步领会到了自然的真谛，你如果不具有自然派系造诣，则获得D级自然派系造诣。  
【乳酸】：乳酸菌发酵带来的酸味，应用在法术上便使法术拥有了一定腐蚀性的特质，你释放的法术具有【熔毁】2。  
【盐腌】：盐是洁净而又基础的物质，也是腌制的基础，你获得额外的一种法术增幅方式：消耗2点能量，使释放的法术具有更强的驱魔效果，提供2点破魔。  
【自时天成】：腌菜的方式还是要由自己掌握，与自然，时间，材料共同组成一场生命的共鸣曲，将精神投入其中，成为其中的一部分，你选择一个魔法副专业，包括其的检定获得等同于其人物内在支线等级\*2的增强检定加值。
