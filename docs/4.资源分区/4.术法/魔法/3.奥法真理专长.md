# 奥法真理专长

  

奥法真理专长

_无数法师文记口述之间，后来者才得以一窥奥法真理之究竟。_

奥法真理专长中记录的所有专长适用于大部分法术的释放和使用，获得2/3/4/5/6级奥法真理专长需要支付对应的6/12/24/80/96点经验值。并满足总条件与该专长前提条件。总条件为获得2/3/4/5/6级奥法真理专长至少掌握了1道法术精熟等级为传奇的D/C/B/A/S级法术。  

派系兼修（2）  
前提：同时学习两个派系  
当你使用一道被法术研发过的某派系法术时，你可以再额外支付等同于法术基础能耗数量的对应能量，至多一次可以再将一种你已掌握的派系效果加入这道法术之中。  
  

跑轰（2）  
跑动，聚能，瞄准，射击。  
当你使用一道动作不超过标准的，无需引导或持续引导的法术时，你可以将这道法术应用到一次移动过程中，这会正常消耗所有所需的动作。（例：玩家星星在一次移动中应用了经济过热，那么他可以在这次移动中使用它，正常消耗标准+移动动作）

魔力源泉（2）  
前提：拥有强化过的魔力池  
当你进行一次价值为10XP的魔导书研读/在法术书上学会一道D级法术时，提升3点魔力池上限。  
  

范围强化（3）  
前提：拥有一道法术精熟程度为传奇的线形法术  
你可以将符合前提条件的线形法术转变为等同于其射程两倍的扇形/锥形，也可以转化为该数值为直径的圆形（即：一道射程为100米的法术，在范围强化后会变为直径200米的锥形，或直径100米的圆形）。  
  

射程强化（3）  
前提：拥有一道法术精熟程度为传奇的，射程为主属性\*米的法术  
你可以将符合前提条件的那道法术射程增加三倍基础射程（即：一道射程为智力\*2米的法术，在射程强化后会变为智力\*5米）。  
  

自动化（4）  
前提：拥有A级物理派系造诣  
在使用一道法术时，【自动化】会赋予这道法术在使用前自动对施法范围内的潜在目标或目标进行定位。成功定位目标的这道法术在攻击时，会以预设的数值进行攻击。若未成功定位，则改为以范围形式进行打击，按照正常转化范围的形式进行转化。  
  

异变化（5）  
前提：拥有A级万变派系造诣  
一回合一次，在你使用一个法术并结算完毕后，在实际产生效果前你可以选择放弃这次检定，用一个消耗能量类型相同，消耗能量数不高于该法术的另一个法术对相同的目标（如果原本的法术为范围性效果，则另一个法术也应当为范围性法术），重新进行检定，这视为你在释放法术的工程中及时改变了能量的走向，紧急修改为另一种法术来应对敌人，因此无需支付第二次法术的能耗，但第一个法术的能耗必须支付。  
你也可以将这种变化的方式用另一种方式使用，但你一回合中只能从这两种方式里选择一种方式使用一次。  
另一种方式是，在你使用单体攻击且正常计算防御的法术时，你令法术的形态发生变化，分成两道形状不变但颜色发生改变的法术，一个是尽可能产生破坏力的黑色魔法，一个是尽可能保持稳定性的白色魔法，从左右两侧袭击目标，你需要在此时选择真正有用的是黑色与白色中的哪一道法术，另一个只是虚假的魔法，但是除了你没有人知道，对方需要进行一次选择，面对黑色的魔法还是白色的魔法，如果对方猜中了真实的法术，则该法术正常计算，只有对方猜错了，真实的魔法才能借此引发额外的效力，若真实的魔法是黑色的魔法，此次攻击造成韧性伤害，若真实的魔法是白色的魔法，此次攻击获得【传奇】特性。
