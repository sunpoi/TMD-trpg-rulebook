# 鹤憩水恋歌

鹤憩水恋歌  
资源类型：可升级血统  
本质：魔幻本质  
购买前提：购买时购买者的特征描述必须为少年/少女，或看起来像少年/少女。

描述：  
朱尼斯克莱恩（JeunesseCrane），也就是青春状态的仙鹤小姐，少女鹤。她既不存在于正常的历史中，也从未被任何人描绘过。仅仅只是在仙鹤小姐的心中留下的一爿面影。仙鹤小姐是源自民间传说的存在，传说「被人类救助的仙鹤，为了报恩而化作妙龄妇人的姿态出现」。因此，它的人类形态并不存在所谓的『少女时代』。  
那么，所谓的少女鹤究竟是什么呢？就结论来说，这是以承载了仙鹤小姐憧憬之心的衣装为依凭，并在U-■■■■■总统的力量作用下现界的幻想人格。从本不应做梦的事物中诞生的夏日之梦。

……那是哪个夏天的事来着？初见的大海。炽烈的阳光。咸咸的海风。潮汐来回的声响。还有年轻人们幸福的笑容。永不能再回还，却也永远无法忘怀，那耀眼如黄金的时光。  
――我没有过去。从未拥有过少年/少女的时光。  
――但是，如果说，若是能与人亲密无间地相伴，共同度过那个夏天的话。若是能有哪怕片刻，一起走过同样的青春的话。  
「想必，那一定是……  
　无比幸福的心境吧……」

D级  
价格：D+600  
描述：  
……今年的夏天，一定也会炎热如常吧。  
「那么，我出发了。」  
「嗯，一路顺风。」  
一位少年/少女脚步轻快地离去，身着泳装。在到达海边之前，就已经迫不及待了。

特性：【梦中雏鸟】【青春能量】【气息顺应】【雏鹤流-入门】  
效果：  
获得鹤憩水恋歌下属能力购买权  
【梦中雏鸟】：  
购买者的种族变为仙鹤。你的外表还是与常人无异，但是你可以以一个自由动作变形为一个体积比你当前体积小2，身着白色羽毛的丹顶鹤，从丹顶鹤变回人也是一个自由动作，这是一个D级变形来源的效果。这并不会让你拥有任何飞行能力，但是展开翅膀扑棱一下还是做得到的！

【青春能量】：  
所谓青春期，便是充满无限能量，肆意展翅翱翔的时期！你开启青春能量池，关键属性为风度+操控,这是一个通用能量池。  
青春能量的恢复方法：1、你可以经过一次完整的长休息后恢复所有能量。  
2、你可以经过一次短休息的冥想后恢复等于风度等级点能量。  
青春能量的基础用法：消耗一点青春能量，获得了等同于基础移动速度的水面速度，持续一轮。

【气息顺应】：  
毫无违和感地渗入对方心灵、让人觉得「他/她在这里是理所当然」的认知改写技能。对好感度在冷漠及以上的目标初见时才能使用，你可以用一个自由动作消耗1点青春能量，对某人声称自己是其过去的同学/朋友/邻居等点头之交的身份，并使用伪装检定与其的意志豁免对抗，如果胜出，对方会认为你是k以前认识的某人（哪怕想不起来也会装作想起来了），这是一个D级心灵来源的思维控制效果。因为是虚假的记忆，所以如果对方发现了你实际不存在的切实证据，这个思维控制会立刻失效。

【雏鹤流-入门】：  
为了应对夏日的一切状况，少女鹤努力修炼剑术，经过某位剑术大师的点拨后自创的流派。每经历一场影片并至少战胜了一名同等级的敌人后，你的敏捷检定获得1点修行加值。

C级  
价格：C+1200  
描述：  
准备万全，为了应对夏日的一切状况，泳装、制服、夏日特工服等衣装已经一字排开。  
――「夏日特工服是什么？ 是凭着『酷炫又帅气！』的气势准备出来的谜之服装。至于制服，当然是和你同一所学校的哦～」  
他/她是这么说的。

特性：【夏日特工模式！】【我的青春期距离感】  
效果：  
【夏日特工模式！】：  
伴随着「想被人看作冷酷角色！」而产生的变身能力。你可以以一个自由动作，变身并进入夏日特工模式，获得普通飞行能力。如果是在人形态，则会伴随着羽毛特效穿上一件黑白相间的紧身特工服。  
【我的青春期距离感】：  
戴上冷酷的假面，来掩饰内心的炽热情感，这就是青春期应有的恰当距离感哟。处于青春期的你对于人与人之间的距离非常敏感，使你在战斗时更容易找到更优势的距离，躲避敌人攻击，甚至保持这种距离感让弥补身体受到冲击产生的劣势。你现在获得基于闪避防御的1点附加成功，并且免疫冲击点数带来的减值。

B级  
价格：B+2400  
「暮色映照的大海，金色的原野，夕阳的尽头，潮风中歌唱的海鸟，也是终将吹散的微风。这夏日的一切，也即将落幕。但是！我一点都不觉得寂寞！」  
「……嗯，其实是说谎啦。心里还是有点，隐隐作痛呢。?到了晚上，一起去BBQ、放烟花吧。借着月光，在海边散步也很棒呢。」  
「是啊，是啊，我和你的回忆，我们的青春，才刚刚开始呢！」

特性：【永恒夏日领域】【雏鹤流-免许皆传】  
效果：  
【永恒夏日领域】：身周时常展开着的夏日与青春的领域，以阳之活力将一切包裹，就算夏天已经结束，夏日的热情也一点不会减！你获得仅能用于使用【轻灵】特性的武器进行攻击和本资源下属技艺的4ap。现在，你的【气息顺应】获得的思维控制效果提升至B级，并可以令其认为你是k曾经关系密切的某人，对于该思维控制获得一个新的使用方法：你现在使用一个自由动作可以花费1点青春能量将该思维控制结合在一次攻击中，对方以此次攻击伤害值为DC进行意志豁免，若你成功，则对方进入【夏日】状态。

【夏日状态】：  
你可以将自己在对方的记忆里的部分做添加或者模糊化，对方通常情况下会对此无法正确认知。列如，你可以让对方相信你是其交往多年的恋人，今天约定一起去海滩游泳约会。修改的内容必须和夏日与青春有关。  
对应的，目标一般来说无法正确认知其中不合理的部分，尽可能的在记忆中模糊掉这次与你遭遇时的正确情。这是一个B级别心灵来源的思维控制能力。

【雏鹤流-免许皆传】：  
经过不懈的努力 你终于将雏鹤流剑法练得炉火纯青，将飞行与攻击结合的淋漓尽致，并获得了祖师授予的免许皆传的资格（因为开山祖师=自己，所以是自己颁发给自己的）。当你持有具有【轻灵】特性的武器并处于【夏日特工模式！】时，飞行能力提升至完美，并且敏捷相关检定的加骰提升一级，至多8加骰。
