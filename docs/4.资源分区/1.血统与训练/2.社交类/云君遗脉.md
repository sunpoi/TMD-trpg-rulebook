# 云君遗脉

云君遗脉

神话本质

描述∶本血统来源于某一平行世界中的上古时期，此方世界的云梦泽大神「云中君」巡天时，见苍生受劫，遂以半身本源云雾化雨济世。雨雾滴落入地脉与濒死凡灵血脉相融。受泽者得其生云架雾之能，神性自此随血脉暗传。因此后世觉醒者皆称云君遗脉

D+600  
朝起见朝雾

    描述∶清流自四肢百骸悄然涌现，带着山巅初晨的凉意与深谷流风的灵动随着血脉在你的经脉间无声奔涌。最终，这涓涓细流汇聚于丹田气海，凝成一泓氤氲不散的云池，此乃【云岚气】初成之象  
    效果∶这是你觉醒能力的第一个阶段，特殊的能量在你的血脉中流淌，最终汇聚成池。你开启【云岚气】能量池，这种特殊的能量能够生成云雾，由作为能量主要来源的【风度】与进行控制的【操控】组成，开启能量之后你可以通过每小时进行一次【风度】检定恢复等同于出目的云岚气，但以此进行恢复的能量同样会进入每日恢复能量的上限，这是一个东方通用能量池

    描述∶心念微动，池中白气便如臂使指般透体而出。它并非蛮横的冲击，而是带着云雾特有的缥缈与莫测。气随心动，一缕凝练如丝的云雾便无声无息地激射而出，快逾疾风，携【贯云】之势直指目标。同时，每一次云气的离体，都伴随着对体内那泓云池精微入毫的感应与调动，这初窥门径的掌控之道，已让你对“云”的驾驭平添了几分浑然天成的熟稔  
    效果∶现在的你可以通过操控并释放出体内的云岚气制造云雾并以此进行攻击，你可以消耗一点云岚气对15M范围内的一个目标进行一次以【操控+专注+专注∶能量感知+学识∶气象学】为检定的攻击，名为【贯云】，造成穿刺伤害。因为你开始学习掌控云雾，你得到操控属性相关检定上的2点修行加值  
    云雾由水汽构成，水生万物，因你体内云岚气的滋养，你的寿命不会随着时间流逝，这是一个D级的不老效果。

C+1200  
万里无云万里阔  
      
    描述∶经年累月的吐纳修习，丹田那方云池已不复初成时的氤氲散漫。池中岚气凝若汞浆，流转间隐有潮声暗涌，透体而出的云雾不再是薄纱般的轻烟，而是翻涌如实质的流瀑。每一次催动云气，周遭空气便陡然湿润三分，仿佛置身深涧幽潭——这过盛的水德之气，已成了你攻伐中无声的利器。当云气如箭离弦贯入敌躯，磅礴水精便随势渗入肌理，寻常之人只觉经脉一寒，周身关节如坠冰窖般滞重难行，衣袍鬓发更在瞬息间凝满冷露，恍若被山雨猝然淋透。  
    效果∶这是因你长久使用云岚气而踏入的第二个阶段，此时因为你长久的练习，你体内的云岚气变得比过去更加凝实水汽充蕴，在视觉效果上也要比之前更加浓重。雾气中含量过高的水汽会使【贯云】击中后将对手的身体打湿，目标需要进行强韧豁免，失败则在三轮内陷入C级、环境来源的潮湿效果。  
并且你领悟了关于云岚气的使用方式，以下能力只能一轮选择一项能力生效，不能连续选择同一个能力  
云隐·在任意一次潜藏检定中使用，消耗一点云岚气，在本次检定中获得两点藏匿上的表现加值  
云踪·在任意一次移动中使用，消耗一点云岚气，在这次移动中获得额外的10M移动距离  
云刃·在任意一次攻击中使用，消耗一点云岚气，在这次攻击中获得两点操控上的士气加值

    描述∶抬首观云时，眸中映出天象流转的轨迹。层云舒卷、星月明晦皆成无字天书，五指虚按苍穹，便能触到大气中水精游丝的脉动。自此五日天机尽在指掌，心念微动间，可引十里流云蔽日，或催三更细雨敲窗。然，此乃顺势而为之术——晴转薄阴如抚琴转调，云化微雨似露坠青叶。若欲逆天时骤改气序，譬如炎阳当空忽降倾盆，则云池翻腾几欲溃散，终非当前境界所能企及  
    效果∶已经初步了解云雾的你可以通过观测天象得知近五天内的天气走向，并且通过控制大气中的云雾简单的改变今天的天气，如晴天换成阴天，多云开始下雨等，但改变前后的天气需相近，你不能凭空使原本酷热的天气突然下起瓢泼大雨

B+2400  
云深不知处  
      
    描述∶丹田云池奔涌如潮，至此方知何为御云登霄，心念未动，足下已有流云自生为阶，身后更幻化出磅礴气旋。无需捏诀观想，腾空之势如呼吸般自然——此乃血脉醒觉的天授之能。凌虚步天时身若飘羽，转折升降皆存乎一心，纵是青鸾振翅亦难及此等穿云透风的灵巧。若求更进一步，只需引云池怒涛贯入双翼，霎时流云染金，遁速之疾竟似快如雷电。攻伐之间，云气更显森寒彻骨。若将六缕精纯岚气注入指诀，击出云雾便裹挟寒露。中者不仅衣甲浸润，连经脉气血都似被水汽充斥。  
    效果∶到达这个阶段你才能算得上是掌控了云雾，现在的你可以通过将云雾构筑在脚下或者创造出云雾翅膀在身后等画风表现在外的方式获得飞行能力且机动性为完美，你可以通过一个自由动作展开，因为这是你血统中与生俱来的能力，若在飞行中你可以消耗3点云岚气强化云雾使这一次的移动距离提高30M，你也可以在攻击中消耗6点云岚气，调动更多的水汽使【贯云】附带的C级潮湿效果升级为B级潮湿效果。

    描述∶你双臂擎天骤引云池倾泻，以身为眼掀起滔天云瀑，浓白雾潮奔涌如活物吞噬天地，目力所及唯余混沌。凡陷云海者，如盲者探渊，云海外者观云海如见白龙盘踞地难以探清，然你立于云海中央时，浓雾竟如活物般缠绕庇主，外敌观你如雾里看花，云中毒瘴蚀骨之风亦难侵云君法体。然此境虽强，却需一息聚势，云海翻腾至巅峰时，方显雾中君临之相   
    效果∶随着你对云雾的操控愈加熟练，你可以消耗六点云岚气与一个标准动作以自身为中心释放出大量的云雾并让这些云雾覆盖半径为40米的圆形范围，持续时间为一场景。在此范围内白茫茫的雾气会阻碍视线并形成一个C级环境来源的浓雾环境，若你身处于浓雾区域之中，还会因为浓雾对你的遮掩得到C级自然本质的反侦测能力，同时你免疫C级的环境来源影响，但需要注意的是，浓雾只会出现在你释放后的下一轮，包括反侦测能力与免疫环境能力。

A+4800  
天接云涛连晓雾

    描述∶丹田云池已化天穹倒影，吐纳间天地水精皆如臂使指。至此境方知，昔日腾霄不过稚子学步。指捻雾丝轻弹，弦响处即有云锥裂空，云岚遮其瞳，雾气化蛟龙，缠筋锁骨如附骨之疽！  
    效果∶终于你踏入了这个阶段，在这个阶段内你对云雾的控制能力来到了无人能比的地步，你的通过消耗云岚气进行的攻击检定获得【9加骰】，并且因为云雾在不停的变化之中拥有了实质，被攻击的目标还会受到等同于伤害出目的【目眩】与【纠缠】异常点数，强韧豁免。因为你长久以来对云雾的控制，现在【贯云】的射程提高至自身的敏感范围。

    描述∶吐纳间神念舒张，方圆百里流云尽成延伸的指尖。昔年十五丈凡雾，今朝已扩作呼吸般自然的疆域，召来弥天云海时，寻常水汽早蜕为伴身神云。白雾障目之效虽如旧，然外力破雾者如蛮牛意图力破混沌，徒劳耳！你闲步于云雾中，俗目窥你则似蠡测沧海。  
    效果∶现在你所释放的云雾因体内蕴含的云岚气变得特殊，你通过消耗六点云岚气与一个标准动作释放出的云雾现在可以覆盖等同于自身敏感距离半径的圆形范围，并且来源从c级环境来源变为A级环境来源，但云雾产生的效果不会发生变化，与C级的浓雾环境一致。同样的，在浓雾环境产生之后你获得的C级免疫环境来源能力上升为A级，C级自然本质的反侦测能力上升至B级。  
你在浓雾区域中发起的攻击具有的加骰提高一级处理（上限8加骰）。无处不在的云雾似乎成为了你的皮肤，你获得基于云雾笼罩范围针对气体流动的非线性强感知能力
