# Auspice Grovefield之民

Auspice Grovefield之民  
资源分类：一次性血统  
本质：魔幻本质  
价格：1000  
特殊的，该资源不占据血统位

描述：  
强化该血统将不可避免的将你的国籍改为中国（或者类似震旦，塞里斯等国家），并且是本地人。  
在传说的黑龙之河的南面的幸运之森生存的居民，在严寒中生存的他们拥有积极热情的心态。

特性：【不冻之契】【热情心态】

效果：  
【不冻之契】：在严寒中前行需要穿着大量厚重的衣物，这使得他们锻炼出穿着厚衣服也能自由行动的能力，你穿着盔甲时减少2点盔甲减值。

【热情心态】：与他人的争吵在最后能够以互称家人来结束的热情心态，你承受非不可避免的情绪异常点数时，先将该异常点数减少1点（如果你在受到一次攻击中承受了多种情绪异常点数，则可以将每个情绪异常点数减少1点，再承受之。注意，不可豁免≠不可避免）。
