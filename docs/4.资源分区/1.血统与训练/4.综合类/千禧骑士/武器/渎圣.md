# 渎圣

渎圣  
D+500  
特异本质  
武器伤害6L 高速3 破甲1  
这是一把通体骨白，却晦暗斑驳的长弓，弓弦有三道之多，根本不适合挽弓搭箭。随着持有者的行动而逐渐亮起的弓身，呈现出明亮而苍白的色彩。  
“手拿弓箭象征着战无不胜，因为他的取胜不必依靠着血腥的斧砍刀刮。”  
“弓弦上没有箭矢，因为他的弓不必射而胜利自来。”  
使用者如常施展弓箭相关技艺与享受对应的好处，但在表现上使用本武器并不需要进行攻击，也不需要真正意义上的使用弓箭，因此这把武器还视为【无限子弹】一样处理箭矢数量。  
使用渎圣进行的所有攻击在表现上都视为被攻击的敌人自行用长钉洞穿自己的身体。其还会受到仅生效一轮的，等同于千禧骑士武器选择的属性附加\*1点的防御减值，自行选择减少的防御类型。

  
渎圣无需主动升级，只需购买本改造下渎圣专属的招式便会自动升级，购买招式本身视为升级武器，在计算评级时以武器处理而不是以招式处理。每个等级的招式不会提供属性点，而是提供标注的武器伤害。

  
渎圣专属招式  
特殊：每个等级的招式效果都可以和其他招式合并生效，但占据动作的效果只能有一个生效，除非你具有额外的动作满足效果，转化在造成的异常点数合计生效，且在画面表现上也只有一种会表现出来，由你自选。  
若无特殊说明，渎圣中所有需要涉及到持续时间的效果，都以持续一场景处理。  
若无特殊说明，渎圣每个招式提升的武器伤害为C8B7A12S17。

  
C+1500  
在攻击前消费一个花招动作，本次攻击使目标陷入【处刑】状态，附带等同攻击伤害出目（即计算防御之后，计算dr，全吸收，能量抗力等减伤之前）的特殊不良点数，强韧豁免，可叠加。  
若目标陷入【处刑】状态，则你在进行攻击时获得自身攻击主属性附加\*5点高速。

  
【处刑】  
这些特殊的不良点数作为一种一档不良点数计算。需要注意的是，在判断异常点数相关的能力时，该异常点数等同于疲乏点数，但该异常点数的来源由创伤来源改为命运来源。  

  
这次攻击的表现方式改为：土地中，空气中，甚至虚无中都伸出了越来越多的剑刃，这些剑刃上篆刻着亵渎和诅咒的文字，如同差分机一般相互组合成一个精美的自适应体积的艺术品刑具将目标囚禁其中，刺穿全身。如果目标因此而死，则头颅将会被斩下。

  
  
B+2000  
你使用标准动作进行一次攻击，攻击前使用主属性进行一次检定，获得等同成功数+2的攻击附加成功，这次检定只计算纯属性提供的数值与附加成功。  
在造成伤害的同时，目标以反射豁免检定进行对抗，失败则被击退差值/米的距离，这个效果每轮只会生效一次。  
这次攻击的表现方式改为：目标单位将会把自己钉在凭空出现像是圆环形状的滚轮上，轮盘如同人类濒临死亡时各种惊恐面容的镂空雕刻黄铜外壳，其中固定着一颗悬浮着不断跳动并喷出漆黑血液的嫣红心脏。

  
滚轮会连带着钉在上面的单位像反方向碾轧差值/米，在这个过程中滑轮、绞盘、绳索组成的绞索架会将其拉扯的四肢开裂，血肉模糊，寸骨尽断，这会使本次攻击附加等同伤害出目的肢体妨碍或麻痹点数，需要在购买时选择。造成的不良点数使用标准豁免，可叠加。  
在滚轮到达冲撞造成的极限距离或撞击到障碍物后，将会爆裂并溅射出大量的血液。

  
  
B+2000  
你使用整轮动作进行一次攻击，本次攻击获得3DP亵渎加值，同时视为接触攻击，附带等同攻击伤害出目的流血异常点数，标准豁免，可叠加。  
这次攻击的表现方式改为：目标自愿被拉入一个凭空出现的，用破旧齿轮和传动轴等物品拼接起来的，如同贵妇人一般的优雅女性魔像裙摆之下。  
然而掂起裙子，打开她的裙摆之时露出的确是数柄尖端镶嵌着沾满了暗红色血液的尖刺机械臂。  
如果目标因此而死，尸体会被完全粉碎。

  
  
A+4000  
武器获得【幻灭】特性。  
本次攻击附带等同攻击伤害出目的燃烧异常点数，标准豁免，可叠加。  
在攻击命中后，你获得【真实视觉】，同时选择一个被命中的目标，目标将自身倒挂在凭空出现的，刻满诅咒文字的黄铜十字架虚影之上（通常意味着要把肢体钉在上面），被烈火焚烧全身。

  
你可以同时攻击射程范围内的所有敌方单位，视为范围攻击。在场所有处于你射程范围内的敌方单位，在被攻击命中后，每轮都必须以意志对抗你的攻击出目。如果在对抗中没能取得成功数，则只能攻击被挂在十字架上的单位，而挂在十字架上的单位也只能攻击其阵营上的友方单位，这是A级命运来源的胁迫能力。

  
  
S+8000  
本招式需要以整轮动作进行攻击，攻击中将决心计入DP中，且此次攻击获得【传奇】特性，结算伤害直接造成韧性伤害。  
攻击中视为增加2400米结算触及，附带等同成功数的恐惧和沮丧异常点数，无需判定命中，直接进行标准豁免。  
在本轮内承受的第一次攻击将会被你直接免疫，这视为S级附身来源的无敌效果。  
攻击的表现形式改为：启动后，在范围内的时间的运转速度降低到接近停止（画风效果），但所有人的意识都处于正常状态，此时你宣告万事万物终结的到来。  
“我于此宣告”  
“当钟声敲响之时，世界将迎来他的终结。”  
“纷乱的色彩，污秽的灵魂，无果的祈祷，燃尽的希望。”  
“连毁灭本身也将消逝。”  
“总会失去 总会走向终点。”  
“永恒已失去其意义。”  
“所有的光都已经熄灭。”  
“赞美吧，歌颂吧，咏唱吧。”  
“呼喊吧，悲鸣吧，嘶吼吧。”  
“汝等命定之时已经来到。”  
巨大的中古机械挂钟出现并悬浮在你的背后，用无数半破损半锈蚀的齿轮拼合而成的巨大盘面上，三根长短不一的崭新的刻针转动着，三秒后即将重合于0点位置。  
当时针，分针，秒针重合于0点之时，厚重凝实的钟声将会响起。  
第一声钟响，触及范围内所有你指定的目标/敌对单位，立即以意志豁免对抗攻击成功数，失败将死去，这是一个S级心灵来源的即死效果效果，因此死去的单位会风化成为充满血腥味的铁锈色砂砾。  
第二声钟响，范围内所有你指定目标/敌对单位开始跑走马灯（濒死回忆）。这些目标以反射豁免对抗攻击成功数，失败将死去，这是一个S级时空来源的即死效果，若因此次死亡，会风化成为充满血腥味的铁锈色砂砾。  
第三声钟响，范围内所有你指定的目标/敌对单位以反射豁免对抗攻击成功数，失败将死去，这是一个S级命运来源的即死效果，且会风化成为充满血腥味的铁锈色砂砾。  
因本能力而死的单位不能通过任何方式复活，这视为S级诅咒来源的禁止复活效果。
