# 业-连续技

特别的，他人的脸、世界尽头与冷酷仙境与丰饶之海也视作本连续技的一部分，它们在这之中被称为阵痛-XX、泡沫-XX、迷惘-XX

若无特别声明，则连续技使用前提均为：在本轮或上一轮使用过【阵痛】、【泡沫】、【迷惘】中的一个，且等级不低于上一个使用的技艺。

所有连续技都至少需使用过他人的脸、世界尽头与冷酷仙境与丰饶之海中的一个作为起始技艺，而使用后直到使用一项对应后续连续技前，都不能使用除他人的脸、世界尽头与冷酷仙境与丰饶之海外的非对应分支连续技。

———【阵痛】——  
阵痛-第四间冰期  
C+1000  
乱风裹挟樱花而起，当有人分心之际突进，破敌一点

效果：  
以标准动作使用，往某一固定方向移动30m，并结合一次普通攻击在本次移动中。使用本曲目时视为具有A级【先制】。  
————  
阵痛-人性的证明  
C+1000  
借由乱花的笼盖，随乱风踏步挥出的一斩，随后遁入飘飞的花絮中

效果：  
以整轮动作使用，进行一次冲锋攻击，此次冲锋效果变更为曲线型，目标以反射豁免对抗，豁免失败则受到差值伤害。本次攻击后，你获得持续一轮的B级特异本质反侦测。  
————  
阵痛-长日将尽  
C+1000  
当舞者被纷落的乱花所遮，只需看客一时愣神，刀光便闪至面前

效果：  
以标准动作预备，在自身回合外，当任意角色进行一次移动或攻击时，可以反射动作调用已有的标准动作响应，往某一距离移动60m，并将一次普通攻击结合其中。使用本曲目时视为具有【打断性】。  
无论本技艺是否成功响应，只要以标准动作预备了，在下一回合开始时便视作了使用过本技艺。（即将AP转化为标准动作的行为存在，并宣言用于本技艺，且不进行其他用途）  
——【泡沫】——  
泡沫-海边的卡夫卡  
B+2000  
樱花纷落，舞者位居舞台中央。归鞘抬手之间，方想邀请他人共舞，那风便从远处袭来。其势锐利，有伤筋坏骨之力。

效果：  
以标准动作使用，对自身敏感范围内的敌人近战攻击一次，均以反射豁免对抗出目，豁免失败则受到差额伤害并被拖拽，位移距离等同于【差值出目+50m】，并在【强韧豁免】上获得【劣化2】，持续一轮。  
————  
泡沫-无限近似于透明的蓝  
B+2000  
然无一人能够应邀，故舞者清台送客。收手出鞘之时，那风便调转势头，将客于樱花一同冲散四处。其劲长久，能阻碍身形活动。

效果：  
以标准动作使用，对自身触及范围内的敌人近战攻击一次，并再衔接一次相同范围的普通攻击，均以强韧豁免对抗出目，失败则受到差额伤害并被冲撞推开，位移距离等同于【差值出目+50m】，并在【反射豁免】上获得【劣化2】，持续一轮。  
————  
泡沫-砂之器  
B+2000  
乱风过后，樱花漫天，却未被清出场地的上空。于是视其纷落...直至目光前方，将其与被裹挟其中的看客一同斩断。

效果：  
随后以标准动作预备，在自身回合外，当任意角色进行一次移动或攻击时，可以反射动作调用已有的标准动作响应，对处于自身敏感范围内的所有目标进行一次攻击。使用本曲目时视为具有【打断性】。  
无论本技艺是否成功响应，只要以标准动作预备了，在下一回合开始时便视作了使用过本技艺。（即将AP转化为标准动作的行为存在，并宣言用于本技艺，且不进行其他用途）  
——【迷惘】——  
迷惘-天人五衰  
A+4000

效果：  
以标准动作使用，对以自身为圆心的半径200m范围内所有敌人造成一次攻击，以意志豁免对抗出目，失败则受到差额伤害并陷入五感剥夺的状态之中（触觉、听觉、视觉、味觉、嗅觉），这视为一个A级环境来源的负面效果，持续一轮。  
————  
迷惘-饱食穷民  
A+4000

效果：  
以标准动作使用，对以自身为圆心的半径200m范围内所有敌人造成一次攻击，且本次攻击的最终出目变为1.5倍，并视作【接触攻击】，以意志豁免对抗出目，失败则被施加【魂牵】，这视为一个A级环境来源的负面效果。下一轮开始时，此范围内视为S级幻象来源的困难地形。以上效果皆只持续一轮。  
————  
迷惘-燃烧的绿树  
A+4000

效果：  
随后以标准动作预备，在自身回合外，当任意角色进行一次移动或攻击时，可以反射动作调用已有的标准动作响应，对目标进行一次攻击后再追加一次普通攻击。使用本曲目时视为具有【打断性】与S级【先制】，且在攻击检定上获得12DP和6点附加成功的技艺加值。  
无论本技艺是否成功响应，只要以标准动作预备了，在下一回合开始时便视作了使用过本技艺。（即将AP转化为标准动作的行为存在，并宣言用于本技艺，且不进行其他用途）
