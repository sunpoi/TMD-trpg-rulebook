# 训练家

名称：训练家  
本质：特异  
分类：可升级训练  
概述：这是一个来自宝可梦世界的特殊职业，尽管在主神空间内宝可梦通常被认为是召唤物，但训练家们依旧选择称宝可梦为伙伴。强化了训练家血统后，你将带着你的宝可梦们一起尝遍各个世界的美食，看遍奇异瑰丽的风景，在寒冷的夜晚抱着宝可梦一起入睡，在艰难的战斗中与它们同生共死！  
或许最终，“你想知道生命的意义吗”这一问题，会由宝可梦告诉你答案。

D级  
等级名称：初出茅庐的新人训练家  
价格：D+600  
特性：【指挥与作战】【我们并肩作战！】【我是来自真新镇的】

效果：  
【指挥与作战】  
当你强化了训练家后，你开启一个社交技能下的新专业：指挥。  
你可以在宝可梦使用某些技艺的时候将指挥专业作为辅助专业添加到宝可梦的攻击检定DP中。

当你强化了训练家血统后，你开启了一个宝可梦的新特性：亲密度。你每和你的宝可梦在一场战斗中并肩作战至结束，亲密度+15。若该场战斗难度较高（比如与同级别boss的战斗），亲密度+25。若该场战斗难度极高（越一个级别的战斗），亲密度+35。战斗难度适用于哪种条件由ST判定。  
某些宝可梦的技能伤害与亲密度有关，某些宝可梦的进化条件则需要亲密度达到某个标准后方能进化。

【我们并肩作战！】  
当你周围30米存在你的宝可梦时，只要你不进行主动攻击行为，场景内敌方单位的单体攻击会更倾向于击破你的宝可梦，这是一个D级幻象来源的类胁迫效果，当你训练家等级提升至C/B/A级时，这一效果的支线等级也会随之提升，但这并不会影响敌人使用他的范围类招式。  
这视为敌人认为你的宝可梦比你更具威胁。

【我是来自真新镇的】  
主神空间的训练家皆视为与小智同出一脉的真新镇，你拥有了真新镇居民那常人难以理解的身体素质。  
你的强韧，反射，意志豁免各+3dp修行加值。你的移动速度提升10米。

C级  
等级名称：闯荡天下的精英训练家  
价格：C+1200  
特性：【我们站在一起就是最强的！】【超级真新人】  
描述：  
“坚持住！耿鬼！”  
“加油啊！皮卡丘！”  
“快躲开！喷火龙！”

【我们站在一起就是最强的！】：当你的训练家血统提升至精英层阶后，你与宝可梦之间便拥有了坚不可摧的羁绊。  
【超级真新人】 ：在外冒险的日子里，你开始逐渐意识到你的身体似乎有些不同寻常。除了能举起令人瞠目结舌的重量外，还有惊人的体力和意志力。  
效果：  
【我们站在一起就是最强的！】  
当进入战斗后，若你与你的宝可梦之间的距离不超过30米，你的宝可梦便拥有防御上的4dp士气加值以及1点来自士气的附加成功。

【超级真新人】  
你可以免疫疲乏点数带来的减值及严重状态。

B级  
等级名称：坐镇一方的道馆馆主  
价格：B+2400  
特性：【双打战术大师】 【提升-我是来自真新镇的】  
描述：  
【双打战术大师】：你已经是一方道馆的馆主了，你的宝可梦战斗从此以后再也不是简单的你来我往互换伤害，你已经掌握并能熟练地将战术运用在宝可梦的战斗之中。  
【提升-我是来自真新镇的】 ：即便你并非真新镇的道馆馆主（真新镇没有道馆），你也依旧是真新镇的人。

效果：  
【双打战术大师】  
现在的你能够在一场战斗中同时与两个宝可梦建立【指挥状态】，即你在一回合中只消耗一个你的标准动作便可以指挥两只不同的宝可梦使用技艺，但这并不能使一只宝可梦一回合使用两个技能。

【提升-我是来自真新镇的】  
你的意志，反射，强韧豁免再获得3dp修行加值。  
你的移动速度再提升40米。

A级  
等级名称：传说中的训练家  
价格： A+4800  
特性：【坚定的背影】 【上吧！皮卡丘！】 【提升-我是来自真新镇的】 【提升-我们站在一起就是最强的！】  
描述：  
你是。。传说之人！  
你的冒险传说已经深入到每一个宝可梦训练家的心，你曾与你的宝可梦一起深入海底，遨游天际。你和宝可梦之间的羁绊被每一位训练家憧憬，他们都将你视为自己的偶像。

【提升-我是来自真新镇的】 ：你的真新魂在燃烧！ 现在的你已经完全觉醒了真新人的血脉。  
效果：  
【坚定的背影】  
每轮一次，当你的宝可梦受到攻击时，你可以使用一个反射动作来打断性的进行一次移动，移动结束时你可以进行一次格挡，若移动结束时宝可梦处于你的触及范围内，则你对这次攻击如同全遮蔽一般替代宝可梦接下所要受到的攻击。此外，你也可以消耗一个反射动作指挥一只宝可梦如本能力作用方式一般替另一只宝可梦挡下攻击。

【上吧！皮卡丘！】  
每轮一次，你可以令一只被你指挥的宝可梦的攻击检定的成功数增加一半。

【提升-我是来自真新镇的】  
你从本特性中获得的意志，反射，强韧豁免修行加值提升至风度/耐力属性取低者。  
你可以免疫纠缠与魅惑的异常点数带来的减值。

【提升-我们站在一起就是最强的！】  
取代之前的效果，当你和你的宝可梦站在不超过30米的范围内时，你与宝可梦的攻击与防御都会获得等于你操控属性点士气加值。
