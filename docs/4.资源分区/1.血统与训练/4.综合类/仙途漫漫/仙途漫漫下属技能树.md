# 仙途漫漫下属技能树

仙途漫漫下属技能树

名称：五行法术  
概述：操纵天地之间五行的基础法术。  
总纲：购买时你需要为五行法术绑定一个东方能量池，之后不能更改。

名称：御金咒  
价格：C+1000  
描述：道门基础的五行法术之一，以金行之力，令敌人手中的兵器脱手而出，为己所用。  
效果：  
主动：这个法术需要以一个标准动作施展，消耗6点真气，你可以选择道门法术施法射程中的一个角色，对其以道门法术施法攻击进行一次击落武器的特定部位攻击，这次击落武器你无需承受攻击特殊部位受到的减值。  
如果你击落武器成功，那么你还可以针对敌人的那件武器进行【夺权】，以此次攻击检定的伤害出目对抗物主的强韧豁免，高于对方（D18/C32/B44/A58/S80）点才能获得这一物品的所有权，这是一个B级别魔幻本质环境来源的【夺权】效果。并且你还能让那件武器立即朝你靠近30米，如果移动到了你的身前，你还可以立刻消费相应的动作，直接对武器进行握持。  
被动：只要你习得了这个法术，由于你对五行之一的掌握，你的道门法术的施法射程增加4米。

名称：生木法  
价格：C+1000  
描述：道门基础的五行法术之一，以木行之力，令藤蔓树木凭空生长，纠缠锁困，使敌人难以逃脱。  
动作：标准  
消耗：6点真气  
距离：道门法术施法射程  
效果：  
主动：这个法术需要以一个标准动作施展，消耗6点真气，你可以选择道门法术施法射程为半径的圆形中的所有敌方角色，对他们以道门法术施法攻击进行攻击，这次法术攻击所造成的伤害会全额转化为纠缠点数。并且在你释放生木法的这一轮内，整个生木法的影响范围内变为D级【困难地形】。  
被动：只要你习得了这个法术，由于你对五行之一的掌握，你的道门法术的施法射程增加4米。

名称：控水术  
价格：C+1000  
描述：道门基础的五行法术之一，以水行之力，令湍流奔涌而出，淹没敌人。  
动作：标准  
消耗：6点真气  
距离：道门法术施法射程  
范围：锥形范围  
效果：  
主动：这个法术需要以一个标准动作施展，消耗6点真气，你可以选择道门法术施法射程为半径的锥形范围中的所有敌方角色，对他们以道门法术施法攻击进行攻击。这次法术攻击所造成的伤害类型改为钝击，对方需以反射豁免对抗伤害出目，否则获得B级环境来源的【潮湿】效果。  
被动：只要你习得了这个法术，由于你对五行之一的掌握，你的道门法术的施法射程增加4米。

名称：引火诀  
价格：C+1000  
描述：道门基础的五行法术之一，以火行之力，令熊熊烈火燃起，焚烧敌人身躯。  
动作：标准  
消耗：6点真气  
距离：道门法术施法射程  
效果：  
主动：这个法术需要以一个标准动作施展，消耗6点真气，你可以选择道门法术施法射程中的一个角色，对其以道门法术施法攻击进行一次攻击。这次法术攻击所造成的伤害类型改为火焰伤害，并且额外在攻击上具有你的真气能量池等级\*3点能量加值和【熔毁7】。  
被动：只要你习得了这个法术，由于你对五行之一的掌握，你的道门法术的施法射程增加4米。

名称：搬土令  
价格：C+1000  
描述：道门基础的五行法术之一，以土行之力，令厚重的土块能够被你轻松搬至身前，阻挡攻势。  
动作：标准  
消耗：6点真气  
距离：道门法术施法射程  
效果：  
主动：这个法术需要以一个标准动作施展，消耗6点真气，你可以选择道门法术施法射程中的一个角色，对其以道门法术施法攻击进行一次攻击。这次法术攻击所造成的伤害类型改为钝击伤害。并且在进行这次攻击之后，你在这轮内额外在防御上具有真气能量池等级\*4点的遮蔽防御和10点反射豁免上的环境加值。  
被动：只要你习得了这个法术，由于你对五行之一的掌握，你的道门法术的施法射程增加4米。
