# 空想上的人格保持者下属技能树

空想上的人格保持者下属技能树

名称：表象「祖先托梦」  
价格：D+500  
分类：主动技艺  
动作：标准  
消耗：2点妖力  
射程：15M  
目标：单体敌人  
描述：有时梦中会见到祖宗托梦来忠告子孙的事，而这张符卡的效果就是让这一幕出现在现实中。  
就算是祖宗也不一定会忠告子孙。在梦中出现的列祖列宗们，其实是无意识中自己警戒的内心产生的假象而已。  
恋的这道弹幕正是让对方想起这些无意识中被镌刻在脑海中的景象。  
再说了，我经常看到我那些如假包换的祖先们站在我枕边，早见怪不怪了。

效果：  
发出高速的精准弹幕，对射程内一个敌方目标进行一次具有3DP速度加值和「高速5」的法术攻击检定，造成远程能量伤害，这次攻击视作具有【连射】

名称：表象「弹幕偏执症」  
价格：D+500  
分类：主动技艺  
动作：标准  
消耗：1点妖力  
射程：15M  
描述：使对手回想起连内心深处都没有记忆的景象，以令人难以置信的高密度弹幕使对手烦恼其中的符卡。  
总觉得这是一张很适合用来打那些长期被弹幕包围的家伙们的符卡。

效果：一次性发出大量弹幕，对射程内最多5个敌方目标进行一次法术攻击检定，这次攻击具有【法破】以及【破甲5】 。

名称：无意识「弹幕的墨迹测验」  
价格：D+500  
分类：主动技艺  
动作：标准  
消耗：2点妖力  
射程：15M  
描述： 这张符卡的弹幕的轨迹似乎因人而异，每一个人看到的弹幕轨迹形状都不一样。  
我问那实际上是什么形状，她说只是随便用了些看起来有内涵的形状而已  
我只觉得那看起来像豹脚蚊的殖民地。

效果：有规律的射击弹幕，对射程范围内的一个敌人进行一次法术攻击检定，敌人进行反射豁免，不造成伤害，而是改为造成成功数的精神束缚点数，标准豁免。

名称：本能「本我的解放」  
价格：C+1000  
分类：主动技艺  
动作：标准  
消耗：6点妖力  
射程：60M  
范围：圆形  
目标：范围内所有敌人  
描述： 符卡发动后，无数像爱心一样的玩意儿从无意识状态下的恋体内射出并漫天飞舞，让人死也不想被这弹幕击中。  
高速射出的爱心在飞过了一段距离后速度就会降低，而弹幕密度则会上升，不过弹幕并没有收缩。  
顺便一说，看起来她本人居然没有察觉到自己射出了弹幕，也没有注意到眼前的对手正在全力躲闪。  
虽然她摆出一副呆若木鸡的表情，但这家伙的脑子里到底装的是什么啊。

效果：将开放的精神最大化释放，  
任凭感情驱使向周围全方向投撒攻击，对范围内的所有敌人进行一次法术攻击检定对方以意志豁免，受到差值点精神严重伤害。

名称：抑制「Super Ego」（超我）  
价格：C+1000  
分类：主动技艺  
前提：上一轮使用【本能「本我的解放」】  
动作：标准  
消耗：6点妖力  
射程：60M  
范围：圆形  
目标：范围内所有敌人  
描述：并非将弹幕收缩，而是直接将对手四周散落着的爱心回收的符卡。这张符卡和「本我的解放」是一组举世罕见的符卡套装。  
如果没有「本我的解放」的话，「超我」就无法发动吗，或者说弹幕的密度会显得很稀薄吧  
虽然这时候的恋表情稍微有点认真了，但我果然还是搞不明白这家伙到底在想些什么。

效果：将弹幕收回，对范围内的所有敌人进行一次法术攻击检定，对方以反射豁免，受到差值点纯能量严重伤害。  
这次攻击获得9点纯能量加值.

名称：反应「妖怪测谎机」  
价格：C+1000  
分类：主动技艺  
动作：整轮  
射程：45M  
范围：锥形  
目标：攻击范围内所有敌人  
消耗：6点妖力  
描述：  
能探测出对手一丝一毫的动摇并将其以图表的形式表现出来的相当不可思议的符卡。  
测谎仪据说是一种能发现谎话的东西，应该是一种和没有说谎的人八竿子打不着的装置吧。  
我这么写，可是为了我的名誉着想，这种测谎仪的反应越大，弹幕越容易躲，这可不是因为我撒了谎啊。

效果：动摇就完蛋了，对面前锥形区域15M范围的敌人进行一次法术攻击检定对方以强韧豁免，受到差值的精神严重伤害  
攻击范围在一轮内视作C级影响心灵来源的困难地形，这次攻击具有+3DP能量加值。

名称：复燃「恋之埋火」  
价格：B+2000  
分类：主动技艺  
动作：标准  
射程：90M  
目标：单体敌人  
消耗：10点妖力  
描述：  
把炭火埋在灰烬之中。因为灰烬有着良好的隔热性能，同时也可以把炭火和空气隔绝开来，所以可以把炭火的能量保存相当长的一段时间。  
之后只要供给氧气，炭火就可以重新燃烧起来。  
古明地恋，紧闭的恋之瞳 。  
与灵梦、魔理沙相遇之后想更加多地知道她们的事情 。  
紧闭的第三只眼似乎变得柔和起来 。

效果：在使用这个技艺的时候，你的眼球睁开了一条缝，对一名敌人进行一次法术攻击检定，这次攻击附带等量的魅惑点数，标准豁免。  
这次攻击中，你可以进行一次重骰，并且选取更好的一次作为结果  
你的这次攻击获得【8加骰】如果已经拥有【8加骰】则变成+5DP+2附加成功

名称：「被厌恶者的哲学」  
价格：B+2000  
分类：主动技艺  
动作：标准  
消耗：10点妖力  
射程：90M  
范围：圆形  
目标：范围内所有敌人  
持续：三轮  
描述：只有在恋的周围剑拔弩张地散布着弹幕的特殊符卡。而且恋会进一步封闭内心，这时别说恋了，除了弹幕以外什么东西都看不见了  
她的理念是不认同身边的弹幕，因此恋才会遭到厌弃。也正因为她封闭了内心，死不认同弹幕，因此弹幕才会显得咄咄逼人。  
身为被厌弃之人，连弹幕都厌弃她。虽然弹幕是没有恶意的，但被厌弃者的哲学观使其认为自己被厌弃全是因为弹幕之故，当然不肯承认自己的过错了。  
此外，对于被卷入这个遭到厌弃的人的符卡弹幕中的我而言，这根本就是在给我添麻烦。

效果：对半径为圆形射程范围内的所有敌人进行一次攻击检定，对方以意志豁免，失败则意味着被击中者会标记自己为被厌恶者，  
此时其会开始攻击周围自己的队友，这是一个B级的心灵来源的支配效果。

名称：地底蔷薇  
价格：A+4000  
分类：主动技艺  
动作：标准  
消耗：14点妖力  
射程：决心\*20  
范围：自己为中心，射程为半径的圆  
目标：范围内所有敌人  
描述：扎心的弹幕。似乎每个人的心事都是很伤人的  
这么说也是呀。我的心事也是哟。要是这张照片  
写不成报道的话我会去找碴儿揍人的

效果：进行一次攻击检定，范围内的敌人以反射豁免，受到差值火焰严重伤害。  
这次检定成功数增加一半且伤害增加一倍，这次攻击附带等量燃烧点数，标准豁免。

名称：小石头帽子?  
价格：D+500?  
分类：轻甲  
盔甲防御：5/5  
特性：【防能量】  
描述：一顶系着打着蝴蝶结黄色缎带的黑色圆顶帽，是恋恋最好的朋友～  
效果：  
帽子即本体：?据说……恋恋的帽子才是她的本体，这是真的吗？  
恋恋十分珍惜她的帽子，虽然该物体样貌与头盔类似，但同时占用护甲位和头盔位，且按照轻甲计算，当你因为任何情况取下时，你会感到一股自内心而来的哀伤，不可避免的承受一点沮丧点数  
伴身的无意识：难道说没人注意到我？这样啊，这下就方便了。当穿戴这顶帽子的时候，你会获得恋恋部分无意识的能力，不过这部分能力相当微弱，仅仅只能让你在躲藏上更加自由而已当你处于潜行状态时，你的移动速度将不在减半计算。

差价升级：C+1000  
盔甲防御：10/10  
特性：【防能量】【防弹】  
伴身的无意识：帽子先生变得更加强大了，因此它继承了更多恋恋与无意识有关的能力，现在带上帽子先生后，其他人就更难注意到你了，除D级时获得能力外，你的潜行检定还额外获得了1dp的加成和1点附加成功  
自由的妖怪：幻想乡的大家都会飞呢，即使没有翅膀或者翅膀只是装饰，但是戴了一顶帽子就能够飞行了？当然可以，因为在幻想乡不能被常识所束缚?你获得飞行能力，机动性为普通 。

差价升级：B+2000  
盔甲防御：17/17  
特性：【防弹】【防能量】  
伴身的无意识：恋恋是会突然插进来玩家操作以外的动作的吓人一跳的角色，因此当你戴上帽子先生后，你的行为也会逐渐向恋恋靠拢，除去C级和D级的全部效果外，你额外获得了新的能力，每轮一次，你可以令某次检定投掷2次，然后d2选其1作为结果。?是会出现超越人类极限的超反应还是突然自己露出大破绽都取决于你的下意识  
“钢盔”：恋恋的朋友会在任何情况下保护着恋恋，虽然看着是一顶布制的帽子，但不知为何，受伤却会产生裂痕，就像一顶钢盔一样当穿戴帽子先生时，坚硬的防御将使你获得【坚韧】特性以及2点遮蔽防御，基于盔甲防御
