# Fangire下属技能

Fangire悠久的历史中，有着各种各样的魔法、战技、身体能力。

——————身体能力——————  
魔族的顶点  
资源类型：技艺  
资源价格：C+1000  
资源效果：Fangire作为魔族生物、十三魔族的顶点，有着比人类数倍以上的寿命，比较难因为自然的衰老死亡。这视为一种【不老】能力。身为Fangire被禁止与Fangire以外的低等种族恋爱与交配……但只要不让Queen发现应该没什么问题。  
作为十三魔族的顶点，fangire自然有着与其他魔族生物作战以征服对方的能力，例如十三魔族之中的幽灵族。因此你具有基于视觉的【灵感视觉】  
Fangire的食物是生命能量，除此以外的摄取并不重要，只要魔皇力池不空，你便无需饮食与睡眠。昼夜行动的Fangire自然也有着【黑暗视觉】  
十三魔族之一的人类尽管寿命与身体能力比其他种族差，但是擅长创造更多的知识与文化，如果疏忽大意地发动战争的话，很可能会产生毁灭整个地球的兵器，因此牙血鬼一族没有选择根绝，而是选择管理。伪造身份混入社会中暗中管理是Fangire对于人类一族的方法，因此进入影片后你会获得能正常使用的身份（地位对标平民），但是没有过往经历，如果去查会发现漏洞，仅附带能证明身份的物品（身份证，健康码），通常而言会具有一份足够生活一月的金钱 ，这是fangire一族的伪装，这视为D级信息来源的背景身份能力。

生命能量储存器官  
资源类型：技艺  
资源价格：C+1000  
资源效果：你的生命形态中有着能够储存生命能量的器官，这意味着你的生命能量不会溢散，因此你无需去摄取生命能量，仍然能正常通过短休与长休恢复魔皇力。由于你自身的生命能量恢复因此比溢散要快，所以你的魔皇力池，可以每小时一次，进行一次沉着属性检定，恢复成功数点能量，这会计入每日的恢复上限。在储存器官的帮助下你的体内自然能存储更多的魔皇力，你的魔皇力能量池上限增加耐力\*1的数值。通过存储生命能量能够延续生命，你具有B级衰老免疫。

牙血鬼顶级的骏足  
资源类型：技艺  
资源价格：B+2000  
资源效果：你拥有牙血鬼顶级的俊足，所以跑得很快，嗯，就这样。每轮一次，你可以以自由动作进行一次基础的移动行为，且你的移动速度增加120M

——————牙血鬼战技——————  
射击战法  
资源类型：技艺  
资源价格：D+500  
资源效果：  
枪械战斗：比起寻常的白刃武器，你与时俱进地更习惯使用枪械类型的武器进行战斗。你可以用敏捷+武技：牙血鬼战技+武学掌握来进行枪械攻击检定。若你为野兽纲，且在B级具有**通用形态能力**，则通用形态能力的描述从：你的肉搏攻击或白刃攻击命中的敌人需要以强韧豁免对抗此次伤害出目 改为 你的枪械攻击命中的敌人需要以强韧豁免对抗此次伤害出目。  
牙血鬼武器：你可以花费一个小时选择一把枪械作为你的牙血鬼武器，更换你的牙血鬼武器同样需要花费一小时，若如此做，则该武器变得可以由魔皇力转化弹药进行战斗，只要你的魔皇力池不空，则该枪械获得【无限子弹】，使用该武器进行攻击时额外具有2能量加值。

兵器战法  
资源类型：技艺  
资源价格：D+500  
资源效果：你使用白刃兵器进行战斗，你可以花费一个小时选择一把白刃武器（或选择你由fangire血统中的 分泌 或 战士等特性提供的白刃武器）作为你的牙血鬼武器，更换你的牙血鬼武器同样需要花费一小时。你的牙血鬼武器被魔皇力锐化，只要你的魔皇力池不空，使用其进行攻击时额外具有5破甲，且获得2能量加值。

肢体战法  
资源类型：技艺  
资源价格：D+500  
资源效果：你善于使用自己的肢体进行战斗，你可以花费一个小时选择一个肉搏武器（或选择你由fangire血统中的利刃等特性提供的肉搏武器）作为你的牙血鬼武器，更换你的牙血鬼武器同样需要花费一小时。只要你的魔皇力池不空，你使用牙血鬼武器进行攻击时额外具有1附加成功，且获得2能量加值。

身体组织  
资源类型：技艺  
资源价格：1000  
需求前置：拥有射击战法、兵器战法、肢体战法之一  
资源效果：若你的牙血鬼武器并非由fangire血统提供的武器，则你可以将其转化为你自身的身体组织结合在你的体内，你可以用迅捷动作将其从体内分泌出来出现在你的手中。

魔皇力浸润  
资源类型：技艺  
资源价格：D+500/C+1000/B+2000/A+4000 （差价升级）  
需求前置：由fangire血统提供的武器 或 身体组织 ，且你能购买的等级不超过你的牙血鬼血统等级。  
资源效果：你的牙血鬼武器由你自身的身体组织构建，这意味着你身体中蕴含的魔皇力越强则你的牙血鬼武器自然会相应地更强，始终为适配你的实力的武器。因此牙血鬼才通常使用自身身体组织构建的武器。  
根据你具有的战法与魔皇力浸润等级获得对应的效果：  
射击战法：使用牙血鬼武器进行攻击时获得2/3/9/2能量加值，A级额外提供7附加成功，使用牙血鬼武器进行攻击时高速提升5/10/15/30  
兵器战法：牙血鬼武器的武器伤害提升3L/6L/12L/24L，使用牙血鬼武器进行攻击时破甲提升1/2/3/6  
肢体战法：使用牙血鬼武器进行攻击时额外获得1/2/4/8附加成功，使用牙血鬼武器进行攻击时威猛提升2/3/7/14，因此获得的威猛若结算后仍有剩余则转化为攻击上的能量加值，至多转化2/3/7/14点。

——————牙血鬼魔法——————  
吸命牙  
资源类型：技艺  
资源价格：D+500  
资源效果：牙血鬼能通过光之獠牙摄取生命能量，自然，这可以作为一种攻击方式。你获得一种基础的施法攻击方式，施法检定的构成为：  
1>决心+学识：神秘学+魔力感知（属于洞察）  
2>沉着+武技：牙血鬼战技+魔力爆发（属于专注）  
3>智力+学识：施法+魔力感知（属于洞察）  
在购买时从以上三者选择一项，不得更改  
射程15M，造成纯能量伤害。吸命牙意味着外放魔力进行攻击的能力，fangire可以无需动作地将吸命牙结合在肉体接触与任意攻击中（不影响实际的检定构成），这意味着你和你的任何攻击方式具有【法破】。  
**\[活化\]**：每轮一次，你可以自由动作支付2魔皇力让吸命牙活化，掠取生命，令本轮中结合了吸命牙攻击具有【吸血1】。

吸血牙结合  
资源类型：技艺  
资源价格：C+1000/B+2000/A+4000 （差价升级，最高不能超过你的fangire血统等级）  
需求前置：吸命牙  
资源效果：将吸命牙结合在战斗中是牙血鬼的本能战斗方式，随着你自身变得强大，吸血牙的能力也会有所提升。因此你结合了吸命牙的攻击额外获得【吸血3】/【吸血4】/【吸血10】，且在命中时令对方流失任意能量3/5/10点  
吸命牙中的**\[\[活化\]改为**：每轮一次，你可以自由动作支付6/10/14魔皇力让吸命牙活化，令本轮中结合了吸命牙攻击额外具有【吸血1】/【吸血2】/【吸血3】，并额外获得2/3/4能量加值

唤醒本能  
资源类型：技艺  
资源价格：C+1000  
使用动作：标准动作  
消耗能量：6魔皇力  
影响范围：触及范围内单一目标  
资源效果：伸手触碰范围内一名目标，催动魔皇力唤醒对其血脉中的斗争本能。这是一个黑暗派系近距离攻击技法术，但不享受派系效果的好处。这个法术具有【接触攻击】性质，不造成伤害，而是造成等量诅咒来源的亢奋点数，强韧豁免。此外对方还需要用意志豁免对抗此次攻击出目，若失败则在当前场景中其因为被自身的斗争本能控制，将不分敌我地进行攻击，这是一个诅咒来源的C级混乱效果。

能量弹  
资源类型：技艺  
资源价格：C+1000  
使用动作：标准动作  
消耗能量：6魔皇力  
影响范围：35M内单一目标  
资源效果：直接发射能量弹进行攻击，这是一个黑暗派系远程攻击技法术。这个攻击本身就是一团能量，因此具有能量武器特性，但只能造成纯能量伤害。且此次攻击额外具有4法强。

——————纹章能力——————  
主教纹章：摧毁物质之力  
资源类型：技艺  
资源价格：A+4000  
需求前置：拥有Bishop纹章  
使用动作：标准动作  
消耗能量：14魔皇力  
影响范围：15M内单一目标  
资源效果：借助左手背的主教纹章发动摧毁物质的冲击波，这是一个黑暗派系远程攻击技法术，产生力场伤害，但不享受派系效果的好处。此攻击只对结构值物体或结构值生物生效。被冲击波攻击的物质必然被彻底摧毁，化为粉尘，这视为一个A级变形来源的即死效果。此次攻击造成韧性伤害，具有【湮灭】，这股力量直接作用于摧毁物质，仅靠坚硬无法阻拦这一击，因此忽略盔甲防御（A级变形来源）。且此次攻击额外具有12破甲

king纹章：红色冲击波  
资源类型：技艺  
资源价格：S+8000  
需求前置：拥有king纹章（真正的王 在此不被判定为king纹章拥有者）  
使用动作：标准动作  
消耗能量：14魔皇力  
影响范围：敏感范围内全部目标  
资源效果：king左手的手背与手掌2处有着纹章，可放出红色冲击波攻击。这是一个黑暗派系远程攻击技法术。此攻击具有极高的破坏力，此攻击获得沉着\*1+3的附加成功，具有【湮灭】，且可以额外重骰两次，取最高值
