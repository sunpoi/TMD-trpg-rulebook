# 时间流民下属技能树

时间流民下属技能树

武器名称：恒时权杖

价格：D+500

概述：这是一根两头镶嵌着钟表的长棍，在通常情况下，这根长棍两端的钟表一个正走，一个倒走，但此时都处于不走的状态。

武器分类：长棍  
武器伤害：7L  
基础特性：长柄，格挡1  
特性：【溯走格】

效果：  
解锁倒走表技艺购买权  
【溯走格】  
驱动恒时权杖需要注入【溯】，在注入【溯】时，会立即失去等量的【溯】，这视为由【溯】驱动的攻击。在注入【溯】后，正走表的大针会立刻前进等同于注入【溯】总量，并在接下来的每一轮中倒退一格，倒退时视为损失了【溯】。注入【溯】是一个自由动作，但每轮最多只能注入D2C6B10A14S18点【溯】，允许注入的总数受恒时权杖等级限制。战斗场景结束时，若还剩余未使用的【溯】，将会储存在恒时权杖中，正走表每小时倒退1格，倒走表立刻清零，每天结束时，若正走表还有数字，则清零。

在使用恒时权杖时，每次还需支付其中储存的至少2点【溯】，这次攻击若成功造成伤害，则还会给那个目标施加C级时空来源的衰老效果，这个效果将持续一个场景，并使倒走表前进2格。

武器名称：恒时权杖

价格：C+1000

武器分类：长棍  
购买前置：D级恒时权杖  
武器伤害：13L  
基础特性：长柄，格挡1  
特性：【溯走格】

效果：  
【溯走格】  
现在，D级效果改为支付其中储存的至少4点【溯】，这次攻击若成功造成伤害，则还会给那个目标施加B级时空来源的衰老效果，这个效果将持续一个场景，并使倒走表前进4格。  
现在，每场使用恒时权杖击杀过同级及以上的敌人影片结束时，使用者都会再获得15年寿命。

武器名称：恒时权杖

价格：B+2000

武器分类：长棍  
购买前置：C级恒时权杖  
武器伤害：20L  
基础特性：长柄，格挡1  
特性：【溯走格】

效果：  
【溯走格】  
现在，C级效果改为支付其中储存的至少8点【溯】，这次攻击若成功造成伤害，则还会给那个目标施加A级时空来源的衰老效果，这个效果将持续一个场景，并使倒走表前进8格。  
从【节约用时】、【今非昔比】、【弄时大师】中选择一项获得，剩余的两项可以每项花费B+2000作为技艺获得。  
·节约用时：当使用者保持老年状态（即总寿命低于1/2）时，节约用时将开始正常生效。使用者的10年寿命可以当作15年寿命来消耗，使用者每消耗15年寿命，就会自然获得5年寿命。这个效果将持续至脱离老年状态为止。

·今非昔比：当恒时权杖的正走表格数在24及以上时，今非昔比将开始正常生效。从此时开始，接下来的每一轮中使用者都无法再继续向恒时权杖中注入【溯】，并在接下来的每个回合开始时获得1AP，这些AP仅能使用恒时权杖提供的能力，每轮开始时，正走表将额外倒退等于目前获得的额外AP的格数，直至正走表中的【溯】消耗一空或获得的AP达到8点为止，在那之后，今非昔比停止生效，但每回合开始时仍然会保留提供的额外AP直至战斗场景结束。

·弄时大师：现在，使用者在使用恒时权杖时，每造成5点伤害还会使恒时权杖的倒走表前进一格，当恒时权杖的正走表格数和倒走表格数相等时，弄时大师将开始正常生效。现使用者将可以反从倒走表中汲取【溯】反哺自身，每轮一次，在一次使用恒时权杖进行攻击并成功造成伤害时，使用者可以立即使倒走表倒退6格，并使自身获得6点【溯】，不占用每天恢复上限。现在，使用者在使用恒时权杖的能力扣除寿命时，可以支付正走表中的至多2点【溯】，少支付5年寿命，每以此法少支付10点【溯】，就会获得5年寿命。这个效果持续至正走表和倒走表格数不同为止。

武器名称：恒时权杖

价格：A+4000

武器分类：长棍  
购买前置：B级恒时权杖  
武器伤害：32L+1  
基础特性：长柄，格挡1  
特性：【溯走格】

效果：  
【溯走格】  
现在，B级效果改为支付其中储存的至少16点【溯】，这次攻击若成功造成伤害，则还会给那个目标施加S级时空来源的衰老效果，这个效果将持续一个场景，并使倒走表前进16格。  
现在，使用者视为一直具有【终时护壁】的效果，无需再以招架触发，对于这次每次触发这个效果的攻击而言，被攻击者无法再自行选择吸收的伤害数，而是必须尽可能的吸收伤害，每受到1点伤害，就会损耗3年寿命，直至无法再吸收为止，在主动使用【终时护壁】时，将改为在其规定上限内每3年寿命吸收2点伤害，同时上限提升至45年。同时，使用者的血量将降低至1点，永远无法提升，永远无法降低血量上限，每因此降低1点血量上限，就会提升3点【溯】的能量上限，但是每第4点血量上限将不生效。

倒走表技艺：  
技艺名：跃时窃法  
价格：D+500/C+1000/B+2000/A+4000（逐级升级）  
分类：被动

效果：  
现在，将恒时权杖造成的衰老达到1/8的效果替换为【每轮一次，当其首次希望转化花招动作时，改为消耗2AP而非1AP，且你同时获得1AP，这些AP只能用于转化为花招动作。】，这个效果每轮只能生效一次。  
倒退2格倒走表进度，下一次使用【溯】免除伤害时，最多可以再多免除1点伤害，并少支付1点【溯】。  
当这项技艺升至C/B/A时，获得以下效果：  
·C级  
现在，将恒时权杖造成的衰老达到1/6的效果替换为  
【每轮一次，当其首次希望转化迅捷动作时，改为消耗4AP而非2AP，且你同时获得2AP，这些AP只能用于转化为迅捷动作。】，这个效果每轮只能生效一次。  
·B级  
现在，将恒时权杖造成的衰老达到1/4的效果替换为  
【每轮一次，当其首次希望转化移动动作时，改为消耗8AP而非4AP，且你同时获得4AP，这些AP只能用于转化为移动动作。】，这个效果每轮只能生效一次。  
·A级  
现在，将恒时权杖造成的衰老达到1/2的效果替换为  
【每轮一次，当其希望转化标准动作时，改为消耗16AP而非8AP，且你同时获得8AP，这些AP只能用于转化为标准动作。】，这个效果每轮只能生效一次。  
当这项技艺升至C/B/A时，改为获得以下效果：  
·C级  
倒退4格倒走表进度，下一次使用【溯】免除伤害时，最多可以再多免除3点伤害，并少支付3点【溯】。  
·B级  
倒退8格倒走表进度，下一次使用【溯】免除伤害时，最多可以再多免除6点伤害，并少支付6点【溯】。  
·A级  
倒退16格倒走表进度，下一次使用【溯】免除伤害时，最多可以再多免除12点伤害，并少支付12点【溯】。

技艺名：曲时步法  
价格：D+500  
动作：移动  
分类：移动技  
消耗：10年寿命  
目标：自己  
持续：一轮

效果：  
此技艺视为一种步法，在移动时使用曲时步法会被视为使用步法进行了一次移动，超越时间的多重身影会与使用者相重叠，使用这种步法会消耗使用者的大量寿命，并使倒走表倒退2格。  
使用曲时步法会消耗使用者的10年寿命，使用者的此次移动时，身后会留下无数处于叠加状态的身影，在使用者的此次移动停下后，这些身影将始终为使用者提供5点遮蔽防御，此次移动再提升15米移动距离，试图穿过这道影墙的目标在通过影墙时需计算此次移动剩余距离，并只能在穿过后移动一半的距离，这是一个D级时空来源的困难地形，同样的，冲锋攻击无法正常穿过影墙，此时它被视为一道足够长的障碍物，除非能够曲线冲锋，否则无法直接通过。以此法制造的影墙会持续至其下个回合开始时。

技艺名：终时护壁  
价格：C+1000  
动作：反射+移动  
分类：应对技  
消耗：2点储存在恒时权杖中的【溯】，并使倒走表倒退4格。  
目标：自己  
持续：立即

效果：  
使用终时护壁是一个类招架动作，这意味着使用者需先保留6AP至回合外，并在使用时转化出一个反射和一个移动动作进行招架。额外支付2点储存在恒时权杖中的【溯】，并使倒走表倒退4格。  
本次招架获得额外效果：对于这次攻击而言，额外再招架至多10点伤害，由被攻击者自行选择招架的伤害数，每招架1点伤害，就会损耗3年寿命，直至无法再额外招架或达到上限（即30年寿命）为止。
