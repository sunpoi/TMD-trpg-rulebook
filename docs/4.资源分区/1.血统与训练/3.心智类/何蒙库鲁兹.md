# 何蒙库鲁兹

何蒙库鲁兹（Homunculus）  
资源类型：可升级血统  
本质：魔幻本质  
概述：何蒙库鲁兹（Homunculus），即人造人，最早有关于其的记载出现在16世纪由帕拉塞尔苏斯所著的炼金术著作《De natura rerum》中。  
在特定的世界观下，制作人造人并不是非常困难的事情，制造出的人造人，被用于多种用途：战斗，研究，或仅仅是作为魔力电池消耗。  
理论上真正的完成体只要给予它成长所需的时间与资源，就可以无限自我进化，蕴含着无限的可能性的，真正实现了与人类相当——不，超越了人类的不可思议的存在。  
所以这个人造人，或者说这个“何蒙库鲁兹”，是并不存在的生命，是主神以众多人的理想，愿望与渴求为蓝图，创造出的的“完美的人造人”，作为血统提供给轮回者进行购买。

D级  
价格：D+600  
特性：【魔术回路】【天赋的睿智E】【人工生命体E】

描述：  
【魔术回路】：“所谓魔术回路就是魔术师体内制造魔力的源泉，而何蒙库鲁兹天然具备着在制作时赋予其的魔术回路，并且有着进一步优化的可能性。”  
【天赋的睿智E】：“象征知能水平的技能，等级越高，其拥有的知能就越强大。这一等级只有低限度的知能。”  
【人工生命体E】：“身为人造生命而言，何蒙库鲁兹毫无疑问是上位种族，即使这一等级只是略微优于寻常人类。”

效果：  
【魔术回路】：魔术回路本身视为魔力池的变体，在获得魔术回路后，你将获得等同自身（耐力+智力）\*1条魔术回路，每1条魔术回路可以在计算能量消耗时为你提供1点能量，因此获得的能量视为【魔力】。当使用过魔术回路对应的魔力后，那些魔术回路会暂时失效，视为你耗尽了其中蕴含的个人魔力。

恢复方式：  
1、经过一次完整的长休息后，恢复所有失效的魔术回路。  
2、经过一次短休息后，恢复关键属性（耐力/智力）较高者\*1条失效的魔术回路。

无论如何，每日因上述恢复方式获得的魔力不会超过魔术回路数量的三倍。

【天赋的睿智E】：你在智力相关检定上获得2DP内在加值；每经历一场影片，加值就会再提升1DP。

【人工生命体E】：你从下述效果选择两项获得：  
\*你的外貌提升1点。  
\*你获得黑暗视觉。  
\*你无需饮食。  
\*你无需休息。  
\*你可以在全环境下呼吸。  
\*你的伤势不会自然恶化，这一项需要选择两次才能获得。

此外，你的种族改为【何蒙库鲁兹】，视为一种类人生命体，也就是人造人，由人类之手而创造的生命。

C级  
价格：C+1200  
特性：【瓶中小人】  
描述：  
【瓶中小人】：满足一定条件的话会发生进化，象征着可以通过培养使其得到进化与改变的能力。  
培养它的花费是巨大的，不过，从结果来看，这样的花费无疑是值得的。  
【人智再演】：本质上是作为瓶中之人所得到的未知之智，可以得到连原本的制造者都不清楚的知识，或许正是有限度的触碰所谓“真理”的证明。  
【塔克温】：炼金术士们相信，真正培养成功的何蒙库鲁兹能够拥有超越人智的知能，甚至比他的制作者们更接近真理，虽然人工制造这样生物的难度远大于他们想像。也有一小部分人认为，何蒙库鲁兹只不过是工具，通过令何蒙库鲁的智慧超越自身进而更接近真理，是不现实也不必要的行为。

效果：  
【瓶中小人】：从你获得购买血统C级后开始计算，每经历一场影片，你就可以下述不同项目中选择一项，最多可以选择三次，部分项目可以重复选择更多次（见下）。

\*获得【天赋的睿智D】，选择第二次上升至【天赋的睿智C】。  
\*获得【人工生命体D】，选择第二次上升至【人工生命体C】。  
\*获得【情感模块E】，选择第二次上升至【情感模块D】。选择第三次上升至【情感模块C】。  
\*获得【人智再演E】，选择第二次上升至【人智再演D】。选择第三次上升至【人智再演C】。

——————————  
【天赋的睿智】：当你获得【天赋的睿智D】时，你在每影片结束后在智力相关检定上获得1个额外附加成功数，但失去该血统D级提供的每影片结束后获得内在加值都能力（保留已获得的DP加值）；当你获得【天赋的睿智C】时，除了具有上述【天赋的睿智D】提供的效果外，你还会在智力相关检定上获得【9加骰】。

【人工生命体】：当你获得【人工生命体D】时，你一次性获得在该血统D级中没有选择的那些项目，同时自选一项属性在对应属性相关检定上获得1DP内在加值；当你获得【人工生命体C】时，你不会因为生命值被严重伤害填满而昏迷。

【情感模块】：E、D、C级均不会提供任何效果。

【人智再演】：当你获得【人智再演E】时，你可以从FATE能力组中选择一项价格不超过D+500分，且不会占据血统位的技艺获得；当你获得【人智再演D】时，若在E级时选择的技艺是逐升技能，则可以将其升级至C级（前提是那项技艺C级的价格不超过C+1000分）；当你获得【人智再演C】时，你需要额外支付3XP，从而获得【宝】属性的效果。

【宝】属性：宝属性是一个特殊计数；你在心智类技能上每有一个因技能等级提供的附加成功数，宝属性的计数+1；你在智力相关检定上获得等同宝属性计数的修行加值（例如：宝属性计数2，就能在上述提及的检定中获得2DP修行加值）。

若你处于型月世界观下，宝属性每1点还会视为5点宝具属性，宝具属性不会提供额外效果，只是用来衡量这一能力作为宝具的等级（虽然ST可能因此为你提供一些额外效果），并且使该能力名称中的等级获得相应变动。

宝具属性10＝E级宝具。20＝D级宝具。40＝C级宝具。60＝B级宝具。80＝A级宝具。100+＝EX级宝具。

————————————  
选择三次并再经历一次影片后，若满足对应前置，则可以获得【塔克温】或【人造人】的效果。若均不满足，则你获得【不死性】。

【不死性】：当你死亡时，你的遗体视为具有等同你生前意志力上限的结构值，若在一个月内遗体的结构值没有被清空，你将会复活，这是一个B级附身来源的复活能力。

【塔克温】：  
获得前置：天赋的睿智C或更高级  
效果：你因天赋的睿智而获得的智力相关检定上的【9加骰】改为【8加骰】；你总是知晓敏感范围内是否存在敌意单位。你总是可以知晓剧本内一个名词指向的目标的大致方位（在某个方向、某个地区），但时间上会有误差，这是一个C级命运来源的【预知】能力。

【人造人】  
获得前置：情感模块C或更高级  
基本上这项会是专注于作为“人”感性的特化，作为一定程度上的“起源之人”试着接入常人的思考。  
效果：你在生理属性/心智属性/社交属性合计每有4点传奇属性，就在全检定上获得1DP表现加值。

B级  
价格：B+2400  
特性：【炼金术之业】  
描述：“水30公升，碳20千克，氨4公升……以及其余15种元素各少许。这就是人体构成的元素表，试试用这些拼出人来吧，还有男人的肋骨、泥巴。将这些制造的过程和材料省却的就是繁衍。”

“如果通过公开制作的方法来消灭制造何蒙库鲁兹的魔术的话，那这个制作方法一定现在还广为人知也就是说一定非常易懂好做。”  
效果：  
【炼金术之业】：在进入影片后，你可以选择在任意一个触及距离的位置属于自己的【神秘性载体】（通常外观类似于瓶子），【神秘性载体】，具有等同你生命值的结构值，此外如同B级物品一样处理，【神秘性载体】同时只能存在一个，这一效果只能用来复活而无法进行其他作用。

友方单位无论是否能接触灵体，始终可以接触到你的【神秘性载体】。若其能接触到你的【神秘性载体】，则可以花费长达一周的时间制作何蒙库鲁兹，这种制作不需要进行检定也没有材料限制，在制作完成后将你的【神秘性载体】放置于何蒙库鲁兹的心脏处，你就会立刻复活，这是一个B级附身来源的复活效果。  
若期间你的【神秘性载体】被破坏，则复活失败。

若你具有C级血统中提及的【不死性】效果，则不会获得这一效果，而是将【不死性】的复活所需时间缩短至一周，并在强韧豁免上获得【8加骰】，具有基于天生防御的【适应】特性。

A级  
价格：A+4800  
特性：【触碰根源】  
描述：所谓根源，是所有现象、因果关系的起点和终点，包含宇宙的终极知识与真理，接触根源者通常被世界规则消除存在，然而仍有例外。作为天然、且有限接触根源的产物——何蒙库鲁兹因而有着强大潜能，这份潜能即使是有限度的运用，也依旧是常人无法企及的力量。  
效果：  
【触碰根源】：若你因之前等级的血统而具有【天赋的睿智】或【人工生命体】或【情感模块】或【人智再演】的能力，则你可以选择其中一项提升至A级，并获得对应效果，除此之外，你免疫A级别信息、影响心灵来源的负面效果。若处于FATE（型月）世界观下，即使你触碰根源也不会受到任何影响。

【天赋的睿智A】：保留之前你已具有相应等级的效果；此外，你在全检定上获得等同自身【智力属性】一半的内在加值；每轮一次，若你对自身智力相关检定结果不满意，则可以进行一次重骰，两者取高处理。

【人工生命体A】：保留之前你已具有相应等级的效果；此外，你免疫不超过A级创伤、疾病、毒素/药物来源效果的影响；当你受到攻击时，若你在计算防御后仍然保留你的天生防御，则攻击你的目标需要在攻击检定上承受【劣化3】；即使你失去了肢体（包括头部等其他要害部位）也能立刻再生。

【情感模块A】：你免疫不超过S级的即死来源效果影响；当你的生命值被严重伤害填满时（即完好生命值因严重伤害归零），你可以使用反射动作（在回合内改为迅捷动作）调用下回合的标准动作，并消费1点意志力，从而启动这一能力，这会正常消耗下回合你的对应AP并影响动作可转化上限。

启动后，你每轮都可以免疫一次攻击，这是一个A级命运来源的【无敌】效果。这一效果将会持续一场景。

显然的，情感模块在提升至A级后无需保留之前等级情感模块的效果（因为在A级之前情感模块并没有提供任何效果，只作为选择人造人的前置生效）。

【人智再演A】：保留之前你已具有相应等级的效果；此外，你可以从FATE能力组中选择一项价格不超过A+4000分，且不会占据血统、改造位的技艺获得，获得的技艺不会额外提供属性，因此若是不提供属性的技艺也无法获得；每轮一次，你进行的智力检定出目增加一半。
