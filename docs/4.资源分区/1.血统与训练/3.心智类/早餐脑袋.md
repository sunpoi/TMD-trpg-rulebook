# 早餐脑袋

早餐脑袋  
血统  
前提：喜欢吃早餐或者过去因为一些麻烦事没空吃早餐的生物  
描述：世界上的很多人因为种种原因，导致他们并没有吃早餐的习惯，主神空间的轮回者们更是不吃早餐的重灾区，他们凭借着的强健体魄，又或者什么不用进食的奇怪能力，每天只吃一顿甚至不吃。而就是这弥足珍贵的一顿，那些家伙也不会选择用在早餐，这简直不可理喻。为了遏制这种不良的习惯，仁慈的早餐之神会循循善诱（物理洗脑），使得自己的信徒爱上吃早餐，这些信徒就是最光荣的早餐脑袋！

  
这是一个信仰导向型能力，早餐脑袋虔诚地信仰着早餐之神

  
早餐脑袋们的身体将不可避免的开始早餐化，最严重的是头部（物理洗脑的副作用）你的天灵盖将会变成一个神秘的早餐机器，每天早上都会弹出对应品种的一早餐脑袋份的早餐，这是早餐之神的赐福。早餐化的身体类型取决于选择的【早餐思维】。早餐脑袋需要在每天享用早餐的时候（早餐必须在早上吃！）向早餐之神祈祷10分钟，否则会承受1点全检定减值，直到你下一次这么做。除了早餐之神，早餐脑袋们都极度崇拜厨艺高超的生物，对他们有着天然的好感。

  
潮湿和寒冷是早餐脑袋最大的敌人，这会让早餐凉掉甚至变得不再可口！早餐身体让你获得了寒冰易伤4，当你的身体超过一半的体积被水浸泡或者被有水元素的魔法攻击，你将失去所有的基础防御，你需要用热源烘烤身体十分钟来重新获得

  
你的物种将改为【早餐生物】

  
早餐脑袋们可以选择一个【早餐思维】来决定自己的发展方向

  
法棍思维  
D级：见习法棍  
价格：D+600  
描述：法棍思维的早餐脑袋们热衷于成为法师或者神棍，这在他们的名字上就有很好的体现。  
据小道消息所说，所谓的法杖不过只是一些可以施法的棍子，早餐脑袋们群情激愤：  
“我们一点都不比那些法棍差，而且我们更加可口！”  
【陈年法棍更硬更可口！】：历经时间的洗涤，法棍身体将变得更加坚硬，你获得1点闪避防御，并且额外获得传奇耐力\*1点闪避防御  
【众生平等】：所有的生物都应该领略到早餐的魅力，你进行的施法攻击视为拥有【法破】特性  
【见习全适应法棍】：合格的早餐脑袋在面对任何敌人都应该表现出最可口的一面，你可以把攻击中的【破魔】转化成等额的【高速】，最多转化6点

  
C级：专家法棍  
价格：C+1200  
描述：早餐脑袋之间的竞争是残酷的，抢走早餐，制作黑暗料理，把盐罐子贴上糖的标签......无所不用其极。  
就是这样的一群武德充沛的早餐脑袋脱颖而出，他们被称为——专家法棍！  
【陷阱专家】：在诡计多端的同胞中生存下来并且保持住自己的风味是非常困难的，你见识过太多的阴险损招，已经有了丰富的经验，免疫B级措手不及  
【我是早餐领域专家，你敢和我对视10秒吗？】长久的历练让你获得了一双善于发现的眼睛，很多对手因为看到你的眼睛就憋不住笑导致偷袭失败。用它可以让你察觉身边潜在的危险，这是一种C级魔幻本质的侦测效果。

  
B级：超菁英法棍  
价格：B+2000  
描述：法棍中的法棍！早餐中的早餐！精英中的精英！早餐世界出现了一个英勇的抽象，你是所有法棍的榜样，超级聪明！超级狡猾！超级可口！总之就是超级超级！超菁英法棍！  
【超级防水！】：你摆脱了影响法棍可口的最大威胁，你的身体已经可以在水里畅游，你免疫B级【潮湿】  
【超级镭射眼】：你的眼睛已经可以洞穿一切，其他见习法棍是这么描述你的，虽然是谣言，但你确实拥有一对炯炯有神的卡姿兰法棍眼，本血统获得的C级魔幻本质的侦测效果提升至B级魔幻本质的侦测效果。  
【超级谨慎】：身为超菁英，你的粉丝一整个大餐桌都坐不下！为了防止被友善的早餐粉丝突然袭击，你锻炼出了无与伦比的超级本能和超级速度！你免疫A级措手不及，获得60米基础速度提升  
【超菁英自适应法棍】：万酱丛中过，片汁不沾身！再难缠的酱料也无法触及你拥有独特风味的身体，即使情况再糟糕，你也能凭借着超级法棍的骄傲抗争到底，你获得基于闪避防御的【适应】特性

  
黑暗早餐法术  
法术：平安果榨菜汁  
类型：攻击技  
价格：D+500  
派系：黑暗  
能耗：2  
成分：动作  
动作：标准  
射程：35米  
范围：等同于射程的直线  
持续：立即  
引导凝聚出平安果榨菜汁向敌人激射而去，看似黑暗，但细细品味其实味道还不错，圣诞早餐老人开发的特供法术

  
一次研发10xp：升级为C级法术平安果榨菜折耳根汁，引导凝聚出更为强大的平安果榨菜折耳根汁向敌人激射而去，被击中的生物一般会面露难色呆立在原地不知所措  
能耗改为6点，附带攻击成功数四分之一的【眩晕】点数，射程提升至55米

  
二次研发20xp：升级为B级法术浓缩平安果榨菜折耳根原浆，引导凝聚出令所有早餐脑袋望而生畏的恐怖流体，温热的恰到好处，黏黏糊糊，湿湿哒哒，让所有生物都能感受到圣诞节的温暖  
能耗改为10点，被命中的敌人需要进行一次意志豁免，DC为伤害值，失败则进入B级影响心灵来源的【潮湿】状态，射程提升至55+传奇智力\*10米
