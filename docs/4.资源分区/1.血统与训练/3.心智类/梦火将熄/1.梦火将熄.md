# 梦火将熄

梦火将熄

D+600  
如果说愤怒是人最难控制的情感，那么悲伤将是人最难摆脱的情感。  
一团火烧尽之后，就只剩无处寻觅的余烬。

在此阶段下，你立即获得踽踽独行、二重奏、暮色

踽踽独行：  
当你进行消耗了意志力的主动行为，且不为基础用法，这要求是招式消耗时，这个特性的效果将立刻开始生效：  
从本轮开始，当你满足条件并使用了意志力的回合结束时，你立即在本场景中获得1点攻击检定上的表现加值，这个效果在每场战斗中将至多重复生效你的决心\*1轮。当你拥有【终局战术】且其开始生效时，改为获得1点基于你正在使用的武器的武器伤害。

二重奏：  
当进入战斗场景时，这一特性将立刻开始生效，由你自行选择【爆裂】和【抑制】的生效轮次，它们将在奇偶数回合交替生效（例：玩家【逆位】恋人进入战斗轮之中，她选择奇数轮次生效【爆裂】，偶数轮次生效【抑制】。在下一场战斗开始时，她将可以重新选择）。  
二重奏还会使你拥有的所有本源碎片不再生效，而是改为选择【爆裂】、【抑制】中的一项等价提升其效果。  
爆裂：处于【爆裂】回合中时，你受到的所有伤害将增加1/3（向下取整），并在消耗意志力获得表现加值时额外获得2点。每等价为3本源点的本源碎片会为【爆裂】提供额外的1点消耗意志表现加值。若你因某些能力在该轮免疫了伤害，则这个效果不会生效。  
抑制：处于【抑制】回合中时，你的攻击检定上始终失去4点附加成功，附加不足时改为每不足1点附加成功获得3DP表现减值，本轮内你在被敌人攻击时，敌人视为承受基于闪避防御的【劣化1】。每等价为3本源点的本源碎片会为【抑制】减少失去1点附加成功。

暮色：  
战斗第二轮开始时，若踽踽独行已触发过，则这个特性将开始持续生效，你立刻获得等同于你的传奇决心\*1点【狂怒】，这个效果将持续一个场景。

C+1200  
在此阶段下，你立即获得踽踽独行+、二重奏+

踽踽独行+：  
在踽踽独行的基础上，将每轮结束时获得的表现加值数从1点变为你的传奇决心\*1点，触发【终局战术】而获得效果同样改变为传奇决心\*1点。

二重奏+：  
现在，【抑制】将额外减少2点附加成功，并将【劣化1】升级为【劣化2】  
现在，你还可以额外支付对应价格为【爆裂】增加更多效果，并正常获得属性：

不定之态  
D+500  
【爆裂】回合开始时，需猜测一次D2结果，若猜对，则本回合内【爆裂】提供的额外意志消耗加值变为1.75倍，本能力将额外为你提供2点【爆裂】期间提供的意志消耗加值。

回归呼唤  
D+500  
处于爆裂回合中时，若消耗了提供表现加值的意志力，则可以将至多不超过等同于你的传奇决心\*2点意志提供的表现加值，每1点表现加值转换为任一系本源碎片各1个。不能超过本源碎片的获得上限，不能在一轮中重复使用，【抑制】回合开始时失去这些碎片。

B+2400  
在此阶段下，你立即获得踽踽独行++、二重奏++、暮色+，并获得长夜将至的购买权。

踽踽独行++：  
在踽踽独行+的基础上，将每轮结束时获得的表现加值数从传奇决心\*1点变为你的传奇决心\*2点，触发【终局战术】而获得效果同样改变为传奇决心\*2点。

二重奏++：  
现在，【抑制】将额外减少1点附加成功，并将【劣化2】升级为【劣化3】，且【抑制】将额外提供基于闪避防御的【超级坚韧】。  
现在，你还可以额外支付对应价格为【爆裂】增加更多效果，并正常获得属性：

荒原骑士  
C+1000  
当你处于【狂怒】状态，且处于【爆裂】回合时，若你进行了冲锋攻击，且你在本回合内触发了踽踽独行，则将立即开始生效，你的那次冲锋攻击将额外获得等同于你的意志池每1点空意志池\*1点表现加值，上限不能超过你的传奇决心\*2，本次攻击还视为具有B级先制。

无罪辩护  
C+1000  
当你使用以决心为主要攻击属性的武器，且在回合外保留了至少6AP时，你可以在一次招架的同时调用下个回合的标准动作对触发招架的目标立即进行一次不附带任何攻击技的攻击，这视为进行了一次打断，并在那个回合获得免疫C级措手不及和谐振，谐振将武器伤害提供的破防转化为闪避防御。

暮色+：  
战斗第二轮开始时，若踽踽独行已触发过，则这个特性将开始持续生效，你立刻获得等同于你的传奇决心\*2点【狂怒】，这个效果将持续一个场景。
