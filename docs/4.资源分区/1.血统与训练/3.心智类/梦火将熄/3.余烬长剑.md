# 余烬长剑

\=========  
余烬长剑

魔幻本质  
介绍：  
这是一把布满灰烬的长剑，带有火星的灰烬会从剑上不断落下，但这既不会在地上留下灰烬，剑身的灰烬也不会因此减少，也因此试图清洁这把长剑的行动都是徒劳的。  
而当抚摸剑身时，普通人能够感受到剑身残留的余温，而走上驾驭暴怒道途的人还会听到这样一句话：“祝火焰燃尽你的敌人，而非你自己”  
价格：D+1250  
效果：  
此剑蕴含着暴怒的力量，只有当使用者走上驾驭暴怒的道途才能发挥完全，否则，它便只是稍些沉重且难用的长剑了。  
余烬长剑为武器伤害6L，具有【破甲2】，体积4的长剑，攻击检定为【决心+武技+专业:剑+辅助专业：武学掌握】。  
当你手持余烬长剑时，对燃烧异常状态的豁免获得+2dp器械加值，并且获得1点火焰抗力。

特殊：  
当剑主拥有“暴怒行者”技艺时，适用以下效果  
剑主每有1点空意志池，便在一场景内攻击上获得1点dp表现加值，当剑主骑乘【马尔斯】时变为2dp。至多叠加2+传奇决心\*1点表现加值。

支付C+1500，进一步释放余烬长剑的潜能，你能够发觉这把剑不仅能遏制火焰向你吐出的火舌，还能吸收火焰储存起来为其所用。（无数值修改，仅优化描述）  
当火焰从剑身燃起，剑身的灰尘被这热腾的气流冲散了不少，飘扬的灰尘也不再转瞬即逝，它们还能传递其所具有的温度。

现在余烬长剑视为武器伤害为14L，具有【破甲6】，体积4的长剑  
每场景一次，以迅捷动作令剑身燃起火焰，进入【暴怒】状态，本场景内你的攻击能点燃目标，这会令受到你攻击伤害的单位额外承受1点不可豁免的燃烧点数。  
当你手持余烬长剑时，对燃烧异常状态的豁免获得+4dp器械加值，并且获得2点火焰抗力。 替代D级的效果。

支付B+2000，释放余烬长剑剩余的潜能，你已经能够直观感受到这把剑对火焰的遏制能力，仅是一般的火焰根本无法接近你，即使接近了大多也会被剑身吸收。（无数值修改，仅优化描述）  
当火焰从剑身燃起，剑身的灰尘倒是完全散去了，你得以窥见这把长剑上一任主人所属的家族“德克里”。

现在余烬长剑视为武器伤害为18L，具有【破甲12】，体积4的长剑  
每场景一次，以迅捷动作令剑身燃起火焰，进入【暴怒】状态，本场景内你的攻击能点燃目标，这会令受到你攻击伤害的单位额外承受4点不可豁免的燃烧点数。替代C级的效果  
当你手持余烬长剑时，效果变对燃烧异常状态的豁免获得+8dp器械加值，并且获得4点火焰抗力。 替代C级的效果

特殊变更：  
剑主每有1点空意志池，便在一场景内攻击上获得3点dp表现加值，当剑主骑乘【马尔斯】时变为6dp。至多叠加2+传奇决心\*4点表现加值。

\===【下属技能】===  
扬起燃灰  
前提：C级余烬长剑  
价格：D+500  
效果：  
以移动动作使用，且进入【暴怒】状态。你令余烬长剑释放更旺的火焰，令你接下来的挥砍能够一并带出剑上燃起的灰尘余烬。  
你的下次攻击命中后，对所有受到你伤害的目标额外施加1点不可豁免的燃烧点数。

无羁怒火  
前提：C级余烬长剑  
价格：C+1000  
效果：  
以标准动作使用，且进入【暴怒】状态。你令剑身的火焰连同你心中的怒火一并激燃，大迈步左右挥砍你触及范围内的所有敌人，以反射豁免对抗。  
本次攻击的加骰提升1级，最多8加骰。命中后视为造成了2点多次攻击减值。

特殊：  
当你骑乘【马尔斯】时，效果改为对触及范围内的所有敌人攻击时，此次攻击获得基于技艺加值的2点附加成功，以反射豁免对抗。其余加成不变。
