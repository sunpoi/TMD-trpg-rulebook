# 暴怒行者

暴怒行者（技艺）  
介绍：  
暴怒行者并非易怒的性情中人，而是当逆鳞被触碰后，心中能够燃起泯灭一切美德的怒火，不计手段实行报复，并守护重要之物的一群人。  
学好运用这份力量，同时不被其吞噬，让暴怒宣泄到应得的人身上，保护你想要保护的一切。而如果你没能守护他们，那就吞噬令你如此的人们。

“我知我所为何”  
价格：D+500  
效果：  
压抑心中的怒火，慢慢清点自己从仇人那失去的一切，是每位暴怒行者踏上驾驭暴怒之路的伊始。清点目标，敲定账单，准备寻仇。  
既是开始，也是结束，属于暴怒行者的漫漫路途，一旦开始，便没有握手言和的余地。只要你的意志池不为空，你因此获得1点【狂怒】

与平静的生活彻底切割，需要莫大的意志与毅力。你获得决心的1内在加值，并且在进行决心相关判定时，获得来自技艺加值的1附加成功。

“我知我怎实践”  
价格：C+1250  
效果：  
收拾行囊，处理财产，与亲朋告别，跟故乡再见。你知道你或许不再有回来的可能，但你知道如果正义不能及时来到，那你便是那最后的“正义”。  
如果最坏的可能还未发生，那便亲手掐灭，若那最坏的可能已经实现...唯余清算。只要你的意志池不为空，你因此获得1点【狂怒】

在你漫漫路途的遭遇中，你已经对各方的小表情和小动作再熟悉不过，在心里总能准备一套临时应对突发情况的方案......又或者交由你的战斗经验来处理。谁在你的名单上，谁是无关的人，你早已了熟于心。  
你获得免疫B级搓手不及的能力，并且反射豁免获得4DP的技艺加值。你现在可以在某次攻击中任意留手。

“我知我无他选”  
价格：C+1000  
效果：  
怀着必报的决心，你不顾亲朋好友的劝阻一意前行，你并不清楚实现你的目标需要多少时间，可能要几周、几月、一年、乃至十几年，甚至你的余生。  
但你坚信这是值得的，纵使这心中的怒火有天也会将你遍体鳞伤的身躯燃尽，但为了身后之物。一切都是值得的...都是值得的！只要你的意志池不为空，你因此获得1点【狂怒】

用怒火处理伤口，以意志支撑身躯，决不能在目标完成前就此倒下，只要你的意志池不为空，你就不会因没有完全生命而陷入昏迷。  
同时你不再从疲劳异常点数上获得减值，并且你的强韧豁免获得4DP的技艺加值。

“我知我向何方”  
价格：D+500  
效果：  
一个，两个，三个...亦或更多？你已经记不清你令多少人付出了代价，也不知自己的漫漫路途还有多久。你只知道，你已经很久没有回去故乡看看了，但你的清算还在继续。  
风餐露宿的日子对你已是日常，只要能消除疲劳感，一时的不适感都是可以忍受的。你上次躺在温暖的家中又是何时呢？哈，早就忘却了，能在这路上陪伴你的唯有你的怒火。  
这视为你获得自适应长休的能力。

如果你已经购买“回应我，马尔斯！”则为此技艺添加D级旅行能力。使用方式如往常一样，高声呼喊【马尔斯】来你身边，骑着它奔驰在城际之间，且无需消耗意志力。  
但以此法召唤的【马尔斯】在进入战斗后会自行离场，在战斗轮内需要你用原方式召唤，你也无法用战斗中召唤的【马尔斯】使用此效果。

“回应我，马尔斯！”  
价格：D+1500  
效果：  
以移动动作使用，暴怒行者呼喊七大罪代表之一的大名，随后一匹鬃毛燃烧着的骏马从你周边的阴影钻出来到你的身旁，可以选择直接上马。  
骑乘【马尔斯】需要每回合结束时消耗1点意志力，在第二回合后，每回合进行一次要求意志一半成功数的意志豁免，未成功则在回合结束时下马且【马尔斯】退出战场。  
当骑乘【马尔斯】时，暴怒行者的冲锋攻击将不再承受基础防御上的减值。且在进行攻击时可以令攻击目标改为触及范围内的所有敌人，  
上马的当轮角色视作已经格挡。

再支付B+2000后，额外获得以下效果：  
当暴怒行者骑乘【马尔斯】时，始终视为格挡状态、移动速度增加决心\*5米、攻击检定获得决心附加\*3点表现加值。  
当暴怒行者骑乘【马尔斯】时，进行的冲锋攻击可同时攻击冲锋路径上的所有单位。

（PS：【马尔斯】并非真正的马匹，只是使用者意志力的投影，因此骑乘【马尔斯】时不需要进行驾驶鉴定（已经通过意志力消耗了），移动距离参考使用者的数据。 ）
