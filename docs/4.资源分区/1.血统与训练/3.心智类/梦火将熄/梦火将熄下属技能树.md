# 梦火将熄下属技能树

梦火将熄下属技能树：

暴怒行者  
概述：暴怒行者并非易怒的性情中人，而是当逆鳞被触碰后，心中能够燃起泯灭一切美德的怒火，不计手段实行报复，并守护重要之物的一群人。  
学好运用这份力量，同时不被其吞噬，让暴怒宣泄到应得的人身上，保护你想要保护的一切。而如果你没能守护他们，那就吞噬令你如此的人们。

技艺名：“我知我所为何”  
价格：D+500  
分类：被动

描述：压抑心中的怒火，慢慢清点自己从仇人那失去的一切，是每位暴怒行者踏上驾驭暴怒之路的伊始。清点目标，敲定账单，准备寻仇。  
既是开始，也是结束，属于暴怒行者的漫漫路途，一旦开始，便没有握手言和的余地。

效果：  
只要你的意志池不为空，你因此获得1点【狂怒】

与平静的生活彻底切割，需要莫大的意志与毅力。你获得决心的1内在加值，并且在进行决心相关判定时，获得来自技艺加值的1附加成功。

技艺名：“我知我怎实践”  
价格：C+1250  
分类：被动

描述：收拾行囊，处理财产，与亲朋告别，跟故乡再见。你知道你或许不再有回来的可能，但你知道如果正义不能及时来到，那你便是那最后的“正义”。  
如果最坏的可能还未发生，那便亲手掐灭，若那最坏的可能已经实现...唯余清算。

效果：  
只要你的意志池不为空，你因此获得1点【狂怒】

在你漫漫路途的遭遇中，你已经对各方的小表情和小动作再熟悉不过，在心里总能准备一套临时应对突发情况的方案......又或者交由你的战斗经验来处理。谁在你的名单上，谁是无关的人，你早已了熟于心。  
你获得免疫B级搓手不及的能力，并且反射豁免获得4DP的技艺加值。你现在可以在某次攻击中任意留手。

技艺名：“我知我无他选”  
价格：C+1000  
分类：被动

描述：怀着必报的决心，你不顾亲朋好友的劝阻一意前行，你并不清楚实现你的目标需要多少时间，可能要几周、几月、一年、乃至十几年，甚至你的余生。  
但你坚信这是值得的，纵使这心中的怒火有天也会将你遍体鳞伤的身躯燃尽，但为了身后之物。一切都是值得的...都是值得的！

效果：  
只要你的意志池不为空，你因此获得1点【狂怒】

用怒火处理伤口，以意志支撑身躯，决不能在目标完成前就此倒下，只要你的意志池不为空，你就不会因没有完全生命而陷入昏迷。  
同时你不再从疲劳异常点数上获得减值，并且你的强韧豁免获得4DP的技艺加值。

技艺名：“我知我向何方”  
价格：D+500  
分类：被动

描述：一个，两个，三个...亦或更多？你已经记不清你令多少人付出了代价，也不知自己的漫漫路途还有多久。你只知道，你已经很久没有回去故乡看看了，但你的清算还在继续。  
风餐露宿的日子对你已是日常，只要能消除疲劳感，一时的不适感都是可以忍受的。你上次躺在温暖的家中又是何时呢？哈，早就忘却了，能在这路上陪伴你的唯有你的怒火。

效果：  
这视为你获得自适应长休的能力。

如果你已经购买“回应我，马尔斯！”则为此技艺添加D级旅行能力。使用方式如往常一样，高声呼喊【马尔斯】来你身边，骑着它奔驰在城际之间，且无需消耗意志力。  
但以此法召唤的【马尔斯】在进入战斗后会自行离场，在战斗轮内需要你用原方式召唤，你也无法用战斗中召唤的【马尔斯】使用此效果。

技艺名：“回应我，马尔斯！”  
价格：D+1500  
动作：移动  
分类：增强技  
消耗：1点意志维持/每轮  
目标：自己  
持续：特殊

效果：  
以移动动作使用，暴怒行者呼喊七大罪代表之一的大名，随后一匹鬃毛燃烧着的骏马从你周边的阴影钻出来到你的身旁，可以选择直接上马。  
骑乘【马尔斯】需要每回合结束时消耗1点意志力，在第二回合后，每回合进行一次要求意志一半成功数的意志豁免，未成功则在回合结束时下马且【马尔斯】退出战场。  
当骑乘【马尔斯】时，暴怒行者的冲锋攻击将不再承受基础防御上的减值。且在进行攻击时可以令攻击目标改为触及范围内的所有敌人，  
上马的当轮角色视作已经格挡。

再支付B+2000后，额外获得以下效果：  
当暴怒行者骑乘【马尔斯】时，始终视为格挡状态、移动速度增加决心\*5米、攻击检定获得决心附加\*3点表现加值。  
当暴怒行者骑乘【马尔斯】时，进行的冲锋攻击可同时攻击冲锋路径上的所有单位。

（PS：【马尔斯】并非真正的马匹，只是使用者意志力的投影，因此骑乘【马尔斯】时不需要进行驾驶鉴定（已经通过意志力消耗了），移动距离参考使用者的数据。 ）

名称：永生羽衣

价格：D+750/C+1000/B+2000【逐级升级】

购买前提：女性

概述：这是一件金红色羽披，其上缀有成片的金红色羽毛，整体由绒毛制成，极其轻便。

类型：轻甲  
护甲防御：5/5

高等级不再重复描述前一级效果。

D级

护甲防御：5/5  
特性：【余火将熄】【东隅日暮】

描述：在此阶段下，永生羽衣的羽毛看起来还黯淡无光，寄宿在其上的力量尚未完全流失。

效果：  
【余火将熄】  
羽衣的力量已几乎消耗殆尽，但失去了火的时代，只能保留它最低限度的热量。穿戴这件羽衣时，使用者将获得免疫C级及以下的潮湿效果。

【东隅日暮】  
现在，使用者会获得一种特殊的，名为【火种】的计数方式，每经历一场影片，且其中曾进行过一次表现分为传奇的扮演，在影片结束时获得1粒【火种】。  
并根据拥有的【火种】数量，参照下表获得增益：  
1粒：羽衣的护甲防御提升使用者的传奇决心\*1点。  
2粒：羽衣将获得免疫多次攻击带来的减值的效果。  
3粒：一回合一次，羽衣的使用者受到攻击时，攻击者将必须重投一次，两次结果取劣。  
4粒：羽衣的护甲防御提升使用者的决心点。

C级

护甲防御：13/13  
特性：【照我以火】【轻灵凤羽】

描述：在此阶段下，永生羽衣已恢复了一部分光和热，行走时会在身后留下尾迹。

效果：  
【照我以火】  
羽衣的力量逐渐提升，已没有最开始般孱弱。穿戴这件羽衣时，使用者将获得免疫B级及以下的潮湿效果。

【轻灵凤羽】  
一轮一次，当使用者进行了一次招架时，可以应用这个能力的效果，使用者将立即视为以极其轻灵的步伐躲闪这次攻击，对于本次攻击而言，使用者视为移出了攻击者的攻击范围。

B级

护甲防御：20/20  
特性：【永生】【桑榆非晚】

描述：在此阶段下，【永生】将真正成为永生羽衣的代名词，相传，东方之神鸟每过一千年便要涅槃重生，而永生羽衣也不例外。

效果：  
【永生】  
在收集齐4粒火种后，一场影片一次，当使用者首次死亡时，则永生羽衣将立刻化作满天飞羽，将使用者包裹入其中，使用者将转化为灵体状态，并在一周后从羽衣中破出，以正常状态复活，这是一个A级时空来源的复活效果。

【桑榆非晚】  
在以【永生】复活后，【桑榆非晚】将开始持续生效直至影片结束。只要使用者仍处于场景中，场景便视为亮度提升至最高，所有直接目击使用者的角色需要以意志豁免对抗DC17，若对抗失败，则其将陷入目盲状态之中，这是一个B级模因来源的目盲效果，使用者再获得完美机动性飞行。

名称：余烬长剑

本质：魔幻本质

价格：D+1250

概述：这是一把布满灰烬的长剑，带有火星的灰烬会从剑上不断落下，但这既不会在地上留下灰烬，剑身的灰烬也不会因此减少，也因此试图清洁这把长剑的行动都是徒劳的。

类型：轻剑  
体积：4  
武器伤害：6L  
基础特性：【破甲2】  
特性：【余烬】【驾驭暴怒】  
效果：  
【余烬】  
此剑蕴含着暴怒的力量，只有当使用者走上驾驭暴怒的道途才能发挥完全，否则，它便只是稍些沉重且难用的长剑了。  
余烬长剑为武器伤害6L，具有【破甲2】，体积4的长剑，攻击检定为【决心+武技+专业:剑+辅助专业：武学掌握】。  
当你手持余烬长剑时，对燃烧异常状态的豁免获得+2dp器械加值，并且获得1点火焰抗力。

【驾驭暴怒】：  
特殊：  
当剑主拥有“暴怒行者”技艺时，适用以下效果  
剑主每有1点空意志池，便在一场景内攻击上获得1点dp表现加值，当剑主骑乘【马尔斯】时变为2dp。至多叠加2+传奇决心\*1点表现加值。

支付C+1500，进一步释放余烬长剑的潜能，你能够发觉这把剑不仅能遏制火焰向你吐出的火舌，还能吸收火焰储存起来为其所用。（无数值修改，仅优化描述）  
当火焰从剑身燃起，剑身的灰尘被这热腾的气流冲散了不少，飘扬的灰尘也不再转瞬即逝，它们还能传递其所具有的温度。

现在余烬长剑视为武器伤害为14L，具有【破甲6】，体积4的长剑  
每场景一次，以迅捷动作令剑身燃起火焰，进入【暴怒】状态，本场景内你的攻击能点燃目标，这会令受到你攻击伤害的单位额外承受1点不可豁免的燃烧点数。  
当你手持余烬长剑时，对燃烧异常状态的豁免获得+4dp器械加值，并且获得2点火焰抗力。 替代D级的效果。

支付B+2000，释放余烬长剑剩余的潜能，你已经能够直观感受到这把剑对火焰的遏制能力，仅是一般的火焰根本无法接近你，即使接近了大多也会被剑身吸收。（无数值修改，仅优化描述）  
当火焰从剑身燃起，剑身的灰尘倒是完全散去了，你得以窥见这把长剑上一任主人所属的家族“德克里”。

现在余烬长剑视为武器伤害为18L，具有【破甲12】，体积4的长剑  
每场景一次，以迅捷动作令剑身燃起火焰，进入【暴怒】状态，本场景内你的攻击能点燃目标，这会令受到你攻击伤害的单位额外承受4点不可豁免的燃烧点数。替代C级的效果  
当你手持余烬长剑时，效果变对燃烧异常状态的豁免获得+8dp器械加值，并且获得4点火焰抗力。 替代C级的效果

特殊变更：  
剑主每有1点空意志池，便在一场景内攻击上获得3点dp表现加值，当剑主骑乘【马尔斯】时变为6dp。至多叠加2+传奇决心\*4点表现加值。

技艺名：扬起燃灰  
价格：D+500  
购买前提：C级余烬长剑  
动作：移动  
分类：增强技  
消耗：-  
目标：自己  
持续：特殊

效果：  
以移动动作使用，且进入【暴怒】状态。你令余烬长剑释放更旺的火焰，令你接下来的挥砍能够一并带出剑上燃起的灰尘余烬。  
你的下次攻击命中后，对所有受到你伤害的目标额外施加1点不可豁免的燃烧点数。

技艺名：无羁怒火  
价格：C+1000  
购买前提：C级余烬长剑  
动作：标准  
分类：攻击技  
消耗：-  
目标：触及范围内的所有敌人  
持续：立即

效果：  
以标准动作使用，且进入【暴怒】状态。你令剑身的火焰连同你心中的怒火一并激燃，大迈步左右挥砍你触及范围内的所有敌人，以反射豁免对抗。  
本次攻击的加骰提升1级，最多8加骰。命中后视为造成了2点多次攻击减值。

特殊：  
当你骑乘【马尔斯】时，效果改为对触及范围内的所有敌人攻击时，此次攻击获得基于技艺加值的2点附加成功，以反射豁免对抗。其余加成不变。
