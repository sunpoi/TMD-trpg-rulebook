# 绘者下属技能树

绘者下属技能树  
绘者下属法术：  
特殊的，在释放下属法术时，若动作为移动动作，则可以结合在维持绘画状态中使用

色彩崩溃  
类型：魔法  
价格：D+500  
派系：万变  
消耗：2点绘能  
动作：标准动作  
射程：15米  
范围：单体  
持续：一场景  
效果：  
该法术不会获得派系效果，以一个标准动作，你剥离你射程范围内的一名敌人的色彩，  
你进行一次攻击检定，敌人需要以一次强韧豁免对抗这次检定的出目，若其失败，则其在敏捷/耐力/决心检定（购买时选择，一经选择无法更改）上受到劣化1影响，并且收到5DP机运减值，这是一个等于法术等级的命运来源的效果。  
这不视作一次攻击，因此不会造成伤害或是异常点数。  
——研发：支付D+500将该技艺升级至C级，  
能量消耗增加至6点，劣化1升级至劣化2，收到的减值提升至11点。  
——研发：额外支付C+1000将该技艺升级至B级，  
能量消耗增加至10点，劣化2升级至劣化3，收到的减值提升至20点。

特殊的：你也可以花费2倍价格购买该技艺（也因此会如同购买2个技艺一般提供属性点），若如此做，则效果升级为：在施法时选择敏捷/耐力/决心检定其中一项上受到减值与劣化效果。

常青树  
D+500  
类型：魔法  
派系：自然  
消耗：2点绘能  
动作：标准动作  
射程：15M  
范围：以自己为中心，以射程/4为半径的圆形区域  
持续：一场景  
效果：  
你绘画出一颗常青之树，并且让其赐福一片区域，  
在范围内的队友在一轮内获得快速医疗1（医疗点12）。  
——研发：支付D+500将该技艺升级至C级，  
能量消耗提升至6点，快速医疗的数值提升至快速医疗2（医疗点24），同时获得1点异常点治疗（医疗点24）。  
——研发：支付C+1000将该技艺升级至B级，  
能量消耗提升至10点，快速医疗的数值提升至快速医疗5（医疗点60），异常点治疗的数值提升至异常点治疗2（医疗点48），并且获得B级的免疫重伤。

雪景球  
C+1000  
类型：魔法  
派系：自然  
消耗：6点绘能  
动作：移动动作  
射程：15米  
范围：以自身为圆心，以射程/4为半径的圆形  
持续：一场景  
效果：  
你使周围披上雪景，这会使范围内的环境变化为暴雪环境，这是一个C级的环境来源的效果，并且使范围内的全部友军获得C级的环境来源免疫和2点遮蔽防御，  
并且你的友军在造成伤害时，会附带1点不可避免的冻寒伤害。  
增幅：额外支付1点能量，使你也会获得对应的好处。

天坠之火  
C+1000  
类型：魔法  
派系：核心  
消耗：6点绘能  
动作：标准动作  
射程：15M  
范围：以目标敌人为圆心，以射程/4为半径的圆形  
持续：一轮  
效果：  
你绘画出一枚从天而降的陨石，进行一次攻击，在这次攻击中，你获得2DP的能量加值，  
并且在攻击结束后，范围内会留下一块熔岩区域，若你不持续施法，则这块熔岩区域会在你的下个回合结束时消失，这是一个C级环境来源效果。

瓦尔哈拉  
D+500  
类型：魔法  
派系：自然  
消耗：2点绘能  
动作：移动动作  
射程：15M  
范围：以自身为中心，以射程/4为半径的圆形区域  
持续：一轮  
效果：  
你绘画出神话中的英灵集结之地，这会为范围内的所有友军提供2点攻击上的增强加值和2点能量防御。

——研发：额外支付D+500将该技艺升级至C级，  
能量消耗提升至6点，提供的攻击上的增强加值额外提升智力上的传奇属性\*2点，能量防御额外提升风度属性上的传奇属性\*1点。  
——研发：额外支付10XP升级该技艺，  
此时除了原本的效果外，还会提供加骰上升1级（至多为8加骰）的效果，并且提供的能量防御额外提升风度属性上的传奇属性\*1点。  
——研发：额外支付C+1000将该技艺升级至B级，  
能量消耗提升至10点，提供的攻击上的增强加值额外提升智力上的传奇属性\*3点，能量防御额外提升风度属性上的传奇属性\*2点。  
——研发：额外支付6XP（这个研发至少需要C级才可购买）  
此时除了原本的效果之外，还会额外提供智力属性\*1点先攻上升。

三原色  
价格：D+500/C+1000/B+2000/A+4000（差价升级）

特殊的，该技艺可以重复购买，若如此做，则召唤的随从会增加一个。

并非每位绘者都满足于抽离现实的色彩  
所以他们开始进一步依据心中的构思于世界中绘画，创作...绝不会停滞于此  
有人更加大胆的下笔，将颜料洒落大地，力求用连贯的绘画出栩栩如生的画作。

“画作不该是静止的，它们应当拥有生命”

该技艺具有万变派系并且需要如常消耗2/6/10/14点魔力，但是其不会获得派系效果。  
于是你紧握画笔，想要如前人般实现这个壮举  
以一个移动动作施展，你开始忘我的创作，纯色的画作在你的绘画下逐渐活络起来，甚至浮现出原本你未曾添加的色彩，并最终不在依赖于你的画笔。  
下一轮开始时，这些画作将继承你的先攻，并于你的身边加入战场。这些画作也视作【随从】。

画作的数量取决于你拥有的【色盘】颜料数量和该技艺的等级，并且画作类型由【色盘】颜料的类别决定。  
若同时具有多个颜料但是技艺等级不足，那么召唤顺序为红绿蓝；  
颜料红——画作【红】  
颜料绿——画作【绿】  
颜料蓝——画作【蓝】

尽管绘者每次绘画都是一次巧思，但正如那些在绘者间广泛传播的泛式一样  
仅需一眼，他们便能记住画作的精髓——但艺术家的高傲让他们拒绝随意复现他人的作品。  
请好好珍惜那转瞬即逝的创作灵感和对自己作品的命名权，【随从】的外貌与名字需要由你来署名。

特别的，本技艺所召唤的随从数量与随从类型取决于你购买的绘者下属奇物【色盘】颜料类别，随从的等级取决于本技艺和【色盘】对应颜料二者的等级取低。  
（例如逆位【恋人】购买了2次C级三原色，D级颜料红，B级颜料蓝，那么使用本技艺召唤的随从数量为二，类型为三原色【红】、三原色【蓝】，随从的等级分别为D级和C级）
