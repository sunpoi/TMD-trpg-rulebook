# 绘者下属奇物：色盘

色盘  
D+1000

这是一个通体骨白的调色盘，这一物品会占据你的手腕、护符位，且你需要以一只不佩戴其他物品的手持有这一物品。  
不要去深究其中的色彩的来源，至少它们永不干涸。  
你的施法攻击获得【法破】，  
开放颜料的购买权，并且根据你已经购买的颜料，获得下列效果：  
最高等级为D级：你获得基于能量防御的【防弹】，  
为C级：获得基于能量防御的【防能量武器】，  
为B级：获得灵感视觉，  
为A级：获得心智类传奇属性上的附加成功\*1点能量防御。

颜料：

红

鲜血，岩浆，太阳  
D+1000

红颜料会为绘者提供颜料等级的核心法术造诣，  
这个等级会使色盘额外提供等同于智力上传奇属性\*2点法术强度。  
——支付D+500将该奇物升级至C级，  
此时该奇物提供在智力检定上的【9加骰】和【毁灭】。  
——支付C+1000将该奇物升级至B级，  
此时该奇物额外提供【加骰上升一级】（最高8加骰），  
并且一轮一次，在智力检定时可以重骰一次，并且俩次出目取高。  
——支付B+2000将该奇物升级至A级，  
原先提供的【9加骰】升级至【8加骰】，【毁灭】升级至【湮灭】，  
并且一轮一次，你的智力检定出目增加一半。

绿

植物，翡翠，极光  
D+1000

绿颜料会为绘者提供颜料等级的自然法术造诣，  
这个等级会使色盘额外提供一轮一次的，等同于心智类属性上的传奇属性\*1点法术能量消耗降低（最低降到1）。  
——支付D+500将该奇物升级至C级  
额外提供基于能量防御的【劣化1】和【坚韧】。  
——支付C+1000将该奇物升级至B级  
上一级效果中的【劣化1】升级至【劣化2】，额外提供给予能量防御的【适应】，  
在受到攻击时，若能量防御未被全部破除，一轮一次，敌人的攻击重新投掷，无论如何，取后一次结果。  
——支付B+3250将该奇物升级至A级  
此时在上一级效果中的【劣化2】升级至【劣化3】，  
在上一级效果中的重新投掷生效时，改为取对你较有利的一次结果，  
每轮一次，收到的近战伤害减少至原先的1/2。

蓝

天空，海洋，眼泪  
D+1000

蓝色颜料会为绘者提供同等级的咒击法术造诣  
这个等级的蓝色颜料会为色盘提供心智类传奇属性\*5M的法术射程，  
并且你的绘池现在开始，每小时一次，进行一次能量池关键属性检定，恢复成功数点能量，这会计入每日的恢复上限。  
——支付D+500将该奇物升级至C级，  
现在你可以用【表达+绘画】作为格挡专业计算格挡加值，如此获得的格挡防御转变为能量防御，  
你获得在伤势被严重伤害填满的情况下也不会因伤势晕眩的能力。  
——支付C+1000将该奇物升级至B级，  
现在你额外获得武器等级\*10M的法术射程  
并且当你的攻击命中时，你可以塑造一个敌人的虚像，这个虚像可以出现在你攻击范围的任意位置，并且攻击虚像等同于击中敌人，这是一个B级别命运来源的魂牵效果  
——支付B+2000将该奇物升级至A级，  
现在你额外获得【生息】效果，  
你获得基于能量防御的【折射】特性。
