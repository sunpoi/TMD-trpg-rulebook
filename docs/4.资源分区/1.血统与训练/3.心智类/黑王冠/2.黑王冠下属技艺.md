# 2.黑王冠下属技艺

黑王冠下属技艺

情绪感知  
D+500  
类型：被动/源石技艺

效果：获得一种非线性感知能力，能够模糊定位周围有思维能力的单位，这表现为你模糊的感知到了对方的思维，以这种感知方式可以感知到灵体，这是一种【灵感视觉】。

——支付D+500升级至C级，

非线性感知能力升级为强感知，并且在使用该感知方式的时候，获得D级特异本质的侦查能力。

——支付C+2000升级至B级，

先前获得的D级特异本质侦查能力升级为B级特异本质的侦查能力，并且在进行侦查检定时，改为以决心而非感知作为检定属性。

——支付B+2000升级至A级，

先前获得的B级特异本质侦查能力升级为A级特异本质的侦查能力，你的敏感范围翻倍。

情绪吸收  
D+500  
类型：被动/源石技艺

效果：【文明的存续】中获得的增强加值现在不在具有上限，  
获得传奇决心\*1点源石能上限。

——支付D+500差价升级至C级，

现在你可以改为令敌人使用意志豁免对抗你的攻击。

——支付C+1000差价升级至B级

若你在购买该技艺后购买过任一源石技艺，那么你的攻击附带等量的精神束缚异常点数。

待调弦的怒火  
B+2000  
类型：被动/源石技艺  
前置：【升变】-青色怒火  
效果:  
强化技艺-青色怒火，现在每场影片结束时，若在当前影片中参与并战胜了一次高一级战斗，那么令青色怒火提供的攻击上的增强加值和闪避防御再次提升传奇决心\*1点。

精神爆发  
B+2000  
类型：被动  
效果：  
当你使用决心作为主属性进行施法时，你的攻击具有【湮灭】【幻灭】。

天瞠  
B+2000  
前置：这个技艺只有使用【青色怒火】才能施展。  
描述：  
\[或许愤怒会毁掉我们，但是在面对恶意的时候，更多的人就是别无选择。在这片大地获得平静之前，没人能扑灭这股怒火。\]  
效果：  
消耗一个标准动作进行2次不附带招式的普通攻击，同时你的攻击结算伤害出目时最多将其中等同于此武器伤害点一半的严重伤害改为造成韧性伤害，这是一个【幻灭】效果。  
并且可以在攻击前将最多9点破甲转化为破魔。

诚挚期许  
D+1000  
效果：  
当你使用血统下属武器【慈悲愿景】时，你可将受到的其他来源的治疗效果传递给施法范围内的一名友方单位，令其替代你享受此次治疗效果的好处，但你无法将这一治疗效果传递给已经受到同一治疗效果的友方单位。举例而言，轮回者A对慈悲愿景的持有者与受伤的轮回者B同时使用了一个恢复能力，令双方立即恢复10点生命值，慈悲愿景的持有者将无法把这一治疗效果传递给轮回者B，因为其已经享受到这一治疗效果。

未完的故事  
C+2250  
前置：这个技艺只有使用【慈悲愿景】时才能生效。  
描述：  
那些离去者的名字，我都记得，那些战场上的恐惧、愤怒和痛苦，我也仍然能触摸到余音。但要在大家面前演讲的话，我更想说些心怀希望的故事。博士，我有点紧张......可以握住我的手吗？  
效果：  
现在在你使用慈悲愿景时，你决心上每有一点附加成功，都会为你提供1点基于偏斜伤害的全伤害吸收。  
同时你会获得3点偏斜防御.

预言的魔王  
B+2000

“那魔王戴着冠冕，拖着长裙，轻声地对我许下诺言”

B级：  
兑换者获得B级时空来源的【魔王】身份，这一身份所代表的是与神为敌者，因此在影片中，如果影片内存在并不皈依一神教信仰的势力，则这些人大概率认识你。这部分人群将会把你视作可以信任的同伴，当你与这部分势力内部人员交流时，他们可能会信任你，并一定程度上采取你的建议，你在影片中可能具有一定数量的追随者，你的追随者的行事风格和身份将会根据你的声望来决定（如美名较多时，你的追随者可能为受到不公对待者后放弃宗教信仰者等仅是不信仰一神教的人员，对应的，在恶名较多时，追随者可能多为愉悦犯等具有恐怖主义色彩的人员），你们会被称为【不信者】，特殊的，若影片的世界观允许，ST可以将你的追随者设定为恶魔等与神对立的种族。  
然而一神教信仰的宗教机构将会固定对你态度为冷漠，其下属人员对你的态度若不低于中立，则降低至中立。

当你的血统等级提升至A级时，这一背景身份的等级也会提升至A级，其身份所带来的效果会在传播范围、象征意义上有所增强。
