# 1.黑王冠

黑王冠  
血统

你看见了耀眼的光，而光渐渐远离，升到天上，留下一片斑驳。  
但你还能记起那些斑驳皆是天体，你想起星星们的家园名为宇宙。  
荣誉，财富，家人，理想.....这些词汇闪烁、盘旋、消散，最终只剩下一个，且仅有一个......  
......  
...  
未来

这一资源作为特殊身份5选取时，可以与天命-亚人共同选取，但会额外占据1点专长点，并且建卡时在耐力属性上减少1点。

源石技艺检定：决心+学识+源石学+专注下属专业：情绪感知

该血统中的同名特性效果效果不变，下不赘述。

D级  
D+600  
能量池：源石能  
特性：【未成熟的魔王】【文明的存续】

源石能：  
在活性化下，释放源石内部储存的大量能量。  
能量池属性：耐力+决心  
1.经过一次短休息的冥想后恢复关键属性较高者点源石能，  
2.经过一次完整的长休息后，恢复所有魔力，每日恢复上限为3倍能量池上限。  
基本用法：  
进行决心相关的检定时，消耗一点源石能，检定中获得1DP增强加值。

选开：  
特殊的，你可以在战斗外消耗一颗源石（100分1颗），加快源石能的恢复，你受到一点不可豁免的毒素伤害，并立即进行一次能量关键属性检定使源石能立刻恢复成功数点，依旧遵循恢复上限。

【未成熟的魔王】  
描述：  
“魔王”会赠他人幻景，魔王传承记忆与技艺，传承情感与知识。  
效果：  
你可以消耗一个自由动作召唤出一顶小小的黑色王冠漂浮在你的头顶，  
黑王冠正如它的名字一般，是一款漆黑如墨的王冠，与我们日常所认知的王冠不同，当拥有者佩戴黑王冠时，该王冠并不会以佩戴者的头作为支撑，而是在一股神秘力量的支持下悬浮在佩戴者的头顶上方，因此黑王冠并不会占据正常的头部装备位置，当佩戴者穿戴头盔时，依旧会悬浮在盔甲上方。  
你获得一种新的源石能使用方法：以一个自由动作消耗2点源石能，选择一名感知范围内的单位，你以侦查检定对抗对方的掩饰检定，若对方对抗失败，则你能够得知当前的情绪倾向是什么（疑惑、不安、急躁等），这是一个D级别、影响心灵来源的读心能力。

【文明的存续】  
描述：  
如果试着回忆，你还记得，从远古的远古开始，从火或者一次狩猎开始，你们是怎么走到了今天。  
效果：  
在第一场影片结束后，你获得一种新的源石能用法：以一个自由动作消耗2点源石能，令你下次决心检定获得传奇耐力\*1点增强加值，  
从该特性中获得的增强加值每场影片都会增加传奇耐力\*1点，最多增加至传奇耐力\*4点。  
通过源石技艺的转化，源石也可以释放出多种形式的能量。  
从下列三种攻击形式中选择一种获得，一经选择无法更改：

\[咒愈师\]  
使用源石技艺检定进行远程攻击，该攻击方式称为源石攻击，造成纯能量伤害，具有15M射程，  
\[近卫\]  
使用源石技艺检定使用白刃类武器进行白刃攻击。  
\[术士\]  
改为使用源石技艺检定作为施法检定，  
获得技艺-源石技艺，

源石技艺  
类型：魔法/源石技艺  
消耗动作：标准动作  
派系：-  
目标：单一目标  
射程：15M  
消耗能量：1点源石能

描述：通过源石施放令物质改变原有性状的技术，常被俗称为“法术”。  
效果：进行一次施法攻击，本次攻击造成纯能量攻击。

C级  
C+1200

特性：【未成熟的魔王】【文明的存续】【奇美拉】

【奇美拉】  
描述：  
“事情结束以后再去指责他们做了什么，又有什么用？想想践踏他们的人，虐待他们的人，屠杀他们的人......仇恨滋生仇恨。有些事从一开始就不该发生。”  
效果：  
在【未成熟的魔王】【文明的存续】中记载的源石能用法不在消耗源石能，并将其效果强化：  
【未成熟的魔王】中的读心效果升级至C级，此时的读心效果可以用于辨别是否撒谎。  
这之后的血统等级每提升一级，该能力的来源等级也会提升一级。  
当能力等级提升至B级时，你的读心能力对抗成功后，能够令你知晓目标当前轮次思想的具体内容；  
当能力等级提升至A级时，你的读心能力对抗成功后，能够令你知晓目标大脑内几乎所有内容，当然，你不可能在一瞬间将对方大脑内所有知识全数吸收，具体所需要的时间由st决定。  
【文明的存续】中提供的增强加值改为常驻生效。  
减少1点意志力上限，并且获得在决心检定上的【9加骰】。

B级  
B+2400

特性：【未成熟的魔王】【文明的存续】【奇美拉】【升变】

【升变】  
描述：  
“远逐者”因不进行同类相食而被逐出其氏族部落，然而却成为了最初的魔王，  
“砌城匠”在哀愁之地建起城市，随后死于他的同行伙伴掀起的焚火叛乱，后者继承了魔王称号，并成为了这片大地的恐惧，人们称呼他为“焕日者”，  
最后叛徒被同行伙伴中的第三人“青色怒火”独自杀穿军阵斩首，于是他成为了第四位魔王，又称为游侠君主，但没走出多远，青色的怒焰再一次遭到背叛，  
“挽歌领唱”试图操纵命运，但她的挽歌最后为自己而唱……  
此为诸魔王之遗物，不知有多少人仅是被选中就癫狂于情感与回忆。  
效果：  
在【奇美拉】中获得的【9加骰】提升为【8加骰】，  
根据在 【文明的存续】 中选择的攻击方式，获得对应技艺：

\[咒愈师\]-慈悲愿景

慈悲愿景  
类型：增强技/源石技艺  
动作：移动动作  
消耗：10点源石能  
持续：一场景  
作用范围：以自身为圆心，以射程为半径的圆形区域  
效果：  
进行一次格挡，随后为作用范围内包括自身在内的所有友方单位提供等同于传奇决心\*2点吸血。

\[近卫\]-青色怒火

青色怒火  
类型：增强技/源石技艺  
动作：自由动作  
消耗：-  
作用范围：触及范围  
效果：  
为作用范围内包括自身在内的所有友方在攻击上提供等同于你【传奇学识+传奇专注/运动/生存技能中最低者之和】的增强加值，并在提供等量的闪避防御。

\[术士\]-情感爆发

情感爆发  
类型：魔法/源石技艺  
动作：标准动作  
派系：-  
消耗：5点源石能  
持续：立刻  
效果：  
进行一次施法攻击，这次施法攻击中获得决心属性\*1点的增强加值，并且最终检定出目增加一半计算。

A级  
A+4800  
特性：【文明的存续】【奇美拉+】【升变】【魔王】

【奇美拉+】  
描述：  
大量对此物种的描述仅存在典籍中一般可通过血亲物种异同来判断，根据遗传学表现度不同，个体之间也存在很大差距。  
在天灾频率骤升的矿石工业时代，政权之间的对立让这种种族混合的产物几乎不可能诞生。当然，除了延续生命的常识外，可能还有一些危险方法可能会诞生奇美拉。  
如果她没骗我的话。  
“从现在起，  
我们要严格按照时间计划进行。”  
效果：  
保留【奇美拉】的效果，现在从闪避、天生、格挡防御中选择一项，你的攻击将无视所选择的那种防御类型，并且每轮可以切换一次无视防御的种类，这是一个A级别、命运来源的效果。

【魔王】  
描述：  
“一个萨卡兹的一生，或许会遭遇千种、万种形式不同的不公，被迫了解苦难......”  
“万般不公已变成常态，构成他们生活的全部......”  
“当他们被问起——你们遭遇的苦难到底是什么？”  
“他们哑口无言。”  
效果：  
保留【未成熟的魔王】中的效果，  
你免疫A级及以下的命运来源效果。  
你获得8点仅能用于血统下属技艺的AP。

S级  
S+9600  
前置：该血统等级只有在持有【终结的实相】以及A级黑王冠的前提下购买

特性：【无终奇语的终语】【休止符】【不容拒绝】【存续先兆】【永恒存续】

【无终奇语的终语】  
描述：  
存在于每个故事尽头，带走每位角色，封闭每种可能，停止每段讲述。你是对终结的想象，亦是所有想象的终结，你是一切，唯独不是你熟悉的自己。  
效果：  
继承【文明的存续】【奇美拉+】【升变】【魔王】的所有效果，并获得S级的命运来源免疫以及S级的心灵来源免疫，你选择的攻击形式的射程提升至敏感范围。  
在演出上，你的存在将会达到当前故事的“水面”之上，你所做的不过是将手伸入水中搅动。

【休止符】  
描述：  
剥夺他人的未来，这是否算是一种杀戮？  
可如果那未来便是死亡，这又是否是一种拯救？  
效果：  
当一名敌人被你的攻击击中，并且将会因此死亡时，他不会死去，而是会被你使用一层柔软的白光覆盖它，随之而来的，是坚硬的源石，在那之后的，则是完全的封存，此时的对方会陷入完全静止态，此时你可以将对方收入你编织的茧房中，若如此做，则对方不会死去，而是会在一日后复活，对方的思维将会被你影响，被你完全控制，这是一个S心灵来源的复活和胁迫效果，如果对方免疫了胁迫效果，则不会受到复活效果影响。  
你具有一整个静滞的世界，于其中，除了你之外，一切都被永恒定格，这个世界的边界是无限的，这是一个S级的异空间效果。

【不容拒绝】  
描述：  
只要还有“萨卡兹”，他们就能握手言和，重归于好。  
于是，在精心挑选下，人们将“萨卡兹”的概念安在了某一个地区上。  
当这个地区的种群被屠尽，他们又开始宣称新的“萨卡兹”所在。  
不停地归类，不停地杀戮。  
但只要还有“萨卡兹”在，他们就还拥有“团结与正义”。  
“萨卡兹”们进入了属于自己的悲惨年代。  
效果：  
你现在可以消耗一个整轮动作启用，你敏感范围内的全部敌人都需要进行一次反射豁免，DC为你的决心+学识+源石学检定出目，如果他们失败，那么他们将会被你拉入你的异空间，  
随后你进入【存续先兆】状态，此时在你的异空间中，S级以下的传送被禁止，当你的完好低于一半时，获得【不应存在之人】，这会为你带来一轮一次，完全免疫某次攻击，这是一个S级的无敌效果。  
每影片一次，当你第一次死亡时，你立刻复活，这是一个S级命运来源的复活效果。  
在【存续先兆】的状态下，你可以以一个标准动作将当前的故事永久裁剪，随后将其永久封存，你对攻击范围内的全部敌人进行一次攻击检定，本次攻击成功数增加一半，造成伤害翻倍，并具有【传奇】特性，这次攻击如同具有【超级贯穿】一样结算，对方无法使用应对技和反射动作调用动作对抗此次攻击，这是一个S级的先制效果，此时敏感范围内的全部敌人需要以一个强韧豁免对抗你的攻击出目，若失败，那么他们会被你连带着异空间一同封入茧房。这是一个SS级的创伤来源的即死效果。

若ST允许，你也可以用该效果影响整个影片，即将整个世界封存。

**_“——所有讲述者终将面对这一刻。，在想象枯竭，灵感丧失之时，盲目地追求结局，最终，迎来自己的末日。散发着温暖光芒的白茧轻柔地包裹住讲述者们，我在他们尚未察觉时便已完成了这一步骤，就像是午后小憩前的困意，轻轻将人引至沉睡。在那之后，一切便不再变动，悲伤就此斩断，欢乐永远停留，死亡也被驱逐，悻悻地从每篇故事中离开，在那之后，如夜空般漆黑的“源石”缓缓攀附，将周边固定，如此一来，便没有外力能够变动故事分毫。时光无尽，被固定的叙事如行星般环绕在它们的终结者身旁，见证着一段又一段故事加入它们的行列。”_**
