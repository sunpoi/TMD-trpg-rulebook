# 黑王冠下属武器——青色怒火

青色怒火

D+500

\[史载，萨卡兹君王奎隆的佩剑，长度约在0.9到1.2米之间，宽约5厘米，材质不明，通体呈黑色，剑身铭刻萨卡兹传统文字，释作“争斗在此止歇”。\]  
\[部分古籍宣称，在奎隆继位前，其佩剑就已于战争中毁坏，日后奎隆所持武器，只是他以法术塑造出的脆弱外形。\]  
\[传说，奎隆有令其敌人不战而退之威。有记述称奎隆之敌在与奎隆作战时仿佛会遭受数倍于武器重量之重压，在物理与意志上同遭奎隆捶打，如若侥幸不死，也多成残废。\]

这是一把武器伤害7L 破甲1的长剑，视作概念武装。  
常态下会作为类似于臂甲的黑色结晶附着在手臂上，你也可以用一个移动动作隐藏这些黑色结晶，此时如果需要使用这把武器，需要先以一个移动动作唤出结晶。

【空性】  
\[须常留意，打破固执自我之见解，观世界为空。如此则可超越死亡\]  
这并不是一把具有实体的武器，正常情况下无法使用。  
当你需要使用这把武器时，需要以一个整轮动作支付2点能量以及1点意志力来为其塑型，只有这样，你才能在这个场景正常使用这把武器.。  
在为这把武器塑型的同时，你可以以自有动作移动不超过移动速度的距离，并且对接触范围的敌人进行一次不附带招式的普通近战攻击。

【授记】  
\[燃灯佛与我授记，作是言，汝于来世当得作佛，号释迦牟尼。\]  
\[善哉，善哉！一切众生皆具如来智慧德相，但因妄想执着而不能证得\]  
这把武器具有【余波】效果，这表现为对敌人的精神和肉体的双重攻击。

【三惑】  
\[界外即是无明惑也，界内即是见思惑也。\]  
\[见思破故得位不退，真谛三昧成。\]  
\[菩萨应知，一切众生具有无量惑障，及对治众生惑障之无量教法\]  
\[菩萨虽以空观破除见思惑，但尚执于空理，致使未能进一步了知众生无边之差别相，故尘沙惑又称着空惑。\]  
由该武器造成的伤害，会对击中的敌人附带等量的【尘沙】异常点和【见思】异常点，分别豁免。

【尘沙】是作为一种特殊的一档异常点存在，要注意的是，在判断异常点数相关的能力时，该异常点数等同于亢奋点数  
人之惑障，如尘沙之多，迷于界内外恒沙尘数法所起之惑障，故称尘沙惑。受到尘沙影响的人物，在防御、先攻、互动系技能判定和所有延长动作的判定上受到减值。  
抵抗不良状态：意志豁免。  
陷入重度状态的人物称作进入尘沙惑。进入尘沙惑的角色无法静下来。他无法进行任何需要他呆在某个地方静下心来的动作，包括禅定、打坐、冥思等，  
并且他的睡眠浅而易醒。他无法通过以上行为获得除自然休息之外的好处（如能力充能或回复能量池）。由于没有耐心，他无法进行周期在分钟以上的延长动作。

【见思】是作为一种特殊的一档异常存在，需要注意的是，在判断异常点数相关的能力时，该异常点数等同于疲乏点数，但该异常点数的来源由创伤来源改为心灵来源。  
迷于界内之理为见，迷于界内之事为思。受到见思影响的人物在力量和敏捷相关判定、基础速度上受到减值，不能冲锋攻击或全力攻击。  
抵抗不良状态：意志豁免。  
当见思点数高过人物的耐力或力量的较高者时，人物进入见思惑。  
见思惑的人物基础速度减半，然后计算其因疲乏受到的减值。此外，他无法转化出移动动作。  
见思惑也被视作进入竭力状态。  
见思同时也视作疲乏点数，与其他疲乏点数叠加计算。  
毁灭性后果：见思惑系取六道生死之相。当人物的见思点数超过耐力或力量的较高值之两倍时，将超出部分作为属性伤害，平摊在身体属性上，优先顺序为力量、敏捷、耐力。  
见思带来的属性伤害不会令对方的属性值降低至0。

C+1000  
在这个阶段下，青色怒火上常存青色幽光，同时有着源石的内化红色纹路  
青色怒火的武器伤害提升至15L，破甲1  
\[文献记载，奎隆继位后，其佩剑便时常散发青色幽光，如同焰息。由于奎隆性格激烈，此剑日后也被称作“青色怒火”\]  
\[有人称奎隆为仁爱君主，任何遭其佩剑划破的创口都不会流血，如若退后，则能苟全性命。\]

【空性】  
\[所言空者：无常，无恒，无不变易真实法故。\]  
\[空者：谓离一切烦恼等故。\]  
\[由我我所、我慢执着、及与随眠，皆远离故；说名为空。\]  
当你为这把武器塑型时，消耗的能量提升至6点，意志力消耗提升至2点。

【授记】  
\[记别者，谓广分别略所标义，及记命过弟子生处。\]  
\[言记别者，谓随余问酬答辩析，如波罗衍拏等中辩。或诸所有辩曾常现真实义言，皆名记别\]  
现在这把武器额外获得【碾压】特性，这代表这武器对敌人的数倍重压。

【棒喝】  
\[上堂，僧问：‘如何是佛法大意？’师竖起拂子，僧便喝，师便打。\]  
现在青色怒火获得一个名为【嗔】的计数器，随着【嗔】的增多，青色怒火上会逐渐出现更多的红色纹路  
【嗔】的获取取决于你回合中的行动，每当你使用标准动作进行攻击检定并且造成伤害之后，都会获得一点【嗔】，每回合最多只能获得一点【嗔】，最多获得3点。  
每点【嗔】都会使在进行全力攻击时额外使武器伤害提升耐力上的传奇属性\*1点，同时使全力攻击失去的防御额外加1。

B+2000  
在这个阶段，青色怒火的伤害提升至22L，破甲1  
在未被使用的状态下，青色怒火会同时浮现在使用者的脑后，呈现出黑色光轮的形态  
\[三千大千路迦，生者必有死期，轮回亦有竟时。浩劫乃至，我不能亦不愿革汰之\]  
\[法成而不权变于因缘，终不失于梦幻泡影\]

【空性】  
\[所知者：谓于众生遍计性所执法中，及法遍计性所执法中，此二遍计性，俱离无性；  
及彼所余无我有性。于诸法中遍计性无，卽是无我性有。于诸法中无我性有，卽是遍计性无。  
卽于此中有及非有无二之性无分别境。智者：谓缘彼境，如实了知。\]

当你为这把武器塑型时，消耗的能量提升至10点。  
此时使用该武器的攻击可以攻击触及范围内的全部敌人。

【三惑】  
\[善男子，空实无华，病者妄执，由妄执故，非唯惑此虚空自性，亦复迷彼实华生处，由此妄有，轮转生死，故名无明。\]  
\[无明者：谓由亲近不善丈夫，闻非正法，不如理作意故；及由任运失念故；于所知事，若分别不分别染污无知为体。\]

现在，由该武器造成的伤害，  
同时会对击中的敌人附带等量的【无明】异常点.。  
【无明】：无明是一种特殊的二档异常点，要注意的是，在判断异常点数相关的能力时，该异常点数等同于晕眩点数  
关键抵抗属性：耐力、决心、沉着  
昏迷痴暗，障蔽中道实相理，于诸法事理无所明了，所以称为无明。无明的角色在攻击、运动、洞察判定、基础速度上受到减值。  
抵抗不良状态：意志豁免  
无明点数超过人物的耐力和决心之较高者，则人物进入无明惑。

无明惑的人物失去意识，视为无助状态，无法主动作任何动作，也无法行动。  

【棒喝】  
现在，在你使用青色怒火战斗时，获得基于闪避防御的【适应】以及A级的免疫措手不及  
当你于一场同级战斗中获胜之后，在影片结束之后，你的全力攻击额外具有【湮灭】，并且可以在移动的同时攻击。  
  
  
A+4000  
前置要求：这把武器必须由【阿纳萨羯磨】和B级的青色怒火，在B级黑王冠的前提下才能支付价格来合成。  
在这个阶段，青色怒火的武器伤害提升到34L，同时攻击上额外具有1点附加成功。  
如果你具有身份背景，那么无论如何，你的背景身份都会固定为【摩诃萨埵权化】，意为大菩萨显灵。  
  
当你使用整轮动作塑型青色怒火后，不再进行攻击与移动行为，而是改为进入【入定】状态。  
  
**入定**  
进入此阶段后你在当前场景获得以下效果：  
  
【不着相】  
\[凡所有相，皆是虚妄。  
佛曰：“我相即是非相，人相、众生相、寿者相即是非相。何以故？离一切诸相即名诸佛。”  
“若菩萨有我相、人相、众生相、寿者相，即非菩萨。”  
或如前人所云：“诸众生无复我相、人相、众生相、寿者相，无法相，亦无非法相。”  
凡夫俗子：心念执着，意想住相。  
故向佛者当破执扫相。  
离相无念，不住于相。  
“无住无相，信心清净，则生实相。”  
何谓实相？  
实相者即是非相  
“若见诸相非相，即见如来”  
是谓实相。  
无相不相  
放眼无相，心见实相\]  
效果：你无法进行主动移动，无法使用或者转化动作，你处于全防御与格挡状态。  
在此期间你的身体会进入莲瓣空间的中心，这是一个A级别、命运来源的异空间，其中无法使用A级别及以下的传送效果，其大小为你的敏感范围。  
与其他的异空间不同，处于异空间外的单位能够目视你留在现实的残蜕，这意味着他们能够建立对你的感知线。  
当你失去此效果从异空间脱离后，你会出现在你残蜕所在的位置。  
  
【清火执】：即于两肩起大烟焰，龙退风静，雾卷云开……忽有烟起，少间便出猛焰，时人谓窣堵波已从火烬。瞻仰良久，火灭烟消，乃见舍利如白珠幡，循环表柱，宛转而上，升高去际，萦旋而下。  
现在，你在每轮结束时可在敏感范围内选择一个目标点，在目标点处生成一朵【尼卢火】。  
当场景内每具有一朵尼卢火存在，你将获得1点基于偏斜防御的全吸收与全异常点数吸收，尼卢火的存在上限至多为你的传奇决心\*3朵。  
在你的行动轮开始时，处于你敏感范围内的尼卢火会对其周围50米内的敌方单位进行一次攻击，这次攻击需要使用反射豁免，其攻击检定的构成等同你以青色怒火进行的不附加招式的普通攻击。多个尼卢火对同单位进行的攻击视为同一次攻击。  
  
  
**自在**  
当你处于【入定】状态三轮或者受到伤害后，你会在下一行动轮开始后进入【自在】状态，这一状态下你失去【不着相】特性并在当前场景内获得以下效果：  
  
【怒目】：是以佛陀开眼、金刚怒目，于空性入尘世涤荡百寰。  
现在你以青色怒火进行攻击时，触及范围提升至你的敏感范围，并能够攻击范围内的所有敌方单位。  
每轮一次，你在进行攻击时，若目标同时处于尼卢火周围50米内，则此次攻击造成的伤害翻倍。  
  
【承三身】：  
每场景一次，当你在场景内的生命值没有任何完好时，你的敏感范围内会随机出现三朵托生莲瓣（体积为5），分别象征你的过去、现在、未来，当有单位处于托生莲瓣之上结束自身回合，其会被传送进入莲瓣空间。  
每日一次，当你承受致死性伤害时，你会免疫这一次伤害，这是一个A级别、命运来源的无敌效果。  
  
**无忧觉**  
当你处于【自在】状态受到致死性伤害后若仍然存活，则会进入【无忧觉】阶段，你获得如下效果：  
  
【菩萨辨】  
捷疾辩，于一切法无碍，故得捷疾辩。  
利辩，有人虽能捷疾，但以钝根故犹不能深入；以能深入故为利。  
不尽辩，说诸法实相，无边无尽。  
不可断辩，于般若中无诸戏论，故无能问难断绝者。  
随应辩，能断法爱，故能随众生所应而为说法。  
义辩，为说趣于涅盘之利益事。  
一切世间最上辩，说一切世间第一之事，即指大乘。  
效果：  
你立即进入莲瓣空间的中心，当你在处于莲瓣空间内，你获得等同你决心属性的临时生命，这一部分临时生命场景内无法刷新。  
当你处于莲瓣空间内进行攻击后，你能够额外进行一次普通攻击，这一效果每轮只会生效一次。
