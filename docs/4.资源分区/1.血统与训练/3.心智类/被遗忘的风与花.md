# 被遗忘的风与花

被遗忘的风与花

  
在被遗忘者的乐园之中，这个女孩作为本应诞生的存在，被这片乐园所遗忘了，她是不幸的，因为她本身或许能拥有更高的人气，她是幸运的，在本被应该删除的所有记录中，她的名字，被保留了下来“冴月麟”

  
D级  
价格：D+600

  
种族？：你或许拥有麒麟的血统，麒麟是中国的圣兽。特别是雌性的麒麟，被称为“麟”。但作为本应存在的第三个自机角色，你更有可能和另外两位主角一样，同属人类，当然这种薛定谔的种族也为你带来了一些好处，可能存在的麒麟的血统为你提供了黑暗视觉和快速医疗1（12医疗点），但你仍旧在任何情况下被视为人类  
同时，你解锁妖力能量池

  
  
所持有的物品？：在你诞生之初，你的身旁便跟随着一把写着唐人茅的六角二胡，虽然你不知道那是否真的是你的武器，但你还是拿起了它“茅”是大祓会(驱邪活动)中消除灾厄的仪式所用的“茅之轮”的原料。看着上面的文字，你不禁思考你存在的意义，你获得了一种新的攻击方式，拨动那把不知是否存在的六角二胡，弹奏那不知是什么风格的乐曲  
你现在可以由【决心+学识+专业：乐理学+辅助专业：回忆】进行攻击， 攻击距离15m，攻击造成精神伤害，且可以额外攻击攻击范围内的2个目标

  
C级  
价格：C+1200

  
被遗忘者：该角色从未实际登场，没有官方造型，一设中没有任何立绘，也没有任何造型的描述。即使在战斗中，你也不断地被他人所遗忘着，这并非你的原因，因为你只是，被删除的各种代码而已，你获得C级科技本质的反侦测，但以此反侦测等级在进行侦测对抗时，即使对方在与受影响单位进行的侦测等级对抗与定位对抗中失败，也能模糊定位

  
升级-种族：麒麟的血脉流淌在你的身体里，即使你可能从未拥有过这个血脉，但它或许已经成长了，你再次获得1点快速医疗（医疗点12）与D级提供的快速医疗叠加

  
升级-所持有的物品？：你越发熟练地能够弹奏那把二胡，看着二胡上面的词汇，和那个唯一留存的名字，你在思考，你是否是来消灾祈福的呢？你的攻击检定变为【决心+学识+专业：乐理学+辅助专业：消灾学】 ，你也可以不选择变化，继续回忆着你的存在本身，这个效果将转而改为令你的攻击检定获得1点附加成功的加值

  
B级：  
价格：B+2400

  
升级-被遗忘者：你的一切在被逐渐完善，你感受到自己的样貌，自己的性格，自己的能力，但是……这真的是你自己吗？你不知道，也没有什么人会知道，你获得B级科技本质的反侦测，但以此反侦测等级在进行侦测对抗时，即使对方在与受影响单位进行的侦测等级对抗与定位对抗中失败，也能模糊定位

  
  
  
职业？：护士，那是你过去的职业吗？既然是护士，那或许就应该帮助他人吧，消灾祈福，如同你那可能存在的麒麟血统一般，为她人带来幸福吧  
现在，你可以将你血脉所带来的快速医疗再次加1，可以以一个自由动作消耗1点妖力，将自身血脉提供的快速医疗效果给予以自身为中心，半径为射程/4的圆形的队友，持续一场景，共用医疗点，  
同时，你也可以为以自身为中心，半径为射程/4的圆形的队友提供全遮蔽效果，持续一轮

  
  
冴月： ZUN喜欢的作家菊地秀行的小说《しびとの剑》系列故事的主人公名为冴月紫靡帝。是战国时代一骑当千的美男子。死后百年通过秘术从黄泉国复活。因此，你或许也拥有类似的能力？  
当你死亡后，你可以花费3个月的时间复活，。在复活过程中以灵体形式存在，除了观察和移动外无法进行更多的动作，该灵体拥有等同于你生命值的结构值，被摧毁则复活失败。这是一个B级命运来源的复活效果

  
A级  
购买价格：A+4800

  
升级-被遗忘者：现在，你得一切只能在《红魔乡》主程序中找到残余的相关代码。如果用文本方式打开 東方紅魔郷.exe，就能在其他自机附近找到“冴月麟”的名字。这是你唯一被证实的事物，你获得A级科技本质的反侦测，但以此反侦测等级在进行侦测对抗时，即使对方在与受影响单位进行的侦测等级对抗与定位对抗中失败，也能模糊定位  
与此同时，因为你已经被遗忘了，意味着所有的改变将不再影响你，你获得A级命运来源的免疫效果，同时，你的射程提升至敏感范围

  
  
第三个主角：在最后一刻前你都将作为第三个主角出现，但实在是没有时间了。即使立绘都已经画好了。但是……最终也还是未采用角色。  
作为本应存在的第三个主角，当你进入影片时，你将获得一个有特殊经历的身份，能解释自身能力来源，并附带记忆，附带特殊渠道的人际关系，附带正常人应有的财产，  
附带符合安排身份的消耗品（需要ST临时安排，无法带出影片），这是一个B级命运来源的背景身份

  
  
风与花：从残余的代码中，可以看到，你的名字后面有着【风】和【花】二字，这或许是你本来将会使用的符卡名字，但你并不知晓，这些有什么用处……风去风来，花开花落，或许注定了你的一生，终将消散，随后，被遗忘  
现在，当你进入一部影片时，你可以从风符和花符中选择一种效果进行携带，在影片中可随时以迅捷动作进行切换  
【风】：你射程范围的友方单位将同样获取你的A级命运来源免疫效果以及该血统提供的A级反侦测效果，并获得异常点治疗2（48医疗点），近战伤害吸收1  
【花】：当你进行攻击时，你的攻击将附带等量的沮丧点数和精神束缚点数，意志豁免
