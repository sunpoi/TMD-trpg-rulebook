# 拟态兽

名称：拟态兽  
类型：血统  
本质：魔幻

概述：拟态兽是魔王大军的进攻的先遣部队之一，主要是使用他们的天赋技能来为魔王大军提供情报和暗杀，它们潜伏在各个种族之中收集情报与干扰敌方决策击杀敌方重要单位，他们的成年本体是一副类似于飞蛾的虫形头部和八只细长的手臂加上略微细小的甲壳身躯（还要一条长着细毛的类似于蜥蜴的尾巴），它们没有性别，繁衍主要依靠伪装成雌性生物，收集其他种族的基因来自我孕育出新的拟态兽，所以拟态兽在拟态的时候会偏向雌性生物的外貌，且魔力的的特殊性质让拟态兽能够伪装成各种职业者，但是它们的身体会比较柔弱，所以一般拟态兽伪装形态下都是伪装治疗者或者远程职业。

D级  
等级名称：幼年期  
价格：D+600  
特性：【本源】【力量】【特质】

描述：  
在成年拟态兽的胎盘中孕育的幼年拟态兽会以和母体相同的种族外表与基因进行拟态，这一伪装会模仿母体种族的正常的生理特征与成长过程，成长中它们会往当前种族的优秀外貌表达中可爱怜悯这一特性的倾向进行成长模仿，这是为了幼年体的安全成长所做的本能拟态。

效果：  
【本源】  
魔力产物：开始你的魔族形态还没完全发育形成，你的基本外表和你建卡时样子差不多，在当前阶段不能违背基因中为了安全成长而做出的本能拟态，但是的种族依然是【魔族】并视为“黑暗生物”且获得“黑暗视觉”特性，而身为魔族的你本能的开启了一种特殊的名为本源魔力的变种能量池（感知+操控）恢复方式与魔力能量池一致，能量池的基础用法是当仅进行一次以感知为主属性的检定时，可以选择消耗1点本源魔力，这次检定获得1点感知属性的检定加值，也可以支付1点魔力，使自身获得1点洞察防御，皆持续一轮，并且你的魔力会随着时间的流逝而壮大，每经过一个影片你都会令自身的魔力愈加成熟，将【本源魔力】能量池上限提升3点

【力量】  
神破：并且获得名为“神破”的施法攻击方式，朝向目标方向轻轻一握，以使用感知+学识+学识下属-专业：灵魂学+洞察下属-能量感知（辅助专业） 进行施法检定，能力的攻击射程为15米，造成精神伤害并且当你度过两场影片且每场影片都经历过最少一次同级战斗，那你的的神破攻击会附带破法效果。

【特质】  
怜悯外表：拟态兽天生是善于根据目标的人性与本能来达成目的，在这特殊特质中幼年体也有其天赋，在小时候就善于利用其优秀的外表（内在外貌加值+5）。

C级  
等级名称：成长期  
价格：C+1200  
特性：【本源】【力量】【特质】

描述：  
经过幼年期的潜伏，你终于可以展现真正的力量，在这个阶段可以显现你自己的拟态兽真身，也可以施展拟态兽专属的魔力技艺，且变为真身时你能获得更强大的力量。

效果：  
【本源】  
魔族真身：在这一时期随着魔力与魔族本质的成长，你的魔族形态已经完全形成，以一个自由动作使用变为真身，你的头部如今会穿变为飞蛾类的毛绒型虫首，巨大的复眼，根据性别与蛾类物种的细微不同有不同差异化的长触须，你的体积依然不变，在真身状态下，你的天生武器伤害若不足3L则提升为3L；但所有看到你真实形态的人都会因此对你降低一级好感（可以以此降低至极端），这是一个D级变形来源的变形能力。  
【力量】  
魔力延伸：随着魔力与技艺的熟练度提高现在你的“神破”攻击射程提升传奇感知属性\*5米。  
【特质】  
拟态伪装：你的伪装能力在这一时期被激活，你现在可以随时随地伪装成他人的模样，也可以伪装出各种种族特征和体积差异不能太多的生物，这是一个C级变形来源的伪装能力。

B级  
等级名称：成长期  
价格：B+2400  
特性：【本源】【力量】【特质】

描述：  
经过漫长的磨练，你的力量与魔力已经圆满了，这一时期光靠时间是无法再进步的，你只能走向一条道路，那是一条充满杀戮的晋升之路。

效果：  
【本源】  
晋升任务：最初购买该等级时，不会提供任何效果。  
从购买开始，你将在主神的见证下，在影片中完成【晋升任务】，进入影片后由主神发布任务，一般是击杀某个有影响力的人物，其任务难度至少与拟态兽本身支线等级相当，此任务由主神发布且作为支线任务存在，当完成该支线任务的影片结束后，你将永久晋升为魔将，你的魔族本体外表将会多出各种尖刺与繁复的几丁质甲壳纹路。

不灭诅咒：获得魔将这一阶位，你会获得魔将的共有能力，你被杀死的时候，你的肉体内将凝聚一颗体积为1的魔核并吸收世界的负面力量，魔核结构值为你的生命值上限，并且在你视野中出现一个13格扇形，以每6天一格的填充一格，当填满13格时，你将重新复活并重返世间，如果魔核被摧毁则复活失败，这是一个B级的诅咒来源的复活能力，在复活期间只能以灵体进行观察和移动。  
【力量】  
弱小粉碎：魔将级别的力量令你在进行神破攻击时拥有更加强大且多样化的攻击方式，当你进行神破攻击时，可以额外消耗6本源魔力切换攻击形态与效果，每轮仅能切换为其中一种，切换后，本轮施展神破必须额外支付等量本源魔力，否则特性效果消失直到重新切换攻击形态。  
无论以何种攻击形态进行神破攻击，命中并造成伤害后，承受伤害的目标总是会流失2点魔力。

\-镇压：选中自身射程内的一个目标，以攻击目标为圆心，效果线改为从上而下的半径3米垂直型圆柱范围攻击，若本次攻击为本轮首次施展神破-镇压，则可以令所有承受这次攻击的目标基于这次攻击伤害额外进行一次强韧豁免，失败则视为向你的方向拖拽靠近豁免差值\*1米。

\-剿灭：以自身为圆心，进行半径等同于神破攻击射程\*1米的圆形范围攻击，若本次攻击为本轮首次施展神破-剿灭，则可以令所有承受这次攻击的目标基于这次攻击伤害额外进行一次反射豁免，失败则视为向你所在方位相反的方向冲撞并被击退豁免差值\*1米。

【特质】  
腐化：现在庞大的魔力可以让你通过腐化生物的思想，被你的攻击命中的敌人需要额外进行一次意志豁免对抗伤害出目，失败则混乱攻击周围生物，会避开攻击者，这是一个B级诅咒来源的混乱效果。
