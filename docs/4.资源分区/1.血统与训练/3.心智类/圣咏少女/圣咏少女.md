# 圣咏少女

名称：圣咏少女  
资源类型：可升级血统  
本质：魔幻本质  
购买前提：创建人物的风度不低于3，且性别为女性。  
概述：如出自《魔塔大陆》世界观下设定，名为蕾瓦蒂亚的女性人工生命体被制造出来，通过与“塔”的精神链接来以歌唱的形式施展【诗魔法】，其功能涵盖医疗、生产、战斗等多个领域。  
所谓“塔”即是一种集合整个魔塔大陆中人类与非人类所制造出来的更强大的人工生命，在这里主神用其他方法代替了与塔的链接，以便于这些被称为“蕾瓦蒂亚”也就是圣咏的少女们发挥力量。  
当然，具体能力与原作并不相同，只是主神同样采用歌唱的类似形式所设计出的一种强化体系。

D级  
等级名称：圣咏预备役  
价格：D+600  
特性：【愿力】【塔语】【诗魔法】

描述：“塔”语也称之为Hymmnos语，多为强调感情色彩，词汇大多和神明，自然，信仰，救赎有关，对于她们来说以“塔”语构成咒文，链接导力，从而施展诗魔法实现自己的愿望是本能一样的事情，而圣咏唱诗班的预备役正是这些新生的“圣咏少女”。  
每个圣咏少女会在经过训练后编入骑士团，以搭档的形式来配备骑士战斗，对抗大陆上对人类造成严重威胁的魔物，于是，与普通人类外貌无异的你默默的苏醒了。  
但是，你知道，你有一个作为圣咏少女必须履行的职责。

效果：  
开启圣咏少女下属技能树  
【愿力】  
你开启愿力，愿力是一个西方通用能量池，关键属性为耐力+决心，它不代表传统意义上他人的愿望力量，而是单纯代表了作为圣咏少女能够在短时间内爆发出多激烈多少感情，以使用诗魔法达成自己愿望的力量，所以被称之为愿力。  
愿力的恢复方式：在一次至少十五分钟的短休息中平复内心情绪，进行一个愿力关键属性检定，恢复成功数点愿力。  
进行一次长休息，愿力恢复至上限。  
强化者一日恢复的愿力上限为能量池上限的三倍。

你可以支付2点愿力，使一次决心检定获得2DP加值。

【塔语】  
你获得一种新的2级语言，名为塔语，语言情感根据句调而音调来表达，语句优美柔和，极其适合表达诗歌，因此相关的咒文也称之为诗魔法。  
你可以把塔语加入诗魔法的攻击检定或者正常歌唱检定当中，塔语在攻击检定中如同武器伤害一样生效，在歌唱检定中等级越高表现的越优美。

【诗魔法】  
诗魔法的检定方式是以决心+表达-歌唱+专注-魔力洞察进行检定。此外，你可以用决心代替操控/风度属性来进行歌唱检定。

你获得了一种新的攻击方式，即以诗魔法检定进行攻击，射程15米，这种攻击形式被称为“Hymmnos Spell”，外在表现为不通过歌唱，而是简单的音节链接塔，并凝聚出攻击，以光束的形式进行攻击，造成光能伤害。

C级  
等级名称：圣咏唱诗班  
价格：C+1200  
特性：【诗魔法】【治愈诗篇】

描述：经过培训后的圣咏少女将会正式加入唱诗班，学习到在这个基础之上更多的咒文，以及Hymmnos Word，也就是所谓的“诗魔法”，而并非简短的咒文。  
你脱离了预备役的身份，与自己的搭档来到了这片星空之下，湖畔之旁，你的歌声打破了这片沉寂，柔和的音色在这里荡漾，扩散，  
你知道，他也知道，你们还不能停下，因为还没能做到，那个愿望，必须实现。

效果：  
【诗魔法】  
所有歌姬均可以使用，通过与自身与“塔”的连接，将自己的愿望具现化的过程。  
Word的多彩程度取决于歌姬自身内心的丰富程度，因此可以由歌姬创作，其威力取决于愿望表达的强烈程度及“塔”对愿望的回应，而不是具体的文字。而你继承了那位圣咏少女的心意，所拥有的诗魔法更倾向于治疗方面。  
你的诗魔法射程提高10米。

【治愈诗篇】  
你获得了另一种有关诗魔法的用法，即每回合以一个整轮动作维持，视为你清唱了一曲温暖，平和，治愈人心的诗篇。  
只要你还在清唱诗篇，就能指定诗魔法射程内一个单位获得6点快速医疗（医疗池72点）。  
你可以选择在维持期间每轮消耗6点能量，若如此做，则快速医疗生效期间，指定的目标还会获得异常点数治疗1（医疗点24）。

研发（C+2000）：你可以通过训练提升诗魔法的射程，购买后将提升决心\*5米的射程距离。

B级  
等级名称：纯血圣咏者  
价格：B+2400  
特性：【Hymmnos Extract】

描述：对于第一世代的Origin歌姬及第二代的β-纯血种来说，她们拥有者次时代圣咏少女不曾拥有的能力，即特定的文字列hymmnos code。可以控制天气，重力，也有可以撼动天地的魔法，以及塔各种环境的控制，塔守卫者（Guardian）的控制等。  
游戏中的hymmnos即指的是Hymmnos Extract。?

你拥有了更多的力量，也承担了更多的责任。  
是烈风凛凛，是满地黄沙，是金戈锃鸣，你与搭档一步又一步，走遍了无数战场，目睹着这个世界的争斗，你的歌声却从未停歇，你明白自己的无力，歌声虽然能治愈伤痛，带走悲伤，却无法修补这个世界的错误，但你仍然坚持着歌唱，那是终将也必将实现的愿望。

效果：  
【Hymmnos Extract】  
在任何一个战斗或非战斗场景中，你可以以一个标准动作清唱有关hymmnos code的诗篇。  
此时，进行一次诗魔法检定，以自身为中心等同诗魔法射程的范围内中的所有角色，都需进行一个意志豁免检定以对抗此次检定出目，若其对抗失败，则将立即进入一个新的空间，浮空高塔林立，俯瞰下方无边无际，永远无法抵达的的大陆。

你与这些角色将进入该空间中，在这片空间中时，你与这些角色不会受到空间之外的角色攻击，若这些角色或其他空间外角色拥有传送能力，则可以照常出入此空间或攻击空间内的角色。这视为一个B级环境来源的异空间效果，持续一个场景。  
在这个场景中，你才可以利用诗魔法来改变环境，控制天气，但具体对人所造成影响的部分只能概括为：强风/高温/潮湿/失重等，必须在购买时即选择是哪种天气与场景环境，这是一个B级时空来源的环境效果。

在这片空间里你视为具有等同基础移动速度的飞行能力，机动性普通。  
无论如何，这里的环境不会对你产生影响，但其他进入的人都会受到影响，即使是你的队友。  
异空间的范围会随着你的移动而移动，保证以你自身为中心，但无论如何其他角色也不会因此脱离边界。

A级  
等级名称：初代圣咏者  
价格：A+4800  
特性：【传承】【和平】

描述：初代的圣咏少女为平息“塔”的怒火，创下了如此的诗歌，以自己的牺牲封印了“塔”，让大陆回归和平，回归到了魔物出现以前的时代，这段诗歌如今随着血脉一同传承于你的记忆之中。  
【和平】：  
“为何力量总要寄宿于无欲之人。”  
“而又为何力量折磨着不喜纷争的美丽之人？”  
“力量的孩子啊……”  
“若您厌恶纷争，希求永远的安宁。”  
“那么就让力量与这身体一同沉眠吧。”

效果：  
【传承】  
你的诗魔法射程提升决心\*4米。

【和平】  
以移动动作维持，你可以反复咏唱着这篇象征希望的诗歌，此时每轮皆进行一次诗魔法DP攻击检定，可以如常造成或不造成伤害。无论如何，射程内的所有敌方必须以意志进行豁免这次本应造成的伤害。

若豁免失败，则立刻承受A级命运来源的睡眠效果（没错不是一个心灵影响），时间持续为永久，只有你可以主动将其唤醒。若ST允许，则你也可以选择使豁免失败的单位失去战意，无法进行攻击或继续战斗。

这视为你清唱了一曲和平，安宁，远离纷争的诗篇。
