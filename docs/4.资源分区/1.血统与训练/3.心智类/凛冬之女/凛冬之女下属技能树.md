# 凛冬之女下属技能树

凛冬之女下属技能树：

法术名：冰爆  
价格：D+500  
动作：标准动作  
分类：攻击技  
派系：核心  
消耗：2  
目标：射程直线上所有敌人  
距离：15m  
持续：立即  
描述：瞬间降温并引爆空气中被聚集的魔力，施法后遗留的无数冰棱可以做为冷饮甜品的原材料。

效果：  
进行一次施法攻击，此次攻击造成冻寒伤害，并获得1点施法鉴定上的附加成功。

研发10xp：

法术名：暴风雪  
动作：标准动作  
分类：攻击技  
派系：核心  
消耗：6  
目标：射程扇形上所有敌人  
距离：50m  
持续：立即  
描述：刺骨的寒风以施法者为起点呼啸而出，卷起漫天雪花，迅速覆盖大地。雪花在半空中旋转拉扯，凝结成无数锋利的冰刃在暴风中疾驰，犹如怒吼的野兽发出凛冽的破空声。

效果：  
进行一次施法攻击，此次攻击造成冻寒伤害。

研发20xp：

法术名：霜星陨落  
动作：标准动作  
分类：攻击技  
派系：核心  
消耗：9  
目标：射程圆形上所有敌人  
距离：75m  
持续：立即  
描述：天空中浮现出一个巨大的冰蓝漩涡，仿佛是另一片宇宙的门户被打开。漩涡深处，一颗散发着寒冷光芒的小型陨星缓缓显现，拉出一道寒光闪耀的冰晶尾焰，夹杂着漫天繁星般无数细小的冰晶坠向大地，掀起一场冰晶风暴。

效果：  
进行一次施法攻击，此次攻击造成冻寒伤害，附带伤害等额冰冻点数。

法术名：涟漪涌动  
价格：D+500  
动作：移动动作  
分类：增强技  
派系：无  
消耗：2  
目标：自己  
持续：一轮  
描述：通过将魔力提前散播的方式，使魔法的威力能更迅速和完整地扩散。

效果：  
使用涟漪涌动后的一轮内，下一道类型为攻击技的法术射程增加30米，并获得5点【高速】。
