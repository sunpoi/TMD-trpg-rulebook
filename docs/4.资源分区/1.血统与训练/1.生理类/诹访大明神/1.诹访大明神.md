# 诹访大明神

诹访大明神

资源类型：可升级血统  
本质：自然本质  
购买前提：性别为女性

概述：  
诹访子是居住在守矢神社的谜之神明，种族为八百万之神。八百万之神没有作为基本的灵魂，是由纯粹的信仰所组成的。拥有着统领祟神的力量，掌握着创造坤程度的能力。所带来的祟神的正体，是在外部世界被叫做御左口的土著神。所谓的土著神，就是只有在狭小的一块范围里被信仰的神明。虽说离开土地会失去信仰，但是限定在本土作战的话，拥有比在广泛范围被信仰的神明（即使是最高等级的神明）更强的力量的也不在少数。  
真正身份是洩矢神。现在常常与诹访地区其他传说的神明（包括一些作祟神）被视为同一位神明，  
不过在东方中，诹访子是能统领作祟神的神。

D级  
等级名称：“青蛙神”  
价格：D+600  
特性：【神具「洩矢的铁轮」】【“ケロ帽”】【「祟り」】  
描述：  
【神具「洩矢的铁轮」】：  
\*《諏訪大明神絵詞》中记载，洩矢神和建御名方神作战的时候所使用的武器是铁轮。  
【「祟り」】：  
描述：神 这个特性上，外形几乎没有意义”。（《求闻口授》）  
目前还只是伪·八百万神，似乎还没受到信仰缺失的影响。  
【「名存实亡的神明」】：  
描述：诹访子的王国曾遭到大和诸神的入侵，所谓来访王国的大和之神，正是神奈子。当然，诹访子进行了抵抗。她使用了当时最先进的铁制武器进行战斗。但是，神奈子施放了大量细小的植物藤蔓，使诹访子所拥有的大量铁轮全部锈蚀粉碎了。

效果：  
【神具「洩矢的铁轮」】：  
视为一件伤害6L的天生武器，若扔出视为一件伤害6L、体积2的投掷武器。

【“ケロ帽”】：  
头盔位视为装备名为“ケロ帽”的斗笠，不可拆卸，不可与其他头盔装备叠加使用。“ケロ帽”现在为装备者提供盔甲防御1/1。可以用来避雨，任何看见过青蛙的智慧生物看见它会第一时间将二者联系起来。

【「祟り」】：  
你掌握自然派系的D级派系造诣。  
开启诹访神技能树。  
你未购买诹访大明神下属专长神樱「湛樱花吹雪」(1)时不能进行施法检定。

【「名存实亡的神明」】：  
你获得毒素易伤3。

C级  
等级名称：八百万诹访  
价格：C+1200  
特性：【源符「诹访清水」】【开宴「二拜二拍一拜」】【“ケロ帽”+】【神具「洩矢的铁轮」+】【创造坤程度的能力】  
描述：  
【源符「诹访清水」】：  
描述：諏訪清水是流经诹访大社上社前宫，被称为「水眼」的御手洗川源头的一条清流。因为流经神域，从古代开始就作为御神水被奉为至宝。起源于千米外的山中，最终汇入御手洗川。  
【开宴「二拜二拍一拜」】：  
描述：你现在作为八百万神的一员，身体不再“那么的”有意义。当然，八百万神与任何一神论宗教里的全知全能神有着根本的区别——八百万神更接近于生物，只是生命的另一种存在形式罢了。

效果：  
【源符「诹访清水」】：  
开启一个名为【信仰】的能量池，其容量为风度+操控。每经过一场影片，这个信仰池便+1上限。受到来自各界诹访地区住民的信仰，你每天最多能恢复等同于信仰池上限三倍的信仰。在一次短休息结束后，你可以选择进行一个信仰关键属性检定，为你恢复成功数点信仰。  
你掌握自然派系的C级派系造诣。

【开宴「二拜二拍一拜」】：  
一轮一次，在你受到伤害时，你可以至多消耗3点信仰以吸收其中的1点严重伤害。  
你的身体完全由信仰支撑着，失去全部信仰或许会消散吧。  
当你的信仰池为满时，你的所有检定都会获得1DP的完美加值。与此同时，当你的信仰归零时，你被视为不再拥有任何完好，直到你的信仰池恢复任意点的信仰。这代表当你再次受到伤害时，直接从扣除韧性计算。

【“ケロ帽”+】：  
头盔位视为装备名为“ケロ帽”的斗笠，不可拆卸，不可与其他头盔装备叠加使用。“ケロ帽”现在为装备者提供盔甲防御4/4。

【神具「洩矢的铁轮」+】：  
视为一件伤害6L，格挡1的天生武器，若扔出视为一件伤害6L的投掷武器，扔出后可使用一个移动动作使其立即回到你手里。每造成1点严重伤害将为你恢复1点信仰，但不能超过每日恢复上限。你的攻击范围内新增一个其他攻击目标。

【创造坤程度的能力】：  
动作：标准  
分类：攻击技  
消耗：2点信仰  
目标：单一目标（包括自身）  
距离：10M  
持续：立即  
描述：诹访子同样是大地之神。“坤”在八卦中表示“地”。  
效果：  
改变范围内土壤的形态，在以下效果中选取一样生效：  
1、令范围内的土地往任意方向移动10米，若土地上有其他单位，其也会跟着往该方向进行移动，其可以通过一个DC7的平衡检定令自身不进行位移。  
2、令范围内的土地长出尖刺，这会令范围内的单位受到10点穿刺伤害，反射豁免。  
3、令范围内的土地在一场景内视为无支线级别的困难地形。

B级  
等级名称：永寿唯一诹访神  
价格：B+2650  
特性：【土著神「小小青蛙不输风雨」】【“ケロ帽”++】【神具「洩矢的铁轮」++】【大地之湖】  
描述：  
【土著神「小小青蛙不输风雨」】：  
描述：她的王国却曾经受到过大和众神的入侵。诹访子使用当时最先进的铁器进行了旷日持久的抵抗，最终在明悟神力的巨大差距后投降。但是，洩矢之国的人们，没有忘记原来的土著神御社宫司神的恐怖，所以坚决不接受这新的神明。名义上支配了这个王国的新的神，在大和的神话里是个只有名字保存下来的神。实际上，诹访子是一直都支配着王国的。现在外面的神明已经基本被大和的神话所替换了，只有她一个人，仍然以古代的方式悄悄地获得着信仰。  
【神具「洩矢的铁轮」++】：  
描述：幕府时代来临前，诹访地区的铁器工艺仍大大超过同时代本州岛其他地区的铁器工艺水平。  
【大地之湖】：  
描述：人们传说富士山曾与八岳比高，八岳比较高。因此愤怒的富士山把八岳削去一块变成了现在的高度。有的故事中还有个后续，是八岳的妹妹蓼科山为此事悲哀叹息，流下的泪水形成了诹访湖。  
但实际上，诹访湖其实是构造运动形成的。

效果：  
【土著神「小小青蛙不输风雨」】：  
只要你的信仰尚未归零，你被视为拥有快速治疗1（12医疗点数），当你被断肢效果作用时，你的四肢将会遵循快速治疗规则而再生（包括头部）。  
你掌握自然派系的B级派系造诣。

【“ケロ帽”++】：  
头盔位视为装备名为“ケロ帽”的斗笠，不可拆卸，不可与其他头盔装备叠加使用。“ケロ帽”现在为装备者提供盔甲防御6/6，并带有1点全伤害吸收。

【神具「洩矢的铁轮」++】：  
视为一件伤害9L，格挡1的天生武器，若扔出视为一件伤害9L的投掷武器，扔出后可使用一个移动动作使其立即回到你手里。每造成1点严重伤害将为你恢复1点信仰，但不能超过每日恢复上限。

【大地之湖】：  
创造坤程度的能力射程提升至80米，范围提升至目标点为圆心，半径为40米的圆。  
获得等同于基础移动速度的遁地速度或固体物质穿行速度，遁力3。

A级  
等级名称：土著神的顶点  
价格：A+5050  
特性：【崇符「洩矢大人」】【“ケロ帽”+++】【神具「洩矢的铁轮」+++】【蛙休「长冬长眠」】  
描述：  
【崇符「洩矢大人」】：  
描述：赤口大人呀，让人分不清她到底是保佑丰收的神还是作祟的神。  
这回又是战神了吗？是不是还有更多的身份呀？（《文花贴DS》）  
洩矢神是持有道祖神、性神、蛇神、守屋神、农耕神、风水神等神性和多种信仰形态的自然神。现在在诹访地区一般将其和洩矢神、建御名方神视为同一神明。信仰范围遍及整个东日本，有多种称呼名讳，近二百种汉字写法。  
似乎是从绳文时代就开始存在至今的远古习俗信仰，作为原始的自然信仰难以确定其起源。另外在诹访地区，也常常将诹访蛇神与「ソソウ神」、水神、山神等的神性视为相同。  
【神具「洩矢的铁轮」+++】：  
描述：相传在诹访大战中，建御名方神则使用藤蔓让铁轮生锈腐朽，从而取得了胜利。也有说法认为指古代制铁技术的变迁，洩矢神象征用自然生成的湖沼铁和低温炉的旧制铁技术，建御名方神象征用砂铁和高温炉（鑪，たたら）的新制铁技术。建御名方神所持的「藤蔓」可能象征着新制铁技术中大量使用的木炭燃料。（《诹访大明神绘词》）

效果：  
【崇符「洩矢大人」】：  
只要你的信仰尚未归零，你被视为拥有4快速治疗（48医疗点数），并提升力量检定上的3点附加成功。  
你掌握自然派系的A级派系造诣。

【“ケロ帽”+++】：  
头盔位视为装备名为“ケロ帽”的斗笠，不可拆卸，不可与其他头盔装备叠加使用。“ケロ帽”现在为装备者提供盔甲防御17/17，并带有1全伤害吸收。

【神具「洩矢的铁轮」+++】：  
视为一件伤害27L，格挡3，威猛5，破甲5的天生武器，若扔出视为一件伤害27L，高速5的投掷武器，具有回力特性。每轮一次，这一武器所造成的1点严重伤害将为你恢复1点信仰，但不能超过每日恢复上限。

【蛙休「长冬长眠」】：  
赤口归于土，御射归于石。  
一影片一次，当你死亡，你的遗体立即被土石包裹。你的意识将化为非具象的八百万神，于七日后重新降临于世界上。在此期间你除了观察与移动外无法进行更多的动作。你的遗体将获得你原来生命值点数的结构值，若被摧毁则复活失败。你每复活一次，信仰池上限永久降低1。这是一个A级、祝福来源的复活效果  
只要你的信仰尚未归零，你无需饮食、无需休息、无需呼吸，且免疫C级衰老。当你的信仰归零，你不被视为‘不再拥有任何完好’，而是视为受到自上一次饮食，休息以来的天数总和加6点数的严重伤害。
