# 永久列兵-朗基努斯

朗基努斯  
血统  
魔幻本质  
购买前置：武技3+，专业：长枪1+

你曾经是罗马的一名百人长，但也只是曾经。  
但好消息是，“神”给予你的诅咒被主神限制，你获得了解脱的机会......  
  
  
D级:尽职的百人长  
D+600  
在古罗马，战争就是士兵们的生活。  
百人长，是屡立战功，并且活下来的老兵。  
而这份强大，却成了这名百人长的悲剧之源......

久经战场的你，手持长枪的战斗方式已经刻入本能，使用长枪的攻击检定附加成功增加1点，手持长枪冲锋时有效距离取最大值计算。  
在战场上，每一刻松懈都可能是致命的。强忍着疲倦和饥饿在战场上厮杀已是常事，这视为你具有【无需休息】和【无需饮食】特性。  
  
  
C级:走向战场的士兵  
C+1200  
购买前置：D级朗基奴斯之枪  
那一天，百人长见到了“神”。  
面目可怖的他/她/它，或许只是想要向尽职的士兵表示敬意，但身经百战的士兵将其认定为威胁，本能的投出了自己的标枪。  
“神”的血沾染了枪尖，顺着枪身流到了手上。

“刺伤神，可是很重的罪行啊。”  
“但，神是很仁慈的，我不会杀你的。”  
“你是个忠于职守的好士兵，就让你，当我永远的小兵吧。”  
“你的双脚将自动引导你走向战场，然后你将为了彰显“唯一之神”的权柄而战，永远徘徊在战场上......”

作为“神”永远的士兵，你不会随着时间流逝自然衰老，但这也意味着，你将为“神”厮杀，直到永远。  
战争锤炼了你的躯体和意志，你在没有完好生命时无需消耗意志即可保持清醒，并且获得了2点基于天生防御的穿刺伤害吸收。  
你无法得知“神”的战争何时结束，在漫长的战争中，你的人性逐渐产生裂痕......  
  
  
B级：圣地的军团  
B+2400  
不老不死，但并非血族，无异能也无念刃。  
无论怎么杀死，在30天至300天之后都会复活（附300年的实验记录）。  
历史上进行过76次完美切割成两半的实验，每次都会产生两个一模一样的新个体，复数个体间协作能力很好，但只有其中之一拥有枪状物。  
———取自圣地：圣地亚哥审判局对异端嫌疑人GS-7728个体“朗基努斯”的报告

“取之不尽，不畏死亡，这不就是一支完美的军队吗？”  
———仇恨女士

经过金属领域的完美对半切割后，你被切割成了若干个个体，人类之躯的你变成了【集群生物】，组织度为你生命值的2倍。即使所有的个体都是你，都有着相同的本领与记忆，但只有其中一个个体碰触到的武器会变成朗基努斯之枪，因此持有朗基努斯之枪的个体被圣地作为队伍的象征，成为发号施令的队长。

在队长死去后，朗基努斯之枪会随机转交给活着的个体。

由于持有朗基努斯之枪的个体只有一名（即被视为队长的个体），其他个体手上的武器会保持原样，因此在成为【集群生物】后，由于被朗基努斯之枪覆盖的武器只有一把，因此视为武器原来的效果正常生效。

你们个体间相互协作，可以以一个整轮动作启用罗马龟甲阵，增加6虚拟体积，为【自身触及范围+5M的半径范围内】所有友方单位提供【全遮蔽】保护，在保护期间，伤害由组成方阵的个体承担。在此状态下，如果所有个体使用长枪进行攻击，则可以在攻击范围内新增一个其他攻击目标，也可以进行冲锋攻击,罗马龟甲阵将持续一个场景。

由于你们是多个相同却又独立的个体，在启用龟甲阵后，身为队长的个体依然可以使用【彰显神权】。

由于“神”的诅咒/祝福，在所有的个体死亡后，如果最后持有朗基努斯之枪的个体的尸体被保存了下来，会在一个月至一年不等的时间后复活，这被视为B级诅咒/祝福来源的复活，随后被引导着前往下一个战场。尸体的结构值相当于原本生命值上限。  
（注：只有B复活的原因其实是因为原典是这么写的，而且我也不想整出不死军团（））

训练（花费6xp）：  
作为军队的你们接受了圣地的军事化训练，漫长的战争生涯让你们快速上手了圣地的制式武器，你们可以使用专业：长枪（武技下属）替换两个武技的下属武器专业（购买时选择），可重复购买。  
  
  
  
A级：活着的传说  
A+4800  
“我是活着的传说，永恒的士兵，所以，我无法成为英雄。”  
“我只是不紧不慢的，永远的，凡夫俗子而已。但英雄的黄金年代，如露水般传瞬即逝。”  
“士兵该做的事，用尸体，为英雄铺平道路。”  
———朗基努斯，于红门之战  
（注：所以血统部分才没有S级）

在经历过千百年的战场后，我逐渐理解了一件事，我只是“神”永远的士兵，我所做的只是前往战场，与敌人厮杀，战死或凯旋，复活，前往下一个战场......  
对朗基努斯之枪的使用越加熟练，在使用长枪类武器进行冲锋时可攻击移动路径上触及范围内的所有地方单位，在组成龟甲阵时的冲锋动作也如此。  
躯体与精神在看不到终结的战场上久经磨练，敏感范围增加一倍，意志、强韧、反射豁免检定各获得2点内在加值，获得3点天生防御，以及基于天生防御的【超级坚韧】，免疫精神性质的一半伤害。

但是，我见到了他们，那些被称作英雄的人。  
果然，英雄只能是人类，那些充满欲望，粗俗而又有趣的人类。  
身为永远的，最普通的士兵，我能做到的，就是为他们铺平道路。  
为了保护能成为英雄的那些人，在组成龟甲阵后，获得【强拦截】特性，每轮一次，将受到的远程伤害减少至原先的一半。

在他们昙花一现的生命中，至少，我能给他们送一份礼物。

下属武器  
朗基努斯之枪  
魔幻本质  
购买前置：D级朗基努斯  
“神的矛与钉”，“命运之枪”。  
传说中此枪曾经刺入神的身体，浸染了神之血。  
得此枪者，可获得权力，主宰他人，使身边的人都臣服于脚下。

这是“神”的祝福，也是“神”的诅咒，你在未持有武器时，触碰到的第一件武器会变成一把螺旋状骑枪，这视为概念武装。  
由于此武器需要由现实基础近战武器转化，根据武器等级依次提升D3C8B11A24S35L武器伤害，在你放下武器后，武器将会从朗基努斯之枪恢复成原本模样。被朗基努斯之枪覆盖的武器将会失去原有的能力与外形，但保留原有的攻击dp。

在经过主神有意的限制后，朗基努斯之枪的力量需要逐步解放，但你无法将不属于自己的武器转换为朗基努斯之枪，至少现在不行。  
（注：这是另外的价格jpg）

D+500  
见到此枪，即见到神明，任何生物必须跪拜。  
但目前只解放了一小部分。  
你需要以一个标准动作指定距离15M内的某个生物使用【决心-专注-专业：彰显神权】 进行一次无伤害的精神攻击，命中后需要与目标进行意志对抗，成功后对方便会向着此枪跪拜，这视为【D级命运来源的思维控制】。此时你的脑后会出现混沌的黑暗光圈，宛如“神”的一部分。

这是“唯一之神”赋予你的权柄，无论对方是何种存在，都不具有不跪拜此枪的权力，除非是不受任何效果影响的“青骑士”。  
（注：这是原典设定，按我的理解“唯一之神”是被视为多个神的唯一神，在原典中，处于生效范围内可以不跪拜的活物只有发动念刃“青骑士”的六剑。）

这是诅咒的象征，你无法摆脱，无法逃避，你作为“神”的士兵带着它走过了千百年的战场。你的彰显神权检定获得1点器械加值，你的意志豁免检定获得2点内在加值。

C+1000  
随着力量的解放，你可以以一个标准动作向半径为射程1/4的圆形范围内所有生物【彰显神权】，思维控制的效果升级为C级。同时在“神”的庇护下，坠落再也不会伤害到你。  
原本包括友方在内的所有生物在神权之下必将跪拜，但由于主神的干涉，被认定为友方的生物将免于此礼。  
这是否与“唯一之神”的权柄冲突？  
又或者......

B+2000  
“神”的权柄被进一步的解放，你现在以一个标准动作可以向半径为15M内的圆形范围内所有生物【彰显神权】，思维控制的效果升级为B级，同时你的彰显神权检定获得【8加骰】。  
在神权的影响下，你使用朗基努斯之枪进行攻击时，获得【余波】，并具有【熔毁5】。

A+4000  
“神”的权柄几乎被完全解放，在“神”的光辉之下，你使用【彰显神权】时的范围半径增加480M，每轮一次，你使用【彰显神权】的检定可以重投一次，取其中最高的一次进行结算，思维控制的效果升级为A级。在“神”的祝福/诅咒下，你不再死于战场之外的事物，这视为【免疫S级即死】，同时，由于“神”的权柄握于手中，你免疫S级及以下震慑，对方攻击你时获得2点劣化，而你攻击对方时将被视为【接触攻击】。

S+8000  
“神”的权柄被完全解放，甚至变强了些。你使用【彰显神权】时的范围半径增加2400M，每轮一次，你可以让使用【彰显神权】时视为具有【传奇】，思维控制的效果升级为S级，你手持朗基努斯之枪时攻击造成韧性伤害。  
你额外获得了8ap用于使用【彰显神权】。你的内心也不再受到邪门歪道的影响，由于“神”的权柄不可亵渎，任何事物都不能窥视其力量，这视为你【免疫S级胁迫、支配、思维控制、解析】。

【额外的权柄】  
花费10/20/40/80/160xp解放

差价升级，需要在购买或升级任意等级的朗基努斯之枪时绑定购买/升级，此权柄等级不能超过朗基努斯之枪的等级。

即使身处影片内，在手上没有任何武器的前提下，你触碰到任何武器都会被转换为朗基努斯之枪，这视为祝福/诅咒来源的D/C/B/A/S级夺权，如果这把武器已经被他人所拥有，你需要以专业：彰显神权检定与对方的意志豁免+物品等级（D14/C26/B38/A48/S66）作为DC进行对抗，对抗成功后该武器便会被转化为朗基努斯之枪归你所有并可以带出影片。

在“唯一之神”的注释之下，你，或者你们在进入影片时获得了祝福/诅咒来源的D/C/B/A级背景身份，方便你或你们前往下一个战场。
