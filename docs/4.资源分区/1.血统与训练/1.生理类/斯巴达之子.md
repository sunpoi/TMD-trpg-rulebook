# 斯巴达之子

斯巴达之子  
资源类型：可升级血统  
本质：神话本质

描述：  
传承自魔界的传奇恶魔剑士-斯巴达的血脉。体内流淌着恶魔的狂血令他们注定于深渊与王座间徘徊。  
当你继承这份血脉的那一刻起，也许背叛与矛盾的因果便已纠缠而上。

D级  
价格：D+600  
描述：  
也许你尚未适应这份血脉所带来的力量，但体内奔涌的魔力却诉说着你将步向的未来。  
强化这一级别血统后，人物的头发将会被血脉所侵染变为银色。

特性：【魔血奔涌】【半人半魔】  
效果：  
解锁魔力能量池，其构成关键属性为力量/敏捷+耐力  
当你选择力量作为魔力的主属性后，你的魔力颜色将表现为红色；当你选择敏捷作为魔力的主属性后，你的魔力颜色将表现为蓝色。  
作为斯巴达血裔，你获得一个魔力的基本用法：支付1点魔力，令你伤势的自然恶化停止一轮。

【魔血奔涌】：  
你在进行生理属性检定时，可以支付1点魔力令这次检定获得2点亵渎能量加值。当你使用这一能力时，红/蓝色的魔力将沿着你的血管纹路在皮肤上流淌。  
你的生理属性每合计有3个附加成功，此能力就可额外支付1点魔力并再获得2点亵渎能量加值。  
【半人半魔】：  
你视为黑暗生物，在判断你的种族时，你始终视为人类与恶魔中较劣的一方。若人类知晓了你的种族，则他们对你的初始态度将下降一级。

C级  
价格：C+1200  
描述：  
当剑刃撕碎魔物的躯体，被庇护之人的恐惧却更甚一层。  
魔血烙印下的你是他们始终无法理解的异类。

【不灭魔躯】：  
魔界最伟大战士的鲜血赋予了你非人的战斗性能，你每次险象环生的从危机中走出，便会更强大一分。

特性：【不灭魔躯】  
效果：  
【不灭魔躯】：  
每场影片结束时，若你在这场影片中完成了以下条件，则会获得相应的好处，除第四项条件外，其他条件必须按次序完成，一影片只会获得一次奖励。  
1、赢得三次同级别战斗；你免疫B级重伤效果。  
2、赢得三次同级别战斗且开启基因锁的战斗；当你不再具有任何完好时，你会获得传奇耐力点快速医疗（医疗点12\*传奇耐力）。  
3、赢得一次高一支线等级的、被动开启基因锁的战斗；你的魔力能量池上限提升魔力关键属性点  
4、购买C级血统后获得过传奇评分；你免疫重伤的等级提升至A级，当你不再具有任何完好时，你会再获得传奇耐力\*2点快速医疗（医疗点24\*传奇耐力）。

B级  
价格：B+2400  
描述：  
作为半人半魔的存在，你的力量来源于你不得不承认的存在——恶魔。但与这些仅凭自身本能去驱使过于强大力量的存在不同，你的恶魔本质受制于你身为人的内心。只要你秉持着身为人的信念，你便绝不会因为力量而迷失自我。

特性：【魔力抗性】【斗争本能】【恶魔本质】  
效果：  
获得【魔力爆发】的购买权。  
【魔力抗性】：躯体的部分魔人化给予你对魔力的抗性，因此你免疫B级及以下的由魔力推动的负面效果。  
【斗争本能】：你免疫B级别措手不及。  
【恶魔本质】：  
你获得魔人槽，这是一个计数器，其初始值为0，上限为6。  
魔人槽会根据以下情况获得：  
1、在同级别战斗中每经过一轮，会获得1点魔人槽。  
2、受到敌方单位攻击，若这次攻击不对你造成任何伤害或是负面效果，获得1点魔人槽  
3、进行的攻击触发了毁灭/湮灭效果，获得1点魔人槽。  
魔人槽会在战斗结束后的十分钟内逐渐清空。  
当你具有魔人槽时，可以一个移动动作消耗10点魔力进入魔人化状态。你被魔化的肢体将会激发出强烈的魔力光芒，接着你的身后会浮现出由魔力构成的、纯粹而可怖的魔人虚影。  
处于魔人化状态下将不能以任何方式获得魔人槽，并在人物每回合结束时消耗1点魔人槽，若魔人槽为0则会退出魔人化状态。  
处于魔人化状态时，你不需要为【魔血奔涌】再支付任何能量即可从中获得好处，且获得的能量加值翻倍计算。  
你进行生理属性检定时加骰提升一级，至多为8加骰。  
当你处于魔人化状态时，获得等同传奇耐力点快速医疗（医疗点12\*传奇耐力）

技艺名称：魔力爆发  
价格：B+2000  
效果：  
购买此能力后，你在进入魔人化时可以将使用的动作提升为标准动作，若如此做，则你在进入魔人化状态时会同时对周围力量附加\*10米范围内的所有敌方单位进行一次近战攻击，受到伤害的单位需要进行一次反射豁免，DC为伤害值，失败则被击倒并在一轮内陷入B级、创伤来源的击飞状态。
