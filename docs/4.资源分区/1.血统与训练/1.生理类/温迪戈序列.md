# 温迪戈序列

D级血统 生食者  
告子曰：“食、色，性也。仁，内也，非外也；义，外也，非内也。”

  
当你流淌的血液中拥有远古的一丝怒吼，你将难以自控地，走向毁灭与自我毁灭。

  
与其说是你选择了茹毛饮血，不如说是无边的欲望选择了你，你将更加偏好吞吃生的食物，你可以获得“求生”技能的1DP内在加值。

  
对于熟食类的恢复道具来说，作用在你身上的效果仅仅能起到一半，通常来说的生食类道具对于你不会产生负面效果，但这仅限于果腹。

  
你将会拥有一种特殊的能量池，被称之为饱腹。  
饱腹的初始值为你的决心+操控，能量池的值会随着时间的流逝而减少（每过一个自然日，即24小时就会减去1），如果到0，你要进行一次意志检定对抗自身的耐力为DC，如果失败，你的意志值减1，难度-1，重复检定，直到成功。

  
你可以选择摄入生肉来补充，一般来说，成年人一餐的肉量结合在短休息中可以使你进行一次关键属性的检定，恢复成功数的饱腹。  
而结合在长休息中的，等同于成年人体重的肉量，可以让你降饱腹恢复至满值，每日最多因此恢复等同上限三倍的饱腹。

  
当你饱腹满时，你的力量，以及投掷检定加2DP内在加值。

  
C级 食尸鬼  
欲望如洪钟，叩击你心的门扉，如果对曾经的珍馐如今已经食之无味，弃之可惜，不妨试试这个。  
你的面色变得铁青，手变得干枯而骨节突出，你的指甲成为了尸解的刀，此时你或许会被视为神话生物。你的风度减2。  
日复一日地异食和在晚上谨慎地饱腹，让你获得了黑暗视觉。

  
你将获得一个技能：食尸  
你吃掉了面前的食物，你可以获得他的大部分信息，例如，死因，死亡的时间，周围的环境，一般由ST决定。更重要的是，一顿美餐可以让你的饱腹回满。

  
你可以食用普通的食物，但是每次食用会使饱腹值-1。  
此外，饱腹值为满值时获得的增益额外加3DP。

  
你的双手指甲提升2L武器伤害，但手部不能再装备任何武器，除非你剪除你的指甲。  
如果你剪除指甲，在本次影片内将不会再长出，而是要等1影片后，或者缴纳100点数重新长出。

  
B级 食人魔  
你不再可以接受腐臭的碎肉，你将重新行走在阳光之下，成为世界的猎手。　  
你之前习惯于昼伏夜出，躲避异样的眼光和追捕，在你重见天日的时候，你不得不忍受阳光的刺痛，遗憾的是，你已经无法接受如此强烈的紫外线了，不过还好，你能接受你溃烂的那张脸。  
如果你不戴面具或者其他遮挡方式出门，那么你的风度将会锁定为1。  
你不会再考虑那些你不喜欢的食物，你对品质的要求，逐渐增高。每次食用普通食物，饱腹值减2，食用之前剩下的“剩菜”饱腹值减1。

  
你将可以结合在攻击中标记一个你的“猎物”，造成真实视觉效果，你对猎物的攻击会附带B级重伤效果。每当你“猎杀”成功，你的意志上限加1，一影片只会成长一次，每当你吃掉你的猎物后，无论如何，你的饱腹回复至满值，每日恢复上限不变。  
此外，饱腹的DP额外提升5点。

  
A级 汉尼拔  
你开始享受了，你给自己冠上了你喜欢的名字，不过很明显，你不是他，不是吗？  
你可以剥下受害者的皮囊，放在你可憎的脸上。如果你那么做了，你将会变成受害者的模样，这是一个A级变形来源的效果。

  
你的能量池转变为“愉悦”，关键属性改为“智力加感知”。  
你尝试脱离低级的吃饱，而是转而去追求心里的愉悦，不过你发觉，你在变化。你懂的了荤素搭配，所以现在，你什么都可以吃了，除了鹿肉，你很讨厌他的味道。  
你在对指定猎物目标进行攻击时，攻击获得魂牵特性。  
每次猎杀成功后，你可以通过进行精细的烹饪，以此来回复你的愉悦，如同恢复饱腹一样结合在短休息或是长休息中。

  
你的愉悦将会影响你全属性的判定，此时只计算属性提供的能量池上限，每5点能量提供你在全检定上的1点内在加值。

  
S级 温迪戈（暂不开放）  
你终究意识到了，你已不在为人，你也想通了，这不是命运的不公，你不是天选的猎手，这只是绝望的雨露均沾，如果你沾染的多了，你就会分给其他人，而如果你在愉悦的欢喜中忘掉他，他将会吞噬你的一切。
