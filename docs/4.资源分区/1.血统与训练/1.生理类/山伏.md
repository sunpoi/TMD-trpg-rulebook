# 山伏

山伏  
描述：日本修验道行者的统称，亦称修验者、山卧、行者，指通过山岳苦修获得神验之法的实践者，修验道融合了神道教、佛教、自然崇拜等元素。虽说如此，但这一原典中的山伏是近似退魔师的存在，以神佛习合为主旨，通过严酷的灵山修行获得力量，抱守古旧的传统，以维护人间平衡，退治妖魔为职责，但对于步入现代的退魔师群体而言，恐怕也只是个古怪、过时的宗教罢了。

D级  
特性：【神佛习合】【神秘感知】【修验僧】

【神佛习合】：神佛习合是日本佛教特有的神道与佛教融合现象。这意味着你获得一个特殊的信仰导向【神佛习合】，可以兼容神道与佛教类型的信仰导向能力。

【神秘感知】：由于长期在严酷的自然环境下修行，你与灵魂已达到高度和谐，因此你开启【灵力能量池】。不同的是，你无需支付灵力也具有【灵感视觉】，同时灵力的基础用法改为：支付1点灵力，你的攻击伤害类型转为纯能量伤害，这是由于你的灵力尚不成熟，只能在一定程度上通过转换从而干涉灵魂的表现。

【修验僧】：修验僧是以古流杖术与格斗技为基础，辅以灵力与妖魔进行战斗的存在。

你在武技专业上开启【杖术】，【杖术】可以正常作为棍类武器的专业使用，也可以仅限使用修验僧的格斗类能力（若有）。此外，你在武技相关检定上还会获得等同【生理传奇属性之和】一半的修行加值。

C级  
特性：【灵性修验】【苦行之路】

【灵性修验】：修验道以严苛苦行著称，旨在突破自我限制、获得超自然力量。然而这些苦行往往因外界情况而受到很多限制，但作为轮回者的你穿梭于不同世界，足以更容易的寻找充满灵性的环境，与重复死境边缘体验的苦行，从而最大限度的发掘体内蕴藏的力量。  
从购买该等级训练开始计算，每经过一场影片，你都会获得1点【巡礼】计数，【巡礼】计数的上限为4。

【苦行之路】：根据巡礼计数，你依次解锁下列效果。

巡礼计数1：解锁【野性直觉】。  
巡礼计数2：解锁【灵性感知】。  
巡礼计数3：解锁【灵力强化】。  
训练计数4：解锁【防卫本能】。

野性直觉：你获得了免疫不超过B级措手不及效果影响的能力。

灵性感知：你获得了C级特异本质的侦测能力，自行选择基于视觉或听觉，若有其他感知方式也可基于相应感知方式，相应的感知方式将视为具有正常察觉到灵体的效果。

灵力强化：你的灵力能量池上限提升等同【沉着属性】\*1点。

防卫本能：当你的回合开始时，你可以无需使用动作进行基础规则中的格挡行为，但无法应用相应的主动类防御技招式。

B级  
特性：【灵性契约】【神威显现】

【灵性契约】：所谓神道崇尚万物有灵论，而你的苦行已得到了万物之灵性的认可，足以与它们建立临时的契约。你可以在任何环境下（包括城市中、庭院等，野外等等）进行短时间的冥想（约15分钟，可以结合在短休息中），与当地的地灵建立临时契约。根据【巡礼计数】，你获得以下效果

巡礼计数1或2： 你可以询问地灵一些简单的问题，例如知晓你所指定的一个名词指向的目标的大致方位（在该地区某个方向、某个位置），但有时候时间上会有误差。这是一个C级环境来源的【预知】效果。

巡礼计数3或4： 你可以请求地灵协助你展开一个简易的防御兼妨碍作用的【结界】，【结界】持续存在一场景，需要你支付一个移动动作并消费6点灵力展开。

结界范围以你自身为中心，等同15+智力属性\*1米展开，范围内的友方单位（包括自身）获得等同【决心属性/2】点力场防御，并且从次轮开始后，结界内范围视为处于B级环境来源的【困难地形】影响效果下，对范围内所有单位生效。你可以选择不展开困难地形的影响，仅提供防御效果。

【神威显现】： 你的灵力与武技深度融合，产生了质变。你解锁以下两项能力，但【神佛之加护】中的每项效果都需要你额外支付40XP才能使用，并因此获得自由属性，如同购买了B级技艺一样处理。

降魔之杖： 你的【杖术】攻击正式附魔。现在，当你支付灵力进行攻击时，伤害类型不会转为纯能量伤害，而是具有【神圣】伤害类型，与原本伤害取优处理，持续时间改为一场景，但灵力消耗会提升至5点。

神佛之加护： 你可以呼唤神佛的加护。以一个迅捷动作并支付6点灵力，你获得以下一项效果，持续一场景：

【金刚力】： 你生理属性相关检定获得等同【生理属性之和/3】点增强加值，并在生理属性相关检定上的加骰提升1级处理（上限8加骰）。

【韦驮天】： 你的基础移动速度提升【敏捷属性】\*5米；你免疫不超过A级困难地形来源效果带来的影响；每轮一次，你进行的第一次移动在计算距离时，视为使用杂技动作进行了移动，这会相应的影响移动距离。

【不动明王】： 你获得等同于【沉着属性/3】点全伤害吸收，基于力场防御。你免疫擒抱状态，并在绊摔/冲撞的对抗中无视体积带来的优势与劣势。特殊的是，解锁不动明王需要额外支付8XP。

A级  
特性：【即身佛】【敕令】

【即身佛】： 你通过极致的苦行，触摸到了“即身成佛”的边缘，这使你获得以下效果。  
超凡入圣：你获得【不老】；当你进入战斗后，每经过一轮，你的全检定获得1dp修行加值，至多获得全属性最低者点全检定加值，结束战斗时清空因此增加的加值。  
万魔不侵： 你免疫A级及以下的所有心灵、即死、诅咒、附身来源效果。  
缩地成寸： 你的移动不再受物理障碍限制，这意味着你获得A级命运来源的【传送移动】能力。

【敕令】：你的话语蕴含万物的灵压，能够直接命令或改变环境。你需要支付80XP才能解锁这一能力，你可以通过言语发动以下敕令，但每个敕令一日只能使用一次，不同效果有着不同的发动动作与持续时间（均视为A级命运来源的效果）。

“止”：你需要保留一个标准动作，并在回合外调用才能触发这一能力。若处于你的结界范围/近战触及距离内（取高）有任一友方单位（包括你自身）受到攻击，可以使用这一能力，你立刻对目标进行一次不附带任何招式的普通攻击，目标在被命中后需要使用意志豁免对抗伤害出目，失败则陷入持续一轮的【震慑】状态；无论目标是否被命中，你都可以指定上述范围/距离内的单一友方目标（或自身）获得一次免疫致死攻击的效果，但这一效果一影片只会生效一次。

“净”：整轮动作使用，在该场景内你在该训练中获得的心灵、诅咒、即死效果免疫提升至S级，并获得免疫不超过S级衰老来源效果的能力。此外，持续时间内你的思想会更接近于“佛”的境界，这会导致任何不超过A级的读心效果对你无效。

“禁”：在展开B级灵性契约中的结界时可以使用，你需要依次消费一个标准、移动、迅捷、花招动作，效果依旧持续一场景，消耗提升14点，但直到次日开始恢复该敕令使用次数时，结界将无法再次发动。  
如此展开结界后，结界带来的困难地形提升至A级，范围内任何不超过A级的传送来源效果失效，结界带来的防御提升至等同决心属性\*1点。

解锁敕令会正常为你提供自由属性，如同购买了一项A级技艺一样处理。

【山伏下属能力】  
基础灵力用法  
价格：D+500/C+1000/B+4000（均独立购买）  
分类：被动  
描述：灵力是对抗妖魔之流必不可少的力量，因此而诞生的是一系列基于灵力的探索与艰苦修行。不过也导致了灵力的修行始终没有被划分出界限，不同等级的方向之间完全不能统一。  
效果：  
【D级】：你的攻击获得【法破】特性，你可以在攻击时额外支付1点灵力，使本次攻击获得1DP增强加值；每经历一场影片，你因这一能力额外支付灵力获得的攻击DP加值就会增加1点。  
【C级】：你使用力量属性进行攻击时，攻击可以替换为【决心】为主属性，并在攻击相关检定上加骰提升1级处理（上限8加骰）。  
【B级】：在一次攻击中若你支付了灵力，则本次攻击获得【幻灭】特性，并附带等同伤害出目\*1点纠缠不良点数，目标需要使用强韧豁免，视为B级环境来源的效果。

灵力修行  
价格：D+500/A+4000（独立购买）  
分类：被动  
描述：通过严酷的修行，尽可能提升肉体承受能力，提升灵力使用效率，在即使在最极端的条件下也能维持修验者的状态。若是将灵力积蓄到极限，还会带来飞跃性的力量。  
效果：  
【D级】：你免疫疲乏不良点数带来的减值与严重后果；每轮一次，当消耗灵力时，你的消耗降低【传奇耐力\*1】点（最低1）。  
【A级】：当你进行消耗了灵力的近战攻击时，本次攻击可以转为范围攻击，并且攻击触及距离提升至等同敏感范围，同时攻击出目增加一半（每轮仅会增加一次），这一能力效果需要你每轮消费一个迅捷动作进行灵力解放来维持。

适应力  
价格：D+500/C+2000/B+2500/A+4000（逐级购买）  
分类：被动  
描述：由于常年身处于严酷环境，你磨砺出了一种独特的适应力，可以使你与环境深度契合，从而无视环境带来的特殊影响，并降低自身气息，避免被人发觉。  
效果：  
【D级】：你获得黑暗视觉，全环境呼吸，自适应短休息能力，免疫不超过D级环境来源效果影响，并具有无支线自然本质的反侦测能力。  
【C级】：保留D级效果，同时反侦测能力提升至C级。你可以支付6点灵力，使免疫环境来源效果等级提升至B级，持续一场景。  
【B级】：保留C级效果，同时反侦测能力提升至B级，并且无需支付灵力，也可以获得常驻存在的免疫不超过A级环境来源效果的能力。  
【A级】：保留B级效果，但反侦测能力提升至于A级。

杖术训练  
价格：D+1000/C+2000/B+4000（逐级购买，效果叠加）  
分类：被动  
描述：基础的杖术训练，等级提高之后可以习得奥传阶段的杖术。  
效果：  
【D级】：你在该训练D级中得到的加值从【生理传奇属性之和】一半提升至【生理传奇属性之和】，并且在使用棍类武器进行格挡时，额外获得【格挡5】。  
【C级】：你在武技相关检定上获得【9加骰】，在使用杖术专业的攻击时，攻击获得【高速10】。  
【B级】：在上述C级中的武技相关检定加骰改为【8加骰】；当你使用棍类武器进行攻击时，武器伤害将提升【力量属性】\*1L。

常备不懈  
价格：B+2000  
分类：被动  
描述：无论何时何地都能应对突然袭击，武道中“常备不懈”的精神的体现。在战斗中也能超越肉眼观察，察觉对手的气息。修炼的是高度的专注力和直觉。  
效果：  
\*野性直觉的措手不及免疫能力提升至A级。  
\*灵性感知的侦测能力提升至B级。  
\*能够知晓敏感范围内是否存在敌意单位，这是一个B级环境来源的预知能力。  
\*防卫本能视为使用了杂技动作进行格挡。

描述：以雷霆万钧之势，正面突破。这是一种极具压迫性的正面强攻技。在与对方对峙的瞬间，抓住其气势的间隙，以强大的决心和力量直接打入对方中线，瓦解其架势。体现了“先发制人”的主动攻击思想。  
效果：  
你进行一次普通近战攻击，本次攻击获得等同力量属性\*1点技艺加值，同时攻击出目增加一半处理，每轮你只能使用一次这一效果。

合打  
价格：B+3000  
动作：见下  
目标：见下  
距离：见下  
消耗：见下  
持续：见下  
描述：与对手的攻击合而为一，在同一时刻完成防御与反击。这是杀活一体”的极致体现。并非单纯的“格挡后反击”，而是在极短的瞬间，使对方的攻击与自己的反击几乎同时发生，用一击同时完成防御和制敌。对时机的要求极高。  
效果：你需要使用一个整轮动作进行一次全防御与格挡，随后在直到你的下回合开始前，若有其他单位对你进行了攻击，则你可以对其进行一次普通近战攻击作为反击，反击只会发生一次，直到你重新消费整轮动作再次摆出这个架势。  
这次格挡会视为使用杂技动作进行了格挡，并额外获得【沉着属性】一半的格挡防御，反击会获得等同【感知属性】一半的技艺加值。

雷光  
价格：B+3500  
动作：移动动作  
目标：自身  
消耗：6点灵力  
持续：一场景  
描述：如闪电般迅疾、耀眼，一击决出胜负的架势。速度与精准的极致。在电光火石之间，以肉眼难以捕捉的速度完成决定性的一击。其动作可能极为简洁，但爆发力、速度和准确性都达到顶峰。  
效果：架势持续时间内，你在攻击上获得等同【敏捷属性】点【威猛】，并具有【A级措手不及】效果影响。

転変  
价格：B+4000  
分类：被动  
描述：一套动作中包含多次变化。当第一次攻击被防御或落空时，能够流畅、迅速地转变为第二次、第三次完全不同的攻击，令对手防不胜防。  
效果：每轮一次，当你在一次攻击时完全没有造成伤害，则可以消费1点灵力触发【転変】；触发【転変】时，你立刻对之前攻击的那个目标再进行一次不附带任何招式的近战普通攻击，本次攻击附带【敏捷属性】一半的【高速】，并造成等同这些高速数值的【多次攻击减值】，具有【余波】特性。

打崩  
价格：A+4000  
动作：标准动作  
目标：单一目标  
距离：触及距离  
消耗：1点灵力  
持续：立即  
描述：通过连续、强力、精准的打击，重点攻击对方浑身上下的关键部位，使其气绝并完全丧失战斗力，从而从根本上“崩溃”其攻防体系。虽然看起来朴素，但基本只靠力量、技巧，不怎么依赖灵力。  
效果：  
你对触及距离内的单一目标进行两次普通近战攻击，第一次攻击附带等同【力量属性/3】点二档肢体妨碍不良点数，第二次附带【敏捷/3】点二档肢体妨碍不良点数，皆为不可豁免，视为A级创伤来源，这两次攻击还具有【幻灭】特性。
