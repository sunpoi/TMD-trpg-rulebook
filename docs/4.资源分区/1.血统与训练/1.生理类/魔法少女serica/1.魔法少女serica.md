# 魔法少女Serica

魔法少女Serica  
资源类型：可升级血统  
本质：魔幻本质  
购买前置：  
1.国籍为中国（或者类似震旦，塞里斯等国家），并且是本地人而非外国入籍。  
2.具有专长-白驹过隙。

概述：  
显然的，与你签订契约的魔法女神对于东方某个国度有着让你感到绝望级别的刻板印象。基于这样的原因，你与传统的魔法少女略有不同，其他的魔法少女则坚持认为你变身后的魔法少女名叫做“Serica”。

D级  
价格：D+600  
特性：【魔法内力】【赤绡初现】  
效果：  
开放魔法少女Serica下属技能树。  
【魔法内力】：你开启了魔法内力能量池，因为契约女神的刻板印象，魔法内力并不能用来驱动消耗魔力的能力或者技艺，而是视作内力的变体。  
特殊的是，在计算前置条件时，魔法内力可以满足对应需求的前置条件，这可能影响到对一些能力的兑换。

魔法内力的关键属性为【智力】+【耐力】。

恢复方式：  
1.经过一次完整的长休息中进行吐纳恢复所有魔法内力。  
2.经过一次短休息的冥想后恢复关键属性较高者点魔法内力。  
无论如何一日中无法恢复超过等同上限三倍的魔法内力。

【赤绡初现】：你可以随时使用一个移动动作，并消耗2点魔法内力进行变身，变身状态持续一场景。

进行变身动作，青金色光流覆盖全身，随着编钟与古琴配乐的主题曲响起，无数金色篆体符咒从虚空浮现，为你编制出一身赤红交领襦裙的服饰，袖口绣有烫金云纹，腰间垂下流苏，玉簪自动绾发成双丸子头，并在发梢上系紧一个中国结。

处于变身状态下，你获得以下好处：  
\*你视为穿戴了一件提供5/5点闪避防御的轻甲，具有基于闪避防御的【防弹】特性。  
\*若你的天生武器伤害不足3L，则提升至3L。  
\*你在生理相关检定上获得2DP增强加值。

处于变身状态下，你获得了魔法内力的基础用法：  
\*消耗1点魔法内力，使下次攻击伤害类型转为纯能量伤害。  
\*消耗1点魔法内力，本轮内基础移动速度+5米。

C级  
价格：C+1200  
特性：【朱明流火】【魔法内力-提升】  
效果：  
【朱明流火】：当你进入该训练D级变身状态时，你可以额外支付一个迅捷动作，并将变身的魔法内力消耗提升至6从而进入【朱明流火】形态，持续一场景。

处于【朱明流火】形态下，服饰的红色变得更加深沉、金色云纹扩大并变得更加繁复，襦裙外层增加了轻薄的、带有火焰渐变（朱红到金）的纱质“半臂”或“披帛”，流苏变得更长，末端系着小巧的金铃。

处于【朱明流火】形态下，你获得了以下好处：  
\*在变身中获得的轻甲闪避防御提升至10/10。  
\*在变身中获得的生理相关检定加值提升至5DP。  
\*攻击造成的伤害可转为火焰能量伤害，附带【能量武器】特性。

【魔法内力-提升】：你对“魔法内力”的基础运用更加精妙，这些新的基础用法将取代原本的用法。  
1.消耗3点魔法内力，你的攻击附带【法破】特性。  
2.消耗3点魔法内力，你的基础移动速度增加15米。

B级  
价格：B+2400  
特性：【玄璜镇岳】【赤霄剑气】  
特性：  
【玄璜镇岳】：从现在开始，进入【赤绡初现】、【朱明流火】只需要一个自由动作，在【朱明流火】的基础上，你可以再消费一个移动动作并支付10点魔法内力进入【玄璜镇岳】形态，持续一场景。此外，无论你处于哪个形态都获得了基于闪避防御的【防能量武器】特性。

处于【玄璜镇岳】形态下，服饰的色调在深朱红的基础上，融入了深邃的玄色，并金线勾勒出山岳、祥云的复杂纹饰。襦裙变为更便于战斗的“袴褶”，流苏被更简洁的玄色玉带取代，中央镶嵌一枚圆形玉佩。

处于【玄璜镇岳】形态下，你获得以下好处：  
\*在变身中获得的轻甲闪避防御提升至20/20。  
\*你获得了基于闪避防御的【坚韧】【适应】。  
\*每轮一次，你可以使用反射动作在回合外进行一次基础规则中的招架行为，针对那次攻击你会视为移出了攻击范围一样处理。

【赤霄剑气】：你学会了将魔法内力高度压缩凝聚，外放形成具有强大切割力或冲击力的赤红色能量锋刃或掌风。魔法女神坚持称这个叫“内力外放”。

因此，你获得了魔法内力新的基础用法：  
1.消耗10点魔法内力，在该场景内你可以凝聚剑气伤人，这视为你可以使用轻投掷攻击检定进行攻击，武器伤害6L，在计算弹药消耗时视为无限子弹，伤害类型为力场伤害。

2.消耗10点魔法内力，在该场景内你的触及距离提升等同自身传奇智力\*5米，并在攻击上额外获得【威猛2】。

A级  
价格：A+4800  
特性：【苍璧御天】【真龙降世破】  
效果：  
【苍璧御天】：在【玄璜镇岳】的基础上，现在你可以使用一个迅捷动作进入【苍璧御天】形态，持续一场景。不同与之前形态对于魔法内力的消耗提升，进入这一形态无需额外支付魔法内力。

处于【苍璧御天】形态下，服饰将以深邃、浩瀚的苍色为主，辅以金色和银白。朱红与玄色并未消失，作为镶边、内衬或能量脉络点缀其中。  
服饰结构更加华丽繁复，外层覆盖着由无数微小符文编织而成的透明或半透明纱衣，肩部、裙摆如同微缩的山川或祥云。玉带中央，是一枚散发着柔和青金色光芒的、完整的“苍璧”形宝玉，发型变成极其华丽的高髻，以镶嵌巨大青玉、珍珠和星钻的复杂金冠固定。

处于【苍璧御天】形态下，你获得以下好处：  
\*在变身中获得的轻甲闪避防御提升至30/30，额外获得基于闪避防御的1点额外附加成功数，并具有基于闪避防御的【折射】特性。  
\*每轮一次，你受到的近战伤害减半处理。

【真龙降世破】：你将全身魔法内力毫无保留地倾泻而出，发动一次威力极大的攻击。由于魔法女神坚持认为最强的东方象征是真龙——因此你在施展这次攻击时会附带真龙之力的投影。

每日仅限一次，你需要结合在一次攻击中消耗等同自身【智力】+【耐力】的魔法内力，此时你的全身被狂暴的赤金能量包裹，龙形虚影在身后凝聚咆哮，然后释放一次极大威力的攻击，这会对你的攻击带来如下的额外效果：  
\*这次攻击覆盖你自身前方等同敏感范围距离（扇形）内，对范围内的所有敌对目标造成伤害。  
\*这次攻击无视闪避/盔甲防御，你需要在购买时进行选择，若选择闪避防御则该能力名称不变，若选择盔甲防御则将【真龙降世破】改为【龙魂贯击】，这是一个A级创伤来源的无视防御效果。  
\*这次攻击命中后目标后，目标需要使用强韧豁免伤害出目，若豁免失败则陷入持续一场景的【惊惧】不良状态，来源为A级创伤。
