# 外向力-极速

外向力-极速  
魔幻本质  
训练  
前置：学习元气能量池并选择敏捷+感知作为关键属性

  
现在，你消耗元气获得的防御不再是无名防御，而是闪避防御

D+600  
元气的运用得到加强  
现在，你掌握运用元气强化自身的能力  
一个花招动作，消耗2点元气，于本场景内，敏捷属性检定额外增加敏捷传奇\*2点高速加值。  
一个自由动作，消耗2点元气，让自身本轮内基础移动速度增加敏捷属性两倍的移动速度。

C+1200  
特性：【极速】【战斗不息】  
现在，你已经初步掌握外向力―极速的基本用法  
【极速】：现在，人物移动将伴随着玫瑰花瓣特效，可以朝着任意方向高速移动，保持高速移动期间，每轮一次，你可以在移动中附带一次攻击，这视为你可以将攻击结合在一次移动中。

【战斗不息】：你进入战斗后，每一次攻击都将带动身体，让自己行动变得更为迅捷，镰刃愈发锋利，每轮一次，当你使用了消耗元气的标准动作技艺进行攻击后，你的移动速度会在当前场景内提升5米，至多因此提升敏捷附加\*20米移动速度。

B+2400  
熟练的运用外向力来辅助战斗，是每一个猎人必须掌握的技能  
特性：【极速】【武器精通】  
【极速】：你获得用于极速下属技艺的2AP；  
对于外向力的开发，你逐渐掌握你能力的本质，能利用外向力使自身瞬间加速，如同玫瑰般，身躯将分子层面的分解并重组，这将使你规避绝大部分物理伤害。

你获得新的元气运用方式：

消耗一个移动动作与4点元气，你于本轮下一次主动进行移动时，身躯将化为分子层面进行高速移动，甚至可以穿越部分墙体，该效果下，你将被视为【虚体】，持续一轮。

以一个自由动作消耗10点元气开启，你在该场景内进行消耗了元气的攻击时，攻击加骰提升一级（至多为8加骰）并具有额外的【高速10】，当你进行移动时，额外具有等同于自身敏捷属性\*1米的移动速度。

【武器精通】：长期的磨合，让你已经对新月玫瑰了如指掌，可以随意在战斗中切换武器形态，现在，当你想切换新月玫瑰的武器形态时，所需要的动作降低一级。

A+4800  
经过信标学院的学习，现在，你已经成为真正的精英猎人  
特性：【极速】【武器精通】  
【极速】：你获得用于极速下属技艺或者进行移动的4AP，在施展极速下属技艺时额外具有2点附加成功；  
对于外向力的开发，你逐渐掌握你能力的本质，能利用外向力使自身瞬间加速，如同玫瑰般，身躯将分子层面的分解并重组，这将使你规避绝大部分物理伤害。  
当你在使用极速能力进行主动位移时以一个自由动作进行分解转变，此时你将被视为【虚体】，若本回合以此种方式过，你在该回合内施展极速下属技艺进行攻击时额外具有【高速10】与1点附加成功，转变为【虚体】状态将会持续至自身下个回合开始，并且在该状态下，在承受攻击时只要自身的闪避防御没有被完全破除，就会使得那次攻击承受【劣化3】。  
对于元气的理解与外向力的开发，你免疫A级定身效果。

【武器精通】：长期的磨合，让你已经对新月玫瑰了如指掌，可以随意在战斗中切换武器形态，现在，当你想切换新月玫瑰的武器形态时，所需要的动作降低一级。  
当你持有新月玫瑰时，你可以从容不迫的应对每一次袭击，你免疫S级措手不及。
