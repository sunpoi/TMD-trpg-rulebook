# 秘铁狂卫

名称：血兽种——秘铁狂卫

D级  
描述：  
你出生在荣耀石堡里面，按照人类的说法，你是族群里面的贵族，但这没有让你高人一等，反而需要经受更多，更残酷的训练。

【疼痛转化】：  
在荣耀石堡的地底训练场，没有木剑和草靶——只有生铁、岩石、以及比你更早学会忍受的同类。你的第一课不是学会格挡，而是学会让疼痛成为身体的第二层皮肤。

【秘铁仪式-初】：  
秘铁仪式是什么？没人知道，老祭祀会让你涂上秘粉，在熔炉外跪坐一晚，随后去证明你自己的勇气，

特性：【疼痛转化】  
效果：  
【疼痛转化】：  
你可以忍耐剧痛点数，至多忍耐4点，随后在战斗结束后恢复，若超过忍耐上限，如常承受剧痛带来的负面影响，除此之外，你拥有1点仅针对冲击点数生效的异常点治疗（6医疗点数）。

【秘铁仪式-初】：  
当你斩杀【三】名敌人的头颅归来之时，你感到一股燥热和力量从腹内升起，你的天生防御额外提升5点，此时你就已经是秘铁狂卫的预备役了。

C级  
描述：  
你经历了秘铁仪式后，你就可以开始进行石堡保卫的任务了，站在石堡的寒风之中，你学会了发力技巧。在巡逻之中，学会了如何保护自己。

特性：【蛮力】【本能自保】  
效果：  
【蛮力】：  
现在当你进行以力量属性为主的攻击时，攻击检定额外具有【9加骰】。

【本能自保】：  
你的伤口哪怕溃烂也不会恶化，这可以令你的伤势自动稳定。进入战斗后，只要你承受过敌人的攻击，就会在三豁免检定上额外获得3dp内在加值，这些加值每场战斗仅会获得一次，持续至战斗结束后失去，直到你亲手击杀【三】名敌人后，这些三豁免加值会常态生效。

B级  
描述：  
石堡告危！天火熔炉全力运转，你感到了不对劲，前方的战报一封接一封传回，狂卫们越来越少，你也被推上了天火熔炉感受仓促的仪式。

【死战不退】：  
你战斗中无法主动撤退，你只知道往前冲！

【背水一战】：  
你的瞳孔在因为紧张而缩成了缝，周围的呼啸声和喊杀声，以及身上的伤口，无时无刻不在提醒你，不拼就死！

【秘铁仪式-始】：  
你开始感受什么叫秘铁，铁片在你手中如鱼得水，它们保护着你，使你能对抗锋利的箭矢和魔法也不会胆怯，

特性：【死战不退】【背水一战】【秘铁仪式-始】  
效果：  
【死战不退】：  
哪怕你的完好生命值被严重填满时也不会因为伤势而昏迷，但倘若你背对敌人的方向远离，每移动3米就会令自身立即承受1点不可豁免的冲击点数。

【背水一战】：  
现在，如果你失去了所有完好生命，你在进行力量属性相关的检定时加骰提升一级（最高8加骰）；此时你拥有的所有【快速医疗】将失效，每以此令1点快速医疗失效，你就能在以力量属性为主的攻击上获得3dp增强加值和1点附加成功，至多以此获益4次（即12dp增强加值与4点附加成功），并且在攻击时额外附带【熔毁5】。  
一旦你拥有了完好生命值那么你就会立即失去该特性所有的益处与劣处。

【秘铁仪式-始】：  
必须完成D级特性【秘铁仪式-初】后才能拥有该特性。  
你如果穿着护甲，你的天生防御在原有基础上再额外提升5点。

A级  
描述：  
石堡前的丘陵上都是狂卫和希尔瓦帝国的铁骑，很幸运，你不是其中之一，你从死人堆里爬了出来，再次站在石堡上面对着铁骑……这是不幸还是万幸呢？

【秘铁仪式-终】：  
现在你真正经历了秘铁仪式，那是血与火之中淬炼出来的钢铁，你终于知道了，为什么部落数年来进行最终秘铁仪式的人寥寥无几。

特性：【秘铁仪式-终】  
效果：  
【秘铁仪式-终】：  
从此刻开始，你可以随意融化对方护甲，将其化为自身所用，你以力量属性进行的攻击能够完全忽略对方盔甲防御，这视为A级诅咒来源的无视防御效果；同时对方的护甲会贴合于你自身，你只要成功以此忽略对方防御，那么对方每拥有1盔甲防御，就能给你自身增加1点天生防御，你一轮中最多可以以此法额外提升24点天生防御，这些防御只会持续一轮。  
并且此时友军攻击被你吸收护甲的目标时，对方也因为薄如蝉翼的护甲而无法防御。对于你成功攻击造成伤害，并且完全忽略对方护甲的情况下，你可以从挥砍/钝击/穿刺中选择一种，被攻击者获得对应伤害类型的5点易伤，该效果造成的易伤类型每轮开始时可以切换一次。

S级  
描述：  
最后的秘铁狂卫死于大陆著名的诺尔德丘陵战役，那场绞肉机战役搅灭了一切秘铁狂卫的信息，包括传承仪式…
