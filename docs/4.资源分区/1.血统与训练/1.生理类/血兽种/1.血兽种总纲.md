# 血兽种总纲

血兽种  
本质：魔幻本质  
传说在古老的诺雅德拉大陆上有两条古龙，一为【诺尔亚西】，二为【古卡拉】，他们的日夜争斗撕裂了封闭的大陆，诺尔亚西化为了双日，古卡拉化为了青琉血月，他们的死亡给大陆带来了无限的生机，其中从诺尔亚西的耳朵里流出一滴鲜血，那滴鲜血创造了最开始的狂兽种……  
这也许是真的，也可能是传说，但狂兽种的确出现了，甚至称霸了一个纪元，但终究还是退出了历史的舞台，现在他们被其他种族成为：兽人或者野兽。

“我们是荒野之子”  
“我们在战场上嘬血而生”  
“仰望双日吧，那是我们先祖在注视着我们”

低阶D+600  
你从部落或者石堡，又或者荒野上出生，你那敏感的鼻子闻着出生周围的一切事物，毛茸茸的兽耳和尾巴为你提供了更为灵敏的听觉和平衡，现在你就是一名部落/石堡/荒野的新生儿了。

【古老血统】现在你拥有了一对兽耳，尾巴，尖锐的犬齿等等……曾经的耳朵已经退化消失或者保留原貌但不会产生任何效果。这些都是血统显现的特征，由你自由选择一个动物的特征具象化在自己身上，同样的你会继承一些该动物特征，比如：猫好奇心旺盛，狼喜欢成群结队活动/狼嚎等……  
同样的，这些仅仅只是画风上的改变，不会为你带来其他效果。  
【职责供卫Ⅰ】所有的狂兽种都会因为自身的特性而得到完全符合的工作，这也是在人族纪元能依旧位居第二霸主的原因。  
购买血统时，从双日祭司、秘铁狂卫、荒野之子中选择一项，选择后不可更改，获得对应等级职责的特性。

中阶C+1200  
你明白了自己的处境，你也开始为自己而谋生考虑，也有少部分开始离开部落走向大陆其他地方……

【职责供卫Ⅱ】获得对应等级职责的特性。

高阶B+2400  
你开始踏入了上层社会，你的力量就是你的敲门砖，现在一个部族或者石堡会完全为你打开大门或者由你领导，一些曾经的敌人也会对你有所敬重。

【兽血沸腾】古老的兽血开始涌动，先祖们的呼唤在你血脉中响起，现在你学会了传说中的力量,你开启基因锁时额外获得3点加值。

【职责供卫Ⅱ】获得对应等级职责的特性。

王阶A+4800  
你已经是现存的王之一，王不会死，王的种族的引领者，王是稀少的，你开始背负王的使命。

【王命于天】你现在进入到一个影片里，可以向ST申请一个支线任务，该任务大部分是让你带领该世界和你一样的兽人走向繁荣，或者拯救该世界兽人，一般由st裁定，当然必定是对该世界兽人有益的，并且是极具挑战性的。  
同样的你不会轻易死亡，你免疫S级命运来源的即死效果。  
一旦你完成任务，那么你会获得5xp的额外奖励。

【职责供卫Ⅲ】获得对应等级职责的特性。

龙阶S+8000  
你现在和一头古龙战力差不多了，现在你是首屈一指的强者，在部族里面，兽兽见你都得恭称你为一声老祖，兽族之道上的先祖们也欢迎着踏上无尽征途的你……

【职责供卫Ⅳ】获得对应等级职责的特性。
