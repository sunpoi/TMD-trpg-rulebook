# 怪物猎人

怪物猎人  
描述：后世之人，回首这荒茫而耀眼的数百年，曾发出过如此评论。  
那是大地，天空，最重要的——住在那里的人们，最充满生机与活力的时代。  
世界，比现代要单纯的多。  
换句话说。  
是狩猎，还是被狩猎。

附录1：该兑换源自《狩猎日记》与其他小说相关内容，即使同样处于怪物猎人世界观下，其内容与游戏有所出入，因此不具有游戏版本中的大部分能力。

附录2：购买B级或B级以上的怪物猎人训练时，你必须拥有见证专长-巨人杀手。

下位猎人  
价格：D+600  
描述：俗称是新手猎人，无论是怎样强大的猎人都是从最初的训练所开始。虽然也有个别从一开始就很强的类型，但那并不能算作常理来看待。

【战斗训练】：习惯让刀刃陷入生物体内，并在个人素质的基础上进行战斗训练。刀具、钝器、弓弩，偶尔也会用到猎枪，猎人基本上都是多面手。

你的全武器专业等级上升1级，若当前使用的武器专业已抵达上限，则获得相应专业相关检定上的1DP修行加值。但在处理其他计入武器专业等级的效果时，这一能力提供的专业等级不会计入其中。这一能力需要支付4XP才能对异种武器正常生效，并且使你在上述武器专业抵达上限后，专业相关检定的修行加值提升至2DP。

当你使用全力攻击时，获得的DP增加1点，失去的防御减少1点，并且无论你使用何种武器都可以进行全力攻击。

【极限生存】：辨别环境，穿越荒野，搜集可食用的植物，并确实的将杀掉的动物烹调，只有习惯这些才能成为猎人。

你在生存技能上开启专业：野外生存。野外生存的检定组成为\[感知+生存：野外生存\]。

野外生存可以用于在野外打猎、带领队伍安全穿越冰原、辨识附近是否住有枭头熊、预测天气、或避开流沙等自然危险等。  
进行野外生存的动作是可变的，不同的野外环境进行这些行为的难度也不同，一些典型用法与DC如下所述：  
\-找到东南西北的方向：需要一个整轮，DC1；如果无法看到天象等常见参照物，DC再+1。  
\-在野外行进中避开危险，如湿地中的沼泽，沙漠中的流沙等：不需要动作，DC3。  
\-预测24小时内的天气：需要消耗1分钟，DC3。  
\-一边打猎和搜集可以吃的东西一边以移动速度的一半速度进行旅行：需要一整天，你每获得1个成功数，就可以提供一个人的食物。  
上述DC都是以现实世界中，有正常的气候和物种丰富度的野外环境（例如，还未开发的神农架林区）为基准的。如果是更加恶劣的环境，则由ST酌情增加DC。

熟练猎人  
价格：C+1200  
描述：脱离新手范畴，在数次或十数次狩猎中存活下来，技艺趋于熟练的猎人。他们能独立应对小型怪物群落或单独的大型怪物，开始形成自己的战斗风格，是猎人公会的中坚力量。

【狩猎本能】：在无数次生死边缘的磨砺中，你对危险的气息、怪物的动向形成了近乎本能的感知。

你总是知晓敏感范围内是否存在敌意单位，这是一个C级环境来源的预知能力。

【武器专精】：经历了实战检验，你对某一种武器的理解已经远超基础，能将武器的特性发挥到更高层次。

在购买该等级训练，并经历一场影片后，你选择一项可以使用【武器】进行攻击的非施法专业，你在进行该专业上相关检定时，加骰提升1级处理（最高8加骰）。

上位猎人  
价格：B+2400  
描述：一言蔽之，很强。能够狩猎超越人类体能极限的巨大怪物，能在无数战斗中生存下来，只有猎人中的猎人才会被作为“上位猎人”而认可。

【巅峰技艺】：并非具有特殊或者说是狡猾的能力，仅仅是基础能力和熟练度却高得惊人而抵达压倒性强大的领域，上位猎人就是这样的存在。

你获得等同感知属性一半的攻击检定加值/洞察防御/三豁免加值，加值类型为修行加值。

【传奇狩猎】：当天灾降临的时候，人们总是会为了生存而竭尽所能地战斗，猎人从来也不会缺少对抗传奇的勇气。这不代表你不会恐惧，而是勇气会鼓舞着你继续战斗。

当你承受恐惧/沮丧不良点数与相应的严重状态时，你可以使用一个自由动作并支付1点意志力，使自身免疫恐惧/沮丧不良点数带来的减值与严重状态，持续一场景。

G位猎人  
价格：A+6800  
描述：上位猎人的上位。超越人类极限的传说级猎人，以血肉之躯与山峦般的古龙正面搏杀。换句话说，G位猎人能凭借一己之力对抗，驱逐，乃至于讨伐改写生态系统的古龙级怪物。

【天灾之躯】：无数次沐浴古龙之血，承受天灾洗礼，五感在宛如风暴的战斗中淬炼至极致……在某种意义上，G位猎人与古龙的区别并不算太大。

你在巅峰技艺中获得的攻击检定加值/防御/三豁免加值提升至【感知+耐力】/2，但获得的防御类型调整为一半是天生防御，一半是洞察防御

【传奇狩猎-提升】：与灾厄战斗在第一线，是对"不可能"的否定意志，也是以人类意志对抗自然灾祸的体现。

从现在开始，你在传奇狩猎中获得是效果不仅是免疫恐惧/沮丧不良点数带来的减值与严重状态，即使没有承受相应异常不良点数，也可以正常消耗意志力激活，并使你获得免疫精神伤害的能力。

选开：生态震慑  
前置：G位猎人  
效果：在怪物猎人的世界观下，你被视为【移动型生态灾难】级单位。这会导致以你为中心，半径等同你的最高内在支线等级\*10公里内，所有非古龙级怪物，皆需通过 DC=你的【耐力+力量】/2的意志豁免检定，否则陷入恐慌（无条件远离你，无法对你进行攻击），此效果为A级心灵来源。你并不能关闭这个效果，除非你具有至少等同于自身内在最高支线等级-1级的反侦测能力，才能抑制这一效果的作用。

【怪物猎人下属能力】  
回避性能  
价格：D+500/C+1500（逐级购买，效果替换）  
限制：每轮一次  
分类：主动  
动作：反射动作  
目标：见下  
消耗：1点意志力  
持续：立即  
描述：翻滚，不停地翻滚。看准距离，面对大型怪物保持距离是猎人的生命线。  
效果：  
【D级】：当你在回合外受到攻击时，可以支付1点意志力打断性的进行一次移动，本次移动距离固定为5米。

【C级】：当你在回合外受到攻击时，可以支付1点意志力打断性的进行一次移动，本次移动距离等同基础速度。

全力回避  
价格：C+1500  
前置：回避性能C级  
分类：被动  
描述：在面对超越人智的威压时，人类的动作会不可避免的陷入僵直，这无关勇气，而是生物本能的反应。既然如此，那就尽可能的让身体动起来，拼命的移动、回避，已经成型的动作即使僵直也会因为惯性而被强行拉扯，即使会导致动作变形，也不会站在原地单方面的承受攻击。  
效果：  
你在反射豁免上获得【8加骰】；当你承受【定身】类效果时，可以立刻使用反射动作触发一次回避性能的效果（依旧受到回避性能次数限制），但这次回避性能的效果不会使你进行移动，而是使你获得持续一轮的免疫不超过B级定身来源的效果能力。你可以在回合内消费迅捷动作来维持这一能力，从而维持上述免疫能力，直到你不再维持为止，这会占用你的每轮使用回避性能的次数。

部位破坏  
价格：C+1500  
分类：被动  
描述：断尾、破头、破翅、破背，猎人们最擅长破坏部位，以削弱怪物的力量。  
效果：  
当你攻击特定部位时，可以忽略等同传奇力量/传奇感知取低的部位减值，并且可以额外造成等同这些忽略部位减值数值的韧性伤害（无论对方是否免疫部位攻击，都能造成这些韧性伤害）。

研发（B+4000）：你在攻击时可以选择目标任意一个肢体部位（不包括核心肢体），命中后额外造成等同力量属性三分之一的不可豁免的肢体妨碍不良点数。

盾技  
价格：C+1500/B+2000（差价升级，效果替换）  
分类：被动  
描述：从现实的角度而言，盾牌的难用程度是出乎意料的。如果只是单手拿着不会妨碍行动的小盾，还能正面承受怪物的攻击，别说难易度了，太可怕了。即使如此，能熟练的将其运用，作为辅助进行牵制的猎人虽然稀少，但依旧存在。  
效果：  
当你使用小型或中型盾牌（不具有双手特性，且武器体积小于你的体积2点或以上）进行格挡时：  
【C级】获得基于格挡防御的DR/-，DR/-的数值等同【耐力属性】的三分之一。

【B级】获得获得基于格挡防御的硬度，硬度的数值等同【耐力属性】的三分之一（硬度代表其拥有与其硬度相等盔甲防御以及DR/-）。

狩猎体质  
前置：耐久力C级或以上  
价格：C+2000/B+2000（差价升级，效果替换）  
分类：被动  
描述：即使受到怪物的猛毒攻击，虽然会受到一定伤害，但还是能若无其事地行动，只要体力没有耗尽，毒素状态就会自然痊愈。既然有这么强的免疫力或抵抗力，大部分的毒都杀不死猎人。不过，这可是新手猎人的普通规格哦……这就是猎人比怪物更像怪物的原因。  
效果：  
根据技能等级，获得免疫不超过B/A级毒素与药物来源的效果能力（对应C，B级技能），并在承受异常不良点数时，减少【传奇耐力-1】点/【耐力属性/4】点（对应C，B级技能）。

传奇狩猎  
价格：B+2000/A+4000（差价升级，效果替换）  
前置：狩猎体质C级或以上，怪物猎人训练B级或以上  
分类：被动  
描述：挑战并战胜被称为“天灾”的古龙种或同等级别超大型怪物的实力与勇气，同样也是荣誉的证明。  
效果：你免疫不超过B/A级环境来源的效果影响（对应B，A级技能），并根据技能等级获得额外效果。  
【B级】：你进行的攻击具有【接触攻击】【毁灭】特性。  
【A级】：你进行的攻击无视天生防御（视为A级创伤来源的效果），具有【湮灭】特性。  
特殊的是，该技能在A级时更名为【古龙之敌】。

ST选开：在怪物猎人的世界观下，该能力会为兑换者提供对应技能等级（B/A）级命运来源的背景身份的能力，根据声望数值给予人际关系，经历，与相应的记忆。

深呼吸  
价格：D+1500  
分类：被动  
描述：猎人的顽强并不只是因为肉体。他们拥有秘传的肉体操作术，能够将吞下的药物在体内立刻分解吸收。简单来说，就是「为了凝聚力量的呼吸」的高级版本……因为轮回者也能做到类似的事情，所以这一效果变成了增强服用药物的效果。  
效果：你在战斗场景中使用怪物猎人相关药物时，若你使用特定类型的药物，将会获得相应的额外效果，每个效果只会在一场景内触发一次：  
1.恢复类（包括恢复能量、血量等）；你额外获得1点快速医疗（医疗点12），这些快疗点会持续至药效结束或一场景（取低）。  
2.防御类（包括提供各种类型的防御或吸收等）：你可以降低等同自身【传奇耐力】\*2点攻击检定DP，在防御上获得等量的天生防御，这些效果会持续至药效结束或一场景（取低）。  
3.攻击类（包括提供各种增强攻击等）：你可以将攻击类药物提供的攻击检定上的加值（若有）中的每3DP转化为1附加成功，最多以此法转化传奇耐力\*1个附加成功，这些效果持续会至药效结束或一场景（取低）。  
4.其他类药剂：你的基础移动速度增加【传奇耐力】\*3米，这些效果持续会至药效结束或一场景（取低）。

冥想  
价格：D+1500  
分类：被动  
描述：只是在基地营休息就能大幅恢复体力，从神经到大脑到内脏，所有东西都彻底进入休眠状态（只要走错一步就是假死状态），陷入连梦都不会作的深度睡眠能力。简单来说，就是「睡前冥想就能得到优质的睡眠」的高级版本。  
效果：你获得【自适应长休】；你进行长休息所需时间减半，并在短休息或长休息时获得【快速医疗1】，对应医疗点数12。

耐久力  
价格：C+2500/B+4500/A+8500  
分类：被动  
描述：经过锻炼的肉体和肉体操作术结合在一起，猎人才能发挥猎人的规格，如果没有这个，猎人也只是比普通人强壮三倍左右。  
效果：  
【C级】：你免疫不超过C级创伤来源的效果影响，你的生命值上限提升等同耐力属性四分之一的数值，你不会因生命值被严重伤害填满而昏迷。

【B级】：你免疫不超过B级创伤来源的效果影响，你的生命值上限提升等同耐力属性四分之一的数值，你不会因生命值被严重伤害填满而昏迷。在攻击你时，目标需要在攻击检定上承受【劣化2】（基于你的天生防御）。

【A级】：你免疫不超过A级创伤来源的效果影响，你的生命值上限提升等同耐力属性三分之一的数值，你不会因生命值被严重伤害填满而昏迷。在攻击你时，目标需要在攻击检定上承受【劣化3】（基于你的天生防御）。

受身  
价格：D+500  
分类：被动  
描述：无论从多高的地方以什么姿势落下，都能安然着陆，就算脸朝下掉下去，只要有缓冲物就几乎不会受伤……如果是从猎人的角度来考虑的话。  
效果：你免疫坠落伤害；当你非主动陷入伏地状态时，所需起身动作下降1级（自由-花招-迅捷-移动-标准-整轮），若导致起身动作等级不超过迅捷动作时，你还可在回合外以反射动作起身。

名门特色  
价格：C+1500  
分类：被动  
描述：虽然各地训练出的新人猎人水平基本一致，但也有着名门的情况。并不是指名门就会更强，相反、非名门的训练区域反而会有独特的几个专属训练项目。而名门之所以是名门，是因为名门的训练项目涵盖了所有猎人所必备的项目学习内容。  
效果：你在学识上开启【怪物生态】专业，并且你在进行生存技能相关检定时，可以将该专业作为加值计入其中。  
当你与任何符合【怪物】相关词缀的单位战斗时（无论是外观，还是内在符合条件皆可，由ST判断），你可以将该专业作为加值计入攻击检定。  
若需要判断加值类型，该技能提供的加值始终视为专业加值。

选开：当处于怪物猎人世界观下，你可以指导其他被视为拥有【怪物猎人训练】的非轮回者单位，使其在专业上开启【怪物生态】专业，其依旧需要自行将XP投入来提升【怪物生态】专业。

鬼人化  
价格：B+3500  
动作：标准动作  
目标：自身  
消耗：1点意志力  
持续：决心/耐力\*1轮（取低）  
描述：鬼人化是类似于肾上腺素过载的状态，新陈代谢与生理循环系统被强制提升到极限，心脏以数倍频率跳动，肌肉纤维超负荷运作，从而爆发出惊人的速度和力量。同时还能捕捉到最细微的气流和震动，从做出那些匪夷所思的动作。  
不过，考虑到处于鬼人化状态下猎人会容易沉溺于武器的触感，难以辨认周遭的情况，从而产生明显的僵直，鬼人化这一手段只能算作有利有弊。  
效果：持续时间内，你的近战攻击额外造成等同力量属性三分之一的同类型伤害，并附带力量属性三分之一点【高速】；当你在持续时间内进行移动时，基础移动速度增加等同【敏捷属性】\*10米。  
只要你处于该效果的持续时间内，就可以同时对近战触及范围内的所有目标（不包括自身）进行攻击。此外，你会在战斗结束时承受等同本次战斗中【进入鬼人化状态轮次】-【耐力属性】\*1点疲乏不良点数，不可豁免。
