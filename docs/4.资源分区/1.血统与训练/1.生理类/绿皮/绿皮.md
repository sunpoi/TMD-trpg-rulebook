# 绿皮

绿皮  
资源类型：可升级血统  
本质：特异本质  
原典：战锤40K  
购买前提：智力不高于1

概述：  
春天来了，可塔克星区的某颗绿色的星球上生机盎然，小小的屁精走过几颗粗壮的大蘑菇，用刚扣完鼻屎的手挠了挠脑袋上的疙瘩，快乐地用史古格水壶给软趴趴的地面浇水。

忽然一只绿手破土而出，一把掐住屁精的大鼻子甩在地上，屁精倒头就哭，它的哭声像是闹钟，叫醒了地下的什么东西；摔在地面上的史古格水壶赶紧爬起来尖叫着逃离了现场。

被屁精一把鼻涕一把泪的哭声吵醒的新生绿皮们用各种各样的方式顶破了地面，在一阵轰隆隆的动静中像是雨后的蘑菇般钻了出来，它们一脸懵逼地环顾四周，然后一致觉得那边抱着鼻子打滚的屁精太他妈吵了。

就这样，一只屁精引发的斗殴在孵化场发生了，现场一片混乱，但是……

绿皮们寻思，打架真快乐。

…………

D级名称：新生绿皮  
价格：D+600  
描述：你的力量还很微弱，距离搞毛二哥还很遥远，但无所谓，你寻思自己该Waaagh一场。  
【绿皮之血】：  
绿皮的本质是一种灵能植物，皮肤是健康的绿色，身高比人类平均身高略高，地包天、罗圈腿，严重的脖前倾，这些因素都导致了新生绿皮的身高看起来没有那么高大，但是及膝的手臂和紧密的肌肉让绿皮们看起来依然充满威胁性。  
【绿皮成长】：  
你的血统带来了源于血脉的战斗知识和无穷无尽的生长空间，你只需要晒晒太阳打打架就可以变得更强。

特性：【绿皮之血】【绿皮成长】  
效果：  
获得【体型成长】的购买权。  
获得绿皮技能树的购买权。  
【绿皮之血】：  
你的体积提升1点并获得1点快速医疗（医疗点12）；你的生物类型转变为【植物】，这意味着你将承受2点火焰易伤。

【绿皮成长】：  
每当一场影片结束时，若你参加过至少一场同级别战斗，你三豁免上获得1点俺寻思加值（士气）以及1dp力量检定上的俺寻思加值（士气）成长。

选开1：主持人可以选择开启这一条，由于你是个灵能孢子体，所以你会不断地向外产生绿皮孢子，这是一种微小的，定期散播到空气中的绿皮特有物，孢子会潜入周围的环境，并在一段时间后蔓延生长出绿色的菌丝，这一行为会导致孢子寄生的地方生长出绿皮生态环境，如果依然没有受到清扫，那么绿皮生态将会开始产生……美味的蘑菇。  
选开2：主持人可以选择开启这一条，如果你在战斗中让队友们全都觉得你很牛逼，你在完成这一场影片之后的体积再增长1点，但这些增长的体积没有任何实际效果，但会让你看起来更高，看起来更Waaagh！

C级  
等级名称：绿皮小子  
价格：C+1200  
描述：经历了一次又一次的Waaagh，你现在已经成长的足以独当一面！你感觉自己强大了太多太多，更好的是，你寻思自己还能变得更强！在那之前，你寻思得给现在的自己选个氏族倾向。  
【绿皮氏族】：  
绿皮氏族主要分为恶月、血斧、死颅、高夫、邪日和蛇咬……说真的，绿皮们自己都寻思不出来：为啥氏族这玩楞能稳定地跨星系组织起来又不松散滴？绿皮的……跨星系组织？这玩楞咋存在滴捏？但这玩意就是存在，而且六大氏族的标志统一，习性相同，当真是搞毛二哥显灵。

【恶月（Bad Moons）氏族】：  
恶月氏族的代表色是黄色，通常都会在盔甲和载具上喷画着一脸狞笑的弯月。恶月的小子是最富有和最狡猾的，富有带来狡猾，狡猾带来富有。而有钱则代表，他们可以更加嚣张地倾泻自己的火力。  
【龅牙级火力资金】：只要有钱，啥玩意买不起？用不上？我管你用不用得上，买！买！买就完事儿了！还有你，看什么看，你的牙也拿来吧你！

【血斧（Blood Axes）氏族】：  
血斧氏族的代表标志是染血的交叉双板斧，常见于武器、载具和旗帜上，血斧的小子们相比起其他氏族更加特立独行和“人性化”，他们会钻研战术、伪装行动甚至主动撤退，深受其他兽人氏族鄙夷。  
【绿皮战术】：俺寻思…万一俺还没到敌人脸上突突，就被敌人突突死了呢？这俺不就亏大了？所以俺想到了个好办法，那些小虾米们不是经常藏起来给俺们一阵突突吗？这不就对了！

【死颅（Deathskulls）氏族】：  
死颅氏族的代表色是蓝色，标志是长有向下弯曲的双角的蓝色搞毛头像。死颅的小子们最大的能力就是捡垃圾、偷东西和俺寻思能拼个啥出来，对于他们来说，干仗只是抢劫的手段，抢劫是为了能更好地干仗抢更多东西。  
【战场劫掠】：死颅的小子们总能从战场上扣点什么东西下来，有东西就顺，东西被钉地上了就连着钉子一起顺，东西实在搬不走就砸了顺点看起来还能用的零件，所谓雁过拔毛。

【高夫（Goffs）氏族】：  
高夫氏族的代表色是黑色，使用黑白撞色马赛克纹路作为填充，标志是长着向上弯曲牛角的黑色搞毛头，也喜欢用红色作为装饰色。高夫的小子们在所有绿皮当中都算个大莽子，莽的让绿皮汗颜、莽的让虾米心颤、莽的让生命不值钱、莽的让恐虐直呼内行。  
【战争成长】：别管那么多，冲上去给丫一大逼兜甩死！这就是俺们高夫！

【邪日（Evil Sunz）氏族】：  
邪日氏族的代表色是红色，标志是太阳的形状和橙色火焰、闪电的涂装。邪日的小子们钟爱的是极速！飙车！噪音！他们就连开飞机都没有起落架！！生命在于运动，而邪日的小子们用生命诠释什么叫TMD极限运动。  
【红色三倍速】：给载具涂上红色，焊上邪日的标志，喷上橙色的速度火焰，然后给TMD发动机来上一大飞脚，这速度与激情！WAAAGH!!!!

【蛇咬（Snake Bites）氏族】：  
蛇咬氏族的代表色是棕色，标志是岩石和弯曲的蛇，喜欢用灰白色的石料作成的原始涂料来装饰自己。蛇咬的小子们鄙夷科技，崇尚原始，那些原始世界里野蛮生长的绿皮部落往往都会因为难以教化打包丢给蛇咬氏族，而蛇咬氏族乐于接受新的野蛮同僚。蛇咬的小子们遵从最古老的绿皮生活方式，导致他们在绿皮中也异常耐造，而且只有蛇咬氏族才有培育绿皮生物的独特技术。  
【蛇咬躯体】：蛇咬氏族的绿皮们崇尚原始生活，这种在绿皮中也算作是野蛮的生活方式需要一副强大的野蛮躯体才能掌控，而蛇咬氏族的绿皮们在成年时往往会用各种有毒生物咬自己，拜此所赐，所有蛇咬氏族的小子们都耐造的很（不耐造的全毒死了）。

——上列条件可以多次满足的，每当达成一条条件时，你就会在该场战斗的回合开始时额外获得用于转化为标准动作攻击或进行绿皮血统下属技艺行为的1AP，该特性至多以此提供8AP。

现在，你进行的近战攻击若不具备范围攻击能力，则提升为对你的触及范围内的所有敌人进行攻击，若具备则不产生改变。  
现在，你进行的投掷攻击若投掷物不具备范围伤害能力，则可以提升为使得投掷物对落点为中心半径15M范围内不分敌我的所有目标造成伤害，若具备则不产生改变。  
现在，你进行的远程射击若不具备范围攻击能力，则提升为基于武器射程对前方120°扇形范围的所有敌人进行扫射，若具备则不产生改变，特殊的，该能力硬性要求此次射击必须具有【连射】特性。
