# OO水晶

  
OO水晶（根据你选择的路线冠以那个巨人的名字）  
分类：奇物  
描述：水晶是每一名巨人的象征与能量核心，某种意义上是和光之巨人的水晶类似的存在，但是光之巨人的核心是那名巨人的弱点与现在身体精神状态的信号，在某个宇宙也曾经被人评价为生命的计时器。不过那也仅仅是光之巨人的情况，暗之巨人的话不存在弱点，也不会在你虚弱的时候闪灯。  
OO水晶是逐级购买，效果替换。  
  
  
phase.1 D+1000  
描述：在这个阶段你真正的开始掌握你的能量核心并尝试控制强化你所拥有的力量。虽然你仅仅是刚刚入门，不过就算是再弱的巨人开始掌握了自己的力量。  
因为你对于能量的控制逐渐娴熟，现在你的光线攻击使用的的武器的伤害提升3L  
特性：【真实世界】  
【真实世界】：你也能看到一些常人肉眼看不到的东西并能触碰到对方，这也是为什么人间体能看到并干涉某些普通人看不到的事物的原因。从现在起你获得了【灵感视觉】可以看到灵体、虚体等一些常态下无法接触的事物，同时因为这幅躯体的特殊性你获得了能干涉它们的力量，现在你的攻击获得【法破】特性。  
  
phase.2 C+1000  
描述：在第一阶段的基础上，你对于能量的掌握与强化更近了一步。现在的话你已经可以进一步增强你光线技的威力并开始接触到了一些其他的应用。你现在可以使用心灵感应的方式与人沟通并共享相应的信息位置。你可以对你的核心中制造了一个只属于你存在于别的次元的特殊空间（画风上）可以方便你储物。  
因为你对于能量的控制逐渐娴熟，现在你的光线攻击使用的的武器的伤害提升7L  
特性：【真实世界】【心灵感应】【四次元口袋】  
【真实世界】：你也能看到一些常人肉眼看不到的东西并能触碰到对方，这也是为什么人间体能看到并干涉某些普通人看不到的事物的原因。从现在起你获得了【灵感视觉】可以看到灵体、虚体等一些常态下无法接触的事物，同时因为这幅躯体的特殊性你获得了能干涉它们的力量，现在你的攻击获得【法破】特性。  
【心灵感应】：通过对念力和能量的应用，你现在可以使用自己的念力跟人进行无延迟的心灵沟通并可共享五公里内对于单一目标的模糊定位，但因为你的力量还不完全成熟现在只能同时有两人通过这种方式沟通。  
【四次元口袋】：很多巨人都会有一个独属于自己的空间，你也不例外，现在你可以把东西储存在独属于你自己的异次元之中，这是一个异次元的体积为100立方米，你放入和取出物品都至少是一个花招动作。  
  
phase.3 B+2000  
描述：在第二阶段的基础上，你对于能量的掌握与强化再次进化了。现在的话你已经可以熟练控制增强你光线技的威力并学会了一些其他的应用。你现在可以使用心灵感应的方式与人沟通并共享相应的信息位置，这个能力在二阶段的基础上得到了极大地增强。你可以对你的核心中制造了一个只属于你存在于别的次元的特殊空间（画风上）可以方便你储物。  
因为你对于能量的控制逐渐娴熟，现在你的光线攻击使用的的武器的伤害提升5L  
特性：【真实世界-提升】【心灵感应-提升】【四次元口袋】【黑暗吸取】  
【真实世界-提升】：你也能看到一些常人肉眼看不到的东西并能触碰到对方，这也是为什么人间体能看到并干涉某些普通人看不到的事物的原因。从现在起你获得了【灵感视觉】可以看到灵体、虚体等一些常态下无法接触的事物，而因为你现在可以用自己的念力加强也辅助你的视觉进行探查所以你现在还拥有了基于视觉的【透视】能力，同时因为这幅躯体的特殊性你获得了能干涉灵体、虚体的力量，现在你的攻击获得【法破】特性。  
【心灵感应-提升】：通过对念力和能量的应用，你现在可以跟人进行无限距离无延迟的心灵沟通并可共享100公里内对于单一目标的模糊定位或者5公里内对于单一目标的的精确定位，而且沟通人数不再受到限制。  
【四次元口袋】：很多巨人都会有一个独属于自己的空间你也不例外，现在你可以把东西储存在独属于你自己的异次元之中，这是一个异次元的体积为100立方米，你放入和取出物品都至少是一个花招动作。  
【黑暗吸取】：因为学习暗之咒术的原因，你的攻击可以吸取对方的生命来回复你自身受到的伤势与状态，你的攻击获得了【吸血2】并在吸血发动后获得1异常点治疗（医疗点 24，异常点数治疗可以治愈三至四档异常点）

描述：在第二阶段的基础上，你对于能量的掌握与强化再次进化了。现在的话你已经可以熟练控制增强你光线技的威力并学会了一些其他的应用。你现在可以使用心灵感应的方式与人沟通并共享相应的信息位置，这个能力在二阶段的基础上得到了极大地增强。你可以对你的核心中制造了一个只属于你存在于别的次元的特殊空间（画风上）可以方便你储物。  
因为你对于能量的控制逐渐娴熟，现在你的光线攻击使用的的武器的伤害提升14L  
特性：【完全世界】【心滴拳听】【四次元口袋 · 真】【生命吞噬者】  
【完全世界】：你也能看到一些常人肉眼看不到的东西并能触碰到对方，这也是为什么人间体能看到并干涉某些普通人看不到的事物的原因。从现在起你获得了【灵感视觉】可以看到灵体、虚体等一些常态下无法接触的事物，而因为你现在可以用自己的念力加强也辅助你的视觉进行探查所以你现在还拥有了基于视觉的【透视】能力，同时因为这幅躯体的特殊性你获得了能干涉灵体、虚体的力量，现在你的攻击获得【法破】特性，此外因为念力更加的强大使得你的敏感范围增加1倍。  
【心滴拳听】：通过对念力和能量的应用，你现在可以跟人进行无限距离无延迟的心灵沟通并可共享对100公里内任意目标的精确定位，而且沟通人数不再受到限制。

【生命吞噬者】：因为学习暗之咒术的原因你的攻击可以吸取对方的生命来回复你自身受到的伤势与状态，你的攻击获得了【吸血6】并在吸血发动后获得2异常点治疗（医疗点 48，异常点数治疗可以治愈三至四档异常点）
