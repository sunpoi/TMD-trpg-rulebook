# 4.防具

超古代肌肤护理  
资源类型：可升级防具  
本质：特异本质

描述：超古代肌肤护理会如同重甲一样占据你的盔甲位，由于其本质你自身的皮肤，所以其提供的防御是天生防御，也不会带来外观上的改变。  
你可以穿戴其他服饰，但是身上带有的超古代巨人特有的刺青将无法隐藏。

特殊：超古代肌肤护理是逐级购买的护甲，每个等级获得的效果将会保留计算。

D级  
等级名称：第一疗程  
价格：D+500  
描述：  
【奥式按摩】：  
因为在光之巨人和怪兽的按摩（击打）和防卫军的温柔的护理（枪击）下你的身体得到了保养。  
特性：【奥式按摩】  
效果：  
【奥式按摩】：  
你获得9点天生防御，获得了2点抵破甲和基于天生防御的【防弹】的特性。

C级  
等级名称：第二疗程  
价格：C+1500  
描述：  
【光子嫩肤】：  
接受了接受了按摩以后怎么能不试试日光浴？在好心的光之巨人和防卫军的崭新科技下，你得到了很.好的“日光浴” （光线洗礼）护理，让你逐渐的放松起来。  
特性：【光子嫩肤】  
效果：  
【光子嫩肤】：  
因为得到了护理，你的身体也进一步的成长，体积增加6点（虚拟体积）。在上个等级获得的天生防御提升至17点，抵破甲提升至5，再获得基于天生防御的【免疫接触攻击】【防能量武器】特性。以及额外获得【自适应短休】。

B级  
等级名称：第三疗程  
价格：B+2500  
描述：  
【射线温泉】：  
在经历了之前一系列的事情后，赶到战场的是太空纯天然射线温泉。  
因为你被放逐到宇宙里以石像的状态度过了千万年岁月，就算是温泉耐久仙人也比不过你。  
特性：【射线温泉】  
效果：  
【射线温泉】：  
在纯天然的宇宙射线中，你的身体同样获得了一些增益，在之前等级获得的天生防御提高到24点，获得基于天生防御的【全伤害吸收1】【坚韧】（如果是达贡会提升为超级坚韧），以及1点快速医疗的恢复效果（医疗点12，可与主轴叠加），即使死后遗体也不会损坏。  
从现在开始，你可以使用运动+光线代替常规技能进行格挡，在血统中获得的变形效果来源等级提升至B级。（现在你的体型可在1-57体积间用自由动作切换）
