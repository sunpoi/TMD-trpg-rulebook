# 暗之巨人主轴

超古代暗之巨人  
描述：很久很久以前暗之一族中有着最杰出的四大暗之巨人，巨人们为了暗之一族的未来，在3000万年前跨越无数光年来到地球寻找那传说中的永恒核心永。  
他们企图得到永恒核心的力量来重铸暗之一族的荣耀，虽然结论上他们失败了并没能得到永恒核心，但他们确实实在在的在宇宙中名为地球的这一颗小小的恒星的历史上留下了油墨重彩的一笔。暗黑勇士特利迦、妖丽战士卡尔蜜拉、刚力斗士达贡、崽种...咳咳打错了是俊敏策士希特拉姆。这正是这四位暗之巨人所留下的血脉与传承。

  
  
价格：D+600  
特性：【漫步深渊】【超古代暗之一族】【尚且幼小的巨人】【光线学入门教育】  
能量池：【黑暗粒子】  
【漫步深渊】：黑暗粒子虽然能带来强大的力量，但同时会让他们的情绪甚至是意志都会偏向极端，掌握这种力量无异于在名为堕落的深渊里漫步，这使得你意志池的上限减少3点。

  
  
【超古代暗之一族】：在你继承了暗之一族的血脉的同时，你的脸上将会出现一些奇异的花纹，你整个人的穿衣风格和首饰也会偏向于比较古典的风格，超古代时期人的人与人的体质可是完全不同的同样夜晚也是笼罩在黑暗之中，因此作为后裔继承了超古代血脉的你获得1点快速医疗（医疗点12）和黑暗视觉。

  
  
【尚且幼小的巨人】：就算是继承了四大暗之巨人的血脉与传承，你尚且只是个幼崽。不过好消息是你正处在绝赞成长期，你每经历1个影片便会在所选择属性检定上获得1点内在加值，这个属性由你选择的道路决定\[特利迦-决心 卡尔蜜拉-风度 达贡-力量 希特拉姆-敏捷\]选定后之后无法改变。

  
  
【光线学入门教育】：虽然还很稚嫩，但是你已经确确实实开始尝试掌握自身所拥有的力量了，你可以随手挥出或者手掌竖直往前伸，会出现一道红色粒子构成的光箭如同剑气一般飞向敌人，这也是你现在掌握的最基础光线技能。你获得了一种全新的攻击模式：  
根据你的道路选择的属性-运动-专业：光线-辅助专业：粒子学（学识下属），光线的武器伤害为3L 射程15米。  
根据你的流派不同，这个光线的伤害类型也不同：  
特利迦、卡尔蜜拉：暗能量  
达贡：火焰  
希特拉姆：声波  
基础的光线能力为之后的高级光线的使用打下了基础，特利迦、卡尔蜜拉、达贡流派在使用招牌光线（特利迦——黑暗哉佩利敖光线，卡尔蜜拉——卡尔蜜拉手指火花，达贡——火焰破碎机）时额外获得1修行加值。至于屑——希特拉姆，绝对不是因为被讨厌的原因所以不会获得加值。

  
  
【黑暗粒子】：开启名为黑暗粒子的黑暗属性能量。与象征着光明的光之巨人相对暗之巨人使用的是黑暗能量。这种能量比起光能量更加难以掌控，黑暗能量如同光能量能释放出独属于它们的光线的同时，还能用于一些奇异的咒术，甚至是掌握心灵。

  
  
关键属性：根据你的道路选择的属性而定+决心 #黑暗特利迦作为唯一转变成光的是个列外他是决心+耐力。  
\[只有某人不存在的街道\]：你初步成功掌握了这个这个属于你的能量，随着你的钻研也是根据你选择的道路的不同给你带来了一些变化。  
\-黑暗特利迦-：特利迦的意志无论什么时候都不会动摇，能量也随着你的心意给出了相应的回答。现在只要你能量池不空，你就可以获得1点决心检定加值。  
\-卡尔蜜拉-：卡尔蜜拉对于能量与技能方面的掌控是最深刻的，她是四人中最特殊的一位，现在你可以消耗2点能量观测你所在星球的天象得知接下来5天的天气。同时如果你已经预测过天气的话，你可以进行简单的引导改变今天的天象，改变前后需接近（如：阴天引导为下雨）。  
达贡和希特拉姆最一开始稍微落后两人没有进一步的能力。

  
  
基础用法：每轮1次，消耗1点能量强化自身的攻击或防御，加值为能量加值。  
恢复方式：通过一次短休进行一次决心检定回复成功数点能量，通过一次长休自动回满，一天最多可以恢复能量池3倍的能量。

  
  
价格：C+1200  
特性：【由你选择的道路】  
【由你选择的道路】：  
从以下四条道路中选择你的巨人形态，同时额外支付750分从主神那里购买你的专属变身器。这之后，你可以变身为相应的巨人形态，变身巨人形态和变回人间体均是一个自由动作。  
你开启巨人形态时，你的体型可在5-21体积间用自由动作切换一轮限定一次。这是一个C级变形效果并且通过这个增加的体积不会给你带来任何实际上的生命值收益，如同虚拟体积一样计算效果。

  
  
【暗黑勇士特利迦】  
暗黑勇士黑暗特利迦是三千万年前的四大暗之巨人之一，披黑色的铠甲，眼神凶恶，实力强大，是超古代暗之巨人的一员，虽然在接触永恒核心后转化成了光之巨人，但他处于暗之巨人时期的强大却是实实在在的。由于特利迦强大的实力与残暴的战斗方式，你的攻击获得了2点士气加值，同时三豁免2点增强加值。

  
  
【妖丽战士卡尔蜜拉】  
妖丽战士卡尔蜜拉是三千万年前的四大暗之巨人之一，妖娆美丽，性感魅惑，是超古代时期暗之巨人的首领。她身体柔软擅长光线的同时擅长暗之咒术，因此你的外貌得到了等同5点的提升。

  
  
【刚力斗士达贡】  
刚力斗士达贡是三千万年前的四大暗之巨人之一，性格粗鲁但十分正直且粗中有细，在遇到问题时也能沉着思考。奉行战斗和力量至上，厌恶给敌人设下陷阱的卑鄙的战斗方式。总是希望以势均力敌的力量来一决胜负，对手的力量越是不相上下，就越能发挥实力。强大的力量带来了强大的体魄，你获得了2点天生防御，同时三豁免2点增强加值。

  
  
【俊敏策士希特拉姆】  
俊敏策士希特拉姆是三千万年前的四大暗之巨人之一，性格狡猾奸诈，与卡尔蜜拉、达贡合作时心怀鬼胎。擅长使用各种策略的智谋战和心理攻势，口头禅是“Excellent”。希特拉姆的战斗风格是以他速度玩弄对手，你在攻击时获得了2点高速，同时只要你在这回合移动过还会获得2点闪避防御。

  
  
B级：  
特性：【新的传说开篇（由你选择的道路-提升）】  
【新的传说开篇】  
在C级选择的道路基础上，你得到了更进一步的传承越来越接近最初的四大暗之巨人。  
巨人在失去生命的时候会化作等体型的石像，石像需要一个月的时间慢慢吸取能量复活，除了观察外无法进行更多的动作，这种形态会获得等同于生命值的结构值，被摧毁则复活失败，这是一个B级变形来源的复活效果。  
现在你开启巨人形态时，基于你选择道路相应属性的检定获得8加骰，如果已有8加骰则提供相应属性的9dp增强加值。

  
  
【暗黑勇士特利迦】  
暗黑勇士黑暗特利迦是三千万年前的四大暗之巨人之一，披黑色的铠甲，眼神凶恶，实力强大，是超古代暗之巨人的一员，虽然在接触永恒核心后转化成了光之巨人，但他处于暗之巨人时期的强大却是实实在在的。由于特利迦强大的实力与残暴的战斗方式，你的攻击获得了【湮灭】。

  
  
【妖丽战士卡尔蜜拉】  
妖丽战士卡尔蜜拉是三千万年前的四大暗之巨人之一，妖娆美丽，性感魅惑，是超古代时期暗之巨人的首领。她身体柔软擅长光线的同时擅长暗之咒术，随着光线与咒术的研究，你的攻击获得了【幻灭】。

  
  
【刚力斗士达贡】  
刚力斗士达贡是三千万年前的四大暗之巨人之一，性格粗鲁但十分正直且粗中有细，在遇到问题时也能沉着思考。奉行战斗和力量至上，厌恶给敌人设下陷阱的卑鄙的战斗方式。总是希望以势均力敌的力量来一决胜负，对手的力量越是不相上下，就越能发挥实力。强大的力量带来了强大的体魄，从现在开始你就算没有完好生命也不会因伤势昏迷同时获得了基于天生防御的【坚韧】。

  
  
【俊敏策士希特拉姆】  
俊敏策士希特拉姆是三千万年前的四大暗之巨人之一，性格狡猾奸诈，与卡尔蜜拉、达贡合作时心怀鬼胎。擅长使用各种策略的智谋战和心理攻势，口头禅是“Excellent”。希特拉姆的战斗风格是以他速度玩弄对手，从现在开始你的冲锋攻击不会再降低基础防御，同时现在你在进行直线冲锋攻击后，若仍有剩余的移动距离，可以沿着冲锋方向继续移动剩余的移动距离。

  
  
A  
特性：【传说中的巨神】【背后的万家灯火】【永恒的碎片】  
【传说中的巨神】：  
在B级选择的道路基础上，你拥有的力量已经逐渐接近原初的那四位巨人。他们或许是祂们，黑色的巨神就算是死亡也不过是暂时的，祂们总是会重新归来。  
你在B级时候的复活效果提升，现在仅需要7天就可以重新复活，这是一个A级复活效果。  
同时被称为黑色巨神的你在宇宙中书写下了独属于你的故事，你在攻击时会附带【扼魂】的效果，这是一个命运来源效果。  
你在C级时的变形效果提升至A级，你现在可以将体积在1-120之间进行变换，这个体积增大同样不提供生命值收益，如果你已经购买了第三疗程，则你道路选则的属性检定上额外增加2点附加成功。

  
【背后的万家灯火】：  
你是永远不会倒下的，因为你的背后是万家灯火（雾）。众人的希望支撑着你（？），你获得了1点快速医疗的恢复效果（医疗点12，可叠加）。  
现在你可以为你身后的目标提供【全遮蔽】 的效果，如果对方已获得来源于你的【全遮蔽】，则你额外获得7点偏斜防御，此效果拥有【强拦截】 。

  
【永恒的碎片】：  
永恒核心之中蕴含着无尽的力量，得到永恒核心的力量甚至能重组宇宙，把宇宙改造成自己想要的样子。你得到了永恒核心的碎片，虽然只是很细小的一点微弱的力量，但这股力量也是非常强大的。  
在你攻击时身上笼罩着星星点点金色的光芒，这是独属于永恒核心的光芒，它将你的身体也染上了金色，你的攻击检定成功数增加一半，此能力为每轮一次。
