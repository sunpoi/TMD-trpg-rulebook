# 巨噬蠕虫

巨噬蠕虫(Dhole)  
特异本质  
神话血统  
“他脚下的地面被庞大的巨噬蠕虫侵蚀着。正好在他向下张望的时候，有一条蠕虫将身躯抬起了数百英尺之高，向他伸出苍白而粘稠的前端。”  
HP洛夫克拉夫特&E霍夫曼普莱斯,《穿越银匙之门》  
巨噬蠕虫是一种形似肉虫的巨大生物，会在地面上挖洞，并栖居其中；它们不是地球上的生物，看起来被带来这里的时间也仅有一个非常短暂的时期。因此地球幸运的没有如其他世界般被它们挖出的洞穴蚕食殆尽。尽管不会受到什么实质性的伤害，但是巨噬蠕虫依然不喜欢光。除了在那些在那些被它们彻底征服的星球上外，白天几乎看不到它们的踪影。巨噬蠕虫和钻地魔虫之间 似乎有某种不为我们所知的关系。而在幻梦境中，也生活着一种叫做巨蠕虫(Bhole)的、和它很相似的生物。

  
特殊的，巨噬蠕虫是上位独立种族，这意味着该血统无法与任何具有信仰导向的资源兼容。

  
D级：子嗣  
D+600  
描述：长如蛇般的躯体，暗色的坚韧表皮，以及在身下随着你移动如触手内侧一样的吸盘，头部看不出眼睛，巨大的圆形口器微张，露出了里面在灰白色粘液中蠕动着的利齿和细密的触须，这就是你现在的样子。  
特性：【巨噬蠕虫】【吞噬成长】【坚韧身躯】  
【巨噬蠕虫】：你在购买该血统后将会立刻从足部开始结晶化，最终化作一颗被华丽的宝石包裹着的，体积为5的巨卵，你会在接下来的六天时间内快速孵化，并在破壳而出后获得全新的身体形态，你失去了四肢取而代之的是身体下方的一排排蠕动的细长触手，现在你眼睛极小，你的头部、脖子和躯体融为了一个整体，你的生物类型不可避免地转变为【神话生物】  
若你购买了装备，需要额外花费250分令其适应你的非人躯体。  
你的意志上限永久减少1点，这视为你作为超越人类的物种幼体，精神尚未成长时产生的衰弱。  
【吞噬成长】：你的口器与全身现在视作3L的天生武器，你可以用武技-口器来以牙齿口器攻击，武技-拳被替换为武技-触须（仅名称变化），武技-腿被替换为武技-尾（仅名称变化），无论以何种天生武器部分进行攻击，总是视为钝击伤害。  
现在，当你在影片内吞噬了至少10体积的新鲜血肉，你在影片结束后会提高3点体积。  
特殊的，由【吞噬成长】成长的体积不提供血量上的增长，如同虚拟体积一般处理。  
彻底吞噬1体积的血肉是一个持续5分钟时间的战斗外行为（包含消化时间），需要吞噬的血肉每增加1体积都会使这个时间提升5分钟。  
任何由主神空间产出的食物或者造物、随从等都不会提供吞噬成长，仅能从影片中的新鲜血肉获得吞噬成长。  
【坚韧身躯】：你的外壳会在穿梭世界的过程中变得愈发坚韧，你获得2点天生防御，每经历一场影片，你都会额外获得1点天生防御。  
在经历三场影片后，你将获得基于天生防御的2DR/-，这个收益仅生效一次。

  
C级：贪食者  
C+1200  
描述：你的存在本身足以成为恐慌的来源，那令人不寒而栗的触须蠕动着，伴随着圆形磨盘般密密麻麻的尖齿摩擦声，昭示着吞噬者的到来。  
特性：【难抵贪念】【提升-吞噬成长】【钻掘】  
【难抵贪念】：你难以抵御源于血脉本能的吞噬欲望，这导致你的意志池永久减少1点。  
【提升-吞噬成长】：你的体型迅速成长，当你购买了该血统等级时，【吞噬成长】立即为你提供6点体积成长。  
现在你可以加速成长，当你在影片中吞噬了足够的新鲜血肉并触发【吞噬成长】时，你可以在D级的基础上，额外成长3点体积，你的体型逐渐夸张，成长速度越来越快。  
特殊的，由【吞噬成长】成长的体积不提供血量上的增长，如同虚拟体积一般处理。  
现在5分钟时间可以彻底吞噬3体积的新鲜血肉。  
【钻掘】：在适应了掘地之后，你的身躯将进一步进化。现在你获得等同于基础移动速度的遁地速度，遁力等级等于你的传奇耐力\*1。

  
B级：庞然巨怖  
B+2400  
描述：当大地震颤着发出哀嚎，当天空被漆黑的阴影笼罩，那并非世界末日之光景，仅是你活动身体的写照。  
特性：【提升-吞噬成长】【提升-巨噬蠕虫】  
【提升-吞噬成长】：你的身躯虽然已经十分庞大，但你已然可以继续成长。当你购买了该血统等级时，【吞噬成长】立即为你提供6点体积成长。  
当你在影片中吞噬了足够的新鲜血肉并触发【吞噬成长】时，你将在C级的基础上额外成长3点体积，是时候将你的存在昭告世界了。  
当你购买该血统等级并第一次触发【吞噬成长】后，你额外成长2点天生防御，当你第二次触发【吞噬成长】时，将额外获得1点快速医疗（12医疗点）。  
当你完成前两次成长，触发第三次成长时，你的身体已经做好了遨游星海的准备，你获得等同于你基础移动速度的完美飞行速度。  
特殊的，由【吞噬成长】成长的体积不提供血量上的增长，如同虚拟体积一般处理。  
现在5分钟时间可以彻底吞噬等同你自身1/2体积的血肉。  
【提升-巨噬蠕虫】：  
你的外壳逐渐成熟，获得基于天生防御的【坚韧】，你以天生武器进行的攻击具有【贯穿】特性。你获得全环境呼吸，并且基于天生防御获得【三防】。

  
A级：钻星者  
A+4800  
描述：经过了漫长的成长岁月，你真正成为了足以在星河之间穿梭的庞然巨兽，行星也会被你的饥饿碾碎为食粮。  
特性：【无尽食欲】【钻星者】  
【无尽食欲】：当你购买了该血统等级时，【吞噬成长】立即为你提供15点体积成长。  
当你在影片中吞噬了足够的新鲜血肉并触发【吞噬成长】时，你将在B级的基础上额外成长10点体积。  
现在，每经历一场影片你就可以永久获得1点快速治疗（12医疗点），无上限。  
特殊的，由【吞噬成长】成长的体积不提供血量上的增长，如同虚拟体积一般处理。  
【钻星者】：你现在可以花费十分钟时间进行加速，在此期间不能进入战斗场景，当加速完成后，你可以以50000公里/时的速度进行跨星系旅行，你也可以选择以10000公里/时的速度遁地穿行以此在当前世界旅行。  
若在旅行期间进入战斗，则旅行速度失效。  
完全成熟的你拥有巨噬蠕虫强大的心灵感知能力，你现在拥有感知到敏感范围内所有生物具体方位的心灵感知，这是一种非线性强感知能力，这也为你提供了免疫A级心灵来源负面效果的能力，同时，足以忍耐太空辐射与高压的躯体也拥有免疫S级石化的能力。  
现在你免疫接触攻击。你的天生武器获得8加骰，【贯穿】提升为【超级贯穿】，同时。  
当你以口器攻击并击杀目标时，你可以立即吞噬其所有血肉；现在你可以立即消化进食的所有血肉。
