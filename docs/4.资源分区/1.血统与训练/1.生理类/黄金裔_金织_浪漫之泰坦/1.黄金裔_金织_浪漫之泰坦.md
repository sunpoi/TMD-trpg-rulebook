# 黄金裔-金织-浪漫之泰坦

黄金裔-金织-浪漫之泰坦  
资源类型：可升级血统  
本质：科技本质

概述：由翁法罗斯的实验产物半神-黄金裔中的其中一位泰坦，其本质为司长浪漫的职责，而追溯到其根本命途则是对应着纯美之星神伊德莉拉。

D级  
价格：D+600  
效果：  
获得【黄金裔-金织-浪漫之泰坦】下属技能树购买权。  
获得【衣匠】购买权。

【天衣无缝】：  
若你选择【黄金裔-金织-浪漫之泰坦】为特殊身份5开局强化，则你还可以额外携带一把迅捷剑入场。  
你需要经过一场影片才能解锁【天衣无缝】的所有效果。  
你获得一种特殊的标记方式，标记被称为【间隙织线】。每轮一次，当你进行的一次以迅捷剑为武器的攻击命中一个目标，并对其造成伤害时就对其施加1层【间隙织线】。

间隙织线：拥有【间隙织线】的目标对你进行的攻击将始终承受等同于【间隙织线】层数的表现减值。你还可以在某次对具有【间隙织线】的目标的攻击中消耗数层【间隙织线】，每消耗1层【间隙织线】，就在本次攻击中获得2点攻击检定上的表现加值。如无特殊说明，即使你的一次攻击中有多个攻击目标，也只能对其中一个目标施加【间隙织线】。

支付10XP，额外获得以下特性：

【织运之竭】：  
只有当你拥有【衣匠】时，这一特性才能开始生效。  
只要【衣匠】不离开你的敏感范围，你就具有基于听觉的D级魔幻本质侦测能力，这是一种非线性强感知方式，并获得【黑暗视觉】和可以与自愿与你建立连接的角色进行心灵沟通的能力，建立连接的角色数量不限，无需借助任何媒介，这视为一个D级信息共享效果。  
购买此特性还会正常给予你属性点，相当于一项D级技艺。

C级  
价格：C+1200  
效果：  
【飞驰之阳】：  
现在，你每轮施加的【间隙织线】上限将提升至2层，这意味这你可以在一轮中对两个目标施加1层【间隙织线】，或对一个目标施加2层【间隙织线】。

支付10XP，额外获得以下特性：

【刺纹之蜜】：  
你在对具有【间隙织线】的目标进行攻击时，还会至多获得等同于你的传奇力量\*4点忽略被攻击目标的闪避防御。  
购买此特性还会正常给予你属性点，相当于一项D级技艺。

B级  
价格：B+2400  
效果：  
【金玫之指】：  
现在，每当你施加【间隙织线】时，你将再额外为自身提供等量层数的【热狂圆舞】。这个效果一轮只能触发一次。你获得的【热狂圆舞】上限为你的传奇力量\*1层。

【热狂圆舞】：当你受到一次攻击时，若你仍保留【热狂圆舞】层数，则你可以立即消耗1层【热狂圆舞】，选择向任意方向移动不超过场上单位携带【间隙织线】层数\*10M的距离，上限为30M。这次移动具有打断性。

支付10XP，额外获得以下特性：

【高举吧，升华的名讳】：  
每轮一次，当你以【衣匠】进行紧急守护时，若【衣匠】与你之间存在10M以上的距离，且最大间隔距离不超过场上单位携带【间隙织线】层数\*10M，上限为30M。则这次紧急守护会使你与【衣匠】交换位置，而非原本的移动至你的身旁，这一行为具有打断性。本轮内，当你被攻击时，攻击你的目标若具有【间隙织线】，则将额外承受【劣化1】。  
购买此特性还会正常给予你属性点，相当于一项D级技艺。

A级  
价格：A+4800  
效果：  
【刺纹之陷】：  
现在，你获得在使用【热狂圆舞】进行移动的同时进行攻击的能力，只要你消耗【热狂圆舞】进行了一次移动，则你可以立即对攻击者进行一次不附带招式的反击。这个效果每轮至多能触发3次，若你的某次消耗【热狂圆舞】进行的移动还移出了攻击者的攻击范围，则你进入【生死疲劳】状态。

生死疲劳：若你进行了一次紧急守护，且攻击你的目标具有【间隙织线】，则对于这次攻击而言，令该目标的【间隙织线】提供攻击减值负面效果翻倍（至多不超过12层），并使其本次攻击必须再重掷一次，由你选择攻击结果。这个效果一轮只能触发一次。
