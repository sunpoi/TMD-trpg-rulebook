# 将光阴织成黄金

将光阴织成黄金  
资源类型：可升级武器  
本质：科技本质

概述：  
「这一缕，是战士的牺牲。」  
「这一缕，是英雄的成长。」 「这一缕，是城邦的繁盛。」 …… 衣匠牵丝引线，汇聚在她的指尖。 「一些英雄的轨迹，已然断裂……」 她挽起另一些金丝，接续缝补。 「一些英雄的命运，仍要向前延展……」

被赋予神性的织机兀自运转，衣匠们曼舞翩跹。千年时光在织机上流过，她早已习惯了如此的等待——

为了那将光阴织成黄金的「灵感」。

D级  
价格：D+500  
分类：迅捷剑  
武器伤害：5L  
体积：3  
专业：迅捷剑  
伤害类型：物理-穿刺  
描述：一个织机的【梭】，其一部分被改造的可以手持，被其曾经的使用者添加上了繁复的花纹，但是可以在一些细节上看出，握把处已经有不小的磨损。  
效果：  
当场景内有角色携带【间隙织线】时，你对其进行的攻击将额外再获得等同于你的传奇力量\*1点高速。  
当衣匠在某轮中移动过时，你在该轮进行的移动速度将再增加等同于你的最高内在人物支线等级\*5米。

C级  
价格：C+1000  
分类：迅捷剑  
武器伤害：11L  
体积：3  
专业：迅捷剑  
伤害类型：物理-穿刺  
效果：  
获得【孤锋千吻】，这是一个攻击技。

技艺名称：孤锋千吻  
动作：标准  
持续：一场景  
分类：攻击技  
效果：  
对你触及范围内的一个目标进行一次迅捷剑攻击，你的本次攻击将会对被攻击者施加2层【间隙织线】，从当轮开始，每个该角色的回合结束时，【间隙织线】再增加2层。你也可以选择在某次攻击时，引爆该角色所携带的所有【间隙织线】，每引爆1层，就获得1点附加成功。【孤锋千吻】的效果不会重复对某一目标生效。

B级  
价格：B+2000  
分类：迅捷剑  
武器伤害：18L  
体积：3  
专业：迅捷剑  
伤害类型：物理-穿刺  
效果：  
获得【共舞吧，命定的衣匠】的购买权。  
当你进行一次攻击时，你可以选择消耗数个花招动作，每消耗一个花招动作，在画风表现上，你就视为对其刺出一剑。这意味着你会在消耗一个花招动作时对被攻击者造成3点多次攻击减值，最多不能超过最高内在人物支线等级\*3点。  
你的攻击检定时，你可以重投一次，两次结果任选。  
衣匠进行过攻击的行动轮中，你在其攻击结束后立即再获得4AP，这些AP只能使用黄金裔-金织-浪漫之泰坦的下属能力。

技艺名称：共舞吧，命定的衣匠  
价格：B+2000  
动作：整轮  
持续：立即  
分类：攻击技  
描述：生命，脆若游丝...  
效果：  
你的衣匠将失去本轮内的独立行动回合，若其已经行动过，则无法使用【共舞吧，命定的衣匠】。  
你的衣匠将在你的回合中与你一同行动，它在本轮内将可以对以你为中心，力量\*1为半径范围内的所有敌人进行一次攻击。你的本次攻击将额外对所有携带【间隙织线】的目标造成的伤害翻倍。
