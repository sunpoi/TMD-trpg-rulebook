# 龙王与勇者之遗骸

龙王与勇者之遗骸  
资源类型：可升级血统  
本质：魔幻本质

概述：  
在这片被龙王之血浸染的土地上，从那滩温热的血液中，缓缓地，仿佛被某种无形之力牵引，一个身影开始蠕动、挣扎。它的形态起初模糊不清，如同从梦境中逐渐凝聚的幻象，但随着时间的推移，轮廓愈发清晰——那是一只生物，既非龙也非世间已知的任何物种。  
它是旧日战士的残灵，也是龙王留在人世的不甘。

D级  
价格：D+600  
特性：【虚妄之龙】【万技为战】  
描述：  
\[少年踏上对抗巨龙的旅途，巨龙走上成王的道路\]

【万技为战】：  
你仍具有勇者的自觉，那些为杀死巨龙的武艺开始在你的灵魂深处响起。

效果：  
【虚妄之龙】：拥有该血统后，你在战斗时肤色会不自觉的呈现为血色。  
在涉及到判断你的种族效果时，在你的原种族与【龙】之间取劣。  
你开启强化能量池【虚妄魔力】  
【虚妄魔力】：  
不该存于世的龙，已然逝去的勇者，龙血中的生命魔力与剑士的气血斗气相结合而成的能量；虚妄魔力取决于宿主身体强度和意志力，其性质接近生命魔力，可以正常的如生命魔力所使用和强化，但区别在于，虚妄魔力的属性构成为耐力\*2。  
恢复方式等同魔力的基础恢复方式，每日至多恢复能量池上限的三倍。

【万技为战】：  
强化后在每场影片结束时你依次获得以下效果，不可重复获得：  
累年之业：战斗的训练使你的武技技能相关检定获得1点修行加值。  
经年之意：对无数场死斗的决心使你发起的具有武技技能构成的近战攻击具有【狂怒1】。  
一瞬之机：大量的战斗经验为你带来了面对战斗永远做好准备的能力，这使你额外获得一枚D10先攻骰。

C级  
价格：C+1200  
特性：【逆鳞之龙】【破鳞极武】  
描述：  
\[少年成为屠龙的勇者，巨龙成为惊世的魔王\]

【逆鳞之龙】：  
你的全身现在长满了逆向的透明龙鳞，你可以通过运转魔力使龙鳞显现出不同的颜色。

【破鳞极武】：  
勇者经年累月的修炼，就是为了穿破龙鳞，直冲要害。

效果：  
【逆鳞之龙】：  
每经历一场影片，逆龙鳞都会在你的身上变的更加坚硬，每场影片结束时，你获得一点【活体护甲】，以该特性获取的【活体护甲】提供的是天生防御而不是盔甲防御。

【破鳞极武】：  
当你使用攻击构成具有武技技能的攻击方式时，这次攻击额外具有5点破甲

B级  
价格：B+2400  
特性：【龙王之魂】【勇者之魂】  
描述：  
\[勇者踏上与龙王的对决，龙王在山顶等待着勇者\]

【勇者之魂】：  
勇者的武艺已经登上顶峰，向着最后的巨龙发起冲锋，二者一正一负，恍若本就对立的事物相撞在一起，其最终的结局便是湮灭殆尽。

【龙王之魂】：  
龙王的力量无比庞大，即使是神兵利器也无法打穿。

效果：  
【勇者之魂】：  
你的攻击视为具有【湮灭】效果。

【龙王之魂】：  
以一个移动动作，你将身上的龙之力激发，这一过程会结合在一次正常的移动行为中，头长龙角，部分鳞片展现金属光芒，尾椎生长出一根带刃龙尾，转变成龙人形态。  
在龙人形态下，你获得以下益处：  
当你使用攻击构成具有武技技能的攻击方式时，这次攻击额外提升4L武器伤害并具有1点额外的附加成功。  
该形态下，无论是否处于你自身的回合，你的刃尾总是视为为你进行了一次格挡行为，在此基础上，由格挡获得的格挡防御转变为天生防御。  
你可以随时以一个自由动作退出龙人状态，每场景只能切换形态一次。

A级  
价格：A+4800  
特性：【无期无法之魂】【永恒永囚之龙】  
描述：  
\[凯旋的勇者隐于无人之地，坠落的龙王困于永世之暗\]  
……吗？

【无期无法之魂】：  
为了击坠龙王，勇者磨炼出了粉碎魔力的武艺。

【永恒永囚之龙】：  
为了毁灭勇者，龙王将自身的力量压缩至极致，此时，当你处于龙人形态时，你的龙角将变得庞大而狰狞，浑身逆鳞暗金流转，你的背后生出强壮的有鳞双翼，你的双眼被漆黑蒙蔽，独留金色竖瞳。

效果：  
【无期无法之魂】：  
当你以格挡姿态承受【魔力】能量驱动的攻击时，若那次攻击的能量等级不高于A级，你可以硬抗那次攻击造成伤害，而以肉身将魔力轰散，令那次攻击带有的所有以魔力驱动的负面效果全部无效化。  
现在，即使不进入龙人形态，你的基础防御与格挡带来的防御类型依然转变为天生防御，并额外获得4点天生防御与天生防御上的4点附加成功，具有基于天生防御的4点全伤害吸收。

【永恒永囚之龙】：为了毁灭勇者，龙王将自身的力量压缩至极致，在【龙王之魂】的基础上，取代原有特性效果。现在当你以移动动作试图进入龙人形态时，需要额外消耗11点虚妄魔力，进行一次移动行为，你可以以此从背后的龙翼中获得机动性为完美的飞行能力，从逆鳞中额外获得24点天生防御并具有基于天生防御的【超级坚韧】特性，从无尽搏动的龙心之中汲取龙血的力量，带来源源不断的生命力，令你获得【快速医疗5】（60医疗点数）。  
只要还处于龙人形态，每个回合开始时，你都可以立即支付11点虚妄魔力，若如此做，你会在本轮内，承受来自外界的攻击时，令那次攻击承受基于你天生防御的【劣化1】，并且在该轮次内有一次机会，令攻击目标为你的一次攻击重投取劣计算。  
原本B级的【龙王之魂】特性中，生长出龙刃尾巴进行常态防御的能力，如今已经不需要变身龙人也可以随时进行，哪怕在战斗外、承受突袭时，都可以瞬间生长出带刃龙尾进行格挡。
