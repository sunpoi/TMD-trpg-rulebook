# 女仆技艺

奇术「Misdirection」  
价格：D+500

动作：以标准动作进行攻击  
目标：单体敌人 

效果：你对一名敌人投掷出飞刀的同时，瞬间从另一个方向发射出一枚新的飞刀，由于敌人的注意力全部放在第一把小刀上，因此很难防备住，你的本次投掷攻击将获得【连射】，同时额外获得4点高速，若该高速结算后仍有剩余则转化为攻击上的速度加值。

奇术「幻惑Misdirection」  
价格：D+500

动作：以标准动作进行攻击  
目标：至多5名敌人

效果：瞬间投掷出大量的飞刀，让敌人一时间无法分清那一把飞刀才是真正攻击的手段，你的本次投掷攻击时，会获得2点附加成功。

幻在「Clock Corpse」  
价格：C+1000  
等级C  
动作：以标准动作进行攻击  
效果：投掷出飞刀，在这一瞬间启动时停（画风上的，并不是真正的时停），随后布置大量的小刀，在时间重新开始流动的时候，所有的小刀会在瞬间同时飞向目标单位。  
使用本技艺时，本次投掷攻击获得2点速度加值，你从本血统中获得的C级措手不及效果将提升为B级，同时获得【9加骰】，若本次攻击成功命中目标单位，则目标需要以强韧豁免对抗伤害出目，若失败，则额外赋予令目标攻击检定获得【劣化1】的效果，持续一轮。

可以额外支付C+2000将该技艺差升为幻幽「Jack the Ludo Bile」，等级变为B级，若如此做，那么该技艺所带来的措手不及效果提升至A级，上述效果中的【劣化1】变为【劣化2】，【9加骰】改为获得【8加骰】。

女仆秘技「操弄玩偶」  
价格：C+1000  
动作：以标准动作进行攻击  
效果：广范围内撒出飞刀，之后令其瞬间变换方向向敌人发动攻击。你的本次投掷攻击可以至多将你所拥有的至多6点高速属性转化为等值的破甲属性，并对你攻击范围内的所有目标单位发动攻击，由于同时射出的飞刀目标足够多，因此本次攻击还附带【余波】特性。

你可以额外支付C+2000将该技艺差升为女仆秘技「杀人玩偶」，令该技艺等级提升至B，若如此做， 那么在使用本技艺时，一轮一次，令本次攻击伤害增加一倍。

你可以再额外支付B+4000将该技艺差升为  
幻葬「夜雾幻影杀人鬼」，令该技艺等级提升至A，若如此做，那么使用本技艺时，你的攻击将附带【传奇】特性。

伤符「Inscribe Red Soul」  
价格：B+2000  
等级：B  
动作：以标准动作进行攻击  
效果：你用双手执匕首（当然依旧投掷检定）对敌人实施连续斩击。尽管射程有限，但发生快、覆盖范围广且附加无敌，是几乎没有破绽的万能技。你的本次投掷攻击会施加持续一轮的A级重伤效果，且额外附带2点不可豁免的流血点数。

你可以再额外支付B+4000将该技艺差升为伤魂「Soul Sculpture」，令该技艺等级提升至A，若如此做，那么使用本技艺时，将附带等同于伤害出目点的流血点数。正常豁免，同时每轮一次，本技艺命中后使目标激爆，基于流血点数。

奇术「Eternal Meek」  
价格：A+5500  
等级：A  
动作：以标准动作进行攻击  
效果：向前方连射大量飞刀。但并没有进行瞄准，而是随意地乱扔，你的本次投掷攻击，选择以下效果中的一项效果获得

【1】：本次投掷攻击获得2点速度加值，从本血统中获得的C级措手不及效果将提升为A级，同时获得【8加骰】，若本次攻击成功命中目标单位，则敌人需要以强韧豁免对抗伤害出目，若失败，则额外赋予令敌人在攻击检定获得【劣化2】的效果，  
【2】：你的本次投掷攻击可以至多将你所拥有的至多6点高速属性转化为等值的破甲属性，并对你攻击范围内的所有目标单位发动攻击，同时附带【余波】特性，一轮一次，在使用该技艺的该条效果时，令本次攻击伤害增加一倍。  
【3】：你的本次投掷攻击会施加持续一轮的A级重伤效果，且额外附带2点不可豁免的流血点数。

幻象「Luna Clock」

价格：B+4000  
等级：B  
动作：迅捷动作  
效果：使用迅捷动作启动，随后的一场景内，一轮一次，在一次攻击后，可以额外进行一次不附带技艺的攻击，表现上为你每轮都获取了一些额外的时间。
