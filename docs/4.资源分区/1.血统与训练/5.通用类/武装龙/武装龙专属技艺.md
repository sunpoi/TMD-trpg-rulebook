# 武装龙专属技艺

武装龙专属技艺

技艺名称：【武装龙的霹雳】  
价格：D+500  
效果：  
武装龙对雷电能量有着较高的适应性，你开始尝试将其注入你的武器，你的【龙武装甲】武器获得【法破】特性  
你获得一种特殊移动方式。在非战斗场景下，你可以进行一次十分钟到1小时的蓄力（具体蓄力时间视移动距离而定），而后化作一道雷霆射入空中，再落下到你要移动的目标地点。  
以此方法可以在一小时内从城市的一端至另一端，也可以在一小时内从一座城市抵达另一座相邻城市，这是一个D级时空来源的【旅行】效果。

技艺名称：【武装龙的袭雷】  
价格：C+1000  
购买前提：【武装龙的霹雳】  
效果：  
你的【龙武装甲】武器充斥雷电能量，获得【能量武器】特性。并且由于雷电能量赋予的磁力，武装会自动悬浮吸附在你身上，这使你免疫【击落武器】效果。

技艺名称：【武装龙的震霆】  
价格：B+2000  
购买前提：【武装龙的袭雷】  
效果：  
你使用【龙武装甲】武器攻击时，雷电能量可以从武器身上延伸发射，使这些武器的攻击触及距离提升60m。  
当你死亡时，你可以转为一朵雷云漂浮在遗体上空，该状态下你除了观察和移动外无法进行更多的动作，经过一个月的时间后可以复活。  
雷云态下你具有等同于你意志值的结构值，若被破坏则复活失败。这是一个B级时空来源的【复活】效果。

技艺名称：【武装龙的万雷】：  
价格：A+4000  
购买前提：【武装龙的震霆】  
效果：  
你的身躯和雷电融为一体，当你进行移动时会化作电光穿透各种障碍，这是一个B级【传送移动】效果  
你创造的装备也彻底和雷电能量融合，你现在使用【龙武装甲】武器进行的攻击可以附带等同伤害出目的【麻痹】不良点数，正常豁免；你进行的攻击具有B级【传送攻击】效果，伤害类型可以转为【雷电】能量伤害。  
此外，你可以吸收战场上已死亡的单位的电能量，你具有【生息】特性。

技艺名称：【铠骑士武装】  
价格：C+1500  
效果：  
你学习了如何强化装备的防御性能，获得这一能力并经过一次影片后，你从C级龙武装甲中获得的盔甲防御提升至18/18，并获得基于盔甲防御的防弹、防能量武器；在那之后，每场影片结束后，你创造的每件【龙武装甲】提供的盔甲防御会提高【1】点。

技艺名称：【冲击钻武装】  
价格：B+2000  
效果：  
你学会了在装备上增加旋转结构以增强攻击力，每场影片后，你创造的每一件【龙武装甲】武器的武器伤害提高【1】点；在处理将未使用的【龙武装甲】武器一半基础属性与特性加入攻击的效果时，这一能力只会计算一次而非多次。  
同时因为你身上武器的威慑力，敌人往往会有所退缩，你的先攻提升10点。

技艺名称：【魔化/玄化武装】  
价格：B+2000  
效果：  
购买此技艺后，你可以选择转变为【暗黑生物】或【光明生物】之一。当你已是【暗黑生物】或【光明生物】，并选择转变为另一类时，你将同时视作【暗黑生物】和【光明生物】，并且在需要进行判定时取劣势效果。  
当你为【暗黑生物】时，你获得【魔化武装】特性：你使用【龙武装甲】武器的攻击附带B级【重伤】效果。  
当你为【光明生物】时，你获得【玄化武装】特性：你使用【龙武装甲】武器攻击命中对手后，对手以意志豁免对抗你的伤害出目，若对抗失败则被你放逐至异空间一轮。这是一个B级时空来源的【放逐】效果。

技艺名称：【白之衣武装】  
价格：B+2000  
效果：  
购买此技艺后，你的全身将覆盖能够隔绝魔法与陷阱效果的白色轻纱（你可以理解为全身白丝）  
你免疫A级以下由魔力驱动的效果影响。  
每轮一次，你的三豁免检定之一可以重新投掷并取更高的结果，无论如何此效果一回合只能让你的三豁免之一重骰取高。

技艺名称：【同盟武装】  
价格：B+2000  
效果：  
习得此技艺后，你的【龙武装甲】可以变形为更加独特的【同盟装备】。  
购买后选择以下五种武装之一，当你装备一件【龙武装甲】时，你可以选择让其获得对应的效果；一件【龙武装甲】只能获得以下的一种效果；若需要处理来源，均视为B级创伤来源。  
你可以重复购买此项直到你的装备的5件【龙武装甲】各获得以下的一种效果。  
①V-喷气机虎：你可以让一件【龙武装甲】增加喷气结构，极大程度增强武装的推动力。这使得该武器获得【超级贯穿】和【接触攻击】效果。

②W-弹射飞翼：你可以让一件【龙武装甲】增加弹射结构，攻击时可以在近距离再次造成冲击。这使得该武器获得【余波】效果；使用该武器进行的攻击命中后，目标采用反射豁免对抗伤害出目，失败则被击飞差值/米，这是一个B级击飞效果（正常具有B级击飞的其他效果），因击飞效果结束后计算坠落伤害的距离时，按照60米计算。

③X-首领加农：你可以让一件【龙武装甲】增加炮管结构，攻击时附带炮火轰击。该武器变为对触及距离内所有敌人造成伤害，可以改为使敌人进行反射豁免而非正常防御该武器进行的攻击，并且具有【湮灭】特性。

④Y-机械飞龙：你可以让一件【龙武装甲】增加龙头结构，龙头形象会震慑对手。这使得你获得【威仪】效果，并且该武器在攻击时附带伤害出目四分之一的【恐惧】和【精神束缚】不良点数，正常豁免。

⑤Z-金属履带：你可以让一件【龙武装甲】增加履带结构，可以在任何地面自由移动。这使得你在装备状态下获得免疫不超过B级环境来源效果影响都能力，同时获得等同于移动速度的游泳和遁地速度（遁力0），并且使用该武器攻击时获得【碾压】效果；你可以额外支付D+500，使遁地速度的盾力提升至6，这一额外支付视为进行了一个D级技艺的购买。
