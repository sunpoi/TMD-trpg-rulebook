# 3.猎人锯肉刀

猎人锯肉刀

物品名称：猎人锯肉刀  
等级：D级  
价格：D+500  
武器类型：刀  
体积：3  
武器伤害：6L  
伤害类型：挥砍  
武器特性：2破甲 1破魔  
描述：  
工厂诡兵器，一般用于猎杀行动  
这把样子能变形为善用离心力的开山刀，让其面对比自己体积更大的怪兽也能发挥自己的优势----让怪兽变迟钝，让你变得更敏捷。  
只不过这是猎人的技巧还是武器的功劳，便不得而知了。  
它那嗜血锯齿就象征着猎杀怪兽。

效果：你可以使用一个花招动作将此武器变形，变形后你的武器触及距离提高1米，若你在进行攻击后使用自用动作将此武器变形，则触发变形斩，在下一次使用此武器攻击时获得2点dp加值。

等级：C级  
价格：C+1000  
武器类型：刀  
体积：3  
武器伤害：11L  
伤害类型：挥砍  
武器特性：3破甲 3破魔  
效果：现在当你使用变形斩时，你获得【吸血2】

等级：B级  
价格：B+2000  
武器类型：刀  
体积：3  
武器伤害：21L  
伤害类型：挥砍  
武器特性：8破甲8破魔，若该武器的破甲以及破魔属性结算后仍有剩余则转化为攻击上的增强加值，至多各转化5点。  
效果：现在你的武器附带【幻灭】，并且当你使用武器以变形斩进行攻击时，你的攻击附带二分之一的肢体妨碍点数。
