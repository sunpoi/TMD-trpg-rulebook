# 0.小动物之友总纲

小动物朋友们  
随从组  
该随从组目标为添加一些无支线的可爱小动物们与一些简单的小动物升级，并且可以在日常之中进行一系列的探查作用。该随从组遵从最基础的随从规则，在战斗阶段的指挥也需要进行进行标准动作的指挥。
