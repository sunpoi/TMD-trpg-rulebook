# 超级小动物- 超级猫猫

超级猫猫  
D级随从  
D+500（也可以从无支线的猫猫随从差价升级过来）  
动物种类：猫科动物  
猫（拉丁学名：Felis silvestris catus），是食肉目猫科猫属 的脊索动物。猫体型小，体色由蓝灰色到棕黄色，体型瘦削，身长0.3-0.5米，全身毛被密而柔软，锁骨小，吻部短，眼睛圆，颈部粗壮，四肢较短，足下有数个球形肉垫；舌面被有角质层的丝状钩形乳突。猫雌雄性个体彼此相似，仅雄性头部粗圆，个体大些。猫的繁殖与生育期长达一生，寿命有12-17年。  
而在经过主神的强化之后，猫猫的能力受到了极大的强化，在从完全没有什么能力的普通猫猫变成了一个具有战斗力和一定侦查能力的超级猫猫

专长：  
猫之本能  
完全被激活的来自于猫猫的本能，被主神强化之后猫猫原有的能力回到本来应该有的情况并且有的还被超级强化。  
猫的眼睛可以看清黑暗之中的物体，猫猫的眼睛具有黑暗视觉的能力。猫猫本身具有在高处掉落也可以减免伤害的力量在主神强化过获得了免疫坠落伤害的能力并且在进行攀爬时可全速移动。

  
猫之利爪  
猫猫本身的爪子经过主神的强化终于发挥出来原本的能力  
现在猫猫的天生武器具有3L的武器伤害和2点破甲（转换出来的破防为破甲而非威猛）并且在进行攻击检定时获得2点内在加值

  
洞察之瞳  
在传说之中猫猫和狗狗都有着对于鬼物等传说生物都有着探测的效果，并且猫猫对于事物的侦查也远远大于正常的人类  
猫猫的眼睛现在可以看见更多的正常情况看不见的东西，获得【灵感视觉】效果，并且猫猫在使用感知属性为主属性的检定中获得2点附加成功
