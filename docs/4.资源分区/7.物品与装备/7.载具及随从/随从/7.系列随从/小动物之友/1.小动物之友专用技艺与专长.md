# 1.小动物之友专用技艺与专长

小动物朋友专用技艺与专长  
基础化形术  
无支线技艺  
250分  
来自于逆位死神提供，为这些无支线的小动物提供化形的能力，但是很抱歉基础的化形术并没有进行变回去的能力只有变成人的能力  
将你的小动物变成对应的人型（什么猫鸟，蛇人，犬娘啥的），但是本质上还是小动物属性和装备位不会改变（你还得操控攻击），现在也能跟你正常交流，体积则是会被固定为4体积，由小体型提高而来的体积不会提供血量。

  
专长：飞行能力（3XP）  
限制：鸟类动物  
赐予鸟类真正可以飞行的能力但是只有笨拙飞行的程度  
但是你可以通过往里面投入3XP进行升级，最多升级为完美飞行（即3次）
