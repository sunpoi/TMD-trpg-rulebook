# 宝可梦技能—一般

  

  
宝可梦技能—一般  
特别说明：一般系技能中物理系技能造成的伤害均为物理伤害。法伤技能造成的均为力场伤害。  
所有一般系技能皆具不有【法破】特性，需使用其他方式获得命中灵体的能力。  
【声明】：具有这一词条的技能是在一个战斗轮开始时决定发动的，除了花费声明动作外，训练家本轮的指挥动作也将在声明后被直接占用并消耗。  
  
  
连环巴掌  
C+1000  
指挥动作：标准动作。  
宝可梦消耗动作：标准动作  
效果：一般系物理技能，该宝可梦进行一次近战攻击检定造成钝击伤害，本次攻击获得等同于宝可梦敏捷属性点的技艺加值，以及1点来自于技艺的附加成功，受到本攻击的目标视为已经承受了4次攻击一般承受多次攻击减值。本次攻击具有【长点射】特性。  
差价升级至B+2000  
差价升级后仍可使用技艺的较低级别版本。  
则本技能攻击中附带的技艺附加成功提升至3点，目标视为已经承受了12次攻击一般承受多次攻击减值，【长点射】提升至【连射】，并获得【接触攻击】特性。
