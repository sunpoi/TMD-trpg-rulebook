# 小山猪

  

小山猪  
  
  
D+500  
独立随从

▓▓▓▓关于属性上的附加成功：  
力量上的每1点附加成功会为其提供力量检定上的3DP增强加值，可携带物品体积额外提升4点  
敏捷上的附加成功达到1点时会获得一个花招动作，达到4点时会将基础的移动动作升级为杂技动作，敏捷上的每1点附加成功会提供3米额外移速  
耐力上的每1点附加成功会为其提供5点额外的生命值  
  
智力上的每1点附加成功会为其提供每天一次，重投某个骰子的能力，但无论其结果如何，都必须接受那个骰子的结果  
感知上的每1点附加成功会为其提供额外的20米敏感范围及洞察技能检定上的1点增强加值  
决心上的每1点附加成功会为其提供2点额外的意志池上限，每达到2点就会赠送2点专注技能检定上的增强加值  
  
风度上的每1点附加成功会为其提供5点额外的外貌上升，并额外提供社交/表达技能检定上的1点增强加值  
操控上的每1点附加成功会为其提供1点操控检定上的附加成功与所有互动技能检定上的1点表现加值  
沉着上的每1点附加成功会为其提供在先攻检定上的额外一个十面骰  
  
▓▓▓▓攻击预设：  
普通攻击预设：力量+武技+牙+天生武器伤害=19+3造成严重穿刺伤害。  
  
宝可梦技艺近战攻击预设：力量+武技+牙+天生武器伤害+训练家专业：指挥+技能效果。  
  
宝可梦技艺投掷攻击预设：力量+运动+投掷+训练家专业：指挥+技能效果。  
  
宝可梦技艺施法攻击预设：感知+学识+魔法+训练家专业：指挥+技能效果。  
  
以上三种攻击预设只在宝可梦学习了具有该预设模式的技艺后方可使用。  
  
  
▓▓▓▓防御预设：基础防御4 5点雷电能量抗力 【5点火焰和冻寒能量抗力（当你选择了厚脂肪特性）】【闪避防御6+1（当你选择了雪隐特性且天气为冰雹时）】  
  
▓▓▓▓性别：公/母（购买时选定其中一种，选定之后无法更改）  
  
▓▓▓▓特性：  
  
【长牙未露不可谓之不利】：虽然此时小山猪的牙还只有雏形，但比起它的小小体型来说，那对长牙已经是令人生畏的武器。小山猪的牙齿视为3L的天生武器。  
  
【地面系宝可梦】：地面系宝可梦是天生的电系克星，小山猪获得了5点雷电能量抗力。  
  
【厚脂肪】：拥有厚脂肪特性的宝可梦在耐热和抗寒能力上的表现要远超其他宝可梦，小山猪获得5点火焰能量抗力和5点冻寒能量抗力。  
  
【雪隐】：作为诞生于冰雪能量中的宝可梦，每当天气为冰雹天气时，小山猪获得4点反射豁免的增强加值，6点闪避防御以及1点基于闪避防御的附加成功。  
  
【迟钝】：拥有迟钝特性的宝可梦在抵御来自于情绪等负面影响的异常点数时拥有更杰出的表现，小山猪不会受到魅惑点数带来的减值及严重状态。  
  
以上三种特性仅在购买小山猪时可以选择其中一种生效，一旦选择便无法更改。
