# 黑翼

黑翼  
C级随从  
C+2000  
该随从可自由行动  
蝙蝠型怪物。4000AP。通过与Knight背后合体给予其飞行能力，擅长从口中放出超音波攻击，以及高速飞行以翅膀作斩击或使出撞击作援护攻击。  
与莲契约之前由士郎的实验于现实世界召唤，莲的恋人惠理沐浴于超音波中。此时本打算将惠理作为猎物捕食，但因与莲契约而最终未能得逞。因为镜怪物的习性在契约后也有可能有捕食惠理，而在一开始被莲警戒着，在之后也对莲产生了信赖感，在呼唤名字后便会出现。  
名称的由来是，意味着黑暗的“dark”与意味着羽翼的“wing”。  
当初是意味着“夜之翼”的“Nightwing”的名称，但因为存在同名的机器人动画而废弃了。

防御预设：  
基础防御：11+2

蝙蝠型怪兽：黑翼是蝙蝠型怪兽，其也具有正常蝙蝠所拥有的能力与特性。超声感官，蝙蝠都具有的特殊感官，这是一种新建的强感知能力，因为超声波蝙蝠可以轻松地探查其他进入感知范围的生物，所以黑翼具有特异本质的C级侦测。飞行生物，作为一个飞行生物，自带着完美飞行特性。怪兽之威，作为怪兽黑翼的翅膀天生便被强化，极其锋利具有着3L的武器伤害。

超声波攻击：黑翼把蝙蝠的超音波进一步强化，使得其可以用超音波直接攻击对方，获得一种以敏捷+武技+音波+身体掌握+音波伤害的攻击检定，该攻击方式的属性为音波伤害，并且由于本身就是蝙蝠所以该音波伤害本身就具有3L的武器伤害，并且因为声音的传递性所以具有40米的攻击射程。

镜怪兽空间：因为镜怪兽天生便在镜世界之中生存，所以自然而然的学会了如何控制自己身体之上的反光面开辟一份独属于自己的10立方米空间的能力，其中不能装独立单位，使用者放入/取出物品至少是一个迅捷动作。

契约兽（黑翼）下属武器：  
模仿契约兽黑翼身上的特性然后靠着契约卡系统的能力制作而成的武器与护具，其中蕴含着契约兽身体之中的特殊能力。在其下购买的武器和护盾在购买后都储存在黑翼的体内，需要一个迅捷动作命令契约兽释放来取出和使用，画风为契约兽释放从天上飞下来你的武器和护盾自动装备在你的身上。

羽翼长矛者  
C级武器  
C+1000  
模仿了“黑暗翼蝠”的尾部的大型长矛。2000AP。像帕塔长剑一样持有保护近身的装甲，亦可作为盾使用。用于向敌人投掷的事也可能。有“隐藏着一击必杀的能力”的设定。  
类型：长矛  
基础属性：11L 长柄 双手 格挡2 破甲5  
体积：4  
【双刀流】：该武器在原本主人的手上都喜欢使用双刀流来进行攻击，并且这把武器也更容易去防御，所以这把武器可以兼容另一把把武器使得伤害得到更大的提升，当你双手各持一把武器时，你可以将未加入攻击的另一把武器的基础属性一半与所有特性加入攻击中  
【近身装甲】：虽然是一把长矛，但是其实看起来更像是一把长枪，在持有该武器时使用者可以使用该武器进行格挡时会获得1点全吸收和1点格挡防御  

羽翼屏障  
C级奇物（披风）  
C+1000  
背后装备的“黑暗羽翼”变化而成的披风。3000GP。与其它的降临卡片不同的，将契约兽直接变化的稀有类型的武装。（但是在神死的修改下变成了契约兽依然存在但是你的后背会多出来一个披风，可以常驻存在）  
【翼之障】：使用者在穿戴该披风的时候身上会被黑翼身上的那种声波气息所笼罩，获得2点全吸收和2点力场防御，使用者还可以主动消耗一个移动动作掀起披风让披风的保护面积大大增加，获得2点全吸收和3点力场防御
