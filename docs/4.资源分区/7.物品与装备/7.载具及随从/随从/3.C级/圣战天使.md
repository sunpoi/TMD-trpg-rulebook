# 圣战天使

  

圣战天使  
C+1250  
特殊的人类，能够进行「同契」、将自己的肉体变为武器的种族。武器化的圣战天使外形、特殊能力多种多样，一般的枪炮火器无法被认为是无法与武器化的圣战天使对抗的。因为武器化的关系，圣战天使消耗的力量依靠一般的方法比如饮食睡眠或者休息时无法回复的，必须依靠特殊的方式才能回复力量（当然，依靠睡眠或者饮食来回复力量的圣战天使也是存在的）。  
  
购买时需选择天使属性：地/水/风/火  
本体为14~22岁的人类女性，无任何特殊能力，外表可自设，相貌最低为3。

【能量充盈】使用圣战天使以一个迅捷动作使能量充盈刀身，下次攻击命中后造成等同于持有者传奇决心\*2点额外的严重伤害，伤害类型根据天使属性（火：火焰，水：冻寒，风：音波，地：钝击）进行转化。  
  
【天使的提醒】圣战天使的提醒在耳畔环绕，使你获得1点闪避防御，以及在先攻上的2点数值提升。
