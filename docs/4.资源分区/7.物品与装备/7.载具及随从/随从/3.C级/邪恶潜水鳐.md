# 邪恶潜水鳐

邪恶潜水鳐  
C级随从  
C+2000  
该随从可自由行动  
鳐鱼型镜怪物。4000AP。拥有在水中活动与空中飞行的能力。所有镜怪物中体重最轻的，但却拥有承载一个假面骑士的强韧的身躯。以两端的鱼鳍作为武器，高速飞行放出强力冲撞攻击是其得意技。  
名称的由来是，意味着“邪恶”的“evil”与意味着潜水者的“diver”。设计原型是漫画版《人造人间电脑奇侠》中登场的“鳐鱼机器人”。

专长：  
鳐鱼型怪兽：其本身是鳐鱼类型的怪兽所以自然具有在水中自由游动和黑暗视物的能力，在水中游泳速度等同于基础移动速度，并且自带水下呼吸和黑暗视觉。因为鳐鱼自带的发电器官，所以使得邪恶潜水鳐的近战攻击会附带二分之一的麻痹点数。而作为怪兽使得邪恶潜水鳐的两个翼有着3L的武器伤害。

飞空生物：虽然是一只鳐鱼型怪兽但是邪恶潜水鳐仍然具有着如同在海里一样随意畅游的在天空之中随意畅游的能力，其具有完美飞行特性。

镜怪兽空间：因为镜怪兽天生便在镜世界之中生存，所以自然而然的学会了如何控制自己身体之上的反光面开辟一份独属于自己的10立方米空间的能力，其中不能装独立单位，使用者放入/取出物品至少是一个迅捷动作。

邪恶尾鞭  
C+2000  
C级武器  
Raia所使用的模仿了“邪恶潜鳐”的尾部的鞭状武器。2000AP。能放出高压电流，被直接打中的人会因冲击而陷入活动停止状态。  
类型：长鞭  
基础属性；11L 轻灵  
体积：3  
高压电流：武器之中附带着邪恶潜水鳐的雷电之力，每次攻击都会附带着其身上强大的高压电流，使敌人感觉到酥麻的感觉，因此你使用该武器进行攻击时会附带二分之一的麻痹点数。
