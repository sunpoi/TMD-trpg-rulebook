# 半龙女仆·洗衣龙女

  

半龙女仆·洗衣龙女  
D+1250  
独立随从  
介绍：一只傻里傻气的可爱龙女仆，主要负责清洗才宅抵内的衣物，经常因为笨手笨脚将事情搞砸，不过谁又会因为这点小事对一只可爱的小龙女生气呢，不可否认的是即便是这种傻里傻气的小龙，也依旧拥有的成为未来龙皇的潜力。  
外貌：其身着一身青色的基调为主的女仆装，淡蓝色的头发被青色的帽子所覆盖，从中露出两只小巧的耳朵和短角，脸部显得有些婴儿肥，两只金色的眼眸以及一条粗壮的毛茸茸的青色尾巴更是平添了几分俏皮可爱。
