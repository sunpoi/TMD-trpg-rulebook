# 蛞蝓猫幼崽

蛞蝓猫幼崽  
D+1250  
独立随从  
介绍：这是一只可爱的蛞蝓猫幼崽，它有着所有蛞蝓猫共有的特征：黑豆般圆乎乎的眼睛，柔软弹性的躯体，蛞蝓似的大尾巴和猫儿似的脑袋瓜.  
你可以在购买该随从时自主设计其外貌特征，如外皮颜色、瞳色、牙齿尖锐与否等等，也可以应用【蛞蝓猫】血统中【个体差异】中的外貌特征，如圣徒的毛茸茸，猎人般的利齿，溪流的外鳃等等。  
通常而言，蛞蝓猫幼崽都是杂食的，并且不会像【矛大师】那样无法正常进食。  
主神出品，必属精品；由此购买的蛞蝓猫幼崽个体强度远超其他幼崽个体，精通与长矛近战和远程投掷，都是未来的狩猎新星。

随从特性：  
【灵活小猫猫】：  
\*蛞蝓猫幼崽可以准确的控制跳跃的落点和距离而无需检定，将跳跃和移动结合起来，每次跳跃的高度不能超过移动距离的1/2；回合结束时仍然需要站在稳固的落脚点上，否则就会坠落。此外，在伏地和攀爬时可以全速移动。  
\*蛞蝓猫幼崽可以用【武技+长矛】代替另【运动+投掷】进行投掷检定。

  
【无尽轮回】  
\*蛞蝓猫幼崽作为随从，即使死亡也会在次日凌晨复活。

  
【狩猎投矛】：你需要额外在主神空间花费300分购入一根狩猎投矛供蛞蝓猫幼崽使用，最多携带2根，该武器数据应用现实武器-近战武器-长枪，但是拥有【投掷】属性。你可以如常为该武器购买武器强化/附魔。
