# 机巧人偶

  

机巧人偶  
为代替人类作战而研发的战斗型机巧人偶，其拥有与人类相似的面容，并可以在一定程度上摸拟人类的感情,但无需将其视为人来对待，其本身只是工具。  
material四型  
D+1250

▓▓▓▓专长：  
机巧之躯:机巧人偶并非血肉之躯,因此在对其进行治疗时,不使用急救以及长线治疗，而是使用手艺+维修进行代替，不对结构值生效的快速医疗也无法恢复其生命。同时机巧人偶不拥有新陈代谢，其动力来源内都所装载的核能电池，因此其不需饮用水食，也无需氧气与休息并免疫无支线级的毒素和疾病。  
定向研发:  
在定向制造时若选择的是射击下属专业，则其获得3dp增强加值同时瞄准上限加1。  
若选择为武技下属专业，你将获得2点高速以及3点威猛，如为肉搏专业则天生武器可造成严重伤害。  
机型缺陷:material四型因设计缺陷导致其密封性有限，在其被液体所包裹时（例如处于水中环境）会导致其部分功能运行异常，这会令其在进行任何检定时承受3dp的环境减值。
