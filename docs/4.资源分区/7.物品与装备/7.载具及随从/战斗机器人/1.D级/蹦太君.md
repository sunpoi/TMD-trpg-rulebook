# 蹦太君

名称：蹦太君  
类型：战斗机器人  
本质：科技  
价格：D+500  
描述：史上最小的AS，本质他应该算是盔甲吧？但某个爆炸狂操控它的时候和脱了它的时候就是两个战力，所以——没错它是战斗机器人。  
体积5  
体积：5  
结构：22   
装甲：12   
载员：驾驶员1  
特性：  
小型AS：就如同描述所说，它是个战斗机器人，别管它多大。  
此系统依靠bug运行：因为其布偶装内建变声系统，当变声系统关掉时会导致系统冲突而关机，至今此问题仍无解。驾驶蹦太君时，你的所有发音都会变成“fumo”，这会让你无法通过语言传达消息。但你不通过发声系统就不会出事，所以手语或者另外配备一个对话器就行。  
单人化战术：因为体型，蹦太君可以正常使用现实基础武器，并且因为某位爆炸狂的改造，你可以绑定一件由你拥有的远程武器，这会赋予其【无限子弹】，一经绑定不可更改。
