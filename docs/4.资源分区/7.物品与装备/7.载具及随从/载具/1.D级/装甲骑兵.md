# 装甲骑兵

  
装甲骑兵  
价格：D+750  
通过魔力变换而武装起来的大型摩托。其速度远远凌驾于车体性能的极限之上。做得未免有些过火了。  
载具类型：轻型载具  
体积：4  
结构值：22  
装甲：6  
速度：基础200 加速度50 最高400  
载员情况：驾驶员1，乘客1，总体积不超过10

附属配件

辅助巡航系统  
价格:C+1250  
类型：奇物  
描述：  
能够快速收集与分析周围的路况，来为驾驶员规划出当前最合理的行驶路径，同时能令驾驶员的操控能更低延迟的反应到车体上，达成类似于‘人车一体’的驾驶状态。缺点就是唤醒需要大量的能量  
效果：  
消耗一个移动动作，往机体注入6点任意通用能量以唤醒辅助巡航系统，接下来的一个场景内，令装甲骑兵获得B级困难地形免疫，机动性提升两级（至多为完美），每轮能以自由动作进行一次载具移动
