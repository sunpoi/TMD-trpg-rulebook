# 梦幻马车

  

梦幻马车  
C+2500  
一辆被存放在魔法手环中的豪华古典马车，拥有四匹施加了飞行魔法的白马拖拽，该马车具备低空飞行能力，移动为良好，移动速度为60米/轮，防御面为20，体积9，结构值为35，梦幻马车会为它的乘客提供全遮蔽效果。  
当结构值降至四分之三时，视为损失一匹飞马，此后每降低四分之一便损失一匹飞马，当所有飞马都被击坠时，梦幻马车将无法再继续行动，如果在空中，则会不可遏制的坠落下去，但梦幻马车的设计者也考虑到了这一点，当梦幻马车坠落时，它将会缓慢自动下落，不会使乘客受到坠落伤害。  
马车内由于魔法原因具备比外表看起来更大的内部空间，总大小为10平米，高2米，具备2吨的承载能力，这视为一个储存空间。内部有奢华的装饰，拥有一个魔法壁炉、魔法餐桌和魔法留声机，魔法壁炉可以将马车内的温度永远保持舒适，而魔法餐桌则可以周期性的通过马车能量生成美味的魔法食物，魔法留声机则是可以播放多种乐曲，并且可以通过魔法手段将车内完全透视观察外部景观。  
备注：马车全速行驶下每小时能耗1点，每生产一次魔法美食能耗1点，每损失一匹飞马速度会降低四分之一。  
备注2：魔法马车无法依靠自身上升到水平面1000米以上的高度。  
表现力：通过手环唤醒马车，手环会在角色周围安全地带打开一个紫色传送门，马车会从该传送阵中跑出并停靠在角色身旁安全地带，马车的长度为4米宽3米高2米，拥有四个硕大的铁质车轮，车身为银白色华贵装饰并且带有魔法夜灯提供照明。马车内的正上方有一颗奥术水晶为整个马车提供魔法能源。
