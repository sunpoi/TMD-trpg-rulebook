# 海啸坦克

（出自命令与征服——红色警戒3）

海啸坦克  
价格：C+1000  
"升阳帝国侵略性十足的主力坦克，能在海洋及陆地上穿梭"  
海啸坦克独特的定位注定它的外貌不同于常规坦克，抛去其近似棱形的炮塔不谈——缩短并后移的两侧履带，以及车体前方单独连接的双排厚履带，也是极其大胆的布局。  
虽然海啸坦克因此总被敌人戏称为“三轮坦克”，但不可否认，它们确实如同海啸般能快速从水面冲上陆地。

类型：重型载具  
体积：14  
结构值：36  
装甲：20  
速度：60  
载员：驾驶员1  
特性：神镜装甲、海上奇袭、纳米修复、重型载具

【神镜装甲】：只要将纳米机器从主炮的操作中解放出来，海啸坦克就能制造出一个冲击防护罩，从外观上看就是一团围绕在车体周身的纳米机器云。  
每场景一次，以一个自由动作宣言使用，将海啸坦克装备的一件武器的武器伤害转化等额的护甲防御，但仅在紧急规避时生效，持续一场景。

【海上奇袭】：海啸坦克渡海的目的并非寻常两栖载具一般为了登陆，而是实打实的水上作战。虽然为此海啸坦克舍弃了不少装甲，但其也真正成为了帝国海权之下的得力干将。  
得益于配备的可翻转浮筒式履带，这视为其获得等同于基础移动速度的水面速度，并遵循载具规则移动。

【纳米修复】：海啸坦克配备的纳米虫群有在交战中临时顶替破损装甲的功能，从而达到战场维修的能力，但这不能修复已经报废的海啸坦克。  
这视为海啸坦克拥有，仅用于自身修复的快速维修1（自带维修点12），维修点的恢复与购买者绑定。

【重型载具】：海啸坦克为整个车组提供全遮蔽效果，其数值为载具体积/被遮蔽者体积取低。 并拥有基于护甲防御的【防弹】

\=====【下属武器】=====  
赖朝88mm穿甲炮  
价格：D+500  
旭日帝国的科技结晶，依赖纳米机械运转的新式坦克炮。在必要时，可以将其具有的纳米机器人用于增强自身装甲。  
伴随着清脆炮声与炮口的蓝色火焰，其连续且精准的射击给予目标持续压制。

武器伤害：8L  
武器特性：【高速5】【穿甲3】【连射】  
伤害类型：物理-穿刺  
射程：200m  
装弹量：5

【高速弹夹炮】：为了弥补装甲的薄弱，海啸坦克的设计师剑走偏锋，选择提高射速来弥补火力上的不足。视为其获得【连射】。

【纳米装弹机】：得益于升阳帝国的纳米科技，海啸坦克的主炮从装弹到射击可以完全仰仗纳米机器系统，大大解放了人力。  
主神的主神科技强化了这些纳米机器，现在其可以辅助填装弹药而无需装填，这视为其拥有【无限子弹】。
