# 秃鹰战车

（出自星际争霸）  
秃鹰战车  
价格：D+500  
“秃鹰战车主要担任侦察和巡逻任务，它所装备的两台大型火力发动机在极限状态下能令其以230英里每小时的速度狂飙，但其的敞开式驾驶舱又很好的克制了驾驶员狂飙。”

类别：轻型载具  
体积：8  
结构值：22  
装甲：6  
速度：基础80，加速度40，最高160（均已计算加成）  
载员：驾驶员1  
特性：悬浮车体、离子推进器

【重力悬浮】：秃鹰战车以其的高机动性和高通过性在早期得到了泰伦军方的采用，虽然随着技术更迭其缺乏防护的缺点越来越明显，但不乏有的势力依旧装备了些许。  
由于其采用的悬浮车体，秃鹰战车可以轻松的通过大部分地形，这视为一个免疫C级困难地形的能力。

【离子推进器】：秃鹰战车傲人的动力系统，配合其的悬浮车体能够轻松跑出远超恶火战车的时速和机动性。如果你够胆的话，把秃鹰当机车开也不是不行。  
当你驾驶秃鹫战车时，你获得2点闪避防御。

\===【下属武器】===  
破片榴弹发射器  
价格：D+500  
秃鹰战车配备的一对车载武器，平时收纳隐藏在车头中，需要时会展开盖板方便发射榴弹。

武器伤害：8L  
武器特性：【高速5】【破甲3】  
伤害类型：物理-穿刺  
射程：200  
装弹数：1

【破片榴弹】：秃鹰战车装配的榴弹威力十足，甚至足矣重创自身。使用本武器攻击时获得1点附加成功。

【备弹充足】：考虑到轮回者没有战地补给的机会，主神因此在车体内准备了充足的弹药，视为其拥有【无限子弹】特性。
