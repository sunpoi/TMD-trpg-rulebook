# 速腾全地形车

  

速腾全地形车  
D+1000分  
科技本质  
腾全地形车是沃坦联盟（短大人！）中令人敬畏的快速反应车辆，专为勘察不适宜居住的外星地貌而设计，可以携带一个完成的步兵班组。 他有三对负重轮，前面是装甲驾驶室，后面是装甲部队舱。双方都可密封并配有单独的NBC系统，用于保护乘员，车身装甲可以抵御小型武器的攻击。窗户为单晶石英制成，带有加强层，其防护效果与装甲板类似，因此视为一种全封闭的重型载具。  
  
体积：20  
结构值：22  
装甲：12  
速度：40米，加速度10米，最大速度120米。  
载员：驾驶员1，乘客6名，总体积不超过40  
动力系统：无限动力  
特性：NBC系统，复杂路面悬挂，重型载具。  
  
NBC系统  
速腾全地形车内装有空调和气体调节装置，视为拥有【三防】，旨在使乘员在运输过程中保持舒适。前驾驶室有两扇车门，车顶有两个逃生舱口，而部队舱的后面有一扇门，车顶有1个舱口。  
  
复杂路面悬挂  
速腾全地形车适合侦察行动与在配合炉裔先锋部队作战时进行装甲闪电进攻战，其三队硕大的全地形轮胎搭载具有提供良好的抓地力，可以在70°斜面爬坡，且车身本身的黑科技坚固悬挂，可以缓冲由不平路面传给车架或车身的冲击力，并衰减由此引起的震动，以保证速腾全地形车能平稳地行驶在各种恶劣地形，具有免疫C级【困难地形】效果。  
  
重型载具  
速腾全地形车为驾驶者和乘客提供全遮蔽效果，并额外为驾驶者和乘客提供2点遮蔽防御。
