# 灵马托雷特

  

灵马托雷特  
本兑换是出自《艾尔登法环》世界中的灵马托雷特，虽然名称是马，但是头顶生有宛如牛类一般的角，同时还能够进行二段的跳跃，协助褪色者跨越险峻的地形。  
本兑换不会附赠载具通用的空间胶囊，作为代替，会赠送一枚戒指，只要消耗一个迅捷动作，就可以将灵马托雷特召唤出来，供自己骑乘。

特性：  
【灵马】 ：托雷特并非普通的马匹，因而它的移动速度提升55米，并且全身视为具有1L的天生武器伤害。  
【二段跳】 ：托雷特在进行跳跃时，可以在空中借力一次，因此托雷特进行的跳跃检定额外具有2点附加成功。
