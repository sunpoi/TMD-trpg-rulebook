# 蟹堡车

名称：蟹堡车  
价格：D+500  
本质：特异  
描述：  
出自《海绵宝宝》，是蟹老板用于宣传蟹堡王制造的特殊车辆，整体外观类似于蟹黄堡，全车采用食品级工艺，根据后面被青蛙鱼怪所吃来看，该车应该能开也能吃。

车辆主体外壳点缀芝麻，形似汉堡皮，两片汉堡皮的中间夹着肉饼、奶酪片、番茄片、洋葱片、生菜以及调味料（根据加油站的两条鱼的说法，大概是番茄酱与芥末酱），轮胎为酸黄瓜，座椅为烤肉座椅，发动机为燃油喷射式油炸锅引擎，还有带双头的油脂分离器，在车辆后端存在一个启动时会不断旋转的小船桨以及一个写着“KK”的小红旗，前端则是两个类似于圆窗的小头灯以及一块挡风玻璃。

开车时需要直接跳上车去进行驾驶，启动时需要将锅铲插入钥匙孔中（油门是叉子、刹车是勺子），再推动中杯汽水（带塑料吸管）的手动挡便可发动，操作十分简单，甚至不需要驾照，使用燃料为食用油，在主神空间的改造下，可以无需添加。

类型：重型载具  
体积：8  
结构值：22  
装甲：12  
移动速度/轮：20米，  
载员：驾驶员1，乘客1名，载员体积不超过4  
动力系统：无限食用油  
特性：【深海航行者】【重型载具】

效果：  
【深海航行者】  
蟹堡车具有跟移动速度/轮相同速度的游泳速度，且为乘客提供水下呼吸能力，同时视为具有【三防】特性。

【重型载具】  
蟹堡车为重型载具。
