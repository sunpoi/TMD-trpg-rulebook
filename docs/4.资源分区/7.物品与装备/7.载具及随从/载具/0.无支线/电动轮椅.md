# 电动轮椅

电动轮椅  
价格：250  
一台经过稍微改装的电动轮椅，使用起来并不比散步的速度快多少，但是整体采用了相当优异的人体工学设计及透气耐用的材质，因而坐在上面十分舒适，即使久坐移动也不会疲劳和损伤皮肤。同时，相比普通的轮椅，它的转向以及减震性能也更加优异，在一般轮椅无法使用的崎岖山路上也能如履平地，因此即使腿脚没问题也不妨碍拿它来代步。此外，轮椅扶手上也提供了多规格的充电口，可供各种小型电子设备如笔记本电脑或手机等充电，并能为它们提供额外的固定支架，在轮椅侧后方也各有额外的储物空间为使用者存放零食之类的东西。轮椅整体色调为白色纯色，风格较为干净整洁，但同时也可定制幻彩涂装或增加其他配饰。无论是想要清冷疏离的文弱感，还是展现飒爽利落的都市风格，它都能轻松贴合使用者的气质。

本质：科技本质  
载具类型：轻型载具  
载具专业：无  
速度：20米  
体积：4  
结构值：5  
装甲：0  
动力：电量可供持续移动8小时，可充电，晴天时则可以使用太阳能板充电，此时视为无限动力。  
载员情况：可供1名体型不大于6的使用者使用。

轮椅本身的可使用遥控器，扶手按键，亦或是使用者意念指令等方式遥控移动，因本身速度与步行差不多所以无需额外使用驾驶检定进行操控。但相应的这也意味着该载具不会在战斗中提供任何的防护和助力，在战斗中从轮椅中起身为一个迅捷动作。

此轮椅也可以由其他人帮助推行，此时移动速度取决于后面人能推多快。当然，你也可以选择自己手摇轮椅。
