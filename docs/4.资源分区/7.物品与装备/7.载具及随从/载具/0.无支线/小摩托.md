# 小摩托

　小摩托  
　　价格：1000  
　　当武器使用前提：力量5  
　　一台带有太阳能板的行李箱改装的电动车。(所以为什么叫做小摩托？)因为本质还是行李箱，所以坐起来并不舒服，长时间坐着估计会很废人。

　　本质：科技本质  
　　载具类型：轻型载具  
　　速度：30米  
　　体积：5  
　　结构值：16  
　　装甲：9  
　　动力：电量可持续移动8小时。可充电，晴天时因太阳能板充电视为无限动力。

      载物情况：小摩托内部可装下总体积不大于3，总质量不超过50kg的物品。

　　载员情况：可供1名体型不大于5的使用者使用。

　　使用者力量大于5，战斗时可视为无支线的重型锤，武器伤害4L，重型武器，双手，格挡3，威猛3，伤害类型钝击。

　　因为是手持行李…小摩托进行攻击，所以采用近战武器检定，采用武技：锤专业，而不是一般的载具攻击检定。  
　　每乘坐小摩托一小时，使用者承受1点可豁免的疲乏点数，同乘者承受2点可豁免的疲乏点数。
