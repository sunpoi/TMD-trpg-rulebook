# 红斗篷

  

红斗篷  
价格：D+500  
分类：轻甲  
盔甲防御：5/5  
盔甲属性：无  
描述：一件有些破旧的亚麻斗篷，上面的血似乎是某种动物的……  
完美的伪装：你的过去是什么完全不重要了，当你披上这件斗篷时，你就是货真价实的小红帽。  
穿戴者获得D级反侦测能力，特异本质来源。不能精确定位单位会认为穿戴者只是一个可怜的，需要帮助的孩子。不能定位的单位会在穿戴者有需要时下意识的装作没看见。
