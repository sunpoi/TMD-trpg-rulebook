# D级护甲-轻甲

护甲-轻甲

H  
红斗篷  
价格：D+500  
本质：魔幻本质  
分类：轻甲  
防御：盔甲防御5/5  
特性：【完美的伪装】  
描述：  
一件有些破旧的亚麻斗篷，上面的血似乎是某种动物的……

【完美的伪装】：  
你的过去是什么完全不重要了，当你披上这件斗篷时，你就是货真价实的小红帽。

效果：  
【完美的伪装】：  
穿戴者获得D级反侦测能力，特异本质来源。不能精确定位单位会认为穿戴者只是一个可怜的，需要帮助的孩子。不能定位的单位会在穿戴者有需要时下意识的装作没看见。

L  
龙皮甲  
价格：D+500  
本质：特异本质  
分类：轻甲  
防御：盔甲防御5/5  
特性：【防能量武器】【超凡抗性】  
描述：  
这件盔甲由亚龙的皮为原材料，经过鞣制与工艺技术的处理后崭新出品，其质感厚重而坚韧，从外观上看来与大衣没什么区别。

【超凡抗性】：  
据说龙族的外皮要比黑曜石所打造城门更坚固五倍有余，更具有的天然具有着针对魔法的抵抗能力。

效果：  
【超凡抗性】：  
虽然这件盔甲的原材料只是亚龙的皮革，但穿戴者依旧视为具有1点全伤害吸收。

R  
热血高校服装（夹钢版）  
价格：D+500  
本质：科技本质  
分类：轻甲  
防御：盔甲防御5/5  
特性：【防弹】【便于斗殴】【属于不良少年的荣耀】  
描述：“这是学校最高的地方，是学校的顶点。今天就在这个天台上一决胜负！”  
“呵，那就开...”（突然袭击）“八嘎呀路！为什么这么硬，你这家伙，也在衣服里塞了钢板吗！”  
（反应过来挡下偷袭）“混蛋，你居然偷袭！等等，‘也’？八嘎，你果然也不是什么好东西！”  
如你所见，这原本只是一件校服，不过经过简单的加工，它便焕然一新，重获新生。

【便于斗殴】：  
为了打架斗殴而生的衣服。

【属于不良少年的荣耀】：  
穿着这件衣服，你就代表着自己的高校，不论校内登顶的手段有多么卑劣，现在你都是学校的老大，学校不良少年的荣耀，需要你来捍卫。

效果：  
【便于斗殴】：  
无论你是否格挡，增加2格挡防御。

【属于不良少年的荣耀】：  
仅在穿着此装备时，你的决心属性检定获得2DP士气加值。

S  
SSC风精底层防护服  
价格：D+500  
本质：科技本质  
分类：轻甲  
防御：盔甲防御2/2  
特性：【三防】【活体护甲2】【万用防护服】【变色】  
描述：  
由史密斯岛野财团(SSC)控股的生物群系栽培世界阿克里梅亚四号(Acrimea IV).上发现的风精是种有机生命形体，似乎可以在几乎任何环境中生存。使用生物工程理论定义的类育种方式，SSC开发出了风精防护服附在拥有者身上形成外装的不育活体风精集合物。风精会紧贴著穿戴者的身体并形成互利共生关系：风精以寄主代谢物维生，并保护寄主免受各种恶劣环境的影响。

这些半生物的紧身防护服可长期穿戴。风精是透明的半液态存在，且可以在不需要时收藏在容器中。它们会清洁寄主身体、协助其自然回复过程并消灭代谢废物。需要时，防护服能够变成不透明或改变其颜色和材质。风精可以覆盖寄主头部，并提供气密和辐射保护，过滤空气和液体，甚至能够让穿戴者在水下呼吸。  
效果：  
【万用防护服】：  
穿戴着视为具有免疫D级环境负面效果影响及无需呼吸的能力。一旦活体护甲所带来的临时生命被击破则立刻失去本特性带来的好处，直到重新恢复为止。

【变色】：  
本盔甲可自由改变透明度与颜色，这并不代表你的颜色也会更改，完全透明仍然等于裸体。

T  
天使的羽翼  
价格：D+1000  
本质：魔幻本质  
分类：轻甲  
防御：闪避防御6/6  
特性：【灵巧】【羽翼】  
描述：  
这是一对翅膀（或者多对翅膀），当你召唤这件盔甲时，天使的羽翼在身后展开，在你的身后，有着圣光的特效，在特殊的世界观，说不定有奇效……  
“快看，那是天使！哦，我的上帝，我终于等到天使降临在我身边了”  
“不要突然在里面夹带奇奇怪怪的东西啊喂”

【灵巧】：  
凭借超高的机动性，你可以灵活的躲避他人的攻击。

【羽翼】：  
天使的羽翼，乃是新的肢体，你需要经过锻炼，才可以使用它进行飞行。

效果：  
【灵巧】：  
这件装备提供的是闪避防御，而不是铠甲防御。

【羽翼】：  
获得等同基础速度的飞行能力，机动性为笨拙。  
现在开始，每经历一场影片，你的飞行机动性上升一级，直到良好。（笨拙-普通-良好）

X  
血肉造物  
价格：D+1000  
本质：特异本质  
分类：轻甲  
防御：无  
占据部位：占据上肢和下肢位置  
描述：  
这是一具由不明血肉拼合而成的盔甲，但同时。这团不定形状的血肉造物会随着穿戴者的意念而变化，从而施展它自身的能力。

效果：  
血肉造物不会提供盔甲防御，在血肉造物即将抵御一次伤害时，盔甲表面会浮现出无数只鲜血淋漓的手，一齐合力抵挡攻击。血肉造物会提供5点格挡防御。

若穿戴者在穿戴时试图释放一道法术成分为语言的黑暗派系法术或【神话典籍记载法术】，血肉造物上将会自动变化出一张嘴用以吟诵而满足语言条件，而无需穿戴者自身满足。

每场影片结束后，穿戴者还可以仅一次从主神处再花费250分购买血肉用以喂养这团造物，若如此做，则血肉造物会再获得1点基于格挡防御的活体护甲。

学院死库水  
价格：D+500  
本质：特异本质  
分类：轻甲  
防御：盔甲防御5/5  
特性：【校园泳衣】  
描述：死库水，即日语“スク水”（即“スクール水着”的简称，英文“school swimsuit”的音译）的谐音，传统款式为连体式紧身泳衣，通常以藏蓝色为主（少数为白色），胸前或下摆处缝有写有学生姓氏的布条，这是萌点之一。  
早期采用棉质材料，后逐渐改用透水性更好的尼龙材质。部分旧式设计在腹部设有“排水口”，用于排出泳衣内积存的水流，这一细节也是萌点。  
那么有没有一件……  
算了，有的，兄弟，有的。

【校园泳衣】：  
很明显，这是件泳装，而且它寄托了“泳装”这一概念本身的诸多情感。

效果：  
【校园泳衣】：  
穿着它理所当然地免疫B级潮湿状态，代价是穿着泳装到处跑会让你承受永久存在不可豁免的2点亢奋点数，很兴奋，不是吗？

特殊的，你可以在购买时根据喜好自由决定这件死库水的配色，男式还是女式，有没有裙边，是分体式还是连体式，有没有排水口或者双层内衬。
