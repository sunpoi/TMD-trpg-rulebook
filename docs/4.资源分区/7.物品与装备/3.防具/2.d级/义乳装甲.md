# 义乳装甲

义乳装甲  
D+500  
大奶子不是女性特有，纸扎投石车就不能有吗？为什么一说大奶子就物化女性，纸扎投石车就不配拥有姓名吗 恋人  
我觉得武装直升机也应该有大奈子 力量  
不可描述的形状的柔软装甲，可以加装于任何雄性生物载具或机械载具，个人推荐武装直升机或纸扎投石机。  
这视为武器伤害3L 提供1附加成功的肉搏类武装，但仅可用于撞击攻击  
使用该装甲作为撞击武器的撞击攻击中，柔软的装甲不造成任何伤害，而是等量的魅惑点数，意志豁免。  
柔软的装甲自然有防御的作用，给载具提供2偏斜防御
