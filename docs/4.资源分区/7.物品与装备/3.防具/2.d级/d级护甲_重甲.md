# D级护甲-重甲

护甲-重甲

B  
步人甲  
价格：D+500  
本质：科技本质  
分类：重甲  
防御：盔甲防御1/1  
减值：盔甲减值2  
特性：【全封闭3】【重型武器】【增加甲叶】  
描述：  
步人甲由1825枚甲叶组成，一般重量达58宋斤（1宋斤等于1.2市斤）以上，同时可通过增加甲叶数量来提高防护力，但是重量会进一步上升。为此到后来皇帝亲自赐命，规定步兵铠甲以58宋斤为限。同时期的欧洲步骑兵的铠甲类型还以锁子甲为主，防护性及重量均不如步人甲。

【重型武器】：  
沉重的盔甲带来不俗的防御同时，也让穿戴者的负重情况更加的恶劣，这有好有坏。

【增加甲叶】：  
步人甲使用了可以添加甲叶来提高防御的技术。

效果：  
【重型武器】：  
此时穿戴者在冲撞/摔绊/击飞的对抗中获得获得2DP加值，穿戴者倒地爬起所需的动作上升一级。

【增加甲叶】：  
步人甲的拥有者可以支付500分购买新的增加甲叶，新的甲叶能够让步人甲的护甲防御提升2/2，并获得1点盔甲减值；可重复购买至多3次。

Q  
骑士重甲  
价格：D+500  
本质：魔法本质  
分类：重甲  
防御：盔甲防御7/7  
减值：盔甲减值3  
特性：【防弹】  
描述：以纯钢打造的古典骑士的盔甲，所具有的多层结构能抵抗大多数攻击，其白色的涂漆是否象征着骑士曾经的信念呢。

【英勇冲锋】：  
骑士的冲锋一往无前，曾经如此，今日亦然。

效果：  
【英勇冲锋】：  
穿戴本盔甲的单位在冲锋时会额外获得2点加值，所提供的加值不计入基础防御减值内。  
此外，在对抗恐惧异常点数所进行的豁免时，还会获得4DP士气加值。

X  
玄金甲  
价格：D+1000  
本质：魔幻本质  
分类：重甲  
防御：盔甲防御7/7  
减值：盔甲减值2  
特性：【玄金秘法】  
描述：  
一架通体金色，具有雕刻纹路的铠甲，最中心有一面护心镜，内里寄宿着某种远古神秘的能量。  
效果：  
【玄金秘法】：  
使用玄金秘法是一个移动动作，这要求使用者先保留至少6AP至回合外，使用玄金秘法是一个类招架行为，可以正常获得招架收益，并会正常消耗反射动作与移动动作，需应对一次攻击才能使用。  
铠甲上的玄秘纹路将全部亮起，并根据此次敌人攻击破防中最多的一种生效效果。  
高速/破甲：针对这次攻击，若穿戴者的护甲未被完全破穿，且仍具有一半及以上的基础/闪避防御，则获得基于盔甲防御的【坚韧】。  
威猛/破魔：针对这次攻击，若穿戴者仍具有三分之一或以上的完好，则获得持续一轮的【免疫多次攻击减值】。  
在使用玄金秘法时，也可以选择再额外投入并消耗数点没有使用的AP，每额外消耗1点AP，本轮内就获得1点基于玄金甲的护甲防御。
