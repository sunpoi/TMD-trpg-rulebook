# 蘑菇盾牌

  

蘑菇盾牌  
D+500  
原典：盾之勇者成名录  
吸收森林中最常见蘑菇解放的形态，带来了植物鉴定的知识。  
类型：轻盾  
武器伤害：1L  
体积：2  
专业：盾  
特性：【格挡8】  
【植物鉴定】：现在，你能够以一个迅捷动作选择视觉范围内的一个植物，通过【智力+生存+求生】检定以对抗植物的支线等级\*6，若目标能进行伪装检定，则可将伪装检定结果加入DC。当你对抗检定通过后，可以解析该植物的大致特征、作用或是某一防御类型。这是一个D级别、信息来源的解析效果。
