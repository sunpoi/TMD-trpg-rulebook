# 龙皮甲

  

龙皮甲  
价格：D+500  
分类：轻甲  
特异本质  
描述：这件盔甲由亚龙的皮为原材料，经过鞣制与工艺技术的处理后崭新出品，其质感厚重而坚韧，从外观上看来与大衣没什么区别。  
盔甲基本属性：盔甲防御5/5  
盔甲特殊属性：【防能量武器】  
超凡抗性：据说龙族的外皮要比黑曜石所打造城门更坚固五倍有余，更具有的天然具有着针对魔法的抵抗能力。  
虽然这件盔甲的原材料只是亚龙的皮革，但穿戴者依旧视为具有1点全伤害吸收。
