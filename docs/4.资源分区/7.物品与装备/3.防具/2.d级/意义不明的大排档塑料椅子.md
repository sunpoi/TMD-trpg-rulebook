# 意义不明的大排档塑料椅子

意义不明的大排档塑料椅子  
D+500  
这是一把武器伤害5L，格挡3，体积3的异形轻盾武器

【意义不明】∶这把椅子是如此的显眼，哪怕没有人坐在上面它也会成为所有人的视线焦点，坐在这把椅子上将会获得3点风度属性检定上的增强加值，但你将不可避免的吸引所有人的注意，同时其他单位会向你说出“xxx，你动不动打开传送门的日子结束了，把阎魔刀给我”这句话，而你必须站起身说出“如果你想要，就得自己过来拿，这规矩你早就懂得”作为回应
