# 乌云踏雪

乌云踏雪  
魔幻本质  
价格：B+3000

分类：重甲

盔甲防御：24/24

盔甲减值：3

描述：漆黑的玄铁为原料，千锻而成的数千枚甲叶组成的全身重甲，为守卫边疆而制作。  
千北寒，天地寂，乌云踏雪行。

魂于冰雪：乌云踏雪会为使用者提供基于盔甲防御的【超级坚韧】，以及【免疫接触攻击】的效果，这是某种意义上盔甲上的守卫之魂与你融合的结果。

于边境的风暴中：乌云踏雪甲叶构成的防护极其全面，但也取决于使用者的意志，乌云踏雪具有【封闭1】，在这基础上，使用者每具有1点传奇决心，乌云踏雪便额外提升1层【封闭】。

雪中行：穿戴乌云踏雪且知道当前地区的气象情况时，穿戴者可以以一个自由动作激发守卫之魂，将场地的天气引导向更寒冷的方向。但这一能力有所限制，比如说只能将阴天引导为细雨，或将细雨引导为细雪等，也就是前后环境效果必须接近且改变较为细微，这个效果每个场景只能使用一次。
