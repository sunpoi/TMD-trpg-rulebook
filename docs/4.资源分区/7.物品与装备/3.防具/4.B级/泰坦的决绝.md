# 泰坦的决绝

泰坦的坚决  
B+2000  
这套护甲由泰坦神族的遗骸锻造而成，表面覆盖着厚重的符文钢板，每一块甲片都刻有古老的泰坦文字，闪烁着微弱的蓝光。穿戴者仿佛化身为泰坦巨人，步履间地面震颤，铠甲散发出令人窒息的威压。  
盔甲防御 :10/10  
盔甲减值：2  
盔甲特性：【盔甲减值2】 【全伤害吸收3】  
【百战泰坦】

【百战泰坦】你每受到1次攻击或者攻击1次则获得1点盔甲防御和1点偏斜防御，这些防御可以叠加，最高叠加至5层，抵达5层之后，你获得额外6点体积，这不会给你带来生命值，同时再次获得8点盔甲防御和偏斜防御。以上效果都需要进入一场战斗才能触发，并且持续也仅仅只有一个场景。
