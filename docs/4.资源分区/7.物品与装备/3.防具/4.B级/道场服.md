# 道场服

道场服  
价格：B+4000  
分类：轻甲  
盔甲防御：18/18  
盔甲特性：【看破】  
描述：一丝不乱之心贯穿要害、饱经锻炼的肉体连酸也能弹开的古代武者的衣服。  
效果：  
【武道之心】：穿戴者免疫多次攻击减值；在处理道场服带来的盔甲防御时，将这些盔甲防御转为洞察防御处理，而非盔甲防御。

【历练之躯】：穿戴者获得等同自身【耐力属性/3】点全能量抗力。
