# 心之钢

  

心之钢  
B+2000  
心之钢是一件盔甲减值为6的重甲，它不会提供任何护甲防御或天生防御。心之钢会提供自然恢复伤势所需时间减半的效果，并再额外提供10点生命值（特殊的，机械载具或构装体/具有结构值的生物无法购买此装备）。  
庞然吞食：当你的接触范围内至接触范围外50米范围内有一名敌人，且该敌人与你已在战斗场景中经过了三轮时，心之钢提供的强力攻击将充能完毕，你对其下一次攻击将视为触发这个攻击效果。此次攻击中，你每有2点【额外生命】就额外获得1DP攻击上的器械加值。并再永久获得等同于你的耐力附加\*1点生命值。心之钢的增加生命值效果一影片只能触发一次。  
神话被动：你每有一个B级奇物/装备强化，就再获得2点体积和2点生命值（包括心之钢在内）。
