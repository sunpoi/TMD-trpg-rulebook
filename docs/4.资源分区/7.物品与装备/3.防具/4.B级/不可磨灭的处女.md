# 不可磨灭的处女

不可磨灭的处女  
B+2000  
这套护甲由纯白的圣银打造，表面刻满神圣符文，每一道符文都流淌着柔和的金光。穿戴者仿佛化身为不可侵犯的圣女，铠甲散发出纯净的光辉，能够驱散一切黑暗与污秽。护甲的核心镶嵌着一颗天使之泪，能够在穿戴者濒危时释放出治愈能量，修复其伤势。  
盔甲类型：重甲  
盔甲防御 :17/17  
盔甲特性:【光明之女】【光烛唱诗班】

【光明之女】这件护甲的盔甲防御转变为能量防御，你穿戴护甲时，你神圣无比，宛如世间的光，你被视为光明生物，你免疫接触攻击，并且任何人攻击你都会因为向光明挥刀而具有【劣化1】。圣吟永唱，歌颂世间之美，只要你不愿意，没有任何生物能够沾污你，你的裙下在任何时候都会视为无尽的光，除非你愿意否则没有人会破开光源窥窃裙下之秘，这视为一个B级命运来源效果。如果有人堪破这种效果，会被你第一时间知晓，如果你无法反抗，可以启用光源，彻底将你躯体烧成灰，这视为一个S级即死效果。

【光烛唱诗班】你以一个标准动作召唤出光烛唱诗班，其实只是一群人形态光源，它们集合起来会散发一定的光，这些光会驱散以你为中心五米内的黑暗，使黑暗环境光源提升1级。她们会吟唱神圣的赞美诗～诗歌会让以你为中心，半径50米内所有友军恢复6点严重伤害（医疗点72），你可以每回合以一个标准动作维持，如果没有维持，唱诗班会逐渐淡化，直到消失。
