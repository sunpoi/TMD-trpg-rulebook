# 流星斗篷

流星斗篷  
魔幻本质  
价格：C+1000  
分类：轻甲（占据斗篷位而非躯干位，但不意味能额外穿戴护甲）  
描述：将一场流星雨的景象呈现在特殊织物上的成果，给与穿戴者来自星辰的祝福  
流星斗篷是一件丝质的深蓝色斗篷，上面点缀着带有弧形荧光的金丝线。在穿戴状态下看到斗篷呈现出的就是类似于具有流星雨的夜空景象。  
盔甲防御：10/10  
盔甲特殊属性：【防能量】【三防】  
【无日之旅者】：星星是在没有太阳的时候才出来的旅行者，你在符合【星辰之旅】条件的环境下可以借由星辰的力量进行移动，这是一种机动性为良好的飞行能力。  
【星辰之旅】：  
1.穿戴者在所在区域不是白天  
2.不是明亮光照环境
