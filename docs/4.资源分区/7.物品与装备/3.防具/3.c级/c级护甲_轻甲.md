# C级护甲-轻甲

护甲-轻甲

H  
唤潮法袍  
价格：C+1000  
本质：魔幻本质  
分类：轻甲  
防御：偏斜防御10/10  
特性：【海潮护盾】【唤潮】  
描述：  
这是一件通体蔚蓝色的斗篷，凝视它时，海的波纹在其上荡漾，行走时，会留下一阵海风的咸腥。  
由导魔特性良好的蛛丝纺织，其上的色彩是以特殊培育的水魔力富集的蓝色藻类提取而出，以海妖们特有的附魔手段为其染色，并留下了【海潮护盾】与【唤潮】两种特殊附魔。

【海潮护盾】：  
穿戴此法袍时，富集的水魔力将环绕在你的周身，使得你犹如置身大海一般，一切的攻击也都将被海潮阻拦。

【唤潮】：  
穿戴此法袍时，富集的水魔力将在你进行攻击时为你提供海潮的助力，汹涌的海浪将吞噬你的敌人。

效果：  
【海潮护盾】：  
海水中富含的氧气将为你供应所需的生存条件，并且当你身处水流中时，你将可以自由活动，这是海赐予你的权利。  
法袍将为你提供偏斜防御，并且给与你全环境呼吸效果和在水面上可以以等同于基础移动速度进行移动的能力。

【唤潮】：  
当你进行施法攻击时，你的攻击将附带【潮湿】，这是一个c级的环境来源潮湿效果，目标以反射豁免，失败则陷入该潮湿状态一场景  
并且，在你进行施法攻击时，层叠的海浪将连绵不绝的冲击对方，你攻击时将额外造成传奇智力\*1的多次攻击减值，并且视为拥有【余波】效果。

K  
空间程式魔导分析战衣  
价格：C+1000  
本质：科技本质  
分类：轻甲  
防御：力场防御10/10  
特性：【空间折跃程式】【空间信息分析】【机械链接】  
描述：  
使用特殊合金制造的银白色流线型贴身战衣，表面有着复杂的蓝色荧光纹路，这是其魔导程式的运行回路，背部的圆盘是空间程式的核心处理器。战衣本身极其柔韧，没有保护能力，而是依靠空间程式进行保护。

【空间折跃程式】：  
穿着者进行移动时可以通过折跃程式进行折跃。

【空间信息分析】：  
通过空间程式对物体的距离、速度、方向进行精密分析，再通过战衣进行警示。

效果：  
特殊的，这套战衣不提供盔甲防御，而是借由空间程式提供力场防御  
【空间折跃程式】：  
穿着者的移动具有C级传送效果

【空间信息分析】：  
穿着者可以忽略智力上的附加成功\*3的【高速】。

【机械链接】：  
若穿着者在驾驶机械载具，则可以以一个自由动作将该盔甲提供的除力场防御外的其他效果转移给载具生效，但此时着装者本身则失去空间程式魔导分析战衣提供的除力场防御外的所有好处。

L  
流星斗篷  
价格：C+1000  
本质：魔幻本质  
分类：轻甲  
防御：盔甲防御10/10  
占据部位：占据斗篷位而非躯干位，但不意味能额外穿戴护甲  
特性：【防能量】【三防】【无日之旅者】  
描述：将一场流星雨的景象呈现在特殊织物上的成果，给与穿戴者来自星辰的祝福  
流星斗篷是一件丝质的深蓝色斗篷，上面点缀着带有弧形荧光的金丝线。在穿戴状态下看到斗篷呈现出的就是类似于具有流星雨的夜空景象。

【无日之旅者】：  
星星是在没有太阳的时候才出来的旅行者。

效果：  
【无日之旅者】：  
你在符合【星辰之旅】条件的环境下可以借由星辰的力量进行移动，这是一种机动性为良好的飞行能力。

【星辰之旅】：  
1.穿戴者在所在区域不是白天  
2.不是明亮光照环境

S  
生物齿轮  
价格：C+1000  
本质：自然本质  
分类：轻甲  
防御：天生防御15/15  
特性：【暗黑真实】  
描述：  
古代遗物之一的人类体内的齿轮，也送你一个吧。

【暗黑真实】：  
隐藏在历史之下的暗黑真相，古代的人类竟是这样的存在吗？！

【暗黑真实】：  
植入生物齿轮，你获得天生防御，仅首次穿戴这一盔甲的单位，生命值将被转换为等量构造值。穿戴者免疫【接触攻击】。

生物齿轮如常占据你的盔甲位，如同穿戴了轻甲一样。  
差价升级至B+3000：  
效果：  
将天生防御提升至22点，同时转化的构造值变为1.5倍，额外获得基于天生防御的【坚韧】效果。

Z  
葬仪黑衣  
价格：C+1000  
本质：特异本质  
分类：轻甲  
防御：盔甲防御10/10  
特性：【防弹】【防能量武器】【肃穆】【退魔】【光明】  
描述：  
葬仪之人——持有不存在于组织中的第七秘法，以令死者重归尘土，驱逐妖魔诡异为己任。他们在执行葬仪行为时所穿戴的法衣即葬仪黑衣。那是所作为尘世之人消灭不应存在于世者，跨越生死的证明。  
另外，还有着即使牺牲自己，抛弃一切也要拔除妖魔的觉悟这层含义。

【肃穆】：  
于生者的世界不应被死者指染，一切本应纯净化，这是法衣所象征的意义，也是设计者衣品的体现。

【退魔】：  
当为黑暗之敌，致力于拔除黑暗，扫清罪恶。

效果：  
【肃穆】：  
穿戴者在风度检定上获得1点神圣加值。

【退魔】：  
穿戴者攻击视为具有【光明】特性，且在攻击中获得1个额外的附加成功。

【光明】：  
仅限针对攻击具有黑暗相关能力或分类为黑暗生物的单位时，追加1点神圣伤害。
