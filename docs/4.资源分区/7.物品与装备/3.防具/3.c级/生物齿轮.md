# 生物齿轮

生物齿轮  
本质：自然本质  
价格：C+1000  
分类：轻甲  
描述：古代遗物之一的人类体内的齿轮，也送你一个吧。  
暗黑真实：隐藏在历史之下的暗黑真相，古代的人类竟是这样的存在吗？！  
植入生物齿轮，你获得了15点天生防御，仅首次穿戴这一盔甲的单位，生命值将被转换为等量构造值。穿戴者免疫【接触攻击】。  
生物齿轮如常占据你的盔甲位，如同穿戴了轻甲一样。  
差价升级至B+3000：将天生防御提升至22点，同时转化的构造值变为1.5倍，额外获得基于天生防御的【坚韧】效果。
