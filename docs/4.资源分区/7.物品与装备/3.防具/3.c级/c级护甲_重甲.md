# C级护甲-重甲

护甲-重甲

H  
红缎带之服  
价格：C+1000  
本质：魔幻本质  
分类：重甲  
防御：盔甲防御10/10  
减值：盔甲减值2  
特性：【抵破甲2】【防能量】【坚韧】【防护火焰】  
描述：龙讨伐队在铠甲下穿着的衣服。要害处装入了金属、比看起来更重而且结实。  
效果：  
【防护火焰】：  
穿戴者获得等同【传奇耐力】\*1点DR/法破与【火焰能量抗力】。

J  
巨人之铠  
价格：C+1000  
本质：特异本质  
分类：重甲  
防御：盔甲防御14/14  
减值：盔甲减值2  
特性：【重量级】【巨人级】  
描述：  
通过祈祷自己能和巨人一样强大的衣服，理所当然的只有在面对巨人的场合才能生效。

【重量级】：  
巨人之铠的重量实在太过沉重。

效果：  
【重量级】：  
穿戴者每轮会失去一个迅捷动作。

【巨人级】：  
穿戴者在绊摔/冲撞/擒抱无视体积带来的优势与劣势，同时忽略他人因体积带来的天生防御，这个效果会被C级或以上的免疫接触攻击效果无效。

L  
炼狱碎片甲  
价格：C+1000  
本质：魔幻本质  
分类：重甲  
防御：盔甲防御12/12  
减值：盔甲减值2  
特性：【焦油哭号】【炼狱】  
描述：  
这套重甲是从深渊熔炉中直接拓印而出的噩梦造物，由数千片锯齿状暗红金属残片拼嵌而成。每块碎甲表面都浮动着类似熔岩静脉的纹路，在暗处会渗出硫磺色的荧光。

【焦油哭号】：  
炼狱金属碎片中的灵魂会被不断折磨，甲胄表面不断渗出千万个灼烧中的灵魂产生的黑色粘液，该种状态下穿戴者的近战攻击和肉搏攻击都将带有这些灵魂被折磨的疲累与疯狂的低语与呼号。

效果：  
【焦油哭号】：  
穿戴者的近战攻击和肉搏攻击附带等量疲乏与亢奋的心灵来源的异常点，可以被正常豁免。

【炼狱】：  
重甲在非战斗状态下可以自由通过灵魂产生的粘液变形为各种服装形态，甚至包括为主人产生一副人脸，进入战斗自动变回原形，这是一个C级的变形来源的伪装效果。

落樱武士甲  
价格：C+1000  
本质：魔幻本质  
分类：重甲  
防御：盔甲防御12/12  
减值：无  
特性：【灵性材料】【自然赠礼】  
描述：  
这件盔甲的造型很接近古典武士铠，但不同于通常的厚重护甲，而是采用可塑性的灵性材料所制作，以保留的一层片甲衣来维持防护，即使是百夫长所使用最华丽最沉重的型号，也比通常的盔甲轻盈。  
其外观通体呈樱粉色，就如同盛开的樱花一般，若是仔细看去，你发现这件甲胄可能真的由植物所织成的材料。

【灵性材料】：  
本盔甲采用特殊材质所制作，会随穿着者的动作和思考而做出反应，在移动和作战时，会不断重塑和变化自身形态，无论如何也不会对行动产生阻碍。

效果：  
【灵性材料】：  
本护甲不存在盔甲减值。

【自然赠礼】：  
穿戴者常时具有自选是否生效的D级反侦测能力，这视为灵性材料随环境而产生变化带来的自然本质反侦测效果。  
在相关剧情判定中，穿戴者将会像樱花瓣一样轻盈，这意味着其免疫自然坠落伤害。

选开：拥有本盔甲的单位购买冒险者大礼包时，将其中的食品改为“小精灵特色风味素饼”，其风味绝佳，不会因为时间流逝而失去品质，总计五块，每块足以维持一人一整天的食水需求。
