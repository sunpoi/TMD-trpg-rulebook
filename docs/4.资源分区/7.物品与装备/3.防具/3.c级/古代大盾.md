# 古代大盾

古代大盾  
价格：C+1000  
分类：重盾  
武器伤害：3L  
武器特性：【双手】【防弹】【格挡15】  
武器体积：4  
伤害类型：物理-钝击  
描述：很好的将人遮住的巨大的盾。由极为柔韧的木材制作、对攻击展现出极高耐性。  
效果：  
以移动动作进行一次格挡并获得基于遮蔽防御的【2dr/-】，在格挡时可以选择对触及范围内的所有友方单位提供【全遮蔽】效果，持续至你的下回合开始；持有该盾牌进行的格挡获得的防御将改为提供掩蔽防御。
