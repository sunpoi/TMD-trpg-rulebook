# 攻城盾

攻城盾  
价格：C+2000  
分类：重盾  
武器伤害：3L  
武器特性：【双手】【防弹】【格挡15】  
武器体积：4  
伤害类型：物理-钝击  
描述：极为巨大而粗犷的盾牌。举着无需一切技术的这把盾、带头冲锋的巨汉普通士兵、据说最终被怪物吃了。  
效果：持有该盾牌的单位在格挡时获得【免疫接触攻击】与基于盔甲防御的【坚韧】特性。
