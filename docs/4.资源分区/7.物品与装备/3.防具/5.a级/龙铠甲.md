# 龙铠甲

龙铠甲  
价格：A+10000  
分类：重甲  
盔甲防御：28/28，盔甲减值3  
盔甲特性：【防弹】【防能量】  
描述：有着双头龙之鳞覆盖表面，传说中英雄齐格的铠甲。据说双头龙能操纵火焰和冷气的魔法，但这件铠甲并没有那么方便的性能。  
效果：  
【龙炎吐息】：该特性仅限穿戴者你具有攻击附带或转化燃烧不良点数时才能生效，可以指定相应的一项（见上述）生效，使那一能力造成的燃烧不良点数提升至四档，但会使那一能力在使用时额外消费一个迅捷动作。

【厚重龙鳞】：该盔甲提供的防御为天生防御，而非盔甲防御；龙鳞会在受到近战攻击时锐化，也会在受到远程攻击时吸收动能/能量进行折射，这意味着穿戴者在受到攻击时，会对攻击者造成【耐力属性】\*1点物理-穿刺/纯能量伤害（对应近战/远程），这些伤害不会被防御抵消，但会正常被伤害吸收与减免类能力减少；每轮一次，当受到攻击时，可以令那次攻击重骰，以后者取代先前的结果。

【龙之诅咒】：“龙铠甲正是屠戮恶龙而铸就的盔甲，恶龙临死前的诅咒始终缠绕于佩戴者之身，夺取穿戴者的力量，并留下最致命的弱点。”  
当穿戴者进入战斗时，会在战斗场景的第一个回合中因恶龙诅咒的束缚而失去一个移动动作，如果穿戴者不是战斗中第一个行动的单位则会改为失去一个标准动作；若穿戴者在受到措手不及/先制效果影响时没能将其免疫，则会因无法及时阻挡最致命的弱点而失去【厚重龙鳞】的效果；任何敌人看到穿戴者的背后皆会意识到弱点所在，攻击穿戴者背后的弱点是一个部位减值0的特殊部位攻击，但只有在攻击具有措手不及/先制攻击时，且没有被你免疫时，才能正常攻击命中这一弱点，从而如同上述效果一样无视【厚重龙鳞】的效果。
