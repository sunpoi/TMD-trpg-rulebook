# A级护甲-轻甲

护甲-轻甲

C  
传说之燕  
价格：A+4000  
本质：特异本质  
分类：轻甲  
防御：盔甲防御26/26  
特性：【次元跳跃】【传说の燕】【燕子】  
描述：  
比恶魔要强很多，可以自由的穿越时间和空间，会喷火，会发射激光，可以超高速的移动，长有鳞片。以这样的燕子为原型而设计的燕尾服，基本可以让穿戴者躲避一切非空间折叠产生的攻击。

效果：  
【次元跳跃】：  
每轮一次，穿戴者可以随时使用一个不消耗AP的反射动作遁入次元夹缝之中，次元夹缝视为一个B级时空来源的异空间效果。  
你可以在自身回合内使用一个迅捷动作离开次元夹缝。  
特殊的是，如果因此离开了敌人的攻击范围，只要对方使用的不是范围攻击，那么你就可以免疫那次攻击，这是一个A级时空来源的无敌效果。

【传说の燕】：  
每经历一场影片，你的声望增加10点，这视为负面声望而独立计算，你若具有其他负面声望则合计计算。  
无论如何，这个声望只有在你主动表明身份时才会生效，不过即使是平时也可能有人把你误认成传说中的那只燕子。

【燕子】：  
你的移动实际上不是正常移动，而是连续衔接的空间移动效果，这视为C级传送移动能力，意味着你可以向空中移动（依旧会因为没有站立点而坠落）。

X  
线稿盔甲  
价格：A+4000  
本质：特异本质  
分类：轻甲  
防御：见下文  
特性：【额外具有基于护甲防御的2点附加成功】  
描述：  
这是一套常人难以控制的强大盔甲，想要具有具体角色形象的人更是难以掌控其线稿之力，  
而线稿这一抽象概念本身就对于故事内经历叙事的角色而言有着跨纬度级别的理解难度。

这一本质导致其展现方式也极其抽象，任何直面一个活生生的人身体部分转化为狂乱飞舞的粗糙线条时，都会在短时间内因认知过载而难以理解现状，陷入短暂的混乱与僵直。这也是这件盔甲的初衷——以存在形式的根本性异常，来瓦解一切基于“现实”与“完成品”的攻击与逻辑。

效果：  
若穿戴者的心智类传奇属性总和小于10，那么穿戴者会在穿戴上护甲的同时，受到S级命运来源的即死效果，表现上为你四散为一团扭曲，无法辨认的细线，随后消失。

因为这件盔甲的反直觉运作形态，导致其他人难以确定穿戴者的本体所在，穿戴者视为拥有A级特异本质的弱反侦测效果，最低只能以此降低至模糊定位。且这个效果无法自主关闭。

除此之外，这些旋转扭动的线稿之间也会存在极大的、未被填充的空隙。这些空隙并非物理意义上的“孔洞”，而是“未被定义”的领域。  
任何穿过这些空隙的攻击，无论其能量、实体还是概念性打击，都会因“命中目标未被定义”而判定失效，无法造成任何伤害。  
但盔甲的核心逻辑会确保穿戴者的“生命线”与“行动线”始终连贯，从而在免疫伤害的同时，完美保留穿戴者的生命与行动能力。  
穿戴者的形态将会转变为大量的扭动的线稿，生物类型转变为【微型生物】。

作为【未完成】的作品，观测者，也就是其的穿戴者的态度和划分将会直接影响该护甲的效果。购买时决定，随后不可更改

若穿戴者认为该护甲【已经完善】，则线稿的画风将会定性。  
为了更好的保护成果，护甲会加固线稿以保证自身的稳定。线稿盔甲的护甲防御增加10点。  
而更坚韧的线稿能够更充分地吸收所受冲击的能量，为穿戴者提供免疫冲击点数减值的效果，且每次受到的伤害中至多令10点伤害降低一级。

若穿戴者认为护甲【需被完善】，那么这幅线稿就是你挥洒想法的舞台。  
为了更佳的描绘蓝图，你必须更加深切的了解这幅盔甲，哪怕这会令你的头脑难以承担。  
现在你可以自主关闭本护甲给予的弱反侦测能力，并能够将线稿描绘成你心所想的模样，转念之间添上了为其他人而看的色彩，乃至模仿的惟妙惟肖，这是一个A级的伪装能力。  
并且由于你与护甲的联系加深...其反直觉的运作状态延伸到你的身体各处，也令你本人的原貌难以被查清，你就是这幅线稿！为穿戴者提供免疫A级解析的效果。并且你可以用线稿缝补自己缺失的部分，再提供类似于四肢立即再生的效果。

Y  
夜蛾之舞  
本质：魔幻本质  
价格：A+4000  
类别：轻甲  
部位：上身  
盔甲防御：26/26+2附加成功  
防御类型：护甲  
描述：  
这是一件灰白相间的三层洛丽塔小裙子，轻盈的白色丝绸，除了作为裙子最内侧填充私密遮盖的蓬松软糕裙以外，从双肩向下沿着双峰之间的浅灰色抹胸顺着腋下绕到后腰形成了一个轻盈的小蝴蝶结。深灰色的布料同时承载了裙撑最外延的上裙表面与收腰的支撑作用，在后腰的下摆流出了两条长而拖地的绫缎拖尾。浅灰色的布料起到了肩撑以及裙子中层形成裙浪般效果的重要组成部分，这件裙子的内衬及抹胸也用到了看似柔软贴身、舒适透气却结实坚硬的灰色类羊毛织物布料，从内衬与腰封之间的缝隙处延伸出两大股灰色布料在后腰的蝴蝶结与拖尾之上形成了夸张的蛾翼形结。  
  
【噩梦的新娘】：  
过去的过去，曾经的曾经，公主幸福快乐的生活着，正如一切故事的开头那般，噩梦降临了。 她与狂暴无序的噩梦成婚，噩梦许她命中注定有一命所存。  
  
【欢跃之舞】：  
特性描述：她翩翩而舞，似飞蛾，决绝，热烈，一如既往地扑向自毁的烛火。  
  
特性：【噩梦的新娘】【欢悦之舞】  
效果：  
【噩梦的新娘】：  
穿着这件护甲会为穿戴者提供三豁免检定上的8dp器械加值，并在三豁免检定上各获得1点附加成功。  
购买该护甲后，在一场影片内亲手击杀一头比自身要高一级支线或更多的【黑暗生物】，影片结束后，这件护甲的裙边上将会铭刻纷繁的黑色纹饰，此时，每场影片一次，当穿着者死后，立即以满状态原地复活，那些承载了命定之死的黑色线条将会从裙摆之上化作纷飞的黑色飞蛾，四散而去，在影片结束后四散的飞蛾会重新回到裙摆上化为纹饰。  
  
【欢跃之舞】：  
进入战斗后的首个回合，你可以消耗一个迅捷动作并支付1点任意能量进入【欢跃动之舞】状态，此时，这件防具的盔甲防御提升4点，并在盔甲防御上获得额外4点附加成功，同时免疫A级影响心理来源负面效果，只要自身能量池不为空，就可以获得4点全伤害吸收，开启该特性时，获得4点临时生命，这些临时生命在破坏后不会完全消失，而是在消耗迅捷动作维持【欢跃之舞】状态时立即刷新。  
特性启动后，每个回合都需要支付一个迅捷动作用与1点与开启特性时使用的相同能量以维持该装备的【欢跃之舞】特性，若因任何原因无法维持该特性生效，失去该特性带来的效果，本场战斗内无法再次获得此特性带来的益处。
