# 传说之燕

传说之燕  
本质：特异本质  
价格：A+4000  
分类：轻甲  
盔甲防御：26/26  
盔甲特殊属性：无  
描述：比恶魔要强很多，可以自由的穿越时间和空间，会喷火，会发射激光，可以超高速的移动，长有鳞片。以这样的燕子为原型而设计的燕尾服，基本可以让穿戴者躲避一切非空间折叠产生的攻击。

  
次元跳跃：每轮一次，穿戴者可以随时使用一个不消耗AP的反射动作遁入次元夹缝之中，次元夹缝视为一个B级时空来源的异空间效果。  
你可以在自身回合内使用一个迅捷动作离开次元夹缝。  
特殊的是，如果因此离开了敌人的攻击范围，只要对方使用的不是范围攻击，那么你就可以免疫那次攻击，这是一个A级时空来源的无敌效果。

  
传说の燕：每经历一场影片，你的声望增加10点，这视为负面声望而独立计算，你若具有其他负面声望则合计计算。  
无论如何，这个声望只有在你主动表明身份时才会生效，不过即使是平时也可能有人把你误认成传说中的那只燕子。

  
燕子：你的移动实际上不是正常移动，而是连续衔接的空间移动效果，这视为C级传送移动能力，意味着你可以向空中移动（依旧会因为没有站立点而坠落）。
