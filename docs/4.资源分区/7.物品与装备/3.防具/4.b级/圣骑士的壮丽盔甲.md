# 圣骑士的壮丽盔甲

圣骑士的壮丽盔甲  
本质：魔幻本质  
价格：B+2000  
分类：重甲  
盔甲防御：24/24  
描述：受到神圣之力呵护的纯白重甲，不仅仅是保护好自己，也要保护好同伴。  
“你的帮助从何而来？你的帮助从造天地的主而来。他必不叫你的脚摇动，白日太阳必不伤你，夜间月亮必不害你。主要保护你，免受一切的灾害，他要保护你的性命。从今时直到永远。”

  
神圣加护：被神圣之力所呵护的盔甲是如此轻盈，因此盔甲失去重甲减值。  
由于神圣之力的作用，穿戴者会获得基于盔甲防御的【适应】与【坚韧】效果。

  
守护意志：当发生任意一次攻击时，穿戴者可以在回合外以反射动作选择触及范围内的单位，给予其【全遮蔽】效果。
