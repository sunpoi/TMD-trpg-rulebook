# 血皮革

血皮革  
价格：B+4000  
分类：轻甲  
盔甲防御：17/17  
盔甲特性：-  
描述：杀死无数魔物的冒险者身穿的铠甲，侵染的血会产生灵魂，回复体力。  
效果：  
【生命循环】：穿戴者获得等同自身【耐力属性】三分之一的快速医疗，每1点快速医疗对应12点医疗池。

【血染魂魄】：穿戴者可以在自身的回合中消费一个移动动作压榨铠甲中蕴藏的魂魄，这会使穿戴者立刻进行一次【竭取】，并在这一回合内获得免疫不超过A级重伤的能力。
