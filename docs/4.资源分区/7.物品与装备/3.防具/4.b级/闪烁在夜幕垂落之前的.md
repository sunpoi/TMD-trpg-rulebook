# 闪烁在夜幕垂落之前的

闪烁在夜幕垂落之前的  
B+2000  
魔幻本质  
分类：防具  
类型：轻甲  
盔甲防御15/15  
全能量抗力1  
描述：以夜幕初合时靛蓝与墨黑交织为底色，裙摆自腰际晕染出焰火燃烧的赤金与橙红渐变，随行走拖曳出星砂褪色般的银蓝余烬。前襟至裙尾缀满手工切割的异形水晶，暗处如星群呢喃，光照下则迸发焰火的灼目碎光。  
这件礼服将剧烈的对比编织进每一个元素，正如夜幕初临时无法定义的第一束光――究竟是星子苏醒的哈欠，还是人间烟火烫穿的孔洞？

【是焰火？】  
　　描述：裙摆是偷缝进暮色的晚霞，那位老裁缝用金线捻着余温尚存的烟花屑，在深蓝绸缎上刺出最后一道橙红弧光。她旋转时，水晶如萤火虫追着未燃尽的火药味在暗纹里簌簌游动。人们说总能在裙裾翻涌的一刹那看到二十年前城郊废弃的烟花厂――少女踮脚接住坠落的火星，而父亲在烫金宣纸上画着永不熄灭的火焰......此刻她抚过腰间灼烫的刺绣，指腹沾满了磷粉与记忆的灰烬。  
效果：回忆如焰火般绚烂，往昔的爱意与父亲亲手编织的礼裙在心底深处埋下温暖的种子，你的意志更加坚定，此刻，你在意志豁免检定上获得6点加值。那件承载着父爱的礼裙，为你增添4点风度检定上的表现加值，让你举手投足间都散发着动人的光彩。你在战斗中可以一个标准动作，将回忆中的那漫天烟火在当下场景中唤出，三轮后，在天空中盛放，场景内的所有单位都可以看见，照亮内心中前行的道路，这不会改变场景中的照明等级。

【还是繁星？】  
　　描述：裙摆盛着整个仲夏夜的星斗，那些曾随焰火升空的玻璃糖纸在岁月里褪成透明星砂，此刻正沿着她锁骨流淌成银河支流。每颗水晶都是未燃尽的火药开出的花――旧铁盒里藏了二十年的哑炮，终于在今夜化作不会灼伤指尖的辰光。当她踮脚捧起一抹夜色，在那裙裾漾起的涟漪中，亿万星子正朝着比烟花更高的地方安静生长。

效果：星光洒落，你身着那件在过去就开始为你的未来所编织的礼裙，裙摆摇曳间仿佛承载着时光的重量。这件礼裙为你提供3点立场防御，让你在未知的浪潮中安然前行。你获得了D级不老效果，B级的免疫衰老使得岁月无法在你身上留下痕迹，让你如同群星般永恒闪耀。感知星辰的低语，你能够观测五天内天气的走向并简单的引导今天的天象。只要你展开双臂便能以完美的机动性飞行，全环境呼吸使你与万物共鸣，这一刻，你既是故事的见证者，也是未来的书写者，在成长的道路上，与星辰共舞，与时光同行。
