# 蓝色纯真之盾

蓝色纯真之盾  
价格：B+4000  
分类：轻盾  
武器伤害：2L  
武器特性：【格挡18】  
武器体积：3  
伤害类型：物理-钝击  
描述：拥有净化被迷惑的眼睛的效果的盾。担任在边境监视入境者和同行者的职务的骑士的持有物。  
效果：持有者在格挡时免疫不超过A级幻象来源效果的影响；当持有者在格挡状态下受到攻击时，获得等同【传奇感知】\*1个防御附加成功数（基于格挡防御），并令攻击者获得基于格挡防御的【劣化2】。
