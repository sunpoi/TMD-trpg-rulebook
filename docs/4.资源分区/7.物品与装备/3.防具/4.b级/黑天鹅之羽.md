# 黑天鹅之羽

黑天鹅之羽  
B+2000  
这套护甲由漆黑的羽毛编织而成，每一片羽毛都闪烁着幽暗的光泽，仿佛夜空中坠落的星辰。穿戴者如同化身为优雅的黑天鹅，步履间轻盈无声，羽毛在风中微微颤动，散发出神秘而危险的气息。护甲的核心镶嵌着一颗黑曜石，能够在危急时刻释放出黑暗能量，遮蔽穿戴者的身形。  
盔甲类型：轻甲  
盔甲防御 :17/17  
盔甲特性:【黑天鹅】【幕间】

【黑天鹅】这件护甲的盔甲防御转变为偏斜防御，你穿戴护甲时，你步态轻盈，仿佛优雅的舞者，你反射豁免提高12点，你的敏捷检定提升6点。  
如果你穿着这件护甲进行舞蹈或者歌唱，那么这次舞蹈或者歌唱获得6点增强加值。

【幕间】任何演员都需要一个换装期，都需要一个休息室……你在战斗中可以使用一个移动动作，直到你下回合行动前，你为自身增添11点遮蔽防御，视觉上是突然间从天而降一个独立黑纱更衣间把你自身笼罩在其中，你曼妙的身影在黑纱下若隐若现。
