# 活盾牌

活盾牌  
价格：B+3000  
分类：重盾  
武器伤害：3L  
武器特性：【双手】【格挡21】  
武器体积：4  
伤害类型：物理-钝击  
描述：生物身体的一部分硬质化变成了大盾牌，有时表面看起来在蠕动。  
效果：持有者在格挡时免疫不超过A级毒素/药物来源的效果影响；持有者在格挡时获得等同自身耐力属性三分之一的快速医疗点数（每1快速医疗点数对应12点医疗池）与天生防御；持有者使用该盾牌进行格挡获得的防御，将会改为天生防御。
