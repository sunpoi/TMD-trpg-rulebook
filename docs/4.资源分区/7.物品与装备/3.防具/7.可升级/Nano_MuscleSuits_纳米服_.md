# Nano-MuscleSuits（纳米服）

Nano-MuscleSuits（纳米服）  
科技本质  
描述：它明显把操作者的能力提高到一个超人类的水平。它是威力强大而且非常灵活的战斗装甲，起初提出的目的是让人类在敌方区域存活，而实质并非是带有致命的目的。允许衣服的适应能力通过各种形式，热，光，辐射，静电，甚至通过碳来吸收能量。此外，Nanosuit的CryFibril人造肌肉纤维可以被程序化的编辑为多种用途（如硬化，以吸收并抵御更多的伤害，肌肉增强，或隐形），从而大大增强战斗性能和物理性能，使使用者做出超人类的壮举。

与原作之中，Nano-MuscleSuits，这一版本没有能源的限制。  
若无特殊说明，则下属内容需要逐级购买。

NanoSuit 1.0  
价格：D+500  
分类：重甲  
盔甲防御：7/7  
盔甲减值：1  
描述：纳米服的初始模式，均衡的提供了强化身体机能，提供防护，适应环境的能力。其他功能性则需要额外购买解锁。  
效果：  
NanoSuit 1.0会为穿戴者提供C级环境免疫能力。  
NanoSuit 1.0会为穿戴者提供生理相关检定上的1DP器械加值。

支付C+1000解锁：NanoSuit 1.0提升为C级护甲，盔甲防御提升至15/15，你可以从【装甲模式】【力量模式】【速度模式】【匿藏模式】中选择一种获得。

你可以再支付C+1000解锁未选择的三项模式。每种模式与其他模式无法同时存在，若一轮内已经触发了一种模式的效果，则你无法获得其他模式带来的好处。此外，变更模式是一个自由动作，每轮只能变更一次。

装甲模式：该模式下，穿戴者获得等同自身【传奇耐力/2】点【全伤害吸收】，与【坚韧】，基于盔甲防御。若该盔甲防御因其他原因发生变动，则上述内容改为基于变动后的防御类型。  
“装甲模式转移NS1.0的所有电容，供应该模式来吸收入射弹的动能和其他破坏性的力量，如热量和危险的辐射水平，有效地使使用者暂时不受到真正的生理损伤，并因此能承受通常是致命的冲击。”

力量模式：该模式下，穿戴者的力量相关检定获得5DP器械加值；当穿戴者进行跳跃时，可以准确的控制跳跃的落点和距离而无需检定，将跳跃和移动结合，同时免疫自然坠落带来的伤害（跳跃的高度不能超过移动距离的1/2；回合结束时仍然需要站在稳固的落脚点上，否则就会坠落）。

“在力量模式时，CryFibril收紧，通过使用者的皮肤注入生化调节物质——很容易被吸收进入血液的提升人类性能力量的物质。同时加入一些纳米机器人来增强各方面力量，只要模式被激活，就会让使用者表现超人类的力量。并且NS1.0会发出明亮的红色，近战攻击造成极端致命的伤害（打击，投掷等等）。”  
“这也提高了跳跃的能力，让使用者能够执行“高跳，”使他/她的跳跃高度达到他们能够达到的有效的极端高度的三倍。”

速度模式：该模式下，穿戴者的敏捷相关检定获得5DP器械加值；穿戴者获得等同于基础移动速度的游泳速度（自带水下自由呼吸能力）；穿戴者的基础移动速度提升15米。

“在速度模式下，使用者吸入氧气量多。这是通过注射的纳米机器人，输入他/她的血液，加速整个身体的血流量，提供更多的氧气到大脑，以及肌肉组织，刺激高度异常的神经反射。”  
“NS1.0的“水力推进器”的速度也增快，在水下，你可以使用你的优势“冲刺”冲出水的表面。”

匿藏模式：该模式下，穿戴者获得C级科技本质的反侦测能力。  
“当匿藏模式激活，纳米服改造其自身的表层，通过使用变殖出一种结晶，结晶能充分吸收和弯折到来的光波，譬如可见光。无线电波（雷达），红外线（热成像），微波或者其他一类的。来呈现出人眼和最严密的监控设备完全不可见的身形。”  

再支付B+3000解锁：Nanosuit 2.0  
你需要在NanoSuit 1.0升级为C级的基础上，才能购买Nanosuit 2.0，Nanosuit 2.0的盔甲防御提升至22/22，视为B级护甲。  
Nanosuit 2.0增加了S.E.C.O.N.D人工AI智能，视为与盔甲等级相同的电子设备（虽然不会提供器用加值）。与NanoSuit 1.0不同，Nanosuit 2.0和穿戴者的身体融合性更高，可以调节身体性能的同时，防护性能与环境危害抵御的性能更高。  
这意味着佩戴者在D级获得的环境免疫能力提升至B级，在生理相关检定上获得等同【耐力/2】点器械加值。

此外，若你已经具有【装甲模式】【力量模式】【速度模式】【匿藏模式】，则可以选择其中之一进行强化。

强化【装甲模式】：提供全伤害吸收从【传奇耐力/2】提升至【传奇耐力\*1】，【坚韧】改为【超级坚韧】。  
强化【力量模式】：提供的力量相关检定加值改为【力量/2】点，在力量相关检定上获得【8加骰】。  
强化【速度模式】：提供的敏捷相关检定加值改为【敏捷/2】点，基础移动速度的提升改为【敏捷\*7】米。  
强化【匿藏模式】：提供的反侦测能力提升至B级，强化这一模式会减少1000分的购买价格。

【额外插件】  
纳米面甲  
价格：D+500/C+1000/B+2000（差价升级）  
描述：纳米面甲是能够接受高频率光波的装备，可启用如夜视视野或者纳米视野。  
效果：购买后，纳米服会为穿戴者提供D/C/B级科技本质的侦测能力。  
在D级时，提供【黑暗视觉】。  
在C级时，提供【透视】，这是纳米视野进行高亮标注的特殊视野，可以穿透障碍物（或者烟雾）标记其他目标。  
在B级时，提供敏感范围增加一倍的效果。

超级计算器  
价格：D+500/C+1000（差价升级）  
价格：超级计算可以通过远程接口来黑掉大多数的电子设备，无论是军用还是民用的。甚至是异形的。它的收发器也可以进行编程来干扰CEPH的通讯。也能够记录视频和环境数据，或是提供音频设备与通信设备，可以接受很广的频率范围。  
效果：  
提供借助通讯器等电子设备远程沟沟通的能力。通讯器等电子设备只是对于纳米服内置电子设备投射的载体，因此距离无限，没有人数限制，只需要设置同一个频道。纳米服本身如同电子设备一样可以进行拨打电话，录制视频等功能，可以类似手机/电脑一样使用。  
在D级时，穿戴者可以通过内置的设备尝试可骇入有线直连的设备，以及在同一个局域网下的设备。  
在C级时，穿戴者可以通过内置的设备尝试以无线信号骇入片区局域网、机密设备。

防护强化  
价格：D+1000  
描述：对于纳米服防护外层的基础强化。进一步提升了对于能量武器的吸收能力，与对投射物的防护性质  
效果：纳米服获得【封闭1】【防弹】【防能量武器】。
