# 魔女的冲锋衣

魔女的冲锋衣  
价格：D+500  
分类：轻甲  
本质：特异本质  
防御：偏斜防御5/5  
升级方式：差价升级  
特性：【怪谈】【百变】【怪谈护体】  
描述：源自某本小说，是书中主角的专属冲锋衣，可以吞噬任何服装，变成任何被他吞噬的服装  
这是一件普普通通的冲锋衣，但视乎又有些不太普通？

【怪谈】：  
作为怪谈，且是最弱小的那一批，它现在极其弱小，但作为怪谈，也不是普普通通的人可以应对的。  
正常人只能看到模糊的轮廓，除非他主动显形，具有灵视的人观察它时，会发现这是一件........红白色冲锋衣？这仅是画风，并不会给冲锋衣带来怎么好处。

【怪谈护体】：  
当你穿着怪谈冲锋衣的时候，能感受到它在保护着你，虽然很微弱，但这确实分担了你不少的压力。

效果：  
【怪谈】：  
作为怪谈，你可以与其沟通交流，同样，如果冲锋衣愿意，队友也可以看见这个冲锋衣，并且与其进行交流，冲锋衣进行交谈时，是以一种气泡+颜表情的形式进行的，在交谈时，冲锋衣头顶会冒出气泡，上面附带颜表情，看到的人会不由自主的理解它想要表达的意识。

备注：需要轮回者自己扮演冲锋衣进行rp，请不要操控冲锋衣去探索地图，st有权拒绝轮回者拿冲锋衣当随从的用法，并且冲锋衣无法离开其主人身边

【百变】：  
作为怪谈，冲锋衣可以“吃掉”任何无支线衣物，并且变换成任意被自己“吃掉”过的衣服，这将消耗你一个花招动作，每轮仅限变化一次，这视为一种D级的变形效果。

PS：不仅仅是衣服，它可以连带裤子，裙子，丝袜，帽子等都可以吃掉，并且变化成对应服饰，这意味着衣物的变形是全身性的，也就意味着你可以瞬间从cos初音的服饰变成cos阿米娅的服饰

【怪谈护体】：  
获得1点活体护甲，并且活体护甲提供的是偏斜防御而非护甲防御

等级名称：怪谈核心（半颗）

分类：轻甲  
本质：特异本质

【怪谈护体】：

冲锋衣吃下这半颗核心后，将进一步被强化，此时冲锋衣视为C级资源

这视为一种D级的变形效果  
在副本中，模仿他人服饰，会让冲锋衣逐渐掌握这份能力；当你将这半颗核心喂给冲锋衣后，再经历一场影片即可使其进一步觉醒：  
作为已经觉醒的怪谈，它现在不单单可以通过“吃”掉衣物进行变化，还可以通过观察来模仿对方身上的穿着，但是由于冲锋衣才诞生，所以可能伪装偶尔会出现纰漏。  
这视为一种D级的伪装效果。

PS：不仅仅是衣服，它可以连带裤子，裙子，丝袜，帽子等都可以吃掉，并且变化成对应服饰，这意味着衣物的变形是全身性的，也就意味着你可以瞬间从cos初音的服饰变成cos阿米娅的服饰
