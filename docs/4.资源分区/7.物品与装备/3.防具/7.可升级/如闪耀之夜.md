# 如闪耀之夜

如闪耀之夜  
资源类型：可升级防具  
本质：特异本质

概述：  
由纯粹的黑夜编织而出的一件外套，仿佛将最深沉的夜幕裁下，凝练成柔软的实体。如闪耀之夜是一件纯黑色的夹克，在远处端详，它是一件纯黑色的夹克，在远处端详，它呈现出一种深邃、凝练的黑色，表面仿佛微带光泽，犹如顶级皮革般沉稳内敛。

然而，凑近了细看，这份纯粹便悄然瓦解。黑色并非单调的幕布，反而像是无限深远的夜空背景。目光一旦聚焦，便能察觉到内里并非空无一物——点点纤细、微弱的星光悄然从织物深处透射出来，并非刺眼的光芒，而是像宇宙尘、或者遥远星云中初生的星子，在深邃的背景上轻柔地呼吸、闪烁。这氤氲的星光并非浮于表面，而是仿佛被一层无形的夜色包裹着、酝酿着，为外套注入了一丝深邃宇宙的灵动与生机。

当你以为指尖将触及那视觉上类似皮革的光滑冷硬时，却意外陷入了一片不可思议的温软之中。那并非人造纤维的柔顺，而是一种极其细腻、微微带着绒感的动物羽毛般的触觉。表面触感细密如幼鸟的绒羽，轻轻抚过，又有一种难以言喻的顺滑与弹性，如同抚摸过鸟类最柔韧的飞行羽尖。每一次触碰，似乎都能感受到那“黑夜”本身的生命力在轻轻回应，微凉却不冰冷，仿佛是刚从暮色笼罩的森林深处取出，带着露水的气息和暗夜的静谧。

穿在身上，它轻盈得几乎忘却了存在，却又以无声的方式彰显着自己的不凡。行动时，那些深藏的星光仿佛受到牵引，在暗色的涟漪中流转、明灭，形成一片只属于你的、流动的微型宇宙图景。它不只是御寒的衣物，更像是一副秘咒披风，将寂静与星辰随身携带，让佩戴者仿佛将苍穹的私密角落悄然裹挟，在人间行走时，亦披着一片会呼吸的夜空。

总纲：  
该护甲为差价升级，高级效果替换低级。

分类：轻甲  
本质：特异本质  
防御：天生防御7/7  
特性：【身披夜色】【宁静】

当你穿上这件防具后，你就像身披一片夜色一样神秘而且深邃。

【宁静】：  
你在穿上这件轻甲后便被夜色所保护，俗世中的一切不会影响到你。
