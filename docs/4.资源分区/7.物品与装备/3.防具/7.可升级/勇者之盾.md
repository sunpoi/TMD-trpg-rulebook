# 勇者之盾

  

勇者之盾  
  
原典：盾之勇者成名录  
价格：D+500  
描述：由抵御“浪潮”的盾之勇者所持有，放弃全部攻击换取防御能力的守护之盾。  
虽然现在只是小盾的模样，但只要持续吸收素材，总有一天会成长为守护世界的坚盾。  
类型：轻盾  
武器伤害：0L  
体积：2  
专业：盾  
特性：【格挡6】  
当你持有这一盾牌时，不可同时持有其他武器、盾牌。  
【盾牌统合】：人物每购买了2个D级别及以上的盾牌，该盾牌的格挡数值就提升1点。人物在购买盾之勇者世界观的其他D级别盾牌时，会额外赠送一面盾之勇者世界观下的其他D级别盾牌，若如此做，则这些盾牌会立即被收纳入勇者之盾。  
【盾牌收纳】：这一面盾牌具有收纳融合其他盾牌的能力，人物在购买了一件盾牌后，可以将这一盾牌融合入勇者之盾，这代表着该盾牌进入了特殊的储物空间，进入该储物空间后的盾牌将无法取出，但不妨碍对盾牌进行附魔、改造或是回收。  
人物可以迅捷动作将收入盾牌的特殊效果、所有基本属性、外观替换勇者之盾除【盾牌收纳】外的特殊效果、所有基本属性、外观。以此方式使用盾牌时，视为你正使用着其他盾牌而非勇者之盾。  
当盾牌被切换以后，原先盾牌当前生效的所有效果会立即中断。  
将其他盾牌切换回勇者之盾是一个迅捷动作。  
  
  
额外支付C+1500  
勇者之盾的格挡数值提升8点，在格挡时具有基于格挡防御的【适应】与1点全吸收。  
盾牌获得以下额外效果：  
【盾牌统合+】：人物每购买了2个C级别及以上的盾牌，该盾牌的格挡数值就提升3点。人物在购买盾之勇者世界观的其他C级别盾牌时，会额外赠送一面盾之勇者世界观下的其他C级别盾牌，若如此做，则这些盾牌会立即被收纳入勇者之盾。
