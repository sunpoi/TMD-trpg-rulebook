# 通行方便

  

通行方便  
通行方便是一件会根据使用者所在环境发生改变的衣服，具有不同等级的型号，支持差价升级成更高级型号。不同型号具有对应等级轻甲级别的防御力，并且高等级的通行方便可以提供更多的方便通行的能力。不过因为通行方便的本身特性，会视为当前形态对应本质物品，但严格意义上来说可以视为全本质物品，并且在受针对时取劣判定。  
  
有点方便级 D+500  
特异本质  
盔甲防御：5  
盔甲类型：轻甲  
  
身份方便：通行方便会在使用者进入影片时伪造一个身份，该身份不会具有过往经历，仅附带可以证明身份的物品，如身份证、健康码、通行证等，方面搭乘一些交通工具，如果被调查很容易发现是伪造的，这视为D级命运来源的背景身份。  
  
衣物方便：通行方便会在使用者进入影片时自动变化成符合时代背景与安排身份的服饰，并改变成对应本质物品，视情况可能占据不同的装备位，这视为D级变形效果。  
  
  
一般方便级 C+1000  
盔甲防御：10  
盔甲类型：轻甲  
  
身份方便：通行方便会在使用者进入影片时伪造一个身份，该身份会具有一段基本不去人社交的过往经历和对应记忆，使用者只能通过这份记忆知道一些常识性知识与当地交通出行的相关信息，附带可以证明身份的物品和用于普通出行的物品，如公交卡、少量零钱、滴滴打车账号等，这视为C级命运来源的背景身份。  
  
衣物方便：通行方便会在使用者进入影片时自动变化成符合时代背景与安排身份的服饰，并改变成对应本质物品，视情况可能占据不同的装备位，此外还能以一个标准动作使通行方便变同类型但不同颜色细节的服饰,这视为C级变形效果。  
  
  
还挺方便级 B+2000  
盔甲防御：17  
盔甲类型：轻甲  
  
身份方便：通行方便会在使用者进入影片时制造并插入一个身份，该身份会具有一段普通人的过往经历和对应记忆，使用者能通过这份记忆知道一些常识性知识、当地交通出行的相关信息，附带正常人应有的财产，如一套住房、一辆车等，这视为B级命运来源的背景身份。  
  
衣物方便：通行方便会在使用者进入影片时自动变化成符合时代背景与安排身份的服饰，并将使用者身上其余不符合时代背景与身份的物品在不影响效果的前提下变化成相符的物品。这种变化会使通行方便变成对应本质物品，占据不同的上肢-躯干或下肢-躯干其中之一的装备位，此外还能以一个标准动作使通行方便变幻成同类型但不同颜色细节的服饰,这视为B级变形效果。  
  
交通方便：在身份方便的延伸影响下，必定会为使用者安排一辆其能够驾驶的轻型载具，这辆轻型载具的外形必定符合时代背景与安排身份，而其各项数据也会与时代背景与安排身份相符，但是存在一个上限。其移动速度上限为74米或160米/轮，防御力上限为基础防御7+1或装甲防御14，生命值上限为24血量或52结构值，并且一定不能为使用者提供全遮蔽效果，视为一辆上限B级的载具。  
  
  
非常方便级 A+4000  
盔甲防御：26+2  
盔甲类型：轻甲  

身份方便：通行方便会在使用者进入影片时制造并插入一个身份，该身份会具有一段符合使用者能力与性格的过往经历和对应记忆，使用者能通过这份记忆知道一些常识性知识、当地交通出行的相关信息、特殊出行联系渠道与人脉，附带正常人应有的财产与符合身份的消耗品，如手雷、弹药等，这视为A级命运来源的背景身份。  
  
衣物方便：通行方便会在使用者进入影片时自动变化成符合时代背景与安排身份的服饰，并将使用者身上其余不符合时代背景与身份的物品在不影响效果的前提下变化成相符的物品。这种变化会使通行方便变成对应本质物品，占据不同的上肢-躯干或下肢-躯干其中之一的装备位，此外还能以一个标准动作使通行方便与交通方便带来的载具变幻成同类型但不同颜色细节的服饰和载具,这视为A级变形效果。  
  
交通方便：在身份方便的延伸影响下，必定会为使用者安排一辆其能够驾驶的轻型载具，这辆轻型载具的外形必定符合时代背景与安排身份，而其各项数据也会与时代背景与安排身份相符，但是存在一个上限。其移动速度上限为94米或200米/轮，防御力上限为基础防御9+1或装甲防御18，生命值上限为31血量或66结构值，并且一定不能为使用者提供全遮蔽效果，视为一辆上限A级的载具。  
  
旅游方便：在身份方便的延伸影响下，必定能获得多种与长途旅行或出行的联系渠道与人脉，并且能够免费、立刻准备出发的类型，其免费上限为在星系内旅行的出行方式，这视为A级旅行。  
  
  
超级方便 S+8000  
盔甲防御：37+8  
盔甲类型：轻甲

  
身份方便：通行方便会在使用者进入影片时制造并插入一个身份，该身份会具有一段符合使用者能力与性格并拥有极高地位且无责任的过往经历和对应记忆，使用者能通过这份记忆知道一些常识性知识、当地交通出行的相关信息、特殊出行联系渠道与人脉，附带近乎用不完的财物，这视为S级命运来源的背景身份。  
  
衣物方便：通行方便会在使用者进入影片时自动变化成符合时代背景与安排身份的服饰，并将使用者身上其余不符合时代背景与身份的物品在不影响效果的前提下变化成相符的物品。这种变化会使通行方便变成对应本质物品，占据不同的上肢-躯干或下肢-躯干其中之一的装备位，此外还能以一个标准动作使通行方便、交通方便带来的载具、身上说有物品变幻成同类型但不同颜色细节的服饰和载具,这视为S级变形效果。  
  
交通方便：在身份方便的延伸影响下，必定会为使用者安排一辆其能够驾驶的轻型载具，这辆轻型载具的外形必定符合时代背景与安排身份，而其各项数据也会与时代背景与安排身份相符，但是存在一个上限。其移动速度上限为154米或320米/轮，防御力上限为基础防御17+3或装甲防御30，生命值上限为54血量或112结构值，并且一定不能为使用者提供全遮蔽效果，视为一辆上限S级的载具。  
  
旅游方便：在身份方便的延伸影响下，必定能获得多种与长途旅行或出行的联系渠道与人脉，并且能够免费、立刻准备出发的类型，其免费上限为在星河内旅行的出行方式，这视为S级旅行。  
  
携带方便：为了方便使用者不会因为携带物品过多影响出行，通行方便能为使用者提供一个无限容量的储物空间，并且能够容纳活物，但是容纳的前提的拥有其所有权的，这视为S级异空间效果。
