# 蜘蛛战衣

蜘蛛战衣  
资源类型：可升级防具  
本质：科技本质

概述：  
购买和升级时可以指定蜘蛛战衣的服装样式，例如初代蜘蛛侠彼得帕克、小黑蛛迈尔斯莫拉莱斯、幽灵蜘蛛格温史黛西等，如果不指定则默认采用彼得帕克的经典红蓝战衣外形。  
总纲：  
战衣效果默认高级取代低级，价格为差价升级。

价格：D+500  
分类：轻甲  
本质：科技本质  
防御：盔甲防御6/6  
特性：【防弹】  
效果：  
提供反射豁免的2DP器械加值。  
穿戴者获得等同于基本移动速度的攀爬速度。

价格：C+1000  
分类：轻甲  
本质：科技本质  
防御：盔甲防御10/10  
特性：【防弹】【防能量武器】  
效果：  
提供反射豁免的2DP器械加值与1个附加成功。  
穿戴者获得等同于基本移动速度的攀爬速度，攀爬或利用蛛丝移动时移速+5M，如果已拥有类似能力，敏捷检定+1器械加值。  
穿戴者获得4根金属蜘蛛附肢，这不能像手一样使用或持握武器，但是可以辅助进行战斗。展开/收起蜘蛛附肢需要一个迅捷/反射动作，这使你在运动上获得2DP器械加值。

价格：B+2000  
分类：轻甲  
本质：科技本质  
防御：盔甲防御17/17  
特性：【防弹】【防能量武器】  
效果：  
提供反射豁免的2DP器械加值与1个附加成功。  
穿戴者获得等同于基本移动速度的攀爬速度，攀爬或利用蛛丝移动时移速+15M。  
穿戴者获得4根金属蜘蛛附肢，这不能像手一样使用或持握武器，但是可以辅助进行战斗。展开/收起蜘蛛附肢需要一个迅捷/反射动作，这使你在运动上获得6DP器械加值和1点附加成功。  
在蜘蛛附肢展开时，你每轮都能以自由动作进行一次移动。

配件：  
奇物名称：翼装滑翔  
价格：500  
描述：  
战衣的两腋部位加装了蛛网翼装薄膜，可以在空中短距离翼装滑翔。  
效果：  
免疫自然坠落伤害。
