# 绝地长袍

绝地长袍  
资源类型：可升级防具  
本质：科技本质

概述：  
绝地武士的长袍通常由外袍和前襟两部分组成。外袍长度从及膝至脚踝可自由定制；前襟则具有两条对称布条挂在身前。此番设计考虑了绝地武士们的穿着舒适性、肢体活动自由度以及对原力的亲和性，兼顾了实用性与功能性。

绝地学徒长袍  
价格：D+500  
分类：轻甲  
本质：特异本质  
防御：偏斜防御0/0  
特性：【防弹】【原力导延】【多功能腰囊】  
效果：  
绝地长袍的材质能确保穿着者免受一定程度的恶劣天气条件，令穿戴者免疫D级别、环境来源的负面效果。  
【原力导延】：  
若穿着者的原力池不空，则该护甲的偏斜防御提升至5/5。角色可以花招动作消耗1点原力在场景内将该护甲的偏斜防御提升至8/8。

【多功能腰囊】：  
角色获得四个仅能放置体积不超过1物品的消耗品栏位。

绝地武士长袍  
价格：C+1500  
分类：轻甲  
本质：特异本质  
防御：偏斜防御0/0  
特性：【防弹】【原力导延】【多功能腰囊】

效果：  
可由绝地学徒长袍差价升级。  
绝地长袍的材质能确保穿着者免受一定程度的恶劣天气条件，令穿戴者免疫C级别、环境来源的负面效果。  
【原力导延】：  
若穿着者的原力池不空，则该护甲的偏斜防御提升至10/10。角色可以迅捷动作消耗3点原力在场景内将该护甲的偏斜防御提升至16/16，并获得基于此护甲防御类型的【适应】。

【多功能腰囊】：  
角色获得四个仅能放置体积不超过1物品的消耗品栏位。
