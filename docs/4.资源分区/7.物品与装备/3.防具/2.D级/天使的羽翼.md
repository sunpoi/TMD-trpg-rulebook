# 天使的羽翼

天使的羽翼  
价格：D+1000  
分类：轻甲  
盔甲防御：6/6  
盔甲特殊效果：无  
描述：这是一对翅膀（或者多对翅膀），当你召唤这件盔甲时，天使的羽翼在身后展开，在你的身后，有着圣光的特效，在特殊的世界观，说不定有奇效……  
“快看，那是天使！哦，我的上帝，我终于等到天使降临在我身边了”  
“不要突然在里面夹带奇奇怪怪的东西啊喂”

  
灵巧：凭借超高的机动性，你可以灵活的躲避他人的攻击，因此这件装备提供的是闪避防御，而不是铠甲防御。

  
羽翼：天使的羽翼，乃是新的肢体，你需要经过锻炼，才可以使用它进行飞行，获得等同基础速度的飞行能力，机动性为笨拙。  
现在开始，每经历一场影片，你的飞行机动性上升一级，直到良好。（笨拙-普通-良好）
