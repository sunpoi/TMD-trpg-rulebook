# 生苔铁板

生苔铁板  
价格：D+1000  
分类：轻盾  
武器伤害：2L  
武器特性：【格挡8】  
武器体积：3  
伤害类型：物理-钝击  
描述：沾满有放松效果的苔藓的盾，放松神经能够增幅与自然的联系，积蓄魔力。  
效果：当你本轮或者上一轮消耗了魔力，持有者在格挡时会同时恢复1点魔力/魔力变体，这一效果每轮只会生效一次，无法使相应能量池的恢复超过每日三倍上限。
