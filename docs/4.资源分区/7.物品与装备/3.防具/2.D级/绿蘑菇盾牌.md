# 绿蘑菇盾牌

  

绿蘑菇盾牌  
D+500  
原典：盾之勇者成名录  
吸收阴暗处的绿色蘑菇解放的形态，带来了调和的知识。  
类型：轻盾  
武器伤害：1L  
体积：2  
专业：盾  
特性：【格挡6】  
【草药调和】：每日一次，你现在能够使用具有治愈功效的药草（价值10分）花费30分钟进行调和获得12份以下药剂：  
\[草药\]  
使用动作：花招  
保质期：一日  
最大持有量：12  
效果：立即恢复1点伤势并且移除身上的D级别毒素来源的负面效果，每轮只能使用一次此药剂。  
  
附加技艺  
  
调和精熟  
D+500  
草药的恢复伤势提升1点，移除的毒素来源级别提升至C级，使用草药以后会额外解除C级别诅咒来源的负面效果。
