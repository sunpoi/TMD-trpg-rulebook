# 骑士重甲

  

骑士重甲  
价格：D+500  
分类：重甲  
魔法本质  
描述：以纯钢打造的古典骑士的盔甲，所具有的多层结构能抵抗大多数攻击，其白色的涂漆是否象征着骑士曾经的信念呢。  
盔甲基本属性：盔甲防御7/7，盔甲减值3  
盔甲特殊属性：【防弹】  
英勇冲锋：骑士的冲锋一往无前，曾经如此，今日亦然。  
因此，穿戴本盔甲的单位在冲锋时会额外获得2点加值，所提供的加值不计入基础防御减值内。  
此外，在对抗恐惧异常点数所进行的豁免时，还会获得4DP士气加值。
