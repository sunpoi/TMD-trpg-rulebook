# 巨人之铠

巨人之铠  
本质：特异本质  
价格：C+1000  
分类：重甲  
盔甲防御：14/14，盔甲减值2  
盔甲特殊属性：无  
描述：通过祈祷自己能和巨人一样强大的衣服，理所当然的只有在面对巨人的场合才能生效。  
重量级：巨人之铠的重量实在太过沉重，穿戴者每轮会失去一个迅捷动作。

  
巨人级：穿戴者在绊摔/冲撞/擒抱无视体积带来的优势与劣势，同时忽略他人因体积带来的天生防御，这个效果会被C级或以上的免疫接触攻击效果无效。
