# 炼狱碎片甲

  

炼狱碎片甲  
魔幻本质  
价格：C+1000  
分类：重甲  
描述：这套重甲是从深渊熔炉中直接拓印而出的噩梦造物，由数千片锯齿状暗红金属残片拼嵌而成。每块碎甲表面都浮动着类似熔岩静脉的纹路，在暗处会渗出硫磺色的荧光。  
盔甲防御：12/12，盔甲减值2  
盔甲特殊属性：无  
焦油哭号：炼狱金属碎片中的灵魂会被不断折磨，甲胄表面不断渗出千万个灼烧中的灵魂产生的黑色粘液，该种状态下穿戴者的近战攻击和肉搏攻击都将带有这些灵魂被折磨的疲累与疯狂的低语与呼号，附带等量疲乏与亢奋的心灵来源的异常点，可以被正常豁免。  
炼狱：重甲在非战斗状态下可以自由通过灵魂产生的粘液变形为各种服装形态，甚至包括为主人产生一副人脸，进入战斗自动变回原形，这是一个C级的变形来源的伪装效果。
