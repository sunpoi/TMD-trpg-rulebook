# 泪滴盾

  

泪滴盾  
价格：C+1000  
分类：盾  
武器伤害：5L  
武器特性：【格挡11】【双手】  
伤害类型：物理-钝击  
体积：4  
描述：泪滴形状的宝石大盾，由于其中蕴含的独特力量，曾颇受勇士之国的骑士们所喜爱。  
殿军的矜持：持有者可以使用标准动作敲击盾牌，进行一次攻击检定，位于持有者30米范围内的所有敌人以正常流程进行一次防御。  
只要攻击至少能造成1点伤害（但本次攻击检定不造成伤害），那些敌人必须以持有者为第一攻击目标，这是一个C级祝福/诅咒来源的胁迫效果。
