# 轻金属盾

  

轻金属盾  
C+1000  
原典：盾之勇者成名录  
由带有魔力的金属制造而成的盾牌，能够很好的抵御能量伤害与魔力效果  
类型：轻盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡11】  
持有者获得基于格挡防御的2点全能量抗力  
持有者免疫C级及以下的、由魔力推动的负面效果
