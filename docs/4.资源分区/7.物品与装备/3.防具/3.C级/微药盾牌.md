# 微药盾牌

  

微药盾牌  
C+1000  
原典：盾之勇者成名录  
吸收了各类药物后解放的形态，可以强化使用药物的效果  
类型：轻盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡11】  
【用药精通】：持有者使用药剂时，根据药剂的效果而获得不同效果：  
你在使用攻击类药剂时，你获得额外的2DP攻击检定和1附加成功。  
你在使用防御类药剂时，你获得同类型的两点防御和1附加成功。  
你在使用恢复类药剂时，你获得持续一场景的不可刷新的1临时生命和快速治疗1（12医疗点数）。  
你在使用其他药剂时，你获得1机运骰加值和3点对抗毒素/药物的强韧豁免上的附加成功。  
同时只能获取一种好处，若药剂符合多种类型，则由你选择一种生效
