# 0.套装概述

  

基于多个物品或强化组成的特殊模板，会根据物品等级或强化等级使同一套装内的所有物品或强化获得额外特性，通常情况下每个人只能使一个套装的效果生效，其它套装会被视为不成套的散件，不会获得额外特性，甚至失去所有效果。若要切换套装的效果需要在战斗外进行。  
套装等级等于套装内支线等级最高的物品或强化  
套装效果需要至少2个套装支线等级最高的物品被穿戴才可生效。若未说明套装效果生效所需要穿戴的物品数量，则默认需要穿戴所有套装物品才可生效。
