# 浪客套装

浪客套装  
概述：某浪迹天涯的著名街溜子最珍贵的物品套装，跟随了对方不知道多长时间却依旧相对来说完好无损，穿上这套衣服或许会触发你的某个傲娇死党对你说：“XXX，我再也不会帮你了！”的标准傲娇发言。当然相应的套装已经被主神贴心的清理消毒杀菌过一遍，拥有洁癖的人也可以放心使用。

资源名称：银制口琴  
本质：特异本质  
资源价格：500  
奇物类型：饰品  
概述：  
这是一个通体银白色外壳，某些地方能看到内部的黄铜色结构的口琴，是曾经某个浪迹天涯的人随身携带的物品。 在这个口琴的银制外壳并不透亮整体呈现一种古旧感，这份古旧反而能让人感觉到他的拥有者曾多少次演奏过它。  
描述：  
【银制口琴】：  
因为原主人对它的爱惜与演奏一缕意志寄宿在了其中，在你演奏的时候引导你演奏出美妙的音律。

特性：【银制口琴】  
效果：  
【银制口琴】：  
你现在用它进行表达的时候表达检定获得了2点表现加值。

资源名称：帽子  
本质：特异本质  
资源价格：500  
奇物类型：头盔  
概述：  
这是一顶略微有些旧但却保养的很好的棕色费多拉帽。整体看起来确实有些饱经风霜，但是却并没有给人一种破旧的感觉。  
特性：【帽子】  
效果：  
【帽子】：  
特性效果：这个帽子配合上你的风衣能更凸显出你的随身一种孤寂的浪客气质，这为你带来了2点相貌提升。

资源名称：风衣  
本质：特异本质  
资源价格：500  
奇物类型：斗篷  
概述：  
这是一件略微有些旧但却保养的很好的棕色的皮质风衣。 整体虽然和帽子同样有些饱经风霜的样子，但也仅仅是因为长时间的使用在皮革上留下了些许折痕，整体来说比较完好没有任何破损。  
特性：【风衣】  
效果：  
【风衣】：  
无论什么时候你的风衣总是能安然无恙没有破损，这为你带来了1点机运骰加值。

技艺：

资源名称：家乡小曲  
本质：特异本质  
资源价格：1750  
概述：  
你用那个口琴演奏出独特的旋律，给人一种有点忧郁的印象。 根据那个浪迹天涯的浪客的说法，这是“一首来自我家乡的歌”。这首曲子略微有些悲伤的旋律中蕴含的情感，就算是不清楚对方是个怎么样的人也会觉得他/她/它是一个有故事的人心生好感。  
特性：【家乡小曲】  
效果：  
【家乡小曲】：  
在非战斗的情况下，你可以对1个对象进行一个表达检定只要有1点便成功，听到的人对你的好感提升一个等级（不能提升到极端），同时对方会因为曲子的伤感获得1点沮丧（正常豁免）。

资源名称：run with the wind  
本质：特异本质  
资源价格：1500  
概述：  
这是某个存在着数据形成生命体的世界中一名叫做大和的少年经常吹给自己同伴听的曲子。悠扬的旋律仿佛伴随着风飘扬自在一般，这个曲子给你带来了治愈，使你的心情放松在任何地方都可以安心休息。  
特性：【run with the wind】  
效果：  
【run with the wind】：  
在非战斗的情况下，你可以同时对2个对象进行一个表达检定只要有1点便成功，听到的人将获得1点快速医疗（12医疗点）持续1天，同时在下一次进行短休息时视为自适应短休。
