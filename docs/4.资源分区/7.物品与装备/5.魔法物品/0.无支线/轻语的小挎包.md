# 轻语的小挎包

物品名称：轻语的小挎包  
资源类型：魔法物品  
本质：魔幻本质  
价格：500  
派系：万变派系  
类型：口令型（口令：轻语魔法店天天打折。）  
描述：  
这是一个黑色的小布包，连接着一条布带，其容量就是一个小布包的容量。  
效果：  
在你激活后，其内部将变为具有6立方米空间。在外部表现不变。放入/取出物品至少是一个移动动作，不能放入有支线物品。
