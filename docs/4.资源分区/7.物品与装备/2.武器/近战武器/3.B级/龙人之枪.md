# 龙人之枪

龙人之枪  
价格：B+4000  
分类：长枪  
武器伤害：18L  
武器特性：【长柄】【超级贯穿】【接触攻击】【双手】【格挡2】  
武器体积：4  
伤害类型：物理-穿刺  
描述：据说是得到龙之血的人类使用的枪。锋刃使用龙牙、能发挥可怕的威力。  
效果：当你使用该武器全力攻击时，你因全力攻击获得的DP提升至等同【力量属性】\*1点，并减少1点因此承受的防御减值。
