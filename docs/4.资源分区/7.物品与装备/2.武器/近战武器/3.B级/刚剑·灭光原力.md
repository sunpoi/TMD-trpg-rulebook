# 刚剑灭光原力

刚剑灭光原力  
本质：魔幻本质  
价格：B+2000  
分类：重剑  
体积：4  
武器伤害：21L  
武器属性：【重型武器】【双手】  
描述：被誉为连光都能够斩灭的强力重剑，同时因为其为持有者带来的刚力而著名，虽然无法斩灭光芒，但斩点别的东西倒绰绰有余。  
刚剑：这把武器能够斩断大部分防护，视为具有等于持有者传奇力量\*2的任意伤害减免忽略，同时在攻击时忽略这个数值一半的防御附加成功。
