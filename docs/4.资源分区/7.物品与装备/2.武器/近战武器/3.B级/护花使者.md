# 护花使者

护花使者  
本质：魔幻本质  
价格：B+3000  
武器伤害：17L  
武器特性：【轻灵】【高速1】  
分类：迅捷剑  
体积：3  
描述：一把华美的决斗剑，包裹决斗剑的蓝纱上绣着精致的蔷薇花，它曾经属于一位浪子，如今握在你的手中。

效果  
【寻欢作乐的高手】：“它曾经属于一位浪子，浪子总是以千奇百怪的理由请假。比如说宝剑心情不好，头发解不开了，宝石找不到了等等……顺带一提，这位浪子还是被公认的最受欢迎的男人。”

你获得一个特殊计数器【放浪之欢】，这一计数器初始数值为【0】。如果在影片内存在类似于【怡红院】【教坊司】【梨园】……等等类似的场景，你将除了完成主线任务外至少一半的时间投入其中，就能在影片结束后使该数值+1，上限4。

【放浪之欢】：每有1点计数，持有该计数的单位在使用本武器进行攻击时，附带1点不可豁免的魅惑不良点数；每有1点计数，持有该计数的单位在初次与其他非轮回者单位接触时，若你符合对方的性取向，则使对方对你的初始态度提升1级计算，但无法改善极端态度，也无法因此使态度提升至极端等级（注.一些明显禁欲的单位很有可能不会受到该能力带来的态度提升影响）。

【浪子的决心】：“为妓女的肉体或者是青睐决斗的傻子你见过，为妓女失去的生命而决斗的傻子还是第一次见到。”

当购买者持有该物品初次进入影片时，根据影片世界观而触发专属解锁支线（这一点需要由ST进行判断），这需要在购买者进入影片中任意一处城市时（这需要ST进行判断，基本上“城市”这个名词指的是任何有多于5000人的聚落（因此，至少像小城市一样大）才会生效。  
无论如何，这一支线完成的难度等同一场（或数场）同级战斗，当你完成这一支线时才能取得【浪子的决心】的效果。若失败，则该影片内无法触发新的支线，需要等到下一个影片时才能重复触发。

范例：  
【火中自由】：奴隶市场燃起了大火！奴隶们点燃此处，并试图为自己挣得自由，然而他们即将面临的是无情的镇压。  
这需要你挺身而出，帮助一定数量的奴隶从中逃生，这可能使你触发多次同级或更低级难度的战斗，并且产生后续影响。

【生死决斗】：有人通过买凶杀人，或是即将亲手杀死一位女性，可能是为了谋财，也可能是因为其他原因，而你刚好目睹了这件事的发生。  
这需要你挺身而出，无论是为了保护还是为其复仇，这会使你面临一场与你同级难度的战斗，并且产生后续影响。

一旦你完成了支线，将解锁这一武器提供的效果。  
\*每轮开始时选择其中一项生效：  
1.支付1点意志力，在攻击上获得等同决心属性的士气加值，持续一轮。  
2.支付1点意志力，在意志豁免上获得等同决心属性两倍的士气加值，持续一轮。
