# 癫狂

癫狂  
价格：B+4000  
分类：重剑  
武器伤害：20L  
武器特性：【重型武器】【双手】  
武器体积：4  
伤害类型：物理-穿刺  
描述：得到巨大攻击力、但一切精神抑制变得无效，据说是已经魔物化的骑士使用的武器，持有者会变得一直为血痴狂。  
效果：  
【强化武器】：当你使用癫狂进行攻击时，其武器伤害提升等同你【力量属性】\*1L，并且获得【余波】特性。  
【为血痴狂】：使持有者无法使用意志力、无法使用低于B级的需要动作使用的招式；持有者进行攻击并造成伤害后，需要在回合结束时进行一次意志豁免，DC为自身决心属性，失败则陷入无法分别敌我的【混乱】状态，自动攻击最近的目标。但若持有者的沉着属性不低于决心，则不会因此陷入混乱，并获得免疫不超过A级心灵来源效果的能力。
