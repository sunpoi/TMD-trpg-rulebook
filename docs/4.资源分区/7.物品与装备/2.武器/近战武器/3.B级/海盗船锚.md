# 海盗船锚

海盗船锚  
价格：B+4000  
分类：斧  
武器伤害：20L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】  
武器体积：4  
伤害类型：物理-挥砍  
描述：古老海盜船的物品。顶端是刃所以能当武器、极重、命中精度也很糟糕。  
效果：持有该武器进行攻击时，可以通过缠绕在船锚一侧的铁链延长攻击距离，这会使海盗船锚的近战触及距离提升【力量属性】\*3米。  
每轮一次，当你使用海盗船锚进行攻击时，你可以宣言进行一次【挥击】，这次【挥击】中可以正常使用招式，攻击范围内的所有敌对单位，具有【破甲20】，同时攻击出目增加一半，但【挥击】只有50%的命中率，这意味着你在宣言【挥击】时需要进行一次1D2，若结果为2则自动失手，失手意味着这次攻击不会造成任何影响，动作也不会返还。
