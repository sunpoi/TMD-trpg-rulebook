# 升阳

升阳  
价格：B+4000  
分类：斧  
武器伤害：20L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】  
武器体积：4  
伤害类型：物理-挥砍/雷电（取优）  
描述：带有黄色的斧刀看起来就像日出所以得名，在古代遗迹发掘、如今保管在皇家博物馆。  
效果：持有者可以消费一个迅捷动作启动效果，持续一场景；持续时间内，持有者的生理属性检定获得6点增强加值，在使用升阳进行攻击时，攻击获得【幻灭】与【余波】特性。
