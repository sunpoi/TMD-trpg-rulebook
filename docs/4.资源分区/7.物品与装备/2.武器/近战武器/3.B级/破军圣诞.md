# 破军圣诞

破军圣诞  
B+2000  
武器  
武器伤害：19L  
分类：戟  
特性：双手，长柄，格挡2  
体积：5

“叮叮当，叮叮当，铃儿响叮当~”  
这是一柄形如圣诞树的长戟，棕色的树干与翠绿的枝叶上挂满了铃铛与彩带，挥动时会发出清脆悦耳的节日铃声。

【圣诞礼物】  
“乖孩子有礼物哦！”  
破军圣诞的铃铛内蕴藏着一个约100立方米的折叠空间，其中一部分为空，另一部分则装满了包装好的礼物。取出或存放物品需要一个花招动作，该空间被视为一个C级储存空间。  
但礼物不一样，你可以以一个自由动作从中取出一份礼物，赠予15米范围内的多个目标，但不包括自己，毕竟哪有圣诞老人送自己礼物的。  
收到礼物的人不会知晓礼物内容——事实上，就连你在赠送前也无法知晓。在送出礼物前，你需要进行一次d2判定，并根据结果决定礼物的内容：

· 若结果为1，目标获得一块糖果。服用后可获得“快速治疗2”效果（24点医疗点），持续一轮，并且当天无需饮食。  
· 若结果为2，目标获得一颗苹果。服用后可获得“异常点治疗1”效果（24点医疗点），持续一轮，并且当天无需饮食。  
同一目标在一轮内无法同时获得两种不同的礼物效果。

【圣夜星陨】  
“孩子们别怕，圣诞老人来了。”  
作为带来礼物的象征，你获得一点机运骰加值。你可以在战斗外选择一个方向将破军圣诞掷出，随后这一武器将会如同流星一般消失在天际，该武器至多向目标方向飞行5000公里，这需要花费10分钟-1小时的时间。当飞行结束后破军圣诞会如同真正的圣诞树般稳稳插在落点，并自动开始播放欢快的圣诞乐曲。  
当破军圣诞落地后你可以花费10分钟-1小时不等的时间蓄力跳跃，如同流星划破天空，以经典的美式超级英雄姿态精准地降落在破军圣诞的旁边。此效果视为一个B级传送类旅行效果。  
一切都是为了更好的送礼物。
