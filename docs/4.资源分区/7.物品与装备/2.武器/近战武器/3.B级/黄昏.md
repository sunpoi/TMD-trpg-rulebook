# 黄昏

黄昏  
价格：B+3000  
分类：斧  
武器伤害：20L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】  
武器体积：4  
伤害类型：物理-挥砍/火焰（取优）  
描述：模仿落日光辉的火焰魔法武器。模仿自警团发现的古代武器创造。挥舞后会从刃的位置放出火焰。  
效果：持有者可以消费一个迅捷动作启动效果，持续一场景；持续时间内，持有者使用黄昏进行的近战触及距离提升【力量属性】\*1米，同时可以对触及距离内的所有敌对单位造成伤害，并在攻击上获得【湮灭】特性。
