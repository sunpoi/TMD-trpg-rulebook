# 赶山鞭

  

赶山鞭  
  
介绍：  
始皇作石桥，欲过海观日出处。时有神人，能驱石下海，石去不速，神人辄鞭之，至今悉赤。阳城山上石，皆起立东倾，如相随状，至今犹尔。  
价格：B+2000点  
赶山鞭具有16L武器伤害，轻灵与3点破魔，3点破甲、触及范围15米。  
赶山：使用者可以以一个整轮动作使用赶山鞭对其攻击范围内的一座山进行一次攻击，若攻击出目\*5的总数大于山的体积，则你视为将山一鞭赶开，使其移动5+检定出目米。  
填海：使用者可以以一个整轮动作使用赶山鞭对其攻击范围内的一片水进行一次攻击，若攻击出目的总数大于这片水的体积，则你视为制造出大量土石，将这片水填成陆地。
