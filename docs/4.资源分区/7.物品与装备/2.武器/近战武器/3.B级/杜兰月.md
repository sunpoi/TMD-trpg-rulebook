# 杜兰月

  

杜兰月  
  
B+2000  
这柄双手大剑的黑色剑柄上镌刻有三枚十字花纹，十字型的剑格下是隐约散发金色光芒的剑身。当你紧握住这柄剑时，能感受到一股坚定的意志从中传来：它绝不容许自己的主人轻易受伤！  
武器类型：重剑  
武器伤害：14L  
特性：【格挡6】、【看破】、【双手】、【沉重】  
恪卫：持有者将常态处于格挡状态，当持有者受到攻击，其先攻值会在一轮内提升10点，若持有者的生命值已经没有完好，则再提升沉着属性点先攻。
