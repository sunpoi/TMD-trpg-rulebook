# 海中雷电

海中雷电

B+2000

我曾见过天空厌弃之地，那里的雷电永不停息，  
此乃其根茎与鲜血，其枝叶与灵魂。

特殊的，这把武器需要使用电鳗进行换购，在使用电鳗进行三场影片之后，可以使用标注的支线换购海中雷电。

这把武器在电鳗的基础上，武器伤害上升至18L，具有【轻灵】。

从正面看去，这仅是一条普通的电鳗，若是从侧面看去，这是一段闪电，而若是从另外一个角度看去，这是一把长矛，不管如何，他依旧使用鞭作为专业。

反叛者：  
“你手中的闪电可不是孤品，持矛者，  
大海将与天空同级。”

这把武器将造成雷击伤害，并且其触及范围扩大20M，在你使用它进行攻击时，会附带等量的麻痹异常点数，正常豁免，同时受击者需要以反射豁免对抗出目，否则陷入B级环境来源的【潮湿】效果中，持续一场景。
