# 单分子纤维

单分子纤维  
价格：B+2000  
分类：线（鞭）  
武器伤害：20L  
武器特性：轻灵，高速1  
武器体积：2  
伤害类型：物理-挥砍  
描述：可以将这个世界上大多数物质给分断的强韧纤维，熟练的操作者可以只动一根手指头，就将敌方解体殆尽。  
效果：该武器的触及可以延长等同你【传奇敏捷】\*10米的距离；当你使用该武器进行攻击时，附带【接触攻击】与【幻灭】特性。
