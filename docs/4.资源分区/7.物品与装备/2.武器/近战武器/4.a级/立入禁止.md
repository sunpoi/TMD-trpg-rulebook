# 立入禁止

立入禁止  
A+4000  
特异本质  
这是一杆……禁止标识？银色的长柄，甚至是空心的，简单的焊接，将红色的禁止标识连接在杆子上，完全就是一杆意味着此路不通的禁止标牌，唯一不同的是，这杆立牌无论如何都不会被破坏，仿佛，这就是"禁止"本身。  
武器伤害：34L  
专业：斧  
类型：斧-长柄  
体积：6  
基础：【破甲7】【长柄】【双手】全力攻击额外失去1基础防御DP+1  
特性：【此路不通】【行动禁止】  
【此路不通】：以一个自由动作将立牌插在脚下，红色的禁制会瞬间包裹整个战斗场景，从高空俯瞰，就像是一个巨大的圆形，包裹整个场地，中间有一条横线；已经加入这场战斗的单位会清晰地感觉到自身处于「战斗场地」中，而战斗场地的范围则是「包括所有战斗单位在内的共同所在场景」，那之后，所有能够抵达「战斗场地」的「通道」都会凭空出现一个或多个红色的禁止立牌，战斗场地内所有能符合「道路」这一概念的事物两旁会每隔半米出现一个禁止立牌；这些立牌不会影响道路的通行性，也不会真正地导致遮挡，对于战斗单位并不会产生任何实际上的效果，但是，所有并未在「战斗场地」展开前加入战斗的普通人和生物（通常以无支线强化为判断）会在靠近这些通道时优先注意到这些禁止标识，他们会下意识认为不能靠近，转而放弃进入「通道」，以此避免普通人进入「战斗场地」；而那些在「战斗场地」展开前就已经在其中的普通人，也会避免继续靠近战场，这些影响是源自于人类对「禁止」这一标识的共同潜意识本能，通过影响潜意识来让普通人不想靠近战场，他们在被提醒前不会注意到这件事，也不会在意突然出现的禁止立牌的合理性，但如果有人进行提醒，被影响的普通人也会意识到这件事，这是一个无支线等级环境来源的思维暗示效果。  
展开「战斗场地」后，可以用自由动作将该武器再拔出来，如常进行使用，而场地变化则会保持到战斗结束。  
只要身处展开禁制的「战斗场地」中，以该武器发动的攻击加骰都将提升一级（至多8加骰）。

【行动禁止】：这个能力是通过对于「禁止」这一概念的延伸，最基础的禁止概念一直附着在这把武器上，所以，当这把武器进行攻击时，承受攻击的敌人需要对这次攻击出目作为dc进行反射豁免，豁免失败将会承受A级时空来源的【迟滞】负面状态，而如果进行这次攻击时，你与攻击目标都处于「战斗场地」中，则这次攻击会额外附带有A级时空来源的【耗竭】负面状态，反射豁免伤害值，失败则在下一轮开始时失去10点AP。  
而更多的延伸则则体现在方方面面，当你展开了「战斗场地」时，这把武器可以对「自身敏感范围内处于战斗场地内的」某一战斗单位进行攻击，由于这把武器的本质与「战斗场景」中的其他标识相同，都是一杆禁止立牌，所以当你试图在禁制环境内对目标进行攻击时，目标身边最近的标识也会如被你挥动进行攻击般，以那个最近的标识对目标造成伤害，就如你的触及范围被延伸到了整个敏感范围一般。
