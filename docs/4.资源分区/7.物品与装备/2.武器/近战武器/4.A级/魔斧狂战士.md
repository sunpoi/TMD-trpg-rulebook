# 魔斧狂战士

魔斧狂战士  
价格：A+4000  
分类：斧  
武器伤害：30L  
武器特性：【湮灭】【全力攻击额外失去1基础防御 DP+1】【攻击额外获得2个附加成功数】  
武器体积：4  
伤害类型：物理-挥砍  
描述：一直渴求血、 在疯狂和清醒之间徘徊的英雄的斧子。能得到可怕的战斗能力、但很少有人能控制。  
效果：  
【英勇一击】：当你在进行攻击前，可以消费一个移动动作进行一次向目标方向的移动（因此也能结合在冲锋中而无需独立消费移动动作，但会导致那次冲锋无法应用招式），在移动后你进行的第一次攻击具有【传奇】特性，并获得等同【力量属性】\*1点【破甲】。

【为血痴狂】：使持有者无法使用意志力、无法使用低于A级的需要动作使用的招式；持有者进行攻击并造成伤害后，需要在回合结束时进行一次意志豁免，DC为自身决心属性，失败则陷入无法分别敌我的【混乱】状态，自动攻击最近的目标。但若持有者的沉着属性不低于决心，则不会因此陷入混乱，并获得免疫不超过S级心灵来源效果的能力。
