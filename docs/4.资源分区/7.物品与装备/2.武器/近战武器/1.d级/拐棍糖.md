# 拐棍糖

拐棍糖

D+1000

武器伤害6L，武器体积3  
武器类型：棍

拐棍糖是一种具有标志性“J”形或手杖状的硬质糖果。最经典的是红白螺旋条纹相间，带有清凉的薄荷口味。它质地坚硬光滑，需要吮吸或含化。作为圣诞节的重要象征，其形状常被联系到牧羊人的拐杖或耶稣（J），红白配色则蕴含宗教意义。它既是节日里受欢迎的甜食，也是常用的圣诞树装饰和节日礼物。

当你使用这根拐棍糖击中敌人时，你可以轻轻敲醒对方沉睡的孩童心思，他使用意志豁免对抗伤害出目，若失败，则他在该场景中会避开你作为攻击目标（除非你已经是最后的目标），这是一个D级心灵来源的胁迫效果。
