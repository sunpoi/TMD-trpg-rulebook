# D级白刃类-棍

白刃类-棍

C  
初岁十节，周始朝露  
价格：D+500  
本质：魔幻本质  
分类：长棍  
武器伤害：6L  
武器特性：【长柄】【格挡1】【法破】  
武器体积：3  
伤害类型：物理-钝击  
描述：用一根一年长出十节的青竹所制成的竹棍，即使年岁虽短，但朝露之气已入，是于雾瘴之处行商者的必备物品。  
效果：只要将初岁十节，周始朝露携带在身上或手持便会生效（这意味着只要带着便会生效，但收纳进储物空间不能生效），携带者将被朝露之气包裹，不受潮气侵入，这是一个免疫C级潮湿的效果。

D  
钉棍棒  
价格：D+1500  
本质：科技本质  
分类：棍  
武器伤害：6L  
武器特性：【长柄】【格挡1】  
武器体积：3  
伤害类型：物理-钝击  
描述：钉着钉子的棍棒，也可以说是长满尖刺的棍棒。战争爆发之后，临时赶出来的武器。  
效果：使用该武器进行攻击时，附带1点不可豁免的流血/剧痛不良点数，可叠加。必须在购买时选择造成的是哪一种，造成剧痛不良点数的版本价格会下降500分。  
【特殊的是，该武器可以在购买时将伤害类型永久性改为物理-穿刺，而不是物理-钝击。】

G  
拐棍糖  
价格：D+1000  
本质：特异本质  
分类：棍  
武器伤害：6L  
武器特性：【长柄】【格挡1】【内在孩童】  
武器体积：3  
伤害类型：物理-钝击  
描述：拐棍糖是一种具有标志性“J”形或手杖状的硬质糖果。最经典的是红白螺旋条纹相间，带有清凉的薄荷口味。它质地坚硬光滑，需要吮吸或含化。作为圣诞节的重要象征，其形状常被联系到牧羊人的拐杖或耶稣（J），红白配色则蕴含宗教意义。它既是节日里受欢迎的甜食，也是常用的圣诞树装饰和节日礼物。  
效果：  
【内在孩童】：当你使用这根拐棍糖击中敌人时，你可以轻轻敲醒对方沉睡的孩童心思，他使用意志豁免对抗伤害出目，若失败，则他在该场景中会避开你作为攻击目标（除非你已经是最后的目标），这是一个D级心灵来源的胁迫效果。

L  
李鹏的臂骨  
价格：D+500  
本质：特异本质  
分类：短棍  
武器伤害：5L  
武器特性：【武之旋风】【僧之智慧】  
武器体积：3  
伤害类型：物理-钝击  
描述：【李鹏者，五樱寺僧人也，以勇武智慧为乡人所敬。人类矮人皆呼为己族，其出身竟不可考。至鹏丧后，遗物多传有魔法。  
李鹏青年时，携锦帛抵市，欲易粮米还寺，路遇食人魔御盲地狱犬当道，胁曰：“弃汝锦帛，方可得生。”鹏笑对曰：“人岂畏犬乎？”食人魔怒，乃纵犬欲伤李鹏。鹏纵跃至树梢，掷袍蒙食人魔，地狱犬以食人魔为李鹏，竟反噬。李鹏乃归。  
李鹏老年时，乡人为盗贼所困，求于李鹏。李鹏携手杖见贼于黄昏，劝其归善。贼见李鹏身形瘦弱，皆晒笑，以十字弓欲害李鹏。鹏以手杖拨挡来箭，皆不中，盗贼震惊。鹏复以手杖疾风连击，盗贼皆昏。鹏乃令贼首为骡马，载乡民钱粮而还。  
李鹏暮年时，为五樱寺主持。弟子求传其道，鹏笑曰：“子若路遇得道者，可击而杀之。”弟子方知道不可传。又复求其武，鹏曰：“吾臂为吾用，子若得吾臂，可尽其用也。”弟子思索良久，未明其深意。待李鹏丧后，竟盗其右臂骨，制双节棍，逃亡，其事终不可考。】

这时一把看起来相当鬼气森森的武器，李鹏的臂骨其实就是著名武僧李鹏的右臂骨，中间还有连接着筋肉，可以当作双节棍来使用。  
虽然式样恐怖，但李鹏的臂骨却并非邪恶物品，相反散发着宁静的感觉。  
效果：  
李鹏的臂骨可以以一个迅捷动作展开/收缩，初始为展开状态（3体积），收缩后为原体积的1/2。  
【武之旋风】：  
在使用李鹏的臂骨进行格挡时，可将臂骨舞之密不透风，从而击落箭矢。人物在在格挡时，额外获得仅对远程投射物攻击的3点格挡防御加值。  
【僧之洞察】：  
在战斗中，来自李鹏的武僧智慧将指引你战斗，你总是能在攻防转换间看破目标弱点，做出优秀的战术决定，这使得你在攻击中可额外获得2DP的洞察加值。

M  
马里奥的管钳  
价格：D+500  
本质：魔幻本质  
分类：短棍  
武器伤害：5L  
武器特性：【水管工之怒】【格挡1】  
武器体积：2  
伤害类型：物理-钝击  
描述：  
外形普通的管钳，上面附着一层火焰，也许是管钳正在愤怒。（这层火焰并非实际的火焰，但会让你摸起来暖暖的）  
这把管钳的主人很有名，他自称是个水管工，但他从不修管道，他只是在管道里钻进钻出，跳来跳去。整日忙着与恶势力战斗、拯救王国、寻找公主什么的。他有时会化装成青蛙、浣熊、甚至地藏菩萨。有时他还会放火球，扔锤子……但他从不！请注意，是“从不”用这件水管工必备的工具。于是，这把管钳愤怒了，它火冒三丈，它要证明管钳完全是可以参加战斗的！  
效果：  
【水管工之怒】：你可以使用一个标准动作，攻击某触及范围内目标的头部，这是一次特殊部位攻击，正常承受减值。这次检定不会造成伤害，而会将伤害的出数转化为晕眩点数。以标准形式豁免，同时附带1点不可豁免的麻痹点数。

Q  
齐眉棍  
价格：D+500  
本质：科技本质  
分类：长棍  
武器伤害：6L  
武器特性：【格挡2】【长柄】【齐眉】  
武器体积：3  
伤害类型：物理-钝击  
描述：这根长兵以白蜡杆制成，轻重合宜，粗有盈把，棍竖直与人眉高度齐，故得此名。  
效果：  
【齐眉】：使用本武器施展棍类技艺时，防御上获得4DP的器械加值。
