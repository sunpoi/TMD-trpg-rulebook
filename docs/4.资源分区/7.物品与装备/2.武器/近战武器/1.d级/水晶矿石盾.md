# 水晶矿石盾

  

水晶矿石盾  
D+500  
原典：盾之勇者成名录  
破碎水晶铸造的盾牌，带来了矿石加工的知识。  
类型：重盾  
武器伤害：4L  
体积：4  
专业：盾  
特性：【格挡6】【双手】  
【精通加工】：你的手艺检定获得1点附加成功。  
【矿物鉴定】：现在，你能够以一个迅捷动作选择视觉范围内的一个矿物，通过【智力+生存+求生】检定以对抗矿物的支线等级\*6，若目标能进行伪装检定，则可将伪装检定结果加入DC。当你对抗检定通过后，可以解析该矿物的大致特征、作用或是某一防御类型。这是一个D级别、信息来源的解析效果。
