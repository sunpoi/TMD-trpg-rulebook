# 多功能铲

多功能铲  
D+500  
武器分类：铲  
武器伤害6L，武器体积3，力量需求3

这是一把多功能的铲子，其铲刃左侧带锯齿，右侧则是刃口，因此你既可以用它锯木头，也可以拿它切菜做饭，  
在铲柄有一个钳子，可以轻易剪短铁丝网，  
使用机关将铲刃弯下，其更是能作为锄头使用，  
并且将其铲柄拆开，内里还有一根鱼线和一根能用来生活做饭的镁棒，拆下来的铲柄更是有着手电筒的功能，并且可以随时组装。

而在有着如此之多的效果之外，这把铲子甚至还能作为盾牌使用，  
如果作为盾牌，其将具有武器伤害2L，格挡6的效果，切换是一个迅捷动作，并且这把铲子具有【防弹】效果，没错，是这把铲子防弹而不是你。
