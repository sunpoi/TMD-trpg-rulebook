# 钉棍棒

钉棍棒  
价格：D+1500  
分类：棍  
武器伤害：6L  
武器特性：长柄，格挡1  
武器体积：3  
伤害类型：物理-钝击  
描述：钉着钉子的棍棒，也可以说是长满尖刺的棍棒。战争爆发之后，临时赶出来的武器。  
效果：使用该武器进行攻击时，附带1点不可豁免的流血/剧痛不良点数，可叠加。必须在购买时选择造成的是哪一种，造成剧痛不良点数的版本价格会下降500分。  
【特殊的是，该武器可以在购买时将伤害类型永久性改为物理-穿刺，而不是物理-钝击。】
