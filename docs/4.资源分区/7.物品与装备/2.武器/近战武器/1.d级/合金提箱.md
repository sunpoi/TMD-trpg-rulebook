# 合金提箱

合金提箱  
价格：D+1000  
分类：轻盾  
武器伤害：2L  
武器特性：格挡8  
武器体积：3  
伤害类型：物理-钝击  
描述：作为盾牌使用的强化提箱，有些娃娃会用吉他盒或者伞来代替。  
效果：  
该武器可以用于储存物品，如同D级储物一样具有10立方米空间，不能装独立单位、放入/取出物品是一个迅捷动作。若将物品储存其中，可以正常通过非超自然的安检或类安检设备，无论其中是否有装着会触发警报类的物品。
