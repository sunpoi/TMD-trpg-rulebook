# 日轮刀

日轮刀  
D+1000  
魔幻本质  
出自《鬼灭之刃》，由不断吸收太阳光的星星绯矿石和星星绯砂铁锻造而成的刀，具有太阳的威力。  
锻造完第一个持剑的人会与日轮刀产生感应，使刀刃变色，当然我们亲爱的tmd主神决定由购买者自己决定刀刃所呈现的颜色。  
武器类别：武士刀  
专业：刀  
武器伤害：9L  
体积：3  
伤害类型：挥砍  
特性：高速1，【斩鬼】  
斩鬼：这把武器对【黑暗生物】进行攻击时额外具有1点熔毁。
