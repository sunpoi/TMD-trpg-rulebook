# 平头铲

平头铲  
价格：D+500  
分类：铲  
武器伤害：6L  
武器特性：长柄，格挡1  
武器体积：3  
伤害类型：物理-钝击  
描述：要埋死人的话，铲子绝对是不可或缺的。如果那些死人还在动的话，铲子也可以让它们乖乖躺下来。  
效果：使用该武器进行攻击时，附带1点晕眩不良点数，不可豁免，可叠加。
