# 孤城剑

孤城 D+750  
剑柄裹着白色的皮革，十字护手则是锯齿型城墙的形状，剑刃上也带有坚硬而有光泽的锯齿。  
孤城是一把具有8L，威猛2，高速1，力量需求3的精金长剑，当面对相应的伤害减免时视为精金材质。  
使用要求：非黑暗生物，剑3，社交3

  
不详征兆：若用这把剑砍中了一个无辜的生物（具体由ST裁定，一般指道德值7以上的生物），那么剑身将不会粘上血，持有者因这不道德的行为而心怀愧疚，受到5DP士气减值，亦无法发动领导攻击。  
直到使用者完成赎罪或将剑彻底丢弃，这个效果才会结束。这是一个D级诅咒效果。

  
专属剑技-领导攻击 D+500  
你勇猛地向对手发动攻击。当你攻击时，你发出呐喊宣示胜利已在眼前。你的攻击可以激励周围的同伴参与战斗，让他们重新焕发精力。  
在你使用普通近战攻击时，若攻击命中，则你周围5米的队友在1回合内对你击中的目标进行下次攻击时，可以获得4DP士气加值，与其他群体增益类加值不叠加。

  
花费C+1250升级，技能效果替换为：  
使用标准动作发动一次近战白刃攻击，若攻击命中，周围3.25米的队友在1个场景内对你击中的目标进行攻击时，可以获得6DP士气加值。
