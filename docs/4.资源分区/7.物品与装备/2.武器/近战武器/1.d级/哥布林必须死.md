# 哥布林必须死

哥布林必须死  
D+500  
魔幻本质  
描述：这是一把普普通通的双刃短剑，正反握都非常舒适，精心包养令它在经历了无数次战斗后锋锐依旧，当它的主人为了击杀哥布林而不断磨练武技时，这把平平无奇的短剑也拥有了自己的执念，执念之深，执念之切，所有拿上它的人都会因此动容，因此而改变。  
分类：匕首  
武器伤害：4L  
基础效果：【轻灵】【投掷武器】  
武器体积：1  
伤害类型：物理-穿刺  
特性：【哥布林必须死】  
【哥布林必须死】：仅在攻击哥布林时，这把武器增加1点器械加值，额外具有【高速5】【破甲5】，每具有1点传奇沉着，就在针对哥布林的攻击上额外获得1dp专业加值。

哥布林：该武器针对的目标【哥布林】可以被分类为，具有绿色或近似颜色，长耳朵，面貌恐怖，大多为秃头，性情邪恶，繁育力强的小型类人生物，或直接针对地精、哥布林等明确物种。
