# 李鹏的臂骨

  

李鹏的臂骨  
特异本质  
价格：D+500  
【李鹏者，五樱寺僧人也，以勇武智慧为乡人所敬。人类矮人皆呼为己族，其出身竟不可考。至鹏丧后，遗物多传有魔法。  
李鹏青年时，携锦帛抵市，欲易粮米还寺，路遇食人魔御盲地狱犬当道，胁曰：“弃汝锦帛，方可得生。”鹏笑对曰：“人岂畏犬乎？”食人魔怒，乃纵犬欲伤李鹏。鹏纵跃至树梢，掷袍蒙食人魔，地狱犬以食人魔为李鹏，竟反噬。李鹏乃归。  
李鹏老年时，乡人为盗贼所困，求于李鹏。李鹏携手杖见贼于黄昏，劝其归善。贼见李鹏身形瘦弱，皆晒笑，以十字弓欲害李鹏。鹏以手杖拨挡来箭，皆不中，盗贼震惊。鹏复以手杖疾风连击，盗贼皆昏。鹏乃令贼首为骡马，载乡民钱粮而还。  
李鹏暮年时，为五樱寺主持。弟子求传其道，鹏笑曰：“子若路遇得道者，可击而杀之。”弟子方知道不可传。又复求其武，鹏曰：“吾臂为吾用，子若得吾臂，可尽其用也。”弟子思索良久，未明其深意。待李鹏丧后，竟盗其右臂骨，制双节棍，逃亡，其事终不可考。】  
  
这时一把看起来相当鬼气森森的武器，李鹏的臂骨其实就是著名武僧李鹏的右臂骨，中间还有连接着筋肉，可以当作双节棍来使用。  
虽然式样恐怖，但李鹏的臂骨却并非邪恶物品，相反散发着宁静的感觉。  
  
李鹏的臂骨伤害5L，收起状态/展开状态体积1/2 ，专业分类为：短棍，造成严重伤害；具有【武之旋风】【僧之智慧】特性。  
  
【武之旋风】：  
在使用李鹏的臂骨进行格挡时，可将臂骨舞之密不透风，从而击落箭矢。人物在在格挡时，额外获得仅对远程投射物攻击的3点格挡防御加值。  
  
【僧之洞察】：  
在战斗中，来自李鹏的武僧智慧将指引你战斗，你总是能在攻防转换间看破目标弱点，做出优秀的战术决定，这使得你在攻击中可额外获得2DP的洞察加值。
