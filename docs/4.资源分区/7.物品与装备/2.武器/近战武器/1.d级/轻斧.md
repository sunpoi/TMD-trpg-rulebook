# 轻斧

轻斧  
价格：D+1000  
分类：斧  
武器伤害：7L  
武器特性：破甲5，高速2  
武器体积：4  
伤害类型：物理-挥砍  
描述：棒子前端附以厚重金属刃的武器。威力卓越，作为工具使用的评价也不错，关键在于其重量与轻盈手感的完美平衡性。  
效果：使用该武器进行全力攻击时额外获得1DP器械加值，同时最多无视2点全力攻击带来的减值。
