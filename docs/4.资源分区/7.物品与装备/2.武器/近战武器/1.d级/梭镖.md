# 梭镖

  

梭镖  
D+1500  
特异本质  
一种接以长柄的单尖两刃刀，木制的长柄上沾着新鲜的泥土，有些生锈的铁头上留着已经干涸的血迹。

武器类别：梭镖  
专业：枪  
武器伤害：7L  
体积：4  
特性：【格挡2】  
【慑敌】：持有梭镖进入战斗时，仅在第一回合，可以以一个标准动作选择7米扇形内的所有目标，对其大喊一句：“狗官，识得这把梭镖吗？”并进行一次攻击检定。目标需以意志对抗你的攻击检定，若失败，则目标只针对于你陷入D级心灵来源的惊惧，持续一轮。  
特殊的，若目标为欺压百姓的走资派、警察、地主阶级、官僚主义，则惊惧持续一个场景且来源等级提升为S。
