# D级白刃类-刀

白刃类-刀

J  
机械长刀  
价格：D+500  
本质：科技本质  
分类：刀/手枪  
武器伤害：刀/手枪 6L  
武器特性：【特殊武装】【能量体运用】  
刀 破甲2；手枪 高速5 短点射 装弹数11。  
武器体积：刀3/手枪2  
描述：这是一把充满科技气息的长刀，刀身轻薄微曲，单刃，仔细看可以看到刀背上的电路装置。  
效果：  
【特殊武装】：  
刀柄结构类似自行车，带有扳机。扣下扳机会像手枪一样顺着刀刃指向的方向发射子弹。  
使用枪部分进行攻击时，会带给对手D级来源的措手不及

【能量体运用】：  
使用时按下像是自行车手刹的东西，便会将装载进刀柄的能量模块发挥效用，使刀刃和子弹附带对应效果，一开始并不会自己装载模块。

附属物品：  
能量块  
价格：500分  
根据购买时选择两种能量攻击形式，同样的，一种能量攻击形式绑定一个攻击模式。例如：刀（火焰能量）枪（冰冻能量）。  
PS：禁止在同攻击模式下声明使用不是该模式下的能量攻击。例如：刀（火焰）打出冰冻伤害效果。

L  
黎明之刃  
价格：D+1500  
本质：科技本质  
分类：刀（鬼头大刀），仍使用【刀】专业  
武器伤害：7L  
武器特性：【锋利】【噬时】和【汲能】  
武器体积：6  
伤害类型：物理-挥砍  
描述：黎明之刃是一柄用Chronophagic（希腊语，意为“吞噬时间”）合金制成的鬼头大刀，作为钛帝国唯一昭和猛男【远见.阿兹纳布尔】的爱刀， 每当远见用它杀生时,他的寿命就会被大大滴被延长，因为这柄妖刀会将死者的生命力转移给其持有者，这也使得远见在经历漫长的时间后，依然能作为赤い彗星 活跃在战场上。  
黎明之刃是远见在追击兽人的过程中，不小心误入某个信仰中东萨满的化外小国的领土范围内找到的，并依靠这把武器击退了一群“凭空出现的怪异生物 。不过讲真，考虑到黎明之刃的制造年代和能伤害恶魔的实际效果，塔拉辛最好检查一下自己博物馆里是不是丢了点什么东西....  
效果：  
【锋利】：  
这把一人多高的武器是如此锋利，可以轻易切开岩石， 黎明之刃具有2点破甲。

【噬时】：  
这把古老的利刃隐藏着一个黑暗的秘密。其刀身是由(chronophagic)合金组成的——每当使用者杀死一条鲜活的生命时，受害者原本所剩的寿命便被偷走并叠加到了使用者本人身上，这使得远见持续的活了将近三个世纪。  
每当人物使用该武器杀死一名生物后，便获得该生物剩余的自然寿命，每部影片中你至多从此特性获得5年自然寿命。在战斗中，人物只要使用黎明之刃杀死任意一名生物后，还可立即获得B级的【免疫衰老】效果，该效果持续一个场景。

【汲能】：  
黎明之刃可以真正的伤害那些来自亚空间的怪异存在，伤害并汲取他们的能量，远见甚至凭借此刀硬刚几下恐虐大魔，让钛军借此侥得喘息之机。  
黎明之刃具有【法破】的特性，可以有效的杀伤那些非实体的存在，且若被攻击目标存在关键属性为全部心智属性的能量池时，则每次成功的攻击，都将使得该能量池的能量流失6点。

R  
日轮刀  
价格：D+1000  
本质：魔幻本质  
分类：武士刀  
武器伤害：9L  
武器特性：高速1，【斩鬼】  
武器体积：3  
伤害类型：物理-挥砍  
描述：出自《鬼灭之刃》，由不断吸收太阳光的星星绯矿石和星星绯砂铁锻造而成的刀，具有太阳的威力。  
锻造完第一个持剑的人会与日轮刀产生感应，使刀刃变色，当然我们亲爱的tmd主神决定由购买者自己决定刀刃所呈现的颜色。  
效果：  
【斩鬼】：这把武器对【黑暗生物】进行攻击时额外具有1点熔毁。

X  
楔丸  
价格：D+500  
本质：特异本质  
分类：刀（半太刀），仍使用【刀】专业  
武器伤害：6L  
武器特性：【格挡2】【精确招架】  
武器体积：2  
伤害类型：物理-挥砍  
描述：从主人龙胤神子·九郎处得到的刀，是苇名的庶家——平田氏的相传之物  
虽曾一度失去，但又重回狼的手中

楔丸的名字里包含着心愿  
即使忍者注定是杀人的宿命  
也不可舍弃仅有的一点慈悲之心……  
这心愿刀刃是否会体察到呢？

效果：  
【精确招架】：你在进行格挡时声明一个数字，这个数字不能大于此次格挡防御值。当敌方对你造成伤害时，若该伤害值等于你所声明的数字，则该伤害中有4点伤害会向下转化一级（韧性-严重-冲击）。  
每部影片结束时，如果你在影片内触发本特性向下转化伤害的效果，则本特性可转化的伤害点数提升2点。

Y  
妖刀  
价格：D+500  
本质：魔幻本质  
分类：武士刀  
武器伤害：6L  
武器特性：【妖刀】  
武器体积：3  
伤害类型：物理-挥砍  
描述：这是一把形如长钉的武士刀，装入漆黑如夜的刀鞘中，亮刀时会散发出暗色的气场。  
效果：  
【妖刀】：这柄妖刀中寄宿着恶魔的力量，需与其中的恶魔签订契约才能使用。在与恶魔签订契约时先投掷一枚40面自然骰，并将出目+60作为额外的肉体寿命。每次借助恶魔的力量进行的战斗结束时都会失去5年肉体寿命，当肉体寿命折损至无时立即死亡，这个效果不会被免衰老影响，也不会影响其他效果。当战斗结束时，如果你的寿命未耗尽，则已消耗的寿命恢复，且不会造成其他后续影响。  
当使用者使用这把妖刀进行攻击时，必须至少使用一个标准动作来借助恶魔的力量进行攻击，这代表着它不能使用其他招式协助攻击。它的攻击会附带重伤效果，重伤会导致被攻击者的伤势无法自动稳定，也无法使用3快疗及以下的快疗点治愈其受到的韧性伤害。被攻击者需以强韧豁免对抗那次攻击的伤害出目，若对抗失败，则会陷入重伤状态直至其经过一次急救处理并累计获得一次长休息才能解除这个效果。无论如何，重伤是一个D级创伤来源的效果。在一场战斗中，持有者在对同一目标进行过一次攻击后，与其签订契约的恶魔便会数一个数字，从4开始向下递减，当其数出1时，持有者可以在下个回合使用一个标准动作将妖刀归鞘，恶魔会数出0并从暗处现身，捕捉那个被攻击者，那个被攻击者需立即以意志豁免对抗持有者上一次攻击的伤害出目，若对抗失败，则立即被恶魔撕咬粉碎，这视为一个D级精神来源的残疾效果。妖刀随着持有者的等级提升，会为持有者提供至多等同于其支线等级\*2的先攻数（D2C4B6A8S10），这在画风表现上视为持有者的行动速度上升。

妖刀-村麻纱  
价格：D+500  
本质：特异本质  
分类：武士刀  
武器伤害：6L  
武器特性：【破甲1】【吞噬灵魂】  
武器体积：3  
伤害类型：物理-挥砍  
描述：附着怨灵的妖刀，其中有亲手弑杀亲生儿子的母亲与御宅族儿子的怨念，持有者会被其中的亡灵逐渐腐蚀灵魂，最终性格突变沦为可悲的二次元御宅族。  
效果：  
【吞噬灵魂】：  
每次攻击时，以一个自由动作调动附着的怨灵，使得这把妖刀展现出紫色的剑气，这次攻击因怨气的附着具有【法破】特性，并附带1点不可豁免的精神伤害。  
特殊的，在一场影片中因此造成的精神伤害总量若高于你自身的完好上限，则你将不可避免地转变生物类型为【御宅族】，这一转变持续一场景，表现为你将极度喜爱二次元，变得懦弱、社恐，只在谈论到ACG相关内容时突然恶疾发作般滔滔不绝地插入话题，在意识到做了什么后只会后知后觉地找个地缝钻进去，永久承受对于外貌高于6的可爱女孩子以及你喜欢的二次元角色的10点不可豁免的可叠加的魅惑点数，这些魅惑点数无法被任何能力改变，在对象额外施加更多魅惑点数前不会立即陷入严重、毁灭状态。

仪式隐骨刃  
价格：D+500  
本质：魔幻本质  
分类：短棍/武士刀  
武器伤害：刀：6l/棍5L  
武器特性：在短棍形态下具有【格挡1】  
在武士刀形态下具有【破甲1】【鲜血狩猎】【法破】  
武器体积：刀3/棍1  
伤害类型：刀形态：物理-挥砍/棍形态：钝击  
描述：一把由松木做成的刻有大量咒文的装饰品，但其异常坚硬到能当短棍使用...  
但如果将短棍的一端扭转一次，该物件真正的形态就会出现，一把无刀谭的骨质武士刀，上面刻满了诅咒的符文。  
想要将该武器转换形态需要一个迅捷动作来扭转棍部位的一端来进行。  
效果：  
【鲜血狩猎】：当使用者的完好生命值一半被严重填满时，流出的生命将会包裹住刀刃部分将其变为生命的血红色，咒文将会发出不详的紫光，此时你立即进行一次D10骰并将数值加入先攻。
