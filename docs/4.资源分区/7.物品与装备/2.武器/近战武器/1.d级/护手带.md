# 护手带

  

护手带  
价格：D+500  
分类：拳套  
武器伤害：2附加  
武器特性：【格挡5】  
伤害类型：物理-钝击  
体积：1  
描述：缠在手上的护手绑带，用于保护持有者的双手应对激烈的战斗。  
护手：使用本武器进行肉搏格挡时，额外获得5点格挡防御（已计入）。  
肉搏武器：基于天生武器的伤害由冲击转为严重。
