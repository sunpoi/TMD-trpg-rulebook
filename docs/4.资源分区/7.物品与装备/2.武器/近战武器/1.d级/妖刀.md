# 妖刀

  

妖刀  
D+500  
魔幻本质  
这是一把形如长钉的武士刀，装入漆黑如夜的刀鞘中，亮刀时会散发出暗色的气场  
这柄妖刀中寄宿着恶魔的力量，需与其中的恶魔签订契约才能使用。在与恶魔签订契约时先投掷一枚40面自然骰，并将出目+60作为额外的肉体寿命。每次借助恶魔的力量进行的战斗结束时都会失去5年肉体寿命，当肉体寿命折损至无时立即死亡，这个效果不会被免衰老影响，也不会影响其他效果。当战斗结束时，如果你的寿命未耗尽，则已消耗的寿命恢复，且不会造成其他后续影响。  
这柄妖刀的武器伤害为6L，分类为武士刀。当使用者使用这把妖刀进行攻击时，必须至少使用一个标准动作来借助恶魔的力量进行攻击，这代表着它不能使用其他招式协助攻击。它的攻击会附带重伤效果，重伤会导致被攻击者的伤势无法自动稳定，也无法使用3快疗及以下的快疗点治愈其受到的韧性伤害。被攻击者需以强韧豁免对抗那次攻击的伤害出目，若对抗失败，则会陷入重伤状态直至其经过一次急救处理并累计获得一次长休息才能解除这个效果。无论如何，重伤是一个D级创伤来源的效果。在一场战斗中，持有者在对同一目标进行过一次攻击后，与其签订契约的恶魔便会数一个数字，从4开始向下递减，当其数出1时，持有者可以在下个回合使用一个标准动作将妖刀归鞘，恶魔会数出0并从暗处现身，捕捉那个被攻击者，那个被攻击者需立即以意志豁免对抗持有者上一次攻击的伤害出目，若对抗失败，则立即被恶魔撕咬粉碎，这视为一个D级精神来源的残疾效果。  
妖刀随着持有者的等级提升，会为持有者提供至多等同于其支线等级\*2的先攻数（D2C4B6A8S10），这在画风表现上视为持有者的行动速度上升。
