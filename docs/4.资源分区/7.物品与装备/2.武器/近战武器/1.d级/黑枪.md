# 黑枪

黑枪  
价格：D+500  
分类：短枪  
武器伤害：5L  
武器特性：【轻灵】【格挡1】  
武器体积：2  
伤害类型：物理-穿刺  
描述：基本上当成长枪变短的版本，特色是可以进行简易自动攻击，而无需依赖使用者的力量。  
效果：使用黑枪进行攻击时，持有者可以不使自己的攻击DP而是使用黑枪的内置预设DP进行攻击，预设DP：25DP+2附加成功（已计入武器伤害）。  
每经历一场影片，黑枪的预设攻击DP增加1点。
