# 过期鲱鱼剑

过期鲱鱼剑  
特异本质

价格：D+1000  
分类：迅捷剑  
武器伤害：5L  
武器特性：轻灵，高速1  
武器体积：3  
伤害类型：物理-穿刺  
描述：一把一直潮湿的不锈钢制迅捷剑，有着强大的特异能力，被砍中的人都将会感受到腐烂的鲱鱼汁液进入自己的口腔与鼻子，尽管实际上并没有这种液体  
效果：使用该武器攻击命中后，额外对目标造成2点不可豁免的心灵影响来源的恶心异常点数。
