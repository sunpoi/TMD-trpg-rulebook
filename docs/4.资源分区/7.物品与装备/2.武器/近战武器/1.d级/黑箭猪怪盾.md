# 黑箭猪怪盾

  

黑箭猪怪盾  
D+500  
原典：盾之勇者成名录  
以白线勾勒的黑箭猪怪为装饰的黑色圆盾。吸收黑箭猪怪背部背的刺解放的形态，带给了持有者快速行动与反伤的能力。  
类型：重盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡7】、【双手】  
持有者的移动速度提升10米  
持有者若处于格挡状态，则敌人以近战攻击命中你时，会受到1点不可豁免的穿刺严重伤害。
