# 初岁十节，周始朝露

初岁十节，周始朝露  
魔幻本质  
价格：D+500  
分类：长棍  
武器伤害：6L  
武器特性：长柄 格挡1 法破  
武器体积：3  
伤害类型：物理-钝击  
描述：用一根一年长出十节的青竹所制成的竹棍，即使年岁虽短，但朝露之气已入，是于雾瘴之处行商者的必备物品。  
效果：只要将初岁十节，周始朝露携带在身上或手持便会生效（这意味着只要带着便会生效，但收纳进储物空间不能生效），携带者将被朝露之气包裹，不受潮气侵入，这是一个免疫C级潮湿的效果。
