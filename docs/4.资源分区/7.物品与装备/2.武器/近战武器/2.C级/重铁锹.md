# 重铁锹

重铁锹  
价格：C+1500  
分类：棍  
武器伤害：11L  
武器特性：长柄，格挡2  
武器体积：4  
伤害类型：物理-钝击  
描述：一体成型金属块制成的沉重棒子，不一定必须是工具，也可以做为攻击能力强大的金属棍棒统称。  
效果：使用该武器进行攻击时，附带等同你自身【传奇力量-1】\*5点威猛，计算目标防御后若该效果提供的威猛仍有剩余，则剩余威猛转为攻击上的等量器械加值。
