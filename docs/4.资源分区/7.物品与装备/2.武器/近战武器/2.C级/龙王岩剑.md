# 龙王岩剑

  

龙王岩剑  
C+2500  
魔幻本质  
  
蕴藏初始雷电的碎石刺剑，蕴含着天空之城的古龙“普拉顿桑克斯”其中片鳞半爪的力量。  
是上没有永恒生命、渺小的龙后裔们，所不具有的强大的威力。  
未知的岩石组成四棱椎体样貌重型刺剑，相比其他刺剑更加粗大，也更长  
  
分类：重剑  
武器伤害12L ，体积4  
  
专用战技：雷云化身  
  
整轮动作结合在一次冲锋中使用，将自身短暂化为闪烁这红色雷电的云雾，飘浮在空中，  
随后向前极速飞驰，将缠绕雷电的剑身贯穿敌人的战技。  
短暂化为雷电的身躯在本轮中视为【虚体】。  
冲刺而至的雷云将对敌人造成无法被盔甲抵挡的雷电伤害【接触攻击】。
