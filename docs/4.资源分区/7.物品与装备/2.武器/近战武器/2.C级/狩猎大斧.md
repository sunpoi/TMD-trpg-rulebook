# 狩猎大斧

  

狩猎大斧  
价格：C+1000  
分类：斧  
武器伤害：12L  
武器特性：【破甲2】  
伤害类型：物理-挥砍  
体积：4  
描述：一柄老旧的单手斧，在上面凝聚了不少血腥味。  
真相：黑色才是童话的内核，美好不过是掩饰。持有者在使用该武器进行攻击时，附带1点不可豁免的恐惧不良点数。  
  
猎狼人：进入影片后，持有者获得【猎人】身份，即狩猎怪物之人。通常这会附带一段记忆，以及足够你生存的金钱，具有过往经历，但不会带来其他好处。
