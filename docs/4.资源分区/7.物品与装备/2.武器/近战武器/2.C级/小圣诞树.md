# 小圣诞树

小圣诞树  
本质：特异本质  
价格：C+1000  
分类：长枪  
描述：一颗小小的家庭装饰圣诞树，机缘巧合之下融入了一个孩子的灵魂，于是他想给自己创造一个快乐的圣诞。  
伤害：3L  
效果：【长柄】、【双手】、【格挡2】、【破甲6】、【威猛6】、【圣诞快乐】、【圣诞老人】

  
【圣诞快乐】  
这颗圣诞树的使命就是创造一个快乐的圣诞节，但它是个武器，所以它不杀人就不快乐，不快乐就要杀人，而别人不让它杀它会很生气。  
当你拿起该武器，你将获得五点不可豁免的亢奋点数；而当敌人将你不留手的攻击出目抵消一半时，你将在接下来五个回合中，对该敌人进行攻击或对抗检定并使用意志力时，额外获得五点表现加值。  
“嘻嘻，我一定要快乐~”
