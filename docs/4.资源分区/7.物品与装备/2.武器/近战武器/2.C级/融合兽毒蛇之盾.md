# 融合兽毒蛇之盾

  

融合兽毒蛇之盾  
C+1000  
原典：盾之勇者成名录  
由矿洞深处奇美拉魔兽毒腺附加的盾牌，带给了持有者抗毒与施加毒素的能力。  
类型：轻盾  
武器伤害：6L  
体积：2  
专业：盾  
特性：【格挡8】  
剧毒耐性：持有者受到的毒素伤害减少3点  
形态转换：当持有者受到近战攻击时，可以反射动作支付3点魔力，将盾牌上的毒蛇活化，令攻击者受到2点不可豁免的毒素点数。
