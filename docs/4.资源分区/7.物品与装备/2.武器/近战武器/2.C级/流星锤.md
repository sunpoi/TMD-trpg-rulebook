# 流星锤

流星锤  
价格：C+1000  
分类：锤  
武器伤害：12L  
武器特性：重型武器，双手，威猛5  
武器体积：4  
伤害类型：物理-钝击  
描述：前端附有铁球的金属锁链，被缠上的对手会因此失衡。  
效果：使用该武器进行攻击时，触及提升等同你【传奇力量\*2米】；每轮一次，你可以使用一个自由动作进行准备，当下次该武器命中目标时，以攻击出目为DC，目标需要进行正常绊摔流程进行对抗，失败则进入伏地状态。
