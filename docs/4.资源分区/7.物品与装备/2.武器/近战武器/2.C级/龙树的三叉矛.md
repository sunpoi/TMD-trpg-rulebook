# 龙树的三叉矛

龙树的三叉矛  
本质：魔幻本质  
价格：C+1250  
分类：长枪  
描述：赋予炎龙，冰龙的龙息之力，据说是使用世界树干的一部分制作而成。考虑到大部分世界的树木本质上都是以世界树（或建木）为蓝本而诞生……在宣传方面不算是欺诈。  
武器基本属性：武器伤害11L  
武器特殊属性：【双手】【长柄武器】【格挡2】  
灼炎化/极寒化：迅捷动作使用，可选择进入灼炎或极寒状态，处于对应的状态下，将免疫C级及以下的炎热/寒冷环境带来的负面效果，持续一场景。  
处于灼炎或极寒状态下，你使用本武器进行的攻击获得5DP能量，以及1个附加成功。灼炎化状态下的伤害类型为火焰能量伤害，极寒化为寒冷能量伤害。
