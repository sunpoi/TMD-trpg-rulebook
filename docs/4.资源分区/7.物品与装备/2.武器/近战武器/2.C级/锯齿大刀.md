# 锯齿大刀

锯齿大刀  
价格：C+1000  
分类：刀  
武器伤害：11L  
武器特性：【破甲6】  
武器体积：3  
伤害类型：物理-挥砍  
描述：将牛刀与挫骨和凿孔结合的恐怖杀人凶器。可以把猎物切斩、串刺成死状凄惨的模样。  
效果：使用该武器进行攻击时，附带1点不可豁免的流血不良点数，可叠加。
