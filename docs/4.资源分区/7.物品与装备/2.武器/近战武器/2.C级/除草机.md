# 除草机

除草机  
价格：C+1000  
分类：刀  
武器伤害：12L  
武器特性：【重型武器】【破甲1】  
武器体积：3  
描述：让刀刃像螺旋桨般旋转，将所触之物皆尽粉碎。尽管作用范围很小，却是对活死人的最强破坏兵器。  
效果：该武器可以同时对触及范围内的所有敌对单位进行攻击；使用该武器进行攻击时，附带等同自身【传奇敏捷/2】\*5点多次攻击减值。
