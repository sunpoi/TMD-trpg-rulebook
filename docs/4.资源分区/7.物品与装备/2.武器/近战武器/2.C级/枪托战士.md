# 枪托战士

枪托战士  
C+1000  
体积5，双手  
这一把很好用的狙击枪，真的，他真的很好用，用过的都说好！！  
这是一把货真价实的狙击枪，虽然不知道为什么其主人给其强化时，强化的是枪托，和枪管硬度，但他这么做，一定有他那个大病吧  
这一武器具有两种形态，进行形态切换至少是一个迅捷动作。  
很好用的狙击枪  
武器伤害5L，专业狙击枪，高速5，破甲3，【毁灭】，双手，穿刺伤害，装弹数6+1

你看，这属性，是不是很强！  
不是，你轮其枪管砸人干什么啊！！！！  
这是狙击枪，不是烧火棍啊！！！  
不知名生物爆鸣！！！！

很好用的狙击枪（托）  
武器伤害12L，专业：枪托战士，威猛5，长柄 双手 格挡2，钝击，【毁灭】  
不是，你强化那些就是拿他当烧火棍用啊！！！  
你对得起你手上的狙击枪吗！！！  
不知名生物爆鸣！！！！  
不是，你怎么又抡起枪管了！！  
Ps：st选开，该武器的近战形态可以使用专业为戟的技艺

枪托战士大改造  

结实的枪管  
D+500  
更加结实的枪管，可以更加粗暴的使用哒  
狠狠的抡起它去砸点什么东西吧  
砸砸砸砸砸！！！  
经过强化，更加结实好用的枪管可以让手臂肌肉更好的发力，这也能给敌人带来精神上更加极致的享受  
枪托战士形态下武器伤害提升2L  
每次攻击会附带一点无法豁免的眩晕  

结实的枪托  
C+1000  
更加结实的枪托，给宝宝一个安心的睡眠  
现在，这把狙击枪在枪托战士形态下，攻击带有【余波】特性，强大的冲击力将对敌人体内造成严重的内伤，短时间难以通过常规手段回复，这意味着被其命中的敌人，会受到B级重伤效果影响  
轻轻敲醒沉睡的心灵，慢慢闭上你的眼睛～  

技艺:出其不意  
C+1500  
每个战斗场景限一次  
出其不意的攻击，往往能带给人惊喜，当然，对于敌人可能就是惊吓  
当你第一次抡起狙击枪朝着人砸去时，敌人往往会被这出其不意的一击打到手忙脚乱  
每场战斗限定一次，当你第一次抡起狙击枪进行攻击时，敌人往往会因为来不及反应，被你当头一棒  
你在本回合内的所有攻击都将附带B级措手不及，并且攻击还将额外附带5点威猛  
需要注意的是，和你战斗过，知晓你战斗方式的人，在下次面对于你的战斗时，不会受到该技艺的影响
