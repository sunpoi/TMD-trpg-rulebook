# 雾之刹那

雾之刹那  
本质：魔幻本质  
价格：C+1000  
分类：镰刀  
描述：雾界少女所使用的武器，她作为被称为瓦尔基里的存在，其职责本应是引导战死的勇士升入瓦尔哈拉之中。  
但对于少女来说，她所做的行为仅仅是无意义，且无差别的将他人杀害，只有这样才能抵达自己想要抵达的地方，也只有这样才能将友人救回。  
武器基本属性：武器伤害11L，破甲1  
武器特殊属性：【轻灵】  
刹那：仿若刹那的一击，生死只在一瞬之间。  
以该武器进行攻击，可以在攻击前与对方进行纯敏捷检定对抗，若对抗成功，则本次攻击取得自身敏捷\*1点高速。

  
断：雾之刹那是断送生命的武器，而并非引导他人的武器。  
因此，使用雾之刹那的攻击将附带等同持有者传奇力量/2点韧性伤害。
