# 月牙斧

月牙斧  
价格：C+1000  
分类：斧  
武器伤害：12L  
武器特性：【破甲3】【毁灭】【双手】【全力攻击时DP+1】  
武器体积：4  
伤害类型：物理-挥砍  
描述：长柄装上斧子的武器。使用双手挥舞、但在斧子中命中很高、拥有高暴击率。  
效果：持有这一武器进行攻击时，持有者的近战攻击触及距离提升等同【力量属性/2】米。
