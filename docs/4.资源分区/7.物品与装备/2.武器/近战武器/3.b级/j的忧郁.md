# J的忧郁

J的忧郁  
价格：B+3000  
分类：电锯  
武器伤害：21L  
武器特性：【异种武器】【重型武器】【幻灭】【威猛2】  
武器体积：5  
伤害类型：物理-挥砍  
描述：看起来就非常危险的，毁灭性级别的武器。链锯齿与齿轮什么的，似乎沾染着血渍。不论如何，它可以激发使用者的破坏冲动，是链锯中的链锯。  
效果：当投掷一次你的攻击检定时，不计算任何能力提升，且出数大于或等于总数的一半（不包含附加成功带来的数值），你可以选择触发【破坏冲动】的效果；持有该武器攻击时，你可以支付3点意志力，获得一次只能用于抵消重骰取劣的重骰取优效果，但每轮只能获得一次。

【破坏冲动】：该效果的那次攻击将获得等同【传奇力量+3】点攻击附加成功数与【力量属性】一半点【威猛】，若这些【威猛】在处理目标防御后仍有剩余，则这些剩余【威猛】转化为攻击检定上等量的器械加值。
