# 银之歌者

银之歌者  
价格：B+3000  
分类：剑  
武器伤害：18L  
武器特性：-  
武器体积：3  
伤害类型：物理-挥砍  
描述：非常漂亮的剑，就像是“白银骑士”一般，闪耀着银色的光芒，其护手设计成展开的羽翼状，又因切裂空气时发出的声音宛如歌声而得名。  
效果：  
【被动】：剑身蕴含的奇异能量使其无比锋利。使用银之歌者进行的攻击附带【智力】\*1点【破甲】。  
【主动】：长剑挥舞时发出的歌声蕴含着魔法力量。你可以使用一个移动动作高举该武器随之咏唱，使以你为中心【20+传奇风度\*5】米范围内的所有友方单位在防御上获得等同你【风度属性】一半的偏斜防御，并在意志豁免上获得【8加骰】，持续三轮。
