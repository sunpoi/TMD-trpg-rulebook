# 维度捕食者之牙

维度捕食者之牙  
B+2000  
分类：收割镰   
武器伤害：17L   
武器特性：【轻灵】【破甲1】  
伤害类型：物理-挥砍   
体积：2  
描述：散发着紫色微光的漆黑小镰刀， 时常有颤动的紫电闪烁，镰刀的刀刃不像是金属物质，而是一种生物的骨骼组织，这把武器针对生物体本身的伤害并不显著，但致命的部分在于，随意的划动便足以撕裂空间本身，通过空间锋利的边缘撕裂敌人。  
【捕食维度】：这种生物的本质是一种超越维度，以维度存在本身为食的高位生物，其利齿的本质就足够锋利到切割空间。当你以该武器进行攻击时，你免疫B级时空来源的效果；当你在攻击时身处于A级及以下的异空间中（或在异空间外针对异空间进行攻击），这次攻击将直接撕裂异空间与现实维度之间的隔阂，扯出一条持续一轮的，从异空间通往原本位面的通道。无论如何，被这把武器影响的个体将失去与空间本身的协调，其难以精确调控自身与空间的定位关系，在这把武器的触及范围内，所有B级及以下的传送词条将完全失效。
