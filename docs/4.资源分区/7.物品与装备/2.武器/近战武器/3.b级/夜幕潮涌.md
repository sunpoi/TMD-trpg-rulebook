# 夜幕潮涌

夜幕潮涌

魔幻本质  
B+2000

分类：匕首  
武器伤害：16L  
武器特性：【轻灵】【投掷武器】【破甲15】  
伤害类型：物理-穿刺  
体积：1

“”无心的月亮升起，海上的潮汐便不在出现。

勇者用夜幕杀死了月亮，带回了潮汐。”

——《月岛故事集：1》

夜幕潮涌是一把由陨铁打造成的附魔匕首，全身漆黑，没有护手环，仅有允许单手握持的刃柄，以及刻满符文的刀刃。

【侠客之夜】：  
夜幕将掩护你，你在黑夜环境下时，你将遁入夜幕，你具有B级魔幻本质反侦测能力。

“夜幕将他送进灯火通明的宅邸，残暴的月亮和星星正在狂欢。”

夜幕将与你同在，每日仅一次，你在黑夜环境下进行的冲锋攻击，其造成的伤害翻倍  
“他靠近，抬手刺向月亮，穿透他亮堂的礼服，穿透他坚硬的内甲，穿透他肥厚的脂肪，刺向心脏。”

“而后无归”
