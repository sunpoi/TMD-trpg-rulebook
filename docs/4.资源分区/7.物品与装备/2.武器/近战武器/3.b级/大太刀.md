# 大太刀

大太刀  
价格：B+2000  
分类：武士刀  
武器伤害：20L  
武器特性：重型武器，破甲10，8加骰  
武器体积：5  
描述：巨大的太刀，考虑到刀身的长度，显然并不是给正常人使用的独特武器。但如果你具有挥舞其的恐怖蛮力，在战场上将掀起血腥的风暴。  
效果：该武器可以同时对触及范围内的所有单位造成伤害，即单体攻击可以改为等同触及距离的范围攻击；这一武器造成的伤害具有B级别【重伤】效果。
