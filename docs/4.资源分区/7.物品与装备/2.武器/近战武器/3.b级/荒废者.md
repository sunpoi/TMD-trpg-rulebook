# 荒废者

荒废者  
价格：B+4000  
分类：镰刀  
武器伤害：20L  
武器特性：【重型武器】【威猛3】  
武器体积：5  
伤害类型：物理-挥砍  
描述：斩割伤口会开始腐败的大镰刀，虽然分量颇重但平衡性极强。来自异界、锋刃使用了名为荒废石的特殊材料。  
效果：  
【剧毒锋刃】：使用该武器进行的攻击，在命中后会造成A级毒素来源的重伤效果，持续存在一场景；当具有【腐败】点数时，这一武器造成的伤害类型转为腐蚀伤害。

【腐败之镰】：持有者可以使用一个迅捷+花招动作进行一次纯运动技能检定（本次检定无法计入除运动技能的数值与附加外的任何额外加值与额外附加，无法因任何效果调整出目），随后获得等同检定成功数\*1点【腐败】，多次检定仅取高处理；当攻击命中后，被命中的单位会获得等同【腐败】\*1点【腐蚀易伤】，这一易伤效果不会叠加，与腐败点数均持续存在一场景；若上述运动技能检定没有获得成功数，则会对自身造成等同运动技能数值的【腐蚀易伤】。
