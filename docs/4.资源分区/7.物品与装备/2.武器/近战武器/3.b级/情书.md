# 情书

【情书】  
类型：武器/拳套  
B+2000  
它穿过熙熙攘攘的人群，穿过桌椅与窗户的缝隙，穿过不理解的层层阻碍，最终抵达你的身边……  
接受与否，这份情感都已经传递。  
这是一双红黑色的露指手套，每只手套上的绳子都精细的绑成了蝴蝶结的样子，每一拳都是一份沉重的爱意。  
【情书】拥有6点附加成功 造成伤害由冲击转化为严重  
伤害类型：基于手部天生武器  
体积：1  
特性：【爱与恨】【最后的情书】【幕后的守望】  
【爱与恨】：未曾说出口的真心，爱与悔恨纷沓而至  
你的攻击将附带攻击伤害等量魅惑点数  
对于拥有对你所造成魅惑点数的目标，你的攻击检定加骰提升一级，至多8加骰  
但对于陷入你造成迷情效果或毁灭性的目标，该目标将优先攻击自己，其次是其他拥有你所造成魅惑点数的目标。  
【幕后的守望】：暴力，是否是爱呢  
当你在战斗中佩戴【情书】每持续一轮，那么其上就会有一枚红黑色的蝴蝶绽放，这会使得你攻击时额外获得2点威猛，上限2轮，战斗结束清空  
【最后的情书】：我还想拍，你的笑容  
你可选择打断性的调用下回合的一个标准动作进行一次类招架动作【反击】，受到攻击时立即进行反击，反击伤害等同于获得的招架伤害抵销数，这些伤害不可豁免，但可以被减免、吸收。
