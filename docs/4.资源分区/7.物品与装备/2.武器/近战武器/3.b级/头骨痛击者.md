# 头骨痛击者

头骨痛击者  
价格：B+3000  
分类：锤  
武器伤害：20L  
武器特性：【重型武器】【双手】【威猛3】【8加骰】  
武器体积：4  
伤害类型：物理-钝击  
描述：为了将魔物的肉块砸碎可怕重锤。因为粘附的肉片的灵魂、体力会提升、拥有高昏迷率。  
效果：使用该武器攻击命中后，目标需要使用强韧豁免对抗伤害出目，失败则承受差值+1点晕眩不良点数，至少造成1点晕眩不良点数（这1点晕眩可以正常被降低承受异常点数类效果降低），视为B级创伤来源的效果。
