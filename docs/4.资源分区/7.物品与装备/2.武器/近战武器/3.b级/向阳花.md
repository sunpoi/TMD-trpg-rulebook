# 向阳花

向阳花  
B+2500  
魔幻本质  
描述：没有什么过不去的坎，纵是无光的夜晚，也终会迎来黎明。  
倘若太阳尚未升起，我们该如何度过漆黑与寒冷？  
那就看看向日葵吧，它一生都在追寻着太阳，当太阳落下，便会注视着彼此。  
当你的心中有爱，就会成为他人的曙光。  
这是一柄体积4，武器伤害22L，具有【重型武器】【双手】【8加骰】特性，分类为重剑的向日葵，攻击时，造成物理-钝击而非物理-挥砍伤害。  
那向日葵摇曳着，金色的，温暖的，支撑着心与身的疲乏困顿，当你持有该武器进行意志豁免时，视为这次意志豁免具有【8加骰】，并且不再因承受满额的严重伤害而陷入昏迷。  
购买该武器后，可以选择一个“能够主动开关，可以对自身或他人生效”的治疗类能力，令那个能力的医疗点数池上限提升自身风度属性\*4点且该快速医疗点数提升1点。
