# 光之剑（激光剑版）

光之剑（激光剑版）  
价格：B+2500  
分类：长剑  
体积：3  
武器伤害：18L，破甲6  
武器属性：【能量武器】【法破】  
描述：某版本中勇者爱丽丝使用的光之剑，是货真价实将光能约束成剑刃的激光剑。  
对一些个体而言，这个版本的光之剑可能比激光炮要更糟糕。  
终结之光：以标准动作进行一次攻击，这次攻击具有【B级先制】并使得最终成功数将增加一半。激光会使被命中灼伤的部位难以再生，视为附带了B级【重伤】效果。  
激光剑：武器在攻击时获得【能量武器】特性。
