# B级白刃类-刀

B级白刃类-刀

A  
暗丸  
价格：B+4000  
分类：肋差（武士刀）  
本质：科技本质  
武器伤害：18L  
武器特性：【高速1】【夜杀】  
武器体积：2  
伤害类型：物理-挥砍  
描述：为了隐藏在暗夜中而制作的黑色刀身武士刀，王国应用这把武士刀使用的技术、造出了黑色刀身的短剑和单手剑。  
效果：  
【夜杀】：持有该武器时，若处于夜晚，则你获得B级自然本质的反侦测能力；当你使用暗丸进行攻击时，你的攻击附带B级措手不及效果，若此时目标对你无法定位，则你可以支付1点意志力，使该措手不及提升至A级；在处理武器分类时，肋差依旧可以作为武士刀使用。

B  
白龙  
价格：B+2500  
分类：武士刀  
本质：特异本质  
武器伤害：18L  
武器特性：【高速1】【破魔1】【法破】【破邪之力】【心无杂念】  
武器体积：3  
伤害类型：神圣  
描述：据说含有净化污染的白龙的魔力的武士刀。没人见过那条白龙的身影、仅被认为是传说存在。  
效果：  
【破邪之力】：持有者免疫不超过A级重伤的效果，持有者在使用白龙进行攻击时，可以使用决心作为主属性进行攻击。

【心无杂念】：持有者在决心相关检定上获得等同【决心/沉着（取低）】一半的器械加值；若持有者传奇决心不低于4，则还会获得决心相关检定上加骰提升1级（上限8加骰）的效果；若传奇决心不低于5，则上述器械加值改为【决心/沉着（取低）】等量而非一半。

D  
大太刀  
价格：B+2000  
分类：武士刀  
本质：科技本质  
武器伤害：20L  
武器特性：【重型武器】【破甲10】【8加骰】【巨刃】  
武器体积：5  
描述：巨大的太刀，考虑到刀身的长度，显然并不是给正常人使用的独特武器。但如果你具有挥舞其的恐怖蛮力，在战场上将掀起血腥的风暴。  
效果：  
【巨刃】：该武器可以同时对触及范围内的所有单位造成伤害，即单体攻击可以改为等同触及距离的范围攻击；这一武器造成的伤害具有B级别【重伤】效果。

M  
魔刀千刃  
价格：B+2000  
分类：刀  
本质：特异本质  
武器伤害：7L  
武器特性：【魔刀千刃】  
武器体积：2

描述：“只攻不守，天下无双。”  
这柄武器仅余下黑木断柄与紫纹残刃…

效果：  
【魔刀千刃】：购买者会在进入影片时立刻回想起自身内心最深处的回忆并处于情绪波动状态，接着该情绪会不可避免的向负面转化，此时购买者需要进行一个意志豁免，DC为自身的决心属性+1。若无法通过，则购买者承受差值的决心属性伤害；如果此时你周围还有其他人，你可能会对他们进行攻击等行为。  
每部影片购买者仅能进入一次魔刀千刃所造成的回忆中。  
当购买者成功通过意志豁免，则真正获得这柄武器的使用权。

在成功通过意志豁免后，魔刀千刃的基础面板和特性转化如下：

分类：刀  
本质：特异本质  
武器伤害：19L  
武器特性：【魔刀千刃】  
武器体积：3  
描述：这柄武器由嵌有紫色眼珠的黑木刀柄、三分之一刻有恶鬼纹路的玄色刀身以及三分之二无数碎片合成的刀刃所组成  
效果：  
【魔刀千刃】：这柄武器的刀刃十分容易碎裂，也因此持有者在以其进行格挡时，无法获得格挡所带来的相应好处。  
持有者在一场景内每实际消耗5点内力以此武器进行攻击，则这柄武器的破甲值在本场景内就提高3点，至多提高21点。  
持有者可将自身的内力灌注入这柄武器中，此时刀身的恶鬼纹路会变为蓝色。每轮消耗6点内力，这柄武器的触及范围会在内力的驱动下碎裂延长40米。当你的内力等级提升至A/S级，这柄武器的延长范围会提升至160/360米。  
在一次战斗中，若持有者在魔刀千刃中合计实际消耗了49点内力，则刀身的纹路会化为紫色，此时以魔刀千刃进行的攻击具有A级别措手不及，该措手不及不会无视闪避防御而转为无视目标的格挡防御，在表现上魔刀千刃化为碎片的刀刃绕过了对方用以格挡的武器，而直接对其肉身造成伤害。

魔刀血鲨  
价格：B+4000  
分类：武士刀  
本质：魔幻本质  
武器伤害：18L  
武器特性：【高速1】【嗜血刀】【钓伏杀】  
武器体积：3  
伤害类型：物理-挥砍  
描述：据说会吸食死者之血的魔刀。武士刀自身会渴求血、能够正确瞄准血管斩击。  
效果：  
【嗜血刃】：该武器具有持有者【传奇决心】的吸血，在攻击命中并触发吸血后，额外造成有等同该武器提供吸血/点韧性伤害，但需要每触发1点吸血才能相应造成1点韧性伤害。这一效果需要在你进行攻击前使用一个迅捷动作激活才会生效，持续一轮。

【钓伏杀】：在一次使用【魔刀血鲨】作为武器进行的攻击检定时，不计算任何能力提升，且出数大于或等于总数的一半（不包含附加成功带来的数值），则触发【钓伏杀】，触发【钓伏杀】会使本次攻击附带【幻灭】与【A级重伤】效果，重伤的持续时间为一场景。

X  
雪饮刀  
B+2000  
价格：B+4000  
分类：刀  
本质：特异本质  
武器伤害：16L  
武器特性：【破甲7】【寒刃】【白露】  
武器体积：3  
描述：  
南山巅上火麟剑，北海潜深雪饮寒。  
可怜两锋未缘见，雪刀封隐孤剑鸣。  
雪饮刀是武林中赫赫有名的神兵，为聂风所属聂家之家传宝刀，与南麟剑首“断帅”所持的火麟剑齐名。刀长三尺七寸，为天下间至寒之物，相传为女娲补天之时，遗留下来的四颗神石之一所铸成。挥舞砍杀之时，寒气森森，刀斩人身，霜寒染白，而不见人血。  
效果：  
【寒刃】：  
只要你的内力槽不为空，你进行的白刃攻击就附带你造成的伤害半数的冻结点数，标准豁免。并且，只要被你使用此刀，寒气便会渗入骨髓，游走经络，因此本刀还具有B级别的【重伤】特性。  
【白露】：  
女娲在舍弃未来铸成“雪饮刀”的补天神石“白露”时，曾言道：  
“去吧！就去人间寻找！但愿你石中之铁能被铸成一柄绝世神锋，但愿你最终能落在一个心地善良、愿为众生幸福而甘于作出牺牲的主人手上，把千千万万活在水深火热的人从苦难中拯救出来……”  
后辗转流入聂风之手，果如女娲所言，陪同聂风一起，为苍生而战。因此，一场影片一次，当你不具有任何完好生命值，并且你进行的战斗是为了对抗危害苍生的邪恶时，才可以使用。  
你消耗3意志，下一次攻击中，雪饮的武器伤害提升15L，破甲提升8，并且在这次攻击中具有3点附加成功。

Y  
妖刀·贯鞘  
价格：B+2000  
分类：武士刀  
本质：魔幻本质  
武器伤害：20L  
武器特性：【破甲5】【妖刀】  
武器体积：3  
描述：这是一把完全由骨骸打造的骨白武士刀，刀护如肋骨莲华，刀柄末端留有骨骼关节状，握时似有隐隐寒气入体，浑身发麻。

传说， 茨木童子的小指骨与牛鬼的一根腿骨都被用作了这把妖刀的锻造当中，锻造这把武器的僧人刀匠埋柳寺胧成在锻造完成后欣喜若狂，试着将这把刀收入专门以妖王的骨头和羽毛制作而成的刀鞘中。

后来，当人们在发现埋柳寺胧成已经七日不见踪影后，于锻刀房发现了他冰冷干瘪的尸体倒在地上，尸体呈现出收刀归鞘的姿态，但刀鞘早已破碎，露出森森白骨。

刀匠死了，死于归刀入鞘。

住持和心腹在研究尸体时发现，埋柳寺胧成死于体内的刀伤，并且所有的血液都被抽干一空，体表不存在伤口。无人能知晓真实原因，只知道这把刀不知为何破碎了刀鞘，最后，住持将其命名为，贯鞘。

这把刀被封存在了刀匠的住处，那里改为了一件小祠堂，这把刀就被镇在弥勒座下。

一百三十年后，当僧人的后人因战乱打开传说中的莲台……

那里早就空空如也。

效果：  
【妖刀】：这把武器的基础效果是，当你攻击先攻低于自身的目标时，额外提升破甲10。

代价是，使用这把武士刀时，你无法正常购买并装配刀鞘，若如此做，则下列效果将无法生效。

现在你可以直接用自己的副手当刀鞘(这也意味着副手无法装配其他物品)，归刀入鞘时，骨刃必须直接从你的手掌中穿刺而过，妖刀会以此汲取你的生命滋养自身，这一过程会使你承受不可避免的8点严重伤害(同时意味着生命值替换为构造值或者不具备血肉的个体无法使用该武器)，残忍的代价也会获得妖刀的回馈；若你如此做，你本回合在进行【拔刀检定】后发起的下一次攻击额外获得等同于这次拔刀检定成功数的破甲，获得的这些破甲在计算破防后仍有剩余则转变为等量的攻击上的器械加值。

若在此基础上，你以此方法归刀入鞘后，自身不存在完好生命值的情况下，且攻击目标先攻低于自身，此武器的攻击额外提升一级攻击加骰(至多8加骰)，并具有等同于耐力属性一半的【吸血】。

Z  
斩首大刀（改良版）  
价格：B+2500  
分类：大刀  
本质：魔幻本质  
武器伤害：18L  
武器属性：【重型武器】【双手】【忍刀】【凶刀饮血】  
体积：4  
描述：  
出自《火影忍者》世界观中的武器。为雾隐村忍刀七人众的七忍刀之一。除了巨大的尺寸和锋利的刀刃之外，还能够在杀敌之时汲取对手的血液，用于修复自身的刀身。  
然而主神所提供的武器往往极其难以在影片世界当中被破坏，因而轮回空间诗歌塔大能【逆位战车】针对斩首大刀的这一特性进行了改良，令其能够通过汲取敌人的血液增加自己的锋利程度。  
效果：  
【忍刀】：斩首大刀是出自于《火影忍者》世界中，忍者专用的武器，因而对于查克拉有着良好的适应性。  
如果你持有称号：木叶忍者中的下属技能“体术忍者”，那么你可以用一个自由动作，令原本用于强化你的肉身的查克拉流入斩首大刀，从而令所有“体术忍者”中注明“以天生武器进行攻击时”生效的效果在使用这把武器进行攻击时也可以生效。  
同时，你可以放弃体术忍者给予你的天生武器伤害提升，令这把武器的武器伤害提升你的查克拉能量池等级\*6L。由于给忍刀保持查克拉的注入的消耗更为巨大，因此如果要令这个效果生效，则你每回合必须消耗4点查克拉。  
同时如果你将“体术忍者”提升到了A级，则你也可以选择放弃“一回合一次，你可以额外进行一次不附带任何招式的以天生武器进行的攻击”的效果，转而选择一回合一次，你可以额外进行一次不附带任何招式的以具有【忍刀】的白刃武器进行的攻击。  
【凶刀饮血】：斩首大刀具有能够吸收对手的血液修复自身的能力，而在经过轮回空间大能的改造之后，这一能力改为可以以敌人的血液增强斩首大刀的锋利程度。  
从现在开始，当你以此武器进行攻击时，你每对对手造成3点实际损耗了生命值的伤害，则你可以视为获得了一层【饮血】。无论如何，你一回合最多只能堆叠12层【饮血】，同时【饮血】的堆叠上限为36层。  
根据【饮血】的层数，你获得如下收益：  
【饮血】累积到12层时，你以斩首大刀进行攻击时，攻击骰的加骰提升一级。  
【饮血】累积到24层时，你以斩首大刀进行冲锋白刃攻击时，不再需要承受冲锋减值。  
【饮血】累积到36层时，你可以每回合消耗8点查克拉，令你的身周缠绕着令人恐惧的恶鬼的气势。在恶鬼气势缠绕的期间，每回合一次，你以斩首大刀进行的白刃攻击的伤害提升一倍。
