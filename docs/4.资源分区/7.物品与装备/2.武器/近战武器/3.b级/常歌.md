# 常歌

常歌  
价格：B+3000  
分类：剑  
武器伤害：18L  
武器特性：【破甲1】  
武器体积：3  
伤害类型：物理-挥砍  
描述：据说是某位工匠为了所爱的蕾瓦蒂亚打造出的传说之剑的仿品，其威力在仿品中也是十分惊人的。  
效果：该武器可以使用风度作为主属性而非力量作为主属性进行攻击；每轮一次，若使用风度作为主属性攻击，在每轮攻击前持有者可以迅捷动作进行一次风度属性检定，本次检定可以正常计入属性本身提供的附加成功数与风度相关属性检定加值，随后获得等同这次检定出目\*1点的攻击检定上的器械加值，这一加值持续一轮。
