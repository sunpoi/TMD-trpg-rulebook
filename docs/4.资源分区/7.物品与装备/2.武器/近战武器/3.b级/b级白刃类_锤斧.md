# B级白刃类-锤斧

B级白刃类-锤斧

C  
苍白之月  
价格：B+3000  
本质：特异本质  
分类：斧  
武器伤害：19L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】【冻结夜空】  
武器体积：4  
伤害类型：物理-挥砍/寒冷（取优）  
描述：被称为冻结夜空之斧、在龙讨伐时发现。斧柄冰冷、因为没有使用者由自警团保管。  
效果：

【冻结夜空】：持有者可以消费一个迅捷动作启动效果，持续一场景；持续时间内，使用苍白之月进行的攻击附带等同伤害出目的冻结不良点数，强韧豁免。当攻击命中后，目标还需要使用反射豁免对抗伤害出目，失败则会陷入【极寒】状态，持续一场景。若需要处理来源，皆视为B级环境来源的效果。

【极寒】：这是一个类暴雪环境效果，处于该状态下的单位会减少50米视觉感知范围，承受的的燃烧点数每轮会降低1点，并具有【冻寒能量易伤2】；当受到苍白之月武器的攻击时，还会额外承受1点不可豁免的冻结不良点数。

H  
海盗船锚  
价格：B+4000  
本质：科技本质  
分类：斧  
武器伤害：20L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】【重锚】【晕陆症】  
武器体积：4  
伤害类型：物理-挥砍  
描述：古老海盜船的物品。顶端是刃所以能当武器、极重、命中精度也很糟糕。  
效果：  
【重锚】：持有该武器进行攻击时，可以通过缠绕在船锚一侧的铁链延长攻击距离，这会使海盗船锚的近战触及距离提升【力量属性】\*3米。  
【晕陆症】：每轮一次，当你使用海盗船锚进行攻击时，你可以宣言进行一次【挥击】，这次【挥击】中可以正常使用招式，攻击范围内的所有敌对单位，具有【破甲20】，同时攻击出目增加一半，但【挥击】只有50%的命中率，这意味着你在宣言【挥击】时需要进行一次1D2，若结果为2则自动失手，失手意味着这次攻击不会造成任何影响，动作也不会返还。

恨天无把、恨地无环  
价格：B+2000  
本质：魔幻本质  
类型：锤  
武器伤害：19L  
特性：【重型武器】 【双手】【威猛3】【双锤】【巨大的幻想】【神力的兵器】  
体积：4  
伤害类型：物理-钝击  
描述：自古以来，历朝历代都会出现这种人，天生神力、生撕虎豹，少部分会成为学识之人（孔），但大部分都成为了一员猛将，而这种人物一般都不喜欢太轻的兵器，而如意金箍棒只有一个而且归了大圣所有，其他人都只能退而求其次——一手提着一个大锤，双大锤挥舞遮天蔽日。而其中最佼佼的大力之人会看着天地叹息“恨天无把，恨地无环”。  
效果：  
【双锤】：购买此武器会同时获得两个大锤【恨天无把】【恨地无环】，你必须双手各握持其中一把才能正常使效果生效，这是一对双持武器，每把都仍然具有【双手】特性，在攻击时，可以额外攻击触及范围内另外一个目标。  
【巨大的幻想】：这两锤可不简单，不是正常兵器那种长杆带骨朵锤头的兵器，而是锤子占据整个武器的一半长度，另外一半为柄，以科学视角来看无法存在的兵器，本质源于人们对大锤的幻想。  
因为其巨大的体积使得其具有【碾压】特性并且武器伤害带来的破防为威猛。  
【神力的兵器】：正因为其幻想的本质，其重量和威力都符合人们对其的憧憬。使用本武器必须使用力量为主属性进行攻击，随着你的力量越发超凡，武器的威力也越发强大。  
该武器额外提升你【传奇力量】\*2L+1L的武器伤害，并且伴随你势大力沉的攻击，被你攻击的目标都感到手中武器发出颤抖需要以力量-武技-对应武器专业对抗攻击伤害，对抗失败则武器脱手，这是一个每轮一次的附带击落武器效果；除此之外，当你以该武器进行全力攻击时，不再减少基础防御。

黄昏  
价格：B+3000  
本质：特异本质  
分类：斧  
武器伤害：20  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】【日落】  
武器体积：4  
伤害类型：物理-挥砍/火焰（取优）  
描述：模仿落日光辉的火焰魔法武器。模仿自警团发现的古代武器创造。挥舞后会从刃的位置放出火焰。  
效果：  
【日落】：持有者可以消费一个迅捷动作启动效果，持续一场景；持续时间内，持有者使用黄昏进行的近战触及距离提升【力量属性】\*1米，同时可以对触及距离内的所有敌对单位造成伤害，并在攻击上获得【湮灭】特性。

S  
升阳  
价格：B+4000  
本质：特异本质  
分类：斧  
武器伤害：20L  
武器特性：【破甲2】【全力攻击额外失去1基础防御 DP+1】【日出】  
武器体积：4  
伤害类型：物理-挥砍/雷电（取优）  
描述：带有黄色的斧刀看起来就像日出所以得名，在古代遗迹发掘、如今保管在皇家博物馆。  
效果：  
【日出】：持有者可以消费一个迅捷动作启动效果，持续一场景；持续时间内，持有者的生理属性检定获得6点增强加值，在使用升阳进行攻击时，攻击获得【幻灭】与【余波】特性。

T  
头骨痛击者  
价格：B+3000  
本质：特异本质  
分类：锤  
武器伤害：20L  
武器特性：【重型武器】【双手】【威猛3】【8加骰】【粉碎头骨】  
武器体积：4  
伤害类型：物理-钝击  
描述：为了将魔物的肉块砸碎可怕重锤。因为粘附的肉片的灵魂、体力会提升、拥有高昏迷率。  
效果：  
【粉碎头骨】：使用该武器攻击命中后，目标需要使用强韧豁免对抗伤害出目，失败则承受差值+1点晕眩不良点数，至少造成1点晕眩不良点数（这1点晕眩可以正常被降低承受异常点数类效果降低），视为B级创伤来源的效果。
