# 零式斩舰刀

零式斩舰刀  
科技本质

D+1000  
武器分类：重剑（使用专业：剑）  
武器伤害：7L  
伤害类型：物理-挥砍  
武器射程：触及  
武器体积：见下  
这是一把尺寸巨大的实体战刀，以同名的特机用武器为原形，主神可以为轮回者提供适合单人使用的版本，其刀身溶入了具有延展性的强化合金，刀柄和各部位装备了姿势制御用喷口。但是，过于沉重的构造和庞大的体积也让一般人根本无法使用。  
特性：【自适应】【对舰兵装】【压制】【重型武器】【双手】【破甲2】

【自适应】  
本武器的体积等于购买者在购买时的体积+1，会因此提升力量需求与触及范围。

【对舰兵装】  
持有者可以用一个迅捷动作展开姿势制御用喷口，在本场景内，以斩舰刀攻击任何体积比你大的目标时，忽略其等于持有者传奇力量-1/点任意伤害减免能力，在对其进行冲锋时获得2DP器械加值。

【压制】  
攻击任何比斩舰刀体积更小的单位，你可以选择将攻击DP转换为附加成功，每3个DP转换为1个攻击附加。

该武器以逐级升级，同名特性效果保留，下不赘述。  
改良-三式斩舰刀

B+2500  
武器分类：重剑（使用专业：剑）  
武器伤害：19L  
伤害类型：纯能量  
武器射程：触及  
武器体积：见下  
特性：【自适应+】【对舰兵装】【压制+】【重型武器】【双手】【破甲7】【8加骰】

【压制+】  
攻击体积比战舰刀体积小的单位时，可将攻击上的Dp以每2DP转换1附加的方式转化为附加成功，转化的附加成功上限为目标的体积（持有者依旧可以使用更低级的效果），在以这种方式进行攻击时，你可以同时攻击等同于触及范围扇形的所有敌人，视为范围攻击。

【自适应+】  
可以使用自由动作提升战舰刀的体积，提升武器体积的不能够超过持有者力量属性。计算本武器力量需求时改为每1点武器体积只增加1点力量需求。
