# 青龙&朱雀

青龙&朱雀  
特异本质

C+1500  
武器分类：武士刀/肋差（使用专业：刀）  
武器伤害：11L  
伤害类型：物理-挥砍  
武器射程：触及  
武器体积：  
青龙-3  
朱雀-2  
描述：  
拥有双属性的武器，看起来像是锯齿一样的刀刃，有点吓人。附带冰与炎属性，攻击力和抗性都很优秀。  
特性：【青龙&朱雀】  
青龙-【破甲5】  
朱雀-【高速5】

【青龙&朱雀】  
青龙&朱雀是成对武器，这意味着你在购买朱雀&青龙后，将会获得一把武士刀和一把肋差，分别名为【青龙】与【朱雀】，区别是前者具有的是【高速5】，后者为【破甲5】，在处理武器分类时肋差依旧可以作为武士刀使用。  
当你使用这对武器其中一把作为主要武器攻击时，可以将持有另外一把的基础属性一半与所有特性加入其中，但只有主要武器自带的破防能力正常生效（青龙对应高速，朱雀对应破甲）；你可以使用一个花招动作激活武器属性，此时若你使用的主要武器为青龙则该场景内这对武器伤害类型转为寒冷，若为朱雀则转为火焰。

【青龙&朱雀可以作为两把武器处理，但在任何永久性导致武器损失/失去的效果中，青龙&朱雀只能视为一把武器处理。】

可差价升级为  
绝影&斩无  
特异本质

B+4000  
武器分类：武士刀/肋差（使用专业：刀）  
武器伤害：18L  
伤害类型：物理-挥砍  
武器射程：触及  
武器体积：  
绝影-3  
斩无-2  
描述：  
双持武士刀，散发着强大的气场。如“断绝阴影、斩断虚无”一般，隐藏着能撕裂一切破坏力的双刀，并让使用者的速度飞跃性的提高。  
特性：【绝影&斩无】【幻灭】  
绝影-【破甲6】  
斩无-【高速6】

【绝影&斩无】  
绝影&斩无是成对武器，这意味着你在购买绝影&斩无后，将会获得一把武士刀和一把肋差，分别名为【绝影】与【斩无】，区别是前者具有的是【高速6】，后者为【破甲6】，在处理武器分类时肋差依旧可以作为武士刀使用。  
当你使用这对武器其中一把作为主要武器攻击时，可以将持有另外一把的基础属性一半与所有特性加入其中，但只有主要武器自带的破防能力正常生效（绝影对应高速，斩无对应破甲）；你可以使用一个自由动作并消费2点意志力激活武器属性，随后在该场景内，若你使用的主要武器为绝影则你的攻击具有【B级措手不及】，若为斩无则具有【接触攻击】特性。

【绝影&斩无可以作为两把武器处理，但在任何永久性导致武器损失/失去的效果中，绝影&斩无只能视为一把武器处理。】
