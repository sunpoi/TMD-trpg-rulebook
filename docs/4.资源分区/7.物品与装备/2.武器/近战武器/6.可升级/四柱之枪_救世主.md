# 四柱之枪—救世主

四柱之枪—救世主  
魔幻本质

该武器以逐级升级强化，同名特性效果不变，下不赘述。

D+500  
概述：  
安息绝杰·玛温消失后遗留下来的武器，外表是一根金色的长枪，枪身刻有四颗鲜红的宝珠。传说被救世主刺穿的人，都会静止不动，陷入彻底的死寂。  
武器分类：长枪 （使用专业：枪）  
武器伤害：4L  
伤害类型：物理-穿刺  
武器射程：触及  
武器体积：4  
特性：【四柱】【格挡4】【破甲4】【双手】【长柄】

【四柱】  
在用本武器攻击时，可以以一个自由动作在身体周围幻化出另外三把四柱之枪与你一起协同攻击。在攻击时可以攻击除你攻击主目标外触及范围内的另外三个不同的目标。  
由于只是幻化，所以仍然视为一把武器，视为同一次攻击。

C+1000  
武器分类：长枪 （使用专业：枪）  
武器伤害：12L  
伤害类型：物理-穿刺  
武器射程：触及  
武器体积：4  
特性：【四柱】【格挡4】【破甲4】【双手】【长柄】【安神】 【9加骰】

【安神】  
一轮一次，在攻击命中后，敌方以本次攻击的伤害值为dc进行一次意志豁免，豁免失败会让敌方陷入短暂的困眠，这会令其损失2AP，这是一个C级心灵来源的【耗竭】效果。

B+2500  
武器分类：长枪 （使用专业：枪）  
武器伤害：20L  
伤害类型：物理-穿刺  
武器射程：触及  
武器体积：4  
特性：【四柱】【格挡4】【破甲4】【双手】【长柄】【安神+】 【8加骰】【散播福音】

【安神+】  
一轮一次，在攻击命中后，敌方以本次攻击的伤害值为dc进行一次意志豁免，豁免失败会让敌方陷入短暂的困眠，这会令其损失4AP，这是一个B级心灵来源的【耗竭】效果。

【散播福音】  
以该武器进行的攻击视为具有B级时空来源的传送攻击效果，并且获得使用者传奇决心\*2+2点破魔，若该破防属性结算后仍有剩余则等量转化为攻击上的士气加值，最多转化传奇决心\*2+2点。

A+4000  
武器分类：长枪 （使用专业：枪）  
武器伤害：32L+1附加成功  
伤害类型：物理-穿刺  
武器射程：触及  
武器体积：4  
特性：【四柱】【格挡4】【破甲4】【双手】【长柄】【安神++】 【8加骰】【散播福音+】

【安神++】  
纵享安息吧……一轮一次，在攻击命中后，敌方以本次攻击的伤害值为dc进行一次意志豁免，豁免失败会让敌方陷入短暂的困眠，这会令其损失8AP，这是一个A级心灵来源的【耗竭】效果。  
安息的力量会模糊他们对于自身力量的感受，忘却存世的烦恼，令他们无法以自我意志抗衡；  
在攻击命中后，敌方以本次攻击的伤害值为dc进行一次意志豁免，豁免失败者将无法使用意志力，无法施展低于A级的任何技艺，这是一个A级时空来源的【扼魂】效果。

【散播福音+】  
将安息的福音散播于整个世界吧，当你以该武器进行攻击时，触及距离视为等同于自身敏感范围。  
将武器递出，便会令其从身后刺出，令忙碌的心脏享受安息，这是一个A级的时空来源的传送攻击效果。

\====================  
如果你拥有C级及以上的训练【安息的试炼】，在将该武器逐升至B时你可以选择将其升级变更为以下形态，一经选择无法更改。

温柔之花—安息

B+2000  
武器分类：长枪 （使用专业：枪）  
武器伤害：0L+7附加成功  
伤害类型：物理-穿刺  
武器射程：触及  
武器体积：4  
在安息绝杰消失之后，你经过众人的推举成为了安息的继承者，代替安息的绝杰继续履行安息的试炼，但这真的是你期望的安息吗？  
本武器的外形变为两枚碧绿色的戒指，需佩戴在双手才能使用。本武器的攻击表现是：佩戴者使用语言和肢体行与对方交流、谈话，用【温柔】方式对待对方来尝试让对方打开心扉，让对方相信只要温柔对待他人，明天会变的更好，从而失去对我方的敌意。  
特性：【格挡2】【破魔3】【双手】【长柄】【安息】

【安息】  
使用该武器时，你的攻击具有时空来源的B级传送攻击，  
在使用该武器进行攻击时，若你在当前回合使用了【安息】魔力的基础用法，则可以使得这次攻击额外附带伤害等量B级影响心灵来源的精神束缚异常点数，以意志豁免；  
如果承受攻击的目标至少因此承受1点精神束缚点数，则目标在攻击上承受使用者传奇决心\*4点士气减值，持续一轮，这是一个B级影响心灵来源的效果。  
每轮一次，你可以以一个花招动作唤醒自身拥有的纹章，若如此做，每拥有一个可以获得【纹章】效果的C级或更高级技艺（差价升级则视为同一个），都将使得自身下一次进行的攻击额外具有1点附加成功，至多以此获得6点附加成功。
