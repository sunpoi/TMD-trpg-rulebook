# 猪猪锻造王

猪猪锻造王  
魔幻本质

概述：虽然叫做锻造王，但为什么武器本身和锻造毫无关系啦？！这一把大锤子怎么看都不算锻造锤吧喂！啊……据说这把武器的使用者是超棒的铁匠？所以叫做“锻造王”？  
……意义不明。  
总之，这是一把猪猪形状的大锤

该武器以逐级升级强化，下不赘述。

D级

D+500  
武器分类：锤（使用专业：锤）  
武器伤害：7L  
伤害类型：物理-钝击  
武器射程：触及  
武器体积：4  
描述：这是一柄双手大锤，以整块的粉紫色魔法宝石雕刻而成，曾是一个可爱且自大的天才锻造师的武器。锤柄是闪亮亮的紫金色金属，锤身则被雕刻成了一只宝石猪猪的样子，那可爱的猪鼻子其实蕴含着夸张的杀伤力，只是现在其中蕴含的力量没有被彻底激活。  
特性：【重型武器】【双手】【威猛5】【铁砧魔力】

【铁砧魔力】  
以该武器进行攻击时额外获得1点附加成功，并且具有【熔毁2】。

C级

C+1000  
武器分类：锤（使用专业：锤）  
武器伤害：15L  
伤害类型：物理-钝击  
武器射程：触及  
武器体积：7  
描述：随着创造力的不断累积，这柄巨锤逐渐展现自己原来的姿态，就连体积都变得夸张的大！猪猪的宝石眼睛都变得炯炯有神起来！在战斗中，那来自锻造之力的灼热总会让人心旷神怡。  
特性：【重型武器】【双手】【威猛7】【铁砧魔力】【创造之铁砧】

【创造之铁砧】  
现在以该武器进行攻击时附带1点创伤来源不可豁免的燃烧点数，同时D级的具有【熔毁2】替换为具有【熔毁5】。

B级

B+2000  
武器分类：锤（使用专业：锤）  
武器伤害：22L  
伤害类型：物理-钝击  
武器射程：触及  
武器体积：7  
描述：现在，这柄流淌着魔力的巨锤重获了自己真正的力量，猪猪的眼睛冒着光，猪猪的鼻子哼着热气！猪猪的屁股后面怎么在喷火啊？！原来是要将所有的敌人全都砸碎呀~！！  
特性：【重型武器】【双手】【威猛12】【铁砧魔力】【创造之铁砧+】【创能崩毁】【9加骰】

【创造之铁砧+】  
以该武器进行攻击时附带1点创伤来源不可豁免的燃烧点数，具有【熔毁10】。

【创能崩毁】  
类型：攻击技  
使用动作：整轮动作  
消耗能量：10魔力  
影响目标：单一目标

效果：进行一次冲锋攻击，这视为你将这把巨锤横在身侧，在迸发的粉紫色高温能量推进下冲锋，该次冲锋有效距离始取最大值计算。  
真正的锻造便是能肆意将盔甲与素材蹂躏的力量，其上缠绕的创造之力将使这次攻击视作【接触攻击】并带有【湮灭】，且加骰提升一级，至多为8加骰。  
在【创能崩毁】攻击结束后，你可以利用锤子剩余的惯性立即对目标进行一次不附带任何技艺的攻击。

特殊的，就算没有魔力能量池也可以购买并使用B级【猪猪锻造王】，但在使用【创能崩毁】时，魔力的消耗将转变为等量不可豁免的严重伤害。
