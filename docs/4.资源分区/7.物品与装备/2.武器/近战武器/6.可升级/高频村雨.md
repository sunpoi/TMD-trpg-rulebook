# 高频村雨

高频村雨  
科技本质

这是由一柄具有看似不详的红色刀刃的高周波武士刀与经特殊改造后可进行装药助推的合金刀鞘组合而成的武器。  
武器为差价升级，同名特性效果替换。

高频村雨  
价格：C+1000  
分类：武士刀  
武器伤害：10L  
武器特性：【神速居合】【杀戮波动】  
武器体积：3  
伤害类型：物理-挥砍  
描述：  
这柄武士刀无时不刻地渴望着鲜血以淬炼其锋芒。驭使这柄妖刀的前提-是持有者拥有能够驯服其杀戮冲动的雷霆极意。  
效果：  
【神速居合】：  
当你使用【起火拔刀】后，可以敏捷代替风度进行拔刀检定。  
【杀戮波动】：  
铸就村雨所使用象征不详的红色金属会影响刀刃上的高周波生成装置，进而产生不规律的高周波波动。若能洞悉其波动的本质并加以控制，武士所进行的居合斩将会变得致命而凌厉。  
当你的先攻高于敌人时，以此武器进行的拔刀攻击将会额外为攻击增加【拔刀检定成功数】点破甲；若是计算防御后破甲仍然有剩余，则你可以将剩余的破甲转变为攻击上的增强加值，至多转换这次拔刀检定成功数点破甲。

高频村雨  
价格：B+2000  
分类：武士刀  
武器伤害：19L  
武器特性：【神速居合】【杀戮波动】  
武器体积：3  
伤害类型：物理-挥砍  
描述：  
经过主神改造后的村雨，血红的刀刃即使在非战斗情况下亦会发出隐约的高周波振鸣。这柄桀骜不驯的兵器只会屈从于具有雷霆极意之人，若使用者能力不足，甚至完全无法发挥其作用之十一。此时的村雨在助推后刀刃上会附着有赤红色的电光。  
效果：  
【神速居合】：  
当你使用【起火拔刀】后，可以敏捷代替风度进行拔刀检定。  
【杀戮波动】：  
铸就村雨所使用象征不详的红色金属会影响刀刃上的高周波生成装置，进而产生不规律的高周波波动。若能洞悉其波动的本质并加以控制，武士所进行的居合斩将会变得致命而凌厉。被村雨所伤者，其伤口会如同刀刃鲜红侵染般显得醒目而狰狞。  
使用者可将武技技能加入拔刀检定中。  
仅当你先攻高于敌人时：  
1、以此武器所进行的拔刀攻击会额外附带有【拔刀检定成功数】点破甲。  
2、若是计算防御后破甲仍然有剩余，则你可以将剩余的破甲转变为攻击上的增强加值，至多转换这次【拔刀检定成功数】\*2点破甲。  
3、处于【杀戮波动】状态下的村雨所造成的伤害，治疗难度加深一级。

下属物品：

合金起火助推刀鞘

资源价格：D+500  
奇物类型：刀鞘  
特性：【起火拔刀】【殁血归鞘】【血振】  
描述：  
这件刀鞘的鞘口上装有一个特制的板机，当你扣动扳机时将会引爆装载在刀鞘底部的小型炸药，为居合斩提供强劲的推动力。同时，刀鞘体部还装载有逆向电磁线圈以帮助武士快速进行【归鞘】。  
效果：  
【起火拔刀】：  
在进行【拔刀术】检定时，你可以以一个迅捷动作扣动扳机起火助推，在计算这一次拔刀检定对先攻与攻击Dp的加值时，可额外计入你用于拔刀检定的属性上的附加成功。  
这之后的首次攻击中，由于装药助推的加速效果，该攻击获得额外的2点【高速】；若是你的先攻顺序高于攻击目标，则你的拔刀攻击在对手看来如同子弹一般无法躲闪，这次攻击获得的【高速】再提升6点。  
【殁血归鞘】：  
在你使用村雨进行【起火拔刀】并成功对敌人造成伤害以后，其波动的杀意将会侵染武士的心神，使得武士无法如往常一般执行平息心湖的【归鞘】。取而代之的是满怀着杀意冲动的【殁血归鞘】。  
在你进行【居合】后，如果这次攻击杀死了一名敌方单位，那么武士可以借助逆向电磁线圈以一个自由动作进行【殁血归鞘】，若如此做，则同一轮内进行的下一次攻击可以满足再次使用【居合】的条件。  
武士在进行【殁血归鞘】以后，该场景内武士所有与心智属性相关联的检定都会降低2dp，这是由于武士自愿接纳村雨剑刃中所蕴含的杀意而导致的D级别影响心灵来源的负面效果，若是武士免疫了该负面影响，则无法进行【殁血归鞘】。  
【血振】：  
当村雨归鞘时，刀鞘上的逆向电磁线圈会急速抽离刀身的鲜血将其雾化以防止血液侵蚀村雨刀刃的锋利，使得归鞘后村雨四周将会怵然爆散出一股血雾。
