# C级白刃类-刀

C级白刃类—刀

C  
除草机  
价格：C+1000  
分类：刀  
本质：科技本质  
武器伤害：12L  
武器特性：【重型武器】【破甲1】【除草】  
武器体积：3  
伤害类型：物理-挥砍  
描述：让刀刃像螺旋桨般旋转，将所触之物皆尽粉碎。尽管作用范围很小，却是对活死人的最强破坏兵器。  
效果：  
【除草】：该武器可以同时对触及范围内的所有敌对单位进行攻击；使用该武器进行攻击时，附带等同自身【传奇敏捷/2】\*5点多次攻击减值。

G  
鬼妖村正  
价格：C+2500  
分类：武士刀  
本质：科技本质  
武器伤害：9L  
武器特性：【鬼妖村正】【试炼】  
武器体积：7  
伤害类型：物理-挥砍  
描述：这是一把红色刀刃，黑色刀柄与刀鞘，刀身上闪烁着电光的高频武士刀，即使是最强大的怪物也能干净利落地切碎。  
但此刻，鬼妖村正的力量还未被唤醒，完成试炼才能逐步解封它的力量。

效果：  
【鬼妖村正】：持有鬼妖村正会让使用者的精神和身体承受负担，在此阶段下，持有鬼妖村正会占据使用者的2点意志上限和1点血量上限。  
使用鬼妖村正进行的攻击会以极快的速度再劈砍、上挑一次，这意味着在进行不附带任何招式的攻击时，鬼妖村正还会至多再使被攻击者承受2点多次攻击减值。同时，使用鬼妖村正进行不附带任何招式的攻击还会在每次成功命中的攻击结束时积累1点武器能量，武器能量无上限，且会在场景结束时自动清空。  
鬼妖村正会随着使用者完成试炼和达到条件而觉醒力量。

当使用者完成了3项试炼，且最高内在支线等级在持有鬼妖村正期间达到了B级，鬼妖村正将升至B级。  
鬼妖村正的武器伤害提升30L，体积提升12点  
同时，使用鬼妖村正时将立即额外再获得剑气波、升龙斩、下坠击三个招式。

剑气波  
需要1点武器能量  
使用者先以一个迅捷动作按下刀柄上的开关转化武器能量，并再使用一个标准动作挥刀斩击，进行一次白刃攻击检定，发射一道剑气波。  
这次挥砍将不会造成伤害，而是由剑气波造成伤害，这道剑气波视为具有【能量武器】、【超级贯穿】，并具有完成的试炼场数\*5米射程，在成功命中且造成伤害后，使用者将立即闪身移动至其背后，其移动上限不能超过使用者单移动距离，这是一个C级创伤来源的传送效果，若命中了多个目标，则选择一个进行闪烁。

升龙斩  
需要1点武器能量  
使用者使用一个迅捷动作准备，然后使用标准动作施展升龙斩，立即向任一水平方向冲刺等同于单移动动作一半的移动距离，在触碰到第一个敌人时停止，先进行一次白刃检定，这次攻击额外附带B级击飞效果和【毁灭】，再立即进行一次跳跃检定，这次跳跃视为助跑跳，跳跃方式为跳高，取出目一半+试炼场数\*5米，你因此而落地时不会受到任何坠落伤害。在这次攻击过后，你会额外获得1点仅能用于释放下坠击的武器能量。

下坠击  
需要升龙斩提供的1点武器能量  
必须在浮空时才能使用下坠击。使用者先使用一个迅捷动作准备，然后使用一个标准动作施展下坠击，立即对一个接触范围内的浮空状态的敌人进行一次白刃检定，此次白刃检定具有【9加骰】，使用强韧豁免进行对抗，同时将自身和该敌人同时坠击向地面，额外下落等同于使用者的单移动距离，若下坠击成功击中地面，则立即对该敌人造成等同于对抗差值的伤害，并对落点为圆心15米半径范围内的所有敌人造成伤害，同样使用强韧豁免进行对抗，使用者不会因此次落地而受到坠落伤害。下坠击额外附带等同于试炼场数\*1点的附加成功。

次元斩  
需要在一场战斗中先后使用过剑气波、升龙斩、下坠击。  
使用次元斩时首先要使用一个迅捷动作将鬼妖村正归鞘，在那之后，使用一个标准动作对触及范围内的所有敌人进行一次白刃检定攻击，最后，使用一个移动动作引爆因快速切割而撕裂的空间。这意味着使用次元斩是一个整轮动作，攻击触及范围内的所有敌人，使用强韧豁免进行对抗，但这些伤害不会立即生效，而是在使用移动动作引爆后开始生效，无数被割裂的时空碎片飞溅，这次攻击将额外附带A级重伤和碾压效果，所有受到伤害的目标将被立即关入破碎的异空间之中，这是一个B级时空来源的放逐效果，这个效果将持续一轮。

【试炼】：  
完成一次试炼需要使用者在不使用任何其他武器的情况下，使用鬼妖村正在没有其他轮回者或NPC助力的前提下亲手击杀一名比其支线等级高1级的BOSS，并活着完成这场影片，视为完成了一次试炼。相应的，为了其他轮回者的体验，主神可以选择为鬼妖村正的持有者单独准备一个BOSS。

J  
锯齿大刀  
价格：C+1000  
分类：刀  
本质：科技本质  
武器伤害：11L  
武器特性：【破甲6】【锯齿大刀】  
武器体积：3  
伤害类型：物理-挥砍  
描述：将牛刀与挫骨和凿孔结合的恐怖杀人凶器。可以把猎物切斩、串刺成死状凄惨的模样。  
效果：  
【锯齿大刀】：使用该武器进行攻击时，附带1点不可豁免的流血不良点数，可叠加。

L  
量产村正  
价格：C+1000  
分类：武士刀  
本质：科技本质  
武器伤害：11L  
武器特性：【高速1】【未来科技】【功耗】  
武器体积：3  
伤害类型：物理-挥砍  
描述：东方名剑的复制品，作为土特产而言相当受欢迎，同款的产品还有量产十拳剑，量产草雉剑等等物品。  
效果：  
【未来科技】：虽然是量产品，但是以碳纳米管骨架上加入稀有金属合金作为剑身，并且内置多项传感器，令这把刀变为了一把绝对的武器。  
使用该武器进行攻击前，你可以先使用一个迅捷动作激活内置传感器寻找敌人的弱点，牵引手臂挥下剑刃。若如此做，本次攻击将附带【接触攻击】特性。  
【功耗】：驱动量产村正的武器效果进行攻击，需要消耗能量。安装的新型能量转换器使其能使用任意通用能量池驱动，在每次启动内置传感器效果时会消费1点能量。

M  
名刀  
价格：C+1500  
分类：武士刀  
本质：科技本质  
武器伤害：11L  
武器特性：【高速1】【名刀】  
武器体积：3  
描述：具有高速挥舞与保证杀伤力的日本刀。要能顺手使用的话，需要达成某种精神统一的境界。  
效果：  
【名刀】：持有者在先攻上获得等同自身【传奇沉着\*4】点提升；当持有者因拔刀检定而获得在攻击上的加值时，可以消费1点意志力，使本轮攻击额外获得等同加值\*1点的【破甲】（上限15）。

Y  
妖刀-夜无月  
C+1000  
分类：武士刀  
本质：特异本质  
武器伤害：11L  
武器特性：【高速1】【夜无月】  
体积：3  
特性：  
描述：上古时期，开创了天王斋无月流的天王宗师，斩月斋的佩刀，当斩月斋死后，一同葬在聊月山山顶的这把刀被月光中的妖怪所染，加上千年战火不断，民间将传说的故事反复传唱，最终令这把刀成为了一柄付丧神，虽然依旧不及传说中能斩断月亮之能，但也拥有难以捕捉的如月光之速。  
效果：  
【夜无月】：这是一把通体漆黑的中长太刀，但在月光的照射下会散发出幽幽银光。当你处于没有月亮的夜晚下，此武器进行的拔刀攻击额外具有等同于拔刀检定成功数的高速，握住这把武器会感觉到微弱的情绪，处于白天时，武器中会传来萎靡不振之感，处于夜晚时，武器则会散发出欢欣雀跃之意，当处于没有月亮的夜晚时，以这把武器发起的攻击会具有【吸血1】。
