# 无礼的代价

无礼的代价  
C+1500  
分类：拳套  
专业：拳/爪  
武器特性：【轻量化】【重击】【附着于吾】【破甲3】【高速3】【破魔3】  
武器伤害：5  
体积:1  
描述：这是一只穿戴在手上的钢铁手套，其实并没有想象中那么重，上面玄奥的的花纹除了让自身轻量化外，还赋予了这手套撕碎敌人力量。同样的力量并不属于这支手套，它只是发挥本来属于你的力量。

【附着于吾】这手套原本并没有力量，它的力量来自于你的自身，这手套提供的武器伤害可以与你自身天生武器叠加，如果你没有天生武器，那么该武器伤害降低为2。

【轻量化】这手套的轻量化，除了优化它自身的如同一只普通的丝绸手套般的重量之外，还优化了你自身的重量，你的移速提升10点。

【重击】这手套是武器，那么它就要发挥它原有的作用，让敌人在你面前如同破木头一般被撕个粉碎，你根据你自身的最高支线强化等级来获得（D1C2B3A4S5）肉搏攻击上的附加成功。
