# 改装电锯

改装电锯  
价格：C+2000  
分类：电锯  
武器伤害：12L  
武器特性：重型武器，双手，威猛4  
武器体积：4  
伤害类型：物理-挥砍  
描述：战争期间，士兵之间的制式近战武器，虽然整体而言异常笨重，但是能很方便的肢解活死人所用的凶器。  
效果：使用该武器进行攻击时，附带等同你【传奇力量-1】点肢体妨碍点数，不可豁免，可叠加；该武器虽然是电锯，但主神提供的版本无需供能，你可以正常进行使用。
