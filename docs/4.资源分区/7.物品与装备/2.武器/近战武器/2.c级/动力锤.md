# 动力锤

  

动力锤  
价格：C+1500  
分类：锤  
体积：4  
科技本质  
描述：通体由冷钢铸造的实心战锤，作为一种需要双手使用的大型武器，质量庞大到令人生畏。由于其具备的多种不同改造，只有近战疯子才能驾驭的了这把武器。  
武器基本属性：武器伤害12L  
武器特殊属性：【重型武器】【双手】【威猛5】【连射】  
链锯改装：它结合了重锤的打击力量和链锯的撕裂特性，其棒状的锤头部有安装有多排旋转的链齿，可以把接触到的肉体撕成碎片。  
因此武器威猛+3（已反馈在属性中），力量需求+1，且隐藏时难度提升1级。  
喷气动力：其开创性的加装了喷气式动力，在外部的数的小孔能够于攻击时启动，推动锤身向不同方向狂野的发起连续打击，砸的七荤八素。  
这种物理学上的设计为为武器提供【连射】属性。  
火药冲击：采用火药动力改良，在打击他人肉体时，其内部安装的底火会猛然爆裂，抡圆了一锤子能把敌人直接打翻。  
使用武器进行攻击前，可以用本次攻击纯主属性（即不包括全检定、相关属性加值）进行检定，成功数/2的加入到本次攻击附加成功中。  
随后进行攻击，本次攻击还视为同时进行冲撞来结算。
