# 以理服人

  

以理服人  
价格：C+1500  
分类：指虎  
武器伤害：4附加  
武器特性：无  
伤害类型：物理-钝击  
体积：1  
描述：冷钢打造的指虎，异常厚实，上面的附魔可以克制超自然的再生能力。  
肉搏武器：基于天生武器的伤害由冲击转为严重。  
有话直说：取代肉搏武器自带的攻击附加等同于附加成功的冲击点数效果，以理服人的肉搏攻击会附带等同于攻击伤害值的冲击点数，强韧豁免。  
附魔：以标准动作使用一个不附带招式的普通攻击才能生效，受到本次伤害的目标承受持续一场景的B级重伤效果。
