# C级白刃类-剑

C级白刃类—剑

A  
暗月大剑  
价格：C+2500  
分类：重剑  
类型：魔幻本质  
武器伤害：14L  
武器特性：【双手】【重型武器】【月光剑】  
武器体积：4  
伤害类型：物理-挥砍  
描述：  
历代的卡利亚女王赠与自己伴侣的月之大剑，为交界地‘传说中的武器’之一，黑暗与星空之神的象征是冰冷的黑暗满月，因此而赠予的大剑，则会是暗月的月光。

【剑身引导着暗夜的宇宙，其锋利的刀刃映射出暗影的光芒。曾经的欧斯罗艾斯随着自己的执着追寻着月光，但却连一眼都没有看见。  
最后的半神舍弃了自己的躯体，在迈入星光后，将此剑重铸并赠与了自己的王，为何流落在主神空间，便不得而知了。】

效果：  
【月光剑】：当你持有这把大剑时，你仅能使用智力+学识+主专业【学识：魔法学】+副专业【专注：星光信仰】以进行一次攻击，并造成冻寒伤害。该攻击的射程为你智力/米  
卡利亚的恩赐：当轮回者使用此武器攻击时，附带当前伤害数值的四分之一的冻寒点数，对方以强韧豁免。当轮回者经历过1场影片之后，则会受到星光的加护，强化此攻击模式，为附带当前伤害数值一半的冻寒点数，对方以强韧豁免。

F  
飞天御剑  
价格：C+1000  
分类：剑  
本质：特异本质  
武器伤害：11L  
武器特性：【破甲1】 【飞天】  
体积：3  
伤害类型：物理-挥砍  
描述：可靠的钢剑，能让人御剑飞行，宛如一名剑仙。  
效果：  
【飞天】：持有者获得飞行能力，机动性普通，飞行速度等同基础移动速度+20米。

H  
火麟剑  
价格：C+2000  
分类：剑  
本质：特异本质  
武器伤害：9L  
武器特性：【破甲5】 【火麟】【邪剑】  
体积：3  
伤害类型：物理-挥砍  
描述：  
南山巅上火麟剑，北海潜深雪饮寒。  
可怜两锋未缘见，雪刀封隐孤剑鸣。  
火麟剑乃是与雪饮刀齐名的神兵利器，相传为断家先祖，天下第一剑“断正贤”在力战四处为祸的火麒麟后，得其一块鳞甲，镶于剑身而成，位列【七武器】与【十二惊惶】之中。  
火麟剑上镶嵌的鳞甲带有麒麟之魔性，能助使用者提升功力，但是也会助长使用者的邪念，使其堕入魔道，因而也被誉为“剑中邪神”。

效果：  
【火麟】：火麟剑在挥舞之时，能够激发剑身上的火麒麟鳞片所蕴含之力量，以灼浪伤人。因此只要你的内力池不为空，你每次攻击时，都会附带1点不可豁免的燃烧点数。  
【邪剑】： 火麟剑中蕴含的魔性会祝你不断提升功力，但也会逐渐侵蚀你的心性。每经过一部影片，你的内力池扩大6点，并在攻击检定上获得1点可以叠加的附加成功，但这也会使得你在每经过一部影片时，获得1点无法豁免，永久存在的亢奋点数。

K  
哭泣恶魔  
价格：C+2500  
分类：重剑  
本质：魔幻本质  
武器伤害：12L  
武器特性：【破甲5】 【双手】【重型武器】【哭泣恶魔】  
体积：4  
伤害类型：物理-挥砍  
描述：深暗的魔王流下悲痛之泪，狂嚎着反叛之语，以世界为敌。这是两柄漆黑巨剑，没有剑尖，其上缠绕着无声的哀嚎，剑首至剑护由漆黑无光的某种增生岩类构成，持握时，心中会蔓延起无名的悲哀与伤痛。

效果：  
【哭泣恶魔】：该武器的所有效果都需要双手各持用一把才能生效，作为双手重型武器，该武器的力量要求因此达到11点。  
其中蕴含的反叛哀歌，将使你在持用这把武器时永久承受不可豁免的5点沮丧点数，可以与其他来源的沮丧点数叠加至严重后果。  
持用该武器时，仅能使用一种攻击方式，消耗一个标准动作进行近战白刃攻击检定，两把重剑将溢出漆黑的魔力，这次攻击dp提升等同于你决心属性值的亵渎加值与传奇力量\*2点破甲，带有【法破】特性，并使攻击的加骰提升一级（最高为8加骰）。  
这是一个攻击技。

L  
龙王岩剑  
价格：C+2500  
分类：重剑  
本质：魔幻本质  
武器伤害：12L  
武器特性：【双手】【重型武器】【雷云化身】  
体积：4  
伤害类型：物理-挥砍  
描述：  
蕴藏初始雷电的碎石迅捷剑，蕴含着天空之城的古龙“普拉顿桑克斯”其中片鳞半爪的力量。  
是上没有永恒生命、渺小的龙后裔们，所不具有的强大的威力。  
未知的岩石组成四棱椎体样貌重型迅捷剑，相比其他迅捷剑更加粗大，也更长

效果：  
【雷云化身】：整轮动作结合在一次冲锋中使用，将自身短暂化为闪烁这红色雷电的云雾，飘浮在空中，  
随后向前极速飞驰，将缠绕雷电的剑身贯穿敌人的战技。  
短暂化为雷电的身躯在本轮中视为【虚体】。  
冲刺而至的雷云将对敌人造成无法被盔甲抵挡的雷电伤害【接触攻击】。

M  
玛雷家行刑剑  
价格：C+1000  
分类：重剑  
本质：魔幻本质  
武器伤害：12L  
武器特性：【双手】【重型武器】【处刑剑】【欧赫剑舞】  
体积：4  
伤害类型：物理-挥砍  
描述：  
日荫城的城主，同为行刑者一族——玛雷家的宝剑。  
“传说中的武器”之一。  
以铃珠猎人身份受令人畏惧的“荆棘”艾隆梅尔，在行刑场抢走此剑，作为自己的武器，并将故乡欧赫的战技赋予其上。  
描述：不知何种材料的剑柄以被暗红色浸透，布满荆棘的剑格延伸出布满陈旧血迹宽大的剑身，其并不如其他剑一般具有锋利尖端，而是变为更适合劈砍的圆刃，对应其行刑剑的名号。  
该剑可以使用魔力、斗气、法力等西方能量池驱动。

效果：  
【处刑剑】：自由动作消耗6点能量启动持续一轮，剑身将满布血色，随着你的心意在空中飞舞，在这一状态下，人物可以操纵玛雷家行刑剑对15米距离内直线中的所有敌人发动攻击，剑身血红光芒大盛后急速飞向攻击目标。

【欧赫剑舞】：整轮动作使用，将6点能量灌入剑中，旋转之后向前方进行冲锋，并释放的战技。  
整把剑会剧烈的旋转并散发出血腥的红光，这次冲锋会将路径上所有敌人均作为攻击目标，并视为【接触攻击】。

魔剑【昂巴尔】  
价格：C+1000  
分类：重剑  
本质：魔幻本质  
武器伤害：11L  
武器特性：【破甲2】【昂巴尔】  
体积：3  
伤害类型：精神伤害  
描述：  
噬身之蛇盟主所赐予的魔剑“昂巴尔”，与剑帝·莱恩哈特的噬岩者成对，由于曾经的使用者与该武器契合度过高，此武器外观变成了一把浑身赤红并且散发着不详气息的单手剑。使用者在拥有这把剑后便会一直听到曾经的使用者呢喃以及飘渺的注视。

‘那个银发女人给了我这把剑，似乎相性太好了以至于这把剑因为我发生了一些微小的变化。’

效果：  
【昂巴尔】：当使用者使用武器进行攻击时，魔剑会尝试强化自己的使用者并且帮助使用者强化本次攻击，使用者在攻击前投掷一个不受任何加成的机运骰，当出目为6-10时，使用者可以使用一个移动动作引发昂巴尔的潜在力量，使下一次攻击获得【9加骰】，并且被攻击者似乎能听到异世界魔人的低语，逐渐沉溺在获得力量的虚幻魔境之中，攻击者将此次攻击的成功数转化为魅惑点数，对方以标准对抗豁免。

当你每次使用昂巴尔攻击一个敌对目标时，持有者通过引发曾经异世界魔人的力量来尝试扭曲被攻击者的心智并且支配对方，对方以意志豁免对抗你对对方造成伤害的成功数，若对抗成功，则无效果，若对抗失败则暂时不触发效果，使用者必须在本回合以一个迅捷动作将昂巴尔归入鞘中，其特殊的音波会回荡在敌对目标耳边并且使对方在一定时间内无法分辨现实与虚幻的边界。

魔剑的使用者可以在被支配者的回合开始时对其下达指令，该命令必须明确且必须在能力范围内的具体行动，若对方选择违抗指令，则会从昂巴尔上爆发出血色的能量，并且使对方获得4点精神易伤，仅生效一次。

本质：魔幻本质  
武器伤害：11L

朽木  
价格：C+1000  
本质：魔幻本质  
描述：  
”那一千年的老树，在村头无所事事了一千年。“  
”我第一眼看到他，就知道他在等谁。“  
”他在等一个剑客，一个带他打败天下剑的剑客。“  
”于是我做了祂，打败二百八十道剑门。“  
”现在该你了。“  
一把用老木头制成的，闪着冷光的剑，一看跟普通的铁剑毫无区别，只有手柄上残存的树皮显示着祂的不凡。  
【千年剑魂】：当你持有\[朽木\]时，千年的厚重将淬炼你的灵魂，并在战斗中展露出来，其将锋芒毕露，明亮的剑光覆盖剑身。  
【生死一决】：当你陷入生死之时，\[朽木\]会发挥出自己真正的力量。  
特性：【千年剑魂】【生死一决】  
武器分类：轻剑  
武器伤害：11L  
基础特性：破甲1  
效果：  
【千年剑魂】：在使用\[朽木\]时，将占据你的2点意志力上限，当进入战斗一轮后，本场景中你使用\[朽木\]进行的攻击具有【毁灭】特性。  
【生死一决】：一天一次，当你不再拥有完好生命值时，本场景内，你使用\[朽木\]进行的攻击具有【接触攻击】特性。  

心影  
价格：C+1000  
分类：重剑  
本质：魔幻本质  
武器伤害：12L  
武器特性：【双手】【重型武器】【掘尸三尺】【暗地一击】  
武器体积：3  
伤害类型：物理-挥砍  
描述：被征服者的苦难。  
这是一把银灰金属色调的细长重剑，数条从握柄处延伸出的扭曲的银色钢铁环绕着中心处略微弯曲的黑色剑身。整把剑像是一个符文，又像是一个符号。  
效果：  
【掘尸三尺】：支付1点意志，进行一次整轮动作攻击，在接下来的一场景或你下一次攻击前（取低）获得C级魔幻本质的反侦测效果。

【暗地一击】：在不能被任何敌对目标精确定位的情况下，你可以用标准动作进行一次攻击。这是一个攻击技能。这次攻击会额外造成3的物理易伤，该易伤效果持续一场景，不可叠加。

Y  
鱼腹剑  
价格：C+1000  
分类：剑  
本质：自然本质  
武器伤害：11L  
武器特性：【破甲6】【遗恨】【鱼腹】  
武器体积：3  
伤害类型：物理-挥砍  
描述：  
屈子当年赋楚骚，手中握有杀人刀。  
艾萧太盛椒兰少，一跃冲向万里涛。  
效果：  
【遗恨】：当你首次在战斗中拔出这柄剑时，若此时的天气环境为雨环境，则这一雨环境将在一轮后向上提升一级（雨-大雨-特大暴雨），当长剑被归鞘或脱手时，此环境将在一轮后重新恢复至原先等级。  
【鱼腹】：使用者可以在任何情况下，只要足以正常使用这柄长剑，就能以一个反射动作用这柄剑杀死自己，一只巨鱼将出现在暴雨之中，一口将使用者的尸身吞下，并消失不见，在本场影片中，使用者将无法以任何形式复活或存在。在影片结束后，结算朝生暮死规则时，这将视为尸身保存完整。若以此法终结自身的生命，每经过一场影片，就获得1点在雨环境中的偏斜防御，提升这些防御的画风表现为逐渐隐没在雨幕中。
