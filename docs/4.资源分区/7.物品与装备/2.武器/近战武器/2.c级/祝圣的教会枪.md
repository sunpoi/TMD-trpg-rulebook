# 祝圣的教会枪

祝圣的教会枪  
价格：C+1000  
分类：长枪  
武器伤害：11L  
武器特性：【长柄】【法破】【双手】【格挡5】  
武器体积：4  
伤害类型：神圣/正能量  
描述：据说是教会的圣职者、为了杀死恶魔而祝圣的枪。实际上、是上位恶魔杀死野蛮下级恶魔的枪。  
效果：  
【恶魔杀手】：使用该武器攻击时，若目标具有【黑暗生物】类型，则你在攻击检定上额外获得等同自身【传奇力量】一半的器械加值与破魔，并额外造成等同这些器械加值数值的韧性伤害。
