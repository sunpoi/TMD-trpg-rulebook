# 魔剑飞空

魔剑飞空  
价格：C+2000  
分类：剑  
武器伤害：11L  
武器特性：【轻灵】【破甲1】  
武器体积：3  
伤害类型：音波（震动）  
描述：被称为「风之剑」、斩击后吹起舒适的风，攻击时能极为准确迅速斩击。  
效果：持有者可以使用一个迅捷动作激活武器能力，持续一场景；持续时间内，当使用该武器进行攻击时，在攻击检定上获得等同【敏捷属性】一半的速度加值，并在攻击上获得等同这些速度加值的【高速】。
