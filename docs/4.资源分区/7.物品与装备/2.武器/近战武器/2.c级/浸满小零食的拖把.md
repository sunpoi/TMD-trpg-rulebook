# 浸满小零食的拖把

浸满小零食的拖把  
价格：C+1000  
分类：长棍  
武器伤害：9L  
武器特性：长柄 格挡 高速3  
武器体积：3  
描述：据说是传奇不死人小骑士第四喜欢的武器，持有该武器能轻易令数十人不敢近身，有万夫不当之力。  
效果：  
【零食浇给~】  
当你的攻击命中后，对方需要进行一次反射豁免，dc为本次攻击出目，若失败则承受【潮湿】效果，这是一个C级诅咒来源的效果，持续一场景

【敌莫敢当】  
当你持有这柄武器并花费一个花招动作进行转棍威慑后，本场景内能够进行近距离攻击的敌方单位都会尽力避免与你进行接触（这是一个画风效果）。  
但若目标承受了这一武器带来的【潮湿】效果，则目标立刻红温，必须优先选择你进行攻击，这是一个C级心灵来源的【胁迫】效果。
