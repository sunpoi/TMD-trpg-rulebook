# 刚健之盾

刚健之盾  
价格：C+1000  
分类：重盾  
武器伤害：11L  
武器特性：【双手】【防能量】【格挡10】  
武器体积：4  
伤害类型：物理-钝击  
描述：圣岩石制作的异国之盾。作为神具保存。有提高生命循环的效果、能增加收到的回复量。  
效果：持有者可以选择自身具有的一项快速医疗能力，使其提升1点并提升其医疗点数12点；若持有者没有快速医疗能力，则该盾牌的效果永久性改为获得快速医疗1（医疗点数12）。
