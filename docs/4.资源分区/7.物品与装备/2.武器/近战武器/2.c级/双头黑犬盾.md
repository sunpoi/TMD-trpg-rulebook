# 双头黑犬盾

  

双头黑犬盾  
C+1000  
原典：盾之勇者成名录  
以双头黑犬作为素材制造的盾牌，其上的黑犬头颅能够赋予持有者警觉的能力。  
类型：重盾  
武器伤害：6L  
体积：4  
专业：盾  
特性：【格挡10】、双手  
【危机感知】：当你的敏感范围内具有敌意单位时，盾牌上的黑犬头颅会开始吼叫以提醒持有者敌人正处于附近。这是一个C级别、诅咒来源的预知效果。  
【犬噬】：当持有者受到近战攻击时，可以反射动作花费3点魔力令犬头撕咬目标，对攻击者造成3点穿刺严重伤害。
