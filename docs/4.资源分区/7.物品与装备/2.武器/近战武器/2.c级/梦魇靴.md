# 梦魇靴

梦魇靴  
位置：脚部   
价格：C+1250  
这双被邪恶战士珍藏的靴子，由漆黑的皮革修剪制成，附有深红色的梦魇鬃毛。害怕梦魇的人，会在看到这深红的鬃毛时回忆起梦魇的恐怖。  
武器伤害：2L+3附加成功  
专业：腿  
分类：肉搏武器  
占据：鞋袜位  
体积：1  
视为天生武器般使用腿专业，伤害由冲击转化为严重  
具有【引火长径】特性。

引火长径：  
当穿戴者进行一次近战肉搏冲锋时，火焰与浓烟从将靴子中涌出，点燃穿戴者通行过的路径，这些魔法火焰可以在任意平面燃烧，包括且不限于水面，冰雪以及飞船的隔热装甲，这会令这一次冲锋攻击同时对路径范围上的所有敌方单位均造成影响，且这一次冲锋攻击造成的伤害转化为火焰伤害。  
引火长径的魔法可以燃烧6秒，随后无害的消散（但也可能如正常火焰般点燃什么，ST决定），若有单位在火焰尚未熄灭时踏入其中，其会如同被这次冲锋攻击影响到一般受到伤害，也就是说一个单位不会因此多次受到同一次冲锋的伤害。
