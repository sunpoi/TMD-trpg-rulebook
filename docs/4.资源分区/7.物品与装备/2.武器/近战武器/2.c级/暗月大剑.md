# 暗月大剑

  

暗月大剑  
C+2500  
魔幻本质  
历代的卡利亚女王赠与自己伴侣的月之大剑，为交界地‘传说中的武器’之一，黑暗与星空之神的象征是冰冷的黑暗满月，因此而赠予的大剑，则会是暗月的月光。  
这是一把武器伤害14 力量需求6 【双手】【重型武器】体积为4的一把石制巨剑。  
月光剑：当你持有这把大剑时，你仅能使用智力+学识+主专业【学识：魔法学】+副专业【专注：星光信仰】以进行一次攻击，并造成冻寒伤害。该攻击的射程为你智力/米  
卡利亚的恩赐：当轮回者使用此武器攻击时，附带当前伤害数值的四分之一的冻寒点数，对方以强韧豁免。当轮回者经历过1场影片之后，则会受到星光的加护，强化此攻击模式，为附带当前伤害数值一半的冻寒点数，对方以强韧豁免。  
【剑身引导着暗夜的宇宙，其锋利的刀刃映射出暗影的光芒。曾经的欧斯罗艾斯随着自己的执着追寻着月光，但却连一眼都没有看见。  
最后的半神舍弃了自己的躯体，在迈入星光后，将此剑重铸并赠与了自己的王，为何流落在主神空间，便不得而知了。】
