# 魔剑【昂巴尔】

  

魔剑【昂巴尔】  
C+1000  
11L 3体积  破甲2  
魔幻本质  
这是一把武器伤害11 3体积 力量需求3  破甲2 攻击造成【精神伤害】

噬身之蛇盟主所赐予的魔剑“昂巴尔”，与剑帝·莱恩哈特的噬岩者成对，由于曾经的使用者与该武器契合度过高，此武器外观变成了一把浑身赤红并且散发着不详气息的单手剑。使用者在拥有这把剑后便会一直听到曾经的使用者呢喃以及飘渺的注视。

‘那个银发女人给了我这把剑，似乎相性太好了以至于这把剑因为我发生了一些微小的变化。’

当使用者使用武器进行攻击时，魔剑会尝试强化自己的使用者并且帮助使用者强化本次攻击，使用者在攻击前投掷一个不受任何加成的机运骰，当出目为6-10时，使用者可以使用一个移动动作引发昂巴尔的潜在力量，使下一次攻击获得【9加骰】，并且被攻击者似乎能听到异世界魔人的低语，逐渐沉溺在获得力量的虚幻魔境之中，攻击者将此次攻击的成功数转化为魅惑点数，对方以标准对抗豁免。

当你每次使用昂巴尔攻击一个敌对目标时，持有者通过引发曾经异世界魔人的力量来尝试扭曲被攻击者的心智并且支配对方，对方以意志豁免对抗你对对方造成伤害的成功数，若对抗成功，则无效果，若对抗失败则暂时不触发效果，使用者必须在本回合以一个迅捷动作将昂巴尔归入鞘中，其特殊的音波会回荡在敌对目标耳边并且使对方在一定时间内无法分辨现实与虚幻的边界。

魔剑的使用者可以在被支配者的回合开始时对其下达指令，该命令必须明确且必须在能力范围内的具体行动，若对方选择违抗指令，则会从昂巴尔上爆发出血色的能量，并且使对方获得4点精神易伤，仅生效一次。
