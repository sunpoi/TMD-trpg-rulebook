# 攻城级动能闷棍

攻城级动能闷棍  
科技本质  
价格：C+1000  
分类：锤  
伤害类型：钝击  
体积：4  
武器伤害：12L  
描述：一把前端十分重的金属机关长棍，由于重心偏移，实际上更应该认为是某种缩小锤头的便携攻城锤。  
攻城级动力闷棍具有【重型武器】 【双手】 【威猛3】【动能强化】【贯穿】  
动能强化：该武器会回收使用者的动能进行更强的攻击，在使用该武器的战斗中，每轮结束时该武器将依次获得以下效果，不进行叠加，战斗结束时清除  
1.该武器将会在攻击时存储的动能将触发机关再给攻击目标造成一次冲击，该武器具有【余波】  
2.该武器现在可以将存储的动能在打击单位时变为超声波震动打击，该武器具有【能量武器】。
