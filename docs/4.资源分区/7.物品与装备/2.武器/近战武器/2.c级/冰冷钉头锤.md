# 冰冷钉头锤

冰冷钉头锤  
价格：C+1000  
分类：锤  
武器伤害：12L  
武器特性：【重型武器】【双手】【威猛3】  
武器体积：4  
伤害类型：物理-钝击  
描述：古代被邻国战士们恐惧的冷血英雄所持有的钉头锤，死后、被作为不详的象征封印到海底。  
效果：持有者可以支付1点意志力并消费一个迅捷动作启动，持续一场景；持续时间内，使用该武器进行的攻击附带等同【传奇力量/2】+1点冻结不良点数，不可豁免。此外，这还会使武器伤害类型在持续时间内转为寒冷伤害。
