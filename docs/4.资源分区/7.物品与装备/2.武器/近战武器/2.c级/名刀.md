# 名刀

名刀  
价格：C+1500  
分类：武士刀  
武器伤害：11L  
武器特性：【高速1】  
武器体积：3  
描述：具有高速挥舞与保证杀伤力的日本刀。要能顺手使用的话，需要达成某种精神统一的境界。  
效果：持有者在先攻上获得等同自身【传奇沉着\*4】点提升；当持有者因拔刀检定而获得在攻击上的加值时，可以消费1点意志力，使本轮攻击额外获得等同加值\*1点的【破甲】（上限15）。
