# 风暴大锤

风暴大锤  
价格：C+1000  
分类：锤  
武器伤害：12L  
武器特性：【重型武器】【双手】【威猛3】  
武器体积：4  
伤害类型：物理-钝击  
描述：通过恶魔让武器含有魔力的锤子。据说光是挥舞就能引起风暴。  
效果：持有者可以支付1点意志力并消费一个迅捷动作启动，持续一场景；持续时间内，持有者的近战触及距离增加【传奇智力】\*2米。  
持续时间内，持有者进行的攻击可以对触及距离内的所有敌对单位造成伤害，并附带1点不可豁免的麻痹不良点数，伤害类型转为雷电。
