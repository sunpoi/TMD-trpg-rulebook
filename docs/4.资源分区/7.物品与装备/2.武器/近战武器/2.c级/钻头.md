# 钻头

钻头  
价格：C+1000  
分类：钻头（长枪）  
武器伤害：12L  
武器特性：长柄，双手，格挡2，破甲5  
武器体积：4  
伤害类型：物理-穿刺  
描述：安装在长柄武器上的，以惊人气势旋转，无视对手防御直接贯穿的强力钻头。可以说是专克高防御目标的武器。  
效果：当你使用该武器进行招架前，你可以无需动作的启动钻头，使本次招架不会减少伤害，而是替换为造成等同本应抵消伤害数值的物理穿刺伤害，这些伤害不能被豁免，但可以被伤害吸收等方式吸收，其他方面则正常遵循招架流程。
