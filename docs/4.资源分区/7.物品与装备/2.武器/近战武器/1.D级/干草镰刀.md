# 干草镰刀

干草镰刀  
d+500  
特异本质  
类型：武器/镰刀  
原典：星露谷、我的世界、泰拉瑞亚  
武器伤害5L 轻灵 破甲1 挥砍 体积2  
割下鲜草，会获得干草。  
干草作用可多了，当垫脚方块，当储备干粮，当动物饲料——  
当你对着鲜草进行收割动作时，它们将变为干草，并储存在你的一个储物空间谷仓中，这个谷仓只有3立方米左右的空间，每攒够1立方米的干草，干草将自动绑成一个干草块，可以堆叠起来当墙壁，隔断，或者当做垫脚草。  
这个谷仓无法储存其他物品，取出干草或干草块时需要一个移动动作。  
不过，不是所有干草都能够当做食物；  
起码普通人没法消化干草……  
每场影片中，你每天第一次收割作物时，收割过后你将额外获得1份小麦，每场影片至多可获得3份小麦，小麦不占用谷仓的位置，而是在你不知道在哪里的物品栏里，小麦可在影片结束后带出影片。  
小麦无法直接食用，但可以通过造物规则将其制作为小麦面包，三份小麦可以制作一份小麦面包。

【小麦面包】  
特异本质  
类型：消耗品  
价格：三份小麦（2xp）  
饱食：以一个迅捷动作使用，食用后将获得饱食度提升，并且在该场景中将自动稳定伤势，不会使其恶化  
满饱食：当你食用小麦面包后，若生命值不为满值，你的生命值将缓慢恢复，每轮回复1点生命值，上限12点，这是一个来源于【快疗1】的生命恢复效果，因为饱腹感充足导致吃不下，同一时间只能食用一个。
