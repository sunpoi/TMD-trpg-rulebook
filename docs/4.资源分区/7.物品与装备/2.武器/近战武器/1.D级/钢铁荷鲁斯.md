# 钢铁荷鲁斯

  

钢铁荷鲁斯  
D+1000  
本质：科技本质  
这是一个单肩包形状的物体，有背带，包面上写着“IRON HORUS”，内部可以储存物品，并且有一个特制凹槽，可以装入一把截短过枪管的霰弹枪（换言之，体积不超过3）。以一个迅捷动作可以利用内置气动装置将这个单肩包展开为一面钢铁制成的长方形防暴盾牌，它的格挡防御为8点，体积4，武器伤害为3L，并可以单手持用，但只有展开时才具有武器伤害和格挡防御。当钢铁荷鲁斯展开时，持有者可以以一个自由动作取出凹槽中装入的霰弹枪，若未使用迅捷动作在战斗场景中展开这面盾牌，则仍需以正常动作取出这把霰弹枪。  
当持有这面盾牌且使用霰弹枪进行攻击时，获得抵近射击的射击技巧，可以将射击结合在移动之中，但移动速度降低至原来的一半，若如此做，则本轮内获得钢铁荷鲁斯所提供的格挡防御，但并不视为进行了一次格挡。
