# 破仑

  

破仑  
D+500  
外形：一个大木车轮，看起来有些腐朽，但不像一个真正的破轮子一样一碰即碎  
介绍：夫项羽，乃拔山盖世之雄，岂有一破仑而不能拿乎？  
武器类型：重盾 3L 双手 格挡9 钝击 体积4  
纵或挡道，乌骓但扬蹄，项羽即可过焉，何需下马将其移开，而后再行？岂不多此一举乎。  
一回合一次，你可以自由动作对你触及范围内的一名驾驶载具的敌人展示手中的这沉重的破仑，引起对方的警惕，你使用力量属性-武技-专业：盾牌对抗对方的决心-专注-专业：意志，若成功，本回合对方感受到这个破轮的威胁不会轻描淡写的略过这个破仑，而会尝试排除这个障碍，也就是只会对你发动攻击，这是一个D级心灵来源的胁迫效果。  
一破仑，百数斤而已，令一、二士卒足以胜之，而何须劳项羽屈主帅之尊躬自动手乎？  
若持有者的力量属性达到11，则能以合适的姿势充分利用破仑的性能，装备破仑时持有者额外具有1点全伤害吸收。  
基于上述，余以为：项羽不必拿破仑。  
ST选开：若存在项羽相关的血统/训练，ST可以静止破仑与该血统/训练同时使用。  
  
民国初年，某学堂招考新生，作文命题为《项羽拿破仑论》。有某生者，于拿破仑素无所闻，见题后极为惶惑，继而忽有所“悟”，遂奋笔疾书，为文曰：  
夫项羽，乃拔山盖世之雄，岂有一破仑而不能拿乎？非不能也，实不必也。  
彼破仑，为何物？其大几许，其高若干？纵或挡道，乌骓但扬蹄，项羽即可过焉，何需下马将其移开，而后再行？岂不多此一举乎。  
退万步言，欲将其去之以利行军，然则彼一破仑，百数斤而已，令一、二士卒足以胜之，而何须劳项羽屈主帅之尊躬自动手乎？于情于理，无乃不合也！  
基于上述，余以为：项羽不必拿破仑。
