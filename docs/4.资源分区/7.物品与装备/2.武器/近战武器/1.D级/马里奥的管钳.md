# 马里奥的管钳

  

马里奥的管钳  
D+500  
魔幻本质  
外形普通的管钳，上面附着一层火焰，也许是管钳正在愤怒。（这层火焰并非实际的火焰，但会让你摸起来暖暖的）

这视为一把短棍，伤害5L，格挡1，你可以使用一个标准动作，攻击一个生物的头部，这次检定不会造成伤害，而会将伤害的出数转化为晕眩点数。以标准形式豁免，同时附带一点不可豁免的麻痹点数。

备注：这把管钳的主人很有名，他自称是个水管工，但他从不修管道，他只是在管道里钻进钻出，跳来跳去。整日忙着与恶势力战斗、拯救王国、寻找公主什么的。他有时会化装成青蛙、浣熊、甚至地藏菩萨。有时他还会放火球，扔锤子……但他从不！请注意，是“从不”用这件水管工必备的工具。于是，这把管钳愤怒了，它火冒三丈，它要证明管钳完全是可以参加战斗的！
