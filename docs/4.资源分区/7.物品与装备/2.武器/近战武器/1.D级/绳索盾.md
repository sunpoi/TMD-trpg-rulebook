# 绳索盾

  

绳索盾  
D+500  
原典：盾之勇者成名录  
如同绳索般呈圆形缠绕的盾牌，具有射出绳索的能力。  
类型：轻盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡5】  
【绳钩】：以迅捷动作使用，对不超过20米距离的指定地点射出绳索并固定，接着将人物自身拉向目标地点。  
  
  
附加技能  
该技艺需要角色自行购买  
  
灵气盾  
价格：D+500  
动作：迅捷  
能耗：2点魔力  
射程：5米  
目标：单个友方单位  
持续：一轮  
效果：在持续时间内给予该友方单位5点力场防御。使用者也可以选择不提供防御，转而在射程内一点生成一个持续一轮的落脚点。  
研发（4xp）：现在此能力也可以对敌方单位使用，若如此做，则不提供防御而是转为以盾牌向任意方向进行一次冲撞，若对方失败会额外被击退10米。  
研发（10xp）：能量消耗提升至6点，给予的防御提升沉着属性点。
