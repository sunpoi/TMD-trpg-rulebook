# 楔丸

  

楔丸  
D+500  
特异本质  
从主人龙胤神子·九郎处得到的刀，是苇名的庶家--平田氏的相传之物  
虽曾一度失去，但又重回狼的手中

楔丸的名字里包含着心愿  
即使忍者注定是杀人的宿命  
也不可舍弃仅有的一点慈悲之心……  
这心愿刀刃是否会体察到呢？  
  
武器类别：半太刀  
专业：刀/剑  
武器伤害：6L  
体积：2  
特性：【格挡2】  
【精确招架】：你在进行格挡时声明一个数字，这个数字不能大于此次格挡防御值。当敌方对你造成伤害时，若该伤害值等于你所声明的数字，则该伤害中有4点伤害会向下转化一级（韧性-严重-冲击）。  
每部影片结束时，如果你在影片内触发本特性向下转化伤害的效果，则本特性可转化的伤害点数提升2点。
