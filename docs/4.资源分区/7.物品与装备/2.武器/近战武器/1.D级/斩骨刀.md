# 斩骨刀

斩骨刀  
价格：D+500  
分类：刀  
武器伤害：6L  
武器特性：破甲2，高速1  
武器体积：3  
伤害类型：物理-挥砍  
描述：锐利的大型刃器统称，不限于菜刀，小刀或铊刀也可以算在此类。  
效果：使用该武器进行攻击，附带1点额外的韧性伤害。
