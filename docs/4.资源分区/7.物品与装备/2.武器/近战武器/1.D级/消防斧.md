# 消防斧

  

消防斧  
价格：D+500  
分类：斧  
武器伤害：7L  
武器特性：【破甲7】  
伤害类型：物理-挥砍  
体积：4  
描述：消防专用，被二次加固后不仅能斩断轻薄的障碍物，还能在丧尸潮中杀出一条血路。  
对半砍：使用消防斧进行全力攻击时获得的DP+2。
