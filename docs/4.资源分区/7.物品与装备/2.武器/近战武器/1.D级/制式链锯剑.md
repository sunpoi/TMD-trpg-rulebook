# 制式链锯剑

  

星界军制式链锯剑  
1000  
这是一把体积3，力量需求3，武器伤害4L，【双手】【重型武器】的重剑  
这种制式链锯剑被大量列装为星界军战士的通用装备，在剑柄处有一个扳机，扣动扳机便可以让链锯剑的剑刃旋转。  
扣动扳机是一个迅捷动作，扣动扳机后，在这个回合结束之前，链锯剑都视为再具有破甲2，并在使用链锯剑攻击时，还会再获得1点武器伤害。  
  
阿斯塔特制式链锯剑  
D+500  
这是一把体积4，力量需求6，武器伤害7L，【双手】【重型武器】的重剑  
这种制式链锯剑被大量列装为阿斯塔特战士的通用装备，在剑柄处有一个扳机，扣动扳机便可以让链锯剑的剑刃旋转。  
扣动扳机是一个迅捷动作，扣动扳机后，在这个回合结束之前，链锯剑都视为再具有破甲2，并在使用链锯剑攻击时，还会再额外造成1点不可豁免的某部位肢体妨碍点数。
