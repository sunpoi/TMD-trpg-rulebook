# 蝴蝶盾

  

蝴蝶盾  
D+500  
原典：盾之勇者成名录  
使用蝴蝶作为素材打造的盾牌，带给持有者抵抗麻痹麟粉的能力。  
类型：轻盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡5】  
持有者承受的麻痹异常点数减少3点。  
以花招动作支付2点魔力，持有者在三轮内免疫C级别幻象来源的负面效果
