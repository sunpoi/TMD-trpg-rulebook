# 拟声怪影盾

  

拟声怪影盾  
D+500  
原典：盾之勇者成名录  
形似蝙蝠的盾牌，带给持有者抵抗与放大音波的能力。  
类型：轻盾  
武器伤害：3L  
体积：2  
专业：盾  
特性：【格挡5】  
持有者受到的音波伤害减少3点。  
持有者在可花招动作支付1点魔力以让触及范围内的一个单位下一次使用音波攻击时获得攻击上的3点器械加值。
