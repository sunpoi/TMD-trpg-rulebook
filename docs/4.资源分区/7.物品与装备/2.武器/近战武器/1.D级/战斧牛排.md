# 战斧牛排

  

战斧牛排  
价格：D+500  
分类：斧  
武器伤害：7L  
武器特性：【破甲2】【威猛5】  
伤害类型：物理-挥砍  
体积：4  
描述：这是一个体积为4的，造型如同战斧一般的带骨牛排，对，你没看错，它确实是一块牛排，附带当作柄握的腿骨。  
  
可食用：上面的牛肉不会变质也不会受到污染，进行烹饪即可变成一份美味新鲜的牛排，吃下去的生物会立刻饱腹且满足一日的营养需求，同时因为过量的营养导致身体机能活跃，无需睡眠。  
牛肉被吃掉后的第二天凌晨0点，新的牛肉会在骨棒上长出。  
  
食品级：通常情况下，该物品即使通过安检也不会怎么样，更不会被认为是一把武器，除非你当面使用它进行攻击。
