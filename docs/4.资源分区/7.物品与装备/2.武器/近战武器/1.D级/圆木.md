# 圆木

圆木  
价格：D+500  
分类：锤  
武器伤害：7L  
武器特性：【重型武器】【双手】【威猛3】  
武器体积：4  
伤害类型：物理-钝击  
描述：战斗中可以使用、但本来不是武器、太重不方便使用。但是、眩晕概率高、打击属性可以突破坚固的甲壳。  
效果：使用该武器攻击命中后，附带1点不可豁免的晕眩不良点数；持有者在攻击时，可以将攻击DP3：1转为攻击检定上的附加成功数，最多转换等同持有者【传奇力量】\*1个附加成功数。
