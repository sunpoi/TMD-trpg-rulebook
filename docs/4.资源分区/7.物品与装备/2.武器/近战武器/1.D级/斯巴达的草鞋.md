# 斯巴达的草鞋

斯巴达的草鞋  
特异本质  
D+500  
描述：一对带有红色绑腿的红底草鞋，比看起来要结实得多，便于行动的同时，还有一股奇妙的感觉在你的胸口酝酿，让你想要在踹人的时候大喊一声：“This is TMD！”  
效果：这是一件占据鞋袜位的武器，还能为穿戴者带来较弱的防护。  
同时，草鞋上寄宿的神秘斯巴达之魂会让穿戴者以腿进行攻击检定时，提供基于天生武器攻击的3点附加成功，攻击伤害从冲击转为严重伤害，护腿提供的加成皆为器械加值。
