# 叶盾

  

叶盾  
D+500  
原典：盾之勇者成名录  
形似芭蕉叶的盾牌。吸收草叶解放的形态，带来了采集的知识。  
类型：轻盾  
武器伤害：1L  
体积：2  
专业：盾  
特性：【格挡5】  
【采集强化】：持有者采集的药草在制造恢复性药剂时，可恢复的伤势增加1点，医疗点不变。  
你的手艺技能检定获得1点附加成功。
