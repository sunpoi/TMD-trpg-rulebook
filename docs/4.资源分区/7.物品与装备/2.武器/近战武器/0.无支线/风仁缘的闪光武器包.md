# 风仁缘的闪光武器包

风仁缘的闪光武器包  
300  
TMD主神不知道从那个时空缝隙里打捞出来的武器包，所有者似乎是个会给武器贴灯条的神奇怪人。  
挑选总价值不高于250的具有基础模板的多把现实近战武器，获得之，它们全都具有自定义灯条改造（在武器的任何地方内置灯条，永不间断的发出霓虹灯光（可自定义颜色）  
你将获得一个体积3的具有夜光涂料的帆布背包，但装体积过大的武器会漏出来。
