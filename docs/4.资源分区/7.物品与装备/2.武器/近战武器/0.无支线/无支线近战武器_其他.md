# 无支线近战武器-其他

无支线近战武器-其他

S  
十八铜人的折凳  
科技本质

资源价格：1000  
武器分类：临时武器  
武器伤害：3L  
伤害类型：物理-钝击  
随处可见的普通折凳，并带有一丝时光的气息，凳腿是已经生了锈的铁管。  
折凳，可藏于民居之中，随手可得；亦可坐于其上，隐藏杀机；由于不算凶器，即使被警察抓了，他们也告不了你。被誉为市井械斗七武器之首。  
特性：【械斗杀器】

【械斗杀器】  
使用时，你需要先从凳子上起身，才能起凳子砸人，这会消耗掉你的标准动作和移动动作，此次攻击中会附带一点流血点数，以标准形式豁免。于此同时，你的此次攻击将视为拥有无支线级别的措手不及。  
特殊的是，该凳子需有人坐在上面才能生效，同时如果非使用者本人坐在上面，则需要进行一次平衡对抗，胜出才能抽出凳子。其余情况由st判断

W  
钨钢除草机  
科技本质

资源价格：1000  
武器分类：巨镰（使用专业：镰刀）  
武器伤害：4L  
伤害类型：物理-挥砍  
武器射程：6M  
“哦，老天这破玩意儿能派上什么用场?”  
“起码他还能拿来清除一下前行道路的杂草，不是吗?”  
反抗军为对抗丛林暴君亚利姆而制造的便宜机械钨钢产品的其中一项，用廉价的钨钢能源核心和钨钢金属废料制成，,或许正如它的名字所言，用来除草的效果似乎比杀敌更好?  
特性：【钨钢能源】

【钨钢能源】  
这把镰刀的纹路上蚀刻着钨钢能源回路，通过核心源源不断的能源供给，使得这把镰刀在挥动时可以发射出一道与镰刀刀刃等宽的绿色能量波，能量波可以飞行6米远，能量波的伤害并不与镰刀本身叠加，旦对命中的敌人只能造成实际伤害的一半 。
