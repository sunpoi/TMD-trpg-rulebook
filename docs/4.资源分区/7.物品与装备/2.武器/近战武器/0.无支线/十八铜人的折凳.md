# 十八铜人的折凳

  

十八铜人的折凳  
1000  
科技本质

随处可见的普通折凳，并带有一丝时光的气息，凳腿是已经生了锈的铁管。

这这视为一把临时武器，伤害3l。使用时，你需要先从凳子上起身，才能起凳子砸人，这会消耗掉你的标准动作和移动动作，此次攻击中会附带一点流血点数，以标准形式豁免。于此同时，你的此次攻击将视为拥有无支线级别的措手不及。

备注：折凳，可藏于民居之中，随手可得；亦可坐于其上，隐藏杀机；由于不算凶器，即使被警察抓了，他们也告不了你。被誉为市井械斗七武器之首。

特殊的是，该凳子需有人坐在上面才能生效，同时如果非使用者本人坐在上面，则需要进行一次平衡对抗，胜出才能抽出凳子。其余情况由st判断
