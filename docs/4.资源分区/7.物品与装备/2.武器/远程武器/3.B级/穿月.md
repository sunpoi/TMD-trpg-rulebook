# 穿月

穿月  
B+2500  
魔幻本质  
描述：这是一把金色的金属长弓，没有弓弦，有一只金色的把手，弓身正中是向前反弓的两枚尖牙般的金属月弧，有红色的魔兽晶核整齐地镶嵌在弓身上，在夜晚会散发出美轮美奂的白光。握着这把由月岭魔金铸成的弦月型金色长弓，仿佛能感觉到月光在自己的体内流淌，无弦的弓在手中等待着，将手搭在那不存在的弦上，魔力会相互吸引，在手中幻化出银白的魔力弓弦与能量箭矢，松手，寂静无声，月光，照耀在敌人穿孔的身躯之后，消散。  
分类：长弓  
武器伤害：22L  
基础效果：【双手】【高速3】【破甲1】  
武器体积：3  
伤害类型：物理-穿刺  
特性：【魔弓穿月】  
【魔弓穿月】：该武器在进行攻击时可以消耗一个花招动作进行一次瞄准；该武器可以正常搭箭射击，武器中的魔力会自动形成弓弦，在射击时，可以支付至多6点魔力，以此换取攻击上的额外【支付魔力】\*2L武器伤害提升。  
仅在具有月亮存在的夜晚场景时，该武器具有以下额外能力：  
无需搭箭，可以直接凝聚月光进行射击，视为具有【无限子弹】，能量汇具而成的子弹具有【法破】特性，并且在月光的加持下，你的敏感范围提升一倍，攻击加骰提升一级（至多为8加骰），在进行攻击时，若自身魔力池不为空，且这次攻击至少消耗了1点魔力，这次攻击额外具有4点附加成功。
