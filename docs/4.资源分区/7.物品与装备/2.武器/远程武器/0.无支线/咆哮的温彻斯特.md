# 咆哮的温彻斯特

  

咆哮的温彻斯特  
750  
特异本质  
武器伤害4L 装弹数11+1 射程40米  
攻击距离不超过10米内的单位时获得2威猛  
咆哮的温彻斯特是一把枪身锈迹斑驳的老式霰弹枪，它在被使用时会发出令人恐惧的、如同咆哮般的枪声，这意味着它即使安装消音器也不能发挥其效力。  
咆哮的温彻斯特的枪柄上有用刀刻下的划痕，枪的前任主人在战场上每击毙一名敌人，就会刻下一道划痕记录战功。而它在每场影片结束后都会再自然的浮现出一道划痕，这些划痕的初始为零条，它每拥有一条划痕，就会在使用咆哮的温彻斯特进行射击时再提供1DP士气加值。  
特殊的，咆哮的温彻斯特也可以作为物品改造的模板使用，花费对应的分数，将此霰弹枪模板移植至其他霰弹枪上，这不会使它的武器伤害被继承，而是会正常提供其效果和已有的配件改装。  
咆哮的温彻斯特可以正常进行枪械改装，每项25分的改装使用250分，每项50分的改装使用500分，仍然遵循改装规则，此时它视为现实霰弹枪价格。
