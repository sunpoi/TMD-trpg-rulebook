# 无支线远程-其他

无支线远程武器-其他

T

特殊武器包  
100  
购买后一次性获得胡椒瓶手喷、四发转轮手枪、组装手弩、刃杖。  
这些袖珍武器也可以单独购买，一把50分。

胡椒瓶手喷  
分类：手枪  
本质：科技本质  
武器伤害：2L  
武器特性：【高速5】【攻击距离不超过15米内的单位时获得2威猛】【装弹等级上升一级】  
武器体积：2  
伤害类型：物理-穿刺  
武器射程：25M  
装弹数：1

四发转轮手枪  
分类：左轮手枪  
本质：科技本质  
武器伤害：3L  
武器特性：【高速5】【使用时视为已经进行过一次瞄准】  
武器体积：2  
伤害类型：物理-穿刺  
武器射程：30M  
装弹数：4

组装手弩  
分类：弩  
本质：科技本质  
武器伤害：2L  
武器特性：【高速3】【组装手弩】  
武器体积：1  
伤害类型：物理-穿刺  
武器射程：15M  
装弹数：1  
效果：  
【组装手弩】：  
使用它的首个回合视为具有无支线等级的措手不及 可以在不使用时拆开存放，组装手弩是一个移动动作

刃杖  
分类：短棍/迅捷剑  
本质：科技本质  
武器伤害：  
短棍 3L  
迅捷剑 3L  
武器特性：【轻灵】【格挡1】 【破甲1】【刃杖】  
武器体积：2  
伤害类型：  
短棍 物理-钝击  
迅捷剑 物理-挥砍  
效果：  
【刃杖】：  
刃杖同时是短棍和轻剑，外观是普通的手杖 刃部分藏在手杖之中 切换刃杖的刃是一个迅捷动作
