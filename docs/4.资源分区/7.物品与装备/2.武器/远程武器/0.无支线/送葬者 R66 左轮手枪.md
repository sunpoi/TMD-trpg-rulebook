# 送葬者 R66 左轮手枪

  

送葬者 R66 左轮手枪  
价格：750分  
本质：科幻  
原典：GTFO  
即使特殊武器的爆发力比其他武器更为优秀，也不意味着可以忽略精准度所带来的优势。这把左轮手枪是为了那些追求精度和爆头的射击者所设计的。依靠高精度和更优秀的弹药补充量，一次爆头就可以直接秒杀脆弱的对手。  
它是一款强大而紧凑的大型左轮手枪，使用6发可拆卸弹巢发射10㎜子弹。它在外观看起来沉重粗糙，事实上也确实如此，为了应对后坐力带来的纵向跳动，枪管的下方加装了整整一排的配重模块，让他的枪管看上去粗壮了不少，它缺少传统的那种扳机护圈设计，设计也相对简单，带有分段式套筒和基础的机械瞄具。送葬者 R66 左轮手枪的保险位于枪身的后侧，在弹巢的斜后方。握把上本该带有所属开发商的LOGO，但不知为何已被磨平。  
  
生产概况  
  
制造商 : 未知  
  
类型：左轮  
  
弹药口径：卡尔 10mm 毫米  
  
最大弹匣容量：6  
  
伤害：4L（大型左轮手枪）  
  
射程：30米  
  
力量需求：4  
  
优势：爆头  
这把枪威力强大，且善于穿透坚硬的钙、磷质结构，并恰到好处的在柔软粘稠的大脑中旋转弹射，从而尽可能的造成伤害。在射击者使用这把枪命中头部时（即对头部的部位攻击，对眼睛和太阳穴的部位攻击也同样满足前提），可选择放弃该次部位攻击所携带的异常点，选择将该次攻击中的造成的，将放弃异常点等量的伤害加深一级，最高为恶性。能以此法转化的上限为1。  
显然的，若目标没有头部或类似的器官，则这一效果无法生效，具体由ST决定。
