# 麦卡顿的疯狂

麦卡顿的疯狂 750  
“用枪管给那群狗娘养的带来深喉般的热情吧！”  
这是一把武器伤害4L的连发霰弹枪。区别于一般的连发霰弹枪，它的基础射程仅有7米，弹鼓却有50发的填弹量，无法进行单发或点射射击，每次攻击都必须以【连射】的方式激发。  
以麦卡顿的疯狂进行的攻击无法如同一般枪械一般获得枪械射击本身带来的的5点高速优势，相反的，当麦卡顿的疯狂攻击处于1米内的目标时，获得5DP器械加值和1点攻击上的附加成功，且此时【连射】特性不会额外波及其他目标；每离远1米，所因此获得加值便会减少1DP，最低为0。
