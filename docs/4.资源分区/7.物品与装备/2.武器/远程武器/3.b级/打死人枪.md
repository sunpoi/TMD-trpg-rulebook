# 打死人枪

打死人枪  
价格：B+2000  
分类：霰弹枪  
武器伤害：20L  
武器特性：高速5，攻击距离不超过15米内的单位时获得2威猛  
弹容量：11+1  
武器体积：4  
速射等级：长点射  
基础射程：40米  
伤害类型：物理-穿刺  
描述：连接复数的超级霰弹枪，同时开火放出无数霰弹的枪。在其风暴范围内的肉块无不成末。  
效果：该武器进行的攻击同时覆盖前方扇形射程范围内的所有敌对单位，目标使用反射豁免而非正常防御；使用该武器进行攻击，额外获得等同你【传奇敏捷】\*1个附加成功，攻击加骰提升1级计算（上限8加骰）。
