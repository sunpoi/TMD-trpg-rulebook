# 百臂巨人

【百臂巨人】  
B+2000  
它们是巨人，有五十颗头颅，一百只手臂，长相丑陋而力大无穷，凶猛而巨大的怪物——  
它们的本质，是带来决定性胜利的庞然巨物  
你获得一张由青铜弓和一只青铜手臂组成的长弓，它坚实，可靠，并且肌肉的纹理明显。  
武器伤害：18L  
基础：【高速3】【破甲1】【双手】【无速射】  
体积：3  
装弹量：1  
基础射程：40米  
持有此弓会使得你拥有百手巨人之力  
【κατν（一百）χερ（手）】  
Centimani，前第马尼  
向他们投出百块山石  
你射出的箭矢将如巨石一般沉重，以这个武器造成的攻击伤害转为物理-钝击。  
并且，每当你以百臂巨人战胜3名同级敌人时，弓身上将出现一只额外的手臂，这些手臂不提供抓握等功能，只是在你攻击时，它们将额外向其他敌人投掷出攻击力相当的巨石——你可以以百臂巨人上的手臂数量的新增一个其他攻击目标效果，上限98，初始即拥有1只手臂。  
【κατγχειρε（百臂巨人）】  
百手的巨人啊，为我带来胜利  
手持此弓时，你可以选择一个自由动作引动与关闭百臂巨人的力量  
变为百臂巨人时，会使得你的体型在你的内在等级为b/a/s时获得9/12/15点体积，这些体积不会提供生命值上的加成，如虚拟体积般计算。  
同时，弓体上的手臂们将进行解体并附着在你的双手手臂上，当你做出拉弓的动作时，它们也将帮助你进行蓄力，这会使得你的力量射击在你的内在等级为b/a/s时获得9/12/15点增强加值，此时百臂巨人依旧占据你的双手武器位。  
力量的出色也会使得你能够将攻击投射到更远的地方，你在力量射击时额外获得60米的额外射程。
