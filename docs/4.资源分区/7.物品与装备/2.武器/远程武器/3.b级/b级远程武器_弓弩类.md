# B级远程武器-弓弩类

B级远程武器-弓弩类

B  
【百臂巨人】  
价格：B+2000  
分类：弓  
本质：神话本质  
武器伤害：18L  
武器特性：【高速3】【破甲1】【双手】【κατνχερ】【κατγχειρε】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：1  
描述：  
它们是巨人，有五十颗头颅，一百只手臂，长相丑陋而力大无穷，凶猛而巨大的怪物——  
它们的本质，是带来决定性胜利的庞然巨物  
你获得一张由青铜弓和一只青铜手臂组成的长弓，它坚实，可靠，并且肌肉的纹理明显。  
持有此弓会使得你拥有百手巨人之力  
效果：  
【κατν（一百）χερ（手）】：  
Centimani，前第马尼  
向他们投出百块山石  
你射出的箭矢将如巨石一般沉重，以这个武器造成的攻击伤害转为物理-钝击。  
并且，每当你以百臂巨人战胜3名同级敌人时，弓身上将出现一只额外的手臂，这些手臂不提供抓握等功能，只是在你攻击时，它们将额外向其他敌人投掷出攻击力相当的巨石——你可以以百臂巨人上的手臂数量的新增一个其他攻击目标效果，上限98，初始即拥有1只手臂。  
【κατγχειρε（百臂巨人）】：  
百手的巨人啊，为我带来胜利  
手持此弓时，你可以选择一个自由动作引动与关闭百臂巨人的力量  
变为百臂巨人时，会使得你的体型在你的内在等级为b/a/s时获得9/12/15点体积，这些体积不会提供生命值上的加成，如虚拟体积般计算。  
同时，弓体上的手臂们将进行解体并附着在你的双手手臂上，当你做出拉弓的动作时，它们也将帮助你进行蓄力，这会使得你的力量射击在你的内在等级为b/a/s时获得9/12/15点增强加值，此时百臂巨人依旧占据你的双手武器位。  
力量的出色也会使得你能够将攻击投射到更远的地方，你在力量射击时额外获得60米的额外射程。

C  
穿月  
价格：B+2500  
分类：弓  
本质：魔幻本质  
武器伤害：22L  
武器特性：【双手】【高速3】【破甲1】【魔弓穿月】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：1  
描述：这是一把金色的金属长弓，没有弓弦，有一只金色的把手，弓身正中是向前反弓的两枚尖牙般的金属月弧，有红色的魔兽晶核整齐地镶嵌在弓身上，在夜晚会散发出美轮美奂的白光。握着这把由月岭魔金铸成的弦月型金色长弓，仿佛能感觉到月光在自己的体内流淌，无弦的弓在手中等待着，将手搭在那不存在的弦上，魔力会相互吸引，在手中幻化出银白的魔力弓弦与能量箭矢，松手，寂静无声，月光，照耀在敌人穿孔的身躯之后，消散。  
效果：  
【魔弓穿月】：  
该武器在进行攻击时可以消耗一个花招动作进行一次瞄准；该武器可以正常搭箭射击，武器中的魔力会自动形成弓弦，在射击时，可以支付至多6点魔力，以此换取攻击上的额外【支付魔力】\*2L武器伤害提升。  
仅在具有月亮存在的夜晚场景时，该武器具有以下额外能力：  
无需搭箭，可以直接凝聚月光进行射击，视为具有【无限子弹】，能量汇具而成的子弹具有【法破】特性，并且在月光的加持下，你的敏感范围提升一倍，攻击加骰提升一级（至多为8加骰），在进行攻击时，若自身魔力池不为空，且这次攻击至少消耗了1点魔力，这次攻击额外具有4点附加成功。

J  
狙击弩  
价格：B+4000  
分类：弩  
本质：科技本质  
武器伤害：18L  
武器特性：【高速3】【破甲1】【双手】【狙击弩】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：1  
描述：  
长距离射击特化、可以从远方隐藏身形射击的弩，主要由暗杀部队使用。  
效果：  
【狙击弩】：  
使用该武器进行攻击时，射程距离增加X个基础射程的距离，X＝【敏捷属性/2】；该武器的攻击附带B级措手不及的效果影响。

S  
岁星监得  
价格：B+2000  
分类：弓  
本质：魔幻本质  
武器伤害：18L  
武器特性：【高速3】【破甲1】【双手】【木德】【岁罚】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：30M  
装弹数：1  
描述：  
岁星者，木也，主春，日之魂、水之精炁，色苍苍而有光。  
岁阴左行在寅，岁星右转居丑，与斗、牵牛晨出东方，名曰监德。  
下御山林草木、龙蛇龟鳖水族、风雷之事，义失者，罚出岁星  
效果：  
【木德】：  
万物生发、济损为舒，以该武器进行射击时，木德生箭，化作碧绿灵箭，视为具有【无限子弹】，手持此弓，可得乙木生生不息之妙，获得【快速医疗3】（36医疗点数)。  
每轮一次，你可以在以木德灵箭射击之后，以自由动作立即朝一名射程内的友军射出一箭，若如此做，该武器提供的【快速医疗3】将会转移给那名友军生效，持续一轮。  
【岁罚】：  
乙木生雷、罚出岁星，在进行射击时，使得本次攻击附着岁星罚力，炁箭化雷，射击伤害类型从物理-穿刺与雷电伤害中取优结算，每轮首次以此发起的攻击将附带额外1点严重雷电伤害；若持有者具备气、炁或类似的东方能量池，可以针对此次攻击额外支付3点能量，令本次攻击再额外附带等同于自身传奇风度的严重雷电伤害。
