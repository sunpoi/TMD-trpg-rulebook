# 卢修斯14型(重型)激光枪

  

卢修斯14型(重型)激光枪  
B+4000  
科技本质  
  
卢修斯XIV型激光枪是地狱枪的一个变种，也是克里格死亡军团掷弹兵的标准武器 用来承担精准射击任务，由于射击时巨大的热量，因此在射击时枪如其名（Hot-shot），所以在枪管中还包含了一个热交换管道的冷却系统，加压的冷却剂将通过管道使枪管降温。但是在不射击时，枪管内的冷却系统却使得枪管温度很低，因此普通人可能要戴上手套防止冻伤，地狱枪在建造过程中需要多种先进的组件，包括增强合金枪管，以承受在发射过程中产生的更大的热量，因此，帝国中只有少数铸造世界拥有生产这些强大武器的技术诀窍或必要的材料，所以也仅仅发给那些经验丰富的战士。  
  
这种枪的使用者通常会背着一个大背包，背包里有一个独立的电源，用来给地狱枪供能并提供更多的伤害输出，在无标准电源包供弹时，该地狱枪本身的所装载的电池只能支持12发射击。和标准的激光枪不一样的是，其配备了一个安全装置，在过热时会自动切断电源，直到枪管再次下降到安全范围内，所以卢修斯14型(重型)激光枪仅可以在半自动模式下工作，每次扣动扳机就发射一枪。  
  
  
重量：3.7KG  
  
长度：1400毫米  
  
体积：3  
  
专业：自动步枪  
  
伤害：20L  
基础射程：200米  
  
弹夹类型：使用标准能量电池供能（可射击12发）  
  
射击类型：单发  
  
瞄具：先进光学瞄准器  
  
先进光学瞄准器 ：卢修斯14型(重型)激光枪拥有一个复杂的先进瞄准镜，包括一个4倍镜以进一步增加武器的精度，这意味着使用该武器瞄准时，人物的【视距】提升四倍，这并不意味着人物的敏感范围的到提升，但仅在使用视觉定位时，人物可以定位四倍自身敏感范围距离内的敌人。  
  
先进激光产生系统：该枪的射击系统允许激光在60兆赫的能量范围内运行，其强大的能量和穿透力，甚至可以击穿地狱惧兽的护甲，卢修斯14型(重型)激光枪 还具有【能量武器】，【超级贯穿】特性。  
\------------------------------------------------------------------------------------------------  
  
标准能量背包  
价格：1250分  
  
重量：15kg  
  
长度：800\*700\*200毫米  
  
体积：3  
  
在由能量背包供能时，卢修斯14型(重型)激光枪在正常的射击功率和武器状态下可以射击200发，当标准能量背包消耗殆尽后，人物可以使用能量背包提供的一个快速充电器，该充电器可以适配任意电源。仅需一个小时就能充满，且在该能量背包供能下，卢修斯14型(重型)激光枪可以使用【短点射】。  
  
值得注意的是，虽然标准能量背包的电池组，可以向激光枪能源电池一样直接依靠热电转换进行充电，但这么做无异有着损坏电源的风险，考虑到其昂贵的造价，如果你看到你的任何队友在这么做，那么不要犹豫，请举起你手中的武器，像是消灭异端一样消灭他。
