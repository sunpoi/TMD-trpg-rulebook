# 反物质步枪

反物质步枪  
价格：B+3000  
分类：狙击枪  
武器伤害：20L  
武器特性：高速5，破甲3，双手  
弹容量：6+1  
武器体积：5  
速射等级：-  
基础射程：200米  
伤害类型：物理-穿刺  
描述：为了贯穿战车装甲而造的巨枪。据说它的威力足以媲美小型导弹。  
效果：使用该武器进行攻击时，附带【湮灭】与【超级贯穿】特性；该武器进行的攻击附带等同你【传奇智力】\*5点额外的破甲。
