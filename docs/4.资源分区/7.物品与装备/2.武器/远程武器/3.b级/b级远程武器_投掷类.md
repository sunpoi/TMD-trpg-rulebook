# B级远程武器-投掷类

B级远程武器-投掷类

H  
黄金追魂蹄（Golden Chaser's Hoof）  
价格：B+2000  
本质：魔幻本质  
分类：投掷武器  
描述：  
由秘金锻造，表面镌刻着流动的符文纹路，在阳光下会折射出刺目的金光的金色马蹄铁，蹄铁内侧镶嵌三颗血色红宝石，呈三角排列，边缘被打磨得异常锋利，重量仅相当于普通铁蹄的一半，便于投掷。相传是古代游牧战神「赫尔卡」的武器，他用此物猎杀了吞噬草原的巨兽。  
  
【无垠游牧】：黄金追魂蹄上面的游牧神将祝福着使用者，但......这不是毫无代价的。  
  
武器伤害：18L  
武器特性：【回力】【法破】【无垠游牧】  
体积：1  
射程：敏捷属性\*5米  
伤害类型：物理-挥砍  
效果：  
【无垠游牧】：使用者的敏捷相关检定获得【8加骰】，且敏捷相关检定获得等同于使用者敏捷属性的器械加值，使用者将被占据1点意志力上限。
