# 【罗兰系列组合乐器】

【罗兰系列组合乐器】  
科技本质

注：罗兰V-Combo VR-730与罗兰Fantom-08并不是同一种武器，因此罗兰Fantom-08无法基于罗兰V-Combo VR-730差价升级。

D级  
罗兰V-Combo VR-730

D+500  
武器分类：见下  
武器伤害：6L  
伤害类型：音波  
武器体积：4  
描述：一把被使用过很久的键盘，即使外行也能看出被主人保养的很好。  
特性：【双手】【乐器】【支架】【野蜂飞舞】【灵魂共鸣之声】【概念化-Oblivionis的决心】

【乐器】  
这把琴是一把乐器，其武器伤害可用于表达技能进行的攻击检定。

【支架】  
使用者可以以一个移动动作打开支架，将这把琴架在原地。此时该法器失去【双手】特性，并且力量需求降低为0（没错，就是没有需求）。但同时琴本身不再能移动，除非你的力量属性已达到其原本力量需求的两倍。  
你可以额外花费500点购买自锁式滚轮，这可以使该法器在【支架】状态随你移动，此时你的移动速度减半处理。自锁式滚轮无法在困难地形生效，无论使用者本人是否具有相关免疫。

【野蜂飞舞】  
该武器具有【长点射】特性。

【灵魂共鸣之声】  
优秀的乐章会引发听者灵魂深处的共鸣。该武器进行的检定获得【法破】。该武器伤害提供的破防类型为破魔。

【概念化-Oblivionis的决心】  
跌落尘埃又如何。弹一曲颂歌，我命只由我。  
在你因心智类异常点数进入严重状态、并恢复正常后，你获得了这把琴的认可。此后你可以把这把琴变为一个拇指大小的挂坠。你可以把该挂坠挂在其他键盘琴上，此时其他琴获得该琴的【野蜂飞舞】【灵魂共鸣之声】特性（若如此做，该资源将视为一项物品改造）。
