# “妖器”银刃

“妖器”银刃  
分类：投掷武器  
描述：  
某杀人鬼小姐在伦敦游猎时所使用的武器，本体为一块精致的怀表，只要本体没被摧毁，就能够消耗魔力来凝聚出投掷用的飞刀，银刃并不是代指拥有的材质，而是代表飞刀上闪耀着的凛冽寒光

初步掌握  
价格：D+500  
描述：  
虽然听起来很麻烦，但操作起来很简单的，输入魔力，想象出飞刀的外形，看，很简单吧，现在就只需要扔出去就好了  
效果：  
该武器会占据持有者的1点意志池上限。  
自由动作消耗1点魔力，在你的手中凝聚出一把持续一轮，体积1的银刃

凝聚而出的银刃拥有6L武器伤害，具有【破甲5】【法破】特性，造成物理-穿刺伤害，在投掷的一瞬间时可以分化为三柄，如【短点射】一般速射。

小有所成（差价升级）  
价格：C+1000  
描述：是不是觉得一把飞刀压制力不够呢，其实很简单，还记得之前构造一把的配比吗，等比例的增加就好了。当然不要忘了，眼要准，手要稳，力度要狠。  
效果：  
该武器会占据持有者的1点意志池上限。  
自由动作消耗1点魔力，在你的手中凝聚出一把持续一轮，体积1的银刃

凝聚而出的银刃拥有，11L的武器伤害与【破甲5】【法破】的特性。

投掷攻击前可以消耗2点魔力，额外生成至多4把银刃，这些银刃在投掷时会幻化成复数，这会令你这次进行的投掷攻击速射等级提升为【连射】。

投掷攻击前可以消耗1点魔力，这次攻击可以同时对攻击范围内的另一个目标投掷银刃造成伤害。

投掷攻击前可以消耗4点魔力，令银刃的【破甲5】改变为【传奇力量】\*5点破甲

登堂入室（差价升级）  
价格：B+2000  
描述：  
最后再教你一个小技巧，将魔力的联系想象为一根根丝线连接在飞刀之间，这样你就能随心的操控它出现在你想出现的任何地方了。还有一个温馨提示，做好魔力管理，方便在任何时候都留有余力。加油活下去吧，继任者～  
效果：  
该武器会占据持有者的1点意志池上限。  
自由动作消耗1点魔力，在你的手中凝聚出一把持续一轮，体积1的银刃

凝聚而出的银刃拥有，18L的武器伤害与【破甲5】【法破】和【9加骰】的特性

投掷攻击前可以消耗1点魔力，额外生成至多9把银刃，这些银刃在投掷时会幻化成复数，这会令你这次进行的投掷攻击速射等级提升为【连射】，并使这次攻击加骰等级提升一级（至多为8加骰）。

投掷攻击前可以消耗1点魔力，这次攻击可以同时对攻击范围内的另一个目标投掷银刃造成伤害。

投掷攻击前可以消耗1点魔力，令本次攻击中，银刃带来的破甲变为【传奇力量】\*5点破甲

投掷攻击前可以消耗3点魔力，令本次攻击获得C级的传送攻击效果

投掷攻击前可以消耗3点魔力，令本次攻击的射程提升【传奇敏捷】\*10m的射程
