# 支援无人机

支援无人机  
价格：D+500  
描述：支援用无人机，上面配备了轻量化的小型机枪，可以盘旋在你周围对目标发动攻击。  
效果：这是一种特殊的武器，画面表现上为盘旋在你周围的无人机（体积4）。  
你可以操纵无人机搭载的轻机枪进行攻击，使用智力+器用：无人机+辅助专业：弹道学进行攻击，武器伤害7L，基础射程为100米，速射等级为连射，攻击具有【高速5】【装弹动作上升一级】【贯穿】【无限子弹】特性，伤害类型为物理-穿刺，可以额外攻击两个目标。  
由于操纵限制，你至少要使用一只手进行操纵，且无人机只能盘旋在你周围，无法离开你的身侧。  
升级（C+1000）：无人机搭载轻机枪伤害提升至15L，现在无人机可以在射程范围内进行曲线机动，使你进行的攻击弹道变为曲线型攻击，这仍然需要你能看到才可以使无人机移动攻击至对应位置（这意味着即使曲线可以攻击到障碍物后，你没有视野线也不行，无人机并不和你共享视野，即使已经可以在射程内移动了）。  
现在的无人机可以搭载你飞行，使你获得等同自身基础移动速度的飞行能力，机动性普通。
