# M45战术霰弹枪

  

M45 Tactical Shotgun?/?M45战术霰弹枪  
  
Tactical Shotgun, 8 Gauge, Pump Action, M45 /?M45八号口径泵动战术霰弹枪，由米斯利亚军械库制造。用于大多数近距离作战场景（特别是在敌占的城市地区），以及登舰行动。大反叛期间，它曾在城市地形环境下的许多军事行动中使用，并受到使用者的广泛好评。它后来在人类星盟战争期间和战后被继续广泛使用。

【制止力】  
M45战术霰弹枪是一部双管式、不可拆卸的内置管状弹仓，使用8铅径弹药。它是一种特殊用途武器，用于交战范围有限的情况，以及不需要过度火力，但需要输出最大制止力的情况。 当使用M45战术霰弹枪对目标造成伤害时，每造成1点伤害，被攻击的对象将被击退1米，对方可以选择以力量+运动：反射进行对抗判定抵挡，或者以敏捷+运动：反射进行对抗避开（或至少抵消冲撞的力道）
