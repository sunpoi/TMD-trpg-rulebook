# 爆弹枪

爆弹枪  
描述：该如何形容这把爆弹枪？其外观粗野到不比将发射装置焊死在把手上好到哪里去，你很难看出这是一把霰弹枪而不是什么别的玩意。  
然而，即使是这种形式的武器却依旧有着良好的性能，只要塞把土就能当做弹药把敌人射成筛子，枪身可以拿来开颅。最关键的是，它可不像是那些娇贵的军械，哪怕是在沙漠中也只需要抖抖枪上的沙子就能继续使用了。

  
价格：D+750  
分类：霰弹枪  
武器伤害：7L  
武器特性：【高速5】【攻击距离不超过15米内的单位时获得2威猛】  
装弹量：11+1  
速射等级：【连射】  
射程：40米  
体积：4  
伤害类型：物理-穿刺

  
【粗犷爆弹】：该武器采用了独特的设计结构，把发射装置焊在了把手上，除了外观风格极其野性外，需要填装子弹的时候你只需要从地上抓一把土或者沙子，石头，又或者是别的东西塞进去就能在剧烈的火光和噪音中将敌人打成筛子，因此该武器具有【无限子弹】特性。

  
【狂野结构】：该武器的枪身特别适合作为钝器使用，因此这把武器还可以作为棍来使用。  
以分类为棍的武器使用时，武器伤害为6L，具有【格挡1】【长柄武器】，伤害类型为物理-钝击，不具有枪械武器的效果，但攻击时可以如同具有【短点射】处理，无需消耗弹药，画面表现为抡起枪身连续殴打。

  
升级：C+1000  
枪械武器伤害提升至15L，作为棍使用时的武器伤害提升至14L，无论是作为哪种武器使用，攻击时额外附带【破甲5】，造成伤害转为使目标强韧豁免，而非正常防御流程。  
在画面上，改良后的爆弹枪性能更加优秀，所发射的子弹散布更加均匀，使对方在避无可避的情况下只能硬抗。
