# 真相

真相  
科技本质  
D+500  
描述：  
"再问你一句，为什么要这么做。"  
"事到如今，还有这个必要吗？"  
"……你不该来这里。"  
"我有我的打算。"  
"到现在你也不愿意说真话吗？"  
"你和我一样。"  
"……"  
"……"  
"人说谎言，"  
"枪说真相。"  
砰。

这是一把银白色枪身，有着牛皮手工握把的左轮手枪，同时只能装填6发子弹，体积3，基础射程30米，武器伤害7L，具有【高速5】的基础特性。  
这把枪的左侧枪管上有一行淡淡的英文"TRUTH"  
这把武器必须以移动动作上弹，这意味着任何降低装填动作的效果对该武器不起作用。  
上弹后的首次瞄准会为该武器额外提供1dp瞄准加值，虽然该武器的装弹量是6枚，但这并没有实际的作用，事实上该武器每次装弹仅能射击一次，这次攻击会具有【9加骰】，每次射击会同时射出5枚子弹，这视为该武器的速射等级为【长点射】，并获得相应的益处。射出的5枚子弹无论装填前形态如何，在射出后子弹的底部会依次写上"T,R,U,T,H"这五个英文单词，枪械中遗留的那枚子弹上会写着"LIE"，若取出这枚固定不会被射出的子弹，该武器的所有特性将不会生效，仅留下基础属性。

此外，该武器附带左轮手枪专用装弹器，这仍然无法使其降低装填动作。
