# 巨鸟狩

巨鸟狩  
魔幻本质  
D+500  
类型：重弩  
伤害：6L  
类型：穿刺  
体积：3（手持形态）/6（架设状态）  
装弹量：1  
射程：40米  
黑金硬木作成的十字巨弩，既有在手持形态，又有变形为临时弩车的形态。  
特性：高速3，破甲1，双手，重型武器，【重装 架设】  
【重装 架设】：巨鸟狩通常只能为手持形态，在战斗中使用者可以以一个迅捷动作展开/收回巨鸟狩的架设形态，在使用者所处的地面上架设具有抵消后座力作用的木轮。  
架设状态下，使用者在攻击时可以将攻击上的至多2点附加成功以1：4的比例转化为攻击检定上的器械加值，如果以此法转化了2点附加成功，该次攻击额外具有5点高速。
