# 咙啸嚎咆

IDEA-E-191-『咙啸嚎咆』  
科技本质  
A+4000  
分类：左轮手枪  
武器伤害：34L附加  
武器特性：【重型】【高速5】【无限子弹】【9加骰】  
伤害类型：音波  
基础射程：20米  
装弹量：无  
体积：2  
描述：在一次偶然的契机下，实验室的试做品忽然勾连到了埋藏在深层地底世界中的古老异兽尸骸，它的咆哮声响彻了约为两个■■市的总面积的范围，造成了至少75万人的死伤，并导致了后续几年内连续数次量级至少为7级的地震的产生，损失不计其数。  
研究所因祸得福，成功从中提取了巨兽残留的意志与其生前的能力，将其作为该武装的异灵(Sprite)『无限悲怆哀嚎苦痛世界之龙(Itzamna)』记录下来，从而打造出具有咆哮之力的武装，编号E-191，代号与异能同名，定为——『咙啸嚎咆(Nukuch rugido yaj óolal)』。  
这是一把通体漆黑的左轮手炮，有金色的弧线在其身作为能源传递的动线闪烁着，这种漆黑的金属在阳光折射下会反射出彩虹般的反射光，主要由特殊的隔音镍钢合金与陶钢结构组成，该武装具有6个装填孔，但根本无需装弹步骤，其内部是通过名为由于搭配其异能的特殊陶钢构造进行击发，产生空腔效应的一瞬间将真空化的气泡作为载体，从中填入挤压并被固化保存的咆哮声，随后将无形的气咆发射出去，命中敌人后，气咆会爆炸并仅针对被命中的敌人产生剧烈的咆哮声，该武装的攻击也因此将造成音波伤害而非子弹的物理穿刺伤害，由于其本质并非压缩空气和真正的声音传递，而是通过异能创造出的固体音波弹，所以在真空和水下依然可以成长射击和造成响应的效果，并且无需装弹，视为【无限子弹】。  
以上，是该武装的基本能力与介绍，在实战中，你可以消耗一个花招动作进行蓄能，每蓄能一次，就会有三个弹舱散发出金光，当蓄能第三、四次时，弹仓会散发出危险的红光；每次蓄力其实就是催动『咙啸嚎咆(Nukuch rugido yaj óolal)』积蓄力量的过程，每以此蓄力消耗一个花招动作，以该武器下一次攻击的射程会提升60米，并额外提供1点附加成功，至多4次；每以此获得1点附加成功，就可以将攻击上的3dp转换为1点附加成功。  
该武装的异灵搭配了自动恶意寻敌系统，在使用该武装进行瞄准时，将会使得下一次攻击额外获得24dp瞄准加值，获得这些瞄准加值后，直到通过攻击将其消耗前不会再次获得。  
你可以通过激活『咙啸嚎咆』来获取更多的战斗优势，分为阶段激活(Step Access)和全肯定(ALL Accept)两种激活模式，在全肯定状态下，也可以启动阶段激活并如常获得攻击上的益处。  
以一个迅捷动作调动该武装内置异灵『无限悲怆哀嚎苦痛世界之龙』的部分力量与展开程式，进入阶段激活(Step Access)状态，该状态下，该武装会从枪口两侧展开一个方形扩音设备，在攻击时，提前将攻击打散以达到将音波直接爆发的效果。若如此做，则下次攻击会基于武器射程效果线为中轴线，对身前60°锥形范围进行不分敌我的锥形AOE攻击，造成音波伤害，如哀龙咆哮的悲鸣，这次攻击加骰额外提升一级(至多为8加骰)，并提供1点额外的附加成功，工具结束后武装的激活阶段结束，直到下一次调用阶段激活状态前扩音模块将回收。  
以一个整轮动作，调动激活程式后完全激活『咙啸嚎咆(Nukuch rugido yaj óolal)』，武装将迅速展开，其表面会出现如古典玛雅文字般的铭文，金色的流光不断变化并复述着《波波尔乌(Popol Vuh)》的创世神话，该武器的射程提升180米，该武器的【9加骰】提升为【8加骰】，武器伤害提升2L，那之后，立即对射程范围内的一个目标进行一次攻击，这次攻击及该场景内接下来的每一次攻击，都将附带攻击等量的沮丧异常点数与来源于【碾压】特性的冲击异常点数，强韧豁免；因此造成的沮丧异常点数会在敌人的身躯上出现由泪水交织而成的古玛雅铭文，若陷入毁灭状态，敌人肉体上铭文将会连接成一篇完整的由玛雅语写就的悲惨故事，并不受控制地流泪与发出哀嚎。
