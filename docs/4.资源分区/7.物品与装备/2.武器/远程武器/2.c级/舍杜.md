# 舍杜

C+2000  
舍杜

  
类型：左轮手枪

  
比起金属的枪械，它的材质更像是某种骨骼。这把从骨质外壳的机械种族Sentient身上扯下来的手炮能快速射出热脉冲，并在电击中爆炸。  
特性：电池弹药，攻击模式切换

  
武器属性：10L，高速5，破甲3，无连射等级，射程30米

  
电池弹药：该武器的弹药会自动随着时间恢复，这意味着你不需要换弹。

  
攻击模式切换：你可以用一个迅捷动作切换舍杜的攻击模式。

  
普通攻击模式：舍杜在此模式下会发射造成火焰伤害的能量弹  
攻击成功命中后额外获得2点燃烧点数

  
范围攻击模式：舍杜改为射击在撞击地面或者敌人后能产生大范围爆炸的子弹  
攻击改为对落点处7.5米范围内所有敌人进行一次攻击检定
