# C级远程武器-投掷类

C级远程武器-投掷类

F  
非洲投刃  
价格：C+1500  
分类：轻投掷武器  
本质：科技本质  
武器伤害：11L  
武器特性：【高速2】【回力】【非洲投刃】  
武器体积：2  
伤害类型：物理-挥砍  
武器射程：检定属性\*5M  
描述：  
是曰：散发满满邪气的凶残武器。运用其上延伸而出的刀刃，进行多样化的攻击与投掷。  
效果：  
【非洲投刃】：  
当你使用该武器进行【召还】行为时，【召还】行为造成的伤害提升等同你【传奇敏捷】\*1点数值，同时忽略1点伤害吸收类能力带来的影响。

L  
连发燃烧瓶  
价格：C+1500  
分类：轻投掷武器  
本质：科技本质  
武器伤害：8L  
武器特性：【高速1】【无限子弹】【燃烧瓶】  
速射等级：短点射  
武器体积：1  
伤害类型：火焰伤害  
武器射程：检定属性\*5M  
描述：  
一打装有燃料的易碎容器，可以一口气进行大量投掷。击中敌方的话可以把对方变成火球，反过来说也可以让自己……  
效果：  
【燃烧瓶】：  
连发燃烧瓶是一种特殊的投掷武器而非爆炸物类型，投掷连发燃烧瓶将会对落点5米以内的所有目标（不分敌我）造成伤害，附带1点不可豁免的燃烧不良点数，可叠加。

烈风竹箭  
价格：C+1000  
分类：轻投掷武器/箭矢  
本质：魔幻本质  
武器伤害：10L  
武器特性：【回力】【高速6】【烈风】  
武器体积：1  
伤害类型：物理-穿刺  
武器射程：敏捷属性\*5M  
描述：于山峰之上摘去的经历数十年风暴不倒的烈风老竹制作而成的竹箭，尾部系着细绳，便于投掷和回收，成品在风中便会自行鸣响。  
效果：  
【烈风】：  
在对先攻低于你的目标进行攻击时，竹箭将唤起箭身内的风暴之蕴，使风暴产生于箭尖，本次攻击具有【毁灭】效果。  
这一武器也可以作为箭矢被弓箭所射出，但只会计入一半的武器伤害与高速，以弓箭的射程取代武器本身的射程。在攻击后的下一轮开始时，这一武器会回到使用者手中
