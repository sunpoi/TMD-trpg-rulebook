# C级远程武器-枪械类

C级远程武器-枪械类

A  
爱人吻  
价格：C+1000  
分类：自动手枪  
本质：特异本质  
武器伤害：11L  
武器特性：【高速5】【无限子弹】【爱人吻】  
速射等级：连射  
武器体积：2  
伤害类型：不造成伤害  
武器射程：30M  
装弹数：11+1  
描述：在锁骨处落下一吻，  
代替眼泪，道一句离别，  
白色的花朵为你铸造丰碑，  
你说过玫瑰很美，  
他们的血将如你所愿。  
这是外观精致的两把改装版M1911手枪。枪身整体呈现暗红色，使用了玫瑰金镀层，抛壳窗则采用镀金边的银白色，左枪的握把部分雕刻着一朵艳丽的红玫瑰，右枪则连着枪身一起纹着一株曼珠沙华。  
效果：  
【爱人吻】：  
这是一对双枪，故你必须同时持握两把手枪才能应用该武器的效果，占据双手位置，可以满足双持条件，但仅视为一把武器。  
该武器不会造成物理伤害，射出的子弹将在击中敌人的身躯后绽放出红色的花瓣，伤害转化为等量的魅惑点数，在判断来源时视为创伤。  
落花无心，流水无情，当持握这对武器时，你不会被C级及以下混乱效果所影响。

D  
打桩机  
价格：C+1000  
分类：狙击枪（打桩机）  
本质：科技本质  
武器伤害：13L  
武器特性：【高速5】【破甲3】【双手】【金属桩】  
武器体积：5  
伤害类型：物理-穿刺  
武器射程：200M  
装弹数：6+1  
描述：  
经过改良的单发填装式狙击枪，可以发射出高速的金属桩，不只可以破解敌方防御，更能将之击飞。  
效果：  
【金属桩】：  
使用该武器进行攻击命中目标后，你可以使用自由动作追加一次冲撞，以本次攻击出目作为DC，目标使用正常冲撞流程进行进行对抗，失败则被击退差值/米，这一效果每轮只能生效一次。  
当目标因该武器效果的冲撞被击退时，则每被击退1米，就承受1点额外的严重伤害，目标最多因此承受等同你自身【传奇智力】\*1点严重伤害。若目标在此途中因障碍物阻拦而无法继续被击退，也会全额承受这一效果带来的额外严重伤害。

地狱火霰弹枪  
价格：C+1000  
分类：霰弹枪  
本质：科技本质  
武器伤害：12L  
武器特性：【地狱火】【无限弹药】  
速射等级：短点射  
武器体积：4  
伤害类型：物理-穿刺  
武器射程：40M  
装弹量：4  
描述：  
“听上去英国佬很有把握。英雄永远是英雄，加布里尔。雕像上的那个英雄可不是我”  
这个后来被称为死神的人，本名加布里尔莱耶斯。他出生于洛杉矶。智械危机开始前，他成为了美国军队的高级军官，是一名久经沙场、受人敬佩的老兵。因技战术精湛后加入了实验性小队守望先锋，并被给予了这把地狱火霰弹枪。在随后升职？黑色守望队长期间，手下遗传学家莫伊拉对其进行了改造，使他获得了某些超乎常理的能力（无论是地狱火霰弹枪还是布里尔莱耶斯本人）。  
制造商：守望先锋  
操作原理：泵动式  
效果：  
【地狱火】：  
\-地狱火霰弹枪的弹道扩散很大，因此，其有效射程要比一般的枪械近得多，在超出基础射程后，在远距离上将几乎不产生任何伤害。不过，因其较大的扩散范围，换来的是对极近距离内单位的伤害，当对距离你不超过15米的目标射击时，地狱火霰弹枪武器伤害提高3L，同时获得破甲5和威猛3。  
特殊的是，在购买本武器后，你会获得无数把完全一致的地狱火霰弹枪，但这些地狱火霰弹枪都视为处于一个特殊的储物空间中，只有你可以从中取出和使用，该武器若交由他人使用，会立刻消失。但是同时生效枪械的只能有一把，取出和存入是一个迅捷动作（存取两把或者一把武器动作相同），重新取出的地狱火霰弹枪总视为满备弹状态。  
需要注意的时：  
\-及时备战专长或类似能力，可以降低地狱火霰弹枪取出和存入的动作需求；

钉影射钉枪  
价格：C+1000  
分类：自动手枪（射钉枪）  
本质：特异本质  
武器伤害：11L  
武器特性：【高速5】【钉影】  
速射等级：短点射  
武器体积：2  
伤害类型：物理-穿刺  
武器射程：30M  
装弹数：11+1  
描述：  
这是一把能够钉住影子的射钉枪，但射钉枪不重要，钉子也不重要，能射出钉子钉住影子，这才是最重要的。  
传说，有一种影子怪物，它会完美的模拟成生命的样子，但影子就是它们的弱点，钉住，锤击，射击，穿刺，都会对他们造成伤害与影响。  
这并非原本的那把射钉枪，而是由钉住影子得来的概念形成的钉影射钉枪，就像原本的射钉枪只是因为钉住过影子而特殊，而它，是因为能钉住影子而特殊。  
效果：  
【钉影】：  
如果在你射击时，如果对方的影子与本体处于你的攻击范围内，并且你的效果线未被阻挡时，你可以选择以一个自由动作将目标转为对对目标的影子进行攻击，这会使得你的本次攻击不造成伤害，而是将攻击点数转化为纠缠点数并且反应到影子的主人身上，目标本体需以强韧豁免来豁免该纠缠点数，并且在豁免时，会使其拥有的能量池中的1点任意能量流失。  
同时，以此武器对影子进行部位攻击时，视为对目标的特殊部位攻击减值忽略2点，若目标在这次攻击中承受了纠缠点数，则对其施加d级创伤来源困难地形效果。

分类：霰弹枪  
本质：特异本质  
武器伤害：10L  
武器特性：【高速5】【破甲3】【攻击距离不超过15米的单位时获得2威猛】【荷鲁斯之眼】  
速射等级：长点射  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：11+1  
描述：  
这是一把原型为伯莱塔1301 Tactical的半自动霰弹枪，枪身上写着“Eye of Horus”，主体为白色+黑色双色调，搭配粉色点缀。枪托、护木、机匣主体为哑光白色，简约干练而不失少女的可爱。  
创造之初，为了适配阿比多斯天灾后高强度的治安管理，普通的伯莱塔显然已经无法满足需求，但少女手中的武器却始终熠熠生辉。这是把会随着主人神秘的解放逐渐得心应手的武器。

【荷鲁斯之眼】：  
使用这把武器进行攻击时，可以将攻击结合进移动中。枪身锯短特制，处于他人触及范围内时进行的攻击高速不会减半。  
作为基沃托斯终身严选，持有者以荷鲁斯之眼进行攻击时获得自身传奇敏捷\*2点器械加值。

Harmony  
价格：C+2000  
分类：左轮手枪  
本质：魔幻本质  
武器伤害：12L  
武器特性：【高速5】【装弹的动作上升一级】【协奏曲】  
武器体积：2  
伤害类型：精神伤害  
武器射程：30M  
装弹数：6  
描述：  
这是一柄有弦和乐键的左轮手枪，它的枪管是小号的铜管，它的扳机是手摇风琴的按弦，它通体黑黄色，在风吹过时会自然发出乐音。  
效果：  
【协奏曲】：  
其在通常情况下不能附带招式进行攻击，以其进行的所有射击检定都需要标准动作才能进行攻击或至少为标准动作，下属四种曲调的效果便有一种可以应用在其中

Berceuse  
造成的伤害将转化为造成冲击点数，但会附带等量的耳鸣点数，使用Berceuse发射的子弹将以声波形式出现，也不会消耗其转轮内填入的子弹。当已经在某个场景内利用此效果进行攻击过时，下一次以Berceuse进行的攻击造成的冲击伤害将等量转化为对被攻击者头部造成的肢体妨碍，以标准形式豁免。

Walzer  
使用Walzer进行攻击时，可以从迂回步的前进与后退中选择一个，当选择前进步时，Walzer的使用者最多可以向前移动15M，当选择后退步时，被攻击者将会向后后退15M。当攻击者与被攻击者之间的距离正好在30米之间时，攻击者将视为对被攻击者有免疫C级措手不及。

Rhapsody  
使用Rhapsody进行攻击时，将再使自身承受2点不可豁免的亢奋点数，当Rhapsody形态发射子弹时，这颗子弹在击中第一个敌人后将开始弹跳，并接连击中该目标15米半径范围内的所有敌人，这些敌人将正常以防御抵御这发子弹造成的攻击。在亢奋点数未达严重状态前，Rhapsody的使用者每有2点亢奋点数，就可以无视1点子弹弹跳时被攻击者的掩蔽防御。

Concerto  
使用Concerto进行攻击时，还必须持有另一把手枪同时射击才能正常进行一次攻击。参与此次攻击的两把手枪将共用拥有的特性，武器伤害取高，攻击范围取低结算。当攻击一个濒死的角色时，可以消耗1点意志进行攻击，若其有一轮一次免疫单体攻击或每日一次及以上时间内免疫一次任意攻击的无敌，则无视；若此次攻击将其击杀，则其无法通过S级及以下的复活能力复活。

J  
寂寞小镇（Lonesome Town）  
价格：C+1500  
分类：左轮手枪  
本质：特异本质  
描述：  
这是一把海军左轮手枪，亮银色枪管，黑檀木握把，枪身整体闪着淡蓝色，枪管上刻着“Waits for my return”  
效果：  
使用寂寞小镇的能力必须首先用一个移动动作将附着的手枪（这意味着寂寞小镇必须绑定一把你已具有的手枪类武器）扳开击锤转动转轮，并使用标准动作射击才能使它的能力生效，这会视为一次正常的攻击；或是用一个移动动作用寂寞小镇对自己进行射击，以为自身带来增益。寂寞小镇一共只有15发子弹，且使用后不会再生。此外，当前持有的子弹数每有16枚，在一次战斗结束后，就会获得1枚子弹。

在使用寂寞小镇攻击时，使用者需让转轮自行旋转，掷一次RD6，根据结果决定触发的效果。寂寞小镇的子弹并不会因附着的手枪子弹减少而无法发射，但附着的手枪若因没有子弹而无法射击，那么寂寞小镇的能力也不能生效。  
寂寞小镇的攻击效果：  
1.在本次攻击中，你可以选择将造成的伤害转化为等量的肢体妨碍点数，当你造成肢体妨碍点数时，可以选择不造成肢体妨碍点数，而是改为削去被攻击者的某一种生理属性等量的等级，最多能将某属性削减至1。这些属性伤害在经历一次长休息后会自然消失。若本次攻击造成的肢体妨碍点数还超过了对方体积，则超过的每1点肢体妨碍点数会同时带来1点剧痛点数，使用强韧豁免以进行对抗。

2.本次攻击命中目标时，同时解析目标施过的一个在等级在B级及以下能力的详细效果，在本场景中，还可以至多再对该目标进行4次解析，且无需再触发本效果，本场景中除第一次随机到本效果外，再次随机到本效果时退回寂寞小镇的子弹和已经被使用的移动动作，不触发效果。

3.本次攻击将改为攻击由使用手枪射程为半径的圆内的所有单位，同时还会造成D级困难地形，持续存在一轮。

4.本次攻击将额外附带5点附加成功和1点攻击上的表现加值。

5.本次攻击将需要以意志豁免进行对抗，对于此次攻击而言，可以至多重掷一次攻击结果，但无论结果如何都需接受第二次的结果，本次攻击额外获得1点攻击上的附加成功和1点攻击上的表现加值。

6.本次攻击在对目标成功造成伤害时，还视为具有「吸血5」和1点攻击上的表现加值。

在使用寂寞小镇对自身释放效果时，直接从下列选项中的一个进行结算，但每个效果每场景只能选择一次。

寂寞小镇对自身使用的效果：  
1.为自身提供持续存在一轮的6点偏斜防御  
2.为自身提供持续存在一轮的2点快速治疗  
3.为自身提供持续存在一轮的2点全吸收  
4.为自身提供持续存在一轮的某豁免检定上的6点表现加值

哨兵  
价格：C+1500  
分类：狙击枪  
本质：科技本质  
武器伤害：18L  
武器特性：【高速5】【穿甲3】【哨兵】【猎头者】【重创】【充能】  
武器体积：5  
伤害类型：物理-穿刺  
武器射程：200M  
装弹数：6+1  
描述：  
由拉斯提莫沙军械库为apex竞赛提供的特殊型号栓动式狙击步枪，射程和伤害都在时间的考验下得到了验证。  
这是一把拥有着哑光黑色的枪身的狙击枪，为了保证精良的射程与杀伤力，采用了独特的枪身设计，枪栓可转动能量附属装置，起到充能和换弹的效果。  
效果：  
【哨兵】：  
设计者出于其对老式栓动单发步枪的情怀，加入了传统步枪的射击模式，使用该步枪击发前需要一个移动动作进行手动上膛、攻击后需要一个迅捷动作进行退弹。  
【猎头者】  
以一个标准动作，对敌对目标的头部进行一次射击，这次射击不计算任何能力提升，且出数大于总数的一半（不包  
含附加成功带来的数值），则此次攻击中每有1点附加，就提升1点（这个效果不会和毁灭、湮灭共同生效）。与此同时，目标的头上会亮起一个字体加重加  
粗的140（充能后将变为176）的数字，这视为一个攻击技。  
【重创】  
被这把枪射中头部者，将受到等同于伤害出目肢体妨碍点数，这不是一个部位攻击。使用标准强韧豁免进行对抗。  
若攻击时，被攻击者已陷入严重状态时，则此次攻击视为进入了毁灭状态，可以以恢复严重状态般恢复，但如果  
造成的异常点数累计达到毁灭状态，则视为进入真正的毁灭状态。  
【充能】  
你可以使用一个移动动作拉动枪栓，并连接一枚小型护盾电池为枪械充能。  
此时枪械可以获得6L的伤害提升，同时获得5威猛，持续一场景

附属物品：  
小型护盾电池：750分一枚  
用于补充护盾能量的能量电池，也可以用于武器的充能

比起金属的枪械，它的材质更像是某种骨骼。这把从骨质外壳的机械种族Sentient身上扯下来的手炮能快速射出热脉冲，并在电击中爆炸。

【电池弹药】：  
该武器的弹药会自动随着时间恢复，这意味着你不需要换弹。  
【攻击模式切换】：

攻击改为对落点处7.5米范围内所有敌人进行一次攻击检定

圣约十字  
价格：C+1000  
分类：霰弹枪/盾  
本质：科技本质

描述：  
在噬日的余波中，萨满铁匠沙之圣者逃入了洪荒，并在其深处建立了一处宫殿锻造厂。他野蛮地抓捕妖邪，将它们丢入自己的炉中，用来铸成他的复仇武器。异火是这些武器中的最后一把，自他那被洪荒扭曲而成的工坊废墟中诞生。  
自圣者死后，异火和他的其他武器作品就散落在了尘世和洪荒各处，但罕有武器能与异火的恶名媲美。它的火焰燃烧着洪荒的力量，扭曲玷污而非吞噬任何被其触碰之物。在八目织者手中，异火夷平了无数领土，将战士化为无脸的怪物。  
异火看起来是一把单发式的燧发步枪，其不能应用使枪械具有【短点射】【长点射】【连射】的特性效果，它每次激发都将喷出喷出色彩斑斓的烟雾和火焰。  
效果：  
【癫火】：  
异火与持有者共同燃烧，异火的持有者在进入战斗后，眼中瞳孔就会变为燃烧着的明黄色，在射击时，除了需要子弹外，使用者还需要支付一点意志，之后明黄色的火焰将缠绕子弹，将弹头融化为无温无型的癫火，该次攻击获得1点额外的附加成功，被攻击命中的单位仅可以以【意志豁免】对抗本次攻击检定。  
【熔毁诅咒】：  
异火在造成伤害后，不会带走血肉或留下创口，而是将命中部位的血肉扭曲成骇人的形状，血肉化作珊瑚、手臂骨刺增生、脸部长满真菌等等。这些额外增生的部位不在算作被命中生物的一部分，而是增生物。在治疗时，这些增生的部位会随着敌人生命值的完好而逐渐脱落。而被异火所杀死的敌人，他的尸体将变成钙化物，珊瑚和真菌的融合体，想要简单的通过视觉、气味和声音来辨识受害者将是不可能的，需要进行一次难度等于异火使用者（决心属性数）的智力+生存：医治延长检定，才能确定受害者的身份，这个效果无法将受害者伪装成另一个人，视为C级创伤来源的诅咒【变形】效果。被此武器杀死的单位将无法以B级及以下的能力进行复活。

描述：  
Z-180 Close Combat Rifle /Z-180散射步枪--通常被称为Scattershot散射枪，是是一种先行者定向能武器，它实际上是一种类似于人类霰弹枪的武器，最初是在先行者-洪魔战争即将结束时作为近距离防御武器使用的。它的火力在相对较近的距离内非常危险，一枪可以发射多个单独的射弹，其最显著的特点是子弹可以在表面反弹 。  
制造商：锻匠总装工场  
类型：霰弹枪  
长度：115.5 厘米（45.5 英寸）  
宽度：10.7 厘米（4.2 英寸）  
高度：34.1 厘米（13.4 英寸）  
弹药类型：硬光弹药  
这种武器具有先行者武器设计的特点：棱角性和金属质感。从它的自动装弹充能过程和悬浮部件可以看出，它的一部分似乎被一种能量场连接在一起。当武器自动装弹充能以及装弹完成后的整枪，可以看到橙色光源；当武器发射时，这些光源会变得更亮。武器采用一个折管式弹仓容纳五个弹药单元。  
效果：  
【先行者】：  
来自先行者的能源科技使其可以提供的硬光弹药视之为无限，以进行5次射击（拥有5发子弹），这会以百分比的形式显示在枪械的悬浮屏幕上，每次射击会累计20%的热量，累计到达百分之百时，需要以一个迅捷动作将枪管向两侧弯折散热，待计数器归0后才可继续射击。显然的，这种武器不具有常规意义上的枪膛和撞针，仅能射击硬光弹药，这些硬光弹药将为攻击提供【能量武器】特性。  
散射枪是架构者护卫队的首选武器 ，设计之初便是为了在在狭窄的环境下作战。Z-180散射步枪一次射击可以射出8枚弹丸，若在射击目标的周围大约一米范围内（不包括地面）具有不小于1平方米的坚硬表面，那么这些弹丸就会在表面上进行反弹，对目标额外造成伤害，视为该次攻击中的1点严重伤害转为韧性。
