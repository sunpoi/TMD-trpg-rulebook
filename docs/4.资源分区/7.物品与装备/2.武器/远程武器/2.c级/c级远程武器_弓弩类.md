# C级远程武器-弓弩类

C级远程武器-弓弩类

L  
连装弩  
价格：C+2000  
分类：弩  
本质：科技本质  
武器伤害：11L  
武器特性：【高速3】【破甲1】【双手】【自校准】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：30M  
描述：  
主神空间特供的连装弩，除了其威力足以力贯金石，具有自我矫正能力之外，还可以在你搭箭前自行生产可供你使用的一次性普通箭矢。  
效果：  
【自校准】：  
该武器具有【无限子弹】特性，这意味着你无需搭箭也可以正常攻击，但你依旧可以正常搭箭并正常消耗弹药从而获得一些额外效果；使用该武器进行攻击，额外获得等同自身传奇敏捷\*2点DP加值，若你对攻击出目不满意，则可以进行重骰，但必须接受重骰结果，重骰能力每轮最多只能使用一次。

M  
明亮射击  
价格：C+1500  
分类：弓  
本质：特异本质  
武器伤害：11L  
武器特性：【高速6】【破甲6】【双手】【高亮】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：1  
描述：  
使用特殊的发光矿石、发射时放出伴随幻觉的不可思议之光的十字弓。  
效果：  
【高亮】：  
使用该武器进行的攻击，附带等同伤害出目的目眩不良点数，目标需要使用强韧进行豁免；当该武器攻击命中后，目标需要再使用意志豁免对抗伤害出目，失败则陷入持续一场景的【混乱】状态，随机选择攻击目标；若需处理来源，上述效果皆为C级环境来源的效果。

N  
X  
玄鹰弓  
价格：C+1000  
本质：魔幻本质  
描述：  
此弓重二石，色玄，有鹰羽相缀。玄者色黑也，鹰者猛禽也，玄水之鹰，上达九天，下潜九渊，无拘束而通达，通变幻无常。  
  
【万玄】：聚玄而变化，得水之形。在特殊情况下，弓箭上鹰羽展开，箭矢转化为玄水阴箭。  
  
【鹰击】：直下云端，力沉而迅捷。  
  
特性：【万玄】【鹰击】  
  
分类：弓  
武器伤害：11L  
武器特性：【高速3】【破甲1】【双手】【万玄】【鹰击】  
武器体积：3  
弹容量：1  
基础射程：40米  
伤害类型：物理-穿刺  
效果：  
【万玄】：当你在进行一次射击时，可以针对这次攻击消耗至多3点内力（或者其他东方能量），此次造成伤害改为冻寒伤害，攻击转变为曲线型。  
【鹰击】：当你适用了【万玄】的那次射击已经进行过瞄准，则攻击额外获得2dp器械加值。  
在进行满足获得加值条件的攻击之前，你可以额外针对此次攻击消耗一个移动动作进行蓄力，若如此做，这次攻击额外具有【破甲10】。  
  
【迅牙猎弓】  
价格：C+1250  
分类：弓  
本质：特异本质  
武器伤害：11L  
武器特性：【高速3】【破甲1】【双手】【狩猎的嗅觉】【狩猎之灵】  
武器体积：3  
伤害类型：物理-穿刺  
武器射程：40M  
装弹数：1  
描述：  
“它很好用”  
“它会咬住你”  
“它会吃掉你”  
这是一把通体白色的弓，上面布满了各种尺寸的兽牙，充斥着蛮荒的气息。  
这是一把来自于未知的，古老的时刻的猎弓  
最强大的猎人赋予了它躯体  
最强大的猎物赋予了它灵性  
它将是你狩猎时最好的伙伴，也是你最好的工具  
它那来自于猎物的本能与狩猎的灵性是那样残缺且混沌。  
但它那残缺混沌的灵性中只有一个想法——让我咬住它！让我咬住下一头猎物！  
效果：  
【狩猎的嗅觉】：  
我嗅到了，这就是你的弱点。  
它们如此明显，而且就像落日余晖一般橙红的果实一样——香甜，可口，怡人。  
在你进入战斗后的第二个回合，迅牙猎弓因狩猎而逐渐活跃的灵性将赋予你敏锐的狩猎直觉，这种直觉使得你能够嗅到猎物的弱点，此时，你对猎物造成的攻击能够将1点严重伤害转为1点韧性伤害，这种状态将会在战斗结束后清空。  
【狩猎之灵】：  
在你的回合时，看选择消耗一个移动动作主动调动猎牙的灵性，这会使得猎牙赋予激发出的箭矢【猎牙】或【追狩】效果，持续一场景。  
效果持续期间，可在【猎牙】与【追狩】中进行效果切换  
切换【猎牙】或【追狩】效果则需要一个额外的迅捷动作进行。  
\-猎牙  
咬住，不要松口  
这就是狩猎的法则  
一道苍白的弧光，犹如猎兽的牙齿钉入猎物的身体中。  
你知道的，你一直都知道，到底攻击哪些部位会给予猎物最严重的伤势，而剩下的，就交给猎牙吧。  
看见了吗？猎牙咬穿它了皮肉，洞穿它了生机。  
至于剩下的残躯？  
就继续啃咬它吧，继续狩猎它吧，它的伤势不会允许它支撑太久，死亡会离它越来越近。  
你的攻击将附带b级的创伤来源重伤效果。  
\-追狩  
猎物会逃跑，但不必担心  
猎牙会留下专属的印记，那是它所标记的猎物  
那就让它逃吧  
然后就顺着猎物气味的方向，顺着那条血腥的长廊跟上它吧  
猎牙的齿痕不会消失太快  
那流出的鲜血就是猎牙进一步的追狩  
看着它慢慢死去吧，它已无路可走。  
你的攻击将附带1点不可豁免流血点数。
