# 荷鲁斯之眼

荷鲁斯之眼  
本质:特异本质  
价格：C+2000  
描述：这是一把原型为伯莱塔1301 Tactical的半自动霰弹枪，枪身上写着“Eye of Horus”，主体为白色+黑色双色调，搭配粉色点缀。枪托、护木、机匣主体为哑光白色，简约干练而不失少女的可爱。  
创造之初，为了适配阿比多斯天灾后高强度的治安管理，普通的伯莱塔显然已经无法满足需求，但少女手中的武器却始终熠熠生辉。这是把会随着主人神秘的解放逐渐得心应手的武器。  
分类：霰弹枪  
武器伤害：10L  
武器特性：破甲3，高速5，攻击距离不超过15米内的单位时获得2威猛。  
弹容量：11+1  
武器体积：3  
速射等级：长点射  
基础射程：40米  
伤害类型：物理-穿刺  
效果：  
使用这把武器进行攻击时，可以将攻击结合进移动中。枪身锯短特制，处于他人触及范围内时进行的攻击高速不会减半。  
作为基沃托斯终身严选，持有者以荷鲁斯之眼进行攻击时获得自身传奇敏捷\*2点器械加值。
