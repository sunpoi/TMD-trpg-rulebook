# 制式激光枪

制式激光枪  
价格：C+1500  
分类：步枪  
武器伤害：12L  
武器特性：高速5，特殊子弹器械减值-1  
弹容量：30  
武器体积：4  
速射等级：长点射  
基础射程：150米  
伤害类型：物理-穿刺  
描述：迸射而出的高热光线。虽然威力有点不够，不过武器的距离优势，却能带来许多意想不到的效果。  
效果：使用该武器进行攻击时，获得【能量武器】特性；当你使用该武器附带的【能量武器】特性时，除了享受远程能量武器的额外高速外，还具有【熔毁5】。  
【理所当然的，该武器使用的是电池供能而非子弹，其他方面等价正常子弹，一个基础30发的弹匣对标的是30发射击的能量电池，因此也没法使用通常的特殊子弹。】
