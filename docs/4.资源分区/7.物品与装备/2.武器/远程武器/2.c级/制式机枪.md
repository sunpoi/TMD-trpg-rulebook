# 制式机枪

制式机枪  
价格：C+1000  
分类：重机枪  
武器伤害：13L  
武器特性：高速5，双手，贯穿，装弹动作上升一级  
弹容量：100  
武器体积：5  
速射等级：连射  
基础射程：150米  
伤害类型：物理-穿刺  
描述：击发出连串的子弹，造成如同弹雨效果般的凶器。如果打在敌方密集之处的话可以获得极高的伤害。  
效果：  
使用该武器进行的攻击，可以转变为对前方等同扇形射程范围的所有敌对目标造成伤害，目标需要使用反射检定豁免，而非正常防御流程。
