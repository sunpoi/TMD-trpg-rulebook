# 猎手之枪

  

猎手之枪  
D+1000  
“不要掺杂任何多余思想。”  
“天平之上不容杂草滋长。”  
“务必保证一切精准测量。”  
“沐浴阳光不容任何遮挡。”  
武器采用基础远程武器中的狙击枪模板，具有全部相关特性，在其基础上提升3L武器伤害，高速额外增加1点。

  
这把狙击枪，通体斑驳而手感粗糙，若是仔细观察会发现枪身整体由呈深褐色，如一根树木削刻而成，但重量相较于正常枪械也不显轻松。  
也许正如猎手所言，公正的代价是如此沉重。

  
精准测量：本武器的装弹数不视为6+1，而是1，但只需要装填一次子弹就可以进行六次射击，在第七次射击时必定命中自己，自身在这次攻击中不允许留手，也不能使用任何降低攻击DP的能力。  
持有者可以通过重新装填来刷新次数，规避第七次射击。  
这意味着即使拥有无限子弹或其他类似能力，也仍然需要进行装填。

  
树荫之下：使用本武器时，必须以迅捷动作上膛，移动动作祈祷后才能使用标准动作攻击，这次攻击中附带4点神圣加值。  
持有者视为具有D级特异本质的反侦查能力，且攻击时针对的目标必须没有发觉自身，即反侦查生效才能进行攻击并享受对应的好处。  
否则，本特性提供加值将暂时失效，武器本身改为标准动作进行普通攻击，直到重新掩蔽起来为止。
