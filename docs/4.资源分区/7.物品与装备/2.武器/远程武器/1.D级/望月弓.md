# 望月弓

  

望月弓  
魔幻本质  
价格：D+500  
这是一把乌木制成的长弓,深沉的黑色弓身典雅大方,无任何多余装饰。唯一与众不同的是弓身上隐隐闪动的月色辉光,就像云雾中若隐若现的明月。  
  
武器类型：长弓  
射程：30米  
武器伤害：6L  
特性：  
进行攻击时以自由动作消耗1点能量（须为东方类能量，在购买时决定。），唤醒望月弓中残留的箭意，令这次攻击获得3点破甲或高速。  
【雪恨】：每场景一次，人物可以迅捷动作承受至多【决心与耐力附加成功点】不可避免严重伤害为代价，将自身血液化作一枚箭矢，这不会改变搭箭需要的动作，此箭矢在进行一次攻击后即会被消耗。  
人物以此箭对单一目标进行攻击时，具有等同于承受的严重伤害的【高速】、【威猛】、【破魔】。  
  
注：望月弓中寄宿着曾经主人的灵魂，其自愿化作弓的器灵帮助使用者报仇雪恨。因此如果有沟通灵体的手段，可能会与其灵魂产生交流。
