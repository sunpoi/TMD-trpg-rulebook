# 入侵者

入侵者  
D+1000  
“这里不欢迎你。”

  
特性：不知悔改、涉险而为

  
这是一把武器伤害6L，蓝色枪身，白色握柄，具有连射的自动手枪。

  
不知悔改  
使用入侵者杀死一个敌方单位时，不知悔改将立刻开始准备生效直至重新装填子弹，装填子弹后，发射的下一发子弹将必须使用一个标准动作进行攻击，这是一个攻击技。  
下一次射击时，不知悔改将立即为此次射击额外提供2点高速和4点武器伤害。

  
涉险而为  
若不知悔改成功击杀了那次攻击的目标，则立即视为进行了一次重新装填，并使不知悔改开始生效。

  
可额外选择支付对应的分数购买下列Perks  
全口径枪膛  
1000  
对射程进行优化的枪管。  
增加30米基础射程。

  
弹跳型弹药  
750  
弹药会在硬质表面上反弹。  
忽略3点环境减值。

  
光面握把  
1000  
降低持有和瞄准难度。  
每轮一次，第一次瞄准时少消耗1AP。
