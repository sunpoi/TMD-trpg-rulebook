# 疾驰的G

疾驰的G  
D+750  
魔幻本质  
来源∶游戏王卡图世界  
类型：左轮手枪  
体积：2  
载弹量：6  
伤害6L  
射程30米  
特性：【高速】5，【应战！】，【G的本质】  
【应战！】∶即使是再渺小的生物，也有着向苍天发起叛逆的勇气！它们源源不断的在你的弹夹内生成，只为宣泄自己的怒火！这把手枪获得无限弹药的特性。现在，你不用担心自己的库存了

【G的本质】∶万千世界，无论是哪个角落都存在着这种奇怪的生物，它们没有任何攻击与防御的手段，能活下来也许是因为小巧身躯以及极快的移动速度，但更有可能是因为那令人不寒而栗的名号，因此，你射出的子弹将不再是子弹，而是一种让你熟悉的蜚蠊科昆虫，虽然它们与你印象中的有些不一样？你的攻击将附带1点不可豁免的恶心的不良点数
