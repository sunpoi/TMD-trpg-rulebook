# 制式狙击枪

制式狙击枪  
价格：D+500  
分类：狙击枪  
武器伤害：8L  
武器特性：高速5，破甲3，双手  
弹容量：6+1  
武器体积：5  
速射等级：-  
基础射程：200米  
伤害类型：物理-穿刺  
描述：具有能够射击超远距离的枪械之一，对人威力不在话下，即使是面对活死人也能一口气打个对穿。  
效果：使用该武器进行攻击时，具有【贯穿】特性，同时忽略1点物理伤害减免类能力。
