# 钛帝国脉冲步枪

  

钛帝国脉冲步枪  
C+3000  
科技本质  
脉冲武器是钛帝国土氏族开发的一种等离子武器家族，其只被火氏族装备。脉冲步枪的工作原理是利用脉冲感应场推动等离子体的致命微脉冲在长射程距离下爆炸。  
  
重量：1.75KG  
  
长度：1200毫米  
  
体积：3  
  
专业：步枪  
  
伤害：12L  
  
持用：双手  
  
基础射程：150米  
  
脉冲武器的工作原理是使等离子体弹壳沿枪管以电磁加速方式加速。实际上，它是个微型质量加速器。发射时，将铁磁性的固体块从弹匣中放出来，并通过电磁感应将其转变成等离子体。将线圈电流以足够高的频率交替，使线圈加热到极端温度对于它来说是很容易的。然后，枪膛螺线管充满电，以极高的速度将新产生的等离子体从喷枪中推出，同时保持其粘结性。随后剩余的等离子体从枪管中喷出，产生相当大的枪口闪光。螺旋形电磁感应设备也有助于维持磁场，从而增加了磁场强度，继而增加射程。螺旋形电磁感应设备的重量也会进一步降低武器的后坐力。如果没有维持电磁场，那么发射的等离子体将全部消散在枪管口处，喷出约20厘米左右。正因为如此， 脉冲武器的射击除了需要实体弹药外，还需能量电池基于供能。  
  
最大弹夹：36固体块。  
特殊的，每支持36发弹药射击需消耗一个标准电池，一个杂技动作或同时消耗一个移动动作和迅捷动作才能重新装填子弹。  
  
连射等级：无  
  
伤害类型：该武器通过释放离子脉冲，以用来气化敌人的装甲与肉体，这使得该武器具有【能量武器】特性。  
  
为时已晚：脉冲枪采用两段式发射的方法来确保其等离子体弹药的命中——当扳机被扣到一半时，一束看不见的带负电荷的粒子会光速击中目标，然随后负电荷会引导全部的等离子体击中目标。这使得受害者在被击中前瞬间发出鬼魅般的光芒，如同死神的最后的垂怜。 人物在使用该武器攻击时，需先进行一次命中检定，若成功造成任何一个成功数，则可在真正的攻击中，忽略目标不超过5点的基础/闪避防御。  
  
  
弹药  
铁磁性固体块：1分50发
