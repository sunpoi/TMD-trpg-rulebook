# 圣约十字

圣约十字  
C+1000  
武器：盾/霰弹枪  
科技本质  
‘我主赐予我神圣之启示’  
‘教皇赠予我制恶之十字’  
‘我当制裁一切不恕罪恶’  
‘我当传播恶主神之福光’  
由新兴教国独立研发的神圣式制恶圣约十字  
这个银红色的巨大十字长处约有六英尺高，宽处约有三点五英尺宽，整体进行了防御特化，能够有效阻断物理意义上的部分影响。  
十字其中一侧装有把手与弹药填充口，能够如同手持盾牌一般手持，并在内部装载了泵动式霰弹发射器，必要时能够在十字架底部发射出赤红的龙息火焰弹，可以轻易对碳基生物造成毁灭性打击。

盾牌模式 此时视为11L武器伤害的重盾  
拥有双手 格挡5特性  
造成钝击伤害 体积4

霰弹模式 此时视为12L武器伤害的霰弹枪  
拥有高速5 攻击距离不超过15米内的单位时获得2威猛 长点射 特性  
造成穿刺伤害 体积4 载弹11+1 射程40米

值得注意的是，使用霰弹模式时请自行备足弹药  
【爆发火力】  
以本武器进行攻击时，可以使用一个自由动作将弹药转化为龙息弹，使得攻击时转为对目标造成火焰伤害。  
【神圣庇护】  
巨大的十字无论如何都可以将你笼罩其中，手持武器时，你获得1点遮蔽防御。  
【盾与枪】  
消耗一个迅捷动作，立刻对武器进行一次【火弹装填】或对武器进行一次【猛击祝福】祷告，并使得武器在霰弹攻击模式与盾牌模式之间进行一次切换。  
——火弹装填  
你当知晓，火焰为主的目光  
此刻，祂注视你，你当感恩  
它将如祂令耶稣基督赦免罪人一般，赦免你的罪  
若进行【火弹装填】，武器此时将被切换为霰弹枪模式，无法视为盾进行相关的动作或技艺的使用，直到进行切换为止，随后，在下一次攻击中，你的攻击将对前方造成7.5米的扇形范围伤害，切换后失效。  
——猛击祝福  
你当知晓，主可令地动山摇  
此刻，我持耶稣基督之象征，罪人当止步于此  
亦或者，将遭受日光（存疑）灼烧（确信）之刑。  
若进行【猛击祝福】，武器此时将被切换为盾牌模式，无法作为霰弹枪进行相关的动作或技艺的使用，直到切换为止，随后，你若在下一次攻击中使用圣约十字的头部进行撞击攻击那么这时你可以一个自由动作令圣约十字朝反方向进行一次助推射击，反作用力会使得你的这次近战攻击视为附带了一次冲撞效果。
