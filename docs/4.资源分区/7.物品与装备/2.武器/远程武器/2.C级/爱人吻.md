# 爱人吻

爱人吻  
C+1000  
特异本质  
描述：在锁骨处落下一吻，  
代替眼泪，道一句离别，  
白色的花朵为你铸造丰碑，  
你说过玫瑰很美，  
他们的血将如你所愿。  
这是具有11L，【高速5】【连射】【无限子弹】，射程为30米的，外观精致的两把改装版M1911手枪。枪身整体呈现暗红色，使用了玫瑰金镀层，抛壳窗则采用镀金边的银白色，左枪的握把部分雕刻着一朵艳丽的红玫瑰，右枪则连着枪身一起纹着一株曼珠沙华。  
这是一对双枪，故你必须同时持握两把手枪才能应用该武器的效果，占据双手位置，可以满足双持条件，但仅视为一把武器。  
该武器不会造成物理伤害，射出的子弹将在击中敌人的身躯后绽放出红色的花瓣，伤害转化为等量的魅惑点数，在判断来源时视为创伤。  
落花无心，流水无情，当持握这对武器时，你不会被C级及以下混乱效果所影响。
