# 网枪

网枪  
价格：C+1500  
分类：步枪  
武器伤害：12L  
武器特性：高速5，特殊子弹器械减值-1  
弹容量：30  
武器体积：4  
速射等级：长点射  
基础射程：150米  
伤害类型：物理-穿刺  
描述：可以发射出混合强力黏剂与削肉网绳的枪械，不仅有着削去皮肉的杀伤力，还可以阻止对方的行动。  
效果：你可以使用一个迅捷动作切换为网绳弹，网绳弹造成的伤害可以转为纠缠不良点数，正常豁免，可叠加。无论是否转换，都附带1点不可豁免的额外额外点数，同时伤害类型转为物理-挥砍伤害。每次使用迅捷动作进行的准备仅持续一轮，一轮结束后需重新准备。  
使用网绳弹不会计算弹药消耗，如同【无限子弹】一样处理。
