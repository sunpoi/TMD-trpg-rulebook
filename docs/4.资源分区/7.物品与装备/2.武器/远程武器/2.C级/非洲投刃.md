# 非洲投刃

非洲投刃  
价格：C+1500  
分类：投掷武器（轻投）  
武器伤害：11L  
武器特性：高速2，回力  
武器体积：2  
速射等级：-  
基础射程：检定属性\*5米  
伤害类型：物理-挥砍  
描述：是曰：散发满满邪气的凶残武器。运用其上延伸而出的刀刃，进行多样化的攻击与投掷。  
效果：当你使用该武器进行【召还】行为时，【召还】行为造成的伤害提升等同你【传奇敏捷】\*1点数值，同时忽略1点伤害吸收类能力带来的影响。
