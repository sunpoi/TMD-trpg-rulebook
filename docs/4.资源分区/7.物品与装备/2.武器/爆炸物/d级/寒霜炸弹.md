# 寒霜炸弹

  

寒霜炸弹  
  
  
D+500/5枚  
依靠着一种未知尚未确定的化学物质，这些便携式炸药能够立刻降低四周区域的环境温度。暴露的组织会立刻被冻结。  
固定伤害：10L  
爆炸范围：10米  
豁免方式：反射  
效果：该炸弹造成的是冻寒伤害，受到伤害的单位需要进行一次强韧豁免，DC为10，失败则受到差额的冻结点数。
