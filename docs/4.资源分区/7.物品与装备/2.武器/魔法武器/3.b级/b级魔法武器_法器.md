# B级魔法武器-法器

B级魔法武器-法器

F  
丰识  
魔幻本质

B+2000  
武器分类：法器  
法术强度：15L  
武器体积：2  
描述：水之魔女的武器，一柄以水魔晶石打造的短剑，以魔力驱动时可以在空中悬浮，其中蕴含的魔力甚至能凝聚为蓝色液态，伴随着湛蓝色的水晶剑身移动漂浮在身体周围。以秘银打造的剑柄与护手是类似于法杖的存在，主要用于传递魔力而不是辅助挥砍。这是一柄提供15点法术强度的短剑，魔晶石剑刃并没有剑锋开刃，只是适用于施法的法器。  
特性：【水之魔晶】【冻结魔咒】

【水之魔晶】  
在以该法器进行法术攻击时，可以消耗一个花招动作，催动水魔晶石的力量改变法术的性质，令这次法术的伤害转变为冻寒属性能量伤害，受到伤害的目标需要进行一次强韧豁免，DC为伤害值，失败则会被武器中附带的液态魔力影响，在一轮内受到B级别、诅咒来源的潮湿效果。

【冻结魔咒】  
以该武器进行的法术攻击附带有伤害量二分之一的冻结点数，强韧豁免，除此之外总是附带2点不可豁免的冻结点数，由于武器中铭刻的魔咒，这些异常点数皆视为诅咒来源。
