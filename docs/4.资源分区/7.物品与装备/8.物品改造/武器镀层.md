# 武器镀层

武器镀层  
一项武器只能有一种镀层。通常来说，镀层会提供给那些没钱购置纯材料打造武器的穷鬼冒险者，但也有些材料本身不适合武器锻造（比如一些质地偏软的材料）。  
综上所述，根据需求市场，主神也推出了经久耐磨的镀层工艺，提供给轮回者使用。

藤蔓  
价格：500  
描述：翠绿色的镀层，上面有着像是藤蔓一样的绑带。  
效果：该武器视为具有【木】材质。当使用该武器进行攻击时，额外获得1点器械加值，使用该武器进行格挡则获得1点器械防御

黄金  
价格：500  
描述：纯金色的武器镀层，可以让你的武器变成土豪金。  
效果：该武器视为具有【金】材质。当携带该武器进行社交行为时，你在社交相关检定上获得2DP器械加值。

冰  
价格：500  
描述：散发着冷意的淡蓝色镀层，人工装饰了锯齿的外观。  
效果：该武器视为具有【冰】材质。你的武器造成伤害类型永久性改为冻寒能量伤害，当使用该武器进行攻击时，额外获得1DP增强加值。

火焰  
价格：500  
描述：散发着热意的镀层，但火红的装饰色泽略暗，无法带来温暖人心的感觉。  
效果：该武器并不能具有【火】材质，镀层无法做到为武器添加火材质属性，但依旧可以使武器造成伤害类型永久性改为火焰能量伤害，当使用该武器进行攻击时，额外获得1DP增强加值。

叶子  
价格：D+500  
描述：带有明显明暗色调变化的翠绿镀层，说是叶子，其实更像豆角。  
效果：该武器视为具有【木】材质。使用该武器的进行攻击可以改为【曲线型】，同时在攻击检定上获得1DP器械加值。

花丛  
价格：D+500  
描述：让人感觉握着武器如同握着一束花的镀层，在握把处采用了明朗的白色。  
效果：该武器视为具有【木】材质。该武器获得常驻的【谐振】特性；一场景一次，你可以使用迅捷动作启动该武器的镀层效果，从而使自身获得【快速医疗2（医疗点数24）】，仅持续一轮。

蓝宝石  
价格：D+500  
描述：虽然是蓝宝石，但镀层的颜色更深，以至于接近紫色，手感如矿石一样粗粝。  
效果：该武器视为具有【石】材质。你在使用该武器进行攻击时，获得等同自身【传奇决心】\*1点狂怒；你在使用该武器进行攻击时，获得【连射】特性，即使是近战武器也可以无需消费弹药的享受这一好处。

黎明  
价格：D+1000  
描述：色泽略暗，偏向暗金光泽的镀层，在弧度的位置选取了乳白色，反衬出整体光泽。  
效果：该武器视为具有【光明】或【神圣】属性，这可能影响到一些黑暗生物的反应。当你持有该武器进入战斗时，你的先攻增加等同自身【传奇决心】个十面骰（即1传奇决心对应1D10先攻增加），这个效果仅持续一轮。  
持续时间内，你使用该武器进行的攻击附带1点额外韧性伤害。

霓虹  
价格：C+1000  
描述：如同五颜六色霓虹灯一样的镀层，相当华丽，仔细看的话会发现表面持续性的光影闪动。  
效果：该武器视为具有【塑料】或【玻璃】材质，购买时自行选择。迅捷动作启动，该武器获得等同你的【（传奇智力-1）\*8】点破防，这些破防必须依次平均分配给破甲，高速，威猛，破魔。；该武器伤害类型永久性改为纯能量伤害，同时可以作为照明棒在夜间为你照明（启动后）。  
破防能力的存在仅持续一轮，新的一轮时仍需要继续支付迅捷动作维持。

闪电  
价格：C+1000  
描述：外观破碎，让人感觉锋利的镀层，雕刻出了闪电的符号，摸着会有麻痹感。  
效果：该武器并不能具有【电】材质，镀层无法做到为武器添加电材质属性。该武器获得【法破】特性，伤害类型转为雷电能量伤害；使用该武器进行攻击时，附带等同你的【传奇智力-1】点不可豁免的麻痹不良点数，可叠加。

樱花  
价格：C+1500  
描述：有着相当华美装饰的粉色镀层，上面精心雕刻了樱花的图案，做工似乎比其他镀层更用心。  
效果：该武器视为具有【木】材质。使用该武器攻击时，触及距离提升等同你【传奇风度\*2米】，同时可以对范围内的所有敌对目标进行攻击；你进行拔刀检定时（详见武技-拔刀师），在拔刀检定中额外获得等同自身【传奇风度】\*2个成功数。  
这些额外成功数转化为先攻时会以1：2的数值转化为先攻，但计算攻击检定加值则只能等量转化而不是1：2。  
若有其他能力会使拔刀术检定带来更多效果，则该镀层不会提供任何额外效果。

日光  
价格：C+2000  
描述：闪耀的纯白色镀层，充满了能量武器的质感。轻拿轻放，小心灼伤双手。  
效果：该武器视为具有【阳光】属性，这可能影响到一些吸血鬼之类生物的反应。该武器获得常驻的【能量武器】特性；当你使用该武器进行攻击时，附带等同自身【传奇智力\*2】+1点熔毁，每因此获得4点熔毁，则再附带1点。
