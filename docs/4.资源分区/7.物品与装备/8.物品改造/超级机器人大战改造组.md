超级机器人大战改造组

EN系统D+750  
在作品《超级机器人大战》中，EN是机体能源剩余的具体量化，机体的一切行动都要依赖于EN，EN没有剩余，机体也就失去了驱动力，无法行动。

开池子EN 初始机体支线等级X20+机体装配武器（非自带）支线等级X10。  
攻击时根据使用的武器/技艺的支线等级获得加值，D1C2B3A4S5，并消耗等额的EN值，若武器和技艺同时使用则叠加消耗计算。该能耗为固定消耗，无法避免。  
移动时固定消耗1点，移速提升5，可以增加消耗，每额外消耗1点EN增加5点移速，增加的上限等于机体的支线等级D1C2B3A4S5，移速也得到相应加成。  
若EN不足以支持能量消耗，则无法做出相应的动作，直到EN得到补充。

气力值 C+1000  
进入战斗时气力值初始为100，上限150下限50。  
战斗结束后气力值回复到100。  
气力值提升方式：  
每次攻击对至少一个敌人造成伤害+1  
受到一次攻击+1  
击杀敌人+3  
友军被击杀+1  
友军击杀敌人+1  
在100气力的基础上，每有5点气力值，攻击，防御（依附于闪避防御）和三豁免获得1DP的士气加值。且当气力值到达120时，攻击时若有消耗意志力，则这次攻击获得机体支线等级\*2点附加成功。  
若气力值低于100，每低于5点气力值，攻击，防御（依附于闪避防御）和三豁免获得1DP的士气减值。

  
适应性测试 D+1250  
驾驶员和机体可以分别购买一次  
购买时获得13点适应点数，用来提升适应性等级，每点适应点可以提升1级适应性。适应性分为四个分类，在相应的场景进行战斗时相应的适应性发挥作用。  
D级适应性全检定受到4点成功数减值，C级适应性全检定受到2点成功数减值，B级适应性不会受到任何减值，A级适应性全检定获得1点附加成功，S级适应性全检定获得2点附加成功              
      D   C   B   A   S  
海 -4   -2   0  +1 +2  
陆 -4   -2   0  +1 +2  
空 -4   -2   0  +1 +2  
宇 -4   -2   0  +1 +2  
驾驶员的适应性加值和机体的适应性加值可叠加（例如：机体为D级的空适应，驾驶员为S级的空适应，则驾驶机体受到的全检定成功数减值为2）

  
购买了EN系统，气力值，适应性测试后才可以购买  
机体改造D+500/C+1000/B+2000/A+4000/S+8000  
每个级别的机体改造可以购买五次，购买时从以下项目中选择三项，但每个选项至多选取三次  
EN改造 D/C/B/A/S EN上限提升5/10/20/40/80  
HP改造 D/C/B/A/S 结构值提升2/4/8/16/32  
瞄准改造 D/C/B/A/S 在破甲/破魔/威猛/高速中选择一项，攻击时获得2/4/8/16/32点  
装甲改造 D/C/B/A/S 护甲提升2/4/8/16/32  
运动性改造 D/C/B/A/S 移动速度提升5/10/40/160/800 反射豁免提升1/2/4/8/16 闪避防御提升1/2/4/8/16  
当同一个改造项目被选定三次后可以直接购买更高一级的同项目改造

当完成5/4/3/2/1项D/C/B/A/S级改造后，在以下自定义改造中选择一项附加在机体上，不可清除不可改变，无法重复获得  
1.计算所有的结构值提升之后，每有3点结构值额外提升1点结构值。结构值提升3点  
2.机体获得快速修复6，医疗点112，每修复1点结构值消耗1点EN  
3.在EN的基础用法上，可额外支付EN获得等量的加值，上限为3倍基础EN消耗，EN上限提升30  
4.全地形适应提升2级，并且提供等同于适应性加值的洞察防御上的附加成功。  
5.受到攻击时可以支付EN抵消伤害，每支付3点EN可抵消1点伤害，伤害抵消上限为10点  
6.获得B级科技本质的侦测等级。且进行洞察检定时获得等同于机体支线等级x3的器械加值。

整备组妖精D+500  
一台机体的战斗需要的不只是驾驶员，还有数十倍的后勤人员，才能保证机体永远处于最佳状态。鉴于大部分轮回者都无法自带战舰和机组人员，于是主神提供了整备组妖精来协助轮回者维护自己的机体  
妖精的全力修复！：当处于非战斗状态时，可以选择使用整备组妖精来为机体进行后勤休整，修整需要一个小时。在此期间，机体每分钟回复1结构值。EN完全恢复，但平摊在每分钟上。  
一次整备至多回复44点结构值，之后需要八小时补充资源才能继续回复结构值。  
妖精的资源节省！：每经过一场影片，一次整备可回复的结构值上限提升8点。
