# 葛拉斯诺晶体系列

葛拉斯诺晶体  
原典：《魔塔大陆》系列  
描述：从很久以前就存在的，在世界各地散落，有着特殊力量的晶体。若是与物品融合，可以制作附有特殊效果的物品；给蕾瓦蒂亚安装的话可以强化诗魔法，每种晶体的力量各不相同。  
晶体等级根据购买支线等级计算，无支线等级不占据槽位。

【若无特殊说明，每件武器/防具可以根据支线等级安装多个装备类-葛拉斯晶体，根据支线等级分别获得D1C2B3A4S5个插槽，晶体也会根据等级占用D1C2B3A4S5个插槽。】  
【葛拉斯诺晶体只能在战斗外拆卸/安装，这需要结合在一次短休息中进行。】  
【对同一单位而言，同种类/效果的葛拉斯晶体仅有一枚会对其生效，可差价升级的葛拉斯晶体也会视为同种类/效果的晶体处理，即使差价升级后效果替换。】

附录1：诗魔法类的葛拉斯诺晶体可以直接安装在蕾瓦蒂亚身上的接口处，与蕾瓦蒂亚的身体互相融合。蕾瓦蒂亚每个诗魔法根据支线等级分别获得D-A级插槽，最高A级的诗魔法具有D-A级的插槽各1个。高等级插槽可以安装更低等级的葛拉斯晶体，但无法安装更高等级的葛拉斯晶体。

附录2：诗魔法用的葛拉斯诺晶体安装后会产生身体负荷，间接造成诗魔法消耗增加的缘故，实际上也只有蕾瓦蒂亚能承受这种负荷，其他存在无法安装也无法融合，即使用某种方式成功强行融合的话，除非满足了前置要求血统为圣咏少女（即蕾瓦蒂亚）。否则也无法使用。  
【根据安装诗魔法-葛拉斯诺晶体，可能会增加愿力/意志力消耗，但可以通过安装专用的晶体减轻这一消耗。】

附录3：无关紧要的说明一下，蕾瓦蒂亚的接口位置通常在锁骨，后颈之类的敏感部位。

【装备类-葛拉斯诺晶体】  
稍稍增加HP  
价格：1000  
分类：奇物  
描述：使生命值上限稍稍提升的葛拉斯晶体。虽然不起眼，但有时候能救你一命。  
效果：持有安装该葛拉斯诺晶体的装备时，生命值上限+1点。

体力增强芯片  
价格：D+500  
前置：稍稍增加HP（差价）  
分类：奇物  
描述：使生命值上限一定程度上增加的葛拉斯晶体，可以由多个“稍稍增加HP”进行合成。  
效果：持有安装该葛拉斯诺晶体的装备时，生命值上限增加等同【装备支线等级】\*1点，强韧豁免增加等同【装备支线等级】\*2点。

价格：C+1500  
前置：体力增强芯片（差价）  
分类：奇物  
描述：使生命值上限大量增加的葛拉斯晶体，可以由多个“体力增强芯片”进行合成。  
效果：持有安装该葛拉斯诺晶体的装备时，生命值上限增加等同【装备支线等级】\*2点；持有安装该葛拉斯诺晶体的装备时，持有者不会因伤势昏迷。

价格：500  
分类：奇物  
描述：使能量上限稍稍提升的葛拉斯晶体。能稍微为你增加一点宽裕。  
效果：选择一项能量池，一经选择无法改变；当持有安装该葛拉斯诺晶体的装备时，使选择的能量池上限增加6点。

兆储存器  
价格：D+1000  
前置：稍稍增加MP（差价）  
分类：奇物  
描述：可以储存较多能量的葛拉斯晶体，在恢复能量的范畴内也可以为你提供帮助。  
效果：  
选择一项能量池，一经选择无法改变；当持有安装该葛拉斯诺晶体的装备时，选择的那一项能量池上限提升等同你【决心/2】\*3点。  
此外，只要你选择的那项能量池不空，你就可以指定一项属性，在相关检定上获得1个额外附加成功数，一经选择无法改变。

太储存器  
价格：C+1000  
前置：兆储存器（差价）  
分类：奇物  
描述：可以储存大量能量的葛拉斯晶体，除了有利于恢复外，也能减少能量消耗。  
效果：  
\*选择一项能量池，一经选择无法改变；当持有安装该葛拉斯诺晶体的装备时，选择的那一项能量池上限提升等同你【（决心/2）+2】\*3点。  
\*每轮一次，当你消耗选择的那项能量时，能量消耗减少等同你【传奇决心】点，最多降低至1，无法与其他能量消耗减少的效果叠加计算。  
\*只要你选择的那项能量池不空，你就可以指定一项属性，在相关检定上获得1个额外附加成功数，一经选择无法改变。
