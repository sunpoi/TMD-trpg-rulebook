# 基础护甲

  

防弹衣 25分  
护甲类型：轻甲  
防弹衣具有【防弹】特性，具有年代修正  
防弹衣的盔甲防御为2/2

全身铸甲 75分  
护甲类型：重甲  
全身铸甲具有【封闭1】，具有年代修正  
全身铸甲的器械减值为3点，防御为4/3

板甲 50分  
护甲类型：重甲（只占据上肢）  
板甲具有1dr/穿刺，无年代修正  
板甲的器械减值为2点，防御为3/3

锁子甲 25分  
护甲类型：轻甲  
在计算攻击者的破甲时，锁子甲最多还可以再额外抵消其2点破甲，无年代修正  
锁子甲的盔甲防御为2/2

防暴服 75分  
护甲类型：轻甲  
防暴服具有1dr/钝击，无年代修正  
防暴服在穿戴者手持盾牌时，还会额外提供至多1点仅限于应对近战攻击才能获得的防御  
防暴服的盔甲防御为2/2
