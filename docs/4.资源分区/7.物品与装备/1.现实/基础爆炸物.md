# 基础爆炸物

基础爆炸物  
  
手雷  
  
按照常理而言，投掷手雷是一件需要力量的战术动作，所以如果试图投掷手雷时，你只能通过力量进行投掷，专业为力量+投掷，射程为10\*力量。特殊的，投掷手雷是需要进行一个前置动作，如拉开拉环，点燃鸡尾酒的引子等等，这回消耗掉你的一个迅捷动作，一般而言，手雷将会在下个回合开始时触发，所以如果被对面捡起来丢回来，那么手雷也会在你身前爆开。  
  
  
莫洛托夫鸡尾酒（2分／瓶）  
  
大名鼎鼎的莫洛托夫鸡尾酒，投掷前需要一个类移动作点燃。  
  
燃烧弹能够在5米直径范围内造成5L火焰伤害，以反射检定抵消。如果受到至少两点伤害者，将会获得一点燃烧点数，以标准方式豁免。  
  
  
马克ii型手雷（2分/一颗）  
  
常见的军用类型手榴弹，需要拔下插环，并松开保险才能引爆。  
  
该手雷能在10m范围内能造成最大杀伤，3L2A穿刺伤害，以反射检定抵消，优先抵消A伤。最高杀伤范围可达50m，但超过10m范围的杀害一律修改为冲击伤害。  
  
  
震撼弹／闪光弹／催泪瓦斯（10分／颗）  
  
最常见的特种手雷，能够制造强烈的声响、闪光，或是催泪瓦斯气体。  
  
震撼弹能够在直径5／10米范围内造成3／1点耳鸣点数，以标准状态抵消。  
  
闪光弹／催泪弹则是造成目眩／恶心点数，强度相同。  
  
这些不良状态会以每2轮1点的速度自行恢复。  
  
  
烟雾弹（10分／颗）  
  
能够大量制造有色或无色烟雾，购买时可以自行决定烟雾是否具有颜色。  
  
烟雾弹能够在直径20米范围内释放大量烟雾，持续2D5轮或被吹散。  
  
烟雾中的所有生物在防御和躲藏判定上获得+2隐蔽加值，但烟雾外的生物对于烟雾中的人来说同样也具有以上加值。  
  
  
RG-60TB型手雷（20分/1颗）  
  
  
又叫做温压手榴弹，它与常规手榴弹的区别在于弹体内部，其内部有引信加少量的爆炸物和240克云爆剂，拉开保险头之后，首先引爆少量爆炸物，将云爆剂炸飞出去，云爆剂将在空中与氧气混合产生二次爆炸。  
  
RG-60TB型手雷将直接对10m范围内敌人造成2l3a的爆炸伤害，使用强韧豁免进行对抗，同时因为2次爆炸将对20m范围内敌人造成2点冲击伤害，使用反射豁免进行对抗。  
  
  
一式氢酸手榴弹（20分/1颗）  
  
由日本研发的奇葩毒气手雷，外壳是玻璃瓶，里面装载着浓度达到80%的液态氧氰酸，遇到空气迅速挥发，毒性极强。  
  
投掷后，包括本轮在内的接下来四轮的结束阶段，半径5米内所有需要呼吸的生物每轮受到2点毒素伤害，但只能进行一次强韧检定来抵消掉头（检定成功数）点伤害。同时在生效的范围内还会造成烟雾弹效果。
