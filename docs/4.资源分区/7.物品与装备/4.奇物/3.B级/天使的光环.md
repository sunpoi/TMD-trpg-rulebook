# 天使的光环

天使的光环  
价格：B+2000  
分类：奇物  
位置：头部  
描述：充满神圣感，戴在头上能够保持精神正常的环。据说圣人为了保持自己的精神而使用。  
效果：佩戴者获得免疫不超过A级心灵来源效果的能力，获得【光明生物】分类，并降低1点意志力上限；佩戴者在决心、沉着相关检定上获得等同【智力属性/2】+1DP器械加值。
