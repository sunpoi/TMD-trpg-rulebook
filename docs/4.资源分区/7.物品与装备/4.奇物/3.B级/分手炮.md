# 分手炮

  

分手炮  
  
B+2000  
  
特异本质  
  
一个外表为1cmX1cm的正方形贴纸，贴纸图案为一颗被子弹打穿的红心。  
描述：一种枪械附加装置，可以贴在枪械的任意位置，使你弹匣中的最后一发子弹必定暴击。  
  
分手炮具有以下三种效果，这些效果每场景只会生效一次。  
  
一、不再  
“要分手了，我将我们的过往都留在这次。”  
  
当你的枪里仅剩一发子弹时，你这次攻击所造成的实际伤害增加一倍。  
  
二、离别  
“分手炮结束后，就走吧，不要回头。”  
  
效果：当你触发了上一个效果后，若你本轮没有移动过，则使你下一次移动的移动距离提升一倍。若本轮你已经移动过了，则你立即进行一次额外的移动。  
  
三、不复  
“分手炮若是多打，那就失去了它名字的含义。”  
  
你可以令一次攻击在投掷成功数后具有【湮灭】特性，若已具有则额外获得9dp的攻击加值，这是器械加值。  
  
  
特别说明：使用本奇物附加到枪械上以后，便无法从该枪械上拆除。即本奇物自动绑定你为其附加的枪械。本奇物的效果无法在【短点射】 【长点射】 【连射】中生效。  
本奇物必须加装在弹夹最大子弹数2及2以上的枪中生效。  
枪械类型指狙击枪/来复枪，步枪，冲锋枪，手枪，霰弹枪，机枪。
