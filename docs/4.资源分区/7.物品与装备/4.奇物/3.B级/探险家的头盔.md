# 探险家的头盔

探险家的头盔  
价格：B+4000  
分类：奇物  
位置：头部  
描述：无论去怎样的秘境、也一定能克服危险归来、因此而出名的阿尔卡姆探险家的遗物。  
效果：装备者获得免疫不超过A级环境来源效果影响的能力；当处理生命值上限时，装备者再额外获得等同【耐力属性/2】+1点生命值上限。
