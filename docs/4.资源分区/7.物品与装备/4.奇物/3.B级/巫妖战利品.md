# 巫妖战利品

巫妖战利品  
B+2000

  
这是一群管理着暮色沼泽、黑暗森林和积雪森林的结界领域的大法师，即使他们的身躯早已化为了白骨。当我们击败娜迦后，便可以进入到巫妖高塔之中，这里的视野不是很明亮，而且乱飞的书本时常扰乱我们登顶的步伐，不过最终我们还是来到了塔顶遇见了这位脾气差劲的大法师……我想想我是被杀干掉的，那些幻影？那些法球？那些召唤的僵尸？还是最后因为得瑟故意吃了那一发大宝剑？

  
该奇物是一个巫妖的头颅，占据头盔位，虽然几乎把你整个头部遮盖起来，但并不会令你的视野受到影响，当戴上此奇物后，你将获得巫妖的部分能力

  
黑暗生物：虽然你只是带了那个骷髅的头颅，但你依旧不可避免的会被认为是那些骷髅的同类，你获得【黑暗生物】特性

  
法术护盾：虽然它本体是召唤五面物理免疫的护盾，但是很显然，我们搞不到这么离谱的效果，所以你的周围只会长出三面金色的护盾来帮你抵御那些物理伤害，当然，法破伤害还是该吃就吃吧，你获得5DR/魔法

  
生命吸取：巫妖第一和第二形态时，若自身生命值不处于全满状态则会吸收周围的怪物（如僵尸、蜘蛛、末影人）来回复自身生命值。被吸收的怪物在巫妖短暂施法后会化作一道白烟消失，当然，你没有这么方便的回血能力，你的攻击获得【吸血1】

  
法球：巫妖在进行3秒的准备后，会朝玩家发射一枚法球，消耗一个标准动作吟唱，随后进行一次施法鉴定，这次检定距离增加15米，来决定发出一枚附带【高速5】【破甲5】的法球，对方反射豁免。

  
仆从召唤：按道理，应该是召唤数名僵尸，与你一起发动攻击，但是不同世界的僵尸情况不一样，再加上这仿制品做不出来这种效果了，因此这玩意就改了改效果，你现在可以花费自由动作召唤一个由方块做成的的僵尸，每个场景限制1个，并且持续时间为一场景或十分钟（取低）

  
瞬移：你知道吗？巫妖死亡时，经验可能不会掉落在其死亡的地方，这是因为巫妖的频繁瞬移导致其死亡时尸体也瞬移到了其他地方。  
每当你发射一枚发球后，你可以瞬间移动到法球消失的位置，瞬移的最远距离不能超过你自身的移动，这是一个C命运来源的传送移动效果
