# 大元素使

  

大元素使  
  
B+2000  
概念武装  
  
“元素在呼唤。命运就在眼前。”  
“光明只是众多道路之一。”  
“心志聚焦于战场！”  
【元素秩序】：  
元素从来不是一个定型的东西，它千变万化，可以存在任何地方，它们是世界构成的基石。而大元素使们是操纵元素的主宰，他们可以清晰地感知到元素的呼吸和流动，数不清的元素簇拥着，等待着主宰的驱使，在某些可以感受或者是观测到能量眼中，你如同元素领主一般令人感到伟岸。在他人的观测之中，你仿佛是元素的化身。  
\- 在进入战斗后，每回合开始时，你需要在灼热（火）、电击（电）、冻寒（冰/水）、腐蚀（土）、音波（风）、光能（光/以太）六种元素类型之中选择一种元素，你任何攻击所造成的伤害都可以转化为选择的元素对应的伤害。  
【元素之拥】：  
对于现在的你而言，最可靠的已经不再是所谓的盔甲或者坚韧的肉体了，而是那些围绕着你的元素们，你放开身心张开怀抱迎接它们，元素们钻入你的护甲和皮肤，化身为守护你的华丽战袍，在你的脉搏中流淌，直到它们和元素融为一体，一开始保护你的元素战袍只是由最纯粹的能量组成，它看上去如同绸缎华服,璀璨夺目，但是元素形成的流苏和隐隐若现的甲叶又在诉说着它那不俗的防御力，而随着时间的推移和局势的变化，纯白的战袍将逐渐的变化成为你所选定那款元素风格，耀眼的光辉开始转化为雷霆又或者是烈焰，而甲叶也化为了元素在战袍上环绕交织组成你所心仪的图案，并且渐渐的覆盖你的武器甚至是头发（发型会发生相应的改变）。  
\- 现在，你可以选择一件你所拥有的盔甲，并花费等于其支线等级数\*1小时（D1-S5）的时间进行元素同调。同调完成后，除了外观上的变化，当你在穿戴此盔甲时，该盔甲提供的防御类型将变为能量防御（类型为所选择的元素对应的类型），并提升盔甲等级（D1-S5）\*5点对应的能量防御，被转变后的盔甲将不再满足轻甲或重甲专长的生效条件。  
备注1：你同时只能同调一件盔甲，当你同调另一件盔甲时，之前的同调将自动解除。  
备注2：如同材质改造、盔甲附魔、步人甲的甲片等这些直接提升盔甲效果的被视为盔甲的本身，这将会被一同转化；而通过技艺等通过使用技巧来提升盔甲使用效率的能力不会被转化。  
【元素律令】：  
“我必须经受这元素洗礼！”  
\- 你能以一个自由动作变身（可以自己设计动作，也可以说一段帅气的话），持续一场景。你可以在该回合所选的单元素基础上，再选择一个元素进行融合（第二个元素类型在选择后，本场景之内不可更改）。  
随后你将化身为那种融合后的元素（比如：灼热（火）+腐蚀（土）=岩浆），此时你所在的地方将形成一个巨大的元素涡流，随元素肆意涌动。元素不知所源，却自四周呼啸席卷而来，而你则是在旋涡之中傲然屹立，元素的力量从你的灵魂涌现而出，此刻，你即是元素，这也会小幅度的影响这个区域的天气变化。  
这是对于元素掌控更进一步的表现。  
在变身后，元素会协助你的进行战斗，他们便是你的耳目，之前遥不可及的距离也变的抬手即可触摸，此时你的敏感范围提升一倍。
