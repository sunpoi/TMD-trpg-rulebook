# 钉刺项链

钉刺项链  
价格：B+4000  
分类：奇物  
位置：护符位  
描述：带刺的项链。返还所受伤害的五分之一、增加攻击力。参考自古相传的带刺头盔、古怪的贵族制作的玩笑之作。  
效果：  
【荆棘】：佩戴者获得基于天生防御的【反震】。

【刺伤】：当佩戴者进行近战攻击时，在攻击检定上获得等同4DP器械加值，攻击附带1点不可豁免的流血不良点数。
