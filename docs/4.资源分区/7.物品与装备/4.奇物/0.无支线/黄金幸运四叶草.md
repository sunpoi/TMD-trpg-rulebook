# 黄金幸运四叶草

  

黄金幸运四叶草  
500  
特异本质  
占据护符位  
一枚黄金制成的，做工精致的四叶草项链，大约有10克重，即使作为世俗物也值上一些钱财，其四叶草内测则雕刻着古希腊语写成的Tyche——La posedanto havos bon?ancon （持有者将拥有好运）的字样  
效果：  
这枚项链可以使得穿戴者抵抗厄运。比如掉落的盘子很少摔碎，丢失的东西更容易容易找到，不小心散落的火花基本不会引起火灾等等，总而言之，这枚项链可以让穿戴者所进行的，鉴定/豁免难度为DC1的行为不会失败，还具有机运骰获得1点加值的效果。携带多个四叶草也不会重复获得好处。  
但是，这块石头对任何他人蓄意的行为都无法生效，也无法防止穿戴者鉴定/豁免难度超过dc1的诅咒或巨大的灾难。
