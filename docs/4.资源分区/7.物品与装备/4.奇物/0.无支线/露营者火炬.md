# 露营者火炬

露营者火炬

名称：露营者火炬  
价格：1500  
分类：奇物  
本质：魔幻本质

描述：  
这是一根约3磅重的魔法火炬

在帐篷构筑期间你可以对火炬说出所需要的帐篷样式、颜色、花纹、形状以及温度，但不管怎么样它都只能是一个最多边长6米高3.5米的方形帐篷，允许任何体积低于7的单位进入  
这个魔法帐篷墙壁和天花板都是由结实帆布架成的，而地板则是一片片光滑平整的石材或木板，它可以在外界（-30摄氏度到30摄氏度）的温度区间内，保证帐篷内20度恒温，之后外界温度每上升或者下降1度，帐篷内温度也会同比变化。  
原本的火炬会在帐篷构装完成之后变成一个奇妙的火炉，火炉中的火焰就像普通火焰一般可以发光发热，但却不需要消耗空气也不会产生烟雾，点燃和熄灭炉火同样需要第一个命令语。

这个帐篷视为一件现实帐篷，展开后最大具有10点体积，如若开启物品破坏规则，则视为具有11点结构值和1点硬度。

一旦火炉再次变回了火炬，那么在接下来的12个小时里都无法再次令其构筑起一个新的帐篷。

特性：【Ignisthral】【Spiranfold】  
效果：

【Ignisthral】  
你可以使用一个迅捷动作说出第一个命令语【Ignisthral】，使其顶部燃起一束魔法火焰，这和普通火炬没什么区别（为其身边6米半径范围提供常光光照，以及其外6米范围的昏暗光照）。当你再次以一个迅捷动作说出这个命令语时，火炬会熄灭。

【Spiranfold】  
你也可以使用一个移动动作将火炬插在地上并说出第二个命令语【Spiranfold】，当你这么做的时候火炬会利用魔法力场在1分钟内逐渐构筑出一个帆布帐篷，在帐篷中休息会视为具有自适应长休/自适应短休。

帐篷一旦构筑完成就会在原地存留12小时且无法移动，再次说出第二个命令语可以令帐篷迅速消散并使火炉重新变为火炬状态，任何不属于帐篷的事物在帐篷消散之后将被放在原本的位置的地面上。
