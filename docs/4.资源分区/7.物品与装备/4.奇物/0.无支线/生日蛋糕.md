# 生日蛋糕

生日蛋糕

名称：生日蛋糕【样品】  
价格：125分  
分类：奇物  
本质：魔幻本质

描述：  
当轮回者结束任务，从影片返回在主神空间休息之时，他们是否会在幕间怀念先前的时光呢？是否会怀念如今只能暂存在回忆的亲朋好友、欢喜冤家与死党呢？啊...当然，那些本来日子就不太好的轮回者大抵是不会怀念这些的。总之，为自己再一次完成任务，安然返回或被队友带回主神空间买下这个小东西吧，今天是你到主神空间的第几天了？你又一共在影片里过了多少天呢？

【真挚而朴素的祝福】：很可惜，这个蛋糕似乎是个样品，因而你无法吃掉它来饱腹，但它本身所传承的意愿是美好的，也因如此，本物品在购买时也可以替换为其他文化中具有相似意义的物品，例如长寿面等。

特性：【真挚而朴素的祝福】  
效果：  
【真挚而朴素的祝福】  
只要你持有这个奇物，每经过一个影片便获得1年寿命。

名称：生日蛋糕  
价格：1000  
分类：奇物/永久消耗品  
本质：魔幻本质

描述：  
当轮回者结束任务，从影片返回在主神空间休息之时，他们是否会在幕间怀念先前的时光呢？是否会怀念如今只能暂存在回忆的亲朋好友、欢喜冤家与死党呢？啊...当然，那些本来日子就不太好的轮回者大抵是不会怀念这些的。总之，为自己再一次完成任务，安然返回或被队友带回主神空间买下这个小东西吧，你是否对望不到头的任务感到厌倦，还感觉自己会倒在任务途中呢？

特性：【美好且长存的期盼】  
效果：  
【美好且长存的期盼】  
当你吃完后，获得【不老】特质。
