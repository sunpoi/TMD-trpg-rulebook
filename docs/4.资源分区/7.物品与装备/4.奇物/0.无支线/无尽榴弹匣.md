# 无尽榴弹匣

无尽榴弹匣

名称：无尽榴弹匣  
价格：1000分  
分类：奇物  
本质：特异本质

描述：  
一个看起来具有一定容量的钢匣，顶部是一个盖子，可以自由取用其中的手雷，一般的，其中只会存在两枚手雷，当你取出之后自动补满。

特性：【无限榴弹】  
效果：  
【无限榴弹】你可以从其中源源不断地取出马克ii型手榴弹，但同一时间，且未引爆时，只能存在两枚马克ii型手雷。
