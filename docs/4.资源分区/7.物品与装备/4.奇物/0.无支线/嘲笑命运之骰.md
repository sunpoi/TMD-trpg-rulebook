# 嘲笑命运之骰

嘲笑命运之骰

名称：嘲笑命运之骰  
价格：1000  
分类：奇物  
本质：特异本质

描述：  
外观上看这是一枚骨白色的六面骰，其中的点数三面为黑色漆，三面为红色漆。这枚骰子是亵渎命运之神的产物，因此受到了命运之神最恶毒的诅咒，即只能骰出最小的点数

效果：  
所以无论你以何种方式投出这枚骰子，无论它处在任何空间任何世界，只要它落地，就必定是1点，这是一个D级命运来源的改变效果。
