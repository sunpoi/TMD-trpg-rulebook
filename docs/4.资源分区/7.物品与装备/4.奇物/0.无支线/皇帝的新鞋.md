# 皇帝的新鞋

皇帝的新鞋  
本质：特异本质  
分类：其他  
体积：1  
重量：忽略不计  
价格：100  
描述：现在你也可以光脚走路不用怕沾到什么了。

  
赤足：现在你的脚不会沾染任何污渍(也不会出汗)，永远处于清洁状态，踩到尖锐物品时，视为穿着普通的鞋子一般。作为一种常见的生活方式，赤脚运动被很多人们所接受。也有极少数的赤脚爱好者选择了在公共场合全环境、全地形终身赤脚的生活方式，完全摒弃了鞋袜。而长期赤脚会出现脚趾间隙变大、脚跟开裂以及足部皮肤逐渐粗糙的现象。但是，有了它，你再也不用担心这些了，你的小脚永远干净洁白。同时，你可以在购买时决定它的类型，比如绑在脚上的丝带或者脚环什么的（你只能在不会将脚掌遮盖或包裹的物体中选择）。

  
同时，这并不影响装备位-鞋，你仍然可以装备鞋类道具，并从中获益，这仅仅是一个画面效果。

  
裸足天下第一：但穿戴者装备的鞋子类装备时，该装的会消失，当然这只是一个画面效果，你依旧视作装备者该装备并享受该装备的加成。在画面上，你的脚会看起来像什么都没穿，摸起来也一样。
