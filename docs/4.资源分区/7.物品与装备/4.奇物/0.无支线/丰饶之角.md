# 丰饶之角

丰饶之角

名称：丰饶之角  
价格：250  
分类：奇物  
本质：魔幻本质

描述：盛满珍贵材料的金角，来自S女士除夕夜赠礼。  
这是一个古老的祝福，包含她的慷慨分享与美好心愿，以及一点闪烁的灵光。

效果：只要持有这个物品，其他人就会认为你是个很有别情雅致的超级富豪，这是个无支线级来源的心灵影响效果。  
如果有人试图使用其中的珍贵材料，并进行长时间研究就会发现神奇的事实：使用其中材料进行的任何行为结果都会自动失败，变成无害的随机没用结果。
