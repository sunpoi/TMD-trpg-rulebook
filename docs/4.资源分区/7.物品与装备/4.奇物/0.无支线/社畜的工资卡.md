# 社畜的工资卡

社畜的工资卡

名称：社畜的工资卡  
价格：500分  
分类：奇物  
本质：特异本质

描述：  
某人：“我的钱呢？”

本奇物购买时为工资卡赋予一个两个字的名字，这视为你从主神处购得了这个名字的社畜每个月放入除了基础生活费之外所有分红和奖金的钱包

“该死，这个月又白干了”

效果：这视为你在任何具有货币这一贸易体系和社会这一组织体系的世界观中，取得一个家庭一个月小康生活的金钱，并且每个月刷新一次。
