# 社畜的工资卡

  

社畜的工资卡  
500  
某人：“我的钱呢？”  
本奇物购买时为工资卡赋予一个两个字的名字，这视为你从主神处购得了这个名字的  
社畜每个月放入除了基础生活费之外所有分红和奖金的钱包，这视为你在任何  
具有货币这一贸易体系和社会这一组织体系的世界观中，取得一个家庭一个月  
小康生活的金钱，并且每个月刷新一次。  
“该死，这个月又白干了”
