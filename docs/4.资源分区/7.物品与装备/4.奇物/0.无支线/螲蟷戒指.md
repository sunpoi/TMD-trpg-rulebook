# 螲蟷戒指

  

螲蟷戒指  
1000  
魔幻本质  
螲蟷会掩饰自己巢穴的穴口 ，祈求追猎者无法发现她。而以螲蟷形象制造的戒指则可以帮助玩家将一扇门藏在幻象的后面。  
动作：如下  
持续时间：一个场景，或下一次太阳滚入地平线，二者取低  
效果：玩家通过一个开口（无论是门、窗户还是个墙上的洞口）时，可以标准动作摩擦戒指将其封闭，使其看起来无法通行（门看上去是锁上的，或者是变得非常坚固，也有可能干脆变成了一堵墙，由玩家施展该物品能力时决定具体表现），持续一个场景。但这幻觉只是视觉上的幻象不会实际影响通过，视为D级幻象来源的伪装效果。
