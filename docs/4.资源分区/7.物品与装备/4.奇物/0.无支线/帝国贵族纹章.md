# 帝国贵族纹章

帝国贵族纹章  
价格：1500  
分类：奇物  
位置：饰品  
描述：骄傲的帝国贵族戴的纹章。但是、新手帝国骑士的任务大半是脏脏的任务、荣耀的证明往往被封藏起来。  
效果：佩戴者的负面声望提升10点，并提升因此获得声望数值一半的外貌，需要注意的是，这部分外貌是由于此徽章的身份象征带来的，若佩戴者不具有可靠的身份来源（至少为C级别、对应为贵族或是富豪的背景身份），则可能会因为身份暴露而在影片内失效。  
注：若剧本世界观中，贵族身份并不具有统治意义，则这个徽章也可能在影片内失效。
