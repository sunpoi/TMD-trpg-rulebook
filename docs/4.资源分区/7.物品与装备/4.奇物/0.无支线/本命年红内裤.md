# 本命年红内裤

本命年红内裤

名称：本命年红内裤-圣诞版  
价格：750  
分类：奇物/内裤  
本质：特异本质

描述：“为什么圣诞礼物是条大红内裤啊！”  
“本命年，当然要穿红内裤，红色能驱邪纳福，保卫自身平安！袜子也要换成大红的，从头到脚都沾喜气！”——某户华人家庭的对话  
一条普通的红色内裤，在获得时会根据获得者的体型改变成合适的型号，并会在内裤的一角用金线绣上获得者生肖的卡通形象。

效果：如果今年是穿戴者的本命年（按照年龄是否可以整除12计算），则穿戴者会在机运骰上获得1点加值，并且穿戴者如果同时穿着了红色的袜子，则额外在先攻上获得2点加值。
