# 夜明珠

夜明珠  
500  
禹凿龙关之山，见一神，蛇身人面，探玉简授禹……  
后掘地得赤圭如日，碧圭如月，又有青玉、白玉、明珠、夜光珠。

作为夜光珠，其发出的隐隐幽光令你看起来贵气十足，你的外貌上升1点，并且在普通人看来，你会显得至少曾经有过优渥生活，这是一个无支线等级的命运来源的背景身份。  
但是你每佩戴着这个奇物经过一天，你的寿命就减少一天，什么？你问为什么？  
原因很简单——会自行发光的，一般都是会让盖革计数器响的东西。
