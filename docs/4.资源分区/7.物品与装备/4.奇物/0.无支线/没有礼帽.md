# 没有礼帽

没有礼帽

名称：没有礼帽  
价格：250分  
分类：奇物/占据头部位  
本质：特异本质

描述：  
没有礼帽是一个透明的布质高礼帽，在正常状态下几乎不能看见，每隔几分钟会自动清洁礼帽上的灰尘和污渍。  
其佩戴形态外观上为你把礼貌带在身上，而不是头顶

效果：  
此时你在人面前作出一些不影响他人利益的没有礼貌的事会被无视，这是一个无支线的信息来源的思维控制效果。
