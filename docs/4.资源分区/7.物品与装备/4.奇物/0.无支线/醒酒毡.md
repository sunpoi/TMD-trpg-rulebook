# 醒酒毡

  

醒酒毡  
10  
魔幻本质  
神仙小说《封神演义》中的宝物  
醒酒毡，为明代神仙小说《封神演义》之中的宝品，一种可以解酒的至宝，乃是由西伯侯姬昌长子“伯邑考”进贡给商纣王的西岐三宝礼品之一。书中有云——“人若酩酊大醉，卧于此毡，不消片刻即会醒酒。”  
醒酒毡何以接触所有无支线的普通酒类所造成的醉酒效果，只要在醒酒毡上躺上15分钟，立刻便可恢复清醒，彻底醒酒，不在眩晕恶心。
