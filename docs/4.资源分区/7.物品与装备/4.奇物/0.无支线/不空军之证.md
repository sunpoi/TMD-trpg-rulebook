# 不空军之证

不空军之证

名称：不空军之证  
价格：250  
分类：奇物  
本质：特异本质

描述：  
不空军之证是一把青蓝色现代钓竿，表面覆盖高强度透明纳米涂层，防水防污同时增强金属光泽，在阳光下折射出细碎星芒。竿身嵌入激光雕刻的流线型暗纹，形似鱼群游动轨迹，触感细腻且在特定角度显现低调奢华。  
导环座与尾堵采用冷光材料，夜间垂钓时散发幽蓝微光，兼具实用性与科幻感，竿上写着4个大字  
“永不空军”

效果：  
使用用不空军之证可在任何长宽深不小于10mm的水池中垂钓，在至多10分钟内就可以钓上一个垂钓者想要的任意食品罐头，但里面的食物不存在任何鱼纲下属生物和超凡生物作为原料。只有吃完一个罐头，才能再次钓起一个有内容物罐头，不然垂钓起的只能是空罐头。  
罐头在钓起半小时后会自然消失。
