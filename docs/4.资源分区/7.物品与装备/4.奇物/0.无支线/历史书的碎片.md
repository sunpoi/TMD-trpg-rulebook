# 历史书的碎片

历史书的碎片  
价格：250分  
分类：奇物  
位置：饰品  
描述：以太过古老不像这个世界的话语写的书的碎片。无法从中得到力量、只能学会其中含有的魔法。  
效果：持有者获得一个新的无支线法术【零飞弹】。  
持有历史书碎片的单位可以将持有的一把现实武器转为法器，法器可以作为施法武器计入法术攻击，但不具有任何效果，只具有等同转化前的武器伤害的法强，且只能指定一把现实武器进行转化。

【零飞弹】：进行一次基础规则中的法术攻击，基础射程15米，伤害类型为纯能量。
