# 维生汤匙

  

维生汤匙 Sustaining Spoon  
200  
如果将这柄平平无奇的餐具放置在空容器中，粘稠的面糊就会填满容器。尽管这种面糊尝起来就像是温暖潮湿的纸板，但是其营养实际上十分丰富，同时含有让任何食草动物、杂食动物或食肉动物维生所需的一切。汤匙每天至多可以产生供四名人类食用的面糊。
