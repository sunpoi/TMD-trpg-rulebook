# 粗草绳头带

粗草绳头带  
价格：250  
分类：奇物  
位置：头部  
描述：流浪的民众寻求新的土地度过雪原时使用的绳子。将绳子紧绑在头上集中精神、保持身心正常。  
效果：佩戴者在购买时选择获得毒素/药物，或心灵来源效果的免疫能力，皆为无支线等级。
