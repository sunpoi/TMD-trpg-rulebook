# 魔具宠爱

魔具宠爱  
价格：1500  
分类：奇物  
位置：戒指  
描述：据说在异界受精灵保护的男人戴的指环，那个男人为了保护主人奉献了一生、一直活到寿命用尽。  
效果：佩戴者获得快速医疗1（医疗点数12）与1点力场防御；佩戴者免疫不超过C级衰老效果影响。
