# 天人的凉鞋

天人的凉鞋

名称：天人的凉鞋  
价格：1000  
分类：奇物/占据鞋袜位  
本质：特异本质

描述：据说是高贵者穿的、古代鞋子。优秀的穿着感觉让动作变得敏捷，大抵是某种安慰剂效应。

效果：穿戴者免疫不超过D级困难地形带来的影响，基础移动速度增加10米，并在敏捷相关检定上获得1DP加值。
