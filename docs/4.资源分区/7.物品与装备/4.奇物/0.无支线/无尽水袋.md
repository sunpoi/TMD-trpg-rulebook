# 无尽水袋

  

无尽水袋  
10  
赫拉喷溅而出的乳汁形成了天上的银河，遗落凡间的一滴落入了某位牧羊人的水袋中，为水袋赋予了神奇的魔力，里面的水源取之不尽用之不竭。  
这是一个皮质的水袋，容量为800ml，无论如何使用，使用者一次最多也只能从中取出800ml的可直接饮用的矿泉水。水袋的更新周期为一个小时，从使用开始计时，无论使用多少，水袋都会在一个小时后重新回归满盈的状态，此时之前取出的水会消失。  
你可以在主神空间支付250分将其变更为任意的能够合理的装水的容器，亦可支付XX分为其内容物添加任何口味，内容物只能同时拥有一种口味。改变的口味只会影响口感，本质仍是矿泉水，因此轮回者们无需担心长期引用亚健康饮品带来的危害，非常健康。
