# 血饥珠

血饥珠

名称：血饥珠  
价格：1000分  
分类：奇物/占据护符位  
本质：特异本质

描述：  
血饥珠赤红如血，清透折光，妖艳夺目。当贴身携带，可时刻吸纳人身污浊血气藏于珠中，人体因此将自发补血净血，从而体质渐强。虽其藏污纳垢，却是以自身净他人心血，当其质地浊污，便再无用处。  
购买后需滴血认主，随后开始计算：

效果：  
购买后经过第一场影片，其耐力属性提升1级。  
购买后经过第二场影片，其自然恢复速度提高1倍，强韧豁免获得2点器械加值。  
购买后经过第三场影片，获得1点快速医疗（医疗点12），强韧豁免获得2点器械加值。

在经过三场影片后，血饥珠全然浊污，不再成长。
