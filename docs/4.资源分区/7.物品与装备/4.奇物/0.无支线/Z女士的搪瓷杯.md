# Z女士的搪瓷杯

Z女士的搪瓷杯  
本质：特异本质  
类型：奇物  
价格：1000分  
描述：Z女士的搪瓷杯，上面有着“劳动最光荣”的字迹，里面还放着一个固定式料包，不知道装的是什么。

养生：当液体在杯内的时间超过半小时后，获得养生特性，喝下杯内液体时生命上限或意志力上限增加1点，直到下次长休息结束。效果无法叠加，每日只有一人可以享受这一好处。
