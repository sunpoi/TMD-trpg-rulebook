# 寂静之缠布

寂静之缠布

名称：寂静之缠布  
价格：250  
分类：奇物  
本质：魔幻本质

描述：由漆黑的圣女所制作的圣水，  
其中蕴含的祝福无法被生者所容许；  
浸泡其中的布匹，拥有了夜神之力。  
用它包裹的武器，轻若无物。

【寂静之布】:这是一条在日光下发黑，在月光下发白的粗缝制布匹

特性:【寂静之布】  
效果：  
【寂静之布】  
你可以将它包裹在任意一把武器上，其会根据所选武器的形状改变自身长度并将所选武器完全缠住 缩小，最后成为一块手帕，或任意体积不大于1的布制饰品，你可以随时以一个移动动作将布匹展开，取出武器。
