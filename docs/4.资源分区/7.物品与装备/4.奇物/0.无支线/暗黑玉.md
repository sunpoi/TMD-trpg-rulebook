# 暗黑玉

暗黑玉

名称：暗黑玉  
价格：250  
分类：奇物/占用饰品位  
本质：特异本质

描述：从魔界大洞因冲击飞来的玉石。详情不明，可以吸入周围的能量。

效果：持有者在攻击时可以指定目标的一项能量池，使其流失1点能量。流失的能量如同正常被其消耗一样处理，但不会为其提供任何好处。
