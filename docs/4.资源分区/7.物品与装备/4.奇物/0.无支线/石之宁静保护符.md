# 石之宁静保护符

石之宁静保护符

名称：石之宁静保护符  
价格：500 分  
分类：奇物/占据护符位  
本质：魔幻本质  
前提：具有魔力池

描述：  
空明兄弟会将时间和起源的魔法定式绘制到玉石护符上，减缓生命的流速，而缺陷就是，法师需要消耗魔力定期给维持法术效果的起源式样供能。  
【因为这个魔法定式比绝大多数的更高效，所以法师只需要花费7点魔力/周。但是如果每有一天没有可使用的魔力供给，法师就会承受1A致命伤害/天。并且加速衰老，衰老速度将到超过其实际年龄。】  
【这个定式并不能停止衰老，也不能抵御衰老带来的效果，仅仅是将衰老对人物身体的影响降低到一个非常小的程度。】

效果：  
携带护符的人能如此大幅地延缓衰老以至于肉眼几乎看不出的程度，并免疫D级时空来源的效果。
