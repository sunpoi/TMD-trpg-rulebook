# 幻色之瞳

幻色之瞳

名称：幻色之瞳  
价格：250分  
分类：奇物  
本质：特异本质

描述：  
一对不知道从被谁放在了空间失物招领处的美瞳，因为长时间没人认领，所以被翻修了下可爱的外包装后重新上架。其能够赋予使用者更为自由的瞳孔色泽。

这副精致的虹膜贴片只需要轻触眼眸，便能悄然融入使用者的双眼，如花瓣落入涟漪般变成眼睛的一部分，无论对自己或其他人来说都无比自然。左眼绽放出深邃的靛蓝色，星河般的微光在其中闪烁；右眼则是温暖的琥珀金色，宛如秋日枫叶间洒下的暖意。佩戴之后你的眼睛会变为如同天生的异色瞳一般。

使用时，你可以随心意改变眼睛的颜色，瞳孔的花纹甚至是动态特效——或许是瞳孔中跃动的火焰，也可能是解读文件时流过眼底的数据光纹。这些变化皆随你情绪和专注点自然变幻，如同你的小小心意如同要从眼底溢出一般。

同时，通过一道简单的意念，「幻色之瞳」还能幻化出半透明镜框，你可以随自己的喜好，将其转变为复古文艺的圆框镜，亦可转变为科技感十足的目镜模式。此模式下的视觉体验与原形态无异，但提供了更广阔的镜面界面，让你能在上面展示各种天马行空的图案和信息显示，呈现更加科幻或优雅的外观。

不过，该物品带来的效果也非全部都是画风，它确实能在一定程度上帮助了使用者处理信息，加快文字内容的阅读速度和理解深度，同时因为其本质并不会改变眼部结构，所以并不会跟瞳术之类的相冲突。

效果：  
使用者获得1点智力检定上的器械加值
