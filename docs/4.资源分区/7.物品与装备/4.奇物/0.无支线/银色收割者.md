# 银色收割者

银色收割者  
价格：250分  
分类：奇物  
位置：头部  
描述：据说多次经历战场的异界佣兵的头盔。据说成长的男人。在那之后、成为了和伙伴一同拯救世界危机的英雄，至少在故事中是这样的。  
效果：佩戴者的意志力上限减少1点，承受的盔甲减值减少1点，并获得1点士气（无名）防御。
