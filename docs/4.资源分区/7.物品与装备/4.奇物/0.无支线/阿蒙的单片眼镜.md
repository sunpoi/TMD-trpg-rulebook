# 阿蒙的单片眼镜

阿蒙的单片眼镜  
500分  
这是主神在一个地牢里，从一个牛仔打扮的尸体上捡到的。它只是一个单片眼镜，但似乎有一点点的力量。  
【记得瞄准】单片眼镜提醒你，开枪前记得瞄准，因为这样会打得准一点。  
每轮一次，你在第一次瞄准时额外获得1点瞄准上限。  
【来点乐子】生活苦涩，不妨来点乐子。  
你在带上这个单片眼镜的时候，会不由自主的想去找点乐子。当然，这只是一个单片眼镜，没有这种力量，但是它给予你了一个找乐子的理由。阿蒙只是希望你过的开心一点。

  
【阿蒙干的】这一看就是阿蒙干的。  
当你带上这个单片眼镜的时候，不管你干了什么，你都可以自由的说出“不是我，阿蒙干的”。当然，这本身没有特殊效果。
