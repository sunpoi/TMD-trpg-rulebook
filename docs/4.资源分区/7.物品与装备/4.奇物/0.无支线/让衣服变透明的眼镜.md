# 让衣服变透明的眼镜

让衣服变透明的眼镜

名称：让衣服透明的眼镜  
价格：250分  
类型：奇物/占据饰品位  
本质：特异本质

描述：一副普普通通太多圆框眼镜，只有土妹子和死宅会戴的款式。  
这其实是一副能让衣服透明的眼镜，真神奇  
你可以在购买这副眼镜时自由选择眼镜的镜框款式与镜片类型。

效果：戴上它之后就可以发现，除这副眼镜外，自己穿戴的所有装备变透明了；这是一个无支线信息来源的变形效果。
