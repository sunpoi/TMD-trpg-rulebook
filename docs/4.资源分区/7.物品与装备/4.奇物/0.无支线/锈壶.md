# 锈壶

锈壶

名称：锈壶  
价格：500分  
分类：奇物  
本质：特异本质

描述：  
“再去一次缺口”，军官如是说道。仿佛还有人相信这一切都是一部宏大戏剧，充满崇高抱负与伟大英雄。趁着军官走开在墙上刻下又一道标记，士兵又从壶中痛饮一大口。他看见自己死在战壕尽头，感觉自己已经半疯。威士忌缓和了恐惧，却无法阻止泪水顺着他的脸颊流下，在复面的污垢中冲出细小的河流。

一件一战时期的随身扁酒壶，用粗糙的手工在“为了胜利”字样上刻下英国国旗和迷人裸女。尽管它是银质的，但却散发着旧铜的气味，还会在触碰它的手上留下锈斑。  
壶中总是充满深琥珀色的威士忌，陈化极久却带有使人不快的金属味。喝光这壶中的酒使自己饮醉可以使凡人看见幽冥

效果：这视为你获得了【灵视】，但也会因为醉酒获得一点不可豁免的【眩晕】，这是D级自然本质的【毒素】效果，免疫该效果将不可醉酒，当然也不会获得【灵视】。
