# 灰烬账薄

灰烬账薄

名称：灰烬账簿  
价格：1000分  
分类：奇物  
本质：特异

描述：  
Reckoners（清算者）是一个欧洲类似于黑手党的组织，他们交易着人类能想到最珍贵的商品――寿命，这被记在首领杜弗尔所拥有的，昕旦的灰烬账簿上进行储存和售卖，这使得他们的成员在常人眼中类似长生者，但事实并非如此。

昕旦是先于太阳而来之神，平息靖声之神，永葆平衡之神。  
昕旦执掌寿命，灰烬账簿属于昕旦，这使得昕旦对使用灰烬账簿的条件近乎苛刻。

人物在账簿上书写应尽量工整，绝不能出现拼写或者格式错误，否则这一页灰烬账簿将被浪费，先于太阳而来者从不容忍粗心大意.

限制：  
当需要获取灰烬账簿内储存的寿命时，在第XIV小时，也就是中午14时将储存有寿命的单页烧毁，这一过程称为：点燃岁月，随后将烧毁后的灰烬敷到目标身体上，从而获得额外的年岁。  
需要注意的是，灰烬账簿给与的仅仅是自然寿命，不会也不能抵抗任何超凡等级的衰老效果。

“……终于完成了。我对着灰烬账簿仔细地填好了十份汇票，接着烧掉代表十年的那张。每一张新票都价值四千两百万次人类的心跳。我觉得自己甚至能听到它们在纸上颤抖的声音……”

效果：

灰烬账簿从外观上看类似一个支票本，目前仅有5页，持有者每经历一场影片，在影片结束后，灰烬账簿便会增长5页。

每页灰烬账簿最大可以储存1年寿命
