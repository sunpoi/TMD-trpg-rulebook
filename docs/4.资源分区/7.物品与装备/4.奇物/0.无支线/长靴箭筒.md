# 长靴箭筒

长靴箭筒

名称：长鞋箭筒  
价格：750分  
分类：奇物  
本质：特异本质

描述：  
一只华丽的长筒靴子……不过似乎被人改造并附魔加长了鞋筒长度，变成了一个一些人喜欢的箭筒。  
受伤者如果在一些现代医院中可以检测出，敌人是伤口处脚气感染。

效果：  
你可以放入普通箭矢，至多放置15支，射完后才能放置第二批箭矢，同时使用箭矢攻击时会附带1点毒素伤害，视为无支线毒素
