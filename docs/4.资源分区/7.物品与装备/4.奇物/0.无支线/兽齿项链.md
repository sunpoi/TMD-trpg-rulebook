# 兽齿项链

兽齿项链

名称：兽齿项链  
价格：500  
分类：奇物  
本质：特异本质

描述：一串造型奇异的护身符，它们来自各方，形状各异，却在掌心驯顺而乖巧。它们代表的习俗、禁忌与故事，你可以预留更多时间倾听，而这份礼物的意蕴，你现在已然明了。

效果：佩戴者在感知检定上获得1DP的器械加值，在侦查检定上获得1DP的器械加值。
