# 力量封印环

  

力量封印环  
50  
特异本质  
  
【封印之环】  
“力量，并非是强者的唯一证明。你选择让它沉寂，等待时机。这个小巧的环便是你力量的封印。”  
这是一款可以根据使用者意愿定义形态的小型玩具。无论是指环、手镯、腿环，亦或是颈环，封印之环都能够根据使用者的审美和需求自由变换。其设计优雅而精致，表面刻有神秘复杂的符文，闪烁着微弱的光芒。  
  
效果：  
封印之环会强制将使用者的力量属性限制在较低的水平，通常为1-2点，不论原始力量值如何强大。在封印状态下，使用者的身体外观和行为更贴近他们最原始的柔弱状态，尽显优雅和纤细。但若是对方C级别及以上的能力对你的力量属性进行探知，仍然会获得真实的信息。  
  
然而，真正的强大从未被完全封印。只需集中意念花费1点意志，使用者即可以一个自由动作解除封印。当封印解除时，封印之环上的符文迅速亮起，光芒绚丽，瞬间如电流般流动，随后伴随着耀眼的光束，环体炸裂为片片碎片，幻化成无数光点在空中散开，仿佛打破了古老的咒术。虽然效果看似不可逆，但实际上它会在1小时内重新凝聚在使用者的手边，待其再次使用。  
  
当封印被解除，周围的空气仿佛都为此凝滞。环体的裂纹在瞬间扩展，符文被激活，整个封印之环上的纹路将会开始泛起璀璨的光芒，紧接着其会裂解或者爆裂开来，随后似有无穷的能量从中涌出。当然这并不会造成任何伤害或影响，只是这画面对其他人而言，仿佛撕开笼罩在你身上现实的幕布，露出了你潜藏的无穷力量。此刻，使用者的身体感受到解放，力量迅速恢复到其原有状态——一场属于真正强者的觉醒。  
  
画风提示：  
此效果仅为视觉展示，解除封印的动作酷炫而不具危险性，可自由定义光影效果、爆裂时的画面风格、甚至纹路的亮度变化等。  
  
【重塑】  
封印之环并不会因一次解封而永久损毁，1小时后，它会重新回到使用者手边，以完美无瑕的状态恢复如初，等待下一次的封印与解封。
