# 滤镜开关

滤镜开关  
滤镜开关  
奇物  
10  
购买后会在轮回手表中安装滤镜app，可以使用一个自由动作开启，为自己添加滤镜。  
可以只添加一个滤镜，也可以多重滤镜叠加，但请注意，滤镜只能给自己加上，并不能给其他单位添加。  
当然，这除了让你看起来不太一样外，并没有其他的效果。

目前收录滤镜：  
1.硬汉特效  
如果你想让自己看起来十分威武，你可以试试这个特效。开启此特效后，你的脸会变成长方形屁股下巴高低挑眉高鼻梁小眼的硬汉面容，这会给周围的人留下“这个人是个硬汉”的第一印象。

2.猫耳特效  
这个特效可以让你变得非常可爱，开启后你的头部会出现霓虹猫耳和猫胡须，有256种颜色可选。不过开启时你说话的声音会不自主的夹起来，句尾也会不自主的加上“喵”。

3.水晶鞋特效  
开启此特效后，你的脚上会出现闪闪发光的水晶鞋，这能让你成为舞会上的一颗明亮的星。但，由于这个特效的判定有些迷惑，水晶鞋不一定会出现在脚上，可能会出现在大脚趾、笔尖、头顶、弯曲的胳膊肘，甚至全身凸起的部位！  
st选开：可以统计开启此特效的人身上除脚以外的凸起部位，并让其投骰子决定水晶鞋的生成位置。

4.假笑特效  
如果你处于某个尴尬的场景，或是需要笑但笑不出来的时候，可以开启这个特效。开启后你的嘴部会呈现咧嘴露齿的笑容，无论你想不想笑，因此可能会让你的表情看上去像是挺到队友对你说“为了我，对他们使用炎拳吧”的样子。但此特效并没有做动嘴的适配，你在说话的时候嘴是不会跟着动的，请务必注意。

5.西瓜条特效  
在你遭遇或是糟心或是难绷的事情后，你可以使用这个特效向他人诉苦。开启后你的正面会变成一个只有双眼和嘴的长条西瓜，你也可以换成柠檬。同时你说的下一句话必然是“家人们，谁懂啊”。  
虽然这样可以让其他人看不出来你是谁，但只要稍微偏一下角度就会看到你的真身，这个西瓜条只是一张贴图罢了！因为这个原因，就算你开启了这个特效也不能隐藏自己的真面目。

6.JOJO画风特效  
这个特效会让你看起来，连次元都不一样。开启后你的全身会变成写实的漫画风格，身上的阴影全部以排线的形式表现，宛如从漫画中走出来一样。同时你的一举一动都会不自主的凹出能够登上时尚杂志的专业平面模特级动作，让你更加的引人注目。

7.超绝复古特效  
此滤镜适合中老年人及复古情怀爱好者，开启此滤镜后，身旁会出现缓缓盛开的低像素复古风花朵，可以从玫瑰和牡丹中选择。  
同时会在你的身前显示超大字号的醒目行楷字样，固定为“花开富贵”。  
花朵和字样会跟着你移动。

8.重大通知专用特效  
如果你有什么重要的事情需要宣布，那么这个特效是你的不二之选。开启此滤镜后，你接下来说出的每一句话，都会以醒目的超大字幕显示在你的头顶，并且附加超炫的火焰效果，在字幕的两旁会出现旋地踏步的老虎和盘旋的火焰东方龙，同时播放超燃的BGM，让周围的所有人都看得到你的重大发言。
