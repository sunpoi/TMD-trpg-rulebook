# 托尼史塔克的零食

托尼史塔克的零食

名称：托尼史塔克的零食  
价格：500  
分类：奇物  
本质：特异本质

描述：  
在托尼，班纳，罗杰三人讨论接下来的作战方针时，托尼史塔克却漫不经心地掏出一袋零食边吃边工作，甚至还递给班纳博士吃。  
接了零食的班纳博士犹犹豫豫地赞同了托尼史塔克的观点。  
这让人不禁怀疑，他到底是真的赞同，还是吃人嘴短拿人手软。

零食袋中的零食可以以任何其他零食补充。

效果：  
在一次说服检定中你可以掏出这袋零食分享给聊天对象一同边吃边聊，如果对方愿意与你一同食用零食，则使你本次说服检定获得+2dp的表现加值。

每场景可使用一次。

掏出零食边吃边聊是一个自由动作+迅捷动作。
