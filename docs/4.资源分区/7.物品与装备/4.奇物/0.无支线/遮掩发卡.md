# 遮掩发卡

遮掩发卡

名称：遮掩发卡  
价格：10分  
分类：奇物  
本质：特异本质

描述：  
这是一个别在发丝上的金色的发饰，外表像金属制成，但质量较轻。发卡像是一片金色的叶子。  
这个发卡上被附加了可以避开视线的附魔。  
限制：  
如果进行一些引人注目的行动，发卡的效果将在本场景内不生效。例如：主动对他人进行魅惑、提醒他人注意自己的相貌、进行某些会吸引大众的目光的行动。

效果：  
如果佩戴者的相貌值大于2，则在他人眼中你的相貌值只有2点。并且所具有的态度提升的效果、在社交相关检定的加成等类似效果不会生效。
