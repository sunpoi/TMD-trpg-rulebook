# 重力修炼室

重力修炼室  
C+1000  
重力修炼室是龙珠系列的知名产物之一，在其中的房间内可以进行重力修炼，但是也配备了正常生活的生活环境与措施并且还配备专用的空间来进行杂物的堆放，在日常之中会存放在布尔玛特制的空间胶囊之中  
重力修炼室：进去后角色每小时进行一次任意生理属性检定，当累计成功数达到该生理检定的属性等级后，从以下效果选择一项获得。（在进入影片时可以对st进行询问如果副本时间较短可以缩减为半个小时，但是需要st确实）  
1、这一生理属性检定获得1点修行加值，每个生理属性一影片只能获得一次此收益。  
2、获得1点天生防御（锻炼属性为力量/耐力）或者闪避防御（敏捷）。一影片只能获得2次此收益。  
3、三豁免获得1点修行加值。一影片只能获得一次此收益。  
每场影片结束时，若你在影片内完成了所有选项的训练，则获得1点自由属性点，仅能分配在生理属性上。  
空间胶囊：该空间胶囊是专门为重力重力修炼室配备的，重力修炼室为一个四十平米2.5米层高的小房子，但是其中空间也可以为你存储物品使用，切记不要放入过多物品，导致无法锻炼。空间胶囊是一个小的胶囊物品，在使用时扔到地上即可，扔下后会展开成为重力修炼室原本的状态。
