# 五行阵盘

五行阵盘  
价格：C+1500  
分类：奇物  
位置：-  
描述：采用五行为基础理念，设计而成的微型阵盘。虽说是阵盘，其主要的功能是指引使用者破除幻阵。  
效果：使用五行阵盘需要至少占用一只手，持有者将免疫不超过B级【幻象】来源效果的能力；持有者可以使用移动动作并消耗2点意志力展开阵盘，阵盘将在该场景内提供五行效果，但五行效果仅针对持有者生效，且每轮中只有一种效果可以生效（比如你先使用了【木】快速医疗恢复生命值，随后在回合内切换成【金】再进行攻击，则本轮内【金】不会产生效力），切换效果是一个自由动作。  
【金】：持有者使用的武器伤害提升3L。  
【木】：持有者获得快速医疗1，医疗点数12。  
【水】：持有者免疫B级重伤的影响。  
【火】：持有者造成的攻击附带1点额外的严重伤害，类型为火焰能量伤害。  
【土】：持有者获得遁地能力，具有等同自身基础移动速度的固体穿行速度，遁力5。
