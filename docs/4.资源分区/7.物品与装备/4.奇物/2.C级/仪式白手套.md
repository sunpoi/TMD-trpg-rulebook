# 仪式白手套

  

仪式白手套  
价格：C+1000  
分类：奇物（手套）  
描述：王女麾下的将领们时常佩戴洁白的手套，并不只是宫廷礼仪的需求与作为魔法物品的便利性，更有着另一层意味。  
宫廷礼仪：佩戴者获得B级措手不及免疫，在社交相关检定中获得风度/2的器械加值。  
未染的手套：佩戴者可以以迅捷动作发动效果以下两种效果之一，需购买时决定可以使用哪种效果，效果持续一场景。  
·攻击检定获得传奇风度\*2点器械加值。  
·防御上获得传奇风度\*2点偏斜加值。
