# 西尔萨诺的易容帽

西尔萨诺的易容帽  
C+1000  
魔幻本质  
奇物  
占据头盔位

一顶饱经风霜的棕色牛仔帽，来自一个已经消失国度的旅人。

星辰已然归家。

【易容】：帽子上的魔法效果，只要佩戴帽子，你可以完全将外形改变为你曾遭遇过的另一个人，包括对方的五官、声音、性别、衣着等等，可以模拟出对应的性征但不会直接改变性别，肢体的数量不会发生改变，体积可以增加或减少至多1点，这是C级祝福来源的伪装效果。

【旧日余梦】：“我每天都会梦到西尔萨诺，我的家——那里有一片金黄色的向日葵花海和白色的风车，我看见我的爱人就站在蓝天之下，戴着一顶草帽，向我挥手，她的篮子里放着我最爱的苹果派。”

一场景仅一次，当你进行一次专注相关检定时，可以重掷一次，取其中较高的一次进行结算 。
