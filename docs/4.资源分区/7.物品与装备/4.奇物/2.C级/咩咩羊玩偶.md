# 咩咩羊玩偶

  

咩咩羊玩偶  
C+1000  
神话本质  
这是一个书包大小的玩偶，外形是一只有着红色眼睛，红色斗篷，一看就敲凶的萌萌小羊，喂，不要用想RUA一下的眼神看这个玩偶，这可是我们的教主大人！  
拥有这个玩偶的你被授予了咩徒的身份。带上这个玩偶进入影片，每周记得花一个小时对我们伟大的教主大人，的玩偶，进行祈祷，简单来说就是闭上眼睛，双手合十，时常咩咩叫一下，虔诚地想象我们的教主伟大的身姿。没有玩偶的普通咩徒也需要这么做，虽然不会对没有玩偶的教徒带来好处就是了。在你如此做了一影片后，我们伟大的教主会听到你虔诚的声音，授予你高级咩徒的身份。  
在成为高级咩徒之后，你应当履行你的职责，让更多的人成为你的同僚，你可以花费1小时的时间面对面地，而非使用电话电视之类的传播媒体，向凡人讲述咩咩教的伟大，以此劝导他们成为我们伟大的咩咩教的其中一员，需要注意的是，这是一个普通的演讲行为，所以需要对方有倾听你演讲的意愿，且不能中途被什么事打断，也就是说你不可能面对面地对一个正要掏枪杀了你的人演讲一小时，或者在马路上对只是路过懒得听你的演讲的路人进行演讲来劝导他们加入我们伟大的咩咩教。在你讲述完毕后你以风度-表达-专业：演讲对抗对方的意志豁免，成功则代表伟大的咩咩教思想通过你的演讲传递到了对方脑中，对方成为了一名普通咩徒，这是一个D级思维控制效果，若在你成为高级咩徒后，累计为伟大的教主带来了三名新的咩徒，在你回到主神空间后，我们伟大的教主将授予你牧咩的身份。  
成为牧咩之后，你略微获得了教主的一丝神力，你的传教行为升级为C级思维控制效果，若在你成为牧咩后，累计为伟大的教主带来了六名新的咩徒，在你回到主神空间后，我们伟大的教主将授予你主咩的身份。  
成为主咩之后，你再一次获得了教主的神力，你的传教行为升级为B级思维控制效果，若在你成为主咩之后，累计为伟大的教主带来了九名新的咩徒，在你回到主神空间后，我们伟大的教主将授予你大主咩的身份  
成为了大主咩的你将成为教主最得力的打工人（划掉）最信任的部下，我们伟大的教主将给你伟大的神力，也就是，传教行为升级为A级思维控制。还不仅于此，每若你在当前世界的教徒有12人及以上，在你死亡时他们的信仰之力会让你以灵体的形式继续存在，这时你只能进行观察与移动，你没有生命值，而是等同于意志池的结构值。在两到三个月的准备后，当前世界的教徒将举办复活仪式，就是围成一圈在一群羊旁边祈祷，一起咩咩叫，从而让成为灵魂的你重塑肉身。  
关于教义：  
我们伟大的教主对于教条什么的并不在意，只要咩徒能祈祷提供信仰就行了，所以你可以随意制定教条，但咩徒的虔诚取决于你的等级，如果你只是一个高级咩徒，却制定教条让咩徒们去杀人，那咩徒会拒绝，甚至声称你在扭曲教义，号召大家对你取而代之。你能制定怎样的教条而不被反抗，请参考通用特性表中各级思维控制能做到的事情的介绍。
