# 勇气的证明

  

勇气的证明  
价格：C+1000  
分类：奇物（护符）  
描述：鼓舞的是她的勇气，刺下利刃的是她的决心，然而她作为勇者的证明，却仅仅只有一枚塑料的玩具罢了。  
勇气：持有者不会因伤势而昏迷，可以将攻击上的至多决心附加点附加成功转为3DP，这是士气加值。  
决心：获得本物品时，自选一个技能或属性上获得9加骰，对应技能或属性上还会获得5DP士气加值。
