# 玛娜石胸针

玛娜石胸针  
价格：C+1000  
分类：奇物  
位置：饰品  
描述：使用了能积蓄魔力的矿石所制作的胸针，不只是魔术师，也受到为了保持集中的战士所喜爱。  
效果：佩戴者在智力相关检定上获得9加骰；佩戴者选择一项心智类技能，在选择技能的相关检定上获得【传奇沉着】\*1个额外附加成功数。
