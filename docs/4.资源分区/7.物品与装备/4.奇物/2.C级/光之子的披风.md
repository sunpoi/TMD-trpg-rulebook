# 光之子的披风

  

光之子的披风  
C+1500  
魔幻本质  
占据斗篷位  
由光之长老赠送与在光之中诞生的孩子的礼物。  
 这是一个暗黄色的表面如丝绸般顺滑的披风，背面有五个白色的光点，光点有三个等级的表示形态，每点能量会点亮一级。披风会自动附着在使用者的肩膀上，无需任何连接物，平常则犹如一件裹身毛毯，将使用者的全身保护起来，飞行时披风会如鸟翼一般舒展。  
此装备只有在角色为光明生物或持有［光明］属性的装备时才可使用此装备  
   
披风上具有一个特殊的能量池—光之能量，这个能量池的初始上限为5点。这个能量池不会因为获得光之宝石外的其他方式扩增，也不会因为本资源以外的任何能力而恢复。  
光之能量只能用于满足此披风的能量消耗。  
在拥有光之能量时，穿戴者具有机动性普通的飞行能力，但在空中时每轮都会消耗1点光之能量，因飞行而失去的光之能量会在停止飞行后的三轮内逐步返还，以此法恢复的能量不会占用光之能量的恢复上限。特殊的，如果穿戴者位于能够大量散发光芒的生物周围，则飞行不会消耗光之能量。  
在雨天使用披风飞行时，消耗的光之能量提升为初始的三倍，且每轮额外失去1点光之能量，这部分额外消耗的光之能量不会返还。如果角色穿着此披风落入水中，则会立即失去所有光之能量。  
   
穿戴此披风时，穿戴者获得6点神圣防御。若披风内不具有任何光之能量，则此披风提供的神圣防御失效。  
在受到伤害时，可以将披风内的2点光之能量视作1点临时生命来抵挡伤害。  
   
光之能量的恢复方式：在战斗外沐浴在充满光源的地方（阳光下、火堆附近、能散发大量光芒的生物附近等）至少十分钟后，光之能量会以每5分钟1点的速度进行恢复，最多恢复至上限。  
每日最多恢复【15+光之宝石】点光之能量。  
每日二次，穿戴者可花费至少30分钟在充满光源处搜寻光之宝石，进行一个机运检定，如果检定通过则穿戴者获得1枚光之宝石。如果穿戴者在拥有宝石达到上限的情况下进行搜寻，即使机运检定通过也不会获得宝石。  
每一枚光之宝石都会为披风提供2点光之能量的上限，穿戴者至多拥有2枚光之宝石。  
每经过一场影片，穿戴者可拥有的宝石上限提升1枚。  
特殊的，如果使用者在一次攻击中使用此披风抵消超过5点伤害，则会掉落rd3枚光之宝石，掉落位置为你敏感范围内的随机位置，掉落的宝石需在三轮内拾取，否则宝石会自然消散。

支付D+500进行升级：飞行机动性提升至良好。且人物每影片额外获得宝石1枚宝石上限。
