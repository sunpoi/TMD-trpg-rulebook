# 护身石

护身石  
价格：C+1000  
分类：奇物  
位置：-  
描述：使用纯净者骨灰、香料、祷文等素材制作而成的护身石，能够在邪祟的力量中保护你的安全。只从材料而言，倒也算是“纯洁”了。  
效果：携带护身石的单位免疫不超过B级【诅咒】来源的效果；携带护身石的单位在决心相关检定上获得等同自身内在【最高支线等级】\*2点器械加值；携带护身石的单位在承受异常点数时，先减少1点在计算剩余异常点数；携带护身石的单位在机运检定上获得1点加值。
