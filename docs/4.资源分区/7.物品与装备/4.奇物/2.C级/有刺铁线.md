# 有刺铁线

有刺铁线  
价格：C+1000  
分类：奇物  
位置：-  
描述：长满尖刺的铁线，挂载这项武器的活死人攻击，可以让对手造成更深刻的伤害。  
效果：  
该奇物需要绑定一项武器使用。对于绑定的武器而言，在使用其进行攻击时，附带等同自身【传奇力量-1】点韧性伤害，1点剧痛不良点数，不可豁免，可叠加。
