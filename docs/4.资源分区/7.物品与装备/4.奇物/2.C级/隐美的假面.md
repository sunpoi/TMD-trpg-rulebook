# 隐美的假面

  

隐美的假面C+2500  
特异本质  
分类：面具  
隐藏使用者真实容貌的假面，任何人在观察使用者容貌时，均无法看到其面容，这是一个D级命运效果。  
当使用者佩戴时，任何无法看到其面容的人在对抗其攻击时都会降低1点闪避防御或基础防御，并且随着回合增加，每回合会为此数值在增加1点，持续至战斗结束。  
  
魔性之貌  
使用者可以以移动动作摘下面具，如果如此做，会为使用者提供攻击检定的1点附加成功，并且随着回合增加，每回合会为此数值在增加1点，持续至战斗结束。  
使用魔性之貌时，不会取消隐美的假面的效果，但是同样，在重新戴回面具时，隐美的假面效果不会继续提升。  
重新戴回面具需要一个移动动作。  
当施展魔性之貌时，会获得等同风度属性的外貌提升。
