# Pyramid Head【三角头】

名称：Pyramid Head【三角头】  
价格：A+4000  
本质：特异本质  
描述：  
这是生存恐怖游戏《寂静岭》中，代表复仇、审判、惩罚等寓意的怪物，所带着的头盔，其头盔形象南北战争时期的刽子手的三角形的头套，有着锐角、边缘尖锐的三角形的头，暗喻着带来的痛苦。

Pyramid Head头盔实际上是一个底边为六边形的铁头盔，后面的底边极短，而前面的底边比较长，；从侧面来看的话，后方的边与前方的边形成大约140°左右的夹角，穿戴者的后脑被完全暴露，不过这本身并非用于作战的头盔，因此也并没有什么阻碍。  
这一巨大的三角形头盔虽然完全封闭，但并不会阻碍穿戴者的视野，戴上它会让其他人完全看不清你的表情，在这一状态下，你无法与他人交流，虽然听得见话语，也能理解，但话语从你口中出现的一刹那，就会变成无意义的混沌呢喃，若是有人以精神链接你，也只能感受到审判，痛苦，惩罚等不具有逻辑性的概念词汇。

【恐怖降临】：你的存在便足以带来强烈的压迫感。  
当你进行审判时，人物可以在召唤出如同数量巨大的甲虫海，庞大的虫海在墙壁，地面，天花板肆意游动，涌向你所指定的一个或者复数个目标，这些锋利漆黑的甲虫避开目标坚硬的外壳，直接钻入受害者的口鼻之中，撕咬血肉，吞噬内脏。

分类：头盔  
特性：【永恒审判】【恐怖降临】  
效果：  
【永恒审判】：  
你可以自由动作对一个可以被你定位的目标进行标记，此后该目标将成为你的审判、复仇、惩罚对象，只有目标接受了审判并且不再需要惩罚时（这一般意味着目标的死亡），你才可以停手，在此之前，你无法死去，也无法摘下头盔.这是A级命运来源的【复活】效果。  
被标记的审判目标同时只能存在一个，这个标记不会给你目标或你带来实际效果，但当你死亡时，若审判标记依然存在，你则不会死去，而是化作不具有除观察移动外动作的灵体，你将继续追踪跟随目标，直到你复生的那一天，这一过程中，你只能靠你本身自然恢复进行恢复，直到到生命全部恢复完好重返现世，这个时间最迟会为一周，一周后将完全复活。灵体状态下，你的结构值等同于你的沉着属性的数值，若有人可以在你灵体状态下将你击杀，你则会彻底死去。

【恐怖降临】:你可以以一个迅捷动作进行一次审判，审判将持续一个场景。  
此时，你敏感范围内的一切死物都会变得腐朽破败，而空气则变得闷热，空间内响起不知名的警笛声，火焰顺着墙角屋檐开始缓慢的燃烧，数不清的甲虫在这片空间司仪游走，你的身躯看起来这更加高大具有压迫感，这一状态下，你可以维持两个和你相同外形的幻象与你一同攻击目标，从而给与敌人你无所不在的错觉，但幻象只是幻象，虽然看上去闻上去甚至摸上去都与真身一模一样，但一旦攻击或者被攻击，其幻象的的本质就会暴露无遗，这是A级【幻象】来源的效果，恐惧降临带来的环境改变无法适用于任何能力的前提。

在恐惧降临时，审判者视为具有【接触攻击】特性；人物可以操控甲虫攻击目标，就如同人物本身进行了一次攻击一样，将消耗相对应的动作，你所能操纵的甲虫海不得超过你20米远，判定上就如同你获得了20米攻击触及距离一样。  
人物也可以操控甲虫，每轮一次，自由动作对20米内的一个或者复数个目标进行一次擒抱，正常对抗，从画面上看，则是无数漆黑锋利的甲虫，迈动肢足，源源不断的堆积攀爬在受害者身上，限制其活动。  
特殊的是，甲虫依然是A级【幻象】来源的效果，但即使受害者可以免疫幻象，但依然会受到攻击和擒抱所产生的相应后果。
