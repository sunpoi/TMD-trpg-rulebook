# 竹蜻蜓

名称：竹蜻蜓  
类型：奇物  
价格：D+750  
本质：科幻  
描述：大家最初对飞行的幻象，大概都来源于这个放在头上就可以自由飞行的竹蜻蜓吧。最初的梦想，最初的梦。ps：按科技来说光在头上施加一个力必然头单独飞了，理论上应该是特异但蓝胖子就该科技所以本质仍然是科技。  
效果：将它带着脑袋上吧，然后童年的梦就实现了，你可以在天上飞行了，但竹蜻蜓的飞行难度十分之高，初次接触的你只能进行笨拙的飞行，每个世界一次你以竹蜻蜓飞行过都可以更习惯飞行，这使得你飞行的机动性提升一级，最多提升为完美。
