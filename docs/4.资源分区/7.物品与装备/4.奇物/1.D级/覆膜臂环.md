# 覆膜臂环

覆膜臂环  
价格：D+750  
分类：奇物  
位置：手腕  
描述：在身体张开对魔法防护膜的手镯，研究未知魔术时炼金术师佩戴以防止事故。  
效果：装备者获得3点力场防御，并具有1点基于力场防御的【全伤害吸收】；装备者免疫不超过D级魔力驱动的效果影响。
