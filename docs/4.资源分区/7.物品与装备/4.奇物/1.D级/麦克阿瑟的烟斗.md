# 麦克阿瑟的烟斗

  

麦克阿瑟的烟斗  
  
  
D+500  
特异本质  
  
这是一个样式复古的木质烟斗，烟斗上有着长年被人使用留下的痕迹。  
烟斗边缘的烟熏色隐隐发青，令人难以猜测这到底是烟草留下的颜色还是硝烟所致。  
  
使用者可以以一个迅捷动作掏出烟斗并叼在嘴上，若使用者如此做了，那么接下来只要烟斗没有离开嘴边，他所有的风度相关检定+2dp，这是器械加值。  
当使用者将嘴上的烟斗拿下来攥在手里，并以“如果当初在XXX的时候，是你来做我的对手的话，那么我将。。。”为开头进行社交检定时，社交检定+2dp，这是器械加值。  
  
理所当然的，将烟斗取下和将烟斗叼在嘴上时不可能同时获得两种加值。  
  
NPC并不会在意你开头时所说的那段经历的真伪，对方只会觉得你是一个有身份有阅历的人，你的话有点道理。  
  
如果在获得了来自烟斗的加值后社交检定成功，那么对方对你的好感度会提升一级，但不会提升至极端。  
  
若社交检定仍旧失败，对方也不会对你产生额外的恶感。  
  
若你将该烟斗转赠给别人，只要对方不对你抱有敌意，那么他将立刻拿起烟斗开始夸你，这是一个D级的思维控制效果。
