# 电岩原石

电岩原石  
价格：D+1000  
分类：奇物  
位置：饰品  
描述：工房用于魔法武器的魔法石原石。直接用也能产生对特定魔法的耐性  
效果：持有者在攻击时，攻击伤害类型转为雷电能量伤害，并附带1点不可豁免的麻痹不良点数，具有【雷电能量抗力3】。
