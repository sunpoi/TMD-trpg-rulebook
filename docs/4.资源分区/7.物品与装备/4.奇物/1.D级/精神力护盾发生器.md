# 精神力护盾发生器

  

精神力护盾发生器  
D+1500  
科技本质  
占据头盔位  
标准的精神力护盾插件，在只有指甲盖大小的精神力发生器上连接着数条神经索，这种允许精神力流通的神经索在植入后，将与受术者的脊椎神经接驳，随后受术者可通过意识控制精神力输出，结合精神力发生器产生球形晶格护盾，从而保护使用者。  
植入精神力护盾发生器插件，需要一个DC3的延长医学检定，单位为10分钟。  
这种发生器由无数金属粒子尺寸的计算机芯片浇筑，可以成倍的放大精神力对现实的扰动效应，其理论扰动值可达原本人物实际输出值得1000万倍以上，其实际成型效果与受精神力者本身能力呈现正相关，越是强大的精神力者，所产生的球形晶格护盾便越是厚重，并伴随显眼的的发光现象。  
  
效果：一轮一次，应对某次攻击时，可以自由动作消耗1点精神力，进行一次【决心-专注（意志）】的精神力检定，人物在本轮获得该次检定的成功数一半（取低）的偏斜防御，本轮所取得的偏斜防御越多，随意志所产生的，包裹人物的球形晶格护盾便愈加的明显。若在本轮中该精神力发生器产生球形晶格护盾被完全击穿，则护盾会在清脆的破裂声中散落一地，直到护盾重新生成前，人物都不再享有护盾所提供的偏斜防御加值。  
  
\*足量的破魔，获类似效果可以击穿晶格护盾，使得晶格护盾失效，但无视偏斜防御的能力则不可，它们只是简单的传过去，具体由ST决定。
