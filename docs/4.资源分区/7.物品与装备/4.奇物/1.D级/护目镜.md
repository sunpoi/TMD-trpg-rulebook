# 护目镜

护目镜  
价格：D+500  
分类：奇物  
位置：头部  
描述：朝着前方逐渐变得像鸟嘴一样尖锐的护目镜，有着保持良好视线的作用。  
效果：佩戴者免疫目眩不良点数带来的减值与对应严重后果；佩戴者具有【黑暗视觉】，并获得2点洞察防御与1个基于洞察防御的防御附加成功数。
