# 辉耀火把

辉耀火把  
价格：D+1500  
分类：奇物  
位置：手持  
描述：有着火之玛娜力量的明亮火把。非常耀眼，会妨碍逃走。  
效果：持有者附近30米范围内，所有单位在处理亮度等级时，均视为处于原场景亮度提升2级处理，但在潜藏相关检定时会额外承受2DP器械减值。
