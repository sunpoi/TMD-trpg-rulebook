# “砰砰”牌香槟

  

“砰砰”牌香槟  
分类：奇物  
价格：D+500  
描述：一簇纯金色的佳酿，老少皆宜的香槟酒，口感略显温和。开启木塞时会发出清脆的声响，那是孩子们在新年最期待的时刻。  
效果：这瓶酒神奇的永远会保留1/3的酒液，而且会在次日0点补充至满。  
持有者可以使用一个自由动作饮下1/3，在接下来的24小时内获得机运检定+2DP的效果，第二次饮用获得全检定的1DP加值，最后1/3的酒液不会提供任何效果。
