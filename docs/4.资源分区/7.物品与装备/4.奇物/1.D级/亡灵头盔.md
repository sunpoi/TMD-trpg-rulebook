# 亡灵头盔

亡灵头盔  
价格：D+500  
分类：奇物  
位置：头部  
描述：据说能消除佩戴者存在感的防具。因扭曲周围认知的效果的副作用、身体循环能力降低。  
效果：佩戴者获得4点盔甲防御；佩戴者获得D级魔幻本质的反侦测能力，但具有的常驻快速医疗下降1点，相应降低12点医疗池（若没有，则改为下降1点生命值上限）。
