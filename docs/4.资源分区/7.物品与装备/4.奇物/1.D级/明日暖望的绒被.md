# 明日暖望的绒被

明日暖望的绒被

价格：D+500

类型：奇物（织物类）

\---

外观描述：

一床看似寻常却充满秘密的儿童绒被，约1.5米×1米大小，被面是深蓝色的夜空底色，上面用银线绣着打瞌睡的雪人、打着哈欠的驯鹿、抱着礼物盒眼皮打架的圣诞精灵。被角缝着一小块磨损的羊毛标签，上面用稚嫩的字迹绣着：“等太阳叫醒我，明天就到了。”

被面上的图案并非静止——当被子被完全展开，那些绣线会泛起极微弱的柔光，雪人的胡萝卜鼻子会随着呼吸般的节奏轻轻起伏，驯鹿的眼睫会缓慢眨动，就像所有生灵都在这床被子里安稳入梦，等待黎明的到来。

被芯填充着某种永远保持恰到好处温暖的未知绒毛，触感如同被云朵拥抱。将脸埋进被子里，能闻到淡淡的甜饼干、热可可和松枝的温暖香气。

\---

背景故事：

这床绒被并非由魔法师或工匠刻意制作，而是在某个平安夜里自然“凝结”最后由圣诞老人亲手编织的奇物。

在每一个圣诞前夜，那些孩子们总是会因为过于期待明天的礼物与团聚，翻来覆去无法入睡。在床边，母亲不厌其烦地对孩子一遍遍安抚，最后为他盖好被子，轻声说：“闭上眼睛，好好睡一觉，明天就会在你醒来的时候准时到来。”

孩子们抓紧被角，怀着最为纯粹的期待，在午夜钟声响起前沉入了梦乡。

那个夜晚，整座城市里无数孩子相似的期待——对礼物、对团聚、对奇迹的温暖盼望——悄然汇聚，在守护中、在暖意里、在圣诞树的光影交织下，每一位孩子的梦境中都悄然出现了一位圣诞老人，他给予他们快乐与安心，圣诞老人也将这些美好的愿景织成了最为温暖的绒被

从此，它拥有了给予“安眠”与“期待”的小小力量，而现在它会出现在那些需要它的旅人手中，给予他们安心的睡眠

\---

特性：【明日之安的沉眠】【晨曦的许诺】【礼物与悄悄话】

【明日之安的沉眠】  
孩子们怀着期待入睡，它们在梦中肆意地畅聊着明日之后的惊喜，悄无声息地微笑浮现在它们熟睡的面庞，稚嫩而又纯真的盼望从他们的心间流出

当你盖着绒被入睡时，将会获得一个名为<深度睡眠>的特殊状态。

该状态下：

使用者的身下将会化为最为舒适的床铺，周围的吵闹和繁杂将会无法打扰，哪怕只是坐立的状态，也能够感受到不输于卧躺的体验，这视为一个自适应长休的能力。  
与此同时使用者的身体会进入到一种独特的高效恢复状态，平常需要8个小时才能完成的长休，如今只需要一半的时间。

注：虽然该绒被现实的尺寸为1.5米×1米大小，但是仍然会完全的盖住使用者的身体，不让使用者受风着凉。

【晨曦的许诺】  
当你从梦中醒来，温暖的绒被让你舍不得起床，但是清晨温暖的目光已经照在了你的眼角，揉揉眼睛，迎接你早已期待的惊喜。

当使用者从绒被中醒来后（即长休完），使用者醒来的第一次感知相关检定将会获得1点无名加值。

【礼物与悄悄话】  
也许你已经不再是那个对未来充满着期待的孩童，但是圣诞老人仍然给予了你一份专属于你的礼物，并且悄悄地带上了一些……悄悄话。

当角色经历过一次圣诞节之后，将会永久获得1点机运骰加值，该加值仅能获得一次
