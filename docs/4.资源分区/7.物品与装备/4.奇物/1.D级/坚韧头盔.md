# 坚韧头盔

坚韧头盔  
价格：D+500  
分类：奇物  
位置：头部  
描述：以魔力交换所得到坚韧的头盔，据说由绝对不会倒下的巨汉战士所戴。  
效果：佩戴者不会因被严重伤害填满生命值而昏迷，这一效果生效时会每轮消耗1点魔力。当你的生命值没有完好时，获得3点盔甲防御。
