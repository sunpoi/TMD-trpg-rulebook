# 湖蓝的小纸条

湖蓝的小纸条  
D+500  
魔幻本质  
  
分类：饰品 原典：地城世界  
描述：这是一张充满魔力的小纸条，上面的字迹还很清晰，这张纸条是一个名为埃赞托的世界中的旅法师湖蓝留下的，旅法师湖蓝一生都在游历整个大陆，她每到一个地方都会留一些纸条给那些需要帮助的人，上面分别写满了她的问候、祝愿、建议和经验。大陆上很多人都有收到过这位法师留下的纸条，但奇怪的是关于这位传奇法师到底是怎样的人却众说纷纭，有人说她是一位年轻的少女，有人说她是一位年迈的老奶奶，甚至还有人说她是一位皮克精女王......不过自从之前一个德拉斯堡的哨兵说看到过她的身影之后，就再也没有人见过这位传说中的法师了，人们纷纷猜测这位传奇的法师到底去了哪里，是隐居了？是去世了？还是已经旅行到另一个位面去了？不过总而言之，她应该确实是离我们远去了，唯一留给人们的就是这些小纸条，这些纸条至今仍然散发着一些魔力，这让人们相信说不定哪天湖蓝又会再次出现在大家面前了。  
特性：【湖蓝的小小赠礼】  
【湖蓝的小小赠礼】：购买该奇物时选择一个专业，那个专业相关检定上获得智力附加\*2点表现加值。  
随时间推移，该奇物就能积蓄魔力，购买该奇物并经历一场影片后，该奇物将额外为携带者提供在进行法术检定的2点器械加值，该奇物最多提供三次成长。
