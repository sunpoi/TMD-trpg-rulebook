# 大脑针织帽

大脑针织帽  
奇物  
占据头盔位  
D+500  
一顶丑丑的帽子，外表跟人类大脑一样，但它只是一顶粉色的针织帽。

只要戴上它，你感到脑袋暖暖的……  
你因此获得了大脑的力量！意志力上限+1，智力检定获得3dp增强加值，你全身暖暖的，免疫D级环境来源的负面效果。
