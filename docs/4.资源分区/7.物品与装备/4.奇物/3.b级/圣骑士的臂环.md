# 圣骑士的臂环

圣骑士臂环  
价格：B+2000  
分类：奇物  
位置：手腕  
描述：首次受到「祝福」、被称为「圣骑士」的豪杰在大崩溃之前代代相传的「神圣」手镯。  
效果：  
【坚固意志】：佩戴者只要进行攻击就能维持自身的坚固的防御，这意味着佩戴者在进行攻击后，视为已经进行了一次基础规则中的格挡行为。

【神圣恩典】：在进入战斗后，在自身每轮结束时获得2点神圣防御以及1点基于神圣防御（无名防御）的防御附加成功数，至多获得沉着属性次本效果增益，场景切换后因此获得的效果全部清空。
