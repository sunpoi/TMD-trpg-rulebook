# 血色披风

名称：血色披风  
类别：奇物（斗篷）  
本质：魔幻本质  
价格：B+2000  
描述：  
血色披风是出自虚天殿的藏宝阁的上古法宝，其造型奇特之极，竟有两层构成。里面是银色丝线编制而成，外面则是一根根奇异的不知名羽毛飞贴上面。这些羽毛在光罩中变幻出暗红的血光，显得诡异之极。在注入法力后可化作猩红血光，可成倍的提高遁速，虽然操控不甚灵活，但其提供的遁速使得金丹期修士穿戴后，拥有不下于元婴期修士的逃遁速度。

效果：  
【被动】：穿戴后获得机动性为完美的飞行能力，飞行速度等同于穿戴者的基础速度。

【主动】：每轮一次，穿戴者注入5点东方能量（真气/内力/气/炁）,此刻，披风血色的光芒大盛起来，似被一股火热的感觉包裹，披风外层的数十根羽毛同时幻化出根根的血色长翎，整件披风都被一团浓浓的血光罩在了其中。穿戴者立刻以自由动作（不是你的回合则是反射动作）进行一次移动，该次移动的最大距离为：你的基础速度\*2倍+50米。但本次移动的机动性下降为笨拙，此时，人物在感知属性上每有1点附加成功，则该次移动的机动性上升一级（笨拙-普通-灵活-完美），最高为完美。
