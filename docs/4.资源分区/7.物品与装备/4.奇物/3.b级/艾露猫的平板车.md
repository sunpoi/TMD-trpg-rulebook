# 艾露猫的平板车

  

艾露猫的平板车（猫车）  
B+6000  
魔幻本质  
猫车是一个出自游戏《怪物猎人》的游戏用语，是指在进行游戏的狩猎时被怪物打死，然后就会被一(两)只猫推着车送回基地，同时会减少狩猎获得的报酬金。猎人力竭倒下被艾露猫用平板车推回营地然后扔到地上这一事件。  
进入影片后可以选择一个地点，当你死亡时，会有两只艾露猫趁着参与战斗单位的松懈时机推着一辆破烂颠簸的平板车把你推走，并让你强行脱离战斗，你被艾露猫推走时视为处于无助状态，直到你回到目标地点前你无法进行任何动作，艾露猫会在一周内将你推回指定地点。在此期间，你每日恢复上限1/6的生命值与意志值，当你的生命状态恢复至满值后你可下车并正常行动。你仍然有可能会在推车期间被攻击。 这视为一个B级影响心灵来源的复活效果。  
到达地点时，艾露猫也许会向你索要一些无关紧要的物品（ST可以选择是否生效）。若你将物品交给艾露猫后，艾露猫会给你提供一颗奇怪的珠子，持有时这颗珠子时，攻击检定获得自己最高内在强化支线等级的器械加值（D1C2B3A4S5）。  
奇怪的珠子最多同时存在两颗，且在影片结束后被艾露猫收回。若你拒绝归还，则艾露猫会拒绝给你推车，直到你归还奇怪的珠子。  
一场影片内艾露猫最多只能将你从死亡中抢救回来两次。
