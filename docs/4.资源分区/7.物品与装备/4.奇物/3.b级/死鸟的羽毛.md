# 死鸟的羽毛

名称：死鸟的羽毛  
类别：奇物（饰品）  
本质：魔幻本质  
价格：B+4000  
描述：  
得自被说是地狱之鸟的魔物的羽毛法器。许多猎人为了得到完整形状的羽毛死了。  
效果：  
持有者免疫不超过S级即死来源的效果影响，并获得【威仪】；当持有者进行攻击时，被命中的目标需要使用意志对抗攻击出目，失败则承受差值\*1点恐惧不良点数，标准豁免，这是一个B级诅咒来源的效果。
