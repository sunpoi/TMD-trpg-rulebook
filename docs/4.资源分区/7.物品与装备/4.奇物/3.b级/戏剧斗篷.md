# 戏剧斗篷

戏剧斗篷  
价格：B+3000  
分类：奇物  
位置：斗篷  
描述：为了让舞台表演的人动作轻快的长袍。性能在战斗中也能发挥、在拍卖中以高价交易。  
效果：  
【轻便移动】：装备者在反射豁免上获得【8加骰】，并在基础移动速度上提升等同【敏捷属性】\*3米 。

【迅敏身手】：装备者在敏捷相关检定上获得等同【敏捷属性/2】点器械加值，并获得等量的闪避防御。
