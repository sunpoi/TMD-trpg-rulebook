# 塞壬的臂环

名称：塞壬的臂环  
类别：奇物（手腕）  
本质：魔幻本质  
价格：B+2000  
特性：【异界加护】【尘之心】  
描述：某位女塞壬的祭具、受到尘和次元扭曲的影响变成了暧昧的存在。已经不是此世之物了。  
效果：  
【异界加护】：佩戴者免疫不超过A级时空来源的效果，同时获得1点力场防御与基于力场防御的1个防御附加。

【尘之心】：佩戴者免疫【为血痴狂】导致的无法使用意志力与相应等级招式的效果，但依旧可以正常承受相应的正面作用。
