# 血抑之徽

血抑之徽  
B+2000  
护符  
奇物  
这枚护符由吸血鬼伯爵用自身精血铸造，最初用于镇压血脉中的嗜血冲动。后被某位叛逃的血族刺客改造，融入“血债血偿”的黑暗契约逻辑。传说佩戴者若过度依赖其力量，最终会沦为护符的傀儡，血肉与灵魂皆被吞噬。

【抑制作用】你佩戴该护符，你便能掩盖自身黑暗，如果你是黑暗生物，那么你佩戴时不视为黑暗生物。

【血偿契约】当你收到伤害高于自身生命值一半且攻击者处于你自身20米范围内，若你此时的盔甲防御未被完全破除，则立即吸取攻击者的血液，对方必须过一个dc16的强韧豁免以阻止血液流出，失败则会受到5点亵渎伤害，  
每造成1点伤害，则获得1点快疗（医疗点12），以此最多获得4点快疗，这些快疗不会叠加，持续时间为一个场景或被吸取目标的死亡。
