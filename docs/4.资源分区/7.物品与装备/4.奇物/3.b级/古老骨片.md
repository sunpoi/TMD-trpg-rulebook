# 古老骨片

名称：古老骨片  
类别：奇物（护符）  
本质：特异本质  
价格：B+2000  
使用限制：  
特性：【若你踌躇不前】【若是心存迷惘】【即便身躯死亡】

描述：  
指甲盖大小的一枚骨片，似乎是从某人的尸骸中剥落，漫长的岁月已经让他风干，却丝毫没有腐朽。  
也许你认为是自己选择购买这枚骨片，实际上是这枚骨片选择了你，他才会到达你的手中，遗骸的主人决定注视你，只是你尚不知晓。  
【若你踌躇不前】：  
当你陷入困惑，不知前路在何方，ST可以让一个头戴荆棘王冠，双手穿洞的身影浮现在你的身侧向你低语，这道身影只有你能看见，这份低语只有你能听到，在你看清这个身影之前这个身影就会消失。

【若是心存迷惘】：  
在困境之下，一份低语便会萦绕在你的耳边：”若心存迷惘，便不要击发“，这是第一次吟唱。  
这份低语会再吟唱一次，再一次……  
低语最后一次吟唱的时候，低语会说：”若心存迷惘，便不要击发，要有所成长……再次，那样的话，通往新道路的大门将会打开“  
你的眼中将隐隐浮现黑色的火焰。

【即便身躯死亡】：  
即便身躯死亡，也能获得永生。那道身影如此低语，在你死亡时赐予名为永恒的恩赐。就像每个圣徒一样，你的尸骸将永不腐朽。

效果：  
【若你踌躇不前】：  
身影的低语将会告知你一个大致的方向或者地点名词，那里也许就是你需要前往的方向，并且时间上的误差不会超过一周。这是一个B级命运来源的预知效果。

【若是心存迷惘】：  
一场景一次，当一场战斗中有一名友方角色失去战斗能力，且你失去了所有完好或失去了一半结构值时，低语会第一次吟唱。  
每当经过战斗轮次中任意一个单位的回合，这份低语就会再吟唱一次，合计吟唱三次。  
吟唱三次后你的第一个回合，这份低语会进行最后一次吟唱。  
吟唱后你的下一次攻击伤害增加一倍，若对方具有反震、折射、反伤、打断性反击等会让你因此次攻击受伤的能力，则在此次攻击时，此次攻击额外获得1附加成功。

【即便身躯死亡】：  
你的遗体不会损坏。
