# 异国护手

异国护手  
价格：B+3500  
分类：奇物  
位置：手腕  
描述：本来是异国重装士兵装备、但因为使用方便、在潜伏者之间也使用。实战中能提高各种各样的能力。  
效果：佩戴者承受的盔甲减值减少3点；佩戴者获得等同【敏捷属性/2】点闪避防御，与等同【耐力属性/2】点的盔甲防御与生命值上限提升。
