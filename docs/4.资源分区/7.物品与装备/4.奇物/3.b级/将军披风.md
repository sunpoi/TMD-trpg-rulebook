# 将军披风

将军披风  
分类：奇物（斗篷）  
价格：B+2000  
描述：王国将军标准配置的披风，男性将领与女性将领的款式有细微区别。这件私人订制的款式采用了瑰丽的玫瑰红色，以魔纹工艺精心处理，提供了许多便利之处。  
从容不迫：穿戴者获得A级免疫措手不及能力。  
魔纹工艺：以一个自由动作，向魔纹中注入能量，形成防护力场。  
最多10点能量，每注入1点获得1点偏斜防御，但每3点就会有1点不生效，效果持续一场景。  
只要注入了能量，这件披风就会为穿戴者提供传奇风度\*2点力场防御与传奇风度\*5点三豁免器械加值。
