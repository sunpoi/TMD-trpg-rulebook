# 拒绝死亡的项链

名称：拒绝死亡的项链  
类别：奇物（饰品位）  
本质：特异本质  
价格：B+3000  
描述：  
通过抑制心灵的高昂、能够回避死亡的纯银项链。某位冒险者从「母亲」处得到、但之后抛弃了。  
效果：  
每七日一次，当一次攻击会将你杀死时，你将免疫那次伤害，这是一个B级命运来源的【无敌】效果。
