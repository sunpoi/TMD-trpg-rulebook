# 真实的眼镜

名称：真实的眼镜  
类别：奇物（头部）  
本质：特异本质  
价格：B+3000  
描述：  
提高佩戴者的集中力、让人决不会被幻觉迷惑的眼镜。据说古代的炼金术师们追求洞察力、在研究时使用。  
效果：  
佩戴者免疫不超过A级幻象来源效果的影响，并在感知相关检定上获得等同【感知属性】一半的加值、加骰提升1级（上限8加骰）的效果；若佩戴者不具有灵感视觉，则获得灵感视觉。
