# 普罗晏比娜的吸尘器

名称：普罗晏比娜的吸尘器  
价格：C+1000  
本质：魔幻本质  
武器分类：法器  
法术强度：3L  
武器体积：2  
描述：整体莹白色的吸尘器，但上面沾染着暗红色的不明污渍

【法术浸润】  
在经历了不知名物质的侵袭后，这把吸尘器获得了神秘的力量。

【生命汲取】  
在你对他人发起攻击后，吸尘器亮起了微弱的光芒，

备注：普罗晏比娜是恶魔城中侍从伯爵的女仆，除了在遭遇入侵者时用跆拳道黑带的身手予以驱逐外，她们平日里最常做的工作就是打扫城堡。普罗晏比娜的吸尘器可以有效地吸收地上或墙上的血液，这和伯爵大人喜欢摔杯子的习惯有着密不可分的联系。每当伯爵看到一名贝尔蒙多家的访客时，他总是喜欢将手中装满血的高脚杯扔到地上，然后再上前和对方来一场基情对削。数百年来一直如此，意义不明。

特性：【法术浸润】【生命汲取】  
效果：  
【法术浸润】：  
这视为一把法杖，以操控-器用-施法进行检定，同时自带3点法强。

【生命汲取】：  
你可以用一个标准动作将其依靠在一个存活的目标旁，将吸尘器打开，这视为你使用了一个攻击技，此时你获得吸血1，这表现为吸尘器缓缓发动，同时被吸取者的皮肤开始干瘪，如同生命流失一般，正常造成伤害。同时，对方需以意志豁免进行标准对抗，并会吸走对方等同于差值的能量，若如果对方没有能量或者已经耗光能量，则直接失去每2点本应消耗能量值数额的1点的意志值。
