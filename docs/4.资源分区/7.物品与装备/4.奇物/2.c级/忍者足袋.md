# 忍者足袋

奇物名称：忍者足袋  
资源分类：奇物  
本质：特异本质  
价格：C+1000

描述：  
异国使用的消除脚步声的鞋。提高各种各样关于动作的能力。

效果：  
占据鞋袜位。  
穿戴者在敏捷相关检定上获得3DP器械加值，基础移动速度提升等同敏捷属性\*1米；每轮一次，当穿戴者进行基础规则中的招架行为时，视为移出了目标攻击范围一样处理。
