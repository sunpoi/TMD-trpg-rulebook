# 拳岩

奇物名称：拳岩  
资源分类：奇物  
本质：魔幻本质  
价格：C+2000

描述：  
很硬。在坑道中发现的神秘古老法器，将各种各样的能力都舍弃掉，竭力战斗。

特性：【历战】【打岩】

效果：  
占据手腕位置。

【历战】：  
装备者的天生武器伤害提升等同【生理传奇属性之和】\*1L。

【打岩】：  
装备者可以消费一个自由动作并支付1点意志力，将意志贯穿于身体之中，使自身在该场景内天生武器获得【破甲10】，具有【贯穿】特性。
