# 七杀碑

奇物名称：七杀碑  
资源分类：奇物  
本质：特异本质  
价格：C+1500

描述：  
这是一个使用中文写着以下内容的碑  
天生万物以养人  
人无一物以报天  
杀杀杀杀杀杀杀

效果：  
购买本奇物后他会与一座神秘的人形雕像一起出现在购买者的主神空间中，当你购买后在影片中每经历一场同级别战斗并斩杀至少一位敌人后收集其头颅或其他标志性物品在影片结束后将其炮制筑成京观，你将会1层【杀心】，每层【杀心】会使你获得1点在攻击上的增强加值，你在每影片中至多可以获得3层【杀心】。  
你即使带回更多的头颅也至多获得3层【杀心】，同时你所筑的京观使你恶名远扬，你在影片结束时获得的美名声望将永久性转为等额的恶名。  
只要你持有这座碑，你的攻击就带有会让敌人难以恢复伤势的效果，这是一个创伤来源的B级【重伤】效果。
