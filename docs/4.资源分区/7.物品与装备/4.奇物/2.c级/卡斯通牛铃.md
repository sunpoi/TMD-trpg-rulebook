# 卡斯通牛铃

名称：卡斯通牛铃  
价格：C+1000  
本质：魔幻  
分类：护符  
描述：  
这是一只体积1的方形牛铃，附带一条绑定的可以自动适应穿戴者脖围的项圈，舒适透气结实耐用，可自定义外观，美观实用。  
米诺陶洛斯一族长久以来的问题便是在于，雄性极其易怒，而雌性极其冷淡，导致这些身强体壮的牛族人在世界当中的人数占比不断下降，就在这时，米诺陶洛斯一族中出了一位，名为卡斯通·我爱喝牛奶·安格斯的天才科学家，通过巧妙的音波疗法和稳定且受米诺陶洛斯一族喜爱的黄铜质感研发出了一种声音清脆动耳的牛铃饰品，受此影响，米诺陶洛斯一族的人们，从此不再躁动、狂暴，不再冷漠、排外，拥有了融入现代社会的能力，并逐渐受到各族的喜爱。

【安抚之铃】：无论该护符特性是否生效，它都可以像正常的牛铃般发出清脆悦耳的叮叮声。

特性：【安抚之铃】  
效果：  
【安抚之铃】：佩戴该护符进行的所有隐匿检定均受到10dp器械减值，检定结果投两次取劣结算。  
该特性的生效前提是满足以下条件：长角、类人型，且体积达到10或性冷淡（取一）。  
若效果生效条件满足，佩戴者若有类似阳痿、肾虚、肾透支、早X（过度敏感）、久X不泄、无感等男科、妇科疑难杂症，只要承受的疾病来源不高于C级疾病来源，都会因佩带此物而痊愈；并且佩带后体力都有明显上升，这会为佩带者带来1+传奇耐力\*1点生命值上限提升。  
佩戴者具有1点异常点治疗（医疗点12）
