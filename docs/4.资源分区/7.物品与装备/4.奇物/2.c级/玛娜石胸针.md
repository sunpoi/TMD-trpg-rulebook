# 玛娜石胸针

奇物名称：玛娜石胸针  
资源分类：奇物  
本质：魔幻本质  
价格：C+1000

描述：  
使用了能积蓄魔力的矿石所制作的胸针，不只是魔术师，也受到为了保持集中的战士所喜爱。

效果：  
占据饰品位。  
佩戴者在智力相关检定上获得9加骰；佩戴者选择一项心智类技能，在选择技能的相关检定上获得【传奇沉着】\*1个额外附加成功数。
