# 勇气的证明

奇物名称：勇气的证明  
资源分类：奇物  
本质：特异本质  
价格：C+1000

描述：  
鼓舞的是她的勇气，刺下利刃的是她的决心，然而她作为勇者的证明，却仅仅只有一枚塑料的玩具罢了。

特性：【勇气】【决心】

效果：  
占据护符位。

【勇气】：  
持有者不会因伤势而昏迷，可以将攻击上的至多决心附加点附加成功转为3DP，这是士气加值。

【决心】：  
获得本物品时，自选一个技能或属性上获得9加骰，对应技能或属性上还会获得5DP士气加值。
