# 骥驹过膝

名称：骥驹过膝  
本质：魔幻  
分类：奇物  
价格：C+1250  
占据位置：鞋袜  
描述：一双靴底为马蹄的过膝长靴，据说是某位道士在炼制黑驴蹄时错把马蹄当成驴蹄而诞生的产物。穿上这双靴子后，靴子就会和着装者的双腿融为一体，让着装者的双腿变成真正的马蹄，并且会自适应的变成着装者希望的关节数量，是否反向与毛发颜色。

效果：着装者会开发出一种全新的“人马合一”的高速移动方式，这会让着装者提升自身传奇敏捷\*4点先攻与30米的移动速度，并且着装者现在可以将攻击结合在一次移动中。
