# 死亡水晶

名称：死亡水晶  
本质：自然  
分类：奇物  
价格：C+1000  
描述：  
这是一种具有半透明质感的晶状矿物，产地位于服部独龙禁忌主星（Forbodulon Prime），那是一颗近乎荒芜的星球，没人知道那里为什么会产出如此造物，但它就是出现了。  
这颗晶体表面折射出朦胧的微光，当它被触摸时，触摸者会看到约十二幅模糊的自身未来的死亡画面同时浮现在视网膜周遭，但这些画面大多只能让你用余光中隐约窥见，无法聚焦，难以全部聚焦清晰。  
每一幅画面都是未来可能的一种死法，这骇人听闻，但你无法保证下一秒会不会有一艘航母跑上陆地碾过来压死你。所以忽视吧，它大多数时候预知的死亡都是不会发生的，除非……  
当你下一刻真的会死或遭受重创使短时间内的未来统一，或是开始进行能威胁到生命的战斗时，十二副画面会在此刻展现惊人的一致——那是你下一刻的受创方式，因为你的未来在当前决定下必死无疑，你也将能看清视网膜画面里发生的一切。

"你的未来源于你的现在，如果你活明白了，它就在不断变化。真相是，任何知道自己确切死法的人，要么无聊透顶，要么马上就要挨枪子儿。"——瑞克·桑切斯  
效果：  
此时，你将预知你下一轮的受创方式，或是敌人下一轮行动的动作如何对你使用，相当于该目标先进行下一轮的行动，在你得知其下一轮即将进行的行为后继续你的行动，这是C级时空来源的未来视效果。并在这种状态下获得两点洞察防御。  
（你的行动有可能会改变你得知的行动结果，例：你得知在你面前的敌人下一轮会对你进行近战攻击后格挡，你选择了移动拉开距离，那么由于其攻击距离不够，他的行动会变为移动到你的面前并进行一次近战攻击）
