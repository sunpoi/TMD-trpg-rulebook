# 死灵之书·伪

  

死灵之书·伪  
  
  
C+1000  
死灵之书是疯诗人阿卜杜拉·阿尔哈萨德于公元730年所写的神秘学百科全书，其内容包罗万象，可以随时使用寻找其中记录的禁忌知识。  
  
但很明显，你能买到的这本书并非正版的死灵之书，只是一本打着死灵之书名号的看起来很普通的书籍。  
  
这是一本体积1的书籍，可以作为施法法器，它如同正版的死灵之书一样，也记载了很多神秘学知识，但阅读这本书会对读者造成更多的精神污染，在战斗场景之外的情况下，永久失去1级专注技能等级和1级专注技能下属的意志专业等级，并阅读其15分钟，可以对一件你希望解析的目标进行一次C级命运来源的解析效果，最多能对某一目标解析3次，这个效果可能会导致某角色失去所有的专注技能等级而导致陷入技能为0的状态，在此状态下，由主持人决定这名角色的状态与生死。当不拥有意志专业技能等级时，无法应用此效果。  
  
在战斗场景中，你可以使用这本书作为施法的法器成分进行使用，若如此做，则你只能使用这本书释放一道固定的法术：恶魔牵引  
  
【恶魔牵引】  
  
使用这道法术需要使用一个标准动作，这本死灵之书会从你的手中自动翻开，不可名状的巨型章鱼触须会从本书翻开的页面中奔涌而出，选择以你为中心，以你的智力\*4米范围内的一个目标，触须会自动试图擒抱那个目标，触须的擒抱和攻击检定是34+4，正常进行对抗，触须的体积是10，如同正常单位一般具有生命和韧性，触须具有31点生命值，以及9+1的基础防御和4点全吸收，如本次攻击未成功造成擒抱，则触须会立即回到书中，下一次使用时需要重新花费标准动作召唤，如擒抱成功，则接下来的每回合你可以花费一个标准动作指挥触须继续行动，若未指挥，则在你使用标准动作时立即解除触须当前正在执行的擒抱动作，它会自动回到书中，下一次使用时需要重新花费标准动作召唤。如触须被杀死，则这本书也会自然毁灭，并在下一天到来或回到主神空间（取近）时自动重新出现在你的身边，同理，直接攻击这本书，它具有的防御和生命值不变，被击破时同样自然消失，并会在下一天到来或回到主神空间（取近）时自然出现。在你的人物等级达到B，且至少经历了一场影片后，触须的擒抱和攻击检定将提升至54+6。
