# 饮血杯

奇物名称：饮血杯  
资源分类：奇物  
本质：魔幻本质  
价格：C+1000

描述：  
异种用来饮血。含有魔物的力量。会使最大HP减少、但战斗中HP再生率提升。

效果：  
占据饰品位置。  
持有者的生命值上限降低传奇耐力点，但会获得等同上述降低生命值上限数值的【快速医疗】与【吸血】，每1点快速医疗对应医疗池12点。
