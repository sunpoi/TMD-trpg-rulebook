# 有刺铁线

奇物名称：有刺铁线  
资源分类：奇物  
本质：特异本质  
价格：C+1000

描述：  
长满尖刺的铁线，挂载这项武器的活死人攻击，可以让对手造成更深刻的伤害。

效果：  
该奇物需要绑定一项武器使用。对于绑定的武器而言，在使用其进行攻击时，附带等同自身【传奇力量-1】点韧性伤害，1点剧痛不良点数，不可豁免，可叠加。
