# 防法师之盾

名称：防法师之盾  
分类：奇物（腰带）  
本质：魔幻本质  
价格：C+1000  
描述：  
深恶痛绝的法师……瞧好吧！我要让你在也打不到我！！！哦，我也是法师啊，那没事了#笑 打的就是你这个死同行！！

【同行之间的诈骗】：  
如果一个法师整天举着一个能量护盾，那才是最蠢的存在！你要悄悄地将这个东西隐藏，绝对不能让那些该死的同行看出来这是盾牌！

【同行是冤家】：  
桀桀桀，他们绝对想不到这个看上去其貌不扬的束腰带将会给你带来多大的帮助，我真是迫不及待地想看到他们目瞪口呆的表情了！！你的腰带将会用能量巧妙的包裹你的全身，并且形成硬化的能量残留借此达到类似于盔甲的效果，但是这个防御将更加着重于抵消对方的破防，因为即使在高超的防御魔法也会被强烈的攻击打破，但是任何一点抵消攻击的手段，都会使攻击弱化，

【除你魔力！】：  
是不是感觉自己的魔源空虚？是不是感觉自己的经脉无力？！哈哈哈哈！是流失！我使用了魔力流失！“除你魔力！！！”

特性：【同行之间的诈骗】【同行是冤家】【除你魔力！】  
效果：  
【同行之间的诈骗】：  
这件奇物将会占据你的腰带位并且看上去就像是一条普通的束腰带，能量显现将会被隐藏在普通的外表之下，不易察觉，同时，你可以借此将你的其他衣物同时伪装，但是不能过于复杂，这视为C级幻象来源的伪装效果

【同行是冤家】：  
你获得3点能量防御，并且获得10点的抵破魔

【除你魔力！】：  
是不是感觉自己的魔源空虚？是不是感觉自己的经脉无力？！哈哈哈哈！是流失！我使用了魔力流失！“除你魔力！！！”你的法术攻击在命中后可以对目标造成8点的魔力流失
