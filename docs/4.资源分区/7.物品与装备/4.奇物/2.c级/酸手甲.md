# 酸手甲

奇物名称：酸手甲  
资源分类：奇物  
本质：魔幻本质  
价格：C+1000

描述：  
异国骑士的装备在炼金术师的实验中被药品侵蚀的产物。进行了战斗用改良、攻击会改为酸属性。

效果：  
占据手腕位置。  
佩戴者进行的攻击改为腐蚀伤害类型，附带1点不可豁免的毒素不良点数；佩戴者的生命值上限提升等同【传奇耐力】+1点。
