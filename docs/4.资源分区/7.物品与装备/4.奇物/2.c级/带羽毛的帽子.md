# 带羽毛的帽子

奇物名称：带羽毛的帽子  
资源分类：奇物  
本质：魔幻本质  
价格：C+2000

描述：  
插有得自雷光峰魔素鸟的魅力羽毛。吸入了多样的灵魂、溢出各种各样的力量，最主要的功能还是提升敏捷性。

效果：  
占据头部位置。  
佩戴者的外貌提升5点；佩戴者在反射豁免上获得等同【敏捷属性】\*1点器械加值，并获得等同基础移动速度的飞行能力，机动性良好。
