# 归乡

归乡（Going home）  
C+1000  
特异本质  
奇物  
归乡是一只金色的萨克斯，但使用者只能用它吹出一首曲子，为《Going home》。

终点站，到了。

到站：每场影片仅一次，面对一个死去不超过15分钟的身前属于现代中华文化圈的完整尸体，你可以对其吹起《Going home》，并在吹完后，对其说出“到站了，xxx（尸体的名字）”。若你如此做，其直到影片结束，其尸体都不会损坏。  
夕归：你可以以一个花招动作吹起/维持演奏《Going home》， 在你演奏期间，你的触及范围的景象将幻化为现代中华文化圈的夕阳下的公交车站/地铁站类场景，这是一个C级幻象效果。  
可选：st可以允许夕归产生的幻象为夕阳下的学校，商城，海洋馆等。
