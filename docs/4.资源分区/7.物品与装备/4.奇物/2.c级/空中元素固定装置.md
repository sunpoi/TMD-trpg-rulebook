# 空中元素固定装置

空中元素固定装置  
C+1000  
某个时空中的兜甲儿在与地狱博士的战斗中战死，不得已只得进行全身机械化改造复活，仅剩下脑和少量神经。此时甲儿的身体搭载了已经去世的如月博士的空中元素固定装置，能够借助空中元素再生出皮肤肉体等等。  
空中元素固定装置是一个小巧的装置，购买后可以在主神空间中安装在任意具有的结构值单位中。  
再生：只要接触空气，空中元素固定装置就能将空中元素转化为固体进行填补，这意味着在接触空气时，你具有1快速维修（医疗点12）。  
魔神闪光：让双拳对撞，在体表再生出一层肉体从而获得人类外形，你可以借此模仿和你的体积差距不超过1的人物的外表，这是一个C级变形来源的伪装能力。  
空中元素固定装置，最大功率！：使用迅捷动作让双拳对撞，让空中元素固定装置变为最大功率，在当前场景中再获得3快速维修（医疗点36）
