# 贤人面具

贤人面具  
价格：C+1000  
分类：奇物  
位置：头部  
描述：保持自己的理性、将精神提高到极限的人的面具。他隐藏自己的脸，一直倾听大圣堂信徒悲惨的话语。  
效果：佩戴者免疫不超过A级混乱、胁迫效果，并在意志豁免上获得【8加骰】。
