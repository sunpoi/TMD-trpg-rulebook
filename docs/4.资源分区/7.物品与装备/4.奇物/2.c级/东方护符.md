# 东方护符

奇物名称：东方护符  
资源分类：奇物  
本质：魔幻本质  
价格：C+2000

描述：  
有将对精神有影响之物消除的力量的异国护符。在那个国家、据说存在解除恶魔诅咒的专家。

特性：【醒神之咒】【破邪之咒】

效果：  
占据饰品位置。  
持有者在决心相关检定上的加骰提升1级处理（上限8加骰），每轮可以使用一个自由动作并消费2点意志力释放下述一项【咒】的效果，每场景仅需因此消费一次意志力即可。

【醒神之咒】：  
以持有者为中心五米半径范围内的所有单位，立刻驱散一项目前正在承受不超过B级心灵来源效果的影响，由持有者选择。此外还会在意志豁免上获得1DP表现加值。

【破邪之咒】：  
以持有者为中心五米半径范围内的所有单位，立刻驱散一项目前正在承受不超过B级诅咒来源效果的影响，由持有者选择。此外还会在强韧豁免上获得1DP表现加值。
