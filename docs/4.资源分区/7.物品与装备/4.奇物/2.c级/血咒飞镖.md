# 血咒飞镖

奇物名称：血咒飞镖  
资源分类：奇物  
本质：魔幻本质  
价格：C+1000

描述：  
《天地玄黄》：“有一类巫术为血咒，血咒之术，均需要以自身气血为媒方可触发，颇为邪门。几百年前有一位高手创造名为血咒飞镖的可怕血咒媒介，将血肉媒介溶于自身后，每当他受伤时，血液都会飞溅而出化作飞镖激射，伤害所有攻击他的敌人。”  
将这枚半寸长的血色飞镖插入血管之中，随后角色每次受到攻击时都会在身周围产生一道血飞镖。

效果：  
占据饰品位。  
角色每次受到攻击时，将自动对攻击的敌人造成1点挥砍伤害。若该目标拥有生理属性为关键属性的能量池，则立刻流失该能量池内的5点能量，如果具有复数，优先选择能量池上限最高者。
