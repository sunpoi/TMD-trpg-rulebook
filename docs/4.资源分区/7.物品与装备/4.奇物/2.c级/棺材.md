# 棺材

奇物名称：棺材  
资源分类：奇物  
本质：神话本质  
价格：C+1500

描述：  
背着、拖着不管怎么看都是棺材的……黑色棺材。某位神明似乎很钟意于这种行为，使背负者可以得到无形的加护。

效果：  
这具棺材视为重甲处理，占用背负者的盔甲位，背负者可以获得12点偏斜防御，同时获得2点盔甲减值。  
背负者获得等同自身【传奇风度\*2】+1点额外偏斜防御。只要你仍然背负着这口棺材，就免疫疲乏点数带来的减值及严重状态，同时免疫不超过C级困难地形对你带来的影响。
