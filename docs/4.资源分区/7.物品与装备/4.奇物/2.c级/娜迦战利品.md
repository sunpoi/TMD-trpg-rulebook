# 娜迦战利品

奇物名称：娜迦战利品  
资源分类：奇物  
本质：魔幻本质  
价格：C+2000

描述：  
娜迦，是我们来到暮色森林时遇见的第一个boss，一头长着绿色鳞片（虽然好像不止绿色）有着人脸的蛇，终日盘旋在自己的庭院之中，即使被迫离开，也会蜷缩着身子试图回到庭院之中，期间我们无法对它造成任何伤害，不过好在，它并不难对付，盾牌可以轻松抵御它的冲撞，并令它陷入眩晕状态，搭高方块也可以让它无法对我们发起攻击，哎等等！这家伙咋红温了！咋把我盾牌撞破了！哎我搭的方块去哪了！  
该奇物一个娜迦的头颅，虽然几乎把你整个头部遮盖起来，但并不会令你的视野受到影响，当戴上此奇物后，你将获得娜迦的部分能力。

【冲撞】：娜迦最基础的攻击方式，或者说它只有这么一种攻击方式。  
【凝视】：当玩家在很高的地方时，娜迦会隔空破坏玩家脚下的方块，所以别试图通过搭高来风筝它。  
【再生】：当娜迦没有受到伤害后，它将逐渐回复生命值并且身体会缓慢再生增长，同时她的鳞片也会为你提供一定防御力。

特性：【冲撞】【凝视】【再生】

效果：  
该奇物占据头盔位。

【冲撞】：  
一轮一次，花费一个自由动作进行冲撞，并且你冲锋时基础防御不再受到减值。

【凝视】：  
当你穿戴该奇物时，你获得一种新的攻击方式，攻击检定为【耐力+表达+专业：凝视+辅助专业：破坏（手艺）】，攻击距离20M，该攻击将获得【贯穿】特性，但只能对结构值单位造成伤害。

【再生】：  
当佩戴此奇物时，你获得快速医疗2（医疗点24）以及2点遮蔽防御。
