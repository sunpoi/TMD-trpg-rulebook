# 月狼挂坠

月狼挂坠  
价格：C+1000  
类型：奇物（护符）  
描述：仿佛一轮弯月一样的皎白狼牙，用一个古朴的绳穿过，对着月亮能看出一句话“至，血誓的盟友”。  
效果：挂饰中月狼的残余之力依然不多，只能帮助你在招架时避开攻击。每轮一次，你招架的时候视为移出攻击范围。此外月狼被称为月狼之因正是其在月色下能发挥更强的力量，在有月光的夜晚环境中，你的近战攻击获得10点环境加值。
