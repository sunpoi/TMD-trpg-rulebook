# 魔法斗篷

魔法斗篷  
价格：C+1000  
分类：奇物  
位置：斗篷  
描述：拥有魔法耐性的极薄外套。形状会产生空气层、能够吸收殴打带来的冲击。  
效果：  
【魔法耐性】：穿戴者在受到法术攻击时，额外获得1个基于偏斜防御的防御附加成功，同时攻击者会在攻击检定上承受【劣化1】，基于偏斜防御。

【抗冲击】：穿戴者免疫冲击点数带来的减值，并获得偏斜防御6，与基于偏斜防御的DR1/-。
