# 染血的凝水归流决

染血的凝水归流决  
奇物/护符  
S+8000  
这原本是一本横练的功法，后来经过某人的改动，这本武学才变成了真正的绝世武学，后来历经波折，持有它的人在后来才将它发扬光大，成为了武林中炙手可热的绝世神功之一。  
这本书书写了不少武学的要义和秘诀，只是上面不知为何有不少地方被鲜血所涂抹，看不清了。

【传承·绝世武学】这本功法在武侠或者修仙世界里，它被视为一本绝世神功。在持有它进入该影片后，大部分人都将知道你持有这本功法，当然在临死之际可以说出一段遗言，把功法传给下一个人，下一个人不一定是轮回者，也可能是世界里某一个土著，无论如何这本武学只要传承下去，它的特性就不会消失，一旦传承失败，那么它的所有特性都会失去，这视为一个SSS级命运效果。

【凝水归流内功】  
这本功法最基础的作用，看起来平平无奇。  
你的内力只要不空，你的感知有多少便能幅增多少点增强加值（包括感知的附加成功）在你生理属性检定上。

传承：  
S+8000  
历代持有此物之人，都会将自身奥义绝学印入其中。  
“不要害怕，我们都在。”  
你可以花费额外的钱将传承印入这本书，又或者作为传承者的你，又把自身开发的绝学印入其中？

【诺艾尔的奥义】  
一个一头金发的外域女子，谁知道她是如何学得这门中原绝学的，但是可以知道的是这本秘籍，是在她手上发扬光大的。

天地人三桥  
这个奥秘是来自众人和自身修行所得，一切皆有定数。  
天桥是最后的登仙之桥，每个影片只能触发一次。  
你可以随时以一个自由动作启用下列任一效果，但每个桥每回合只能各用一次。  
人桥:将2倍内力上限（最高21点）的增强加值在下一轮攻击检定之中。  
地桥:将内力幅增到1250米内另一个人身上，他会获得与你人桥一模一样的增强加值，持续1轮。  
天桥：在受到致死攻击时，免疫该次攻击，并且恢复全身状态至完好，这是一个S级时空来源的复活效果。
