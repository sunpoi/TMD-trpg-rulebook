# 谁的手指？

谁的手指？  
本质：特异本质  
价格：D+1000  
分类：护符  
描述：像是活着一样的指尖可以帮你自动按摩，促进淋巴循环并使体力和活力也跟着一起恢复，基本上很难陷入弱化状态。  
效果：佩戴者获得1点异常点数恢复能力（医疗点24）。此外，由于指尖按摩全身带来的活跃效果，兑换时可以选择在生理大类属性中选择1项属性上得到3DP的增强加值。
