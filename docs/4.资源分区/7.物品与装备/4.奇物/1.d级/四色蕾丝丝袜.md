# 四色蕾丝丝袜

四色蕾丝丝袜

名称：四色蕾丝丝袜  
价格：D+500  
类型：奇物  
本质：特异本质

描述：  
这是几条八成新的蕾丝丝袜，因为占库存所以【便宜】卖了，因为丝袜的发明是为了凸显男性的腿型，所以推荐男性购买。  
因为库存很多，所以我们支持散货购买，从以下四种丝袜中选择两个给你的左右脚即可，当然，你也可以选择两次购买同样的丝袜，两条腿穿一样的丝袜的确也很协调。

【黑丝】：

描述：黑色丝袜更吸热，也就更易出汗

效果：你使用腿脚进行肉搏攻击时附带1点不可豁免的创伤来源的恶心点数，若你左右脚都是黑丝，则叠加为2点，特殊的，ST可以根据对方的癖好改成等量的亢奋点数。

【白丝】：

描述：白色丝袜略显清纯，让对方对你蕴含的力量进行误判。

效果：所以你使用腿脚进行肉搏攻击时获得5【威猛】，若你左右脚都是白丝，则第二条白丝的效果改为，若你具有力量上的附加成功，则腿脚进行的肉搏攻击附带1点不可豁免的创伤来源的疲乏点数，这是被可爱的小脚猛砸胸口的结果。

【肉丝】：

描述：肉色丝袜若隐若现，让被踢的目标陷入你到底穿没穿丝袜的疑惑，这让他的反应略显迟钝

效果：你用腿脚进行的肉搏攻击视为具有D级措手不及，且获得1【高速】。若你的两只脚都是肉丝，则第二条的肉丝改为，若你具有敏捷上的附加成功，则你使用腿脚进行的肉搏攻击附带1点不可豁免的创伤来源的目眩点数，因为敌方会陷入疑惑：“到底穿没穿？刚才踢的是左脚还是右脚？”

【红丝】：

描述：红色丝袜充满诱惑

效果：这让你额外获得等同于风度上附加成功的防御上的闪避加值，若你的左右脚都是红丝，则会发挥完全不一样的效果，不会获得闪避上的防御加值，而是对风度不大于自己的敌方角色进行使用腿脚进行的肉搏攻击时可以不造成伤害，转而造成等量的魅惑点数进行勾引。

此外，在穿着蕾丝丝袜时，你的风度相关检定获得1器械加值。
