# 红丝绳

红丝绳  
价格：D+500  
分类：奇物  
位置：手腕  
描述：象征约定的红丝绳，寄托思念，寄托生的希望。牵住你的手腕，只愿你不会先一步踏入那永世隔绝的黄泉。  
效果：佩戴者免疫不超过A级的即死来源效果；佩戴者在三豁免上获得1点器械加值。
