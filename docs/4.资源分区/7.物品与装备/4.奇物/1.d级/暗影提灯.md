# 暗影提灯

名称：暗影提灯（DARK LANTERN）  
价格：D+500  
分类：奇物  
本质：魔幻本质

描述：  
这盏由暗影秘使们所设计的黑色灯笼放射出昏暗的秘术之光。  
这件物品是一盏带着灯盖的灯笼，由秘银和黑铁细工打造而成。其上覆盖的水晶板则是色彩鲜明的金黄色。  
暗影提灯用其秘术之光创造出昏暗环境，并可以改变四周环境光的亮度，使其变为昏暗  
光线闪烁的比正常的火焰更慢，并且其中火焰之中发出的噼啪声更像是沉重的脚步声。

特性：【激活】  
效果：  
【激活】：点亮提灯需要说出一个命令词，例如“阴影”或“黑暗”。激活灯笼是一个整轮动作。你也可以用一个标准动作打开灯盖并吹灭其中的火焰，从而中止提灯的效果，一个充满暗影能量暗影提灯可点亮8个小时，在暗影能量消耗殆尽后，提灯需要在黑暗环境内花费4个小时来重新充满。  
当灯点亮时，会使得周围60米范围内的环境变为昏暗（即使原本为明亮或是黑暗），起效范围为半径60米的球体，这是一个D级环境来源的效果。
