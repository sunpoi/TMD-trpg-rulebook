# 外置金丹

外置金丹  
本质：魔幻本质  
分类：奇物（手串）  
价格：D+1000  
占用：手腕与护符位  
前提：具备能量池  
描述：还在为你的能量不够而担忧吗？还在因为你无法容纳更多的回路而苦恼吗？还在因为无法直接得到简单直接的超强外挂而绝望吗？从现在开始你无需担心了！为了应召广大人民的需要，我方公司【内力改革】全新推出一款产品，此物由活人炼制，去除经络属性，压缩成完美大小，居家外出都能够给予您慢慢的安全感！还不快来买吧！

效果：  
画风上这是一串由多个直径相同的珠子制作成的手串，结实耐用，颜色艳丽，带着若有若无的香味：扩大2点指定的一项通用能量池。  
购买后外置金丹会自动匹配你当前所拥有的东方能量池（如内力，真气等，若有多个，选择其中之一），后【金丹】将会为你增加能量池副属性\*1点的能量上限，在原理上，你花费时间逐渐再【金丹】内部构筑出来了与自己相似的能量回路。  
在你佩戴【金丹】的那一刻，【金丹】内部自带的金丹立场将会立刻包裹你的全身，为你提供微弱的防御，毕竟一个好的产品，首先需要确保宿主的存活：若绑定的能量池不空，则获得2点力场防御，  
金丹带来的不仅仅是广大的基底，还带来了持久的恢复能力，在原理上，在【金丹】具备你的能量回路后，他便会自行运作，为你长久的提供能量补给：指定你所绑定的能量池，每小时一次，进行一次能量池关键属性检定，恢复成功数点能量，这会计入每日的恢复上限。
