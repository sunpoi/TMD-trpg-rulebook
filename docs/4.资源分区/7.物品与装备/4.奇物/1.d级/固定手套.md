# 固定手套

名称：固定手套  
价格：D+500  
分类：奇物/占据手腕位  
本质：科技本质

描述：固定武器的特殊手套，除了防刃、防武器滑脱外，也包括了绑带等配件。其实是施加了固定武器的特殊效果，因此即使武器不在手部也能正常生效 。

效果：佩戴者【免疫击落武器】；进行格挡时，佩戴者额外获得1点格挡防御。
