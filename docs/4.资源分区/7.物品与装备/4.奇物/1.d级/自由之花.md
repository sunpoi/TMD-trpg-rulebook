# 自由之花

自由之花

名称：自由之花  
价格：D+500  
分类：奇物/占据饰品位  
本质：魔幻本质

描述：  
“何塞啊何塞，可怜的何塞，狠心的何塞。就算重活一世，就算了无身形，还是要杀掉这只自由的鸟吗？还是要杀掉这无拘的爱吗？”

这是一捧蓝色的花束，花种未知，闻起来并无花草的芳香，却有着莫名令人熟悉的气息。携带它将会占据饰品位

效果：  
使携带者免疫擒抱状态的影响，同时在意志豁免上获得2DP器械加值。
