# 成长之钥

名称：成长之钥  
价格：D+500  
分类：奇物  
本质：魔幻本质

描述：这是一把微闪着银光的密钥，大约有25厘米长，在它接触到一柄武器/法器后（无论其为何材质），便会永久融入其中，无法再取出。

效果：  
获得成长之钥加持的武器获得等于人物支线等级对应级别的武器模板，这会令武器伤害/法术强度提升至（不包括附加）（D6C11B18A27S38）点  
若是仅提供附加成功的武器，则武器提供的附加成功会提升至（D2/C4/B7/A12/S16)点  
若附加的武器被其他能力所提升了武器伤害、法术强度、附加成功，成长之钥的提升会在这些能力提升之后再进行结算。若武器同时具有法术强度、武器伤害、附加成功，仅提升数值较高（1点附加成功按照2点武器伤害/法术强度计算）的那一项。

特殊的，如果st允许，这柄神秘的钥匙也可以用于概念武装
