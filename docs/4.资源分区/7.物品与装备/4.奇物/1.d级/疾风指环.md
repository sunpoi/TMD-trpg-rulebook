# 疾风指环

疾风指环  
价格：D+500  
分类：奇物  
位置：戒指位  
描述：据说使用原理不明剑技的古代高手戴的指环。据说老年的高手在偏僻&村的森林中、一人努力修行。  
效果：佩戴者每经历一场影片，就在使用剑专业的相关检定上获得1DP修行加值；佩戴者在进入战斗后每经过1轮，都会获得1点洞察防御，至多获得15点，场景切换时清空。
