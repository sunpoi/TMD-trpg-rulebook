# 玛莲妮亚的大卢恩

玛莲妮亚的大卢恩

名称：玛莲妮亚的大卢恩（伪）  
价格：D+750  
分类：奇物/占据戒指位  
本质：特异本质

描述：

【玛莲妮亚是玛莉卡女王与拉达冈生下的子嗣。  
此大卢恩原本该是最神圣的一个。】

……这不是“碎片君王”玛莲妮亚的大卢恩，只是沾染了些许气息的仿制饰品。

【义手剑士的反抗意志】  
其中残留的反抗意志

【水鸟剑舞】  
因为卢恩弯弧的稀少，制作此物的工匠将弯弧的功能融入了这件物品本身。

特性：【义手剑士的反抗意志】【水鸟剑舞】【飞翼】  
效果：

【义手剑士的反抗意志】

佩戴者获得1点异常点数恢复能力（医疗点24）

【水鸟剑舞】

消耗一个移动动作，仿佛使用卢恩弯弧一般捏碎大卢恩，使之带来恩惠。  
此时，你失去【义手剑士的反抗意志】效果，改为攻击获得破甲5，【飞翼】，持续一场景。

特殊的，在场景结束时，大卢恩的碎屑将再次恢复完好并回到你的手中。

【飞翼】

当你受到至少1点严重伤害时，获得持续一轮的吸血1。
