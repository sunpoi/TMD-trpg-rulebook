# 便携式检修仓

  

便携式检修仓  
D+500  
科幻本质奇物  
  
购买时绑定使用者，绑定者需为具有结构值的单位。  
收纳时体积1，展开时体积为绑定者体积+1。无限能源。  
绑定者可以进入维修舱，这时维修舱会对绑定者进行长线修复。  
修复检定的dp值默认为21+2。  
使用维修舱进行的长线修复的每次判定从1小时缩短到15分钟。  
你需要为检修舱准备修复用的材料，检修舱至多可放置修复36次的材料。
