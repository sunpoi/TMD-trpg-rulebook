# 魔术师的帽子

魔术师的帽子

名称：魔术师的帽子  
价格：D+500  
分类：奇物/占据头部位  
本质：魔幻本质

描述：谁都能得到熟练魔术师般的醇厚玛纳，不成熟的魔术师为了使用强力魔法会戴上。

效果：佩戴者的智力检定获得3点增强加值；佩戴者在施展法术，法术攻击检定获得等同【传奇智力】\*1DP器械加值，并在魔力消耗上减少等量的数值（下限1），每轮只能因此减少一次消耗。
