# 卢恩符文手套

卢恩符文手套

名称：卢恩符文手套  
价格：D+500  
分类：奇物  
本质：魔幻本质

描述：  
远古北方大地，不知名的游魂所遗留的手套，在残留的暗红色的血迹上，有着由秘银所描绘的花纹，仔细看那些花纹，便能发现其中所蕴含的神秘。  
这是为那些不曾研究魔法，却向往魔法力量的人所制造的道具。  
但那些洗不去的暗红色，似乎在诉说什么.....

这个手套的造型为只遮盖手掌和第一指节的露指手套，使用者以标准动作，在战斗外进行一次附魔，用带着手套的食指在实体投射物上方书写出代表空间的卢恩符文，这次书写不需要接触所要附魔的物品，这意味着，即使是一个山岭巨人，也可以用它宽大的手指，在一发0.38毫米的子弹上附魔。  
从表现上看，手套使用者书写两个卢恩字母后，发出荧光的字母会在虚空中浮现，并旋转缩小，印在附魔物上，形成银色的花纹，写出的卢恩符文会仅能被书写者的意志所激发。  
人物附魔的投射物数量不具有限制，但效果最多只能维持24小时，在超过时效后符文会无害消散

效果：  
在使用投射物攻击时，人物需自由动作支付1点意志激发其上的符文，随后投射物在从发射器射出的一瞬间，将跨越地形阻碍，传送到攻击发起者可以定位的一个，没有被占据的地点或目标身前，若攻击发起者拥有可以无视阻碍的侦测手段（如透视），该次攻击可直接命中位于载具或其他封闭掩体内的目标，同样，这也允许攻击发起者在封闭的掩体内，在不损害掩体的前提下向外射击（前提同样是拥有定位封闭掩体外目标的手段），这是C级时空来源的【传送】效果，使用附魔投射物的该次攻击，具有【法破】的效果。
