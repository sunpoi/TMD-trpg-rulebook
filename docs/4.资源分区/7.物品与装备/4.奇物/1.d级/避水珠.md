# 避水珠

名称：避水珠  
价格：D+500  
分类：奇物/占据饰品位  
本质：魔幻本质

描述：水不能溺，火不能焚。那些儿去不得？

效果：佩戴者获得对B级及以下潮湿效果的免疫，获得水下呼吸的能力。
