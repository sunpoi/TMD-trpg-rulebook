# 功德木鱼

功德木鱼  
D+500  
奇物  
一个个看起来普普通通的木制木鱼，配着一把木头小锤，用小锤敲击木头的时候会发出嘟嘟嘟的声音，并会在角色头上显示功德+1的字样。  
你可以通过木鱼来获得功德，每次敲击功德加1，但你因此获得的功德最多只能存储100点，如果你把木鱼转让给其他人，你的功德也会同步转移给他们。你可以消耗功德来获得以下的效果，持续一场景。  
消耗20功德:你攻击【黑暗生物】时获得1dp士气加值。  
消耗40功德:你攻击【黑暗生物】时获得1dp士气加值和1点抵破魔。  
消耗60功德:你攻击【黑暗生物】时获得1dp士气加值和1点抵破魔并在【黑暗生物】攻击你时获得1点力场防御。  
消耗100功德:你攻击【黑暗生物】时获得1dp士气加值和1点抵破魔并在【黑暗生物】攻击你时获得1点力场防御，若你因为【黑暗生物】的攻击满足了开启基因锁的条件，那么这次基因锁的开启获得1点机运加值。  
特别的是，在战斗中你也可以通过消耗一个移动动作来快速敲击木鱼来快速积攒功德。这会让你在一轮内获得至多60功德,如果一轮获得的功德超过了这个量会被佛祖认为恶意刷功德导致功德清零。  
获得功德的过程是不可以被孤魂野鬼肆意打断的，因此在获得功德的过程中你被视为拥有仅对于【黑暗生物】生效的攻击劣化一级，这是基于力场防御的效果，持续一轮。
