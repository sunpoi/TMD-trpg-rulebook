# 魔具哈克特

魔具哈克特

名称：魔具哈克特  
价格：D+500  
分类：奇物/占据头部位  
本质：魔幻本质

描述：惊人的动作、不管从多高跳下也能活下来的不怕死的人喜欢的帽子。

效果：装备者在敏捷相关检定上获得3DP器械加值；在进行攀爬时，装备者无需通过检定也能正常攀爬移动，并且免疫坠落伤害。
