# 防护斗篷

名称：防护斗篷  
价格：D+500  
分类：奇物/占据披风位  
本质：魔幻本质

描述：这件由秘银丝线编织、内衬古老符文的斗篷散发着微光。穿戴者能感受到一股无形的守护之力充盈全身，使其在面对各类物理冲击、精神侵袭乃至环境剧变时，防御能力显著提升，意志更为坚定

特性：【守护秘文】  
效果：  
【守护秘文】  
此时穿戴者的三豁免获得4dp器械加值。同时，斗篷还提供了一层微弱但全面的能量防护屏障，穿戴者还获得1点近战伤害吸收与1点全能量抗力。
