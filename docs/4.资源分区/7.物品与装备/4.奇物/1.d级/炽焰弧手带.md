# 炽焰弧手带

名称：炽焰弧手带 Bands of the Blazing Arc  
价格：D+1000  
本质：特异本质  
占据：手套位  
描述：  
这对金属手带制作精良。每个手带的关节处都镶有一个小小的红色水晶。 蚀刻在金属表面的火焰图案使手带呈现出火热的外观。  
这对金属制的手带能爆发小范围的锥型火焰。  
当手带被激活时，每个手带的水晶都会发出明亮的光芒，就好像从佩戴者的手中喷出灼人的热浪。之后还能闻到微微的硫磺气味。  
效果：  
将双手并拢，以标准动作令两个手带互相接触，你能用思维命令激活手环，并进行一次施法攻击检定（前提是具有施法攻击方式），对人物正前方15米锥形范围内造成火焰伤害，该火焰伤害可以正常影响灵体，视为具有【法破】特性。

若人物具有精神力，则可消耗2点精神力增幅，使得本次射程变为决心属性\*3米
