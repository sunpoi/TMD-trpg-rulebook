# 口袋树海

口袋树海  
本质：特异本质  
价格：D+750  
分类：护符  
描述：口袋大小的树海，被绿意盎然治愈了身体，能够提高佩戴者的恢复能力。  
由于口袋树海是很正常的绿植，并不会发生与你抢氧气的问题，请放心携带吧。  
效果：佩戴者指定自身的一个能力，该能力具有的快速医疗点数提升自身传奇耐力\*1点。每因此提升1点快速医疗，医疗池点数扩大12点。  
此外，口袋树海具有提供氧气的能力，佩戴着获得全环境呼吸效果。
