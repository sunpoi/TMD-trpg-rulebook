# 韦尔奇乐牌汽水机

  

韦尔奇乐牌汽水机  
D+500  
韦尔奇乐牌汽水（F-05-52）是一台体积6闪亮的银灰色自动售货机。它的两侧各站着一个长得很像煮熟的大虾的体积为6的守卫，它们穿着灰蓝色的吊带工装裤和黄色的防水靴。自动售货机的出货口有一罐天蓝色的汽水，上面还有“免费（FREE）”的标语。  
在你购买了这个奇物后，你可以在非战斗场景中的街道或走廊中发现这台汽水机，但你只能同时在一个地方发现它，如果轮回者小队中有复数轮回者购买了韦尔奇乐牌汽水机，那么也只会同时发现一台。  
在找到这台汽水机后，它的购买者可以以一个自由动作按下按钮来试图购买一瓶免费的韦尔奇乐牌汽水，紧接着，在12s内，会有金属碰撞的声音，数罐不同口味的韦尔奇乐牌汽水会从取物口滚落出来，但不会出现葡萄味的。数量等同于今日未饮用过韦尔奇乐牌汽水的，在汽水机旁的总人数。  
必须在非战斗场景中才能饮用汽水，也不能将汽水带出场景。  
喝下汽水，可以立刻治疗1点严重伤害（有韧性伤害时优先治愈韧性伤害）和1点意志，所有自愿喝下汽水的人都能享受这个效果。喝下汽水是一个花招动作，每个人每天只能自愿喝一瓶韦尔奇乐牌汽水，如果超过这个数目，就会不可避免的沉沉睡去，并且场景内其他试图观测他的人眼前都会出现卡通像素风格的波浪船图案，这视为一个幻象来源的D级效果，并在他彻底睡着后消失。自愿喝下第二罐汽水的人会发现这是葡萄味的汽水，并在醒来时随机出现在一个周围合理的、宽敞的、尽可能都是水的环境中（如海、大湖），由主持人决定，这视为进行了一次旅行，在旅行过程中，若以非常规方式醒来，则会发现自己正被两只煮熟的大虾抬着前往未知地点，试图对大虾进行攻击时，它们会在一轮内以逃窜的形式消失，且你会被留在一个未知的地点。
