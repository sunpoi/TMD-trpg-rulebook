# 染血斗篷

染血斗篷  
价格：D+500  
分类：奇物  
位置：斗篷  
描述：沾满血的斗篷。佩戴者可以获得强韧的精神力。在残酷的实验中虽然滥用镇静剂、但也有以自己的精神力忍耐下来的人。  
效果：穿戴者在意志豁免上获得【8加骰】，并在决心相关检定上获得1DP表现加值。
