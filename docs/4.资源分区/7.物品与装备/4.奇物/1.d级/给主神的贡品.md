# 给主神的贡品

给主神的贡品  
奇物  
D+500  
这是一个神龛，里面供奉着主神。  
平时会以一块刻着主神形象（大光球？）的令牌，在轮回者怀着虔诚之心呼唤主神之名时便会变成神龛。  
神龛木制的底座上有着神龛本体和一个盘子，神龛的造型和材质根据主神的形象改变。轮回者需要往盘子上放置符合主神胃口的贡品并向主神祈祷，在虔诚的祷告后，主神将会降下祝福。  
启动祝福是一个花招动作，该场景内，进行了上贡的轮回者在场景内随机获得以下效果中的一项：  
\*攻击轮回者的单位，会在攻击上承受劣化1（基于机运防御）。  
\*基于机运防御的1点全伤害吸收。  
\*从破甲/破魔/高速/威猛中选择一项，在攻击时获得5点对应破防能力。  
\*快速医疗1（医疗点数12）。  
场景内只会有一个轮回者能够享受这一好处，如果有任一轮回者在场景内已经使用了这一奇物，其他轮回者即使上供也无法享受到效果。
