# 他是谁？

【他是谁？】  
d+500  
奇物  
想法源自斩服少女中出现的实体文字以及动画中常常出现的场景或角色出场时用于助于人物识别的文字。  
是在抵达新环境或见识到新的角色后会在身旁或身前出现文字的奇妙现象。

【他是谁？】在你来到一个地方时，或你见到了一个特殊的，拥有戏份的人时  
在你能看见的位置会出现一行实体字展示这个地方的名字或这个人的姓名。  
这视为一个命运来源的C级预知效果  
【我是谁？】同时，在你被他人侦查到时，会在你的身前也会出现一串带着你姓名的字体，这视为给予你1点遮蔽防御。
