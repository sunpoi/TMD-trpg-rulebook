# 受虐的玛娜指环

受虐的玛娜指环

名称：受虐的玛娜指环  
价格：D+500  
分类：奇物/占据戒指位

描述：据说越是接近死亡、身心就越平静、头脑就越清醒。死囚魔术师直到最后一刻之前佩戴的装饰品。

效果：佩戴者在完好生命值被严重伤害填满后，若在当轮或者上一轮使用了魔力，便可以恢复1点魔力这会计入每日恢复上限，这一效果每轮只生效一次。此外，在满足上述条件的情况下，佩戴者的智力相关检定获得等同【沉着属性/2】点器械加值。
