# 世界上最软的小猫软糖

世界上最软的小猫软糖

一次性消耗品  
500

这就是世界上最可爱，最软，最美味的小猫软糖！  
惊世的可爱扣人心弦，绝顶的美味诱人放错！  
不管如何，在你将它放进口中的那一刻，你就是世界上最幸福的人！

吃下这枚小猫软糖会为你提供持续一轮的临时生命4。  
但是随后你会因为失去了最可爱的小猫软糖受到5点沮丧异常点数，意志豁免。  
另外，如果你持有小猫软糖但是不食用，那你每小时需要进行一次DC为1的意志豁免，  
若失败，则你没能抵扣小猫软糖的可口诱惑，将其囫囵吞下。
