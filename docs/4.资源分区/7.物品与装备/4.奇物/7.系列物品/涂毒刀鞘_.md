# 涂毒刀鞘

名称：涂毒刀鞘  
类别：奇物  
价格：D+500  
效果：在购买时决定所收纳的兵器，并在此时定制其外形。无论收纳那种兵器，在其收纳刀鞘当中一轮后，会自动运行其中的机关并为兵器涂毒，并且支持同时保存多种类型的毒在刀鞘中，也支持自动涂毒时决定涂多少种毒，并同时生效，而刀鞘对毒的总容纳上限为100份。  
下属的毒素来源均为毒素/药物来源。

名称：弱效急性毒  
价格：500分/10份  
效果：当涂有弱效急性毒的兵器对目标造成伤害时会使目标获得1点不可豁免的毒素异常点，每次涂毒只会生效一次攻击。此毒只能用于涂毒刀鞘使用。

名称：中效急性毒  
价格：D+500/10份  
效果：当涂有中效急性毒的兵器对目标造成伤害时会使目标获得3点不可豁免的毒素异常点,每次涂毒只会生效一次攻击。此毒只能用于涂毒刀鞘使用。

名称：强效急性毒
