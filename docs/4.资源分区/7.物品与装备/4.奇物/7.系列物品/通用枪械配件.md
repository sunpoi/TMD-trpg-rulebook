# 通用枪械配件

通用枪械配件  
过去的段子说的好，看那灯塔国大兵手上的圣诞树一样的烧火棍是他们强悍单兵战斗能力的一部分，更是灯塔国财力的象征。我们的主神在经过多次考量后，最后决定为TMD主神空间的轮回者们提供相关配件。  
虽然对于轮回者而言，拿着圣诞树一样的烧火棍并不代表他们的实力强劲，但至少能显露出他们的财力如何。

安装本资源的配件时，不受减益的配件数量至多为（枪械体积-1），之后每超出1个配件便在射击相关检定上蒙受1DP器械减值。不可安装重复类型配件，除非特别提及。  
计算配件数量时，“现实远程武器”中除了“左轮装弹器”“弹鼓”外的枪械辅助物品一并计入，并适用上条所述规则。如果与其他提供类似效果的资源一同安装，且无特殊声明差异，则一并计算之并依照上条。

————【瞄具】————  
红点瞄准镜  
价格：250  
描述：红点镜，部份型号又名反射式瞄准镜，亦作内红点，是一种无放大率的光学瞄具，有光亮的瞄准点（通常为一红点）  
效果：进行瞄准时提供2点瞄准加值，但不能超过瞄准上限。

全息瞄准镜  
价格：500  
描述：全称激光全息衍射式瞄准镜， 是一种直接观察弹着点并用弹着点作为瞄准标志的革命式速瞄瞄具。瞄准快而射击精准。也能用来辅助侦查。  
效果：进行瞄准时提供2点瞄准加值，但不能超过瞄准上限；使用瞄具进行侦查相关检定时，提供1DP器械加值（透过瞄准镜侦查，因而要求与人眼相似的成像原理）

侧位瞄具架  
价格：250  
描述：一类安装在枪身两侧的瞄具架，通常搭载低倍瞄具，方便射手侧身射击或迅速切换瞄具倍率。  
效果：为安装的枪械提供一个额外的瞄具位，最多安装两个。

特殊的，本配件不计入配件数量。

主动型夜视仪瞄具  
价格：250  
描述：通过发射红外光线并接收其反射回来的光线来形成图像。这种夜视仪在完全黑暗的环境中也能工作，但成像距离和清晰度可能受到限制。  
效果：使用本瞄具观察时，视为获得【黑暗视觉】效果。

特殊的，本配件可作为改造为其他瞄具购买，并提供相同效果。并使计入配件数量+1。

热成像瞄具  
价格：2500  
描述：即热成像瞄准镜，一种利用目标与背景之间的温差形成热图像的瞄具设备。在夜战中发挥着较大作用。  
效果：透过本瞄具观察时，获得基于温差识别目标的强感知方式并视为拥有【黑暗视觉】，例如瞄准在烟雾中的生物、正在工作的机械或某一热源。

特殊的，本配件可作为改造为其他瞄具购买，并提供相同效果。并使计入配件数量+1。

————【枪口】————  
消焰器  
价格：500  
描述：通过减少枪口焰的方式，从而提高射击时的隐蔽性，避免射手被察觉位置的枪口装置。此外还有些许减轻后坐力的功效。  
效果：因射击而令某人与你进行侦测对抗时，你的潜藏检定获得1点DP器械加值。此外，当你进行任何速射时（短点射/长点射/连射）再获得1DP器械加值。

消音器  
价格：500  
描述：通过改变推进气体的排溢方式，有时也通过降低弹头的射速，来达到抑制声音的效果。  
效果：因射击而令某人与你进行侦测对抗时，你的潜藏检定获得2点DP器械加值。此外，你在你的回合内可以以一个自由动作拆卸/安装消音器。

补偿器  
价格：500  
描述：补偿器主要用途是增加弹道距离的稳定性，使枪械在射击时更容易控制。具体来说，通过减少枪械射击时产生的后坐力，来提高射击的准确性和稳定性。  
效果：在射击检定上获得1DP器械加值，如果这次射击进行了任何速射（短点射/长点射/连射）则再获得1DP器械加值。

退制器  
价格：1000  
描述：退制器多用于大口径机枪、半自动和自动枪械上。在减小后坐力的同时，还会产生较大的冲击波、声响和火焰，对射手来说容易暴露位置，  
效果：在射击检定上获得1DP器械加值，如果这次射击进行了任何速射（短点射/长点射/连射）则再获得2DP器械加值。但因射击而令某人与你进行侦测对抗时，你的潜藏检定获得2点DP器械减值。

枪口收束器  
价格：500  
描述：收束器主要作用是减少鹿弹的散布范围，使弹着点更加集中，从而增加射程和提高伤害。通过控制弹丸在出膛瞬间的方向，使弹丸收束在更小的角度之内，减小散布面积，提高射击精度。  
效果：使安装的霰弹枪增加2点威猛。

特殊的，本配件仅限霰弹枪安装，并且射击独头弹时不生效。

————【枪管】————  
短程枪管  
价格：250  
描述：顾名思义，即相对于标准枪管长度有所减少的枪管。牺牲了相当的精准性与射程精度，换取了更容易藏匿的体积。  
效果：枪械体积-1，最多减少到3。

加长枪管  
价格：1000  
描述：顾名思义，即相对于标准枪管长度有所增加的枪管。通过增加子弹在枪管内的受推动距离，从而提高子弹的初速度。不过枪管并非越长越好，要按需考虑。  
效果：枪械体积+1。安装的枪械基础射程翻倍，如果是步枪/狙击枪则射程增加30m。

卡宾枪枪管  
价格：1250  
描述：一类为短距离快速交手而生的枪管，以牺牲了子弹部分的初速度与射程为代价，用比标准枪管更短的枪管方便枪手快速抬枪射击。  
效果：枪械体积-1。手持安装本配件的枪械时，你的先攻检定获得2DP器械加值，并且你从身上取出该武器的动作降低一级（自由-花招-迅捷-移动-标准-整轮）

特殊的，本配件仅限步枪/狙击枪/机枪安装。

————【枪托】————  
特殊的，手枪类武器不可安装本分区配件

无枪托构造  
价格：750  
描述：通过把枪托部位与枪身整合，使得全枪长度缩短，加快了枪手举枪瞄准的速度。当然，直接把枪械的枪托拆掉也有类似效果，一般会配备一个枪托垫。  
效果：你从身上取出该武器的动作降低一级（自由-花招-迅捷-移动-标准-整轮）

撞火枪托  
价格：1000  
描述：一种用于半自动武器的一体化配件，通过一并更换枪托与后握把，使半自动武器能够连续发射子弹，从而提高射速及杀伤力。  
效果：提升枪械的速射等级至连射。

CQB枪托  
价格：500  
描述：基于CQB需求特化的一类枪托，在保有原本功能的前提下增加了打到敌人身上的分量。  
效果：本配件视为短棍

————【护木】————  
特殊的，手枪类武器不可安装本分区配件

木质护木  
价格：500  
描述：最早，也是最经典的护木，例如大名鼎鼎的AK74就是使用的木质护木。  
效果：射击不受减益的最大配件数+1，且本身不计入配件总数。但当你拥有【潮湿】状态或处于潮湿环境下时，本配件正常计入配件数。

轻质护木  
价格：1000  
描述：随着材料学的进步，以更可靠的性能与相对轻盈的质量取代了木质护木。同时也为搭载更多战术配件预留了空间。  
效果：射击不受减益的最大配件数+2，且本身不计入配件总数。

装饰护木  
价格：1000  
描述：这类护木经常是金银或容易雕刻塑形的材质，表面有优美的雕花或纹路，多在工艺品或象征枪械上出现。  
效果：射击不受减益的最大配件数+1，且本身不计入配件总数。向他人展示护木时，你接下来对他们的社交相关检定获得2DP器械加值，但不能用于攻击检定上。

————【握把】————  
特殊的，手枪类武器不可安装本分区配件。

垂直握把  
价格：750  
描述：经典款握把，主要功能为减小枪支在垂直方向（即上下方向）的后坐力，使枪口在垂直方向上的抖动变得更为稳定。适用于需要稳定射击的情景，如近战扫射。  
效果：射击不受减益的最大配件数+1，且本身不计入配件总数。当你进行任何速射时（短点射/长点射/连射）再获得1DP器械加值。

三角握把  
价格：750  
描述：主要功能为减小枪支在水平方向（即左右方向）的后坐力，使枪口在水平方向上的抖动得到控制。适用于远距离射击或快速调整枪口的场景，如步枪点射。  
效果：射击不受减益的最大配件数+1，且本身不计入配件总数。对单个目标射击后，可迅速调转枪口同时对射程内另一个单位进行一次相同的攻击，这会正常消耗弹药。

拇指握把  
价格：1250  
描述：主要功能为增加开镜速度，同时略微减小枪支后坐力。适用于需要快速瞄准进入状态的场景。  
效果：射击不受减益的最大配件数+1，且本身不计入配件总数。每场景的首次瞄准所需的动作降低一级（自由-花招-迅捷-移动-标准-整轮）。

阻手器  
价格：500  
描述：在C型夹钳握持法广泛传播后，图方便将三角握把简化之后的产物，其目的就是为了在C型夹钳握持时能够卡住拇指不令其后移。  
效果：本身不计入配件总数。对单个目标射击后，可迅速调转枪口同时对射程内另一个单位进行一次相同的攻击，这会正常消耗弹药。

————【战术配件】————  
激光指示器  
价格：500  
描述：用于辅助枪手瞄准目标的配件，最初是红色，但因对人眼伤害太强而后改成了绿色、蓝色或紫色等版本。  
效果：启用激光指示器时，进行瞄准时瞄准加值+1，瞄准上限+1。但某人与你进行侦测对抗时，你的潜藏检定因此获得1点DP器械减值。打开/关掉激光指示器是一个迅捷动作。

安装多个激光指示器增益不叠加，但负面效果可叠加。

战术手电  
价格：100  
描述：比起寻常的手电，战术手电的续航时间更久，光束更集中，此外没有任何不同。  
效果：同手电一样提供照明。但某人与你进行侦测对抗时，你的潜藏检定因此获得1点DP器械减值。打开/关闭战术手电是一个自由动作。

安装多个战术手电增益不叠加，但负面效果可叠加。

两脚架  
价格：250  
描述：两脚架可以有效降低射击时的后坐力影响，提高射击精度，特别是在需要长时间稳定射击或远距离射击时。  
效果：在伏地或拥有合适的掩体时，展开两脚架后，射击相关检定获得1DP器械加值。展开/收起两脚架是一个迅捷动作。

三脚架  
价格：500  
描述：相比两脚架，三脚架体积更大，但也能为机枪、狙击枪等枪械提供稳定的射击平台。经常单独移动携带或安装在固定位置。  
效果：部署三脚架需要一小块相对平坦的地形，使用安装在其上的枪械时，射击相关检定获得2DP器械加值，部署/收起三脚架是一个迅捷动作。

特殊的，本配件只供步枪/狙击枪/机枪使用，并且视为随身物品而非枪械配件，故不计入总数。

————【特殊配件】————  
冲锋枪套件  
前提：非左轮，是手枪，体积不大于2  
价格：750  
描述：通常包括更长的枪管、更大的弹匣、以及可能的自动射击机构。以用于将手枪转换为具有更高射速和火力的冲锋枪。  
效果：将枪械体积变为3，本枪械在使用时视为冲锋枪而非手枪，可以正常安装护木/握把。武器伤害参考安装配件的武器，其余特性参考冲锋枪模板。拆卸/安装套件是一个标准动作

卡宾枪套件  
前提：非左轮，是手枪，体积不大于2  
价格：1750  
描述：通常包括一个更长的枪管、一个可伸缩的枪托以及其他战术配件，不过TMD版不提供额外战术配件。以提高手枪的射程、精度和稳定性，同时使其更适合户外或远距离作战。  
效果：将枪械体积变为3，本枪械在使用时视为步枪而非手枪，可以正常安装护木/握把。武器伤害、基础射程参考安装配件的武器，且基础射程翻倍，其余特性参考步枪模板。拆卸/安装套件是一个标准动作
