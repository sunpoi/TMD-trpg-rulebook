# 仿龙血软糖

六一纪念糖果-仿龙血软糖  
本质：魔幻本质  
物品-奇物  
500/3颗  
包裹着一滴假龙血的软糖，实际上是由一些抗寒药剂做成的，祝福勇敢的孩子无惧寒风。  
食用一颗软糖后，一小时内食用者免疫寒冷环境的效果.。
