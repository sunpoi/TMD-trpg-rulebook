# 特供薄荷糖

六一  特供薄荷糖  
500分/3个  
消耗品  
这是由最纯粹的阴灵精气混合鲜榨薄荷汁辅以百合，川贝母，枇杷叶，菊花……所制成，具有清热下火，养阴止咳的功效，当然，最主要的还是纯净阴灵精气带来的深入灵魂的凉爽。  
食用后会为使用者清除4燃烧点数异常，但因为小作坊下料就是猛，你会受到不可豁免的1点冻结异常点数。
