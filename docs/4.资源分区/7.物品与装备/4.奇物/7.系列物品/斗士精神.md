# 斗士精神

斗士精神  
从假面骑士还有龙珠世界走过之后【逆位】死神又来到了数码宝贝的世界，看着这里由数据产生的生物感觉超级的神奇，之后看见了斗士精神这种让人类变成数码兽的神奇物品所以便学习制作，就产出了量产的斗士精神！  
Ps：该系列依旧为差升，但是你也可以直接跳级买C

  
火之斗士精神：  
初生的人型斗士精神（火）  
D+500  
D级奇物  
虽然进行了量产但是还是因为数码兽的设定所以恶趣味的制作了这种类似于成长期级别的斗士精神，但是样子是火神兽的样子约等于刚出场的火神兽的级别，仅仅是对于身体进行强化顺便提供一些装甲进行防御但是对于火焰的力量使用没有那么熟练。该斗士精神的外貌为一个黑色半身像头发为黄色上面穿着红色的装甲。  
火之斗士！（火神兽）：你使用一个标准动作调用火之斗士精神之中的力量变身为火神兽，该状态会改变你的外貌变成为火神兽的样子，头发为黄色，身上穿着黑色底衣外面有着红色的盔甲，在变身后你的攻击会缠绕着那来自与数据之网之上防火墙的火焰所以你的攻击属性变为火焰属性，而因为你的身体从普通的人类的躯体被提升为了那数据躯体成长期的数码宝贝的强度，你的身体素质也从人类完全超脱而出变成了非人级别的存在，在进行生理属性相关的检定时获得5点内在加值。
