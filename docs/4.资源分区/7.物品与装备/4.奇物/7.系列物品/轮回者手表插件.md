# 轮回者手表插件

轮回者手表插件

描述：  
一个轮回者手表上至多安装一个插件，若购买了多个插件，可在空间内进行切换。

名称：量子通讯  
价格：1000分  
分类：奇物  
本质：科技本质

描述：采用量子技术传递消息，保证零时差无延迟，在任何情况下都可以使用，物有所值。  
效果：安装本插件的通讯器，可以无视传递距离的进行沟通，沟通方式包括文字、语音、图片、视频，要求自设联络频道。

名称：夜视仪  
价格：500分  
分类：奇物  
本质：科技本质

描述：市面上最广为流传的夜视仪，有多种类型的款式（包括但不限制于头盔、眼镜片、望远镜等）  
效果：持有者视为具有黑暗视觉来计算视野。

名称：观星仪  
价格：500分  
分类：奇物  
本质：科技本质

描述：市面上座广为流传的神秘学道具，有水晶球，塔罗牌等多种形式提供装神弄鬼的选择。  
效果：持有者可随时进行占卜天象，推测出未来五天的天气走向。

名称：真.观星仪  
价格：1000分  
分类：奇物  
本质：科技本质

描述：观星仪的升级版，不光可以装神弄鬼，而且真的可以进行一些特定的仪式以得到效果。  
效果：持有者在占卜天象后，可以通过引导使天气改变，但前后天气必须接近，如阴天改为下雨。

名称：窥灵镜  
价格：1000分  
分类：奇物  
本质：科技本质

描述：能够窥视宇宙女鬼……啊不，是灵体幽魂等存在的磨制镜片，通常被打造成便于携带的手镜。  
效果：持有者视为具有灵感视觉。

纳米喷雾器-增值  
价格：1000分  
分类：奇物  
本质：科技本质

描述：市面上的稀罕货，只需要喷洒就能够使植物飞速增长，是如今各大农业产物的好帮手。  
特殊的是，本物品虽然是喷雾剂但其实纳米机器会自循环所以不会用尽，如果市面上售卖仅限24小时的类型都是因为公司为了多赚钱下的程序设计。  
效果：可催生植物迅速增长，以肉眼可见的速度长大，但植物的体积不超过10。

名称：纳米喷雾器-水下  
价格：1000分  
分类：奇物  
本质：科技本质

描述：市面上最广为流传的产品，即使是旱鸭子只需要轻轻一喷，就能够如同鱼儿一样在水下随意游动。  
效果：持有者获得等同基础移动速度的游泳能力，并具有水下呼吸能力，但不提供抵抗水压效果，且只有一人份。

名称：纳米喷雾器-急救  
价格：1000分  
分类：奇物  
本质：科技本质

描述：市面上最广为流传的产品，能够迅速止血，稳定伤势，并提高手术效力  
效果：持有者伤势自动稳定，在接受长线医疗时，所需的长线治疗时间减半。
