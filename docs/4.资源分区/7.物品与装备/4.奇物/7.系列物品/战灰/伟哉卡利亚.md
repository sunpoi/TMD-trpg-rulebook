# 伟哉卡利亚

伟哉卡利亚

  
价格：C+1250  
描述：《艾尔登法环》世界中存在的能赋予武器属性，战技的战灰之一。在取得时需要选定自身持有的一个武器，将战灰赋予其上。只有在装备那个武器时，才可以使用战灰的战技。  
这个战灰可以附着在所有的剑类武器上。  
一柄武器仅可以被赋予一个战灰。战灰所选定的武器可以在一次长休时进行变更。  
这个战灰可以通过“战灰：卡利亚大剑”差价升级而来。  
【魔力】：战灰所附带的属性之一，每在智力上具有一点附加成功，就在附加了本战灰的武器的攻击上获得1点器械加值和1点破魔。  
【战技：伟哉卡利亚】：喻为卡利亚王室象征的战技。将剑身化为魔力大剑，再向下挥舞。借由蓄力发动，能提升威力。  
以一个标准动作，消耗1点意志，你凝聚出魔力构筑的大剑，以手中的武器进行一次白刃攻击，这次攻击额外获得5米射程，并且具有10点威猛和【法破】特性，如果你的威猛在将对应防御破除殆尽后仍有剩余，你可以将至多5点威猛转化为dp。  
你可以在这次攻击之前消耗一个移动动作，进行一次蓄力，还会使得这次攻击具有4dp的能量加值和【余波】特性。
