# 狩猎巨人

狩猎巨人

  
价格：D+1000  
描述：《艾尔登法环》世界中存在的能赋予武器属性，战技的战灰之一。在取得时需要选定自身持有的一个武器，将战灰赋予其上。只有在装备那个武器时，才可以使用战灰的战技。  
这个战灰仅可以附着在不造成钝击伤害的近战武器上。  
一柄武器仅可以被赋予一个战灰。战灰所选定的武器可以在一次长休时进行变更。  
【优质】：战灰所附带的属性之一，每在力量和敏捷上合计具有6点属性，就在附加了本战灰的武器的攻击上获得1点器械加值。

  
【战技：狩猎巨人】：挑战体型比自己大的敌人而有的战技。压低身形，稳住姿势，向前踏出，接着顺势由下而上突刺敌人。  
以一个标准动作，消耗1点意志，你以手中附加了本战灰的武器进行一次攻击，这次攻击中，你的攻击造成突刺伤害，如果本身武器造成突刺伤害，则会额外在攻击中获得1dp技艺加值。  
这次攻击额外具有5点威猛，并且附带C级别的【击飞】效果。
