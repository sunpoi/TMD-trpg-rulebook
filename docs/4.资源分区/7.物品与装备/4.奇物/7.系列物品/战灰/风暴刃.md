# 风暴刃

风暴刃

  
价格：C+1000  
描述：《艾尔登法环》世界中存在的能赋予武器属性，战技的战灰之一。在取得时需要选定自身持有的一个武器，将战灰赋予其上。只有在装备那个武器时，才可以使用战灰的战技。  
这个战灰可以附着在所有的剑类武器上。  
一柄武器仅可以被赋予一个战灰。战灰所选定的武器可以在一次长休时进行变更。  
【优质】：战灰所附带的属性之一，每在力量和敏捷上合计具有6点属性，就在附加了本战灰的武器的攻击上获得1点器械加值。  
【战技：风暴刃】：史东薇尔的失传战技。将武器缠绕风暴刃，朝前方释放风暴。可以连续施放。  
以一个整轮动作，你以手中的武器进行一次白刃攻击，这次攻击额外获得25米射程，并且具有5点高速。由于风暴带来的冲击，本次攻击还会具有【余波】特性。  
此外，风暴刃可以进行连环的射出，增强对于敌人的压制力。因此只要上回合你使用过了风暴刃，本轮你继续使用风暴刃攻击时，你的武器伤害视为上升6L，并且在以风暴刃进行的攻击上获得3点附加成功，想要维持这个效果，你不能以风暴刃以外的方式攻击。
