# 荒野行者

名称：荒野行者  
类别：系列奇物  
本质：自然本质  
购买前提：拥有和平使者柯尔特

名称：“斯特森”式的宽沿高顶毡帽  
类别：奇物  
价格：C+1000  
描述：这种帽子顶部向下凹陷，而两侧帽沿则稍稍往上翻翘。除了戴上它的人显得潇洒干练外，还具有多方面的实用功能，不仅可以遮挡风霜雨雪、烈日冰雹，还是现成的水瓢、饭碗、枕头。  
除此之外，它还能为荒野行者提供一个机会，绝处逢生的机会。  
效果：  
每场景一次，当荒野行者面临致死攻击时，可以反射动作令敌人错误判断荒野行者的要害，在面对这次攻击时，荒野行者视为进行了一次招架并移出攻击范围，这之后这顶毡帽将会被该攻击给击落，直至场景结束才可拾起。当这顶毡帽被佩戴时，佩戴者每高于初始相貌5点且总计不高于40点时，就可以获得2点在风度检定上的表现加值。

名称：印花方巾  
类别：奇物  
价格：D+500  
描述：花方巾大都用棉布或丝绸制作，底色比较单一，主要有红色，也有蓝色和黑色，上面印着各种小图案，色彩鲜艳图案美丽的丝绸印花大方巾最为冒险者所钟爱。印花方巾几乎成为荒野行者的象征，它的用途十分广泛，擦汗、洗脸、包扎伤口、遮挡灰尘、传递信号，同时，它亮丽的颜色，也是富有个性的装饰。荒野行者们对自己的方巾都倍加珍惜，每到有水的地方，他们都要把方巾仔细清洗干净，然后顶在头上或握在手中，策马前行，让风吹干。荒野行者死后，同伴都要在其脸上盖上一块他生前最喜爱的干净的印花方巾，使他在另一个世界的孤寂旅途中仍有最忠实的伙伴。  
效果：  
这件印花方巾会为荒野行者表达、运动、社交、求生方面的检定带来2DP表现加值。  
同时这片方巾可以为荒野行者免疫C级别环境来源的负面影响。

名称：幸运币  
类别：奇物  
价格：D+2000  
描述：这是一枚颜色驳杂的黄铜硬币，似乎是原料被随意敲击而制成的。  
正面潦草的刻画着女性面庞的笑颜，背面则是一个凸起的环形轮廓，中心有一个点样凹槽。  
荒野行者拾起血泊中的左轮，扳开转轮取出一枚致命的哑弹。  
效果：  
在战斗开始时你投掷一个rd2，若出目为1，则你在为【Clover】选择幸运数字时，可额外选择1个幸运数字。  
当荒野行者面临致死危机时，他可以以一个迅捷动作透支下一回合的标准动作对危机来源进行一次攻击，这个行为具有打断性。  
若是在这次攻击中他选择触发【Clover】，则在选择幸运数字时，他可以额外选择2个。

名称：一块怀表  
类别：奇物  
价格：D+500  
描述：这是一块镶嵌有美丽女性脸庞照片的怀表，打开表盖就能看到照片，上过发条后还会有清脆的音乐声响起直到发条走时结束  
效果：  
荒野行者可以以一个移动动作打开这枚怀表的表盖，清脆悠扬的音乐会在这个场景中回荡。荒野行者需选择一名在射程范围内的角色，向其发起决斗挑战，对方需以意志豁免对抗DC为10的标准，若失败，则这个场景内，双方只能相互攻击直至其中一名角色死亡。怀表的持有者对被选中者的攻击视为获得3DP士气加值。则视为一个C级心灵来源的胁迫效果。
