# 龙灾之下

名称：赴死者  
类别：奇物（鞋靴）  
本质：魔幻本质  
价格：B+2500  
特性：【赴死】【归根】  
描述：赴死者是一双黑色长筒靴，上面占满了泥点，穿戴者总是会闻到一股味道，具体味道跟穿戴者的家乡有关。  
“那个地方已经被龙毁灭了。”  
“但那是我的家乡。  
效果：  
【赴死】：归乡途，亦是赴死旅。每场影片一次，赴死者可以为穿戴者抵挡B级及以下任何来源的效果，这是一个完美来源的免疫效果。  
【归根】：除了穿戴者，没有单位能够脱下赴死者，当穿戴者死去，可以选择让赴死者发动以下能力，但这会使得作用在其身上的其他复活效果失效，但这并不为一个不可复活效果。  
1.赴死者将会强行让穿戴者遗体保持站立的状态，同时赴死者将与遗体所在位置最近的稳定地面位置长出植物类的树根固定。  
2.赴死者现在无法被任何方式损坏，直到影片结束回到主神空间后失去该效果，赴死者视为穿戴者的100％完整遗体，且在暮死规则中按受到严重伤害和异常点都为0计算。

名称：咒织圣衣  
类别：防具-轻甲  
本质：魔幻本质  
价格：B+2000  
盔甲防御 :17/17  
特性：【不再遗忘】【飞沙走石】  
描述：咒织圣衣是一件土色的麻布织的长袖布衣，上面布满了紫色的魔力丝线的花纹。  
“龙王的仆从将龙血灌入大地，举行仪式”  
“掌管大地的土元素自此失去了自己的记忆，自己的名字，在本属于自己的大地上游荡”  
“直到一只队伍到来”  
效果：  
【不再遗忘】：你的回忆检定获得5点器械加值和1点附加成功，且免疫A级及以下的混乱效果，大地的守护者回忆起了自己的真名，不再迷茫，不再遗忘。  
【飞沙走石】：你可以在任何的土石风沙间移动，你可以在任何硬度为3以下的固体材料中进行正常的移动，且现在你的视觉侦测方式能过透过土石之间，这是一种【透视】效果。在战斗中若你在自己的回合中移动过，风沙将掩埋你的身型，直到你下一轮行动开始前免疫擒抱状态。

名称：誓约之戒  
类别：奇物（戒指）  
本质：魔幻本质  
价格：C+1000  
特性：【誓约】【誓言不绝】  
描述：誓约之戒是一环赤金色的戒指，在中间雕刻着一段文字，能够为穿戴者提供增强意志的力量  
“我，...在此立约”  
“我们将铭记过去的教训”  
“开辟未来的道路”  
“我们将团结一心，消灭永恒的仇敌”  
“无论遭遇何种困难，无论承受何种苦难”  
“我们都将与龙族斗争到底”  
“我们将放下彼此的争端，不计自身的得失”  
“为我们共同的未来”  
“献上我们的忠诚，我们的生命，我们的灵魂”  
“我们将自己的名字，用我们的鲜血，写在大地之中”  
“共同的意志，共同的真理，共同的思念，共同的誓言”  
“以吾之名，立下盟约”  
“吾族未绝，则盟约不绝，吾友未灭，则盟约不灭”  
效果：  
【誓约】：所有立下过结盟约定或有契约的单位，若其在第一次遇见持有戒指者时，基础好感不低于冷漠，则其好感针对持有戒指者提升1级，最高不会高于友善。  
【誓言不绝】：只要持有者具有至少1点传奇决心，持有者的决心属性相关检定就会获得1点器械加值。当持有者的生命值与意志力池皆不为满时生效，持有者在进行消耗1点意志力的行为时可以从该饰品中抽取力量代替，每轮至多抽取1次，该饰品至多可以抽取持有者的传奇决心+1次，消耗的次数可以在一次长休中恢复，当持有者执行可以回复意志力的方式时，也可以选择以1：1的方式复原该效果的触发次数。持有者受到使自己意志力流失的效果，也会优先减少该效果的触发次数来抵抗。

名称：陨灭之石  
类别：奇物（戒指）  
本质：魔幻本质  
价格：C+1000  
特性：【咒灵之躯】【告死】  
描述：陨灭之石是一块有着墨绿色花纹的类似戒指状的石头。  
“咒灵无法被杀死，只能被驱逐，用神圣的能量去驱逐它”  
“可惜人类并不明白这一点”  
“他们对自己掌握的力量太自信了”  
“咒灵能通过漫长的时间，一点点侵蚀一个地区，即使它的形态被打散，但它的危险却加剧了。遍布到洞窟的每一个角落。”  
“这座不可攻陷的堡垒还是沦陷于死亡的诅咒”  
效果：  
【咒灵之躯】：你具有4dr/银，基于你的偏斜防御。  
【告死】：你的攻击如同咒灵一般带来死亡，选择你的一种攻击方式，当投掷一次该攻击方式的攻击检定前，投掷一次自然十面骰，出目为8、9、10时，你的本次攻击检定加骰等级提升1级，至多为8加骰，额外附带等同你感知属性一半的破魔。
