# 爱莲娜的圣钉

名称：爱莲娜的圣钉  
价格：S+16000  
本质：特异本质  
位置：奇物  
描述：  
这是一个基督教信仰导向物品。  
使徒时代最后的遗香，圣者爱莲娜的圣钉，这枚3.5英寸的染血铁钉，代表着其所受到的苦难与主给予的不死祝福!  
这枚给与爱莲娜近乎不死之身的圣钉伴随了她四处传道的苦难一生，无尽的痛苦是信仰的铁证？亦或是神的考验？  
使用者以一个移动动作将这枚圣钉插入心脏，随后无尽的荆棘将从使用者的心脏涌出，吞噬血肉，淹没骨骼，直到只剩一副皮囊。你将成为神的使徒！神的行者！以及....神的怪物。

【苦难的再生者】：将圣钉插入心脏后，象征痛苦的荆棘将遍布使用者全身，正如同朗基努斯之枪的穿透一切，圣杯实现任何愿望，任何一件圣物都有着其“传奇的本质”，而爱莲娜圣钉的“传奇的本质”便是不死的再生性。每当每当你受伤，苍白带血的枝条将从伤口处涌出，如同榕树一般扭曲的生命力攀附凝结，顷刻间化作新血肉，肢体..与头颅。而当你被彻底杀死时，他人会惊讶的发现你尸体内没有一丝鲜血，身体如同陶瓷般碎裂，化作渺小的灰随风散去。

特性：【苦难的再生者】  
效果：  
你的生物类型始终视为【人类】，并在相关判定上额外具有【圣人】特质。

【苦难的再生者】：从现在开始，你具有等同耐力属性点快速医疗（医疗点12\*耐力点），同时，你还具有传奇耐力的异常点治疗（医疗点120点）。只要心脏处的圣钉依然存在，你便绝不会轻易死去，每次战斗结束时，将圣钉提供的所有治疗能力的医疗点恢复到上限。当使用者死亡时，圣钉会以你的心脏为支点，10轮之内成长一颗参天的巨木，枝条扭曲蔓延向上，将你的尸体将会被包裹其中，在24小时后，巨木枯萎，你则在树芯中得以重生。这是【S级命运来源】的复活效果。

巨木具有S级物品一样的硬度，结构值为你的血量上限\*1，若巨木在24小时内被击破，你则被彻底的杀死，这一奇物也被永久性破坏。

任何一人都可以通过一次dc为66的学识检定发现这枚圣钉的意义与作用，随后便可进行一次针对心脏的部位近战攻击，你仅可以以敏捷+武技对抗，对方的任何一点成功数都视为将圣钉拔出，而你则理所当然的失去爱莲娜圣钉带来的全部效果，并立刻承受等同于耐力值的恶性伤害。  
ST选开：在地球中世纪或者基督教盛行的地方，学识鉴定可以变为dc6。
