# 3\. 仙烟丹

消耗品名称：仙烟丹  
资源分类：消耗品  
本质：魔幻本质  
价格：C+1000  
数量：3颗  
生效轮次：使用者的传奇风度\*轮/场景结束取低

描述：  
使服用者获得使自身灵力或任意东方背景类型能量外放并变成可笼罩自身的灵烟的能力，外放灵烟产生的消耗近乎于无。

效果：  
获得等级与灵力或能量池相等但最高为B级的魔幻本质反侦察能力。  
修仙世界效果：持续时间提升至（传奇风度+1）\*轮。
