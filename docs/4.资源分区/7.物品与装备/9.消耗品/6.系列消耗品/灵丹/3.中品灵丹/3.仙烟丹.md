# 3.仙烟丹

仙烟丹  
C+1000  
数量：3颗  
描述：使服用者获得使自身灵力或任意东方背景类型能量外放并变成可笼罩自身的灵烟的能力，外放灵烟产生的消耗近乎于无。  
生效轮次：使用者的传奇风度\*轮/场景结束取低  
效果：获得等级与灵力或能量池相等但最高为B级的魔幻本质反侦察能力。  
修仙世界效果：持续时间提升至（传奇风度+1）\*轮。
