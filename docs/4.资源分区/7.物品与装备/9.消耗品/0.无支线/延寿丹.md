# 延寿丹

  

延寿丹  
1000  
"囊中有药逢人度，腹内新诗遇客吟；丹粒能延千载寿，漫夸人世有黄金。”  
效果：服食后，即可增寿千载（广告效果），固龄当下。  
这是C级【免疫衰老】的效果 且服用者寿命不会随时间而流逝。
