# 延寿丹

消耗品名称：延寿丹  
资源分类：消耗品  
本质：神话本质  
价格：1000分

描述：  
"囊中有药逢人度，腹内新诗遇客吟；丹粒能延千载寿，漫夸人世有黄金。”  
服食后，即可增寿千载（广告效果），固龄当下。

效果：  
服用后，获得C级【免疫衰老】的效果，且服用者寿命不会随时间而流逝。
