# 聚灵石

聚灵石  
500  
普普通通的灵石，看起来就是一块微微散发白气的手掌大小的普通石头，因为蕴含灵气所以经常在修仙界作为货币使用。在灵石上刻画聚灵阵，就能将灵石作为源源不断补充灵气的容器，在某些修仙界是常用的道具。  
携带聚灵石后，只要身处现实世界（梦境世界之类的地方可能无法汲取灵气），无需任何行动即可每小时进行一次你的灵气能量池关键属性检定，恢复成功数点能量。因此方式恢复的灵气会计入每日恢复上限。  
当你以此方式恢复了10次灵气后，这一物品会因灵气耗尽自动失效。
