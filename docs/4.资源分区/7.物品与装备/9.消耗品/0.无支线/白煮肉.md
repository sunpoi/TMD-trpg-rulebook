# 白煮肉

消耗品名称：白煮肉  
资源分类：消耗品  
本质：自然本质  
价格：250分

描述：  
一块平平无奇的肉，来自于各种禽类或者兽类，出自一个新手厨师，尽管他只是焯了下水而已。  
“我觉得沾点酱油也能吃。”  
除了肉类的荤腥味外没有别的味道，但一股热流从你体内升起！

效果：  
只能在战斗外使用，可以食用2次。  
吃下后你恢复3点内力。
