# 白煮肉

白煮肉  
250  
一块平平无奇的肉，来自于各种禽类或者兽类，出自一个新手厨师，尽管他只是焯了下水而已。  
“我觉得沾点酱油也能吃。”  
除了肉类的荤腥味外没有别的味道，但一股热流从你体内升起！  
只能在战斗外使用，可以食用2次  
吃下后你恢复3点内力。
