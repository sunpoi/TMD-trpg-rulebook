# 醇香的桂花酒

消耗品名称：醇香的桂花酒  
资源分类：消耗品  
本质：自然本质  
价格：500分

描述：  
琥珀色的桂花酒，入口清新醇和，天然的桂花香带着丝丝绵甜的酒味充满口腔，回味无穷。饮用后使人回味无穷。  
选开：在一些武侠或者仙侠世界中，这瓶酒可以送给一些剑客增加好感度，至于为什么玩剑要喝酒？我怎么知道？据说在仙剑奇侠传了里面可以换取蜀山剑仙的秘籍？

效果：  
摄入方式：口服。  
这酒只能在战斗外使用。  
你在喝下此酒后，必须过一个DC10的强韧，如果没有成功，则直接醉倒在地，这视为一个S级创伤来源的效果。  
选开：将瓶酒送给一些剑客，态度看情况提升一级。
