# 回城卷轴

回城卷轴（消耗品）  
250  
英雄真正需要的东西。 

一捆记载了回城魔法的卷轴，使用一次就会被消耗殆尽。  
需提前花费10分钟在某地设置一个坐标，  
随后激活回城卷轴的时候，在10分钟到1小时不等的时间后（根据距离而定），  
该传送具有一个最大距离，最大距离约为一小时的引导后从城市一段到另一端，或者一小时的引导后传送到相邻城市，  
你会完成施法，这会在你面前展开一个传送回坐标的传送门，这是一个D级的旅行效果。

远行靴  
占据鞋袜位  
D+500

“足生双翼，上天入地”

这是一双有着神奇魔力的靴子，在兑换这双靴子之后，你的基础移动速度上升敏捷属性\*1M。  
你可以下意识知道你的所有轮回者队友的位置（除非对方主动屏蔽了你），  
选择你使用回城卷轴时可以选择轮回者队友的身边为传送的落点坐标，并且回城卷轴现在不再消耗，但仍承受回城卷轴的距离限制。

你也可以选择让轮回者队友的位置在轮回者手表上以小地图的形式显示，若如此做，那么你们还能通过轮回者手表进行交流。

——额外支付D+500将该奇物差升至C级。

在使用回城卷轴时，你能选定的传送范围从轮回者队友身边扩大至只需要知道具体坐标，  
原先的传送最大距离扩大，当前你可以从大陆内的一端传送至另一端，这是一个C级的旅行效果，  
你的基础移动速度上升敏捷属性\*2+8M。
