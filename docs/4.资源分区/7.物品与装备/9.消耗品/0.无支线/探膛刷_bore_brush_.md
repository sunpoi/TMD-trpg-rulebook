# 探膛刷（Bore Brush）

消耗品名称：探膛刷（Bore Brush）  
资源分类：消耗品  
本质：魔幻本质  
价格：150分

描述：  
这种螺旋形金属杆上有着经过炼金处理的刷毛，其能溶解早期或近代火器枪管中的污垢和弹渣，并在抽出时在枪管内壁上留下一层薄薄的保护层。

效果：  
这仅对使用火药推进的枪械有效，无法对电磁武器，能量武器或者有魔力等其他能源推动的枪械起效。使用探膛刷是一个花招动作，其保护层能使该枪械的下一次射击获得+3器械加值。该效果持续1场景或直到枪械下一次开火为止，每场景只能使用一次探膛刷。  
一个探膛刷可以使用3次。
