# 手卷雪茄

消耗品名称：手卷雪茄  
资源分类：消耗品  
本质：自然本质  
价格：250分

描述：  
四根装的主神特供版提神用品，看起来像是18世纪美国西部某个土工作坊里的手工品，用白磷火柴划着便会散发出呛人的烟气。

效果：  
一份包含四根。  
用一个花招动作便可以将一根卷烟叼在嘴里并点着，它将持续燃烧一个场景，在任意一个场景结束时将熄灭且无法再点燃，在此期间内，使用者将获得至多3点意志豁免上的检定加值。
