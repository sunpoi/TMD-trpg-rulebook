# 手卷雪茄

  

手卷雪茄  
250  
四根装的主神特供版提神用品，看起来像是18世纪美国西部某个土工作坊里的手工品，用白磷火柴划着便会散发出呛人的烟气。用一个花招动作便可以将一根卷烟叼在嘴里并点着，它将持续燃烧一个场景，在任意一个场景结束时将熄灭且无法再点燃，在此期间内，使用者将获得至多3点意志豁免上的检定加值
