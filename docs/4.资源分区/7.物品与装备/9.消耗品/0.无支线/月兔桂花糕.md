# 月兔桂花糕

消耗品名称：月兔桂花糕  
资源分类：消耗品  
本质：神话本质  
价格：500分

描述：  
一碟花香馥郁的糕点，为纪念秋夜满月的赠礼。赏月时，不妨摆上这碟点心，倾听一个古老悠远的传说。它关于月中玉兔，关于禳灾送吉，它作为一个节候的祝福被主神挂牌售卖。  
“桂花糖浆风味浓郁，请酌量添加。”  
理所当然的，轮回者并不会致死量的糖浆致死。

效果：  
一盘月兔桂花糕可以享用三次，在战斗外享用时可以选择加入致死量桂花糖浆并食用，然后让自身在接下来的一场景期间获得快速医疗1（医疗点12）。

变体：你可以在购买时选择枫糖口味，枫糖口味的月兔桂花糕（？）在享用后，效果改为每轮恢复1点任意一个通用能量池，直到本次累计恢复6点为止，每日恢复上限依旧为3倍能量池上限。
