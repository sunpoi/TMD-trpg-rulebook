# 龙肝瓜

消耗品名称：龙肝瓜  
资源分类：消耗品  
本质：神话本质  
价格：250分

描述：  
《洞冥记》  
有龙肝瓜，长一尺，花红叶素，生于冰谷。所谓冰谷素叶之瓜。仙人瑕丘仲采药，得此瓜，食之，千岁不渴。瓜上恒如霜雪，刮尝，如蜜滓。及帝封泰山，从者皆赐冰谷素叶之瓜。

效果：  
当人物吃下一颗龙肝瓜后，在余下千年的岁月中，都不再口渴，可以无需要喝水便能维持生命。
