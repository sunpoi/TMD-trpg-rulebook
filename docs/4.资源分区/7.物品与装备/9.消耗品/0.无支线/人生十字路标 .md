# 人生十字路标

  

人生十字路标  
  
  
无支线奇物/消耗品  
1000  
由指向不同的银色和黑色箭头组成的交叉路标状项链。因为形状的原因会戳到自己，所以一般被当作挂饰使用。  
  
当情绪异常进入毁灭状态时，进行一次意志豁免，DC为自身决心值。  
若检定成功，消去全部进入毁灭状态的情绪异常点数（即变为距进入毁灭性后果还差1点的状态）。奇物变成【银色涅槃之路标】。  
若检定失败，受到3点永久的决心属性伤害。奇物变成【黑色崩毁之路标】。  
  
“彷徨在人生的十字岔路，崩毁与涅槃只差一步。”  
  
  
  
银色涅槃之路标  
项链/护符  
由【人生十字路标】的黑色箭头变为银色变化而来。  
决心检定+2。  
每日一次，你可以将一次决心检定重新掷骰，以新的结果取代原先的结果。  
  
“只要心不迷惘，条条大路康庄。不论道阻且长，向着旭日起航。”  
  
  
  
黑色崩毁之路标  
项链/护符  
由【人生十字路标】的银色箭头崩坏，仅剩黑色箭头变化而来。  
你无法脱下【黑色崩毁之路标】，你在对抗影响心灵效果的掷骰中-2DP，每日太阳落山后的第一次掷骰双骰取低。  
这是一个C级诅咒来源的效果。  
注：是否允许以一次无意义掷骰消耗这次双骰取低，以st判断为准。  
  
“在黑暗之中辗转反侧，伴随深沉的梦魇入眠。自我折磨吧，直到生命终结的那天。”
