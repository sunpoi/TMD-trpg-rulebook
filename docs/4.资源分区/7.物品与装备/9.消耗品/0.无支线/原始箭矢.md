# 原始箭矢

原始箭矢  
分类∶消耗品  
价格：500/3支  
词条∶破甲5  
简介∶部落里数一数二的巧手制作，来之不易

名称∶螺壳箭矢  
分类∶消耗品  
价格：500/3支  
词条∶射程增加20米，但是如果无法击碎对方盔甲防御，那么对方不会受到伤害。  
简介∶由吃剩的完整螺壳制作，轻巧的质量和流线的外形使得射程更远

名称∶碳化箭矢  
分类∶消耗品  
价格：500/3支  
词条∶破甲1 高速3  
简介∶为了硬度而降低韧性的产物，优点是制作方便

名称∶鲨齿箭矢  
分类∶消耗品  
价格：D+500/9支  
词条∶造成2点不可豁免的流血，高速2  
简介∶珍贵的鲨鱼牙齿，由临海部落的巧手制作

这些箭矢并没有什么不同，除了制作方便之外。  
如果你使用造物，所需的材料只有木棍，鸟类羽毛，石头等制作而成，简单快捷，且造物dc减少2点。  
你甚至可以在副本中制作，前提有足够的手艺和说服st。
