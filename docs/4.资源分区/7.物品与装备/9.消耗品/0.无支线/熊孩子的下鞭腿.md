# 熊孩子的下鞭腿

熊孩子的下鞭腿  
价格1000  
消耗品  
一本普普通通的技能书，拍了即学，不会提供任何属性加值。  
使用完3次后，该技能直接消失。  
介绍：源于某本小说主角曾经获得过的消耗型技能  
限制：对方必须是人形生物  
你可以消耗一个移动动作选择一个触及范围内的一个人形单位，与对方进行一次【力量/敏捷】+【武技：肉搏相关专业】/【运动：平衡】的检定对抗，失败者会进入倒地状态。  
本次绊摔无视体积带来的优势与劣势  
本次绊摔鉴定获得2dp的鉴定加值的同时，获得等同于感知附加两倍的dp士气加值
