# 大脑冷冻式冰淇淋

消耗品名称：大脑冷冻机式冰淇淋  
资源分类：消耗品  
本质：特异本质  
价格：10分/盒

描述：  
这是一盒有许多彩色雪球糕组成的体积2的大冰淇淋！  
该物品在打开后必须在2小时内吃掉！不然它会融化！当然如果零度以下的环境能保存更久。

效果：  
食用一盒冰淇淋是一个标准动作，你每吃一个这种冰淇淋都可以将一点严重伤害回复到完好。  
但每次都需要投掷一个rd2，如果出1则无事发生，出2则会受到冰冻诅咒！你的智力属性会受到2点属性伤害，持续一天。
