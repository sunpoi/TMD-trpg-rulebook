# 法力回复技巧

抄录页：法力回复技巧  
500  
魔幻本质

  
前置条件：使用该魔法道具需要使用者至少拥有4智力与3学识，否则，展开抄录页将是一个需要消耗1点能量的标准动作。

  
描述：一张用麻绳捆起来的，手写羊皮纸抄录页，纸上的文字似乎是某个人匆匆抄录下来的，所用语言毫无原型和具体结构，也无法读懂，主神保留了其无法阅读的特性，因为一旦将其翻译成可以被看懂的内容，这张失去神秘性的抄录页将会彻底失效。  
而且就算翻译了，据说上面写的东西也狗屁不通。

  
效果：以一个迅捷动作，将抄录页展开对准自己或者10米范围内另一个目标并念出口令“菲密斯概论”，抄录页上的文字会闪烁出一道蓝色光芒没入其体内，吸收了蓝色光芒的人将会立即得到4点能量恢复，这可以作用于任何已开启的能量池。  
该魔法道具在使用3次后会自动焚毁，无法被修复。
