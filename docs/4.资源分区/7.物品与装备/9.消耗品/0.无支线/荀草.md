# 荀草

消耗品名称：荀草  
资源分类：消耗品  
本质：神话本质  
价格：200分

描述：  
《山海经》  
青要之山…有草焉，其状如葌，而方茎黄华赤实，其本如藁本，名曰荀草，服之美人色。  
荀草，或叫苞草，山海经中的草，菅似茅，方茎，黄花，赤色果，服之可以美容颜。

效果：  
服下荀草，便可免疫所有不超过D支线的的肌肤疾病，并使相貌提升1点。
