# 失嗅药

失嗅药 160/片  
原典：《粪海狂蛆》  
描述：降低嗅觉和味觉的药物，让人从粪臭中暂时解脱出来。  
效果：此效果持续时间内，角色失去嗅觉和味觉，包括灵敏嗅觉在内的基于嗅觉的侦测能力将会失效，同时亦会失去基于味觉的侦测能力（如果有的话）。  
作为好处的，当角色承受恶心点数的时候，第1点不会带来减值，但计算是否进入重度和毁灭时依然如常计算。  
在持续期间内任意时刻，以一个自由动作支付1点意志力，可以使本场景内额外忽略2点，最多支付1次。  
“虽说是可以让人闻不到粪味的药，但其实也是用大便做的呢，真是讽刺。”——博士。（此段建议跟蛮荒最后那条一样涂黑）
