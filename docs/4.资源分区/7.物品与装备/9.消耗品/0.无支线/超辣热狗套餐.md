# 超辣热狗套餐

  

超辣热狗套餐  
500  
特异本质  
  
这是一个超级辣的热狗，辣得要命！  
附赠一瓶冷藏牛奶，给追求刺激的你一点小小的安慰。  
  
吃下热狗是一个标准动作，它实在是太辣了！所以吃下热狗的人会承受一点不可豁免的剧痛点数。  
吃下热狗以后可以迅捷动作张嘴喷火，对面前一个锥形范围内的所有单位造成8L火焰伤害，反射豁免，射程15米。  
  
战斗结束后可以喝下牛奶来恢复超辣热狗造成的剧痛点数（仅能用于恢复来源为超辣热狗的剧痛点数）。
