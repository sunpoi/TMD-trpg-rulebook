# 跑鞋肉（生）

消耗品名称：跑鞋肉（生）  
资源分类：消耗品  
本质：特异本质  
价格：500分

描述：  
一块和跑鞋形状差不多的肉块，上面白色的脂肪甚至勾勒出X耐的标志！

效果：  
你需要进行一个DC5的厨艺烹饪！只要成功了，那么这块肉会变成——跑鞋肉排！  
【跑鞋肉排】：在战斗中一个自由动作，吃下去后获得35点移动速度，持续一个场景。
