# 元素沟通药丸

消耗品名称：元素沟通药丸  
资源分类：消耗品  
本质：魔幻本质  
价格：500分

描述：  
这是一瓶9颗装的药丸，拥有与对应元素进行短暂沟通的能力，以减少施法消耗。

效果：  
这瓶药丸共有9颗。  
吃下一颗药丸，仅限于使用魔力池进行施法才会正常生效。在当前场景中，每轮的第一次施法消耗将会自动减少2点，最低不会减少到0。在同一场景中重复食用并无叠加效果。

药物疲劳：  
一档：每2轮的第一次施法减少2点。  
二档：每3轮的第一次施法减少2点。  
三档：每3轮的第一次施法减少1点。  
四档：没有任何作用。
