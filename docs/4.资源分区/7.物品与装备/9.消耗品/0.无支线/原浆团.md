# 原浆团

消耗品名称：原浆团  
资源分类：消耗品  
本质：特异本质  
价格：500分/3枚

描述：  
由蔓延在土地之上的原浆精炼过后得到的团子，对人类来说仅能果腹，但对于成为非人的生物来说有恢复生命的能力。

效果：  
在战斗外吞服一枚团子后接下来一场景使用者具有快速维修1（12医疗点）。
