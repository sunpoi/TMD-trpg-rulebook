# 马草

名称：马草  
价格：250  
本质：特异本质  
分类：消耗品  
描述：  
这是一把干枯的草料，茎秆呈黄褐色，带着细碎的穗头，散发着淡淡的干草清香。凑近细看，每一根草叶的边缘都隐约泛着一层极淡的金色光泽。这是不知名人士从某个农耕文明偷回来的战马用特殊草料

虽然名为马草，但万能的主神让它也能作为正餐被人消化。当你吃下它，会尝到一股带着青草香气的微甜，口感粗糙有些卡嗓。在别人的面前吃下它会让人觉得你的脸看起来有点像马脸，这并不会有什么实际作用，只是有点像而已。

吃下马草后，你能清晰感觉到四肢变得轻快有力，同时，你每一步落地都有一股细微的弹力从脚底涌现，并带着点点金色流光。  
效果：  
吃下马草后，你获得持续1h的5m移速加成，持续1h。
