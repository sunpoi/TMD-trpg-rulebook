# 可以成为mama的专属巧克力

名称：可以成为mama的专属巧克力（马年特供）  
价格：1000分  
本质：特异本质  
分类：消耗品  
描述：  
因为某个神的特性产生的神奇巧克力（fgo的新活动巧克力真就叫这个名字），会让收到这个巧克力的人对赠送者产生“想成为ta「mama」“的想法。  
当你向一个目标送出这份巧克力时，无论对方是否接受，你都会因为对所爱之人送出了这份心意而获得满足感。

效果：自由动作使用，你可以随时向一个目标送出这份巧克力，无论对方是否接受，你都将获得【激昂】效果，持续一场影片。如果你持有技艺【指露为mama】，则被你送出巧克力的单位也可以作为被你选择为「mama」的目标。  
每场影片你只能送出一份此巧克力，对于接受者来说，这份巧克力只是一个没有效果的普通甜点而已。
