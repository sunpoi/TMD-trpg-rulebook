# 七星灯

  

七星灯  
500  
七盏锈迹斑斑的灯，若能摆为北斗星，点亮后主灯七日不灭，也许会发生一些奇迹吧。  
若寿命即将走到尽头，亲手点亮七星灯并且维持七日不灭，则可延寿若一纪(12年)。期间若是灯被任何原因熄灭，则立即受到反噬，寿尽而亡。

“亮，再不能临阵讨贼。悠悠苍天，何薄于我……”
