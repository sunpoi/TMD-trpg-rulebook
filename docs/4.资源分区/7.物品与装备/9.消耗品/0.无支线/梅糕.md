# 梅糕

消耗品名称：梅糕  
资源分类：消耗品  
本质：自然本质  
价格：250分/块

描述：  
这是一块绿色的梅糕，为什么是绿色的？哦，不是因为这是青梅，而是因为它发霉。  
恶心的苦味口感充斥你的喉管和口腔。这会激活你全身的求生细菌（？），为了活下去，这是你为了活下去进行的挣扎。  
特殊的，你也可以在购买时换成绿豆糕，若如此做不会产生任何效果变化，只是发霉的菌丝会变成白色的。

效果：  
以自由动作一口吞下去服用，你将不可避免地受到该场景内无法抵消的2点恶心异常点数，可以和其它来源的恶心点数正常叠加。  
在接下来一个场景中，你的移动速度提升10米，基因锁检定获得额外1点成功数。
