# 梅糕

梅糕  
250/块  
描述：这是一块绿色的梅糕，为什么是绿色的？哦，不是因为这是青梅，而是因为它发霉。  
效果：以自由动作一口吞下去服用，恶心的苦味口感充斥你的喉管和口腔，你将不可避免地受到该场景内无法抵消的2点恶心异常点数，可以和其它来源的恶心点数正常叠加，这会激活你全身的求生细菌（？），为了活下去，你在接下来一个场景中，你的移动速度提升10米，基因锁检定获得额外1点成功数，这是你为了活下去进行的挣扎。

特殊的，你也可以在购买时换成绿豆糕，若如此做不会产生任何效果变化，只是发霉的菌丝会变成白色的。
