# 一盘美味的水饺

一盘美味的水饺  
250  
消耗品  
一种常见的美食，正常来说一卖，卖一盘，一盘8个。寓意美好，令人感慨，也是团圆的象征。  
正常来说这种水饺由蔬菜，不知名的肉类还有面粉制作的粉皮制作而成。制成后放入锅中蒸煮便可吃食，空间售卖的都会是完全煮熟的，放心不是预制饺就对了。

  
战斗外使用  
吃下这盘美味的饺子  
你在半小时内恢复6点严重伤害，3点恶性伤害。
