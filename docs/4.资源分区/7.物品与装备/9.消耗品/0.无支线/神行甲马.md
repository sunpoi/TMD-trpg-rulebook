# 神行甲马

神行甲马  
类型：消耗品  
价格：500  
本质：魔幻  
描述：此乃梁山一百单八将，天罡星之天速星——神行太保戴宗能日行八百里的宝贝。不过这四片甲马比起戴宗所拥有的而言较差，只能发挥一次用处。  
效果：花费10分钟将四片甲马分别贴在左右脚的两侧，而后额外花费10分钟就可以60公里/小时的速度进行旅行，而且甲马还有特别的能力能让你渡水如履平地，持续一日，这是一个无支线等级的旅行特性。当处于旅行状态时，人物默认失去所有动作，不可进入战斗轮（换句话说就是进入战斗则旅行速度失效）。
