# 兵粮丸

兵粮丸  
10  
兵粮丸用红萝卜、荞面粉、麦粉、山芋、甘草、薏苡、糯米粉，全部磨成粉末，浸泡在酒中三年，待酒蒸发后，揉成桃子核一般大小，一天仅吃一粒，就能保证体力口也！

  
你在战斗外或者战斗内一个迅捷动作吃下一颗兵粮丸后，三天不用吃饭，同样也拉不出屎，无法进行正常排泄。  
这意味着，持续时间内你无法正常豁免【恶心】这个异常状态。
