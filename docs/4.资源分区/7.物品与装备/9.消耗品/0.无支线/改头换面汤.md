# 改头换面汤

改头换面汤  
消耗品  
750分  
改头换面汤是一碗根据饮用者的过去经历味道会发生改变的陶瓷碗装着的约一盏量的无色液体。  
主神空间是一个奇妙的地方，你的过去的罪恶，善行，恩与仇，恨与爱，都已经远去。  
那代表着你过去的皮囊为什么不修整一下？  
如果需要，饮下这碗汤吧。  
【改头换面】：饮下改头换面汤后，你的基础外貌会被强制提升为你的当前风度属性等级或5点中取高，你可以重新更改你的样貌，这意味着你可以更改你的立绘或者相貌描述。
