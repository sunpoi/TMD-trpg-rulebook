# 伤药

消耗品名称：伤药  
资源分类：消耗品  
本质：科技本质  
价格：500分

描述：  
购买者可以购买效果相同的变种。例：原素瓶、核子可乐、红药水、急救药。

效果：  
九颗装的小药瓶，在吞服一颗药后的两分四十秒内缓慢恢复使用者12点伤势。  
药物疲劳：  
一档：恢复9点伤势  
二档：恢复6点伤势  
三档：恢复3点伤势  
四档：没有任何作用
