# 伤药

  

伤药  
500  
九颗装的小药瓶，在吞服一颗药后的两分四十秒内缓慢恢复使用者12点伤势。购买者可以购买效果相同的变种。例：原素瓶、核子可乐、红药水、急救药  
药物疲劳：  
一档：恢复9点伤势  
二档：恢复6点伤势  
三档：恢复3点伤势  
四档：没有任何作用
