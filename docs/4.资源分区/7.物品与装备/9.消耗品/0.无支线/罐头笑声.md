# 罐头笑声

  

罐头笑声  
1500分/个  
录音的笑声播出来总是千篇一律的，就像罐头食品吃起来总是同一味道那样，人们将电视节目中播放的那些提前录制好的笑声称为“罐头笑声”。自其发明以来的影史来看，它一半是伟大发明，一半是噪音机器，故其售卖也分为两个版本。  
伟大发明:外观上来看这是一个普通的铁质方形罐头，外包装彩印着“伟大发明”四个字，罐头轻若无物，摇晃时不会发出任何声音。你可以在你社交检定失败时使用这个罐头，使用时你以一个移动动作打开罐头，罐头会迅速释放出一段录好的笑声，此笑声斑驳掺杂，各色年龄性别似乎都有。此时你只需表达出对你刚刚进行的社交内容的否定，则你的社交对象就会将其视为一次无伤大雅的玩笑，这是一个D级的心灵来源效果。并且使用者获得了再进行一次社交的机会，此次社交结果会取代上次的失败结果，但你仍然需要正常花费社交所需要的时间与动作。  
噪音机器:外观上来看这是一个普通的铁质圆形罐头，外包装彩印着“噪音机器”四个字，罐头轻若无物，摇晃时会发出“咚咚咚”的闷响。你可以在战斗时使用这个罐头，使用时你以一个移动动作打开罐头，罐头会迅速释放出一段录好的笑声，此笑声斑驳掺杂，各色年龄性别似乎都有，同时使用者也会不自主的随其哄堂大笑，听到的人会被其突兀且意义不明的冲击感所激怒，将其视为下一次攻击的对象，这是一个C级环境来源的胁迫效果。
