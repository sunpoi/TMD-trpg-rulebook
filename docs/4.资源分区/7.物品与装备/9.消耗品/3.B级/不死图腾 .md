# 不死图腾

不死图腾  
B+2000  
占据护符位  
图腾在古代被视为氏族的保护神和象征，而这块凝聚了众人祈愿的不死图腾所象征的，则是那超越生化的希望。  
这是一块通体金黄方正，上方镌刻着一张奇怪大鼻子方型人脸的黄金图腾，佩戴他你不会获得任何的加成，仿佛这只是一块很普通的装饰品，但是一旦你受到了足以导致你死亡的伤害，这一块图腾会自动跳出来为你挡下这次致命伤害，这被视为一个B级祝福来源的无敌效果。不死图腾在挡下伤害之后将会破碎成一堆绿色的光粒并融入你的身躯,这将立即令你获得仅持续一轮的8点额外临时生命并令你将在当前的场景内再获得4点快速医疗（医疗点48）与B级的免疫环境效果。
