# 崩坏能

  

崩坏能  
本质：特异本质  
  
崩坏能是一种侵蚀性极强的能量，其诞生与消亡对于人类而言是几乎不可观测的黑箱。崩坏能的大量产生让世界文明崩落解体，甚至有人猜测这一能量的背后有着更深邃的崩坏意志在操纵着一切。  
  
非律者或者不具有崩坏抗性的生物接触到崩坏能就会被其侵蚀、变异，人类被侵蚀后会成为丧尸一般的死士，动物被崩坏能侵蚀后一般会变成楼房高的崩坏兽……被侵蚀而没有消亡的个体往往会服从崩坏意志，开始消灭人类。  
  
经过天命和逆熵组织长期的研究和律者展现的一些能力来看，崩坏能可以转换成物质、生成力场、激发能态甚至使用权能……几乎可以说是无所不能的万能神力或者说这本就是世界为了消灭人类而彰显出的诸相。  
  
一般情况下，具有崩坏能池的生物将被“世界的意识”（盖亚等）视作友善；对于“人类的意识”（阿赖耶等）而言，你将被视为邪恶，在某些同人类意识链接紧密的人（带领人类族群抵御外敌的主角等）的眼中，你的形象可能会污浊、怪异或不可名状化。值得注意的是，这些效果都是在极为宏观的层面上生效的，在大部分时候都不会引发有益或有害的影响。  
  
  
决定崩坏能大小的关键属性为风度+沉着（律者）/耐力（非律者）  
风度象征着世界的本源意志对于崩坏能拥有者的亲和力。  
沉着象征着崩坏能拥有者以自身意志产生的对于崩坏能引导与掌控程度。  
耐力则象征着并非世界意志的选中者被崩坏能侵蚀后，凭借自身的身体容受承载崩坏能的上限。  
  
崩坏能的恢复方式：  
1、每次长休息后，崩坏能会恢复至满值。  
2、每进行一次短休息，进行一次崩坏能关键属性检定，恢复成功数点崩坏能。  
特殊的，当剧本背景中，世界意志正宏观的进行针对并消灭人类这一物种的行动时，崩坏能在每个小时都会恢复风度属性点。  
  
崩坏能的元素属性  
你在开启崩坏能时，在以下元素属性中选择一种：  
火焰、冻寒、雷电、力场  
选择元素属性后不可变更。  
  
崩坏能的基本用法：  
当你进行攻击时，你可以自由动作消耗1点崩坏能，令这次攻击的伤害类型转化为你的元素属性。  
  
崩坏能适应  
500  
你开启崩坏能，并获得基础用法。  
  
崩坏能研发  
D+500  
当角色进行攻击时，若这次攻击使用了崩坏能基础用法，则在这次攻击中再获得3点(火焰/冻寒/雷电/纯粹）能量加值与【法破】特性。  
  
如果角色一轮内在同一位置使用了至少3点崩坏能，这一地点在会出现细小的空间裂缝，这是由于空间被崩坏能侵蚀导致的。  
  
  
通用技能树  
  
闪避  
D+500  
受到攻击时，自由动作支付2点崩坏能强化自身，在应对这次攻击时获得4点闪避防御或者反射豁免加值。  
若这次攻击不对你造成任何伤害，立即对敌方造成2点崩坏能元素属性严重伤害。  
  
极限闪避  
D+500  
前提：闪避  
角色现在可消耗2点崩坏能以移动动作准备一次极限闪避。  
极限闪避： 当角色受到攻击时，可消耗反射动作打断性的进行一次移动，距离为敏捷\*2米。此外，在应对这次攻击时，角色如同进行了招架一般，抵消【运动】技能等级点由这次攻击造成的伤害。当进行了极限闪避后，人物需要再次进行准备才可使用。
