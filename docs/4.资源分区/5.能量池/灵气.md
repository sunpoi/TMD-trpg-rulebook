# 灵气

灵气  
分类：东方通用能量池  
本质：自然本质

描述：  
灵气，现代修仙世界观中经常有修仙者吸纳灵气进行修行的设定。发挥了类似货币的作用，将从太阳，从月亮，通过不同功法，从不同的自然环境与自然物质中获取的力量统一成了一种能量。在不同的世界观下这种能量有不同的名字，真气、灵气、仙力等等，将这种主要依靠沟通自然获取的能量统一为灵气，具体的资源中类似的能量是否属于灵气，则取决于制作者有没有写明。

效果：  
关键属性：感知+决心。感知代表汲取灵气的能力，决心代表驾驭灵气的能力。

灵气的恢复方式：灵气是自然界中的能量，只要身处现实世界（梦境世界之类的地方可能无法汲取），无需任何行动即可以每小时感知/6点的速度恢复.必要时你可以打坐的方式主动汲取自然界的灵气，当你进行一次短休息时，你额外恢复决心/6点灵气。灵气一日的恢复上限为能量池上限的三倍。

灵气购买方式：

技艺名称：通脉法  
价格：750  
描述：  
这是一门基础功法，打通体内灵脉，产生容纳与运输灵气的通道，从而能汲取、存蓄灵气。  
效果：  
开启能量池：灵气。  
获得灵气下属能力购买权。

灵气下属能力：

技艺名称：基础神通  
价格：D+500  
描述：  
在某些世界观中修仙者基本都会的神通。  
特性：【灵气护体】【灵气附着】【辟谷养生】  
效果：  
学习此资源如同正常习得了一个D级技艺。  
【灵气护体】：  
每回合一次，在受到攻击时，你可以自由动作使用2点灵气拦截对方的攻势，使你在面对这次攻击时获得3点能量防御。  
【灵气附着】：  
在进行从你发动的从自身相连的物质进行的攻击时，你可以同时消耗2点灵气附着在攻击上，令此次攻击具有【法破】。  
（举例来说，远程操控无人机的攻击因为自身与无人机不相连，无法将灵气传递到无人机上，不满足“从自身相连的物质进行”这一条件；骑乘生物载具时生物载具使用自身动作发动的攻击则不满足“你发动的”这一条件。这两种情况均不能进行灵气附着。一般的施法、肉搏、投掷、射击、白刃攻击，都满足这两点，所以一般均可进行灵气附着。）  
【辟谷养生】：  
你可以将消耗2点灵气，使自身一天之内无需呼吸、饮食、休息。
