# 查克拉

查克拉  
类型：能量池  
本质：魔幻本质

描述：  
查克拉是人体内精神能量与身体能量相互结合产生的东方通用能量。随意的结合身体能量与精神能量并不会产生查克拉，查克拉需要一个人的身体能量与精神能量相互调和平衡的状态下才可引导而出。也是因此，习练查克拉是一名忍者的必修课。

效果：  
查克拉的关键属性是【操控+耐力】，其能量池大小与其他能量不同，查克拉能量池上限是关键属性之和的平均值\*2。  
恢复方式：当人物进行一次长休息，恢复所有查克拉能量。  
当人物进行一次短休息，恢复耐力与操控属性平均值的查克拉。  
每日恢复的能量上限不能超过能量池大小的三倍。

查克拉的属性  
查克拉主要分为火、水、风、雷、土为五大性质。通常忍者的查克拉属性在这五大性质之中。  
但在此之外还有阴、阳二种查克拉属性。  
在开启查克拉能量池后，人物投掷一个D5的自然骰，根据出目决定人物的查克拉属性。  
人物可以在主神空间花费1xp任意切换自己的查克拉属性。  
人物只能学习并使用自身查克拉属性所对应的忍术。

查克拉购买方式：

技艺名称：开启查克拉  
价格：500  
购买前提：人物的操控属性与耐力属性等级相同  
效果：  
人物开启查克拉能量池。  
获得查克拉下属能力购买权。  
基本用法：花费1点查克拉，人物一轮内可以在任何液体上站立行走，但依然会受到液体本身性质影响。

技艺名称：血继限界  
价格：500  
购买前提：具有查克拉能量池  
描述：  
当一名忍者同时具有多种查克拉属性时，不同的查克拉属性会融合并产生新的遁术。血继限界一旦修炼成功，就会改变忍者的血脉，并可以遗传给自己的后代。  
效果：  
选择一个你不具有的查克拉属性，拥有这一查克拉属性。

查克拉下属能力：

技艺名称：忍印修炼  
价格：D+500  
购买前提：具有查克拉能量池  
描述：  
忍者施术时结印的目的是借由手势对体内查克拉进行引导，以更高效的释放自己的忍术。  
效果：  
购买该技艺时不获得自由属性点。  
当忍者空置两只手进行满足结印成分的忍术检定时，获得6点法术强度。当查克拉能量池等级提升至C/B/A/S级时，获得的法术强度会提升至11/18/27/38点。
