# 蛊元

  

蛊元  
  
巫蛊师催动巫蛊术的能量，也是蛊物巫蛊力量的来源，它发自你的真灵，与你的生命力交融，具有与万物结合改变现实的力量。  
蛊元的关键属性是决心+耐力。  
每天恢复能量关键属性之和的3倍，平均分配到每时每刻。  
蛊元是一个东方通用能量池。  
基本用法：在进行决心、生理属性相关检定时，可以消耗1点蛊元获得1dp能量加值。
