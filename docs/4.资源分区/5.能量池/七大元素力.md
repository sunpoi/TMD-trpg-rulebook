# 七大元素力

七大元素力  
类型：能量池  
本质：魔幻本质

描述：  
我们都还没来得及说再见，所以...我不会把它当作离别。  
【战争】已经开始了，是上一场战争的延续。  
众神为欲望的轮廓镀上七种光辉。以此昭示，他们的权柄可以被企及。  
而现世的基底里埋藏着阴燃的残骸，那是对僭越者的警示  
【高天之上的神座，从来不是为你预留的位置】  
但僭越之人啊，不要就此驻足。  
谁都不能隔岸观火。  
看吧...

效果：  
七大元素力是一种特殊的能量池，其本质是由构成世界的基础属性：火、水、风、雷、草、冰、岩这七种元素组成，而使用者需要至少通过一种特殊装置来获得操控元素力的能力。  
特殊的，当你购买了该能量池后，你会获得一个特殊装置（你可以叫它什么不用多说了。）该装置视为护符或者戒指，占据护符或者戒指位。该装置可被夺权以及击落，若被夺权或击落后，使用者失去该能量池以及该涉及该能量池有关的所有的技艺/血统/训练的一切好处。（包括可以获得该能量池的技艺/血统/训练）

所有的元素力恢复模式皆为  
1：进行长休息后恢复至满值  
2：进行一次短休息，进行一次元素力对应关键属性的检定，恢复成功数点元素力  
若有特殊情况，则会告知，下不赘述。

特殊的，若你通过别的资源获取了七大元素力相关的能量池，仅仅只获得能量池的基础使用方法和恢复方法，其额外增强仅有购买七大元素力能量池才可获得。  
七大元素力每个能量池都相对独立，不视作同一种能量。

七大元素力购买方式：  
无论哪种属性的元素力，使用者购买时的价格均为D+500。

技艺名称：【捕风的异乡人】  
描述：  
守护自由城邦千年的巨龙 终于对自由产生了迷茫。  
被自由之神命令的自由，还能称之为自由么？  
效果：  
使用者开启了能量池【元素力：风 】属性为：风度+感知  
当你获得了【元素力：风】，你便获得了风元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改。  
1.你的攻击伤害性质永久转变为音波伤害。  
2.你的移动速度提高5米

你获得了【元素力：风】的基础使用方法  
1.自由动作消耗1点【元素力：风】，你的下一次攻击检定获得1点高速。  
2.自由动作消耗1点【元素力：风】，你获得未选择的那项能力直至该轮结束。

你获得了风的认可，你现在由于拥有风元素的亲和力，你获得了全环境呼吸的能力。  
风将会是你战斗时最坚实的伙伴，你将不会受到任何来自箭矢和子弹的伤害：只要你的元素池不为空，那么你获得基于偏斜防御的【防弹】。

技艺名称：【辞行久远之躯】  
描述：  
众目睽睽之下，【契约】之神遭人谋杀  
在最后的时刻，他将签订终结一切契约的契约。  
效果：  
使用者开启了能量池【元素力：岩】属性为：耐力+操控。  
当你获得了【元素力：岩】，你便获得了岩元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为钝击伤害。  
2.你获得了一点天生防御

你获得了【元素力：岩】的基础使用方法  
1.自由动作消耗1点【元素力：岩】，你应对下一次攻击时，获得1点天生防御。  
2.自由动作消耗1点【元素力：岩】，你获得未选择的那项能力直至该轮结束。

你获得了岩的认可，由于现在你拥有岩元素的亲和力：现在你只要该能量池不为空，你便获得了基于天生防御的【防能量】。  
现在当你进行格挡时，你的格挡行为带来的防御类型将更改为天生防御。

技艺名称：【千手百眼，天下人间】  
描述：  
将军长生不灭，幕府锁国之期亦无尽头。  
追求【永恒】之神  
在世人眼中见到了怎样的永恒。  
效果：  
使用者开启了能量池【元素力：雷】属性为：沉着+操控。  
当你获得了【元素力：雷】，你便获得了雷元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为雷电伤害。  
2.你在攻击范围内可以新增一个其他攻击目标

你获得了【元素力：雷】的基础使用方法  
1.自由动作消耗1点【元素力：雷】，你进行下一次攻击时，获得1点破甲。  
2.自由动作消耗1点【元素力：雷】，你获得未选择的那项能力直至该轮结束。

你获得了雷的认可，由于现在你拥有雷元素的亲和力：现在你只要该能量池不为空，你现在的攻击检定附带【法破】【短点射】特性。

技艺名称：【虚空劫灰往世书】  
描述：  
智慧是【智慧】之神的敌人  
知识，是无知之海表面漂浮的诱饵  
学城的学者正在催生愚行  
而神的智慧对此并无意见  
效果：  
使用者开启了能量池【元素力：草】属性为：智力+操控。  
当你获得了【元素力：草】，你便获得了草元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为毒素伤害。  
2.你扩大自己的【元素力：草】3点。

你获得了【元素力：草】的基础使用方法  
1.自由动作消耗1点【元素力：草】，你的下一次智力检定获得1点增强加值。

你获得了草的认可，由于现在你拥有草元素的亲和力：现在你只要该能量池不为空，你便可以催生体积不超过10的无支线植物。

技艺名称：【罪人舞步旋】  
描述：  
【正义】之神热爱法庭上的一切闹剧  
甚至渴求审判诸神  
但她非常明白，唯有【天理】不可与之为敌  
效果：  
使用者开启了能量池【元素力：水】属性为：风度+决心。  
当你获得了【元素力：水】，你便获得了水元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为腐蚀伤害。  
2.你在先攻检定上获得2点增强加值。

你获得了【元素力：水】的基础使用方法  
1.自由动作消耗2点【元素力：水】，在下一次受到攻击时，你受到的1点伤害降低1级。  
2.自由动作消耗2点【元素力：水】，你获得未选择的那项能力直至该轮结束，若你选择的是1效果，则改为获得2点破魔。

你获得了水的认可，由于现在你拥有水元素的亲和力：现在你只要该能量池不为空，你便可以在水上行走，移动速度等同于你的基础移动速度。并且你获得【自适应长休】。

技艺名称：【炽烈的还魂诗】  
描述：  
战争的规则刻写在生物体内  
败者成为战火的余烬，而胜者重燃  
【战争】之神将这秘密告知旅人  
因为她有这样做的理由。  
效果：  
使用者开启了能量池【元素力：火】属性为：力量+操控。  
当你获得了【元素力：火】，你便获得了火元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为火焰伤害。  
2.你的攻击检定获得1点破甲。

你获得了【元素力：火】的基础使用方法  
1.自由动作消耗2点【元素力：火】，下一次攻击时，你的攻击检定获得狂怒1。  
2.自由动作消耗1点【元素力：火】，你获得未选择的那项能力直至该轮结束。

你获得了火的认可，由于现在你拥有火元素的亲和力：现在你只要该能量池不为空，你现在获得【激昂】。

技艺名称：【无神怜爱的雪国】  
她是人再也不会去爱的神  
她是再也不会去爱人的神  
人跟随她的原因  
是相信她终有一日能对【天理】举起叛旗

当你获得了【元素力：冰】，你便获得了冰元素的祝福，你可以在以下增强中选择一个获得，选定后不可更改  
1.你的攻击伤害类型永久转变为冻寒伤害。  
2.你若持有武器，则你的武器虚拟体积提高3点。

你获得了【元素力：冰】的基础使用方法  
1.自由动作消耗2点【元素力：冰】，下一次攻击时，你可以使1点严重伤害转化为韧性伤害。  
2.自由动作消耗1点【元素力：冰】，你获得未选择的那项能力直至该轮结束。

你获得了冰的认可，由于现在你拥有冰元素的亲和力：现在你只要该能量池不为空，那么你的伤势就不会自然恶化。
