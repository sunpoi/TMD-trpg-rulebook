# 妖力

妖力  
分类：东方通用能量池  
本质：特异本质

描述；  
妖力作为妖怪常用的能量，其能量性质更为诡异多变。在能量的使用过程中，可能会被他人认为使用者的阵营偏向更接近妖类。

妖力购买方式：

价格：D+500  
效果：  
强化者开启妖力能量池，能量池的关键属性为风度+耐力。  
妖力的恢复方式为：  
1、在阴暗的环境/夜晚下进行一次短休息，恢复关键属性检定点妖力。  
2、在午夜十二点、月食发生时、天地异象产生时、可恢复能量池上限的妖力。  
无论如何，一日内只能恢复至多等于能量池上限3倍的妖力。

并获得妖力的基础用法：  
【迷常】：  
只要妖力池不空，人物在非战斗场景下会令他人下意识的认为自己是一位人群中十分常见的普通人（如在农村时是一位村民、在城市中是一个上班族/学生等），如果人物表现的比较突兀，则目击者可以进行一个DC为7的意志豁免，豁免通过则本能力失效。这是一个D级别影响心灵来源的思维控制效果。

【惑心】：  
在人物施加可豁免的情绪异常点数时，可以支付2点妖力，使得该情绪异常点数上升2点。（该流程在豁免进行前。）
