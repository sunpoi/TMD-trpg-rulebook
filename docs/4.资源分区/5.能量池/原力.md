# 原力

原力  
分类：能量池  
本质：自然本质

描述：  
原力是一种存在于万物之中的超自然力量，它无处不在而无所在。原力的哲学性观念源于“气”或者“道”等禅意概念，因而超越了常理与凡人的认知。  
原力天然具有两面性：光明与黑暗  
原力的光明面象征着事物的存在、生长、发展，任何事物的存在与发生皆与光明原力息息相关。  
原力的黑暗面象征着事物的改变、矛盾、冲突，因而黑暗原力的存在是宇宙间循环所不可或缺的。  
光明与黑暗原力会相互生克：毁灭意味着复苏，苦痛意味着痊愈...生与死、存在与湮灭，彼此之间循环不息。

效果：  
原力的性质：原力具有光明、中立、黑暗三种性质，在开启原力能量池时，通常会决定一个角色的原力性质。

原力性质通常会因为以下情况而变更：  
光明原力在进入亢奋异常点数的毁灭性后果后，原力性质会永久的转变为黑暗。  
黑暗原力在角色愿意舍弃自我生命而去拯救他人后，原力性质会永久的转变为中立。

原力的关键属性：感知+沉着（光明）/操控（中立）/力量（黑暗）

原力的基本用法：  
光明：角色在进行感知或者沉着属性检定时，支付1点原力，在这一检定上获得1点能量加值。  
中立：角色在进行感知或者沉着属性检定时，支付1点原力，在这一检定上获得1点能量加值。  
黑暗：花招动作支付1点原力，在这一轮内的攻击检定上获得2点亵渎加值。

恢复方式：  
1、每次长休息后，恢复所有原力  
2、在进行一次短休息后，恢复关键属性检定成功数点原力  
每日恢复的原力不超过能量池上限的三倍。

原力购买方式：

技艺名称：原力觉醒  
价格：500  
效果：  
角色开启原力能量池，性质为中立。
