# 元气

元气  
魔幻本质  
能量池

D+500  
（Remnant世界观下的元气变体）  
各个世界观下元气所指皆有不同，在树不（dun）子世界（Remnant）的核心设定，Aura（元气）是源自生物灵魂的生命能量，需通过特定仪式激活后可形成防御力场、治疗创伤并转化为攻击手段。其进阶应用为外像力（Semblance），即消耗元气发动的个性化超能力，如念动力或力量强化，具备唯一性?。  
在树??子世界里，以抗击戮兽、保护人民为天职的猎人被公认为世上最伟大的战士。他们不但精通各种武器和肉搏战，这些勇士们还掌握了一种更强大的力量——元气（Aura），在尘晶（Dust）被发现之前是人类与戮兽对抗的主要手段。

真正的战士与其他生物的区别在于其强化与控制元气的能力。元气首先起到的就是在向外释放精神作为力量时保护身体、偏转伤害的防御作用。在战斗中同时投入身心时，元气就像一层防御力场，被动的覆盖着使用者，能够大幅减轻身体对冷热和疼痛的感觉，保护战斗人员撑过致命一击。当元气护罩被彻底击破时，会产生特殊的视觉效果和声音。

如果陷入苦战，战士还能靠另一种方式来使用元气——外向力（Semblance），这个词用于形容投射元气的更具体形式。  
外像力需要消耗元气来发动，理论上元气一旦耗尽，则外像力也将无法发动。

元气池会根据人物最高内在强化等级具有额外加成，为D5、C10、B15、A20、S25。  
关键属性：从以下组合中选择一个作为关键属性，选择后无法更改，并根据选择获得对应的外向力  
外向力-极速：敏捷+感知  
外向力-雕文：智力+感知  
外向力-残影：决心+感知  
外向力-炽烈焚天：力量+耐力  
外向力-极性操控：操控+感知  
外向力-镜像：操控+风度

恢复方式：  
1.在短休息时，恢复等同关键属性之和的一半点元气。  
2.在长休息后，恢复所有元气。  
无论如何，每日恢复的元气上限不会超过能量池的3倍。

元气的基础用法：自由动作消耗1点元气，令自身在一轮内获得1点无名防御。

元气池强化  
D+500  
每经历一场影片，元气池上限扩大3点，以此成长的次数上限为感知属性的一半。  
新增元气能量回复方式：每小时进行一次元气能量池的关键属性检定，恢复成功数点元气，这会计入每天能量回复上限（即三倍能量池上限）。  
现在，只要你的元气池不为空，就会始终获得1点无名防御。
