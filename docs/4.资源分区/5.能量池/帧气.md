# 帧气

内驱式能量灌输冲件I型——“帧气”  
分类：东方通用能量池  
本质：科技本质

帧气购买方式：

价格：D+600  
描述：  
拥有一个人造能量储存池是对人体进行机械化改造的入门项目，没有它，所有的其他模块都无法顺利搭载。  
在人造丹田技术的基础上，这种试作I型“帧气”开始陆续被应用在原型体上，“帧气”提供的冲能更大，而内部转换效率也更高，适合对所有模块提供驱动能量。  
效果：  
获得一个“帧气”组件会占用改造位。  
“帧气”的容量上限为12，初始容量为决心+沉着之和，从D级开始，决心和沉着合计每比12高出2，这个容量上限便会再增加2点，“帧气”还会随着强化机体的最高支线强化等级而自我提升D5/C10/B15/A20/S25点（逐级替换）。  
“帧气”会参与所有搭载模块的调用，也可以主动被提取出以为机体自身提供保护。开启一个主动用法：在受到一次攻击时可以使用一个反射动作调动1点“帧气”，获得2点全伤害吸收。  
在强化机体支线等级升级为C后，可以不消耗动作并改为消耗2点“帧气”，但无论如何，应对一次攻击只能使用一次这个用法。在一场战斗结束后，可以选择开始制造新的“帧气”，在1D10分钟内将“帧气”完全恢复至满，一天内至多可以恢复三次。
