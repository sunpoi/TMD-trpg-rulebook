# 波纹气功

  

  
波纹气功  
最初是早期人类为了对抗的远古的柱之男开发的利用太阳能量的技术，在柱之男被卡兹灭族后依旧传承了下来。

  
波纹呼吸法  
通过呼吸来驱动波纹。这是一切波纹招式的基础，本技艺中其他一切能力都需要此技艺作为前提。  
D+500  
获得强化能量池，波纹  
波纹是通过呼吸产生的能量，需要在战斗中保持平稳的呼吸才能使用，所以也被称为勇气的力量。  
能量池属性：力量+决心。  
恢复方式：波纹只需短暂休息即可恢复，在战斗结束后休息十五分钟即可将波纹恢复至满，但一天中至多恢复能量池上限三倍的波纹。  
特殊的，波纹需要呼吸与血液才能使用，必须是呼吸系统与体液循环系统完好且没有被改造者才能使用，虽然不一定只有人类才能使用波纹，但必须为以血肉组织组成身体（一般来说具有生命槽）且具有血液与呼吸能力（具有呼吸能力不一定需要呼吸）的生物才能使用。波纹是太阳的力量，黑暗生物使用波纹时每回合受到5点韧性伤害，每消耗1点波纹还要额外受到1点韧性伤害。波纹需要呼吸才能使用，所以若处于不能呼吸的状态，例如水里，太空中，也不能使用，但若你具有全环境呼吸能力（注意，不是无需呼吸能力），或者佩戴氧气面罩之类的东西，则仍可以使用。若你在使用波纹气功时陷入无法呼吸的状态，例如被打穿气管，则你仍能用体内残留的波纹最后使用一次波纹能量。  
  
ST选开：最后的波纹：特殊的，若你的波纹能量不够，你可以选择以生命为代价释放最后的波纹，你可以无视能量消耗释放一次波纹技能，但相应的，你将不可避免地死亡，这种死亡除了主神无法复活。

  
波纹疾走（仙道波纹疾走）  
D+750  
将波纹传导出去进行攻击，这是波纹在攻击上最基础的应用。  
你可以通过击打物体令其作为介质来传递波纹，从而对靠近该物体的目标造成伤害，这视为让你的肉搏攻击具有【贯穿】效果，你至多以此法传播1M，特殊的，虽然攻击正常计算，但你可以选择不对介质造成伤害。波纹对吸血鬼来说是致命的，若被传入波纹的是黑暗生物，你每有2点决心上附加成功就将1点伤害转化为韧性伤害。加持了波纹疾走的攻击伤害类型变为正能量类型伤害。  
你可以通过修炼提升对波纹的应用，通过支付XP来提升波纹疾走，需要按顺序购买  
11XP：C级  
传播距离改为决心\*1，  
22XP：B级  
【贯穿】改为【超级贯穿】，射程改为决心\*4M。

  
———前置为波纹疾走的技艺————  
银色波纹疾走  
将波纹通过金属进行传递，也可以通过敌人的武器攻击对方  
B+2000  
消耗：6波纹能量  
现在你在使用白刃武器、枪械、投掷武器、弓弩时均可使用波纹疾走，但枪械与投掷武器这类远程武器不会因波纹疾走获得额外的射程，并且加持了波纹疾走的攻击可以视为接触攻击，这表现为波纹传过对方的盔甲直接攻击其肉体。  
在受到天生武器攻击或近战武器攻击这类在攻击的瞬间会通过武器将攻击者与被攻击者连接的攻击时，你可以以反射动作调用下回合的标准动作来进行一次特殊的招架，除了正常的招架效果以外，你可以同时进行一次攻击，这次攻击必须加持波纹疾走，且不计算武器伤害。

  
太阳波纹（山吹色波纹疾走/金黄色波纹疾走/紫棠花色波纹疾走）  
B+2000  
通过拳头直接释放的绝招，颜色是最接近太阳的金色波纹，攻击力是波纹疾走系列中最强的。  
在对触及范围内的敌人进行加持了波纹疾走的肉搏攻击时可以支付7点波纹能量使用，在击打的同时从拳头中释放的是最接近太阳的波纹，这次攻击具有湮灭效果，并获得等同于决心\*1的能量加值，若对方为黑暗生物，再将力量与决心较低者的附加成功数的伤害转换为韧性伤害。

  
绯红色波纹疾走  
C+1000  
通过摩擦产生高热效果的波纹。  
在进行波纹疾走加持的攻击时，你可以支付3波纹能量让波纹产生高热效果，用以烧伤对方，这将为你的这次攻击带来等同于决心上附加成功的能量加值，并附带力量上附加成功点的烧伤点数。

  
机体紊乱  
C+1000  
通过波纹让对方的机体产生紊乱，从而强制令其去执行某个动作。  
只能对具有肉体的敌人使用，由于这种招式极为复杂，你同时只能让一名目标陷入紊乱，你消耗5点波纹能量使用标准动作对触及范围内的一名敌人击出拳头，使用力量+武技+肉搏进行对抗，对方使用反射豁免，成功则你成功击中对方的躯体，随即放出波纹精准的影响对方的机体，在当前场景下，对方在不危及自身生命的情况下必须向你所指定的方向进行移动，这是一个B级创伤来源的胁迫效果。  
波纹的应用会随着战斗逐渐强大，当你在一场影片内进行至少三次同级难度的战斗（杰克、两名骑士）和一场高一级的战斗（DIO）并获得胜利后，从下一场影片开始，这个效果会变为A级创伤来源的胁迫效果，你可以令目标在当前场景中不断尝试对某个目标进行擒抱，即使是他的友方，特殊的，可以令其擒抱正在运行的机器，从而产生破坏作用，即使这些行为会危害他自身。

  
究极深仙脉疾走  
B+2000  
前置：仙道波纹疾走升至B级  
继承我的意志吧！究极！深仙脉波纹疾走！  
这便是究极的仙道。使用标准动作，消耗10点波纹能量，将自己的毕生功力传递给触及范围内一名友方，在使用之后你便会无可避免地死去，即使是主神也无法将你复活，这名友方将获得你在波纹气功上所有已获得的能力，其本就具有的能力则返还一半的XP，这个XP只能用于波纹气功中技艺的购买。在传功的同时你还会对其治疗等同于你决心上附加成功点的生命值，若你在踏入究极的仙道仍修行了一影片，则你深厚的功力不会进行如此低效的治疗，而是用一种脱胎换骨的方式立刻令其完全恢复，即使他已经失去生命，这视为一个S级的复活效果。  
ST选开：虽然因为都会导致死亡，所以该技艺即使在ST选开最后的波纹时也不应一同使用，但选开最后的波纹的ST可以选开该技艺可以适用最后的波纹，这样比较还原。

波纹治疗  
D+500  
使用波纹进行治疗，这是波纹原本的用法  
消耗2点波纹能量，使用标准动作，为自己或触及范围内的某人进行治疗，对其治疗自身决心上附加成功+1的生命值，医疗点为（决心上附加成功+1）\*12。

  
弹簧拳  
D+500  
前置：波纹治疗  
让骨头脱臼来延长攻击距离，疼痛就用波纹来缓解  
进行肉搏或白刃攻击时，你可以自由动作支付2波纹能量来让这次攻击的触及距离延伸力量上的附加成功+决心上的附加成功M，这种突然延伸的攻击距离会让人猝不及防，这是一个D级的措手不及效果。

  
波纹防御  
D+500  
让波纹布满全身进行防御，在进行格挡时支付2波纹能量以额外获得力量属性一半的格挡防御。  
10XP：前置：波纹疾走  
你可以再支付3点波纹能量（合计5点），下一个使用白刃攻击或肉搏攻击对你造成伤害者因为被传导波纹而受到你决心上附加成功点正能量伤害。

  
生命磁场  
D+500  
我能感受到红酒的波纹，从酒杯传到手臂，再通过身体传到地面，我能感受到那家伙的生命振动！  
你获得一种新的感知的方式，通过波纹感知到生命磁场，这是一种强感知方式。这种感知方式可以感知到一切至少部分由肉体组成的且可以自发进行运动的存在，所以即使是尸生人也可以被感知到，这种感知方式可以穿透墙壁进行感知，但只能感知到存在与位置对于颜色之类的细节则无法感知。  
10XP：  
你可以通过生命磁场进行吸附，通过这种方式你可以在任何树木或者涂油的墙壁爬行，如果你学会了银色波纹疾走，则可以在没有涂油的墙壁爬行。你也可以用生命磁场在水面行走，即获得等同于基础移动速度的水面速度。你也可以将波纹当做强力胶使用，使用波纹将两个以上的物品连接起来，这视为一个制造物品能力，正常进行造物检定，你只能以此制造无支线物品。通过对生命磁场的锻炼，你的青春得以保留，只要你坚持练习，你的寿命就不会因为时间而自然流逝，这是一个D级不老能力，不过一定要记得坚持，不然……

  
一瓶植物油  
10  
理论上波纹无法直接在金属上进行传递，但银色波纹疾走又的确是在金属上传递的波纹，不过设定嘛，味道挺好的。这是一瓶植物油，使用迅捷动作淋在剑上就能像银色波纹疾走一样在近战武器上使用波纹，如果是弓弩，则需要额外花费一个迅捷动作在射击前对箭矢进行淋油，枪械同理，但ST有权裁判子弹因此直接炸膛。  
一瓶油可以无限使用，但只能用于对武器或墙壁的涂抹，如果想用这瓶植物油做别的事，则会因为神秘的主神意志导致一滴油都流不出来，对此我觉得很可惜，听说这种油做菜挺好吃的。
