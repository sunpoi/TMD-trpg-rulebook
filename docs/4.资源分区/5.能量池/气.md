# 气

  

气  
气是存在于生物之间的根本能量，其性质会随使用者而出现变化，可能偏向暴烈，也可能偏向阴和。  
  
关键属性：沉着+耐力  
  
恢复方式：气可以通过一次完整的长休息恢复至能量池上限。也可以在短休息时进行一次能量关键属性的检定，恢复成功数点气。  
  
  
气的感知  
D+500  
你开启气能量池  
你现在获得一种基于气对事物进行感知的能力。  
只要你气能量池不空，你就能在敏感范围内对生物进行模糊定位，这是一种非线性感知方式，同时具有D级别、特异本质的侦测等级。  
气的基本用法：你可以支付2点气，令你在洞察检定中获得3dp能量加值，持续一轮。  
  
气的精进  
C+1000  
前提：气的感知  
现在你通过气的感知方式可进行精确定位，如同具有【透视】效果一般，气感知方式的侦测等级提升至C级。  
气的基本用法得到了提升：你可以支付6点气，令你在一轮内的洞察检定与防御上获得6点能量加值。
