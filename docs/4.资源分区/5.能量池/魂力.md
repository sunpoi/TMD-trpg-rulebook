# 魂力

名称：能量池：魂力  
本质：魔幻  
类型：能量池

描述：魂力是斗罗星的人通过其特有的先天能力武魂吸收天地元气而形成的能量，具有极强的兼容性。

效果：  
魂力是一个通用能量池，其关键属性为【智力+操控】

魂力能量池不可重复开启。

魂力的恢复方式：  
1、短休息时，进行一个关键属性检定，恢复成功数点能量  
2、进行长休息以后，能量池恢复满值。  
无论如何，一日内恢复的能量值不能超过上限的三倍。

魂力的基本用法会根据获得魂力方式的不同而不同。

名称：废武魂觉醒  
价格：500  
类型：技艺  
描述：在主神的帮助下进行的武魂觉醒，但觉醒的是基本没什么作用的废武魂，但仍有魂力存在。

效果：你选择一种杀伤性低的动物或日常物品作为你的器/兽武魂，开启魂力能量池。

魂力的基本用法：  
当你选择的是日常物品时，你进行相关物品的侦查检定时可以消耗1点魂力获得1点能量加值。

当你选择的是动物时，你可以消耗1点魂力进行武魂附体一轮，在身体上体现出该动物的特征，本轮你的威吓检定获得1点能量加值。

名称：唐门弟子  
价格：500  
类型：称号  
描述：你修行了超级阉割版玄天功，自然是一位唐门弟子。

效果：作为唐门弟子，你可以在任何时候声称你是一名唐门弟子，并在面对该死的人的时候声称其有取死之道，但这个身份并没有什么卵用，这是一个无支线心灵来源的背景身份，由于修行了超级阉割版玄天功，你的魂力能量池上限提升3点。

名称：魂力属性：火/冰  
价格：D+500  
类型：天赋类技艺  
描述：自身武魂所带有的属性反应到魂力上的体现，不同的属性有不同的表现形式：  
火：高温，火焰以及剧烈反应  
冰：低温，冻结以及降速

效果：  
选择一种属性，获得其对应效果，该技艺可重复购买，但必须选择不同项。

火：你消耗魂力的攻击额外具有【熔毁5】，如果你额外消耗1点魂力，该次攻击伤害性质改为火焰伤害。

冰：你消耗魂力的攻击额外附带1点D级环境来源的不可豁免的冻结点数，如果你额外消耗1点魂力，该次攻击伤害性质改为冻寒伤害。
