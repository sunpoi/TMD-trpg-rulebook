# 真气

  

真气  
真气是一种东方通用能量池  
  
真气的关键属性是沉着+感知  
  
真气的恢复方式：  
1、在一次至少十五分钟的短休息中进行一个真气关键属性检定，恢复成功数点真气。  
2、花费8小时进行打坐，沟通天地阴阳之机，令真气重新充盈丹田，恢复能量池上限的真气，这同时视为进行了一次长休息。
