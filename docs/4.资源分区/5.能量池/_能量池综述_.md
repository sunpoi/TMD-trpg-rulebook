# 能量池

  

  
【能量池说明】  
能量是某些能力或技艺发动的前提，当人物获得后以能量池的形式计算其大小。如无特殊说明的情况下，人物可以同时拥有多种能量。每种能量都有对应的方式，如以那两种属性决定能量池大小、如何恢复能量、能作为那些能力或技艺的发动前提、是否具有基本用法等。  
  
【能量池获得】  
人物可以通过各种方式与形式获得各种能量池，但是对于已经获得过能量池，再次获得并不会有任何加成。  
  
【能量池每日恢复上限】  
一种能量池可能会拥有多种恢复方式，但是正常情况下，每日恢复的能量不得超过能量池大小的三倍。  
  
【通用能量池、能量池变体、强化能量池】  
通用能量池：正常的能量池之间每1点能够发挥的作用在理论上是一样的，因此可能出现一种能量可以作为多种能力或技艺发动的前提、一种能力或技艺可以用多种能量作为发动的前提等类似的情况，这些情况需要以能量、能力、技艺的描述决定是否存在，在无特殊说明的情况下，能量池都为通用能量池。  
能量池变体：以某种能量池为基础，具有一定程度不同的能量池。虽然本质相同，属于同一种能量，但其使用方法、能量池的关键属性、恢复方式……都有可能不同。当你开启了某个能量的多个变体时，该能量池可以同时满足多个变体能量的消耗需求、使用需求。例如：变体魔力可以作为魔力来满足使用通用能量（或是西方能量）的需求  
强化能量池：其大小会根据人物最高内在强化等级具有额外加成，为D5、C10、B15、A20、S25。各级别能量池大小相互取代  
  
【减少能量池消耗的技能 】  
若未特殊说明，减少能量池消耗的技能最低将消耗减少到1点。  
  
【能量池的分类】  
能量池的分类仅与该能量的关键属性有关：  
心智类能量：心智+心智  
代表：SP、蓝色法术力  
身知类能量：心智+生理  
代表：暂无  
生理类能量：生理+生理  
代表：暂无  
精魄类能量：生理+互动  
代表：妖力  
互动类能量：互动+互动  
代表：梦力  
灵魂类能量：心智+互动  
代表：暂无  
  
【能量池等级】  
能量池等级通常由以下方式决定：  
1、能量关键属性较低者的附加成功，当一个能量的关键属性较低者的附加成功为1/2/3/4/5时，能量池等级为D/C/B/A/S级。  
2、开启这一能量的内在资源最高支线等级。  
可能有资源具有特殊的能量池等级计算方式。  
多种能量池等级计算方式取高者。  
  
【能量种类上限】  
一个单位可同时拥有的能量种类（变体能量与原能量视为同一种能量），通常只有1种，该单位每具有2点传奇耐力会提高1种可持有的能量上限。这是由于多种性质的能量在人物体内互相冲突导致的。一些能量可能会兼容其他能量。
