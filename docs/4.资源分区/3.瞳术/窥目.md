# 窥目

窥目  
描述：眼睛即是心灵的窗户，通过窥视心灵的窗户从而得知他人的所思所想……窥目并不是那么方便的东西。窥目是一种与生俱来的才能，是天赋与社交能力的一种体现，同时也包括了微表情、气氛的察觉力。

虽然分析起来很复杂，但概括而言就是能“大体上把握他人的性格”“当前的情绪变动”“危机的氛围”等多方面能力综合体。由于多数性能都需要能看到他人的眼睛才能生效，窥目的作用其实相当微妙。

D级：差不多能读懂氛围的程度。  
D+600  
特性：【心灵窗口】【危机感知】

【心灵窗口】：通过目视他人的双眼从而大体把握性格是一件相当困难的事情。在这一阶段，你能读懂的充其量也就是对方当前明显的情绪而已。

你获得D级信息来源的读心能力。你可以在一次社交行为中额外进行一次侦查检定，对抗目标的伪装检定，若成功则你可以了解到目标当前的最明显的情绪（例如高兴，厌恶等情绪之一）。特殊的是，这一能力如果想要生效必须是你在能看到目标眼睛的时候，并且如果对方是盲人，或者干脆没有正常意义上的眼睛，则该效果无法生效。

【危机感知】：对危机的感知是窥目是副产物，也就是能够感觉到不妙的气氛，通俗来说就是能感觉到“杀气”。虽然不能未卜先知，也没法追溯来源，但至少可以在一定程度上对危险做出反应。

你免疫不超过D级措手不及的效果影响；你在攻击检定上获得2DP表现加值，并获得2点洞察防御。

C级：能把握性格的程度。  
C+1200  
特性：【目视心灵】【危机一发】

【目视心灵】：目视他人的心灵是一件很复杂的事情，虽然如今你能大体把握他人的性格、情绪、精神状态，但所得到的感觉相当模糊。

简单的例子：“只是看了一眼她的眼睛，你立刻就明白：她已经没救了。”精神状态-疯狂。情绪：狂气。

普通的例子：“她在灰暗的蓝色眼瞳看着你，那是一种放弃了一切的眼神。”情绪-平静，精神状态-绝望。

稍微复杂的例子：“她用如海洋般深邃，似星空一样闪耀的瞳孔看着你，然而你莫名的感觉她的眼睛中映照不出任何感情。”情绪-？，精神状态-？，性格-？，这种情况下最有可能的是对方根本就没有正常人性，或是处于被操纵状态。

你在【心灵窗口】中获得的读心能力提升至C级，效果改为你可以在一次社交行为中额外进行一次侦查检定，对抗目标的伪装检定，若成功则你可以感受到对方的心理活动、情绪、精神状态（参考上述例子内容的表现），并且能对目标的性格做出评估结论（例：1.虽然看起来很温和，但内在是冷酷无情的性格。2.虽然态度恶劣，其实是因为心理创伤导致的结果，本质应该是个懦弱的人。以此类推）。此外，你可以在战斗中每轮一次，以自由动作进行上述侦查对抗伪装的读心检定，并得知目标当前情绪与精神状态。

但是，你并不能真正读取目标的想法内容，因此你并不知道目标思想的具体内容，只能自行推论。其他限制依旧不变。

【危机一发】：在最危险的关头，你总是能提前进行预知。

你在【危机感知】中获得的免疫措手不及能力等级提升到C级；你在进入新场景时，可以选择以自由动作消费传奇感知一半的意志力（在S级时再额外支付1点意志力），使【危机感知】的效果额外再提供攻击与防御上的1个附加成功数（基于洞察防御）。

B级：看破敌人行为的程度。  
B+2400  
特性：【看破的才能】【先知先觉】

【看破的才能】：并非只是能看穿精神状态、心理活动，把握性格，也能在战斗中真正意义上的看破对方使用能力的底细，从而更好的做出应对。

你获得基于洞察防御的【洞彻】，这一效果需要你每轮使用一个迅捷动作进行使用并维持。

【先知先觉】：你彻底掌握了对危险氛围的感知能力。

你总是知晓敏感范围内是否存在敌意单位，这是一个B级信息来源的【预知】效果；你在【危机一发】中获得的措手不及免疫能力提升至B级。

额外支付B+2500分：你在先知先觉中获得的免疫措手不及等级提升至A级，并获得等同感知属性一半的攻击表现加值/洞察防御；你获得一种非线型强感知方式，能够正常定位对自身产生威胁的事物，包括但不限于有敌意的敌对目标、试图攻击自身的目标、即将爆炸的手雷等情况。
