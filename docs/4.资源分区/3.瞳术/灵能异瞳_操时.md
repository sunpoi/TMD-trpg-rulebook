# 灵能异瞳：操时

  

灵能异瞳：操时  
  
介绍：出自《灵气复苏：七日尸潮》中的能力，是在人类终结地上的阴尸乱世之后，新崭露头角的人类议会强者安娜·克罗克所拥有的异瞳。这种异瞳能够目视“时间”，看向过去与未来。如果单纯要论及破坏力，大概无法与当时已经成为传说强者的叶封的“霜光”相提并论，但这据说也是因为全力施展这一异瞳的能力可能会导致时间结构的崩坏，从而对人类造成重大的损害所导致的。  
由于这个世界当中的异瞳，本质上是一种特化了的灵能者能力，所以这个兑换内还包含了一些轮回者应用该瞳术所必须的对灵能力本身的开发。  
  
D级：操时初开  
价格：D+600  
描述：你开启了灵能异瞳：操时，这会使得你的眼部化为一种宛如被晨雾笼罩的淡银灰色。在银灰色中，有无数冷金色的碎芒在你的眼中流转——他们是自时间沙漏中泻出的金沙。不过现在你还无法通过调动灵能进行干涉时间的“注视”。  
【灵能力】：你现在开启一个名为灵能力的能量池，关键属性是感知+决心。并且你获得一个基础用法，你可以在一次决心属性参与的检定上支付一点灵能力，获得2dp的能量加值。  
【时流洞察】：虽然现在还不足以进行能够干涉时间的“凝视”，但是现在的你的眼睛，也能够看到来自过去与未来的零散信息。  
当你目视一个事物时，如果你想要得知这个事物过去的经历（比如得知一具尸体的死因，一个物品先前被谁持有过等），你可以集中意念，从而通过自己的目视看到相应的“过去”。这个向前目视的时间范围最大到15分钟之前，这是一个D级别的时空来源的信息回溯效果。  
目视的来自未来的信息碎片能够警示你即将到来的危险。你具有C级别的免疫措手不及。  
  
  
C级：凝视之力  
价格：C+1200  
描述：你适应了操时的力量，当你下定决心进行“凝视”时，你的视线将没入时光的长河之中，读取过去和未来，但承受信息的，仍然是你作为人类的大脑，因而这也会给你带来极大地负担。  
【操时凝视】：你现在可以在洞察技能上开启一个名为“灵能凝视”的专业。  
你现在获得了购买操时下属的“凝视之力”技艺的权限，作为最基础的用法，你可以花招动作支付至多你掌握的“凝视之力”下属技艺数量\*2点灵能，每点支付的能量会提供2点幸运防御，持续一轮。  
你每开启一个“凝视之力”你就能从下述的增益中选择一项获得，不能重复获得相同选项的增益。  
1.你掌握了更多操控时空的力量，你看穿过去与未来的能力也得到了增强。  
获得此项增益后，你从D级得到的信息回溯效果提升到C级，可以向前回溯的时间延长到1个小时，同时你从D级获得的免疫措手不及的能力提升到B级。  
2.你的异瞳能够看穿时空的扭曲以及未来与因果如何被操控，并且让自己不被修改未来事物发展的能力所影响。  
获得此项增益后，你具有C级别的时空来源效果免疫和C级别的命运来源效果免疫。  
3.你的异瞳能够看到来自过去与未来的信息。  
你可以知晓敏感范围内是否存在敌意单位，这是一个C级别的时空来源的预知能力。  
4.你能够间断性的改变对你不利的结果  
每轮一次，你能够重新投掷洞察检定，以后者结果取代先前结果。  
  
  
B级：“我拒绝不利于自己的未来。”  
价格：B+2400  
描述：大多数的凡人，终其一生都在逃避厄运的纠缠——病痛，受伤，死亡，这一切都令他们感到害怕。  
然而你所拥有的灵能力可以让你逃避这一切。  
以这双眼睛目视的未来，有对自己有利的未来，也有对于自己不利的未来。  
去把对于自己不利的一切以这双眼瞳的力量进行“蔑视”，那么你便能够让对你不利的“未来”无法成为“现在”。  
现在，你可以以一个标准动作进行一次特殊的全防御。这次全防御中，你的基本防御的防御类型会转化为幸运防御（但是依然是基本防御），同时你通过这次全力防御不会获得闪避防御增加的效果，取而代之，你额外获得等同于感知点的幸运防御，并且你具有基于幸运防御的防御上的【劣化2】。  
这在视觉上表现为，宛如你眼瞳中美丽金沙一般的金色光点环绕在你的身周，将一切靠近你的“不幸”与“痛苦”弹开，你抵挡攻击依靠的不再是自己个人的力量去打落或者躲开，而是依靠操纵时间的力量使【命运】与你同在。  
你在进行这次格挡的同时还可以令70米半径圆形范围内的所有敌人以意志豁免与你的决心+洞察+专业：灵能凝视检定进行对抗。如果对方在对抗当中失败，则会被你所编织的命运所限制。  
在被你所编织的未来限制的情况下。对方如果想要对你进行攻击，则进行的攻击会自动改为以除你与发动攻击的敌人以外的在场任意角色为目标，因为对你造成伤害的未来在时空层面上为你所否定，这是一个B级时空来源的胁迫效果。  
  
  
A级：灵能真谛  
价格：A+4800  
描述：暂不开放  
  
专属技能：  
  
异瞳视觉  
价格：D+750  
描述：异瞳给你最基础的增幅，就是你的眼瞳的性质发生了转变，令你的视觉展现出各种神奇的效果。  
只要你的灵能力池不为空，那么你的视觉就具有D级的侦测等级，并且具有黑暗视觉和灵感视觉。  
异瞳开发：当你获得了“灵能力成长”之后，你灵魂所蕴含的更为庞大的灵能力也会使得你的异瞳视觉获得进化。  
你可以支付D+750，使你的异瞳视觉差价升级到C级（这个技艺会就此变成C技艺，提供对应的内在属性，下同）。这会使得你的侦测等级提升到C级。此时你还可以选择消耗一个迅捷动作，花费1点灵能力，使得受到下一次进行的凝视攻击所命中的单位将被你施加【真实视觉】。  
  
异瞳开发II：当你获得了B级瞳术主轴之后，你灵魂所蕴含的更为精纯的灵能力让你的你的异瞳视觉获得进一步的提升。你可以支付C+1500，使你的异瞳视觉差价升级到B级。这会使得你的侦测等级提升到B级。  
现在你在受到攻击时，可以支付3点灵能，令你在应对这次攻击时具有决心/2点洞察防御。  
  
异瞳开发III：当你获得了“灵能真谛”之后，你对于灵能力的本质的体悟让你的双瞳所能够发挥出洞察一切的力量。你可以支付B+2000，使你的异瞳视觉差价升级到A级。这会使得你的侦测等级提升到A级，并且你的敏感范围因此增加一倍。  
  
凝视之力：加速  
前提：C级：凝视之力  
价格：C+1000  
描述：以异瞳的力量干涉时间的流速，令自身爆发出超乎寻常的速度。也可以用于直接对敌人进行进攻，加速对手躯体的器官，部件的运转，令对方快速陷入疲乏，从而令对方崩溃。但加速时间与进行未来推演相同，都会令你的灵魂承受压力。  
这个能力一共有两种用法。一轮之中，只能选择其中的一种发动。  
1.以一个迅捷动作，消耗6点灵能力，并且承受1L不可减免的精神伤害。你可以发动该能力。在本轮中，你的移动速度提升25米，并且你在攻击上具有3dp的速度加值和【高速10】。  
2.以一个标准动作，消耗6点灵能力，你可以发动该能力。你可以对15米范围内的单体敌人进行一次以决心+洞察+专业：灵能凝视+专业：能量爆发为构成的攻击。这次攻击附带等额的疲劳与恶心点数，这表现为你以加速时间让对手的器官超出极限地运作到出现不良反应。同时对手还需要以强韧豁免对抗你这次攻击的出目，如果对抗失败，则会立刻减少对方10年寿命上限，并且陷入衰老状态，你至多令受影响目标的寿命上限减少四分之一，这是一个C级时空来源的衰老效果。  
  
异能开发：当你拥有了B级别的该瞳术主轴之后，你可以支付B+2000，强化你的异瞳加速时间的能力，以取代原本的效果。  
在开发后，“凝视之力：加速”仍然具有两种用法。仍是一轮之中，只能选择其中的一种发动。  
1.以一个迅捷动作，消耗10点灵能力，并且承受1L不可减免的精神伤害。你可以发动该能力。在本轮中，你的移动速度提升85米，之后立即进行一次移动，并且你在攻击上具有5dp的速度加值和【高速15】。  
2.以一个标准动作，消耗6点灵能力，你可以发动该能力。你可以对35米范围内的单体敌人进行一次以决心+洞察+专业：灵能凝视+专业：能量爆发为构成的攻击。这次攻击附带等额的疲劳与恶心点数，这表现为你以加速时间让对手的器官超出极限地运作到出现不良反应。同时对手还需要以强韧豁免对抗你这次攻击的出目，如果对抗失败，则会立刻进入B级别的迟滞状态。同时对方还会减少对方80年寿命上限，陷入衰老状态，你至多令受影响目标的寿命上限减少二分之一，这是一个B级时空来源的衰老效果。  
  
  
凝视之力：迟缓  
前提：C级：凝视之力  
价格：C+1000  
描述：以异瞳的力量干涉时间的流速，令对手的攻击变得迟缓从而给自己留出更多的反应空间。但减缓时间与进行未来推演相同，都会令你的灵魂承受压力。  
以一个移动动作，消耗6点灵能力，你可以发动该能力，在发动该能力时，你如常进行一次格挡。在这次格挡的持续时间内，你能够抵消8点【高速】，你在反射豁免上具有【8加骰】，并且在防御上额外具有传奇感知\*3点的洞察防御。  
  
异能开发：当你拥有了B级别的该瞳术主轴之后，你可以支付B+2000，强化你的异瞳迟缓时间的能力，以取代原本的效果。  
以一个移动动作，消耗6点灵能力，你可以发动该能力，在发动该能力时，你如常进行一次格挡。在这次格挡的持续时间内，你能够抵消24点【高速】，你在反射豁免上具有【8加骰】，并且在防御上额外具有传奇感知\*6点的洞察防御和基于此洞察防御的【超级坚韧】。  
  
  
凝视之力：回溯  
前提：C级：凝视之力  
价格：C+1000  
描述：以异瞳的力量干涉时间的运作，令自己的身体恢复到先前完好的状态。只要持有了这双名为操时的眼睛的话，那么时间就无法令你衰老。当与你相关的一切逝去，你才会发现，无尽的时间也许意味着无尽的煎熬。  
购买了这一能力之后，你获得【不老】特性，这一效果常驻生效。  
消耗一个迅捷动作和6点灵能力，你可以令你在这一轮中具有【快速医疗5】（医疗点数60）。  
异能开发：当你拥有了B级别的该瞳术主轴之后，你可以支付B+2000，强化你的异瞳回溯时间的能力，以取代原本的效果。  
消耗一个迅捷动作和10点灵能力，你可以令你在这一轮中具有【快速医疗12】（医疗点数144）和肢体立刻重生的能力，这在画风上表现为你重新让时间倒流到了肢体仍然在你身上的时候。  
  
  
  
凝视之力：削删  
前提：C级：凝视之力  
价格：C+1000  
描述：以异瞳的力量干涉时间的运作，令自己短暂地从当前的事件段中剥离而出，从而达到删除事情发生的过程，直接达到事情的结果的效果。  
以一个移动动作，消耗6点灵能力，你可以发动该能力。在发动这一能力时，你可以同时进行一次移动。只要发动了这一能力，你在本轮中进行的移动变为C级别的时空来源的传送移动。在本轮内，你可以将一次攻击结合在传送移动之中，这次攻击具有B级别的【先制】效果，并且额外具有3dp表现加值，这在画风上表现为你在时间线之外行动，随后回归这条时间线，突兀地抵达你到达地点的结果，以这样的手段令对手对你的攻击来不及做出防御。  
  
异能开发：当你拥有了B级别的该瞳术主轴之后，你可以支付B+2000，强化你的异瞳削删时间的能力，以取代原本的效果。  
以一个移动动作，消耗10点灵能力，你可以发动该能力。在发动这一能力时，你可以同时进行一次移动。只要发动了这一能力，你在本轮中进行的移动变为B级别的时空来源的传送移动。在本轮内，你可以将一次攻击结合在传送移动之中，这次攻击具有A级别的【措手不及】效果，并且额外具有3dp表现加值和【湮灭】，这在画风上表现为你在时间线之外行动，随后回归这条时间线，突兀地抵达你到达地点的结果，以这样的手段令对手对你的攻击来不及做出防御。  
  
  
时空演算  
前提：具有至少2个C级凝视之力下属技艺  
价格：C+1000  
描述：你的异瞳能够通过对于过于和未来的探知，凝视到那一个对你来说是“胜利”的未来。  
你必须要消耗一个标准动作，并且消耗6点灵能力才能使用这一能力，同时发动这一能力还会给你带来3L不可豁免的严重伤害，如果判定伤害类型，则这次伤害的类型属于精神伤害，因为这本质上是由于你以人类的脑部构造理解更高维度的时空构造而导致的灵魂过载。  
发动这一能力后，你以决心+洞察+专业：灵能凝视+专业：能量爆发和15米范围内的任意一个角色的意志豁免进行一次对抗。如果这次对抗当中，你胜出，那么你可以以操时对无数时间分支当中的对方一切行动可能进行演算。  
“时空演算”在发动的本轮不会带来任何好处，而等到两轮过后（这个两轮包括发动效果的这一轮在内，例：你在第一轮使用这一能力，则在第三轮此能力会起效），到来的那一轮的未来便被你完全掌握。被你进行操时演算的对象需要先进行下一轮的行动，在你得知其下一轮即将进行的行为后继续你的行动，你的行动有可能会改变你得知的行动结果。（例：你得知在你面前的敌人下一轮会对你进行近战攻击后格挡，你选择了移动拉开距离，那么由于其攻击距离不够，他的行动会变为移动到你的面前并进行一次近战攻击），这是一个B级别、时空来源的未来视效果。此外，在那一轮中，你的攻击对被进行了“操时演算”的对象额外具有感知属性\*2点的洞察加值，并且你在攻击上额外具有【高速15】。
