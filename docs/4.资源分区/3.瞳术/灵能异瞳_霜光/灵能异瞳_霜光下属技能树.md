# 灵能异瞳：霜光下属技能树

专属技能：

异瞳视觉  
价格：D+750  
类型：主动  
描述：异瞳给你最基础的增幅，就是你的眼瞳的性质发生了转变，令你的视觉展现出各种神奇的效果。  
效果：  
只要你的灵能力池不为空，那么你的视觉就具有D级的侦测等级，并且具有黑暗视觉和灵感视觉。

异瞳开发：当你获得C级瞳术本体之后，你灵魂所蕴含的更为庞大的灵能力也会使得你的异瞳视觉获得进化。你可以支付D+750，使你的异瞳视觉差价升级到C级（这个技艺会就此变成C技艺，提供对应的内在属性，下同）。这会使得你的侦测等级提升到C级。此时你还可以选择消耗一个迅捷动作，花费1点灵能力，使得受到下一次进行的凝视攻击所命中的单位将被你施加【真实视觉】。

异瞳开发II：当你获得B级瞳术本体之后，你灵魂所蕴含的更为精纯的灵能力让你的你的异瞳视觉获得进一步的提升。你可以支付C+1500，使你的异瞳视觉差价升级到B级。这会使得你的侦测等级提升到B级。  
现在你在受到攻击时，可以支付3点灵能，令你在应对这次攻击时具有决心/2点洞察防御。

异瞳开发III：当你获得了“灵能真谛”之后，你对于灵能力的本质的体悟让你的双瞳所能够发挥出洞察一切的力量。你可以支付B+2000，使你的异瞳视觉差价升级到A级。这会使得你的侦测等级提升到A级，并且你的敏感范围因此增加一倍。

异瞳应用  
D+500  
类型：被动  
描述：你将成长之后更为庞大的灵能力注入异瞳。  
效果：  
你在进行攻击时可额外消耗2点灵能，这使得你的【霜光凝视】所进行的攻击的可以选择对扇形范围内的所有单位进行攻击。

热力学原理  
价格：C+1000  
类型：被动  
描述：霜光的力量的本质是纯粹的超自然。然而对于热力学知识的了解可以帮助你更好地引导霜光的力量，分析这双眼睛所看到的流动着冷与热的世界。  
效果：  
你在使用洞察技能进行检定时（包括但不限于侦查，搜查，或者使用凝视进行攻击等行为），只要你在学识技能上具有热力学专业，你就在检定上获得智力/2点的内在加值，这表现你用经过灵能力开发的脑域实现了精准精算，强化了异瞳的应用。  
此外，只要你应用热力学原理经过了一场影片，并且赢得了一场同级战斗的胜利，那么你就会因为在战斗中对于热力学理论在霜光的能力应用上有了更深入的理解，这会令你的洞察技能检定获得【9加骰】。

异瞳开发：当你获得B级瞳术本体之后，你可以以更细致的运用将霜光所蕴含的冻结之力与冷热的学识相结合，通过支付“B+2000”，将“热力学原理”逐级升级到B级别。此时你进行的“凝视”攻击所附带的冻结点数变为等同于伤害点而非附带二分之一，而且你获得针对冻结点数的减值以及严重状态的免疫。  
由于你对世界运行规律的把控。你的洞察技能检定原本获得的【9加骰】改为【8加骰】。  
你现在进行的冻结更为深入，可以将对手彻底地冰晶化，因而当你进行“凝视攻击”时，现在可以将附带的冻结异常点数转为附带等额的晶化点数，标准豁免。

脑域开发  
价格：B+2000  
类型：主动  
描述：这个兑换只有当你的学识技能等级达到了11级，并且智力达到16点时，才能够购买这个兑换。  
伴随着对于异瞳的应用和灵能力对灵魂的反哺不断加深，你的智力达到了前所未有的水平。  
当然，你所拥有的只是超越常人的脑力水平，如果要说到谋略和智力等方面，更加需要的东西也许是知识和经验。不过你现在确实可以用自己的大脑最好地将自身异瞳的能力驱如臂使，以最小的代价去将异瞳的性质进行最大程度的发挥。  
效果：  
你可以消耗一个移动动作，消耗9点灵能力，宣言你进入“脑域演算”状态，持续一个场景。在脑域演算状态下，每经过一回合，你对于当前战斗中，所有敌人的状态，战斗的环境，所有你需要调整的技巧，能够去攻克的弱点，都会得到进一步的了解，因而你可以每回合结束后，选择一个演算效果获得，这些效果的持续时间和“脑域演算”状态相同。  
【演算效果：机先】：运用超级的脑域能力，你可以预判到对手在面对你以霜光凝视时的闪避意图。你进行的“凝视”的攻击具有A级别的【先制】效果。  
【演算效果：查知】：运用超级的脑域能力，你可以精巧地控制你的灵能力，并且将你在攻击时打入对手身体的灵能力当做声呐一样使用，将攻击的反馈进行处理，并且得到情报。受到你进行的“凝视”攻击的单位需要同时进行一次强韧豁免，DC为攻击造成的伤害值。若对方失败则你能够获得其防御构成、豁免构成、移动速度、移动方式。这是一个B级别、信息来源的解析效果  
【演算效果：矫正】：运用超级的脑域能力，你可以紧急地将自身不恰当的行动予以修改，并且得出最优的结果。你进行的“凝视”的攻击时，可以重新投掷一次攻击骰，并且从两次攻击骰的结果中选择更好的一个取用。

守恒突破：假想冰晶  
价格：C+1000  
类型：主动  
描述：霜光所觉醒的力量，同时也是“心胜于物”的直接体现，通过自身的想象，打破物质的守恒，来创造出宇宙中本不存在的质量。当然由于霜光本身的性质，这种创造只被局限于创造冰晶上。不过就算是这样，这种能力也足以担当守护战友的护盾。  
效果：  
你指定自己或者是20米范围内的一名友方单位，可以消耗一个移动动作和6点灵能，发动这个技艺，使其在这一轮内获得等同于你决心点的遮蔽防御，这在画风上视为你召唤出冰晶环绕在他的四周，为他抵挡攻击。并且由于冰晶的环绕，还会使得这个遮蔽防御具有基于其自身的【封闭1】。

异瞳开发：当你获得C级瞳术本体之后，你灵魂所蕴含的更为庞大的灵能力也会使得你以异瞳构筑冰晶的能力更为强大，能够凭空产生更加厚重宽广的冰块。你可以支付D+500进行异瞳能力的开发（这个效果会视为一个D技艺，提供对应的2点内在属性，下同）。这会使得你在释放假想冰晶时可以使效果对象获得基于遮蔽防御的【适应】，并且获得的遮蔽防御提升3点。

异瞳开发II：当你获得B级瞳术本体之后，你可以将异瞳制造冰晶的范围扩大得更为广大，使其保护复数的目标。你可以支付C+1000进行异瞳能力的开发。这会使得“守恒突破：假想冰晶”变为以半径55米的圆形范围内的你指定的单位为目标的群体增益。

异瞳开发III：当你获得B级瞳术本体之后，你可以用更高质量的灵能力制造出更加坚硬厚实的冰晶，用以抵挡抵挡爆炸的冲击波和各种扫射而来的投射物。你可以支付C+1000进行异瞳能力的开发。这会使得你在释放假想冰晶时可以额外支付2点灵能力，使效果对象获得的基于遮蔽防御的【封闭】的点数提升你的传奇决心＊2点，并且额外获得基于遮蔽防御上的1点全伤害吸收。

世界构筑：纯白荒原  
前置：灵能真谛、热力学原理、守恒突破：假想冰晶  
价格：B+2000  
类型：主动  
描述：世界构筑，是在种类繁多的灵能者能力中，居于顶点的能力。将一定范围内的灵魂与物质的边界模糊，将物质的世界拖入自身的精神。用古武中内家练气的术语来形容，这便是“天人合一之境界”。如果用入侵地球的噬灭虫的说法，这是在进入究极的，堪称掌握宇宙真理的神明才会拥有的第十三级灵能之前所必须经过的晋升的阶梯之一。  
以霜光所展开的世界构筑，是一片不断被飘落的纯白光辉覆盖的空无一物的原野，所持有的根本的理念是“荒芜”。即就是“活跃”“繁荣”“演变”等理念的对立面，将这些事物停止的存在。  
要展开世界构筑，所必须的条件是与真实的世界沟通——即就是吟唱“宏大之声”。  
所谓的“宏大之声”并非现存的生物的灵智所能理解的话语，而是更为伟大的意志在现在的宇宙中留下的遗物。因而人类只能以残篇般零散的语句将其吟唱。  
效果：  
展开霜光“世界重构”所需要的“宏大之声”为九章，人物需要以整轮动作将这九个词语说出，并且每说出一个词语，人物都需要支付1点灵能力。九个词语分别为：“银色的沙漏”，“尸骸”，“摇篮”，“失聪”，“缄默的契约”，“灵魂刑场”，“涂抹虚无的底色”，“终点”。  
随即，在咏唱后的下个回合，人物可以一个自由动作展开“世界重构：纯白荒原”，它对于世界的覆写持续一个场景的时间。你同时只能展开一个纯白荒原。  
以人物咏唱时的位置为中心，半径为敏感范围内的区域，将会变成不断飘落纯白光芒的荒芜雪原。在雪原内，移动的概念将被压制和阻止，任何处于荒原内的角色想要在雪原中进行移动时，他们会感觉到自己进行前进之后最终一切变为纯白，然后等到他们的意识又一次从纯白中找回了形体时，他们会发现他们回到了移动前的位置。这视为范围内所有角色受到了一个A级别影响心灵来源的困难地形效果，并且只要处于纯白荒原的范围内，，所有受到困难地形影响的单位将无法使用A级别及以下的传送能力。  
当你处在雪原中时，你的凝视会得到强化，其射程提升至你的敏感范围。当有单位在这一区域内死亡后，这表现荒芜的法则由内而外地剥离他们的生命，将他们的生机飞快地破灭，消散，最终化作粉碎的冰尘，融入苍白的原野。
