# 灵能异瞳：霜光

名称：灵能异瞳：霜光  
类型：瞳术  
本质：魔幻本质

概述：出自《灵气复苏：七日尸潮》中的能力，是能够与统领数万阴尸的“尸王”进行对峙的A级灵能者叶封所持有的“灵能异瞳”。这种异瞳能够以视线为媒介，能够操作冻结，或者凭空凝结出物理法则无法解释的冰晶，位列这个世界当中十大灵能者能力的第五名。  
由于这个世界当中的异瞳，本质上是一种特化了的灵能者能力，所以这个兑换内还包含了一些轮回者应用该瞳术所必须的对灵能力本身的开发。

D级  
等级名称：霜光初开  
价格：D+600  
描述：  
你开启了灵能异瞳：霜光，这会使得你的眼部变为冰蓝色，并且在其中出现宛如冰晶结构的纹理。不过现在你还无法通过调动灵能进行进攻性的“注视”。  
【热能洞察】：  
虽然现在还不足以进行将事物冻结的“凝视”，但是现在的你的眼睛，也对周遭热量的变化洞察无遗。

特性：【灵能力】【热能洞察】  
效果：  
【灵能力】：  
你现在开启一个名为灵能力的能量池，关键属性是感知+决心。并且你获得一个基础用法，你可以在一次决心属性参与的检定上支付一点灵能力，获得2dp的能量加值。

【热能洞察】：  
你的双眼堪比最为先进的热成像仪器。现在你的视觉变为强感知，并且现在你因为对任何攻击引发的热能变化的敏感，视为具有C级别的免疫措手不及。

C级  
等级名称：凝视之力  
价格：C+1200  
描述：你适应了霜光的力量，当你下定决心对于事物进行“凝视”时，超自然的寒冷将会出现，将对手体内的一切“活性”进行冻结。

特性：【霜光凝视】  
效果：  
【霜光凝视】：  
你现在可以在洞察技能上开启一个名为“灵能凝视”的专业。  
以一个标准动作，消耗6点灵能力，你进行一次施法攻击，这次攻击的构成为决心+洞察+专业：灵能凝视+专业：能量爆发，射程为35米和你的敏感范围取低。这次攻击具有你的传奇决心\*3点法术强度，造成冻寒伤害，需要以强韧进行豁免，附带一半的冻结点数。这个攻击方式在本资源中被称为“凝视”攻击。

B级  
等级名称：灵能真谛  
价格：B+2400  
前提：这个级别的本瞳术只有你在获得了下述技能树下的“灵能力成长”和“灵能力精炼”之后才能获得。  
描述：  
灵能力，是自人类的灵魂当中涌现出的力量。在能够吞噬一个星球上的全部生物，并且将正因为有些被地球的意志选中的人类的身体经受住了天地间灵气回潮的洗礼，并且将自身的灵魂留在了体内，所以才得以操纵这股超越常理的力量。  
灵能力并非只是驱使异瞳的工具，而是异瞳的本质，是星球在面对外来的入侵者唤醒的身为智慧生物的人类在心灵中内藏的希望，现在的你的双眼体现了这一点。

特性：【灵能觉醒】  
效果：  
【灵能觉醒】：这一能力只有在你完全掌握一阶基因锁后，主动以迅捷动作主动启动一阶基因锁时才能发动。此时你在一阶基因锁中失追加如下效果：  
如果人物为自己进行的决心属性检定支付意志，可再获得决心/2点增强加值，但也会因此承受4点不可豁免的韧性伤害，这个伤害类型是精神伤害，表现为你为了爆发出更为强大的灵能力导致了脑域和灵魂的损伤。  
每影片一次，当这个场景中有你的友方单位（一般为和你一同度过了一场一场影片以上的轮回者队友）死亡时，并且该队友持有灵能力或其他包含心智属性的能量池时，若你的意志力剩有最大值一半以上，你可以用标准动作消耗所有剩余的意志力，牵引已经逝去的生命所包含的灵魂的力量，突破个体灵魂的极限，进入仅持续一轮的“神之境界”。在你发动这个能力时，如果场景内的角色持有“灵感视觉”的话，会发现逝去者的灵魂涌入你的身体，随即你的身体发散出奇异的生命力的光辉。  
由于“神之境界”对肉体的庞大负担，你只能在“神之境界”中进行一次以“凝视”进行的攻击。  
在这次“凝视”的攻击中你将会冻结对方的“因果”，也就是是说，对方的过去无法抵达现在，同时现在也无法抵达未来，即使想要进行行动，也会发现自己的行动无法产生对应的结果，这会使得这次攻击具有一个S级别的命运来源的【先制】效果。  
随后，你再进行这次“凝视”攻击本身，特殊地，这次凝视不能由强韧豁免，而是直接经过防御，但是由于步入“神之境界”的你所具有的宛如能够摧毁一切的庞大灵能力将物理法则也一并扭曲，无视所谓坚硬和脆弱的区别，所以这次“凝视”攻击能够忽视所有天生防御和盔甲防御，由于依靠对于现有物理法则的破坏达成这一效果，这两个忽视防御的效果均为A级别的时空来源的效果。  
同时，由于你投入这一击中的庞大的灵能力，已经超越了人类这一物种的极限，这次攻击检定的出目还会变为原来的1.5倍，并且最终在计算伤害时，伤害翻倍。  
当你以这次攻击杀死敌人时，首先由于你攻击的庞大威力，这次攻击会将对手的物质结构从细胞层级一个不剩地杀灭，因此死去的目标无法以S级及以下的方式复活，这是一个A级别的创伤来源的禁止复活效果。  
并且由于你在“神之境界”下，获得了短暂的俯瞰所有可能，一切世界线，万事万物的开端与结束的能力，这次攻击还会剥夺对手现在存在的“根基”之后存在的“权利”，因此死去的目标无法以S级及以下的方式复活，这是一个S级别的命运来源的禁止复活效果。  
只有同时免疫了这两个免疫复活效果，对手才能试图在被这踏入“神之境界”的一击杀灭后进行复活。

A级：踏向混沌的海洋  
价格：A+4800  
描述：暂不开放
