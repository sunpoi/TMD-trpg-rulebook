# 金钟罩

  

这是一门佛门功法，使用者若想学习这门功法必须要对佛学拥有一定的认识和了解才能修行此功法。  
（具体表现为学习入门内功需要学识下属佛学1，学习核心内功需要佛学3，学习顶级内功需要佛学5）  
同时此功法也要求修练者不得故意屠戮无辜，贱淫妇女，偷盗抢掠等行为，  
（无意中伤到无辜或恶人拿无辜之人当作人质以及可以对恶人拿走他们的东西不算。）（注:主要是由st判断为大概符合我们本世界善恶观的善或恶）  
否则功法反噬，失去修练此功法的一切技能，且无法修练佛门功法。

  
入门内功:D+600  
解锁一个名为内力的能量池，其上限为耐力+力量。回复此能量槽需要诵经冥想修行一小时，可进行一次关键属性的检定，恢复成功数的內力。 也可以在长休后恢复至满。每日以此法恢复上限不能超过总上限三倍。  
被动技能  
铁布衫:你的筋骨坚实，皮肤受金铁之器刺之犹如皮革一般，全身都能抵抗外力之攻击。获得等于力量上附加成功数的DR/穿刺。

核心内功:C+1200  
被动技能  
你的铁布衫升级为获得力量上每点附加成功的DR/—。  
金钟罩:你的皮肤犹如铜像般的颜色，而身体也犹如铜像般坚不可摧。你的拳头视为伤害7L的天生武器。  
主动技能  
金刚劲:以一个自由动作启动，花费三点内力，持续一个场景或15分钟。你可以让金钟罩的内力凝聚在你的肢体上, 使你的肢体皮肤泛起金色的光芒，在使用肉搏挌档时视为具有\[格挡值6\]特性的天生武器。

顶级内功:B+2400  
被动技能  
金刚明王身:你的身体散发出恐怖的力量，身如山岳，立于大地之上,身体皮肤好像佛像一样，散发金色的光芒，犹如金刚明王一般。  
获得等于你耐力的天生防御。  
金钟罩的效果提升为13L。获得4AP。
