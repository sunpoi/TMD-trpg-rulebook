# 井盖

  

井盖  
D+500  
出自。。。谁TM拿了个井盖过来当盾牌啊！#摔（注：请不要在影片世界偷井盖，大多数井盖都没有本井盖的防御能力）  
重盾 体积4 3L 双手 格挡10  
使用井盖进行格挡时额外获得1点防御上的附加成功。  
注意：在持有本道具时建议不要靠近没有井盖的井，否则你可能会被当成偷井盖的。
