# 暴走轻机枪

  

持久的暴走轻机枪  
（D+500）  
“暴走这逼枪，除了烂成一坨屎的腰射，其他方面都挺不错的”  
暴走作为一款王勇防务（Wonyeon Defense）推出的全新旗舰款机枪，凭借其稳定可靠的性能，深受IMC和边境反抗军的喜爱。而由于其优异的性能，kg博士对其进行了一次改良，以便于其更加持久的作战。至于这次改动是否与宇宙作战者权益保护法冲突，就不在博士的考虑范围内了。  
于是，996作战就此诞生，战士们苦不堪言，军事家们则乐开了花，当然，这也是后话了......  
\--------------------------------------------------------------------------------------------------------------------  
这是一把造成6L伤害的轻机枪（步枪），装弹量为30，基础射程为150，可以进行短点射/长点射/连射，体积为3，使用和自动步枪一样的改造和弹药。  
［过载］：使用迅捷动作，将铝热剂手雷塞入枪身的存储器内中，使枪械过热，此时的枪械额外获得贯穿，和无限子弹和破甲1的特性，你在此状态下的攻击获得3dp的器械加值持续时间一轮  
［加班］在过载状态下，你永远不会感到疲惫，也不需要休息。  
6\*1.25+5

铝热剂手雷  
250分/1枚  
铝热剂手雷最开始是什么人制作出来已经无从考究了，但从它在IMC和边境反抗军中的受欢迎程度中，可以看出这款堪称手雷在投掷物历史上的重要地位。
