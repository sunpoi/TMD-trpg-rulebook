# 死神终临

  

死神终临  
D+1500  
魔幻本质  
“凡品又如何，我自会从平凡走向传奇”  
  
武器类别：狙击枪  
专业：狙击枪  
伤害：8L  
体积：5  
特性：高速6 破甲3 双手  
【我迟早可以杀了你】游走与生死之间的经历，让你愈发能掌握这把武器潜藏的威能  
每经历一场影片，你在使用这把武器进行攻击时获得1点器械加值  
【命定的死亡送于你】锁定猎物，可别让待宰的羔羊跑了  
在使用这把武器时，你获得1AP，这1点AP仅能用于这把武器的瞄准。  
【贯虹的子弹射向你】死亡会格外关注你的敌人，前提是你得有耐心  
如果你在一次攻击之前对一个敌人瞄准了至少三次，那么那次攻击具有【贯穿】，并再获得1点破甲。
