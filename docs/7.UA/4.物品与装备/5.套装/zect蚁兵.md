# ZECT蚁兵

  

ZECT蚁兵  
ZECT蚁兵，是日本特摄剧《假面骑士Kabuto》中登场的量产战士。ZECT为了应对异虫而开发的假面骑士系统以外的量产型设备由ZECT的队员装着而成，以团队作战和支援ZECT所属的假面骑士战斗为主。  
D+500  
分类：轻甲  
科技本质  
盔甲基本属性：盔甲防御5/5  
特性：【防弹】【黑暗视觉】  
Combat Boots：在难以站稳的山林和岩石场地也不容易滑倒，能够保障步行时的安全性。获得D级特性的困难地形免疫  
  
  
  
  
机枪剑  
ZECT蚁兵的通用武器、装备3000发的空尖弹的轻机枪。重量9.8kg。安装于右腕处。白刃战时，尖端展开被内藏了的能切开厚度30mm的铁板的UTS钢铁制的格斗专用刀片。  
  
D+500  
科技本质  
  
机枪剑：每轮可以通过一次花招动作转换机枪模式与剑模式  
机枪状态：  
武器伤害：8L  
速射等级：连射  
伤害类型：穿刺  
效果：高速5 无限子弹 贯穿  
  
剑状态：  
武器伤害：7L  
伤害类型：挥砍  
效果：破甲3  
  
在同时装备两件装备时，盔甲防御提升1，武器两状态都提升1L伤害
